# wiki coverage 台帳

更新日: 2026-03-28

## 目的

この台帳は、`_book/_作成メモ/目次_確定版.md` の細目と、公開 wiki の対応関係を `1細目1行` で管理するためのものです。

今後は、この台帳を基準にして次を判断します。

- どの細目が公開側で未対応か
- どの細目が章ページ止まりか
- どの細目が `知識 / 出題 / 演習 / 要約` まで進んでいるか
- 次にどこを掘ると `完全網羅` に近づくか

## 状態の意味

- `独立ノードあり`: 細目に対して、公開側に独立した着地点がある
- `章ページ+演習あり`: 独立ノードではないが、章ページと演習ページで追える
- `章ページで対応`: 章ページ内で扱っている
- `章内に一部含む`: 周辺論点の中で触れているが、独立した着地点ではない
- `未対応`: 現時点で公開側に明確な着地点が見当たらない

## いますぐ優先する不足

- `第2次試験`: `設問索引` を起点に、`事例Ⅰ〜Ⅳ` の悪文例、改善例、短文骨子を厚くし、`事例Ⅳ` と次年度の行へ広げる
- `横断付録`: `重要用語集` と `重要公式集` は略語、言い換え、事例別入口まで揃ったので、各章ハブ、事例ページ、索引からの逆流を増やす
- `章ページで対応の残り`: `受験ガイド` と `第2次試験 overview` は確認順と時間配分を補強済みなので、今後は必要に応じて索引または個別ノードで補う
