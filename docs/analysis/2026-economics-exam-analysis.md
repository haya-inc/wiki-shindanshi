# 経済学・経済政策 過去問分析：必要な知識と思考法の体系

> 作成日: 2026-03-28
> 分析対象: R2〜R7（2020〜2025）全175マーク（150問＋R7の25マーク詳細）
> 目的: 各テーマで**どのような知識**と**どのような思考法**を身につけるべきかを整理する

---

## 分析フレームワーク

本分析では、過去問を以下の3軸で整理する。

**軸1: 必要な知識の種類**
- **K1 定義・用語**: 概念の正確な定義（例: GDPに何が含まれるか）
- **K2 グラフの形状・動き**: 曲線の傾き、シフト方向、交点の意味
- **K3 数式・公式**: 乗数公式、弾力性の計算式、指数の計算法
- **K4 因果メカニズム**: 「AがBを引き起こし、Bの結果Cになる」という連鎖
- **K5 制度・データ**: 経済白書や統計データの読み方、制度の仕組み

**軸2: 求められる思考法**
- **T1 正誤判定**: 選択肢の記述が定義と合致するか一つずつ検証
- **T2 グラフ読解**: グラフの形状・面積・交点から経済的意味を読み取る
- **T3 計算実行**: 公式に数値を代入して正確に計算する
- **T4 因果推論**: 条件変化→中間変数→結論の多段推論を組み立てる
- **T5 場合分け**: 条件（弾力性の大小、財の種類など）で結論が変わるケースを整理する

**軸3: 罠パターン**
- **Trap-A 逆方向**: シフト方向や効果の正負を逆にした選択肢
- **Trap-B 条件すり替え**: 前提条件（固定相場/変動相場、短期/長期等）を変えた選択肢
- **Trap-C 部分正解**: 途中まで正しく最後だけ間違う選択肢
- **Trap-D 混同誘発**: 似た概念（名目/実質、代替効果/所得効果等）を混同させる選択肢
- **Trap-E 計算ミス誘発**: 分母分子の入れ替え、符号ミスを誘う数値設定

| R3 | 4,5-6,7-8,9,10 | 一時金乗数・45度線・IS-LM（流動性の罠）・貨幣乗数・金融政策 | L2-L4 |
| R4 | 4,5,6-7 | 絶対所得仮説・生産物市場均衡・45度線分析 | L1-L3 |
| R5 | 7,8,9,13-14 | 45度線・IS曲線・LM曲線・国債理論 | L2-L4 |
| R6 | 5,6,7,8,11-12 | 消費関数・貨幣需要・減税乗数・ビルトインスタビライザー・IS-LM | L1-L4 |
| R7 | 7(2問),10(2問) | 45度線（デフレギャップ+租税乗数）・IS-LM（均衡判定+傾きと政策効果） | L2-L4 |

### 必要な知識

**K1 定義・用語**
- 限界消費性向（MPC）、限界貯蓄性向（MPS）、MPC + MPS = 1
- 流動性選好（取引的動機・予備的動機・投機的動機）
- クラウディングアウトの定義と発生メカニズム
- ビルトインスタビライザーの定義（比例税の自動安定化機能）
- 流動性の罠（LM水平）、投資の利子非弾力性（IS垂直）

**K2 グラフの形状・動き**
- **45度線図**: 総需要線の切片（基礎消費+投資+政府支出）と傾き（限界消費性向）
- **デフレギャップ/インフレギャップ**: 完全雇用GDPとの乖離を45度線上で読む
- **IS曲線**: 右下がり。財政拡張→右シフト。傾きはMPCと投資の利子感応度で決まる
- **LM曲線**: 右上がり。金融緩和→右シフト。傾きは貨幣需要の所得感応度と利子感応度の比
- **IS-LMの交点**: 生産物市場と貨幣市場の同時均衡
- **交点以外の領域**: 4象限それぞれで「超過需要/超過供給」を判定する能力（R7第10問設問1）

**K3 数式・公式**
- 政府支出乗数: 1/(1-c) （定額税）、1/(1-c(1-t)) （比例税）
- 租税乗数: -c/(1-c)　※政府支出乗数より絶対値が小さい
- 均衡予算乗数: 1（定額税モデル）
- IS曲線の導出: Y = C(Y-T) + I(r) + G → Yについて解く
- LM曲線の導出: M/P = L(Y, r) → rについて解く

**K4 因果メカニズム**
- 財政政策の波及: G↑ → AD↑ → Y↑ → L(Y)↑ → r↑ → I↓（クラウディングアウト）
- 金融政策の波及: M↑ → r↓ → I↑ → Y↑
- LMが急勾配 → 金融政策が有効、財政政策はクラウディングアウトが大きい
- ISが急勾配 → 財政政策が有効、金融政策の効果は限定的
- **R7で問われた因果**: 「限界消費性向が大きいとIS曲線はどうなるか→なぜGDP拡大効果が変わるか」

### 求められる思考法

**T2 グラフ読解**: IS-LMのグラフで、各点がどの象限（IS右/左 × LM上/下）にあるかを判定し、財市場・貨幣市場それぞれの過不足を読む。R7第10問設問1はまさにこの能力を問うた。

**T4 因果推論**: 「パラメータの変化 → 曲線の傾き/位置の変化 → 政策効果の変化」の3段論法。R7第10問設問2では「投資の利子感応度が大きい → IS曲線が緩やか → 金融政策のGDP拡大効果が大きい」という推論が必要。

**T5 場合分け**: 流動性の罠（LM水平）vs 古典派（LM垂直）で結論が正反対になる。定額税 vs 比例税で乗数が変わる。これらの場合分けを正確に使い分ける。

### 典型的な罠パターン

- **Trap-A**: 「金融緩和でIS曲線が右にシフトする」（正: LM曲線が右シフト）
- **Trap-B**: 定額税モデルの乗数を比例税モデルに適用させる
- **Trap-C**: 「財政拡張→Y増加→r上昇」まで正しいが「→I増加」で逆にする
- **Trap-D**: IS曲線の傾きを決める要因とLM曲線の傾きを決める要因を入れ替える

### 学習の優先度と深さ

**最優先（毎年出る）**: 45度線の均衡GDPの導出、乗数の計算、IS-LMグラフの読解
**次点（2〜3年に1回の深い問い）**: 各象限での超過需給の判定、パラメータ変化と政策有効性の関係、流動性の罠

## 2. 時事統計・データ読解（6年平均2.2マーク → R7で3マークに増加）

### 出題実績
| 年度 | 問番号 | データの種類 | 形式層 |
|---|---|---|---|
| R2 | 1,2 | 政策金利推移、貿易統計 | L2 |
| R3 | 1,2 | 実質GDP国際比較、国債保有者内訳 | L2 |
| R4 | 1,2 | ジニ係数・再分配、実質GDP成長率・寄与度 | L2 |
| R5 | 1,2,3 | GDPシェア、経常収支推移、家計金融資産 | L2 |
| R6 | 1,2,3 | 名目国内総支出内訳、労働生産性、消費者物価 | L2 |
| R7 | 1,2,3 | 一般会計税収推移、男女別雇用者数、対内直接投資残高 | L2 |

### 必要な知識

**K1 定義・用語**
- GDP（名目/実質）、GNI、NNPの違い
- 経常収支の4項目（貿易・サービス収支、第一次所得収支、第二次所得収支）
- 消費者物価指数（CPI）、GDPデフレーター、コアCPIの定義
- 完全失業率の定義（労働力人口に占める完全失業者の比率）
- ジニ係数の意味（0=完全平等、1=完全不平等）

**K5 制度・データ**
- 日本の税収構造: 所得税・法人税・消費税の推移と比率（R7第1問）
- 雇用形態: 正規/非正規の男女別推移（R7第2問）
- 直接投資の業種別構成（R7第3問）
- 経済白書・経済財政白書の主要グラフ
- 主要国のGDP規模・成長率の概算値

### 求められる思考法

**T2 グラフ読解**: 時系列グラフで「どの線がどの系列か」を、数値の大小関係や変曲点の時期から特定する。「消費税は段階的に上昇するステップ関数」「法人税は景気に連動して変動」のような系列の特性を知っているかが鍵。

**T1 正誤判定**: 「2010年以降、Aは増加傾向にある」のような記述を、頭の中のデータイメージと照合する。

### 典型的な罠パターン

- **Trap-A**: グラフの2本の線を入れ替えた選択肢（例: 所得税と法人税の線を逆に）
- **Trap-C**: 「2000年から2020年にかけて一貫して増加」→実際は途中に減少期がある
- **Trap-D**: 「名目GDPは増加」と「実質GDPは増加」を混同させる

### 学習戦略

この分野は**暗記で解く問題ではない**。重要なのは「日本経済の大まかな姿」を知っていること。具体的には：
1. 税収三本柱の大小関係（消費税 ≒ 所得税 > 法人税、近年は消費税が最大）
2. 正規雇用は男性>女性、非正規雇用は女性>男性
3. 経常収支は黒字だが貿易収支は赤字化傾向（第一次所得収支が稼ぎ頭）
4. 日本のGDP世界シェアは低下傾向

**対策法**: 試験直前1〜2ヶ月で、経済財政白書のグラフを中心にざっと眺める。細かい数値ではなく「方向」と「転換点の年」を押さえる。

## 3. 消費者行動・企業行動（6年平均4.3マーク / 毎年必出）

### 出題実績
| 年度 | 問番号 | 具体テーマ | 形式層 |
|---|---|---|---|
| R2 | 13,14,15,16 | 予算制約線・無差別曲線・所得効果代替効果・総生産物曲線 | L1-L3 |
| R3 | 17,18,19,24 | 無差別曲線・予算制約最適消費・所得変化の消費行動・労働供給 | L1-L3 |
| R4 | 17-18,19-20 | 利潤最大化・最適生産・生産要素投入 | L2-L4 |
| R5 | 17,18,19 | 短期費用曲線・生産可能性曲線・エンゲル曲線 | L2-L3 |
| R6 | 15,16,17-18,22-23,24 | 予算制約・等費用線等産出量・費用曲線供給・労働供給・負の所得税 | L2-L4 |
| R7 | 14,15(2問),16 | スルツキー分解・限界費用の性質・MCグラフ面積・独占利潤最大化 | L2-L4 |

### 必要な知識

**K1 定義・用語**
- 正常財（上級財）、劣等財（下級財）、ギッフェン財の定義と判別基準
- 代替効果（常にマイナス方向）と所得効果（財の種類で方向が変わる）
- 限界費用（MC）、平均費用（AC）、平均可変費用（AVC）の定義
- 規模の経済・規模の不経済
- 等産出量曲線（アイソクアント）と等費用線の接点＝費用最小化

**K2 グラフの形状・動き**
- **無差別曲線**: 右下がり・原点に凸・交わらない。限界代替率逓減
- **予算制約線**: 傾き = -Px/Py。所得増加→平行シフト、価格変化→回転
- **スルツキー分解のグラフ**: 価格変化→補償予算線（代替効果）→新予算線（所得効果）の2段階移動
- **MC曲線**: U字型。AC・AVCの最低点を通る
- **MC=AC交点**: 長期均衡価格（ゼロ利潤点）
- **MC=AVC交点**: 操業停止点
- **面積の読み方**: 価格×数量=総収入、MC曲線の下の面積=可変費用、差が利潤（R7第15問設問2）

**K3 数式・公式**
- MRS（限界代替率）= MUx/MUy = Px/Py（最適消費条件）
- MC = dTC/dQ、AC = TC/Q
- MCがACを下回る→ACは低下中、MCがACを上回る→ACは上昇中
- 利潤最大化条件: MR = MC（完全競争ではP = MC）

**K4 因果メカニズム**
- スルツキー分解の論理:
  - 代替効果: 「相対価格が変わった→安い方に代替する」（必ず価格低下した財を増やす方向）
  - 所得効果: 「実質所得が変わった→消費量が変わる」（正常財なら増、劣等財なら減）
  - ギッフェン財: 所得効果が代替効果を上回る劣等財
- 完全競争企業の短期供給曲線 = MC曲線のAVC以上の部分

### 求められる思考法

**T2 グラフ読解**: MCとACの位置関係から利潤の正負を判定する。R7第15問設問2では「グラフの面積として利潤を読む」能力が問われた。価格線とMC曲線に囲まれた面積 = 利潤（または損失）。

**T5 場合分け**: 「この財は正常財か劣等財か」で所得効果の方向が変わるため、スルツキー分解の結論が変わる。R7第14問では上級財のケースが出題されたが、下級財やギッフェン財のケースも出題される可能性がある。

**T4 因果推論**: 「限界費用が一定 → 供給曲線は水平 → 供給の価格弾力性は無限大」のような多段推論（R7第15問設問1）。

### 典型的な罠パターン

- **Trap-A**: 代替効果の方向を逆にする（「価格上昇した財の消費を増やす」）
- **Trap-D**: 所得効果と代替効果を混同させる（特にギッフェン財の議論で）
- **Trap-C**: 「MC < AC のとき」まで正しい条件設定で「ACは上昇中」と誤結論
- **Trap-E**: 面積計算で「総収入-総費用」ではなく「総収入-可変費用」を利潤として計算させる

## 4. AD-AS・フィリップス曲線・国際マクロ（6年平均3.7マーク / 毎年必出）

### 出題実績
| 年度 | 問番号 | 具体テーマ | 形式層 |
|---|---|---|---|
| R2 | 8,9,11,2 | 失業種別・価格賃金硬直性・マンデルフレミング・貿易統計 | L1-L4 |
| R3 | 11,12,13,14 | 輸出入のGDP影響・マンデルフレミング・雇用失業・TFP | L1-L4 |
| R4 | 8-9,10,11,12,23 | AD-AS完全雇用・景気循環・金利平価・自然失業率・資本移動 | L1-L4 |
| R5 | 2,10,11-12 | 経常収支推移・変動為替の円安要因・開放経済マクロ | L2-L4 |
| R6 | 9,10,13 | 為替レート決定・マンデルフレミング・自然失業率仮説 | L1-L3 |
| R7 | 11(2問) | 経常収支黒字要因・為替レートとマーシャル＝ラーナー条件 | L1-L4 |

### 必要な知識

**K1 定義・用語**
- AD曲線（右下がり）、AS曲線（短期は右上がり、長期は垂直）
- スタグフレーション: AS左シフト → 物価↑ かつ GDP↓
- フィリップス曲線: インフレ率と失業率のトレードオフ（短期）。長期は垂直（自然失業率仮説）
- 自然失業率: 構造的失業＋摩擦的失業。需要不足失業を除いた失業率
- マンデル＝フレミングモデルの前提: 小国開放経済、完全資本移動

**K2 グラフの形状・動き**
- **AD-AS図**: ADは右下がり、短期ASは右上がり、長期ASは垂直
- AD右シフト（拡張的政策）→ 短期: 物価↑・GDP↑、長期: 物価↑のみ
- AS左シフト（供給ショック）→ 物価↑・GDP↓（スタグフレーション）
- **フィリップス曲線**: 短期は右下がり、期待インフレ率↑で上方シフト
- **マンデルフレミング**: 変動相場制で金融政策有効/財政政策無効（逆は固定相場制）

**K3 数式・公式**
- マーシャル＝ラーナー条件: |ηX| + |ηM| > 1 で円安が貿易収支を改善（R7第11問設問2）
- 購買力平価: e = P/P*
- 金利平価: i = i* + (eᵉ - e)/e

**K4 因果メカニズム**
- 変動相場制での金融緩和: M↑ → r↓ → 資本流出 → 円安 → NX↑ → Y↑（有効）
- 変動相場制での財政拡張: G↑ → r↑ → 資本流入 → 円高 → NX↓（相殺、無効）
- Jカーブ効果: 短期では円安で貿易収支悪化、中長期で改善

### 求められる思考法

**T4 因果推論**: マンデルフレミングは「政策→金利→資本移動→為替→純輸出→GDP」の5段の因果連鎖を追う必要がある。固定相場/変動相場で結論が正反対になるので、前提条件の確認が必須。

**T5 場合分け**:
- 固定相場 vs 変動相場 → 有効な政策が入れ替わる
- 短期フィリップス曲線 vs 長期フィリップス曲線 → トレードオフの有無
- 完全資本移動 vs 不完全資本移動 → クラウディングアウトの程度

**T3 計算実行**: マーシャル＝ラーナー条件の数値検証（R7第11問設問2）。輸出入の弾力性を与えられ、条件を満たすか判定する。

### 典型的な罠パターン

- **Trap-B**: 固定相場制の結論を変動相場制の問題に適用させる（最頻出の罠）
- **Trap-A**: 「円安で輸出増加」を常に正しいとする（Jカーブ効果の無視）
- **Trap-B**: 短期フィリップス曲線の結論を長期に適用させる
- **Trap-D**: 需要インフレ（AD右シフト）とコストプッシュインフレ（AS左シフト）を混同させる

| R7 | 16,17 | 独占利潤最大化（MR=MC vs 総収入最大化）・外部不経済（死荷重） | L2-L3 |

### 必要な知識

**K1 定義・用語**
- 完全競争の4条件（多数の売り手買い手、同質財、自由参入退出、完全情報）
- 独占: プライスメーカー、MR < P（需要曲線の下にMR曲線）
- 自然独占: 費用逓減産業（AC右下がり）。3つの規制方式（限界費用価格/平均費用価格/二部料金制）
- 外部性: 私的限界費用 vs 社会的限界費用。ピグー税 = 外部不経済の単位当たりコスト
- コースの定理: 取引費用ゼロ→当事者間交渉で効率的配分が達成される
- 公共財: 非競合性＋非排除性。クラブ財（排除可能・非競合）、共有資源（非排除・競合）
- 逆選択（事前の情報の非対称性）vs モラルハザード（事後の情報の非対称性）

**K2 グラフの形状・動き**
- **独占のグラフ**: D（需要）曲線の下にMR曲線。MR=MCの生産量で価格はD曲線上
- **独占の利潤**: (P - AC) × Q の長方形の面積
- **外部不経済のグラフ**: 社会的限界費用（SMC）は私的限界費用（PMC）の上方。死荷重=三角形
- **関税のグラフ**: 国際価格→関税価格の上昇で消費者余剰が4面に分割される（R7第18問）
- **独占の「総収入最大化」vs「利潤最大化」**: 前者はMR=0の点、後者はMR=MCの点（R7第16問）

**K4 因果メカニズム**
- ピグー税導入 → PMCにtを加算 → 新MCがSMCに一致 → 社会的に最適な生産量に
- 自然独占の限界費用価格規制 → P=MC → 赤字発生（ACがMCより高い）→ 補助金が必要
- 逆選択のメカニズム: 情報劣位者が平均品質で価格設定 → 高品質財が退出 → 市場崩壊

### 求められる思考法

**T2 グラフ読解**: 独占のグラフでは「どの面積が何を表すか」を読む能力が重要。R7第16問では「MR=MCの点」と「MR=0の点」を比較して、利潤最大化と総収入最大化の違いを判定する。

**T4 因果推論**: 外部性→市場の失敗→政策介入の3段論法。「ピグー税を課すと、なぜ社会的最適に近づくか」を因果で説明できるか。

**T5 場合分け**: 3つの規制方式の比較（限界費用価格 vs 平均費用価格 vs 二部料金制）はそれぞれの長所短所を整理する。

### 典型的な罠パターン

- **Trap-D**: 「利潤最大化」と「総収入最大化」の混同（R7第16問の核心）
- **Trap-A**: ピグー税で「生産量が増える」と誤導（正: 生産量は減る）
- **Trap-D**: 逆選択とモラルハザードの混同
- **Trap-B**: 自然独占の規制方式で、限界費用価格と平均費用価格の帰結を入れ替え

### 必要な知識

**K1 定義・用語**
- 需要曲線のシフト要因: 所得変化、関連財価格、嗜好、人口
- 供給曲線のシフト要因: 技術進歩、原材料費、補助金・課税
- 代替財（一方の価格↑→他方の需要↑）、補完財（一方の価格↑→他方の需要↓）

**K2 グラフの形状・動き**
- 価格規制（上限価格）→ 超過需要 → 品不足

**K3 数式・公式**
- 弾力性の計算: 点弾力性 = |(dQ/dP) × (P/Q)|
- 弾力性 = 1 のとき総収入最大（MR = 0）
- 弾力性 > 1: 弾力的（価格↓→総収入↑）、弾力性 < 1: 非弾力的（価格↓→総収入↓）

### 求められる思考法

**T1 正誤判定**: R7第12問のように「代替財の価格上昇→需要曲線が右シフト」のような基本命題の正誤を判定する。ここでは4つの記述を一つずつ検証する。

**T5 場合分け**: R7第13問のように弾力性のさまざまなケースを判定する。「弾力性がゼロ→需要曲線は垂直」「弾力性が1超→価格低下で総収入増」「長期は短期より弾力的」「完全競争企業の需要曲線は水平→弾力性無限大」。

### 典型的な罠パターン

- **Trap-A**: 需要曲線の「移動」と「上の移動」の混同（所得変化→曲線シフト、自身の価格変化→曲線上の移動）
- **Trap-D**: 代替財と補完財の効果を逆にする
- **Trap-E**: 弾力性の計算で分母分子を入れ替える

- ナッシュ均衡: 互いに相手の戦略を所与として、自分の戦略を変えるインセンティブがない状態
- 支配戦略: 相手がどの戦略をとっても自分にとって最善の戦略
- 囚人のジレンマ: 支配戦略の組がパレート最適でない
- パレート最適: 誰かの効用を下げずに他の誰かの効用を上げられない状態
- ミニマックス戦略: 最悪のケースを最小化する戦略

- 2×2の利得行列で各プレイヤーの最適反応を矢印で示す

**T5 場合分け**: 同時ゲーム vs 逐次ゲーム、1回ゲーム vs 繰り返しゲームで結論が変わる。囚人のジレンマでも繰り返しゲームなら協力が可能（フォーク定理）。

### 典型的な罠パターン

- **Trap-D**: ナッシュ均衡とパレート最適の混同（囚人のジレンマではこの2つが異なるのが核心）
- **Trap-C**: 支配戦略が存在しないケースで「支配戦略均衡」と答えさせる

- ライフサイクル仮説: 生涯所得を均等に消費する
- 恒常所得仮説: 一時所得の変動には消費を変えない

## 思考法別の鍛え方

### T2 グラフ読解（最重要: 全配点の30〜50%）

**鍛え方**: 教科書のグラフを見ながら、以下を口頭で説明する練習
1. 「この曲線は何を表しているか」
2. 「曲線が右にシフトするのは何が起きたときか」
3. 「この面積は何を意味するか」
4. 「交点は何の均衡を意味するか」

R7では52%がグラフ問題だった。グラフを「見て理解できる」だけでなく、「グラフから情報を読み取って推論する」能力が必要。

### T4 因果推論（次に重要: 全配点の20〜30%）

**鍛え方**: 各テーマの因果連鎖を「→」で書き出す練習
- 金融緩和 → M↑ → r↓ → I↑ → AD↑ → Y↑
- 円安 → 輸出↑ → NX↑ → AD↑ → Y↑（ただしJカーブ）
- ピグー税 → PMC↑ → MC=SMCに → 生産量↓ → 死荷重消滅

3段以上の連鎖を暗記ではなく「なぜそうなるか」で理解すること。

### T5 場合分け（差がつく思考法: 全配点の15〜25%）

**鍛え方**: 「条件が変わると結論が変わる」ケースを対比表にまとめる
- 固定相場 vs 変動相場 → 有効な政策が逆転
- 正常財 vs 劣等財 → 所得効果の方向が逆転
- 短期 vs 長期 → フィリップス曲線の形状が変化
- IS急勾配 vs LM急勾配 → 有効な政策が変化

## 罠パターン対策チェックリスト

試験本番で使えるセルフチェック:

1. **「シフト方向は合っているか」**: 右と左、上と下を一度立ち止まって確認
2. **「前提条件は何か」**: 固定/変動、短期/長期、定額税/比例税の確認
3. **「似た概念を取り違えていないか」**: 名目/実質、代替効果/所得効果、利潤最大化/総収入最大化
4. **「計算の分子分母は正しいか」**: 弾力性、物価指数、乗数公式
5. **「因果の最後の一段は正しいか」**: 途中まで合っていて最後だけ逆にされるパターン
