## 1. 完全に欠落している論点（35個）

### 企業行動（生産・費用）の深い理解に関わる論点（9個）

#### 1.1 等量曲線（等産出量曲線）と等費用線の詳細
- **Reference 記載箇所**：1.3 企業行動と供給曲線
- **欠落内容**：
  - 等量曲線の定義と形状（原点に凸）
  - 技術的限界代替率（MRTS）の定義：MRTS = ΔK/ΔL = MPL/MPK
  - 等費用線の式：TC = wL + rK、傾き：-w/r
  - 費用最小化条件：MRTS = w/r
  - 各要素の1円あたり限界生産物の等値条件：MPL/w = MPK/r
- **出題傾向**：計算問題、図表問題で頻出
- **重要度**：極めて高い

#### 1.2 コブ・ダグラス型生産関数
- **Reference 記載**：Q = A × L^α × K^β、規模に関する収穫性分類
- **欠落内容**：産出弾力性（α, β）の意味、規模性の判定（α+β の値）
- **出題傾向**：計算問題、規模の経済性との関連
- **重要度**：高い

#### 1.3 限界生産物逓減の法則と MP曲線・AP曲線の関係
- **Reference 記載**：MP > AP → AP上昇、MP = AP → 最大、MP < AP → 下降
- **欠落内容**：具体的な曲線形状、位置関係の詳細説明
- **出題傾向**：図表問題、正誤判定
- **重要度**：高い

#### 1.4 短期費用構造の詳細（AFC, VC, TC の区分と関係）
- **Reference 記載**：AC = AFC + AVC、MC曲線の通過順序（AVC最低点 → AC最低点）
- **欠落内容**：
  - MC曲線が通る順序の**理由説明**（AFC逓減のため）
  - 総収入・総費用曲線による利潤最大化の幾何学的判定
- **出題傾向**：図表問題で非常に頻出
- **重要度**：極めて高い

#### 1.5 長期平均費用曲線（LAC）と規模の経済性
- **Reference 記載**：LAC の定義（短期AC群の包絡線）、スケールメリット、最適規模、規模の不経済性
- **欠落内容**：LACの形状とその意味、原因の詳細説明
- **出題傾向**：自然独占論、産業組織論での頻出
- **重要度**：高い

#### 1.6 範囲の経済性（Scope Economies）
- **Reference 記載**：複数製品を同一企業で生産する方が費用が低い、C(X₁, X₂) < C(X₁, 0) + C(0, X₂)
- **欠落内容**：定義と計算方法
- **出題傾向**：高度な問題

#### 1.7 供給の価格弾力性
