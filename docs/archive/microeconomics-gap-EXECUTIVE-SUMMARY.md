## 最重要 優先度 S（7 個）

直ちに補足すべき論点：

1. **等量曲線と等費用線の詳細**
   - 見出しはあるが内容が空の状態
   - MRTS = MPL/MPK、費用最小化条件 MRTS = w/r を追加

2. **MC曲線が通る順序とその理由**
   - 事実は知られるが、理由説明が欠落
   - AC = AVC + AFC であり、AFC が逓減するため、AVC 最低点 < AC 最低点

3. **スルツキー分解 vs ヒックス分解の明確な区別**
   - Wiki では分解の説明が浅く、ヒックス分解は完全欠落
   - 補助線の位置による違いを詳細化

4. **損益分岐点と操業停止点の経済的意味**
   - 短期継続条件 P ≧ AVC の重要性を強調

5. **自然独占と価格規制の 3 つの方法**
   - 完全に新規追加が必要
   - 限界費用価格規制、平均費用価格規制、二部料金制の比較表

6. **独占市場の死荷重の図形化**
   - 死荷重が「Qm〜Qc の範囲で需要曲線と MC に挟まれた三角形」であることを図で明示

7. **関税の死荷重（2 つの三角形）**
   - 死荷重が 2 つの三角形であることを図解
