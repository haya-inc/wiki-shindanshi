## 主要発見

### 最も重要な欠落（優先度 S）

| 論点 | 理由 |
|------|------|
| 自然独占と価格規制 3 方法 | 診断士試験で毎年出題の可能性が高い政策問題の中核論点 |
| 独占市場の死荷重の図形化 | 図表問題で頻出、三角形による図解が必須 |
| スルツキー分解 vs ヒックス分解 | グラフ問題で頻出、補助線の位置の理解が必須 |
| 等量曲線と等費用線の詳細 | 見出しはあるが内容が空（= 見落とし） |
| MC曲線の通過順序の理由 | AFC 逓減のメカニズムが説明されていない |
| 損益分岐点と操業停止点 | 短期継続条件 P ≧ AVC の重要性が浅い |
| 関税の死荷重（2 つの三角形） | 図表問題で頻出 |

### 浅い論点の共通課題

- **計算パターンの欠落**：式は説明されるが、具体的な数値計算例がない
- **グラフの詳細化不足**：概念は説明されるが、形状や位置関係の細部が不足
- **表形式による体系化の不足**：複数項目の比較表がない

### Wiki の強みとして保つべき点

- **基本的なフレームワーク**の整理（9 つのメインページ + ハブ）
- **ナッシュ均衡と支配戦略の正確な区別**
- **比較優位の説明**（絶対優位との区別を強調）
- **グラフ・比較問題の型分類**（6 つの型が明確）
