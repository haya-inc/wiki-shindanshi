主要発見
================================================================================

【最も重要な欠落（優先度S）】
1. 自然独占と価格規制3方法 → 診断士試験で毎年出題の可能性が高い
2. 独占市場の死荷重の図形化 → 図表問題で頻出
3. スルツキー分解 vs ヒックス分解 → グラフ問題で頻出、補助線の理解が必須
4. 等量曲線と等費用線の詳細 → 見出しはあるが内容が空（見落とし）
5. MC曲線の通過順序の理由 → AFC逓減のメカニズムが説明されていない
6. 損益分岐点と操業停止点 → 短期継続条件P≧AVCが浅い
7. 関税の死荷重（2つの三角形） → 図表問題で頻出

【Wiki の強み】
- 基本的なフレームワークがしっかり整理
- ナッシュ均衡と支配戦略の区別が正確
- 比較優位の説明が的確
- グラフ・比較問題の型分類が明確

【Wiki の課題】
- 計算手順が大きく不足（弾力性、余剰、機会費用）
- 曲線の詳細な形状説明が浅い
- 市場の失敗と政策の具体的メカニズムが不足
- 寡占理論の詳細モデルが完全に欠落
