## 購買管理 ── 資材の調達

購買管理は、必要な資材を適切な品質・価格・納期で調達する活動です。

### 購買管理の 5 原則

| 原則 | 内容 |
| --- | --- |
| 適正な品質 | 要求仕様を満たす品質 |
| 適正な数量 | 不足も過剰もない量 |
| 適正な時期 | 生産計画に合ったタイミング |
| 適正な価格 | 品質・納期を踏まえた妥当な価格 |
| 適正な供給者 | 信頼できる取引先 |

#### 各原則の詳細

**適正な品質（品質）**
最安値の資材が常に最適とは限りません。不良率が高い安い部品を使えば、納入後の検査コスト、不良による交換対応、場合によっては製品の品質問題が発生します。調達時点で「品質仕様」を明確に指定し、受入検査で確認することが不可欠です。

**適正な数量（需給計画の精度）**
少なすぎれば生産ラインが停止し、多すぎれば過剰在庫になり、在庫管理コストや陳腐化リスクが高まります。生産計画の需要予測の精度を高め、在庫水準を最適化する必要があります。

**適正な時期（リードタイム管理）**
納期遅延は生産スケジュール全体に影響します。調達先のリードタイムを把握し、生産計画から逆算して発注する「MRP（資材所要量計画）」思考が重要です。

**適正な価格（トータルコスト最適化）**
単価だけでなく、輸送費、在庫管理費、受入検査費、品質問題の修正コストなど、総所有コスト（TCO）で判断する必要があります。見かけの価格が安くても、納期遅延による外注増加や品質問題が多ければ、実際のコストは高くなります。

**適正な供給者（継続性と安定性）**
単一供給源依存はサプライチェーンリスクになります。一方、複数調達は管理コストが増えます。調達先の経営安定性、品質管理体制、技術力を総合評価し、戦略的に選定・維持することが重要です。

### 集中購買 vs 分散購買

| 比較軸 | 集中購買 | 分散購買 |
| --- | --- | --- |
| 発注の主体 | 本部が一括 | 各工場・拠点が個別 |
| メリット | 大量発注で価格交渉力向上 | 現場ニーズに即応 |
| デメリット | 現場の個別対応が遅い | 重複発注、管理コスト増 |
| 向く場面 | 共通資材、標準品 | 特殊資材、緊急調達 |

#### 集中・分散の判断フロー

```
調達する資材は何か？
  ↓
【共通部品・標準品】（複数の工場で使える）
  → 集中購買（本部一括）で大量発注→価格交渉力UP、単価低減
  ↓
【工場特有の専用部品】
  → 分散購買可能だが、同じ部品なら集中の方が有利
  ↓
【急な増産対応や仕様変更】
  → 分散購買で即応（柔軟性重視、コスト二の次）
  ↓
【特殊加工や小ロット品】
  → 分散購買（現場との密な調整が必要）
```

**実例**：自動車メーカーは「共通部品（ボルト、ネジ）は本部集中購買」「専用部品は各工場で調達」という使い分けをします。

### 調達先選定の評価基準

| 基準 | 見るポイント |
| --- | --- |
| 品質 | 不良率、品質管理体制、認証 |
| コスト | 単価、輸送費、トータルコスト |
| 納期 | リードタイム、遵守率 |
| 柔軟性 | 急な増減への対応力 |
| 経営安定性 | 財務状況、事業継続リスク |

## 外注管理 ── 工程・作業の委託

外注管理は、自社の工程や作業の一部を外部に委託し、品質・納期・コストを管理する活動です。

### 外注の種類

| 種類 | 内容 | 例 |
| --- | --- | --- |
| 能力外注 | 自社の能力不足を補う | 繁忙期の増産対応 |
| 専門外注 | 自社にない技術を活用 | 特殊加工、めっき処理 |

## 内製 vs 外注の判断

### 判断基準

| 判断軸 | 内製が有利 | 外注が有利 |
| --- | --- | --- |
| コスト | 固定費の活用（余力あり） | 変動費化（需要変動対応） |
| コア技術 | 自社のコア技術 → 内製維持 | 非コア工程 → 外注可 |
| 品質管理 | 厳密な管理が必要 | 標準的な品質で足りる |
| 需要変動 | 安定需要 | 変動が大きい |
| 設備投資 | 既に設備がある | 新規投資が必要 |

### 内製・外注の損益分岐点と TCO（総所有コスト）

内製と外注のどちらが安いかは、生産量によって変わります。また、単価だけでなく輸送費・検査費・在庫管理費・品質問題への対応費まで含めた「総所有コスト（Total Cost of Ownership）」で判断する必要があります。

#### 基本的な損益分岐点計算

```
【考え方】
  内製コスト = 固定費 + 変動費 × 数量
  外注コスト = 外注単価 × 数量

  損益分岐点 = 固定費 ÷ (外注単価 − 変動費)

【計算例①：基本パターン】
  内製: 固定費 500,000円、変動費 300円/個
  外注: 800円/個

  損益分岐点 = 500,000 ÷ (800−300) = 500,000 ÷ 500 = 1,000個

→ 1,000個未満なら外注が安い、1,000個以上なら内製が安い
→ 1,500個の場合、外注コスト 1,200,000円（1,500 × 800）
              内製コスト 950,000円（500,000 + 300 × 1,500）
              内製が 250,000円安い
```

#### TCO（総所有コスト）を含めた判断

損益分岐点で内製が有利でも、外注先の選定時には「単価」だけでは不十分です。

| 費用項目 | 内容 | 見落としやすい例 |
| --- | --- | --- |
| 購入価格 | 部品の単価 | 最安値業者を選んだが… |
| 輸送・梱包費 | 調達地から工場までのコスト | 海外調達の送料が単価を上回る |
| 検査・受入コスト | 受け入れ時の検査・データ確認 | 不良率 5% の業者は本当に安いか |
| 在庫管理費 | 保管、金利、陳腐化リスク | リードタイムが長い→在庫が増える |
| 品質不良対応コスト | 納入後の不良対応、顧客対応 | 数件の納期遅延で全体スケジュール崩壊 |
| リードタイム | 調達期間、問い合わせ対応速度 | 急な増減対応が遅い = 緊急手配コスト |

**実例**：一見 500円安い海外メーカーを選んだが、輸送費 300円、検査コスト 100円、不良率高で対応費 200円。実際のコストは国内業者より高くなった。

#### 計算例②：外注単価変動時のシナリオ分析

外注先の単価が交渉によって変わった場合、損益分岐点もどう変わるか？

```
【基本想定】
  内製: 固定費 500,000円、変動費 300円/個

【シナリオ A：外注単価 800円/個（現行）】
  損益分岐点 = 500,000 ÷ (800−300) = 1,000個

【シナリオ B：外注単価を 700円に交渉成功】
  損益分岐点 = 500,000 ÷ (700−300) = 500,000 ÷ 400 = 1,250個
  → 分岐点が上がる（内製の有利性が減る）

【シナリオ C：外注単価が 900円に上昇】
  損益分岐点 = 500,000 ÷ (900−300) = 500,000 ÷ 600 = 833個
  → 分岐点が下がる（少量で内製有利になる）

結論：月産 1,000個の場合
  A: 内製コスト 800,000円 vs 外注 800,000円 → ほぼ同等
  B: 内製コスト 800,000円 vs 外注 700,000円 → 外注有利
  C: 内製コスト 800,000円 vs 外注 900,000円 → 内製有利
```

この分析を「感度分析」と呼び、外注先交渉や需要予測の不確実性を判断するときに活用します。

## 外注管理のポイント

### 外注先の管理項目

| 管理項目 | 内容 |
| --- | --- |
| 品質管理 | 受入検査、工程監査、品質データの共有 |
| 納期管理 | 進捗確認、リードタイム管理 |
| コスト管理 | 見積もり精度、コスト低減活動 |
| 技術管理 | 図面・仕様書の明確化、技術指導 |

### 外注管理のリスク管理

外注に委託する際には、以下のリスクに対処する仕組みが必要です。

#### 品質リスク
- **受け入れ検査の徹底**：納入時に全数または抜き取り検査を実施。不良率が高い場合は改善計画を要求。
- **工程監査**：外注先の製造工程に立ち入り、品質管理体制を確認。定期的な監査予定を設定。
- **品質データ共有**：不良内容、改善状況の報告を月次で確認。継続的な改善（PDCA）を促進。

#### 納期リスク
- **複社購買**：1社に依存せず、複数の外注先を確保。一社が納期遅延しても対応可能にする。
- **進捗確認**：発注後の工程進捗を定期確認（週次 or 月次）。遅延の早期発見と対応。
- **リードタイム管理**：外注先の能力（月産能力、リードタイム）を把握し、生産計画に反映。

#### 技術・知財リスク
- **コア技術の保護**：自社のコア技術は内製維持。外注するのはコモディティ化した工程のみ。
- **図面・仕様書の管理**：提供する図面に「機密」「目的外利用禁止」などの明記。不用意な流出防止。
- **技術指導**：外注先の技術力が不十分な場合、社員を派遣して直接指導。共同開発のような信頼関係を構築。

#### 経営リスク
- **経営安定性確認**：定期的に外注先の財務状況、組織体制を確認。倒産や事業廃止に備える。
- **BCP（事業継続計画）**：震災やパンデミック時に外注先が操業不能になった場合の代替先を確保。

### 下請法の概要と親事業者の義務

「下請代金支払遅延等防止法」（下請法）は、親事業者が下請事業者を不当に扱うことを禁止する法律です。製造委託や加工委託、修理委託が対象になります。

#### 適用範囲（資本金による区分）

| 取引形態 | 親事業者 | 下請事業者 |
| --- | --- | --- |
| 製造委託・加工委託 | 資本金 3 億円超 | 資本金 3 億円以下 |
| 製造委託・加工委託 | 資本金 1,000 万円超〜3 億円以下 | 資本金 1,000 万円以下 |
| 運送・個別配送委託 | 資本金 1 億円超 | 資本金 1 億円以下 |
| 情報成果物作成委託 | 資本金 5,000 万円超 | 資本金 5,000 万円以下 |

「製造委託」とは、カスタマイズ製品や部品の製造を外注すること。建設業や物品レンタルなど一部業種は対象外です。

#### 親事業者の 4 つの主要な義務

**① 書面交付義務**
- 発注時に「発注書」を交付。口頭発注は禁止。
- 記載内容：品名、数量、納期、代金、支払日、支払方法など。

**② 支払期限と支払遅延禁止**
- 代金支払期限は「納入日から 60 日以内」。
- この期日に全額支払わなければならない（一部でも未払いは違反）。

**③ 支払遅延時の遅延利息**
- 万が一支払が遅れた場合、年利 14.6%（日歩 0.04%）の遅延利息を付ける義務。
- 「やむを得ない事由」がない限り遅延利息の免除はできない。

**④ 書類保存義務**
- 発注書、納品書、請求書などを 2 年間保存。
- 下請法違反の立証時に必要。

#### 禁止行為（11 の代表例）

| 禁止行為 | 具体例 |
| --- | --- |
| 買いたたき | 市場相場より著しく安い代金を強要 |
| 手形長期化 | 支払期限を 60 日より延長 |
| 一方的なキャンセル | 納品予定日直前にキャンセル（正当理由なし） |
| 代金減額 | 納入後に理由なく代金をカット |
| 返品強要 | 不具合でないのに返品させ、修理費を請求 |
| 買い叩き的値引き | 他社との競争を名目に著しい値下げ要求 |
| 納入者負担による加工 | 必要な加工費を下請に負担させる |
| 検査費用の押し付け | 自社の検査費を下請に負担させる |
| 受入遅延 | 納品受け取りを長期間遅延させる |
| 原材料の強制供給 | 特定メーカーの原材料のみ使用を強要 |
| 報告義務の強要 | 過度な経営情報の開示要求 |

**注意**：下請法違反は中小企業庁が指導・勧告の対象となり、悪質な場合は改善命令や課徴金（売上額の 0.5～3%）が課される可能性があります。詳細は [経営法務](/business-law) を参照。

## 損益分岐点グラフの読み方

内製と外注のコストを視覚的に理解するため、グラフで表現します。

```
コスト
  ↑
  │     内製コスト（直線：固定費500k + 変動費300×数量）
  │    /
800k │   /
  │  /
  │ / 外注コスト（直線：800×数量）
  │/━━━━━ 分岐点 1,000個
  │
  0├────────────────→ 生産数量
    0   500  1000  1500  2000

【読み方】
  - 分岐点より左（1,000個未満）：外注コストが下（外注有利）
  - 分岐点より右（1,000個以上）：内製コストが下（内製有利）
  - 感度分析で外注単価が変わると、右側の直線の傾きが変わり、
    分岐点の位置が左右に動く
```

## 典型的なつまずき

- `購買` と `外注` を同じ調達活動として一括で覚える → 購買は **モノ**（資材）、外注は **コト**（工程・作業）
- 最安値の調達先が常に最適だと考える → **品質、納期、安定供給** を含むトータル評価が必要
- 外注先管理を価格交渉だけだと思ってしまう → 品質、納期、技術の **継続的な管理** が重要
- 内製・外注の判断をコストだけで行う → **コア技術の保持** は長期的に重要
- 損益分岐点を超えれば必ず内製が良いと判断する → 設備投資の回収期間や需要の不確実性も考慮
- 下請法を親会社に適用される法律だけと考える → 自社が下請になることもあり、自社が守る側に回ることもある

## 問題を解くときの観点

- いま問われているのは資材（購買）か、工程（外注）か
- 重いのは `価格` か `品質` か `納期` か
- 内製維持と委託活用のどちらが自然か
- コスト比較だけでなく戦略的判断を求められていないか

## 確認問題

<Callout title="問1：内製・外注の損益分岐点">
    内製の固定費 600,000円、変動費 200円/個。外注単価 500円/個。損益分岐点を求め、月産 1,500個の場合どちらが有利か答えよ。

    **解答**：損益分岐点 = 600,000÷(500−200) = 2,000個。1,500個 < 2,000個なので外注が有利（外注750,000円 vs 内製900,000円）。
</Callout>

<Callout title="問2：集中購買と分散購買">
    「複数の工場で共通に使用する標準ボルトの調達コストを下げたい」場合、集中購買と分散購買のどちらが適切か。

    **解答**：集中購買。共通資材・標準品は本部で一括発注することで、大量発注による価格交渉力を発揮できる。
</Callout>

<Callout title="問3：外注の種類">
    「自社では対応できない特殊な表面処理を外部の専門業者に委託する」のは、能力外注・専門外注のどちらか。

    **解答**：専門外注。自社にない技術を活用するための外注。能力外注は自社の処理能力不足を補うもの（繁忙期の増産対応など）。
</Callout>
