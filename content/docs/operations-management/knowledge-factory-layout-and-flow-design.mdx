## まずイメージをつかむ

想像してみてください。

**コンビニの棚配置と工場レイアウトの類似性**

コンビニの弁当売り場：
- 商品（弁当）はお客さんの手に取りやすいように、**通路に沿って一直線**に並んでいます
- 扱う弁当の種類は多く変わりますが、**棚そのものはいつも同じ位置**です

工場の**製品別レイアウト**：
- 製品は生産工程の流れに沿って、**一直線に工程を通る**
- 少品種を大量に作るため、ラインは製品ごとに固定です

一方、**修理工場や町工場**：
- 色々な修理品が持ち込まれ、それぞれ異なる作業（旋盤、溶接、検査）を必要とします
- 修理品は「**必要な工程がある設備のところへ、その都度移動**」します

これが工程別レイアウトの考え方です。

この「品種・量」と「配置戦略」の関係が、工場レイアウト設計の中核です。

## レイアウト類型の比較

| 類型 | 配置の考え方 | 向く生産 | メリット | デメリット |
| --- | --- | --- | --- | --- |
| 製品別レイアウト | 製品の流れに沿って設備配置 | 少品種大量 | 流れがスムーズ、搬送距離短 | 品種変更に弱い |
| 工程別レイアウト（機能別） | 同種の設備をまとめて配置 | 多品種少量 | 品種変更に強い | 搬送距離長、仕掛品増加 |
| 固定式レイアウト | 製品を固定し設備・人が移動 | 大型・重量物 | 製品移動不要 | 作業者の移動が多い |
| セル生産 | U字型などに多工程を集約 | 多品種小ロット | 柔軟性＋流れの両立 | 多能工が必要 |

### 製品別 vs 工程別の判断基準

```
品種が少なく量が多い → 製品別レイアウト（ライン）
品種が多く量が少ない → 工程別レイアウト（ジョブショップ）
中間 → セル生産（グループテクノロジー）
```

### レイアウト選択の判断フローチャート

以下のツリーに沿って、対象工場のレイアウトを判断します。

```
┌────────────────────────────────┐
│  P-Q分析を実施                 │
│  ・生産量の分布を確認          │
│  ・品種数を確認                │
└────────┬─────────────────────┘
         │
         ↓
   ┌─────────────────────┐
   │ 大量品の集中度は？  │
   └─────────────────────┘
    /       |       \
   /        |        \
(高)       (中)     (低)
 /          |        \
│           │         │
↓           ↓         ↓
┌─┐      ┌──┐      ┌──┐
│製│  確認│セ│   工│工│
│品│→ →│ル│ →│程│
│別│②の│生│③ │別│
│→│質問│産│の│→
└─┘│   ├──┤要│
   │   │要素│└──┘
   │   └──┘
   ↓      ↓

② 搬送距離が重要か？＆品種切替の頻度は？
   （YES, YES）→ 製品別（流れ重視）
   （NO, YES）→ セル生産（柔軟性+効率）
   （YES, NO）→ 製品別（ラインは動かない）

③ セル生産の採用条件は全て満たされるか？
   ✓ 類似工程グループが存在
   ✓ 多能工育成体制がある
   ✓ グループ当たり月 100 個以上の量
   （YES）→ セル生産
   （NO）→ 工程別レイアウト
```

**判断のコツ**

| 判断項目 | 製品別へ傾く | 工程別へ傾く |
| --- | --- | --- |
| 品種数 | 少ない（1～3 種） | 多い（10 種以上） |
| 生産量 | 品種当たり 1,000 個/月以上 | 品種当たり 100 個/月以下 |
| 工程順序 | ほぼ全品同じ順序 | 品種によって異なる |
| 搬送距離への関心 | 高（コスト削減が重要） | 低（多少の搬送は許容） |
| 設備の専用度 | 特定製品向け専用設備 | 多品種対応の汎用設備 |
| 柔軟性への要求 | 低 | 高 |

## P-Q分析 ── レイアウト選択の前提整理

P-Q分析は、製品（Product）ごとの生産数量（Quantity）を棒グラフで可視化し、レイアウト方式を決める基礎データにする手法です。

### P-Q分析の手順と数値例

まず、以下の具体例で流れを押さえましょう。

**例：精密機械部品メーカーの月間生産実績**

| 製品 | 月産数量（個） |
| --- | --- |
| 製品A（ギア）| 1,000 |
| 製品B（シャフト） | 500 |
| 製品C（スプロケット） | 50 |
| 製品D（カムシャフト） | 30 |
| **合計** | **1,580** |

**P-Q分析の 3 ステップ**

```
① 生産量で降順にソート（完了）
② 累積比率を計算

   製品A: 1,000 / 1,580 = 63.3%
   製品A + B: 1,500 / 1,580 = 94.9%
   製品A + B + C: 1,550 / 1,580 = 98.1%

③ 棒グラフ化＆グループ分け

   左端（0～70%）：製品A → 大量単一品 → 製品別レイアウト
   中間（70～95%）：製品B → 準大量品 → セル生産/グループテク
   右端（95%～）：製品C/D → 少量品 → 工程別レイアウト
```

### P-Q分析の読み方と判断

$$
\text{P-Q分析の判断基準} = \begin{cases}
\text{製品別レイアウト} & \text{左端に少数の大量品が集中} \\
\text{工程別レイアウト} & \text{広く分散して右端に少量品が多数} \\
\text{セル生産・GT} & \text{左端と右端の両方が存在}
\end{cases}
$$

**なぜこれで判断できるのか**

- **大量品**が存在 → その品目のために **製品別ライン**を組む投資が回収できる
- **少量品**が多数 → 工程ごとに設備をまとめた **工程別レイアウト**で柔軟に対応すべき
- **両方ある** → 大量品はラインで、少量品は工程別で処理 → セル生産で折り合いをつける

## SLP（Systematic Layout Planning）

SLP はミューサーが提唱した体系的なレイアウト設計手法です。部門間の関連性を評価し、近接させるべき部門を決めます。

### SLP の全体像

SLP は以下の 6 ステップを順序通りに進めます。重要なのは、**P-Q分析で大方向を決めたあと、部門間の物の流れと関係性を細かく分析する** ことです。

```
① P-Q分析 → レイアウト方式の方向づけ（少品種大量か多品種少量か）
② 物の流れ分析（From-To チャート）→ どの部門間で物が移動するか定量化
③ アクティビティ相互関係図（近接性の評価）→ A/E/I/O/U/X で評価
④ 関係線図（スペース考慮なし）→ 近接度を線で結ぶ
⑤ スペース相互関係図（面積を加味）→ 実際の部屋の広さも考慮
⑥ レイアウト代替案の作成と評価→ 複数案を試して比較
```

### 具体例：町工場の引っ越しプロジェクト

**現状**：親方が経営する自動車部品工場。現在の工場が手狭になり、新しい敷地への移転を検討。部門は以下の 5 つ：

| 部門 | 説明 |
| --- | --- |
| 旋盤工程（Lathe） | 金属を削って形成 |
| 研磨・仕上げ工程（Grinding） | 旋盤後の精密仕上げ |
| 溶接工程（Welding） | 部品を溶接で接合 |
| 検査・組立工程（Inspection） | 精密検査と最終組立 |
| 仕上げ塗装（Painting） | 完成品の塗装 |

**ステップ① P-Q分析**

P-Q分析の結果：大量品 2 種（自動変速機部品A、同B）と、少量特注品 3 種。
→ 判断：大量品 2 種は製品別ラインを用意し、特注品は工程別で対応。

**ステップ② From-To チャート（物の流れ）**

月間の部品移動数（単位：パレット）：

| From | To | 量 |
| --- | --- | --- |
| 旋盤 | 研磨 | 150 |
| 研磨 | 溶接 | 90 |
| 溶接 | 検査 | 80 |
| 検査 | 塗装 | 70 |
| 旋盤 | 検査（特注品直行） | 40 |
| 旋盤 | 塗装（部品による） | 20 |

→ **最も流れが多い**：旋盤→研磨→溶接→検査→塗装

**ステップ③ アクティビティ相互関係図（近接性評価）**

各部門の関係性を A/E/I/O/U/X で評価：

| 部門 1 | 部門 2 | 評価 | 理由 |
| --- | --- | --- | --- |
| 旋盤 | 研磨 | **A**（絶対） | 旋盤→研磨の物の流れが最大（150）。工程順 |
| 研磨 | 溶接 | **E**（重要） | 研磨→溶接が 90。ただし工程間距離は許容 |
| 溶接 | 検査 | **A**（絶対） | 溶接欠陥は即検査が必須。品質リスク高 |
| 検査 | 塗装 | **I**（重要） | 検査合格品のみ塗装。流れは 70 |
| 溶接 | 塗装 | **X**（望ましくない） | 溶接の高温・飛沫と塗装の清潔環境が相反 |
| 旋盤 | 溶接 | **U**（不要） | ほぼ直接流れなし。研磨を経由 |

**ステップ④⑤ 関係線図とスペース配分**

近接度に基づいて、配置案を作成：

```
┌─────────────┬─────────────┐
│   旋盤      │   研磨      │
│   (30坪)    │   (20坪)    │
├─────────────┴─────────────┤
│   溶接      │   検査      │
│   (40坪)    │   (25坪)    │
└─────────────┬─────────────┘
              │
        ┌─────┴──────┐
        │   塗装     │
        │   (20坪)   │
        └────────────┘

配置の工夫：
- 旋盤と研磨を隣接（A評価）
- 溶接と検査を隣接（A評価）
- 溶接と塗装は廊下を挟んで対角配置（X評価回避）
```

**ステップ⑥ 評価と比較**

この案のメリット・デメリット：

| 評価項目 | スコア | 根拠 |
| --- | --- | --- |
| 搬送距離の短さ | ◎ | 主流（旋盤→研磨→溶接→検査→塗装）がほぼ直線 |
| 品質管理 | ◎ | 溶接直後に検査できる配置 |
| 柔軟性 | ○ | 特注品の旋盤→検査（直行）も対応可能 |
| 騒音・安全性 | ◎ | 溶接と塗装が離れている |
| 総合評価 | 採用 | 代替案との比較で最適 |

### 近接性の評価基準

| 記号 | 意味 | 判断 | 例 |
| --- | --- | --- | --- |
| A | 絶対的に近接 | 必ず隣接させる | 旋盤→研磨（流れが多い）、溶接→検査（品質必須） |
| E | 特に重要 | できるだけ近く | 検査→梱包（直後の処理） |
| I | 重要 | 近い方がよい | 研磨→溶接（工程順） |
| O | 普通 | 特に制約なし | 材料置き場と工程群 |
| U | 不要 | 離れていてもよい | 旋盤と塗装（ほぼ流れなし） |
| X | 望ましくない | 離す | 溶接（高温）と塗装（精密）、危険物と事務室 |

### SLP での注意点

- **流れが多い**だけでなく、**品質・安全の視点**も A/E 評価に含める
- **代替案を複数作って比較**する（完璧な案は通常存在しない）
- **スペース制約**を後付けで考えるのではなく、早期に入れておく（⑤の段階で調整）

## グループテクノロジー（GT）

グループテクノロジーは、類似する部品や製品をグループ化し、グループごとにセルを構成する考え方です。多品種少量生産でも、類似品をまとめることで量産に近い効率を得られます。

### GT の概念：なぜ必要か

通常、多品種少量生産は工程別レイアウト（ジョブショップ）で対応します。しかし、以下の問題が発生：

- 製品 A は旋盤→研磨→溶接のルート
- 製品 B は研磨→溶接→旋盤のルート（違う順序）
- 搬送距離が長くなり、仕掛品が増加
- 工程間での待ち時間が増加

**GT の発想**：「似たような形の部品」「似たような工程」をグループ化すれば、そのグループだけで閉じた「ミニ製品別ライン」が作れるのでは？

### GT コードの基本

GT では、部品を**コード化**して類似度を判定します。

**例：自動車部品メーカーの GT コード体系**

$$
\text{コード} = \text{[形状][寸法][材質][工程]}
$$

| コード | 意味 | 例 |
| --- | --- | --- |
| 1-A-1-1 | 円筒形、小（φ10-20mm）、アルミ、旋盤のみ | ピン、スペーサ |
| 1-A-2-2 | 円筒形、小、アルミ、旋盤+研磨 | 高精度シャフト |
| 1-B-1-3 | 円筒形、中（φ20-50mm）、鋼、旋盤+溶接 | 連結軸 |
| 2-A-1-1 | 箱形、小、アルミ、旋盤のみ | ブロック部品 |

同じコード（または近いコード）の部品群が **1 つのセル**を形成します。

### セル生産方式（GT の実装形態）

GT コードで分類した部品グループごとに、**U 字型に設備を並べ**、1～3 名の多能工が全工程を担当します。

**セル内の配置（例：コード 1-A-2-2 グループ用セル）**

```
    【材料投入】
        ↓
┌──────────────────────┐
│      旋盤             │
│    (入口・出口)      │
│      ↓               │
│      ↓               │
│    研磨機             │
│      ↓               │
│    検査台             │
└──────────────────────┘
        ↓
   【完成品】

多能工 1 名が、朝から晩まで同じセル内で：
- 旋盤操作 → 研磨操作 → 検査 → 梱包

製品Aが完成→製品Bに移る（品種切替）も、セル内で完結
```

### GT の適用条件と効果

**適用できる条件**

| 条件 | 理由 |
| --- | --- |
| 部品の形状・工程が類似するグループが存在する | グループ化の前提 |
| グループごとに 3～5 種類の工程機械がセルに入る | 閉じた小ラインが成立 |
| 多能工の育成体制がある | 1 人が複数工程を担当 |
| 月単位のロット数が十分（グループあたり 100～500 個/月） | セルの稼働率を確保 |

**期待される効果**

| 効果 | 数値例 |
| --- | --- |
| 搬送距離の削減 | 工程別比で 50～70% 短縮 |
| 仕掛品の削減 | 工程別比で 40～60% 削減 |
| 生産リードタイムの短縮 | 工程別比で 30～50% 短縮 |
| 品質向上 | 1 人が最後まで責任を持つため、不良の早期発見 |

### GT とセル生産のまとめ

- **グループテクノロジー** = 部品をコード化して類似グループを特定する技術
- **セル生産方式** = GT で分類したグループごとに、U 字型セルと多能工で生産する実装方法
- **対象生産形態**：P-Q分析で「中間帯」（大量品と少量品が混在）の場合に特に有効

## 搬送距離の計算 ── From-To チャートから定量評価へ

レイアウト案が複数あるとき、**搬送距離の合計コスト**で比較することがあります。

### 計算例：2 つのレイアウト案の比較

**シナリオ**：先ほどの町工場の引っ越し。以下 2 つの案を比較。

| | 案 1：工程順直線配置 | 案 2：スペース都合で溶接を東側に移動 |
| --- | --- | --- |
| 配置 | 旋盤→研磨→溶接→検査→塗装 | 旋盤→研磨→検査→塗装（溶接は別棟） |
| 想定距離 | 各工程 10m 間隔 | 検査と溶接が 50m 離れる |

**From-To チャート（月間搬送数）を再掲**

| From | To | 量 | 案1の距離(m) | 案2の距離(m) |
| --- | --- | --- | --- | --- |
| 旋盤 | 研磨 | 150 | 10 | 10 |
| 研磨 | 溶接 | 90 | 10 | 30 |
| 溶接 | 検査 | 80 | 10 | 50 |
| 検査 | 塗装 | 70 | 10 | 10 |
| 旋盤 | 検査（直行） | 40 | 30 | 30 |
| 旋盤 | 塗装（部品） | 20 | 50 | 50 |

**搬送距離コストの計算**

$$
\text{搬送距離コスト} = \sum (\text{搬送量} \times \text{距離})
$$

**案 1：直線配置**

$$
\begin{align}
\text{コスト} &= (150 \times 10) + (90 \times 10) + (80 \times 10) + (70 \times 10) + (40 \times 30) + (20 \times 50) \\
&= 1,500 + 900 + 800 + 700 + 1,200 + 1,000 \\
&= 6,100 \text{ パレット・メートル/月}
\end{align}
$$

**案 2：溶接を東側に移動**

$$
\begin{align}
\text{コスト} &= (150 \times 10) + (90 \times 30) + (80 \times 50) + (70 \times 10) + (40 \times 30) + (20 \times 50) \\
&= 1,500 + 2,700 + 4,000 + 700 + 1,200 + 1,000 \\
&= 11,100 \text{ パレット・メートル/月}
\end{align}
$$

**結論**：案 1 が案 2 より搬送距離で **5,000 パレット・メートル/月コスト削減**。年間では 60,000 パレット・メートルのムダが削減される。

この定量化が、SLP の最終判断を支える重要な根拠になります。

## 典型的なつまずき

- 工程別レイアウトを大量生産向けと誤解する → 工程別は **多品種少量** 向け
- セル生産と製品別レイアウトを同じものとして扱う → セルは **多品種小ロット** 向け、ラインは **少品種大量** 向け
- SLP を単なる記号暗記で終える → **手順**（P-Q分析→物の流れ→近接性評価→スペース配分）の流れが重要
- P-Q分析をパレート図と混同する → P-Q分析は **レイアウト選択** のため、パレート図は **品質改善の重点特定** のため
- 固定式レイアウトを工場の固定設備と混同する → 固定するのは **製品**（船舶、航空機など）

## 問題を解くときの観点

- 品種と数量はどうなっているか
- 重いのは `柔軟性` か `直線的な流れ` か
- 距離や搬送のムダを減らしたいのか
- SLP の手順のどの段階を問われているか

## 確認問題

<Callout title="問1：レイアウト類型の選択">
    「生産量は少ないが品種が多く、製品ごとに使用する設備が異なる」工場に最も適したレイアウト類型はどれか。

    **解答**：工程別レイアウト（機能別レイアウト）。同種の設備をまとめて配置し、品種変更に柔軟に対応する。
</Callout>

<Callout title="問2：P-Q分析と生産量の判断">
    ある精密機械部品メーカーで以下のデータが得られた：

    | 製品 | 月産数量 |
    | --- | --- |
    | 製品X | 800個 |
    | 製品Y | 400個 |
    | 製品Z | 80個 |
    | 製品W | 20個 |
    | 計 | 1,300個 |

    P-Q分析を行い、累積生産量が 90% に達するまでの製品群に適したレイアウトを答えよ。また、その理由を述べよ。

    **解答**：製品X, Y, Z が対象（累積：800 + 400 + 80 = 1,280 個、比率 98.5%）。**製品別レイアウト**が適切。理由：最も生産量の多い 3 製品（全体の 98%）について製品別ラインを構成すれば、搬送距離が最小化され、工程順が安定する。少量品（製品W）は別途工程別で処理する柔軟性を持つ。
</Callout>

<Callout title="問3：SLP の近接性評価と配置">
    SLP で部門間の近接性を評価した結果、以下の組み合わせがある場合、配置をどのように決めるか。また、その根拠を述べよ。

    | 部門 1 | 部門 2 | 評価 | 理由 |
    | --- | --- | --- | --- |
    | 旋盤工程 | 研磨工程 | A | 物の流れが月 200 パレット以上 |
    | 溶接工程 | 塗装工程 | X | 溶接の火花・騒音が塗装品質を阻害 |
    | 検査工程 | 梱包工程 | E | 検査直後に梱包（品質確保） |

    **解答**：以下の配置原則に従う：
    1. **旋盤と研磨は隣接**させ、搬送距離を最小化する
    2. **溶接と塗装は廊下やスペースで明確に離す**（棟を分けるなど）
    3. **検査と梱包はできるだけ近く**配置し、検査→梱包の流れを直線化する

    これにより、物流効率と品質リスク管理のバランスが実現される。
</Callout>

<Callout title="問4：搬送距離コストの比較（計算問題）">
    以下のデータで、2 つのレイアウト案の搬送距離コストを比較し、より効率的な案を選べ。

    **月間搬送量（パレット）**

    | From | To | 搬送量 |
    | --- | --- | --- |
    | 投入 | 旋盤 | 120 |
    | 旋盤 | 研磨 | 100 |
    | 研磨 | 溶接 | 100 |
    | 溶接 | 検査 | 90 |
    | 検査 | 梱包 | 85 |

    **配置案**

    案A（直線配置）：投入→旋盤(10m)→研磨(10m)→溶接(10m)→検査(10m)→梱包(10m)
    案B（スペース重視）：投入→旋盤(10m)→検査(40m)→梱包(10m)、かつ研磨と溶接は別途

    案A の搬送距離コストを計算し、案A が採用される理由を述べよ。

    **解答（計算）**：

    案A：
    $$
    \begin{align}
    \text{コスト} &= (120 \times 10) + (100 \times 10) + (100 \times 10) + (90 \times 10) + (85 \times 10) \\
    &= 1,200 + 1,000 + 1,000 + 900 + 850 \\
    &= 4,950 \text{ パレット・メートル}
    \end{align}
    $$

    案B（旋盤→検査の直行を仮定）：
    $$
    \begin{align}
    \text{コスト} &= (120 \times 10) + (100 \times 40) + (100 \times \text{迂回}) + (90 \times 40) + (85 \times 10) \\
    &\approx 8,000 \text{ パレット・メートル}（概算）
    \end{align}
    $$

    **採用理由**：案A は案B より 3,000 パレット・メートル/月（年間 36,000）のコスト削減が期待でき、工程順の一貫性が高い。スペース制約がある場合でも、検査エリアの配置工夫で案A を実現すべき。
</Callout>
