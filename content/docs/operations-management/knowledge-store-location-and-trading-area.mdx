## まずイメージをつかむ

**シナリオ：新しいカフェの出店場所をどう決めるか？**

あなたは新しいカフェチェーンを経営しており、3つの候補地を検討しています。

1. **A駅前（商業集積型）**：既に商店街がある駅前。月々の家賃は高い（30万円）が、駅利用客がいて人通りがある。
2. **県道沿い（独立型）**：ロードサイドの空き地。駐車場が広く、家賃は安い（10万円）。ただし人通りはゼロに近く、看板と広告に頼る。
3. **近隣SC内（計画型）**：新しくできたショッピングセンター内の区画。家賃は中程度（15万円）。SCの集客力に乗れるが、営業時間やメニューの自由度が限られる。

さらに、競合が既にいます。
- 駅前には老舗コーヒー店（売場面積150㎡、客歴30年）がある。
- SCには別チェーンのカフェ（売場面積200㎡、新規）がある。

**問い**：
- A駅前に出店したとき、既存コーヒー店との商圏境界はどこになるか？
- 各候補地のカフェから客が何km範囲で来るか？
- 各候補地で、どのくらいの売上が見込めるか？

これらの問いに、**ライリーの法則**（2都市の吸引力）、**コンバースの法則**（商圏境界）、**ハフモデル**（来店確率）で答えます。

## 学習のポイント

- `立地類型` は出店環境の分類
- `商圏分析` は来店範囲の把握
- `ライリー` は「吸引力の比」、`コンバース` は「境界」、`ハフ` は「確率」──何を求めるか

## 試験で何が問われるか

- 立地類型の違いを説明し、業態別の最適立地を判断できるか
- コンバースの法則で **商圏境界** を計算し、解釈できるか
- ハフモデルで **来店確率** を計算し、複数店舗を比較できるか
- 商圏の階層構造（第1次～第3次）を理解しているか

## 店舗立地の3大類型と業態別最適立地

### 1. 商業集積型（商店街、駅前、ショッピングセンター）

**特徴**：複数の店舗が一箇所に集積している立地。駅前商店街、郊外ショッピングセンター、百貨店内などが該当。

| 項目 | 説明 |
| --- | --- |
| 集客メカニズム | 集積自体が強い吸引力。各店舗は集積の恩恵を受ける |
| 売上構成 | 目的来店（目当ての店に来る）+ 買い回り（他の店も覗く） |
| 賃料 | 高い（駅前30-50万円/月級） |
| 最適な業態 | 百貨店テナント、高級ブランド、カフェ、ファッション |
| 着眼点 | SCの来客数、客層、営業時間制限 |

**メリット**：集積の集客力を利用でき、客は自動的に来店。高級品や新ブランドでも販売可能。

**デメリット**：賃料が高く、営業時間やメニューの自由度が限られ、競合が常に目の前。出店するには企業審査が厳しい。

---

### 2. 独立型（ロードサイド）

**特徴**：県道や国道沿い、駐車場併設の単独出店。ガソリンスタンド、外食チェーン、ドラッグストア、家電量販店など。

| 項目 | 説明 |
| --- | --- |
| 集客メカニズム | 看板、広告、ロケーション（「この道で唯一の店」） |
| 売上構成 | ほぼ100%が自力集客。周辺人口で決まる |
| 賃料 | 低い（10-20万円/月級） |
| 最適な業態 | ドラッグストア、牛丼チェーン、ガソリンスタンド、大型一次産業店 |
| 着眼点 | 通行量（特に車）、駐車場台数、夜間照明、周辺人口密度 |

**メリット**：賃料が安く、営業時間やメニューに自由度がある。駐車スペースが広く、大型フォーマットに適している。

**デメリット**：自力集客が必須。訴求力のない看板では客が来ない。初期のブランド認知が低い業態には不向き。

---

### 3. 駅前・駅ナカ（交通結節点型）

**特徴**：駅ビル、駅構内、駅直結ビル。通勤通学客の流動が前提。

| 項目 | 説明 |
| --- | --- |
| 集客メカニズム | 交通結節点の強制通過。「朝の通勤ルート上」が価値 |
| 売上構成 | 時間帯が偏る（朝7-9時、夕方5-7時に集中） |
| 賃料 | 非常に高い（50万円以上/月） |
| 最適な業態 | スターバックス、コンビニ、弁当屋、キオスク、ベーカリー |
| 着眼点 | 通勤客数、乗車率、昼間の人口（平日vs休日のギャップ） |

**メリット**：確実な流動人口。高級品の露出に最適。朝夕ラッシュの時間帯に人口が集中するため「つかみの営業」に強い。

**デメリット**：賃料が高く、昼間や休日の客付きが弱い。営業時間が制限される。

---

### 業態別：最適立地の判断表

| 業態 | 第一選択肢 | 理由 |
| --- | --- | --- |
| カフェ・喫茶 | 駅前 or SC内 | 立ち寄り客、Wi-Fi需要 |
| 牛丼・ファストフード | ロードサイド | 低賃料、大型厨房、駐車場 |
| 高級フレンチ | SC内 | 客層の絞り込み、集客力 |
| コンビニ | ロードサイド or 駅ナカ | 24時間営業可能、通行量 |
| ドラッグストア | ロードサイド | 広い売場面積、駐車場必須 |
| ブランド専門店 | 駅前・商業集積 | ステータス感、客層のマッチ |
| 居酒屋 | 駅前 | 夜間の人通り |
| 本屋 | 駅前 or SC | 立ち寄り客、滞在時間 |

    2. **競合状況**：既存の同業者は何店舗か。距離は
    3. **駐車場**：台数は十分か。料金は

## ライリーの法則（小売引力の法則）

### 公式と意味

2つの都市が中間の町の購買力を吸引する割合は、**人口に比例し、距離の2乗に反比例**します。

$$
\frac{B_a}{B_b} = \frac{P_a}{P_b} \times \left(\frac{D_b}{D_a}\right)^2
$$

| 記号 | 意味 |
| --- | --- |
| Ba/Bb | A市とB市の吸引力の比。Ba/Bb = 2なら、A市がB市の2倍の吸引力 |
| Pa/Pb | A市とB市の人口の比 |
| Da/Db | 中間の町からA市・B市までの距離 |

### 「なぜ」を理解する

<Callout title="ライリーの法則の直感的理解">
    **人口が大きい都市ほど吸引力が強い**理由：人口が多い = 店舗数・品揃え・情報が多い = より多くの顧客を集めやすい。

    **距離が遠いほど吸引力が弱い**理由：遠い = 移動に時間・コスト・手間がかかる = 利便性が低い。**距離は2乗で効く**（2倍遠いと、1/4の吸引力）。
</Callout>

### 2都市間での客の引き合いの数値例

**シナリオ**：A市（人口40万人、距離20km）とB市（人口10万人、距離30km）がある中間の町M（人口5万人）の購買力を引き合う場合。

$$
\frac{B_a}{B_b} = \frac{400,000}{100,000} \times \left(\frac{30}{20}\right)^2
$$

**計算手順**：

1. **人口比**：400,000 ÷ 100,000 = 4
2. **距離比**：30 ÷ 20 = 1.5
3. **距離の2乗**：1.5² = 2.25
4. **吸引力の比**：4 × 2.25 = **9**

**解釈**：
- A市がM町の購買力を吸引する力は、B市の**9倍**
- M町の人口5万人のうち、約90%がA市で、10%がB市で買い物をする（粗い見積もり）
- A市は人口4倍（×4）だが、距離も遠い（1.5倍遠い）ため、最終的には9倍の吸引力

**別シナリオ**：距離が同じなら？

A市とB市がM町から同じ距離にある場合（Da = Db）：

$$
\frac{B_a}{B_b} = \frac{P_a}{P_b} \times 1 = \frac{P_a}{P_b}
$$

→ **人口比だけで決まる**。距離効果が消える。

## コンバースの法則（商圏分界点）

### 公式と意味

A市とB市の間の**商圏境界（分界点）**の位置を求める公式です。「どこから先がB市の商圏か」という地理的な分岐点を計算します。

$$
d = \frac{D}{1 + \sqrt{\frac{P_b}{P_a}}}
$$

| 記号 | 意味 |
| --- | --- |
| d | A市から商圏境界までの距離 |
| D | A市とB市の距離 |
| Pa | A市の人口 |
| Pb | B市の人口 |

### 「なぜ」を理解する

<Callout title="コンバースの法則の直感的理解">
    商圏は人口の大きさで「引き寄せられる」位置が変わります。

    A市が大きいほど、境界はB市に近づく（A市の商圏が広がる）。

    B市が大きいほど、境界はA市に近づく（B市の商圏が広がる）。

    距離が同じなら、2つの都市の「力関係」で境界が決まります。
</Callout>

### 商圏分界点の図的イメージ

```
A市 ━━━━━━━━━━━━━━ B市
人口40万     分界点(d)     人口10万
          ← d →|
          ↑
     A市からの距離
```

分界点より左がA市の商圏、右がB市の商圏。A市の方が人口が大きいため、分界点はB市に近い。

### 計算例1：基本的な例

**シナリオ**：A市（人口160万）とB市（人口40万）の距離が30kmのとき、商圏境界はどこか。

```
【データ】
  A市の人口 Pa = 160,000人
  B市の人口 Pb = 40,000人
  A市とB市の距離 D = 30km

【計算】
d = 30 ÷ (1 + √(40,000/160,000))
  = 30 ÷ (1 + √0.25)
  = 30 ÷ (1 + 0.5)
  = 30 ÷ 1.5
  = 20km

【解釈】
  → A市から20kmの地点が商圏境界
  → A市の方が人口が多い（4倍）ため、商圏はA市側に大きく広がる
  → B市から見ると10km地点が境界（30−20=10km）
  → つまり、AB間30km中、A市が20km、B市が10km を支配
```

### 計算例2：複雑な例（両都市の人口が近い場合）

**シナリオ**：A市（人口90万）とB市（人口10万）の距離が40kmのとき、商圏境界は。

```
【データ】
  Pa = 90,000、Pb = 10,000、D = 40km

【計算】
d = 40 ÷ (1 + √(10,000/90,000))
  = 40 ÷ (1 + √(1/9))
  = 40 ÷ (1 + 1/3)
  = 40 ÷ (4/3)
  = 40 × 3/4
  = 30km

【解釈】
  → A市から30kmの地点が商圏境界
  → A市の人口が9倍なので、境界はB市に大きく寄る
  → B市の商圏は、たった10kmしかない
```

### 計算例3：人口が同じ場合（A市 = B市）

```
Pa = Pb（人口が同じ） のとき：
d = D ÷ (1 + √1) = D ÷ 2 = D/2

→ 中点（AB中間）が分界点
→ 対称的に、各都市の商圏は半分ずつ
```

### 読み方のポイント

- **人口が大きい都市ほど商圏は広い**（分界点が相手側に寄る）
- **「d」はA市からの距離**なので、A市が大きいほど `d` も大きくなる
- **小さい都市は、非常に狭い商圏しか持たない**（上記例2でB市は10kmのみ）

## ハフモデル（確率的商圏モデル）

### 公式と意味

消費者がある店舗を選ぶ**確率**を、店舗の売場面積と距離から求めるモデルです。「必ずA店に行く」ではなく「70%の確率でA店、30%の確率でB店」という確率的な行動を扱います。

$$
P_{ij} = \frac{S_j / T_{ij}^{\lambda}}{\sum_{k=1}^{n} S_k / T_{ik}^{\lambda}}
$$

| 記号 | 意味 |
| --- | --- |
| Pij | 消費者iが店舗jを選ぶ確率（0～1.0、%で言うと0～100%） |
| Sj | 店舗jの売場面積（㎡） |
| Tij | 消費者iから店舗jまでの距離（km）または移動時間（分） |
| λ | 距離抵抗パラメータ。通常λ=2（距離は2乗で効く）。λ=1なら直線的、λ=3なら距離に極度に敏感 |

### 「なぜ」を理解する

<Callout title="ハフモデルが「確率的」である理由">
    現実の買い物客は「A店が最も便利だから絶対に行く」のではなく、「A店の方が便利だから、70%の確率で行く」という選択をします。気分、セール、友人との約束など、モデルに入らない要素で30%の確率でB店を選びます。

    ハフモデルは「複数の選択肢がある中で、相対的に選ばれやすい確率」を計算するもので、「100%的中する予測」ではありません。
</Callout>

### λ（距離抵抗パラメータ）の感度

| λ値 | 特徴 | 現実的なシーン |
| --- | --- | --- |
| λ=1 | 距離が直線的に効く | 距離感覚が鈍い（インターネット通販） |
| λ=2（標準） | 距離は2乗で効く | 日常的な店選び（スーパー、カフェ） |
| λ=3以上 | 距離に極度に敏感 | 手間や時間を極度に嫌う（コンビニ、駅ナカ） |

距離が**2倍遠い**と：
- λ=1なら、吸引力は1/2に低下
- λ=2なら、吸引力は1/4に低下
- λ=3なら、吸引力は1/8に低下

### 計算例1：2店舗の基本的な比較

**シナリオ**：消費者Xが、自宅から店舗A（売場1,000㎡、距離5km）と店舗B（売場2,000㎡、距離10km）のどちらを選ぶか。

```
【データ】
  店舗A: 売場面積 1,000㎡、距離 5km
  店舗B: 売場面積 2,000㎡、距離 10km
  λ = 2

【計算手順】
1. 各店舗の「効用値」を計算（面積 ÷ 距離²）
   店舗A: 1,000 / 5² = 1,000 / 25 = 40
   店舗B: 2,000 / 10² = 2,000 / 100 = 20

2. 効用値の合計
   合計 = 40 + 20 = 60

3. 各店舗の選択確率
   P(A) = 40 / 60 = 2/3 ≒ 66.7%
   P(B) = 20 / 60 = 1/3 ≒ 33.3%

【解釈】
  → 店舗Bの方が面積は2倍だが、距離が2倍なので距離の影響（2乗で4倍弱くなる）が大きい
  → 結果として店舗Aの選択確率の方が高い（66.7% vs 33.3%）
  → 消費者Xは、100回買い物に行く中で、約67回A店、33回B店に行く
```

### 計算例2：3店舗以上の複雑な例

**シナリオ**：消費者Yが、3つの店舗から選ぶ場合。

```
【データ】
  店舗A: 売場面積 500㎡、距離 2km
  店舗B: 店舗B: 売場面積 300㎡、距離 3km
  店舗C: 売場面積 800㎡、距離 5km
  λ = 2

【計算手順】
1. 各店舗の効用値
   店舗A: 500 / 2² = 500 / 4 = 125
   店舗B: 300 / 3² = 300 / 9 ≒ 33.3
   店舗C: 800 / 5² = 800 / 25 = 32

2. 効用値の合計
   合計 = 125 + 33.3 + 32 = 190.3

3. 各店舗の選択確率
   P(A) = 125 / 190.3 ≒ 65.7%
   P(B) = 33.3 / 190.3 ≒ 17.5%
   P(C) = 32 / 190.3 ≒ 16.8%

【解釈】
  → 店舗Cは売場面積は最大（800㎡）だが、距離が遠い（5km）ため、期待度は低い
  → 店舗Aが最も近く（2km）、売場も十分なため、選択確率が圧倒的に高い（65.7%）
  → 店舗BとCはほぼ同等の低確率
```

### 計算例3：λの感度による違い（同じデータで、λを変えた場合）

**同じ2店舗を、λ=1とλ=3で比較**：

```
【データ】
  店舗A: 売場1,000㎡、距離 2km
  店舗B: 売場1,000㎡、距離 4km
  （面積は同じ、距離だけ異なる）

【λ=1の場合】
  A: 1,000 / 2 = 500
  B: 1,000 / 4 = 250
  P(A) = 500/750 ≒ 66.7%、P(B) ≒ 33.3%

【λ=2の場合（標準）】
  A: 1,000 / 4 = 250
  B: 1,000 / 16 = 62.5
  P(A) = 250/312.5 = 80%、P(B) = 20%

【λ=3の場合】
  A: 1,000 / 8 = 125
  B: 1,000 / 64 = 15.625
  P(A) = 125/140.625 ≒ 88.9%、P(B) ≒ 11.1%

【比較】
  λが大きいほど、距離の短い店舗が圧倒的に有利になる
  λ=1なら、A と B の確率差は 33.4%
  λ=3なら、A と B の確率差は 77.8%
```

### ハフモデルのポイント

- **売場面積が大きいほど吸引力が強い**（分子に面積）
- **距離が遠いほど吸引力は弱い**（分母に距離を乗せる）
- **距離は λ乗で効く**（通常λ=2なので、2倍遠いと1/4の吸引力）
- **λが大きいほど、距離に敏感**。近い店舗が圧倒的に有利になる

## 商圏の階層構造（第1次～第3次）

実際の店舗は、単一の「商圏」ではなく、**複数の層からなる商圏**を持っています。距離に応じて、来店客の比率と行動が異なります。

### 3層の定義

| 層 | 距離の目安 | 来店比率 | 特徴 |
| --- | --- | --- | --- |
| **第1次商圏** | 徒歩圏（500m-1km） | 60-70% | 日常的な来店、立ち寄り客 |
| **第2次商圏** | 自転車・バス圏（1-3km） | 20-30% | 目的来店、買い物ついで |
| **第3次商圏** | 車圏（3km以上） | 10%以下 | 計画来店、大型店舗への来店 |

### 業態別の商圏の広さ

| 業態 | 第1次商圏 | 第2次商圏 | 第3次商圏 | 理由 |
| --- | --- | --- | --- | --- |
| **コンビニ** | 500m | 1-2km | ほぼ0% | 近さが最優先。少し遠いと利用しない |
| **スーパー** | 800m-1km | 2-3km | ～5% | 日常利用だが、週1-2回の計画来店 |
| **ドラッグストア** | 1km | 1-2km | ～3% | 目的来店だが、距離の影響は受ける |
| **カフェ** | 500m-1km | 1-2km | ～3% | 立ち寄りが多いため、近さが重要 |
| **高級レストラン** | 1km | 2-5km | 5-10% | 計画来店が多く、距離への耐性あり |
| **百貨店・SC** | 2-3km | 3-10km | 10-30% | 大型施設は広い商圏を持つ |

### 例：スーパーXの商圏分析

```

【第1次商圏（500-800m、徒歩圏）】
  人口：20,000人
  来店比率：70%
  来店客数：約14,000人/月
  行動：「夕食の買い物」「日用品」が主体

  行動：「週末のまとめ買い」「特売日の目的来店」
```

## ライリー・コンバース・ハフの使い分け

| モデル | 求めるもの | 入力データ | 出力 |
| --- | --- | --- | --- |
| ライリーの法則 | 2都市の吸引力の **比率** | 人口、距離 | 吸引力比 |
| コンバースの法則 | 商圏の **境界地点** | 人口、都市間距離 | A市からの距離 |
| ハフモデル | 特定店舗の **選択確率** | 売場面積、距離、λ | 来店確率(%) |

## 典型的なつまずき

<Callout title="つまずき1：公式を覚えても、何を判断するための式か説明できない">
    **❌ 誤り**：「ハフモデルの公式は Pij = ... 」と式を丸暗記

    **✓ 正解**：
    - ライリーは「比率」（A市がB市の何倍の吸引力か）
    - コンバースは「境界」（どこから先がB市の商圏か）
    - ハフは「確率」（70%の確率でA店、30%でB店）
</Callout>

<Callout title="つまずき2：商圏分析をマーケティング戦略と混同する">
    **❌ 誤り**：「20代女性向けのターゲット設定が必要」と答える

    **✓ 正解**：商圏分析は「来店範囲の物理的把握」。ターゲット設定は別の問題。
    まずは「誰が」ではなく「どこから来るか」を押さえる。
</Callout>

<Callout title="つまずき3：コンバースの公式の分母を逆にする">
    **❌ 誤り**：d = D ÷ (1 + √(Pa/Pb)) ← 分母が逆

    **✓ 正解**：d = D ÷ (1 + √(Pb/Pa)) ← **分子の方が大きい都市（Pa）を左に**
</Callout>

<Callout title="つまずき4：ハフモデルで距離を2乗し忘れる">
    **❌ 誤り**：効用値を「面積 ÷ 距離」で計算（λを無視）

    **✓ 正解**：λ=2なら「面積 ÷ 距離²」。λ=3なら「面積 ÷ 距離³」
</Callout>

<Callout title="つまずき5：立地条件と商圏の広さを同じものとして扱う">
    **❌ 誤り**：「駅前立地なので、商圏は広い」

    **✓ 正解**：立地は「出店場所の特性」（駅前、ロードサイド等）。商圏は「来店範囲」（第1次～第3次）。
    駅前立地でも、コンビニなら商圏は狭く（500m）、百貨店なら広い（5km以上）。
</Callout>

## 確認問題（5問）

<Callout title="問1：コンバースの法則（基本計算）">
    A市（人口 90,000人）とB市（人口 10,000人）の距離が 40km のとき、A市からの商圏境界を求めよ。

    **解答**：
    d = 40÷(1+√(10,000/90,000))
    = 40÷(1+√(1/9))
    = 40÷(1+1/3)
    = 40÷(4/3)
    = **30km**

    **解釈**：A市から30km地点が商圏境界。A市の方が人口9倍なので、境界はB市に大きく寄る。
</Callout>

<Callout title="問2：ハフモデル（2店舗の確率計算）">
    消費者から店舗X（売場 500㎡、距離 2km）と店舗Y（売場 800㎡、距離 4km）がある。λ=2のとき、店舗Xと店舗Yの選択確率をそれぞれ求めよ。

    **解答**：
    - 効用値：X = 500/2² = 125、Y = 800/4² = 50
    - 合計 = 125 + 50 = 175
    - P(X) = 125/175 = **約71.4%**
    - P(Y) = 50/175 = **約28.6%**

    **解釈**：店舗Yは売場が1.6倍大きいが、距離が2倍遠いため（距離²の効果で4倍弱い）、店舗Xの選択確率が高い。
</Callout>

<Callout title="問3：ハフモデル（3店舗の複雑な比較）">
    消費者Zから見て、店舗A（売場 600㎡、距離 1.5km）、店舗B（売場 900㎡、距離 3km）、店舗C（売場 1,200㎡、距離 5km）がある。λ=2のとき、各店の選択確率を求めよ。

    **解答**：
    - 効用値：A = 600/2.25 ≒ 266.7、B = 900/9 = 100、C = 1,200/25 = 48
    - 合計 = 266.7 + 100 + 48 = 414.7
    - P(A) = 266.7/414.7 ≒ **64.3%**
    - P(B) = 100/414.7 ≒ **24.1%**
    - P(C) = 48/414.7 ≒ **11.6%**

    **解釈**：店舗Cは売場が最大だが、距離が遠いため選択確率が最も低い。店舗Aが最も近く、かつ売場も十分なため、選択確率が圧倒的に高い。
</Callout>

<Callout title="問4：モデルの使い分け">
    次の4つの問い（①～④）のそれぞれに、ライリーの法則、コンバースの法則、ハフモデルのうち、どれを使うか答えよ。

    ① 「A市と B市が、中間の町Mの購買力を引き合う力の比を知りたい」

    ② 「A市とB市の間で、商圏の境界（A市からの距離）を知りたい」

    ③ 「自宅から5km以内に3つのスーパーがあるとき、その消費者がどのスーパーを選ぶ確率を知りたい」

    ④ 「来年、新しいショッピングセンターが出店されるが、既存スーパーの来店客数がどうなるか予測したい」

    **解答**：
    ① **ライリーの法則** ← 「比率」（「何倍」）を求めるから
    ② **コンバースの法則** ← 「境界地点」（「どこから」）を求めるから
    ③ **ハフモデル** ← 複数店舗から「来店確率」（「70%、20%、10%」）を求めるから
    ④ **ハフモデル** ← 新店舗の出店による各店舗への「来店確率の変化」を予測するから
</Callout>

<Callout title="問5：立地類型と商圏の組み合わせ">
    以下の3つの業態について、最適な立地（商業集積型、独立型、駅前）と、おおよその商圏範囲（第1次：500m-1km / 第2次：1-2km / 第3次：2km以上）を答えよ。

    ① 居酒屋
    ② ドラッグストア
    ③ ブランド専門店（高級ブランド）

    **解答**：
    ① 居酒屋 → **立地：駅前** **商圏：第1次主体（500m-1km）。夜間の人通りが必須。**
    ② ドラッグストア → **立地：独立型（ロードサイド）** **商圏：第1次+第2次（1-2km）。広い駐車場と周辺人口が必要。**
    ③ ブランド専門店 → **立地：商業集積型（駅前・SC内）** **商圏：第1次+第2次+第3次（1-5km以上）。ステータス感と集積の集客力が必須。**
</Callout>
