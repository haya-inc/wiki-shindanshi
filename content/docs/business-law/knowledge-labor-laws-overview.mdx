## このページの役割

<Callout title="このページの役割">
    このページは、`使用者がどこまで労働条件を決められ、どこから法規制が入るか` を統一軸で整理する教材型ノードです。労働関連法規の全体像を **三層構造**（最低基準法 → 契約・就業規則ルール → 団体関係・職場環境）で解説し、試験で問われる計算・判例・誤答パターンまで網羅します。
</Callout>

<Callout title="法規制の優先順位（最重要）">
    労働条件では以下の優先順位が絶対です：`労基法 ≧ 労働協約 ≧ 就業規則 ≧ 労働契約`。労基法を下回る合意は**全て無効**で、労基法の基準が自動的に適用されます。これが全ての出題の軸になります。
</Callout>

## 学習のポイント

- **第一層**：労働基準法が`最低基準法`（強行法規）であることを押さえる
- **第二層**：労働契約法で`就業規則と個別契約の関係`を理解する
- **第三層**：労働組合法で`不当労働行為`を類型化する
- **第四層**：ハラスメント防止が`労務管理義務`だと整理する
- **第五層**：その他重要法規（派遣法、育介法、均等法など）を体系化する

## 試験で何が問われるか

- 労働基準法が強行法規だと説明し、下回る合意の効果を判定できるか
- 労働協約 > 就業規則 > 労働契約の優先順位を正確に適用できるか
- 就業規則の変更が「合理的」かどうかを7要素で判断できるか
- 解雇権濫用法理（16条）と雇止め法理（19条）を実務判例で区別できるか
- 無期転換ルール（18条）の5年ルールと労働者申込みの条件を正確に押さえているか
- 団体交渉拒否・支配介入・不利益取扱いの不当労働行為を類型化できるか
- ハラスメント（パワハラ6類型、セクハラ対価型・環境型）を事業主義務として説明できるか
- 時間外労働の上限規制（月45時間・年360時間、特別条項年720時間）を計算できるか
- 割増賃金率（時間外25%以上・深夜25%以上・休日35%以上）の複合計算ができるか
- 月60時間超の時間外労働が50%以上になる局面を判定できるか

## 労働基準法：最低基準の強行法規構造

### 基本原則：強行法規としての絶対性

労働基準法1条は、労働条件の最低基準を定める法律であり、**これを下回る合意は全て無効** となります。使用者がいかに有利な契約を用意しても、労基法の基準が自動的に適用される構造です。

```
労基法2条2項：
労働契約その他労働関係に関する
協約その他の合意で、
労基法に反するものは、
その部分について無効とする。
```

**三層構造での位置付け**：

| 優先順位 | 法規等 | 効力 | 試験での見分け方 |
| --- | --- | --- | --- |
| **第1位** | 労働基準法 | 最低基準（下回る合意は無効） | 「8時間」「40時間」「36協定」など絶対基準 |
| **第2位** | 労働協約 | 労基法と同等以上の基準で拘束力あり | 労組との団体交渉で成立した協定 |
| **第3位** | 就業規則 | 労基法・労協約を下回らなければ有効 | 会社が一方的に定めた基準 |
| **第4位** | 労働契約 | 就業規則を下回らなければ有効 | 個別の雇用契約書 |

**具体例で理解する**：

- 会社が「月45時間以内の時間外労働」と就業規則に明記しても、36協定がなければ時間外労働は全て違法 → 労基法の基準が自動適用
- 個別契約で「裁量労働制（月給45万円で時間外手当なし）」と合意しても、実際には9時間労働で7時間が時間外 → 割増賃金を支払うべき
- 就業規則で「年次有給休暇は法定6日」と定めた → 法定10日の方が優先 → 自動効

### 労働時間規制：法定労働時間と36協定

#### 法定労働時間の基準

```
労基法32条：
使用者は、労働者に、
休憩時間を除き1週間について
40時間を超えて、
労働させてはならない。
```

また、1日については8時間を超えることは禁止（実務では1日単位と1週単位で判定）。

**時間外労働の要件**：
- 36協定（労使協定）の締結・届出が必須
- 協定なしの時間外労働は全て違法
- 時間外労働をさせる**度合い**をあらかじめ定める必要あり

#### 2019年改正による時間外労働の上限規制

2019年4月から、時間外労働の上限が法律で明確に規制されました：

| 規制項目 | 基本上限 | 特別条項での上限 | 備考 |
| --- | --- | --- | --- |
| **月の時間外労働** | 45時間 | 月の場合の上限なし（別項で規制） | 原則として月45時間を超えてはいけない |
| **年間の時間外労働** | 360時間 | 年で720時間（月平均60時間） | 複数月ある場合、いずれかが80時間超はNG |
| **複数月での上限** | — | 2ヶ月〜6ヶ月の平均が80時間超NG | 「月100時間」は高々2ヶ月まで |
| **連続超過** | — | 月100時間超が2回まで | 3回以上は協定違反 |

**試験での計算パターン**：
- A社：「特別条項で月100時間、3月続く」→ これは違法（月100時間は2ヶ月まで）
- B社：「月45時間×10ヶ月、月100時間×2ヶ月」→ 年720時間、各月の上限OK、複数月平均70時間（80時間以下）→ 合法
- C社：「月50時間×12ヶ月（年600時間）」→ 年360時間以下→ 合法

### 休憩時間規制

```
労基法34条：
1. 使用者は、労働時間が6時間を超える場合
   において少なくとも45分、
   8時間を超える場合において
   少なくとも1時間の休憩時間を
   与えなければならない。
```

**重要ポイント**：
- 休憩時間は「一斉付与」が原則だが、実務的に認められる例外あり
- 休憩時間は労働時間に含まない（給与計算から除く）

### 休日規制

```
労基法35条：
使用者は、労働者に対して、
毎週少なくとも1日の休日を与えなければならない。
ただし、4週間を通じ4日以上の休日を与える場合は、
この限りでない。
```

**重要な判別**：
- **法定休日**：週1日（またはその代替として4週で4日）
- **所定休日**：就業規則で決めた休日（法定以上）
- **休日出勤**：割増賃金35%以上の支払いが必須

### 割増賃金の計算と複合

割増賃金率は法律で最低基準が決められており、これを下回る合意は無効です。

#### 基本的な割増賃金率

| 時間帯・条件 | 割増率 | 注記 |
| --- | --- | --- |
| **時間外労働** | 25%以上 | 1日8時間超、週40時間超の場合 |
| **深夜労働** | 25%以上 | 夜10時～朝5時（実務では異なる場合あり） |
| **休日労働** | 35%以上 | 法定休日での労働 |
| **月60時間超の時間外** | 50%以上 | 2019年改正で大企業対象（中小企業は猶予） |

#### 複合的な割増の計算

労働が複数の割増対象になった場合、**高い率の方が適用される**（加算されない）：

**例1**：深夜の時間外労働（22時～23時の1時間）
- 時間外25% + 深夜25% = 最高50%（加算ではなく、より高い率を採用）
- ただし、基礎単価の計算法により異なる場合がある

**例2**：休日深夜労働（日曜日の深夜2時間）
- 休日35% + 深夜25% = 最高60%

**試験での頻出パターン**：
- 月の時間外が70時間、うち深夜が20時間 → 深夜分は50%超（月60時間超該当）
- 法定休日に夜勤（8時間） → 35%（最高率だから休日35%が優先）

### 年次有給休暇

#### 発生要件と日数

```
労基法39条：
1. 使用者は、その雇い入れの日から
   起算して6ヶ月間継続勤務し、
   全労働日の8割以上出勤した労働者に対して、
   継続し、又は分割した10日の年次有給休暇を与えなければならない。
```

**重要な数字**：
- 6ヶ月継続勤務 + 8割出勤 → 初回10日
- 以後1年経過ごとに加算：11日→12日→14日→16日→18日→20日
- 最高20日（6年勤続以上）

#### 2019年改正：5日の年次取得義務

2019年4月から、使用者は**年に最低5日** の年次有給休暇を「計画的に」取得させる義務が生じました：

```
労基法39条7項：
使用者は、前各項の規定により
年次有給休暇を与える場合においては、
その5日については、
その年度から指定した日時に
与えなければならない。
```

**試験での見分け方**：
- 「有給休暇の日数」と「実際の取得日数」を区別
- 残日数が10日以上ある人でも、実際取得が5日未満なら違法

### 解雇と解雇予告

#### 解雇予告の要件

```
労基法20条：
使用者は、労働者を解雇しようとする場合は、
少なくとも30日間前にその予告をするか、
30日分以上の平均賃金を支払わなければならない。
```

**重要なパターン**：
- 予告30日 = 合法
- 予告15日 + 15日分の手当 = 合法
- 予告なし + 30日分の手当 = 合法
- 予告なし + 手当なし = 違法

**例外**：
- 試用期間中（14日以内の予告で可）
- 天災事変などの不可抗力

### 就業規則の作成・周知義務

```
労基法89条：
常時10人以上の労働者を使用する使用者は、
次に掲げる事項について就業規則を作成し、
行政官庁に届け出なければならない。
```

**必須記載事項**：
- 始業・終業の時刻、休憩時間、休日
- 給与の決定・計算・支払い方法、支払日
- 退職（定年、解雇手当含む）

**周知義務**：
- 掲示、書面交付、電子データ提供など
- 周知されていない規定は効力が制限される場合がある

## 労働契約法：契約と就業規則の効力規制

### 基本三原則

労働契約法は、労働契約の基本的なルールを定めており、労基法と並ぶ重要法規です：

1. **合意原則**：使用者と労働者の合意が基本（ただし労基法・就業規則の枠内）
2. **均衡考慮の原則**：労働条件を定める際、就業規則・個別契約のバランスを考慮
3. **仕事と生活の調和**：長時間労働の防止、柔軟な働き方の配慮

### 就業規則と労働契約の効力関係

#### 労働契約法7条：就業規則が個別契約を上回る場合

```
労働契約法7条：
使用者が就業規則の周知があり、
かつ合理的な内容である場合、
個別の労働契約に優先して
就業規則が適用される。
```

**三層構造での再確認**：

```
労基法 > 労協約 > 就業規則 > 労働契約
```

- 労基法の最低基準を下回る就業規則は無効
- 就業規則が個別契約に優先（ただし労基法以上の水準であること）

**試験での判別**：
- 採用時に「月給30万円」で契約 → 後から就業規則で「月給25万円」に変更 → 個別契約の30万円が優先（不利益変更）
- 採用時に就業規則未説明 → 労働者が有利な内容を選べる

#### 労働契約法8条：就業規則の不利益変更（最重要論点）

```
労働契約法8条：
使用者が、就業規則の変更により、
労働者の労働条件を一方的に低下させる変更を
しようとする場合、
その変更後の就業規則が合理的なものであれば、
労働者の同意がなくても
変更後の規則が適用される場合がある。
```

**「合理的な変更」の7要素判定**（判例準拠）：

1. **変更の必要性**：企業の経営環境悪化など正当な理由があるか
2. **変更の内容の相当性**：不利益の程度が経営上の必要性に相応しいか
3. **労働者への代償措置**：転職、退職金の上乗せなど代替措置があるか
4. **労働組合や労働者代表との交渉**：十分な説明と協議が行われたか
5. **他社での同業慣行**：業界標準との比較
6. **労働条件の性質**：給与・勤務地・職務内容の重要度
7. **変更の段階性**：急激な変更か段階的な変更か

**試験での頻出事例**：
- 「経営難のため給与30%カット、給与表の改定」→ 労働者代表の同意なし、代償措置なし → 不合理で無効
- 「定年を65歳に延長、給与は60歳以降20%減、業界標準」→ 労働者代表と協議、代償あり → 合理的で有効

### 解雇権濫用法理（労働契約法16条）

解雇は、労働法上最も強い権利行使（労働関係の最終的な終了）であり、厳格なルールが課されます：

```
労働契約法16条：
解雇は、客観的に合理的な理由を欠き、
社会通念上相当であると認められない場合は、
その権利を濫用したものとして、無効とする。
```

**「合理的な理由」の類型**：

| 解雇類型 | 合理的な理由 | 試験での見分け方 |
| --- | --- | --- |
| **普通解雇** | 能力不足・勤務不良・経営危機 | 改善指導、段階的処分が必要 |
| **懲戒解雇** | 就業規則の懲戒規定違反・著しい非違 | 「懲戒解雇」が事前に定められている必要 |
| **整理解雇** | 経営上の必要性・人員削減 | 人選基準の合理性、代替手段の検討が必須 |

**整理解雇の4要素（判例基準）**：
1. 解雇を回避するための経営上の真摯な努力
2. 被解雇者の選定基準の合理性・公正性
3. 被解雇者への説明と協議
4. 社会通念上の相当性

**試験での典型パターン**：
- 「A さんは仕事が遅いから解雇」→ 改善指導・配置転換がない → 権利濫用で無効
- 「経営難で人員削減、賃金の高い順に解雇」→ 人選基準が定量的でない → 権利濫用で無効

### 雇止め法理（労働契約法19条）

有期雇用契約が複数回更新されると、実質的には無期契約と同視される場合があります：

```
労働契約法19条：
有期労働契約が通算3年以上継続して更新されると、
その雇止めには、解雇と同等の理由が必要とされる。
```

**雇止めと解雇の区別**：

| 軸 | 有期契約の雇止め | 無期契約の解雇 |
| --- | --- | --- |
| **契約形態** | 期間満了に伴う終了 | 使用者の一方的な終了 |
| **法的根拠** | 労働契約法19条（反復更新の場合） | 労働契約法16条（権利濫用法理） |
| **必要理由** | 通算3年以上なら解雇と同等 | 客観的に合理的な理由 |
| **判例動向** | 通算3年で実務的にほぼ無期視 | より厳格に適用傾向 |

**「通算3年」の意味**：
- 連続する3年間ではなく、複数契約を合算した期間
- 例：1年契約を3回更新 = 通算3年 → 雇止めに解雇権濫用法理を類推

### 無期転換ルール（労働契約法18条）【2013年改正で新設】

```
労働契約法18条：
有期労働契約が通算5年を超えて反復更新された場合、
労働者の申込みにより、
当該有期労働契約は、
期間の定めのない労働契約に転換する。
```

**最重要ポイント**：
- **5年超**が門（3年ではない）
- **労働者の申込み**が必須（使用者の義務的適用ではない）
- 申込みは「いつでも」可能（時期制限なし）
- 転換後は無期契約で、解雇権濫用法理が適用

**試験での計算**：
- 2010年4月から3年契約を更新：2010年4月＋2013年4月（2回目）＋2016年4月（3回目）→ 通算6年→ 5年超該当 → 2016年4月時点で申込み可能
- 5年6ヶ月で初めて申込み → 無期転換成立は申込み時点

## 労働組合法：団体関係と不当労働行為

### 労働三権と基本的枠組み

```
日本国憲法28条：
勤労者の団結する権利及び団体交渉をする権利
並びに団体行動をする権利は、
これを保障する。
```

労働三権は**憲法で保障** されており、労働組合法がこれを具体化します：

#### 三権の概要

| 権利 | 定義 | 具体的内容 |
| --- | --- | --- |
| **団結権** | 労働組合を組織する権利 | 労働組合の設立、加入、維持 |
| **団体交渉権** | 労働条件について使用者と交渉する権利 | 給与・労働時間・休暇等の交渉 |
| **団体行動権** | 争議行為を行う権利 | ストライキ、作業遅延、ボイコット等 |

### 不当労働行為の4類型（最重要）

不当労働行為は、労働三権を侵害する使用者の行為です。労働委員会の救済対象になります：

#### 1. 不利益取扱い（労働組合法7条1号）

```
不利益取扱い：
労働者が労働組合の組合員であること、
又は組合活動をしたことを理由として、
解雇、降給、配転、昇進差別等の
不利益な取扱いをする行為
```

**具体例**：
- 組合加入を理由に給与カット
- ストライキ参加者を解雇
- 組合幹部候補者を左遷

**試験での見分け方**：
- 「勤務成績が悪いから」という理由付けがあっても、実質は組合活動が原因 → 不利益取扱いで無効

#### 2. 黄犬契約（労働組合法7条2号）

```
黄犬契約：
労働者が労働組合に加入しない、
又は脱退することを雇用条件にする行為
```

**具体例**：
- 「うちの会社は組合がない会社です。同意してくれますか？」と誓約書署名を強要
- 「組合加入禁止」を雇用契約に盛り込む

**現代での頻出**：個人事業主化、フリーランス契約による組合加入の実質的排除

#### 3. 団体交渉拒否（労働組合法7条3号）

```
団体交渉拒否：
労働組合又はその代表者が、
労働条件その他に関する事項について、
誠実に団体交渉をすることを求めた場合に、
正当な理由のない拒否
```

**重要な判定軸**：
- **「正当な理由」の有無**が争点になる
- 「話し合う予定がない」は正当な理由ではない
- 「組合代表者が国籍外国人」というだけでは正当な理由にならない

**試験での頻出パターン**：
- 労組が「給与交渉」を申し入れ、使用者が「会社の経営判断だから交渉対象外」と拒否 → 不当労働行為（給与は典型的な団交事項）
- 労組の提案が「非現実的」→ でも話し合い拒否は不当労働行為

#### 4. 支配介入（労働組合法7条4号）

```
支配介入：
労働組合の結成、運営若しくは保持に関し、
経費の援助その他の行為をする、
又は労働組合の活動を妨害する行為
```

**具体例**：
- 使用者が「親睦会」名義で金銭援助 → 実質的には使用者友好的な組織化
- 管理職が「別の組合を作ってくれ」と勧誘
- 組合運動を監視・報告させる

**「経費援助」の怖さ**：単なる物質的援助だけでなく、施設提供、通信インフラ提供なども該当

### 労働委員会による救済

不当労働行為があると、労働委員会が救済できます：

| 救済内容 | 具体的措置 | 試験での位置付け |
| --- | --- | --- |
| **原状回復** | 不当に解雇された者の復職、差別待遇の是正 | 実質的な救済 |
| **金銭補償** | 不利益期間の給与支払い、慰謝料相当額 | 原状回復が難しい場合 |
| **中止命令** | 不当労働行為の中止、再発防止措置 | 予防的措置 |

**手続**: 不当労働行為の申し立て → 労働委員会の調査 → 斡旋 → 調停 → 審問（審判） → 救済命令

## ハラスメント防止と職場環境整備

### パワーハラスメント（事業主の防止措置が必須）

2019年の改正労働施策総合推進法により、パワーハラスメント防止が事業主の義務化されました：

```
事業主は、職場におけるパワーハラスメントを
防止するため、雇用管理上必要な措置を講じなければならない。
```

#### パワハラの定義と6類型

パワハラは「優越的な立場を背景にした言動」で、業務上必要な範囲を超えるものです：

| 類型 | 具体例 | 対象者 |
| --- | --- | --- |
| **身体的攻撃** | 殴打、蹴り、物投げ | 全ハラスメント共通 |
| **精神的攻撃** | 人格否定、脅迫、侮辱 | 最頻出 |
| **人間関係からの隔離** | 無視、村八分、異動強要 | グループ外しとして頻出 |
| **過大な要求** | 不可能な納期、重大な見直し | 管理職の権限濫用パターン |
| **過小な要求** | 意図的な無職状態、左遷 | 退職強要として頻出 |
| **個の侵害** | 私生活への過干渉 | SNS チェック、恋愛関係への干渉 |

#### 事業主の防止措置（4つの柱）

1. **方針明確化・周知啓発**：ハラスメント禁止方針を就業規則に明記、研修実施
2. **相談体制整備**：相談窓口の設置、プライバシー保護、相談者保護
3. **事後の迅速対応**：事実確認、被害者サポート、加害者処分
4. **再発防止**：職場環境改善、行為者の教育

**試験での見分け方**：
- 「上司による指導」と「パワハラ」の境界 → 業務上の必要性、対象者の理解可能性、態様の相当性で判定
- 「パワハラだから解雇」ではなく「事前の防止措置がなかったのか」が問われる

### セクシュアルハラスメント

```
セクシュアルハラスメント：
労働者の意に反する性的言動で、
労働者がそれを理由に
不利益を受けたり、
職場環境が悪化したりする行為
```

#### 2つの類型

| 類型 | 定義 | 具体例 |
| --- | --- | --- |
| **対価型** | 採用・昇進などの人事行為を対価に性的言動 | 「昇進させるから」と迫る |
| **環境型** | 性的言動が職場環境を著しく害する | 工事現場での裸体画像掲示 |

#### 事業主の措置

パワハラと同等：相談窓口、事実確認、被害者保護、行為者処分、再発防止

### マタニティハラスメント（妊娠・出産関連）

```
妊娠・出産・育児休業等に関するハラスメント：
労働者が妊娠・出産したことを理由とする
不利益取扱いや、
育児休業取得を阻害する言動
```

**具体例**：
- 妊娠報告時に「戦力外」と言われる
- 育児休業取得予定者を配転
- 時短勤務制度の利用者を昇進対象外とする

**男女雇用機会均等法との関係**：
- マタハラは均等法の性差別と重なる論点
- 事業主の防止措置は均等法でも規定

## その他の重要労働法規（概論）

### パートタイム・有期雇用労働法：同一労働同一賃金

```
2020年改正により、短時間・有期雇用労働者の
不合理な待遇差の禁止が法律化
```

**禁止される待遇差**：
- 基本給の大幅な差
- 賞与・手当の支給区別（合理的理由がない場合）
- 教育訓練機会の制限

**合理的な待遇差の要素**：
- 職務内容の相違
- 配置転換の可能性の相違
- 勤続年数の相違

### 労働者派遣法：派遣三者関係

派遣労働は、派遣元と派遣先に分かれるため、以下の関係規制：
- **派遣禁止業務**：建設、警備、医療等の一部業務は派遣不可
- **派遣期間制限**：原則3年（同一派遣先での派遣期間上限）
- **雇用契約義務**：派遣元での雇用契約が必須

### 育児休業・介護休業法

| 制度 | 期間 | 対象 |
| --- | --- | --- |
| **育児休業** | 子が1歳まで（最大2歳） | 男女問わず対象 |
| **パパ育休** | 最大4週間（子が1歳2ヶ月まで） | 配偶者の出産予定から取得可能 |
| **介護休業** | 通算93日（分割可） | 家族の介護が必要な場合 |

**試験ポイント**：育児休業中の給与は「無給」が原則（ただし雇用保険の失業給付に準じた給付あり）

### 男女雇用機会均等法

- **募集・採用の性差別禁止**
- **配置・昇進・教育訓練での性差別禁止**
- **間接差別の禁止**：一見中立的だが結果的に一方の性を不利にする条件

## 法規制の優先順位：三層構造での整理

試験では、常に以下の優先順位を適用してください：

```
労基法 ≧ 労働協約 ≧ 就業規則 ≧ 労働契約
```

### 層別での判定フロー

#### 第1段階：労基法違反か

- **時間外45時間超（月）や360時間超（年）**か → 違法（上限規制違反）
- **36協定なしの時間外労働**か → 違法
- **割増賃金25%未満**か → 違法
- **年次有給休暇10日未満**か（6ヶ月継続勤務者）→ 違法

労基法違反なら **即座に無効**。下位の就業規則・労働契約は関係なし。

#### 第2段階：労働協約の基準か

労働協約（労組と使用者の団体交渉の結果）がある場合：
- 労協約 > 就業規則 > 労働契約
- 労協約の基準を下回る就業規則は無効

#### 第3段階：就業規則が「合理的」か

就業規則が有効になるための要件：
- **周知**されていること
- **合理的な内容**であること
  - 不利益変更の場合：7要素で合理性判定
  - 新規規則の場合：労基法以上であること

#### 第4段階：個別労働契約で上乗せ

- 就業規則を上回る条件なら有効（労働者に有利）
- 就業規則を下回る条件なら無効→就業規則が適用

---

## 典型的なつまずき

- **法規制の優先順位を逆にする**：「就業規則に書いてあるから」で労基法を下回る条件を認める
- **36協定の意味を誤解**：36協定があれば「無制限に時間外OK」と勘違い（2019年改正で上限規制あり）
- **割増賃金の複合計算**：深夜+時間外で「50%」と加算してしまう（最高率で統一）
- **就業規則の不利益変更を一律判定**：「不利益変更は全てNG」と過度に保護する（合理的なら有効）
- **解雇権濫用と雇止めを混同**：有期契約での終了と無期契約での解雇を区別しない
- **無期転換ルールの5年を3年と誤記**：「3年の反復更新で無期転換」は誤り
- **ハラスメントを「倫理」の問題だけで終わらせる**：事業主の防止措置義務が法的には最重要
- **不当労働行為の類型数を誤記**：4類型（不利益取扱い、黄犬契約、団交拒否、支配介入）をしっかり区別

---

## 問題を解くときの観点

1. **法規制の層を同定する**：労基法か、契約・就業規則か、団体関係か、職場環境か
2. **優先順位を確認**：労基法 > 労働協約 > 就業規則 > 労働契約で上位の基準を適用
3. **合理性を多角判定**：
   - 不利益変更なら7要素で判定
   - 解雇権なら客観的理由と社会通念上の相当性
4. **数字を計算**：36協定・割増賃金・有給休暇・無期転換の5年など、具体的な計算
5. **ハラスメント防止措置が「できていたか」**：違反判定だけでなく、予防措置の有無
6. **団体関係か個人か**：不当労働行為なら団体が関与、ハラスメントなら個人被害
7. **事例Ⅰと組織文化をリンク**：人事制度、評価システム、育成方針などとの整合性

## 確認問題

<Callout title="問1：労働基準法と就業規則の優先順位">
    A社は、経営難を理由に就業規則を変更し、全従業員の月給を30%カットすることを決定しました。労働組合との団体交渉を設定する計画でしたが、経営者判断で「経営判断であり交渉対象外」として一方的に実施しました。

    1. この給与カットは有効か
    2. 労使協定(36協定)がなければ時間外労働はどうなるか
    3. 労働者側が「不合理な変更」として争った場合の判定基準は何か

    **解答**：
    1. 無効の可能性が高い。不利益変更として、(1)必要性、(2)内容の相当性、(3)代償措置、(4)労働者代表との協議、(5)業界慣行、(6)条項の性質、(7)段階性の7要素で判定され、協議なし・代償措置なしでは合理性を欠く可能性がある。
    2. 36協定なしの時間外労働は違法。労基法32条（週40時間）の枠を超える労働は原則禁止で、36協定の締結・届出が必須。協定なければ全て時間外手当の対象になる。
    3. 就業規則変更の合理性を7要素で総合判定：企業の経営状況、人選基準、従業員への説明、労組・代表者との協議程度、同業他社の状況、給与の重要性、急激性か段階性かなど。一方的決定は最も合理性を欠く。
</Callout>

<Callout title="問2：割増賃金の計算と複合">
    B社の従業員Xは、月の時間外労働が70時間（日中50時間、深夜20時間）、その月の月給は40万円です。中小企業で月60時間超の割増は50%です。

    1. Xの当月の割増賃金（時間外部分）をいくらか計算せよ
    2. 36協定で「月100時間、年720時間」と特別条項を定めた場合、この月の70時間労働は合法か

    **解答**：
    1. 時間外50時間は25%（通常）、深夜20時間のうち月60時間超部分は50%：月60時間 - 50時間（通常時間外） = 10時間が50%対象 → (40万÷160時間) × 50時間 × 1.25 + (40万÷160時間) × 10時間 × 1.50 = 625万円分 + 150万円分。ただし実務的には基礎単価の算定法により異なる。
    2. 月70時間は月45時間の上限を超過(25時間超)。36協定で月100時間と定めても、2019年改正の上限規制(月45時間が原則、特別条項でも複数月平均80時間など)に違反する可能性がある。協定の妥当性を検証する必要があり、単に「協定あれば合法」ではない。
</Callout>

<Callout title="問3：不当労働行為の類型判定">
    C社では、労働組合が「給与交渉」を申し入れましたが、経営者は「給与は経営判断であり、団体交渉の対象外」と拒否しました。加えて、組合幹部のAさんが人事考課で「Dランク」となり、昇進対象外になりました。

    1. 給与交渉拒否は不当労働行為か
    2. Aさんの人事考課は不利益取扱いか、それとも正当な人事判断か
    3. 労働委員会の救済内容は何か

    **解答**：
    1. 団体交渉拒否に該当する可能性が高い。給与・労働条件は典型的な団交事項であり、「経営判断だから交渉対象外」という理由では正当な理由にならない。使用者は誠実な交渉義務を負う。
    2. Aさんが組合幹部だからという理由で不当に低い考課をつけた場合は、不利益取扱い(労組法7条1号)に該当する。ただし、客観的な成績不良など別の正当理由があれば不当労働行為ではない。実質的に「組合活動」が原因かを判定する。
    3. 労働委員会は: (1)原状回復(昇進、給与遡及支払い)、(2)金銭補償(不利益期間の給与)、(3)中止命令(交渉応諾、再発防止措置)などを命じる。調停が不調に終われば審問・審判に移行。
</Callout>
