本文書は、19年間の出題傾向分析を「具体的な問題例」で補強するもの。

## 事例1：Trap-B「条件すり替え」の実際（会社法から）

### 典型的な Trap-B 問題の構造

```
【問題文】
A社（公開会社）は、現在の取締役会構成を変更することを検討している。
以下の記述のうち、最も適切なものはどれか？

ア. 取締役の任期を2年から3年に延長するには、取締役会の決議が必要である
イ. 監査役の設置を廃止するには、株主総会の決議が必要である
ウ. 取締役会の設置を廃止するには、定款変更が必要である
エ. 監査役の人数を2名から3名に増員するには、取締役会の決議が必要である
```

### 誤答パターン（Trap-B の典型）

**受験生が引っかかりやすい選択肢**：
- ア. 「あ、任期延長は定款事項だから会社法で決まっているはず…でも取締役会で決めるのかな」→ **条件の「定款に書いてあるかどうか」を見落とす**
- エ. 「監査役を増員するなら取締役会か…いや株主総会か…」→ **「設置」と「人数変更」の条件が違う**ことを読み落とす

### 正解への思考フロー（Trap-B対策）

```
条件の確認フロー:
1. 「定款に記載必須か」を確認 → YES なら「定款変更+株主総会」
2. 「法定事項か」を確認 → YES なら「会社法で決定権者が指定」
3. 「経営判断か」を確認 → YES なら「株主総会 or 取締役会」
4. 「誰が決めるのか」に注目 → 「取締役会」「監査役会」「株主総会」の三者択一

問題文の細部を読む:
- 「公開会社」→ 取締役会・監査役会が必須
- 「廃止」「設置」「変更」で条件が異なる
- 「2年から3年」は定款事項 → 定款変更が条件
```

### 正解

**イが正解**。理由：
- 監査役の「設置」は法定事項
- 会社法327条により「株主総会の決議が必要」
- 設置廃止も設置同様、株主総会決議が必須

### 受験者への Trap-B 対策

1. **決定権者フロー表を暗記**
   ```
   【四層の決定権構造】
   ├─ 定款：定款記載事項 → 定款変更（株主総会決議）
   ├─ 法定：会社法で決定権者を指定 → 指定権者の決議
   ├─ 経営判断：取締役会のレベルで判断可能 → 取締役会決議
   └─ その他：株主総会＝最高決定機関
   ```

2. **条件キーワードに注目**
   - 「設置」「廃止」 ← 法定事項
   - 「人数変更」「任期延長」 ← 定款事項
   - 「経営判断」「判断権」 ← 取締役会決議
   - 「必須」「廃止禁止」 ← 会社法で決定

## 事例2：Trap-D「混同誘発」の実際（知財分野から）

### R07 で出題された Trap-D の例

```
【問題文】
X社は、社内で開発した「デジタルマーケティング支援システム」の権利保護を検討している。
以下の記述のうち、著作権について最も適切なものはどれか？

ア. 著作権は自動発生し、著作物の創作時点で権利者が決定される
イ. 著作権の保護期間は、著作者の死後70年（法人の場合は公表後70年）である
ウ. 著作権は著作者の同意なく譲渡できない
エ. 著作権は特許権と同じ「産業財産権」に分類される
```

### Trap-D の誤答ポイント

**選択肢ア**：正解候補
- 著作権は「登録不要」で自動発生
- 創作時点で著作者に帰属
- **特許権との相違** = 特許権は「出願→登録」という手続が必須

**選択肢イ**：Trap-D（混同誘発）の最大の誘発肢
- 著作権の保護期間は「著者死後70年」で正しい
- **しかし選択肢ウでの「譲渡禁止」は誤り**→ 著作権は譲渡可能
- 多くの受験生は「ア＆イで正解」と考えて、イを選んでしまう

**選択肢ウ**：Trap-D（混同誘発）
- 著作権は「譲渡可能」である
- **道徳権（著作者人格権）は譲渡不可だが、経済権（著作権）は譲渡可**
- 特許権と混同（特許権は譲渡可能）

**選択肢エ**：明らかな誤り
- 著作権は「著作権法」で保護される権利
- 特許権・商標権は「産業財産権」
- **著作権は「著作物隣接権」的な位置づけ**
- しかし、初学者は「著作権も権利だから…」と混同する可能性

### 正解

**ア が正解**。理由：
- 著作権は登録不要で自動発生（特許と異なる）
- 創作時点で著作者に帰属（登録制度がない）
- これが「産業財産権」との根本的相違

### R07 実績データでの Trap-D 激増の背景

R07 での Trap-D 40% という高さは、**知財分野で権利の相違が「出題の中核」に昇格**したことを示唆。

- **著作権 vs 特許権 vs 商標権** の三者択一が頻出
- 権利の「自動発生 vs 登録要件」「保護期間」「譲渡可能性」での混同
- **複合問題化**：会社法×知財の「従業員発明の権利帰属」問題も増加

## 事例3：L層進化の実際（2015年会社法改正の影響）

### H27-H30期の「L2への転換」を示す問題例

```
【H27 出題問題（L2：制度理解を要求）】
公開会社 B社は、経営判断原則の適用を受けるか否かについて、監査役会から質問を受けた。
以下の判断フロー(実際は図示)のうち、経営判断原則の適用要件として誤っているものはどれか？

経営判断原則の適用要件：
1. 取締役が会社のため善意で行為
2. 合理的な情報収集の下で判断
3. 因果関係のある判断
4. ~~誤った判定でない~~（＝判断の妥当性は不問）

※実際は図示で複数段階のフロー
```

### L2（制度理解）の特徴

この問題が「L2」とされる理由：
1. 単なる「経営判断原則とは何か」ではなく
2. 「経営判断原則の**適用条件**は何か」という多段階判定
3. さらに「誤った判定でない」という否定形での理解が必須

### H19 との比較（L1中心だった時代）

```
【H19 同様の問題（L1：定義暗記レベル）】
「経営判断原則」について最も適切な説明はどれか？
ア. 取締役の経営判断に対する司法統制を緩和する原則
イ. 取締役の経営判断の結果責任を問わない原則
ウ. 取締役の裁量経営を保障する法的原則
エ. 経営判断が不当な場合、取締役は個人責任を負う

→ 定義をそのまま知っていればよい（記憶力で対応可能）
```

### 15年間での難易度の実質的な変化

| 要素 | H19時代（L1） | H27時代（L2） | 変化 |
|------|-------------|-------------|------|
| 問題文の長さ | 40字 | 150字+ | ↑ 3-4倍 |
| 条件数 | 単一条件 | 複数条件（フロー図） | ↑ |
| 正誤判定の階数 | 1段階（「それは何か」） | 3段階（「いつ、だれが、どう判定」） | ↑ |
| 背景知識の深さ | 表面的理解 | 実務的適用判定 | ↑ |

## 事例4：相続出題の増加傾向（2019年民法改正の影響）

### R2-R6 での相続出題の実際

**2020年度（R2）以降、相続が「準鉄板テーマ」に昇格**

```
【R2 出題例】
配偶者がいる場合の相続分配について、以下のうち最も適切なものはどれか？

ア. 配偶者の相続分は常に「1/2」である
イ. 子が複数いる場合、配偶者の相続分は「1/3」である
ウ. 婚外子と嫡出子では相続分が異なる
エ. 配偶者がいる場合、相続人は配偶者と子のみとなる

【出題の背景】
- 2019年民法改正で「配偶者の相続権拡大」という大改正
- R2（2020年4月施行）から直後の出題
```

### 相続の「段階的な出題増」パターン

| 時期 | 出題内容 | レベル | 背景 |
|------|--------|--------|------|
| H19-R01 | ほぼ出題なし or 1問/2年 | - | 改正前の標準内容 |
| R02-R03 | 基本的な相続分配 | L1-L2 | 改正直後の「知識定着」 |
| R04-R05 | 改正内容（配偶者相続権拡大） | L2-L3 | 改正内容の「実務適用」 |
| R06-R07 | 相続×会社法（経営者の相続） | L3 | 「複合問題化」 |

### R05 での複合問題例（予想）

```
【R05 複合問題例（推測）】
経営者 C氏（会社代表取締役、60歳）が亡くなった。
C氏は妻と成人の子2人を遺す。会社はC氏名義の株式で成り立っている。

以下のシナリオについて、最も適切な対応はどれか？

ア. 妻が相続分「1/2」を株式で受け取る場合、経営権は妻に移る
イ. 相続人全員の同意なく「遺産分割」をすることはできない
ウ. 配偶者相続権の拡大により、妻は「1/2超」の受け取りが可能
エ. 相続放棄をする場合、債権者への通知は不要である

【この問題が「複合」である理由】
- 相続分配（民法）+ 株式相続（会社法）+ 経営権（商法）の三領域
- 単一領域の知識では対応不可
```

## 事例5：出題形式の変化（条文知識型 → 事例問題型）

### 1990年代的な「条文知識型」

```
【古い形式】
会社法 330条について、最も正確な説明はどれか？

ア. 取締役会は3人以上で構成
イ. 取締役会は月1回以上の開催が必須
ウ. 取締役会の議事録は10年間保存必須
エ. 取締役会の決議は全員一致が原則

→ 条文を読んで、その文言を正確に知っているか」のテスト
```

### 現代（R05-R07）の「事例問題型」

```
【現在の形式】
D社（非公開会社、取締役会設置）の取締役会で、以下の決議を予定している。
各決議について、取締役会での決議が「必須」なのはどれか？
複数ある場合はすべて選べ。

a. 会社資産である不動産を売却する
b. 定款変更を提案する
c. 副取締役会長を新設する
d. 損失補償契約を締結する

【正答（複数）】
- a. 資産売却は「取締役会決議が必須」（会社法 362条4項）
- b. 定款変更は「株主総会決議のみ」→ 取締役会は不要
- c. 役員設置は「定款変更必須」→ 取締役会は不要
- d. 損失補償は「取締役会または株主総会」で可能 → 取締役会決議で足りる

→ 「複数の状況を組み合わせて」「各々で決議権者が異なる」ことを「判定」する
```

### 変化の本質

| 視点 | 1990年代型 | 2020年代型 |
|------|----------|----------|
| 測定対象 | 「条文の記憶」 | 「状況判定の能力」 |
| 問題長 | 50字程度 | 200字+ |
| 解き方 | 「条文を引く」 | 「複数条件を組み合わせる」 |
| 推論段数 | 1段階 | 3-5段階 |
| 正誤判定 | 単一肢 | 複数肢 or 記述式（大学型） |

## 事例6：Trap パターンの進化（単純 → 複雑）

### H19 の Trap パターン（単純型）

```
【問題】
著作権について、以下のうち誤っているものはどれか？

ア. 著作権は著作者の死後50年まで保護される
イ. 著作権は契約により譲渡できる
ウ. 著作権は自動発生し、登録は不要である
エ. 著作権は特許権と同じく「産業財産権」である

【Trap分析】
- エ. が明らかな誤り → Trap-D（混同誘発）の典型
- しかし、Trap が「1個」（エのみ）
- 受験生は「エ以外は全部正しい」と直感的に判定可能
```

### R07 の Trap パターン（複雑型）

```
【問題】
X社従業員 Y が、業務で開発した発明の権利帰属について、以下のうち最も適切なものはどれか？

ア. 従業員発明は常に会社に帰属する
イ. 従業員発明でも、契約で従業員に帰属させることが可能
ウ. 従業員発明の場合、従業員は「相応の対価」を請求可能（特許法35条）
エ. 発明が業務範囲外でも、会社が資料や施設を提供していれば「従業員発明」である

【Trap分析】
複合Trapが多層化:
- Trap-B：「相応の対価」の定義が不明確（「いくら」か「いつ」か）
- Trap-D：従業員発明 vs 通常の特許 vs 商標の「帰属ルール」が異なる
- Trap-C：「相応の対価」は「請求権」だが「払う義務」とは異なる
- 問題文が「複合」で「多段階判定」が必須
```

### Trap の進化系統

```
【レベル1】H19-H23：単一 Trap
└─ 「これは明らかに誤り」という肢が1つ

【レベル2】H27-H30：複合 Trap（2種類が一肢に混在）
├─ 正しい部分（ア. で正解）
└─ 誤った部分（イ. の一部が誤り）

【レベル3】R01-R07：多層 Trap（3つ以上の Trap が複雑に絡む）
├─ Trap-B（条件読み取り）
├─ Trap-D（権利の混同）
└─ Trap-C（正解の一部は含まれるが全体は誤り）
```

## 事例7：法改正による「出題テーマの交代」

### 2006年会社法改正の長期影響（H19-H30）

```
【改正の要点】
- 旧・取締役会（会社の統治機関）の廃止可能化
  → 会社法では「設置必須」に変更（公開会社）
- 監査役制度の強化
  → 会社法で「監査役会必須」（一定規模）
- 委員会設置会社制度の導入
  → 経営と監督の分離

【出題への影響】
- H19（施行直後）：取締役会・監査役の「定義」が大量出題（25問中6問）
- H23-H26：「定款で決める」「法定事項」などの区別が深掘り
- H27-H30：「経営判断原則」など、改正後の「実務判定」が主流化
- R01以降：改正内容は「当たり前の前提」に
```

### 2015年会社法改正の短期集中的影響（H28-R02）

```
【改正の要点】
- 役員責任の拡大（取締役の損失補償責任）
- 経営判断原則の法定化（判例を成文化）
- 監査役の権限強化

【出題への影響】
- H28（施行直後）：「役員責任」「経営判断原則」が集中出題（14問中3問）
- H29-R01：実務的な判定問題へ移行（事例問題化）
- R02以降：改正内容は「標準的な知識」に降格
```

### 改正の「出題ライフサイクル」

```
施行年 +0年 → 施行年 +1年 → +2年 → +3年 → +5年 → +10年

【Y = 出題頻度（%）】

大改正（H19）:
30% ── 28% ── 25% ── 22% ── 18% ── 10%
↑          ↑改正テーマは
施行後      3-5年で「準鉄板」化
直後        し、その後緩やかに減衰
集中出題

中程度改正（H28）:
20% ── 18% ── 15% ── 12% ── 8% ── 5%
↑
施行後
1-2年の
集中出題
```

## 事例8：出題難度と配点の相関（R03での配点変更）

### R03 での配点体系変更の意味

```
【R02まで】
100点満点 = 4点 × 25問
├─ 易問（L1）：4点 ×12問 = 48点（確保しやすい）
├─ 中問（L2）：4点 × 10問 = 40点
└─ 難問（L3）：4点 × 3問 = 12点（確保しにくい）

【R03から】
100点満点 = 5点 × 20問
├─ 易問（L1）：5点 × 8問 = 40点（得点源が減少）
├─ 中問（L2）：5点 × 10問 = 50点
└─ 難問（L3）：5点 × 2問 = 10点

【含意】
- 「1問あたりの配点を上げる」= 「1問の失敗がより大きな減点になる」
- つまり「易問での確実な得点」の重要性が低下
- 代わりに「中問（L2）での正確な判定」の重要性が上昇
- この変更が R03 で L2 が 70% に急増した背景
```

## 事例9：受験生の相談カテゴリ別の出題対応

### 典型的な受験相談と出題との対応

```
【相談①】「条文を暗記しているのに、問題が解けません」
→ これは「L1は身についているが L2が不足」という診断
→ 対策：「複数条件フロー図」の作成

【相談②】「選択肢が似ていて、区別がつきません」
→ これは「Trap-D（混同誘発）」に引っかかっている
→ 対策：「権利相違表」（商標×著作権×特許のマトリックス）

【相談③】「いつもラスト5問が分からない」
→ これは「L3（複合問題）」に対応していない
→ 対策：「複数領域の統合問題」の演習

【相談④】「2015年改正以降がわかりません」
→ これは「改正内容の実務適用」が理解不足
→ 対策：「経営判断原則」などの実例研究
```

## 事例10：学習段階別の出題対応戦略

### 初学者向け（0-3ヶ月）：L1 定義暗記

```
【やること】
1. 会社法：「取締役」「監査役」「株主」の定義
2. 知財：「商標権」「著作権」の定義
3. 民法：「相続」「契約責任」の定義

【やらないこと】
- Trap 対策（まだ出題パターンを知らない）
- 複合問題（基礎知識がない）
- 法改正の細部

【目標得点】
```
