## 学習のポイント

このページを読む前提知識として、知的財産権の全体構造（特許、実用新案、意匠、商標との違い）を把握していると理解が深まります。

著作権学習の進め方：

1. **著作物とは何か** → 無方式主義・属地主義の原則
2. **著作物の種類** → どんな創作が保護対象か
3. **著作権の内訳** → 人格権と財産権の違い
4. **法人著作** → 職務著作の5要件
5. **保護期間** → 計算ルール（特に2018年の改正）
6. **権利制限** → 許諾不要で利用できる場合
7. **著作隣接権** → 著作権以外の関連保護

## 試験で何が問われるか

以下の5つの軸から出題されます：

- **発生の仕組み** → 登録制度がなく、創作と同時に発生すること
- **権利の内容** → どのような利用行為が権利侵害になるのか
- **期間の計算** → 著作者の死後何年か、法人著作はいつまでか
- **権利の譲渡可能性** → 人格権と財産権で異なること
- **実務との接点** → 職務著作や引用ルールの実際の活用

## 著作権法の基本原則

### 無方式主義（登録不要）

著作権法は **登録制度を採用しない** のが基本です。著作物は、特許や商標と異なり、創作した瞬間に自動的に著作権が発生します。

```
創作 → 著作権発生（登録不要）
```

- 著作物を完成させた時点で、誰がどこで作ったか関わらず著作権が存在する
- 登録がなくても権利侵害を主張できる
- 登録は「登録しておくと証拠になりやすい」というメリットに留まる

### 著作物は「表現」を保護する

著作権が保護するのは、**創作的に表現されたもの** です。アイデア自体は保護されません。

| 保護される | 保護されない |
| --- | --- |
| 日本語での小説表現 | ストーリーのアイデア |
| 特定の作曲法 | メロディーの概念 |
| プログラムのコード | アルゴリズムそのもの |
| 写真の映像表現 | 被写体となった現象 |

別の言葉で同じアイデアを表現したり、異なるアルゴリズムで同じ機能を実装したりすれば、著作権侵害にはなりません。

### 属地主義

著作権は、**各国の法律に基づいて発生・存続します**。

- 日本での創作物は、日本著作権法で保護される
- 外国での創作物は、その国の法律で保護される
- ベルヌ条約などの国際条約により、外国人の著作物も日本で保護される

## 著作物の種類

著作権法第10条は、保護される著作物として以下の9つのカテゴリーを列挙しています。

### 1. 言語の著作物

- 小説、論文、脚本、詩、エッセイ など
- 日本語に限らず、あらゆる言語で表現されたもの

### 2. 音楽の著作物

- 作曲（楽曲自体）
- 作詞（歌詞）
- 編曲も保護される

### 3. 舞踊・無言劇の著作物

- バレエ、現代ダンス
- 振付け
- 無言劇の演技構成

### 4. 美術の著作物

- 絵画、版画、彫刻、工芸美術品
- 美的享受を目的とした著作物

### 5. 建築の著作物

- 建物のデザイン、構造の創作的表現
- 建設される建物そのもの

### 6. 図形の著作物

- 地図、図面、設計図、グラフ
- 学術的・実用的な図表

### 7. 映画の著作物

- 映画フィルム、映像全体
- 動画表現（テレビ放送のための映像も含む）

### 8. 写真の著作物

- 被写体選択や構図、照明、タイミングに創作性がある写真
- 通常の記念写真も保護されている

### 9. プログラムの著作物

- コンピュータープログラム（ソースコード、オブジェクトコード）
- アルゴリズムやプログラミング言語自体は保護されない

### 特殊な著作物

#### 二次的著作物

既存の著作物を基にして新たに創作された著作物：

- 翻訳作品（英語の小説を日本語に翻訳）
- 改編作品（小説を映画化）
- 編曲作品（既存の曲をジャズにアレンジ）

二次的著作物は、原著作物とは独立した著作権を持ちます。ただし、二次的著作物の利用には、原著作者の許諾が必要です。

#### 編集著作物

複数の著作物や資料を集めて編成した著作物：

- 年鑑、事典、教科書（複数の記事や写真を組み合わせたもの）
- 雑誌、新聞（多くの記事を編成したもの）
- アンソロジー（短編集）

編集著作物の著作権は、その編成・構成の創作性を保護します。掲載される各著作物の著作権とは別に保護されます。

#### データベースの著作物

データベースの編成・構成に創作性がある場合：

- 大量の統計データの独特な配列
- データの分類・整理方法が創作的な場合

#### 共同著作物

複数の著作者が共同で創作した著作物：

- 複数の作詞家・作曲家による楽曲
- 複数の著者による共著本

共同著作物は、各著作者の寄与部分を分割できない場合、著作者全員の同意がないと利用できません。

## 著作権の構成：人格権と財産権

著作権は、大きく2つの柱で成り立っています。この区別が試験での重要ポイントです。

### 著作者人格権（著作者のみが持つ権利）

著作者人格権は、著作者の **人格と名誉を守る権利** であり、著作者以外は持つことができません。

#### 1. 公表権（せんぴょうけん）

著作者だけが、著作物をいつ、どのような形で公表するか決める権利です。

- 未発表の原稿を勝手に出版されることを防ぐ
- 企業内で作られたが未公表の設計図を勝手に公開されることを防ぐ

#### 2. 氏名表示権（しめいひょうじけん）

著作物の利用の際に、著作者の名前（実名でも芸名でも）を表示するよう要求できる権利、または名前を表示しないよう要求できる権利です。

- 「著作者：山田太郎」と表示させる
- 「匿名で発表したい」という要求も認められる

#### 3. 同一性保持権（どういつせいほじけん）

著作物の内容や題名を改変されることを防ぐ権利です。

- 詩の一部を削除されることを防ぐ
- 歌詞の一部を変更されることを防ぐ
- 小説の結末を変えられることを防ぐ

ただし、建築物の改築のような「著作物の本質的な改変でない改修」は認められます。

#### 人格権の重要な特性

| 特性 | 説明 |
| --- | --- |
| **譲渡不可** | 人格権は売却や譲受ができない（著作者に永遠に帰属） |
| **一身専属** | 著作者の人格に密接に結びついているため、著作者本人のみが行使可能 |
| **放棄不可** | 人格権を放棄することはできない（ただし行使しないことは可） |
| **相続不可** | 著作者の死後、人格権は消滅する |

例外：著作者が死亡した場合、人格権の行使は「その著作者と社会的に密接な関係にあった者」が代理できます（著作者の遺族など）。

### 著作財産権（譲渡・ライセンス可能な権利）

著作財産権は、著作物から **経済的な利益を得る権利** です。人格権と異なり、買売や譲渡が可能です。

#### 1. 複製権

著作物をコピーする権利を独占します。

- 印刷（本、パンフレット、ポスター）
- 録音・録画
- デジタル複製（データ化）
- 複写機やスキャナでのコピー

複製は、「目的」や「形式」を問わず、複製された時点で権利侵害になります。

#### 2. 上演権・演奏権

著作物を公開で演じたり、演奏したりする権利を独占します。

- 演劇作品の舞台公演
- 楽曲のコンサート演奏
- ダンス作品の舞踏

ライブ配信も上演・演奏に含まれます。

#### 3. 公衆送信権

著作物を、インターネットやテレビなどで多数の者に届ける権利を独占します。

- ウェブサイトでの公開
- YouTube などの動画配信
- テレビ放送
- ラジオ放送

最近の改正では、この権利が強化され、ストリーミング配信やダウンロード配信も含まれます。

#### 4. 口述権

著作物を、話し言葉で公開すること（朗読や講演）を独占します。

- 書籍の著者が、自分の本を朗読して公開する
- ラジオでのニュース原稿の読み上げ

#### 5. 展示権

美術作品や写真などを、公開で展示する権利を独占します。

- 絵画をギャラリーに展示する
- 彫刻を屋外に設置する

#### 6. 頒布権

映画作品のみに適用される権利で、映画を複製物のかたちで販売・貸出しする権利を独占します。

- DVDやブルーレイの販売
- ビデオレンタル店での貸出し

注意：映画以外の著作物には頒布権は適用されません。

#### 7. 譲渡権

著作物の原本や複製物を、有償で第三者に譲渡する権利を独占します。

- 絵画の原本を売却する際、絵画を買った人は、その絵を別の誰かに売ることができるが、著作者は売却のたびに利益を得る権利がある（ただし、限定的に適用）

#### 8. 貸与権

著作物の複製物を、有償で貸与することを独占します。

- 図書館での書籍貸出し（実際には制限あり）
- ゲームソフトのレンタル

#### 9. 翻訳権・翻案権

著作物を別の言語や形式に変換する権利を独占します。

- 小説を映画化する（翻案）
- 小説を他言語に翻訳する（翻訳）
- プログラムを改変する（翻案）

#### 財産権の重要な特性

| 特性 | 説明 |
| --- | --- |
| **譲渡可能** | 財産権は売却、譲受、ライセンス供与が可能 |
| **相続可能** | 著作者が死亡した場合、相続人に相続される |
| **期間制限** | 保護期間が限定されている（詳細は後述） |
| **部分的譲渡** | すべての権利を一括譲渡することも、特定の権利のみを譲渡することも可能 |

## 著作者人格権と著作財産権の比較表

| 観点 | 著作者人格権 | 著作財産権 |
| --- | --- | --- |
| **何を守るか** | 著作者の名誉と人格的利益 | 著作物からの経済的利益 |
| **主な権利** | 公表権、氏名表示権、同一性保持権 | 複製権、上演権、公衆送信権など |
| **譲渡** | 不可（著作者に専属） | 可能（売却・ライセンス可）|
| **相続** | 不可（著作者死後消滅） | 可能（相続人に相続） |
| **放棄** | 不可 | 可能（権利を手放せる） |
| **保護期間** | 著作者の死後も一定期間代理行使可 | 著作者の死後70年（原則） |

## 法人著作（職務著作）

企業の従業員が職務上著作物を創作した場合、その著作権は従業員ではなく、法人（企業）に帰属する場合があります。これを **法人著作** または **職務著作** と呼びます。

### 法人著作が成立する5つの要件

著作権法第15条では、以下の5つの要件をすべて満たす場合、著作者は法人となります：

#### 1. 法人発意（ほうじんはつい）

著作物の創作が、法人の発案・指示に基づくこと。従業員の個人的な創意工夫ではなく、法人として企画・指示したものである必要があります。

- 企業が「マニュアルを作成せよ」と指示して作られたマニュアル
- 会社が「広告キャンペーン用の動画を制作せよ」と指示した動画

#### 2. 業務従事者

その著作物の創作に業務として従事した者（従業員）によって創作されていること。

- 出版社の編集部員が本を編集する
- ソフトウェア企業のプログラマがプログラムを開発する

#### 3. 職務上（しょくむじょう）

創作が、従業員の職務範囲内で行われていること。

- マーケティング部員が、その部門の職務として営業資料を作成
- 研究所の研究員が、その研究の一環としてレポートを作成

#### 4. 法人名義

著作物が法人の名義で公表される、または法人が著作者として公表すること。

- 出版物が「ABC株式会社著」と表示される
- ソフトウェアが「XYZ会社製」として販売される

#### 5. 別段の定めがない

雇用契約や就業規則で「著作権は従業員に帰属する」など、異なる規定がないこと。

当事者間で明示的に「著作権は著作者個人に帰属する」と約束されていれば、法人著作は成立しません。

### プログラムの法人著作の特例

プログラムについては、上記5要件の中の **「法人名義公表」要件が不要** です。

- プログラムが非公開（社内用）であっても、法人名義で開発されていれば、法人著作と認められます
- これは、プログラムが多くの場合、社内のみで使用される性質に対応した例外ルールです

### 職務発明との違い

**職務著作** は著作権法のルール、**職務発明** は特許法のルールです。

| | 職務著作（著作権法） | 職務発明（特許法） |
| --- | --- | --- |
| **権利帰属** | 法人に帰属（デフォルト） | 従業員に帰属（法人に譲受権がある） |
| **デフォルト** | 5要件を満たすと自動的に法人著作 | 従業員発明者に帰属 |
| **契約の役割** | 別段の定めで法人著作を回避可 | 職務発明は適切な対価があれば法人帰属可 |

## 著作権の保護期間

著作権は、いつまで保護されるのか。これは、著作者の死亡時期や著作物の公表時期によって異なります。

### 保護期間の原則（2018年のTPP法改正以降）

#### 一般的な著作物

著作権の保護期間は、**著作者の死後70年** です。

```
著作者の創作 → 著作者の死後70年間保護される → 消滅（パブリックドメイン化）
```

例：A氏が2000年に小説を発表し、2050年に死亡した場合、著作権の保護期間は2050年から70年後の2120年までです。

#### 共同著作物

複数の著作者による著作物の保護期間は、**最後の著作者の死後70年** です。

```
共同著作者A、Bが創作 → Bが2050年に死亡（Aはすでに死亡）
→ 2050年から70年後の2120年まで保護
```

#### 法人著作（法人名義で公表された著作物）

法人著作の保護期間は、**公表後70年** です。

```
法人名義で2000年に公表 → 公表から70年後の2070年まで保護
```

企業が著作者の場合、著作者個人の死亡は関係なく、公表時点から計算されます。

#### 映画の著作物

映画作品の保護期間も、**公表後70年** です。

```
映画が2010年に公開 → 公表から70年後の2080年まで保護
```

映画は通常、複数の著作者（脚本家、監督、撮影者など）を持つため、「公表後」で統一されています。

### 旧法との違い（2018年改正のインパクト）

2018年のTPP対応著作権法改正により、保護期間が大幅に延長されました：

| | 改正前 | 改正後 |
| --- | --- | --- |
| 一般的な著作物 | 著作者死後50年 | 著作者死後70年 |
| 映画・法人著作 | 公表後50年 | 公表後70年 |
| 改正時期 | - | 2018年1月1日 |
| 遡及適用 | - | 改正時点で保護期間内の著作物に適用 |

この改正により、海外（特に欧米）との保護期間が統一され、国際的なコンテンツ流通が容易になりました。

### パブリックドメイン化

保護期間が満了した著作物は、**パブリックドメイン** となり、誰でも自由に利用できます。

- 夏目漱石の『吾輩は猫である』（著作者死後70年超）
- 歌舞伎の古典演目

パブリックドメイン化した著作物は、複製、翻案、映画化など、自由に創作・改編できます。

## 著作権の制限（許諾なく利用できる場合）

著作権は強力な権利ですが、すべての利用が権利侵害になるわけではありません。社会的な必要性から、**著作権者の許諾なく利用できる場合** が定められています。

### 1. 私的使用のための複製（第30条）

著作物を、個人的に、家族など限定的な範囲で複製する場合は、著作者の許諾が不要です。

**許諾不要な例：**
- 書籍を自分で手書きで写す
- テレビ番組を自宅で録画
- CD から自分用に MP3 に変換

**許諾が必要な例：**
- 複製物をインターネットで配布する
- 複製物を販売する
- 営利目的での複製

重要な制限：**技術的保護手段（DRM）を回避してコピーすることは、この例外に該当しません**。違法ダウンロードなどは、私的使用でも許諾が必要です。

### 2. 付随対象物への複製（第30条の2）

別の著作物に付随して複製される場合。

例：映画の背景に映り込んだポスターや、ニュース報道に映り込んだ看板。これらを許諾なく複製・放送できます。

### 3. 情報解析による複製等（第30条の4）

AI 学習のためのデータマイニングなど、情報解析目的での複製は、著作物の著作権者の利益を不当に害しない限り、許諾が不要です。

例：テキストマイニング、統計分析での大量データ利用。

### 4. 引用（第32条）

既存の著作物を、新たな著作物の中で参考資料として引用する場合。

**引用が認められる条件：**

- **主従関係**：引用される部分が「従」、引用元の著作物が「主」であること
- **明示性**：「ここから引用」「引用元：〜」と明確に示すこと
- **引用の必然性**：なぜその著作物を引用する必要があるのかが明確であること
- **出所明示**：著作者名と出所を明記すること

**引用の例：**
- 論文で、他の研究者の論文を参考にして引用
- 教科書で、著名な著作物の一部を参考として掲載
- ニュース記事で、声明や公開情報を引用

### 5. 教育目的での利用（第33、34、35条）

学校教育での利用について、一定条件下では許諾が不要です。

- 学級内での教材としての複製
- 教科書への掲載（著作者に利用料を支払う前提で許諾不要）
- 授業中の上演・演奏

### 6. 図書館での利用（第31条）

図書館が、資料提供や利用者への複製提供を行う場合、一定条件で許諾が不要です。

- 貸出用資料の複製
- 利用者リクエストへの複製提供

### 7. 試験問題での使用（第36条）

大学入試や検定試験などの問題文に著作物を引用する場合。

- センター試験での文学作品の引用
- 資格試験での図表の引用

### 8. 非営利上演・演奏（第38条）

営利目的でない上演・演奏の場合、著作者の許諾が不要（ただし著作隣接権は別）。

例：学校祭でのコンサート、非営利団体での演劇公演。

### 9. その他の重要な制限

- **時事問題の報道（第41条）**：ニュース報道での著作物使用
- **美術作品の改築（第47条）**：建築物の改築・改修
- **障害者向けサービス（第33条の2）**：視覚障害者向けの拡大文字版など

## 著作隣接権

著作権とは別に、著作物の創作に関わった周辺的な関係者も保護されます。これを **著作隣接権** と呼びます。

### 著作隣接権が保護する主体と権利

#### 1. 実演家（じっせんか）

演奏者、歌手、舞踊家、俳優など、著作物を実際に実演する者。

**保有する権利：**
- 録音・録画に対する同意権
- 映像化への対価請求権
- CD などの複製物販売からの対価請求権

**保護期間：** 実演日から70年

#### 2. レコード製作者（れこーどせいさくしゃ）

レコードやCD などの音声録音製品を製作した者（レコード会社など）。

**保有する権利：**
- 複製権
- 公衆送信権（ストリーミング配信での対価請求権）

**保護期間：** 発行（公表）から70年

#### 3. 放送事業者（ほうそうじぎょうしゃ）

テレビ放送やラジオ放送を行う事業者。

**保有する権利：**
- 無線通信による再放送への同意権
- 有線放送への再放送への同意権

**保護期間：** 放送日から50年

#### 4. 有線放送事業者（ゆうせんほうそうじぎょうしゃ）

ケーブルテレビなどの有線放送を行う事業者。

**保有する権利：**
- 再放送への同意権

**保護期間：** 放送日から50年

### 著作権と著作隣接権の関係

音楽CD の販売では、複数の権利者が関わります：

- **著作権者（作詞家・作曲家）** → 作品の創作者として著作権を保有
- **実演家（歌手）** → 歌唱を実演した者として著作隣接権を保有
- **レコード製作者（レコード会社）** → CD 製作者として著作隣接権を保有
- **放送事業者** → ラジオで放送する際の著作隣接権を保有

ストリーミング配信での収益は、これら複数の権利者に配分される仕組みになっています。

## プログラムの著作物

プログラムは、著作権法で明確に著作物として保護されています。

### 保護される範囲

- **ソースコード** → 人間が読む形式のプログラム
- **オブジェクトコード** → コンパイル後の機械語形式のプログラム

### 保護されない範囲

- **プログラミング言語** → Java、Python などの言語自体
- **規約・仕様** → HTML の標準仕様など
- **アルゴリズム** → ソートやハッシュなどの計算方法

別のプログラミング言語で同じアルゴリズムを実装すれば、著作権侵害にはなりません。

### プログラム著作権の実務的側面

企業開発では、**法人著作ルール** が重要です：

- 企業の指示で従業員が開発したプログラム → 企業が著作者
- 独立系プログラマの受託開発 → 契約で著作権帰属先を決定
- オープンソースプログラム → ライセンス条件に従う（GPL など）

## 典型的なつまずき

### ❌ つまずき1：著作権は登録しないと発生しない

**誤り** → 「著作権を得るために登録が必要」と思う。

**正解** → 著作権は、創作した瞬間に自動発生します。登録は、後々紛争が生じた場合に「著作権が自分にあること」を証明しやすくするためのオプションです。

### ❌ つまずき2：著作者人格権と著作財産権の混同

**誤り** → 「著作者人格権も売ることができる」「著作財産権は相続できない」と思う。

**正解** → 人格権は譲渡不可・相続不可（著作者専属）。財産権は譲渡可能・相続可能です。

### ❌ つまずき3：すべての著作物に同じ保護期間

**誤り** → 「著作物は常に著作者死後50年」と覚える。

**正解** → 一般的には死後70年ですが、法人著作・映画は公表後70年です。2018年の改正で50年→70年に延長されました。

### ❌ つまずき4：引用と複製の混同

**誤り** → 「出所を明記すれば、どの量でも引用できる」と思う。

**正解** → 引用は主従関係が必要です。引用部分が「従」で、著者の著作物が「主」である必要があります。

### ❌ つまずき5：著作隣接権を著作権と同等と考える

**誤り** → 「実演家も著作権の著作者である」と思う。

**正解** → 実演家、レコード製作者、放送事業者は著作権ではなく、著作隣接権を保有します。保護期間も異なります。

### ❌ つまずき6：プログラムの法人著作で名義公表要件

**誤り** → 「プログラムも法人名義で公表する必要がある」と思う。

**正解** → プログラムの法人著作は、公表・非公開を問わず、法人発意・職務上の5要件で成立します。

## 問題を解くときの観点

### 1. 問われているのは「何」か

**「発生」「内容」「期間」のいずれかを先に判別**

- 「著作権が発生するか？」 → 無方式主義、創作と同時に発生
- 「誰が権利を持つか？」 → 法人著作の5要件、著作隣接権の主体
- 「いつまで保護されるか？」 → 保護期間計算（死後70年、公表後70年など）

### 2. 権利の性質を判別

**「人格権か、財産権か」から、譲渡・相続の可否を判定**

- 「著作者は財産権を譲渡できるか？」 → はい。譲渡可能な著作財産権
- 「著作者の遺族が人格権を行使できるか？」 → 有限（人格権は著作者死後も一定期間代理行使可）

### 3. 著作権制限の適用可能性を検討

**「許諾なく利用できるか」を見極める**

- 「個人用の複製ですが、違法ダウンロードではないので大丈夫？」 → DRM を回避していれば違法
- 「学校の授業で本を複製してもいい？」 → 学級内教材用なら許諾不要（第33条）

### 4. 他の知的財産権との違いを意識

**特許、商標、意匠などとの違いを常に念頭に**

| | 著作権 | 特許 | 商標 |
| --- | --- | --- | --- |
| 登録 | 不要 | 必須 | 必須 |
| 保護対象 | 表現 | 技術（発明） | ブランド |
| 保護期間 | 死後70年 | 出願後20年 | 登録後10年（更新可） |

## 確認問題

<Callout title="問1">
    次の記述のうち、著作権法上、正しいのはどれか。

    **A** 著作権を得るためには、著作物を著作権事務所に登録する必要がある。

    **B** 著作物は、創作と同時に著作権が自動発生する。

    **C** 著作権の保護期間は、すべての著作物において著作者の死後50年である。

    **D** 著作者は、著作財産権を他人に譲渡することはできない。

    ---

    **解答：B**

    著作権法は無方式主義を採用しており、著作物は創作した時点で自動的に著作権が発生します。登録制度はありません。また、保護期間は一般的には著作者の死後70年です（2018年改正で50年から延長）。著作財産権は譲渡可能です。
</Callout>

<Callout title="問2">
    企業 X が、従業員 Y に対し「当社のマニュアルを作成して欲しい」と指示し、Y がこれに従ってマニュアルを作成しました。マニュアルは企業 X の名義で公表されました。この場合、マニュアルの著作権は誰に帰属するか。

    **A** 従業員 Y

    **B** 企業 X

    **C** 従業員 Y と企業 X の共同著作

    **D** マニュアルには著作権が発生しない

    ---

    **解答：B**

    これは職務著作（法人著作）の典型例です。（1）法人発意（企業が指示）、（2）業務従事者（従業員が業務として従事）、（3）職務上（業務範囲内）、（4）法人名義（企業名義で公表）、（5）別段の定めなし の5要件をすべて満たしています。したがって、著作者は企業 X であり、著作権は企業 X に帰属します。
</Callout>

<Callout title="問3">
    著作者人格権と著作財産権に関する記述のうち、正しいのはどれか。

    **A** 著作者人格権は譲渡可能であり、著作財産権は譲渡不可である。

    **B** 著作者人格権は譲渡不可であり、著作財産権は譲渡可能である。

    **C** 著作者人格権と著作財産権は、ともに譲渡可能である。

    **D** 著作者人格権と著作財産権は、ともに譲渡不可である。

    ---

    **解答：B**

    著作者人格権（公表権、氏名表示権、同一性保持権）は著作者の人格に密接に関わるため、譲渡も放棄もできません。一方、著作財産権（複製権、上演権、公衆送信権など）は著作者の経済的利益を保護するもので、売却やライセンス供与など譲渡が可能です。
</Callout>
