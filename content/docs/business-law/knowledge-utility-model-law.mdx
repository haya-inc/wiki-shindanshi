## このページの役割

<Callout title="このページの役割">
    実用新案法は、特許法に並ぶ重要な知的財産権制度です。このページでは、実用新案の保護対象（考案、物品の形状・構造・組合せのみ）、無審査登録主義の仕組み、特許法との本質的な違い、実用新案技術評価書の役割、そして実用新案登録に基づく特許出願の制度を体系的に学びます。診断士試験では、保護対象の限定や権利行使の仕方を含めた正確な理解が必須です。
</Callout>

<Callout title="学習前の確認">
    実用新案を「簡易版特許」と雑に理解すると、試験で引っかかります。特に「審査の有無」「権利行使の方法」「保護対象の限定」の3つが特許と決定的に異なります。これらの違いをテーブルで整理し、具体例で確認することが合格への第一歩です。
</Callout>

## 学習のポイント

1. **実用新案の定義と保護対象**
   - 考案とは：技術的思想の創作（高度な発明性は不要）
   - 保護対象：物品の形状、構造、組合せのみ
   - 非対象：方法、プログラム、物質そのもの

2. **無審査登録主義**
   - 方式審査のみで登録（新規性・進歩性は問わない）
   - 登録の速さが特許との大きな利点
   - 一方、権利の安定性は不確定

3. **特許法との体系的比較**
   - 審査のあり・なし、存続期間、補正可能性、出願公開
   - 権利行使時の差異（技術評価書の必須化）
   - 法的効果の違い

4. **実用新案技術評価書**
   - 権利行使の前提条件
   - 提示義務と損害賠償責任
   - 権利者の戦略的判断

5. **実用新案登録に基づく特許出願**
   - 優先権の活用
   - 3年以内の手続きと権利の付け替え

## 試験で何が問われるか

- **基本問題**：実用新案の保護対象を特許・意匠と区別できるか
- **比較問題**：審査、期間、補正可能性など体系的な違いを説明できるか
- **応用問題**：技術評価書の役割と権利行使のリスクを判断できるか
- **複合問題**：実用新案から特許への切り替え戦略を評価できるか

## 実用新案法の概要

### 実用新案とは

実用新案は、**物品の形状、構造、または組合せに関する技術的思想の創作**を保護する知的財産権です。特許法と異なり、発明の「高度性」を求めません。

**実用新案の定義（実用新案法第3条）**
- 発明ではなく「考案」：技術的思想であること
- 創作性：完全なオリジナルであること
- 実用性：産業上の利用可能性があること

### 保護対象の正確な理解

実用新案が保護する対象は、**物品のみ**です。

| 対象範囲 | 保護される | 保護されない |
| --- | --- | --- |
| **物品の形状** | ○ 自動車のボディ形状、容器の形 | - |
| **物品の構造** | ○ 機械の部品配置、接合方法 | - |
| **物品の組合せ** | ○ 複数部品の組み合わせ方 | - |
| **方法・プロセス** | ✗ | ○ 製造方法、処理工程 |
| **物質そのもの** | ✗ | ○ 化学物質の新規組成 |
| **プログラム** | ✗ | ○ ソフトウェア |
| **ビジネスモデル** | ✗ | ○ サービス方法 |

この**物品に限定される**という点が、実用新案の最も重要な特徴です。

## 無審査登録主義：実用新案権の最大の特徴

### 方式審査のみで登録される

実用新案は、**方式要件（出願書類の形式）の確認のみ**で登録されます。

**実審査項目（実施される）**
- 出願書類の形式が整っているか
- 出願費用は払われているか
- 出願人の表示は正確か

**実体審査項目（実施されない）**
- 新規性はあるか（先行技術との比較なし）
- 進歩性はあるか（当業者の知識との比較なし）
- 実施可能性は十分か

### 登録から権利行使までの流れ

1. **出願** → 2. **方式審査（1〜3ヶ月）** → 3. **登録** → 4. **権利行使時に技術評価書の提示**

この流れにより、実用新案は特許より**迅速に権利化**できます。ただし、権利の安定性は確保されません。

### 無審査登録主義のメリット・デメリット

| 観点 | メリット | デメリット |
| --- | --- | --- |
| **迅速性** | 1〜3ヶ月で登録 | - |
| **コスト** | 審査に時間がかからない | - |
| **権利の安定性** | - | 新規性・進歩性が未確認 |
| **権利行使** | - | 技術評価書の提示が必須 |
| **訴訟リスク** | - | 無効の可能性が高い |

## 特許法との体系的比較（最重要）

実用新案法と特許法の違いを体系的に整理することが、試験合格の鍵です。

### 保護対象の違い

| 比較軸 | 特許法 | 実用新案法 |
| --- | --- | --- |
| **保護する概念** | 発明 | 考案 |
| **高度性の要件** | あり（進歩性、非自明性） | なし（技術的工夫があればよい） |
| **物品保護** | ○ | ○ |
| **方法保護** | ○ | ✗ 物品のみ |
| **化学組成保護** | ○ | ✗ 物品のみ |
| **プログラム保護** | ○ 限定的 | ✗ |

### 審査と登録の違い

| 比較軸 | 特許法 | 実用新案法 |
| --- | --- | --- |
| **審査方式** | 実体審査（実施） | 無審査登録主義 |
| **新規性審査** | ○ 実施 | ✗ 未実施 |
| **進歩性審査** | ○ 実施 | ✗ 未実施 |
| **先行技術調査** | ○ 実施 | ✗ 未実施 |
| **登録までの期間** | 平均2〜3年 | 平均1〜3ヶ月 |

### 存続期間の違い

| 比較軸 | 特許法 | 実用新案法 |
| --- | --- | --- |
| **権利存続期間** | **出願日から20年** | **登録日から10年** |
| **期間計算の起点** | 出願日（早い） | 登録日（遅い） |
| **保護期間の実質** | 出願から最大20年 | 登録から最大10年 |
| **商業化までの時間** | 最初の2〜3年は権利なし | ほぼ即座に権利あり |

出願日から登録日までに1〜3ヶ月かかるため、**実用新案の実質保護期間は登録日から最大10年**と考えるべきです。

### 出願公開の有無

| 比較軸 | 特許法 | 実用新案法 |
| --- | --- | --- |
| **出願公開** | ○ あり（出願から18ヶ月） | ✗ なし |
| **登録公報** | ○ あり（権利成立後） | ○ あり（登録後） |
| **秘密性** | 出願から公開まで秘密 | 登録まで秘密（1〜3ヶ月） |

実用新案は登録されるまで秘密とされるため、競争相手に情報が漏れにくいというメリットがあります。

### 補正可能性の違い

| 比較軸 | 特許法 | 実用新案法 |
| --- | --- | --- |
| **補正（修正）** | ○ 可能（条件あり） | ✗ 原則不可 |
| **実務的意味** | 拒絶理由に対応可能 | 出願時の記載が決定的 |
| **権利の柔軟性** | 高い | 低い |

実用新案は補正ができないため、**出願時の明細書の記載が極めて重要**です。

### 権利行使の方法の違い

| 比較軸 | 特許法 | 実用新案法 |
| --- | --- | --- |
| **権利行使の方法** | 直接行使可能 | 技術評価書の提示が必須 |
| **提示の義務** | ✗ なし | ○ あり |
| **権利主張の根拠** | 特許登録 | 特許登録 + 技術評価書 |
| **提示を怠った場合** | - | 損害賠償請求される可能性 |

## 実用新案技術評価書の仕組みと重要性

### 技術評価書とは

実用新案技術評価書は、特許庁が実用新案登録に関する考案につき、**その新規性、進歩性、実施可能性について評価を示した文書**です。

**技術評価書の内容**
- 新規性があるかどうかの評価
- 進歩性（創作容易性）の判断
- 実施可能性の確認
- 先行技術に対する比較分析

### 権利行使時の技術評価書提示義務

実用新案法第29条の2により、実用新案権者が自身の権利を行使する際、**相手方に技術評価書を提示しなければならない**という重要なルールが定められています。

**提示が求められる場面**
- 侵害警告書の送付
- 差止請求訴訟の提起
- 損害賠償請求訴訟の提起
- その他の権利行使

### 提示を怠った場合の法的リスク

実用新案技術評価書を提示せずに権利行使（侵害警告など）を行った場合、以下のリスクが生じます。

**実用新案法第29条の3**
- 権利者が相手方に対して不当な利益を与えるおそれのある警告または請求をした場合
- 相手方に対して損害賠償責任が生じる（不正競争防止法と同様の扱い）
- 損害賠償額の算定：相手方が被った損害全部

### 権利者の戦略的判断

実用新案権者は、権利行使前に必ず技術評価書を取得して検討する必要があります。

**検討フロー**
1. 実用新案登録を取得
2. 侵害の事実を確認
3. 技術評価書を請求（特許庁に申請、約1ヶ月）
4. 技術評価書の内容を分析
5. 新規性・進歩性が認められそうであれば権利行使
6. 評価が不利であれば、権利行使を慎重に判断

**権利者の法的責任**
- 有利な評価を得たとしても、それは参考に過ぎず、最終的には裁判所が判断
- 技術評価書を提示すれば権利行使が完全に保護されるわけではない
- 訴訟では相手方が無効理由を主張し、裁判所が改めて判断する

## 実用新案登録に基づく特許出願

### 優先権制度との併用

実用新案権者が、後から特許権を取得したいと考える場合、**実用新案登録に基づく特許出願制度**が利用できます。

### 要件と手続き

**実用新案法第13条の規定**

| 要件 | 内容 |
| --- | --- |
| **基本出願** | 実用新案登録出願が先に行われていること |
| **手続き期限** | 実用新案登録出願の日から3年以内に特許出願をすること |
| **優先権の主張** | 実用新案登録出願の出願日を基準として優先権を主張 |
| **明細書の同一性** | 特許出願の明細書は、実用新案登録出願と同一であること（通常） |

### 実用新案権の放棄

実用新案登録に基づいて特許出願をした場合、以下のルールが適用されます。

**実用新案法第13条第2項**
- 特許出願があれば、実用新案権者は実用新案登録を放棄する必要はない
- ただし、実務的には両立できない場合が多い
- 特許が登録されれば、実用新案権は消滅する

### 活用シナリオ

**シナリオ1：迅速な権利化と長期保護の両立**
1. 急いで実用新案登録出願をする（1〜3ヶ月で権利化）
2. その間に特許出願の準備をする
3. 実用新案登録から1年以内に特許出願（優先権主張）
4. 特許が登録されれば、より強力な長期保護（出願から20年）

**シナリオ2：市場反応を見ながら判断**
1. 実用新案登録を取得して市場に投入
2. 市場での反応を見て、本当に価値があるか判断
3. 価値が高ければ特許出願（3年以内）
4. 価値が低い場合は実用新案のみで対応

### 法的効果の注意点

| 状況 | 法的効果 |
| --- | --- |
| **特許出願中の実用新案権** | 権利は併存（両方有効） |
| **特許登録後** | 特許権が有効であれば、実用新案権は通常放棄 |
| **特許出願が拒絶された場合** | 実用新案権は有効に存続 |
| **出願期限（3年）を超えた場合** | 特許出願はできない（優先権も失われない可能性） |

## 典型的なつまずき

### つまずき1：期間だけで覚えている

**誤り**
「実用新案は10年、特許は20年」という暗記のみ

**正解**
- 実用新案：**登録日から10年**（登録まで1〜3ヶ月かかるため実質短い）
- 特許：**出願日から20年**（出願直後から権利有効）
- 出願から実質的に保護される期間を比較することが重要

### つまずき2：保護対象の違いを無視している

**誤り**
「実用新案も物品の方法も保護される」と考える

**正解**
- 実用新案は**物品の形状・構造・組合せのみ**
- 方法は絶対に保護されない
- 「形」を守るのか「やり方」を守るのかの判別が試験での頻出問題

### つまずき3：技術評価書を「権利の強さの証明」と誤解

**誤り**
「技術評価書があれば権利は絶対に有効」

**正解**
- 技術評価書は参考情報に過ぎない
- 提示しないと損害賠償責任が生じるというのが目的
- 訴訟では裁判所が独立に判断する

### つまずき4：実用新案から特許への切り替えを「無条件」と考える

**誤り**
「いつでも特許出願に切り替えられる」

**正解**
- 出願から**3年以内**に限定
- 期限を超えると特許出願できない
- 時間的制約がある戦略的判断

### つまずき5：無審査登録主義を「権利が強い」と誤解

**誤り**
「審査がないので、権利が強い」

**正解**
- 無審査登録主義は**迅速性が特徴**であって、強さではない
- むしろ新規性・進歩性が未確認なため、権利は脆弱
- 訴訟では無効になりやすい

## 問題を解くときの観点

### 観点1：保護対象の判定フロー

```
⬇ 保護すべき対象は何か？
├─ 物品の形状・構造・組合せ？ → 実用新案の対象
├─ 製造方法やプロセス？ → 実用新案の対象外（特許または何も保護されない）
├─ 化学組成？ → 実用新案の対象外（特許のみ）
└─ 外観デザイン？ → 意匠法の対象（実用新案ではない）
```

### 観点2：法的効果の違いを見分ける

**「権利が登録される」という事実だけで判断しない**

- 特許：実体審査を経ているので、新規性・進歩性に一定の信頼性がある
- 実用新案：無審査登録なので、新規性・進歩性は未確認（訴訟で問題になりやすい）

### 観点3：権利行使時の責任を意識する

実用新案で権利行使する場合、必ず**技術評価書を提示しているか**を確認

- 提示あり：正当な権利行使
- 提示なし：損害賠償責任の可能性

### 観点4：時間制約を考慮する

実用新案登録後、特許出願を検討する場合：
- 3年以内であれば特許出願可能
- 3年を超えると不可
- 市場での成功度合いを見極める期間として実用新案を活用

### 観点5：特許との比較選択

**実用新案を選ぶ理由**
- 迅速な権利化が必要（1〜3ヶ月）
- 物品の形状・構造の工夫のみ
- 一時的な競争優位の確保

**特許を選ぶ理由**
- 長期の安定した保護が必要（20年）
- 方法や化学組成も保護したい
- 実体審査による権利の強さが必要

## 確認問題

<Callout title="問1">
    実用新案法では、保護対象を「物品の形状、構造、組合せ」に限定しています。次のうち、実用新案で保護される可能性があるのはどれか。

    A. 自動車のエンジン部品の新しい形状
    B. 医薬品の新しい化学組成
    C. 半導体の製造方法
    D. ビジネスモデルとしてのサブスクリプション方式

    **解答**
    **正解：A**

    自動車のエンジン部品の新しい形状は「物品の形状」に該当し、実用新案で保護されます。

    B は化学組成（物質）であり、物品の形状・構造・組合せではないため、実用新案の対象外です（特許で保護される可能性があります）。

    C は製造方法であり、「方法」は実用新案の保護対象外です（特許の対象です）。

    D はビジネスモデルであり、産業上の利用が限定的であり、実用新案の対象外です。

</Callout>

<Callout title="問2">
    実用新案と特許の登録までのプロセスについて、次の記述のうち正しいのはどれか。

    A. 実用新案は無審査登録主義であるため、特許より権利が強い。
    B. 特許は実体審査を受けるため、新規性と進歩性が確認される。
    C. 実用新案は審査期間が長いため、登録まで2〜3年かかる。
    D. 特許と実用新案は、共に新規性・進歩性の審査を受ける。

    **解答**
    **正解：B**

    特許は実体審査を受けるため、新規性と進歩性が確認されます。これにより、特許は権利の安定性が高い特徴があります。

    A は誤りです。無審査登録主義だからこそ、実用新案は新規性・進歩性が未確認であり、むしろ権利が脆弱です。迅速性が特徴であって、強さではありません。

    C は誤りです。実用新案は無審査登録主義であるため、登録までは通常1〜3ヶ月です（特許が2〜3年かかるのに対して、圧倒的に短い）。

    D は誤りです。実用新案は無審査登録主義であり、新規性・進歩性の審査は受けません。特許のみが実体審査を受けます。

</Callout>

<Callout title="問3">
    実用新案登録に基づく特許出願について、次のシナリオを考える。実用新案法第13条の規定に基づく正しい記述はどれか。

    A. 実用新案登録出願から5年以内であれば、特許出願をすることができ、実用新案登録出願の日を優先日として主張できる。
    B. 実用新案登録出願から3年以内に特許出願をすれば、実用新案登録出願の日を優先日として主張でき、特許が登録されれば実用新案権は自動的に消滅する。
    C. 実用新案登録を取得した後、特許出願をする場合、実用新案権を放棄することが必須条件である。
    D. 実用新案登録出願から3年を超えた後に特許出願をしても、実用新案登録出願の日を優先日として主張できる。

    **解答**
    **正解：B**

    実用新案法第13条により、実用新案登録出願から3年以内に特許出願をすれば、実用新案登録出願の日を優先日として主張できます。また、特許が登録されれば、実用新案権は通常放棄されます（または自動的に消滅）。

    A は誤りです。期限は3年以内であり、5年ではありません。

    C は誤りです。実用新案登録を取得した後、特許出願をする場合、実用新案権の放棄は必須条件ではありません。特許が登録されれば実用新案権は消滅しますが、事前の放棄を要求されません。

    D は誤りです。3年を超えた後に特許出願をした場合、実用新案登録出願の日を優先日として主張できません。3年以内という時間制約があります。
