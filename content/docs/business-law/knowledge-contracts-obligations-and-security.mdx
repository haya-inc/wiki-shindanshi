## 契約の成立と分類

### 契約成立の基本要件

契約は、**申込み** と **承諾** の意思表示が合致することで成立します（諾成契約の場合）。民法は当事者の自由を最大限認める原則を採用しており、これを **契約自由の原則** と呼びます。

#### 契約自由の原則：4つの自由

1. **締結の自由**：契約を締結するかしないかを自由に決定できる
2. **相手方選択の自由**：誰と契約するかを自由に選べる
3. **内容の自由**：契約内容を自由に定めることができる（ただし公序良俗の制限あり）
4. **方式の自由**：契約の成立に特定の方式は不要（ただし一部例外：宅建業法など特別法）

#### 申込みと承諾

- **申込み**：契約締結の意思を表示し、相手方の承諾により契約が成立することを望む意思表示
- **承諾**：申込み者の提示した条件のままで契約に応じる旨の意思表示
- **合致**：申込みと承諾の内容が完全に一致することが必須（民法526条）

申込み後、申込者が撤回しようとした場合：
- **到達前なら撤回可能**（民法525条）
- **到達後なら原則撤回不可**（民法525条1項）
- ただし、申込みが「いつでも撤回できる」旨を明示した場合は撤回可能

### 契約の分類と主要類型

#### 1. 双務契約 vs 片務契約

**双務契約**：両当事者が相互に債務を負う（売買、交換、賃貸借、請負、委任など）
- 特徴：同時履行の利益（民法536条）、危険負担の規定が適用される
- 試験ポイント：一方の履行が不能になった場合、他方も給付義務を免れる可能性がある

**片務契約**：一方のみ債務を負う（贈与、寄託、使用貸借など）
- 特徴：同時履行の抗弁が認められず、危険負担は債権者が負う傾向
- 試験ポイント：贈与は諾成ですが、承諾後に撤回できない（民法550条）

#### 2. 有償契約 vs 無償契約

**有償契約**：一方の給付に対して対価がある（売買、賃貸借、請負など）
- 特徴：相手方の責任が厳格（瑕疵担保責任、契約不適合責任が適用される可能性が高い）

**無償契約**：対価がない（贈与、使用貸借、準委任の一部など）
- 特徴：無償提供者の責任が軽減される傾向
- 試験ポイント：贈与者の瑕疵担保責任は極めて限定的

#### 3. 諾成契約 vs 要物契約

**諾成契約**：意思表示の合致のみで成立（売買、賃貸借、請負、委任、贈与など）
- 特徴：多数の契約がこれに該当

**要物契約**：意思表示に加えて、物の交付その他の給付が必要
- 消費貸借（金銭などを実際に貸す必要）
- 使用貸借（物を実際に貸す必要）
- 質権設定（物を質権者に占有させることが必要）

### 主要契約類型の比較表

| 契約類型 | 当事者の義務 | 双務/片務 | 有償/無償 | 諾成/要物 | 試験での最重要ポイント |
| --- | --- | --- | --- | --- | --- |
| **売買** | 売主：物の所有権移転・引渡し、買主：代金支払い | 双務 | 有償 | 諾成 | 契約不適合責任、危険負担、代金減額請求 |
| **請負** | 請負人：仕事の完成、注文者：報酬支払い | 双務 | 有償 | 諾成 | **仕事の完成責任**、委任との違い、契約不適合責任 |
| **委任** | 受任者：事務処理、委任者：報酬支払い | 双務 | 有償 | 諾成 | **完成責任なし**、準委任、請負との区別 |
| **準委任** | 受任者：事務処理、委任者：報酬支払い | 双務 | 有償 | 諾成 | 法律事務以外の事務処理、委任に準じた扱い |
| **賃貸借** | 賃貸人：物の使用収益、賃借人：賃料支払い | 双務 | 有償 | 諾成 | 継続的給付、敷金、原状回復、修繕義務 |
| **贈与** | 贈与者：財産移転、受贈者：受領 | 片務 | 無償 | 諾成 | 撤回禁止（民法550条）、瑕疵担保責任極限、書面性 |
| **消費貸借** | 貸主：金銭等貸付、借主：返還義務 | 双務 | 有償（利息あり場合） | **要物** | 金銭の引渡しが必須、法定利率の適用 |
| **使用貸借** | 貸主：物の使用、借主：無償返還 | 双務 | 無償 | **要物** | 借主の責任が重い、貸主の瑕疵担保責任なし |
| **交換** | 両当事者：一定物の移転 | 双務 | 有償 | 諾成 | 売買に関する規定を準用（民法559条） |

### 請負と委任の区別（試験での頻出）

#### 請負（民法632条）

```
請負人が当事者に対して仕事の完成を約し、
当事者がこれに対してその仕事の結果に
対して報酬を約する
```

**特徴**：
- **仕事の完成が必須**：最終的な仕事が完成しなければ報酬請求権がない（民法633条）
- **結果責任**：どのようなプロセスで実現するかは請負人の自由だが、完成の責任は負う
- **契約不適合責任が適用**：仕事が契約内容と異なる場合、請負人は損害賠償や解除に応じる
- **例**：建築工事、修理業者、ソフトウェア開発など

#### 委任（民法643条）と準委任（民法656条）

```
委任とは、当事者の一方が法律事務を
処理することを相手方に委託し、
相手方がこれを承諾することによって
その効力を生ずる
```

**特徴**：
- **プロセス重視**：善管注意義務（民法644条）を尽くすことが主要義務
- **仕事の完成責任なし**：完成できなくても責任追及は限定的
- **報酬請求権**：プロセス処理を終えれば報酬請求可能（仕事の完成不要）
- **准委任**：法律事務以外の事務処理にも委任規定を準用（民法656条）
- **例**：弁護士、税理士の法律事務、コンサルティング、医師の医療行為など

#### 判別方法

| 判別軸 | 請負 | 委任 |
| --- | --- | --- |
| **約束する内容** | 仕事の完成 | 事務処理そのもの |
| **報酬請求権** | 完成時のみ | 処理実施時 |
| **完成できなかった場合** | 報酬請求不可 | 処理費用は請求可 |
| **瑕疵担保責任** | あり（契約不適合） | なし（故意・重過失除く） |
| **判例が示す区別基準** | 仕事の最終成果物が存在するか | サービス提供そのものが目的か |

**試験での頻出事例**：
- コンサルティング契約：「売上増加」という成果を約束 → 請負の可能性
- 医療サービス：「治癒」を約束していない → 委任（准委任）
- ソフト開発：「稼働するシステム納品」が条件 → 請負
- 監査業務：「監査実施」が義務で結果は事実判定 → 委任（准委任）

## 意思表示の瑕疵

契約の成立には両当事者の真実の合致が前提ですが、意思表示に欠陥がある場合、契約は無効または取消可能になります。民法は5つの瑕疵類型を定めており、それぞれ効果と第三者保護ルールが異なります。

### 5つの瑕疵類型の比較表

| 瑕疵類型 | 定義 | 内容的有効性 | 第三者への対抗 | 善意第三者保護 | 2020年改正 |
| --- | --- | --- | --- | --- | --- |
| **心裡留保** | 真実の意思と異なる表示をしたが、相手方がそれを知らない | 有効（原則） | 対抗可能 | あり | 変更なし |
| **虚偽表示** | 当事者が共通の意思で真実と異なる表示（ペテン目的） | 無効 | 対抗不可 | あり（善意取得可） | 無効性が強化 |
| **錯誤** | 表示者が真実と異なる認識で表示（表示者の勘違い） | **取消可能**（旧：無効） | 対抗可能 | あり | **重大改正** |
| **詐欺** | 相手方が意図的に虚偽陳述で相手を騙す | 取消可能 | 対抗可能 | あり（被詐欺者に利） | 変更なし |
| **強迫** | 相手方が暴力・脅迫で意思決定を強制 | 取消可能 | **対抗不可**（善意第三者にも対抗可） | **なし** | 変更なし |

### 各瑕疵類型の詳説

#### 1. 心裡留保（民法93条）

```
意思表示は表意者がその真意ではないことを知って
したときは、その意思表示は有効とする。
ただし、相手方が表意者の真意を知り、
又は知ることができたときは、その意思表示は、
その真意に従う。
```

**具体例**：
- A：「この絵画を100万円で売却します」と言うが、実は売却意思がない（買値が50万円で値上がり期待の表示）
- 相手方B が、A の本当は売るつもりがないことを知らない場合→ 有効に成立
- 相手方B が、A の真意を知っている場合→ 無効

**試験ポイント**：
- 有効が原則だが、相手方の「知または知ることができた」が問題
- 「知ることができた」は客観的に判断される傾向

#### 2. 虚偽表示（民法94条）

```
相手方と通じてした虚偽表示は、
その意思表示については無効とする。
```

**特徴**：
- **当事者間の共謀**が前提：双方が真実ではないことを知っている
- **典型的事例**：所有権移転登記を行うが、実は贈与ではなく担保目的の場合

**第三者保護**（民法94条2項）：
```
前項の規定による無効は、善意の第三者に対しては、
その効力を失わない。
```

- 善意の第三者が虚偽表示に基づく権利を取得した場合は保護される
- **例**：A と B が虚偽表示で土地を移転、B が C に売却→ C が善意なら C の所有権は保護される

#### 3. 錯誤（民法95条）【2020年改正で重大変更】

**改正前**：無効
**改正後**：取消可能

```
意思表示は、次に掲げる錯誤に基づくものであった場合には、
取消すことができる。
１　表示内容についての勘違い
２　動機についての勘違い（法律行為の基礎となっている事情が
現実と異なる）
```

**重大改正の意義**：
- 錯誤で無効とされると不動産登記が混乱（第三者が登記を信頼できない）
- 取消可能にすることで、「取り消すまでは有効」として第三者保護を強化
- ただし、取消権を行使すれば無効に遡る

**4つの取消要件** (民法95条3項)

1. **通常の人なら同一の錯誤に陥るか**（客観的重大性）
2. **表示者が重大な過失で陥った錯誤でないか**（表示者責任）
3. **相手方が詐欺的勧誘をしていないか**
4. **当事者間の危険負担契約（「ノークレーム」など）がないか**

**試験での頻出事例**：
- A：「この牛は黒毛和牛と思った」→ 実は乳牛だった：通常の人も同じ錯誤に陥る可能性 → 取消可能
- A：「この絵画は本物ダ・ヴィンチと思った」→ 実は贋作：専門知識ない買い手が通常陥る錯誤 → 取消可能
- 「現物確認、ノークレーム」と明記：錯誤による取消権の制限

#### 4. 詐欺（民法96条）

```
詐欺により意思表示をした者は、
その意思表示を取消すことができる。
```

**構成要素**：
- **欺罔行為**：虚偽陳述または積極的な欺く行為
- **因果関係**：その欺罔により相手方が誤信
- **意思決定の瑕疵**：誤信に基づいて意思表示

**第三者への対抗**：
- 取消権は相手方に対抗可能
- 相手方が善意でも対抗可能（詐欺は強迫より軽い扱い）

**詐欺と強迫の区別**：
| 軸 | 詐欺 | 強迫 |
| --- | --- | --- |
| 手段 | うそ・欺き | 暴力・脅迫 |
| 相手方の態様 | 誤信（騙される） | 恐怖（脅される） |
| 善意第三者に対抗可能 | 可能 | **不可** |

#### 5. 強迫（民法96条）

```
強迫により意思表示をした者は、
その意思表示を取消すことができる。
```

**特徴**：
- **暴力または脅迫**が必要：「殴る」「キャリアを破壊する」など
- **有効性**：強迫の種類や程度により判断
- **親族による強迫**も該当（民法96条2項）

**善意第三者保護の欠落**（民法96条2項）：
```
相手方が強迫者から利益を受けたときは、
その強迫により意思表示をした者は、
強迫者に対してのみ取消権を行使することができる。
```

- 詐欺の場合と異なり、善意第三者にも対抗可能
- 理由：強迫は人の自由意志を奪うため、最も保護が必要

## 代理

代理制度は、一人の者（代理人）が他の者（本人）の名義で意思表示をおこない、その効果が本人に帰属する仕組みです。試験では、**無権代理**と**表見代理**の判別が重要です。

### 代理の3つの類型

#### 1. 任意代理

本人と代理人の契約（委任契約など）に基づく代理

**要件**：
- 本人からの代理権授与（委任状など）
- 代理人が代理権の範囲内で行為

**効果**：
- 代理人の意思表示が本人に帰属
- 本人が直接意思表示したと同じ法的効果

#### 2. 法定代理

法律の規定により当然に生じる代理（親権者の子に対する代理、成年後見人の被後見人に対する代理など）

#### 3. 復代理（民法104条）

代理人がさらに他の者に代理させる場合

```
代理人は、特別の授権を受けた場合を除いて、
その代理権を以って他人のために行為をすることが
できない。
```

**要件**：
- 本人からの「復代理権授与」の明示的授権
- または、代理人が本人のために必要な行為をする場合（急迫の事情など）

### 無権代理（民法113条）

代理権がないのに代理人が他人の名義で意思表示をした場合

**効果**：
- 原則として本人に帰属しない
- ただし、本人が追認すれば有効に遡る（民法113条2項）

**本人の地位**：
- 契約相手方に対して追認/拒絶を決定可能
- 無権代理人に対して損害賠償請求可能（民法117条）

### 表見代理

代理権がないが、外観上代理権があると相手方が信じるのは合理的な場合、相手方を保護する制度

#### 表見代理の3つの類型

**（1）民法109条：代理権の範囲外での行為**

```
代理人が代理権の範囲外の行為をした場合、
本人は相手方に対して追認を拒むことはできない。
```

**条件**：
- 本人が「代理人にこの行為をさせる」と相手方に信じさせた
- 相手方が善意

**例**：
- A が B に「1000万円まで売買交渉の代理権を与えた」と伝えた
- B が実際は500万円の物を「1500万円で売却」と言った
- 相手方が A が了承していると信じて契約→ 表見代理成立、本人 A に対抗可能

**（2）民法110条：代理権消滅後の行為**

```
被代理人が代理人を解任したときは、
被代理人は、遅滞なく、その旨を第三者に通知し、
又は代理人が有していた代理権の書面を返還させなければならない。
これをしない場合には、その代理人がまだ代理権を有するものと
信じて当該代理人とした行為をした者に対し、
本人はその行為の効力を否定することができない。
```

**条件**：
- 本人が代理人を解任した
- 本人が解任を第三者に通知しなかった（または代理権書面を回収しなかった）
- 相手方が、代理人がまだ代理権を有すると善意で信じた

**例**：
- A が C に「B に売買代理権を与えている」と言った
- 後で A が B の代理権を解任した
- A が第三者 C に通知しなかった
- C が B と契約→ 表見代理成立、本人 A に対抗可能

**（3）民法112条：代理権授与表示の悪用**

```
前条の場合のほか、被代理人が、代理権を有しない者に対して、
代理権があるかのような表現をもってこれを与えたかのように示した
場合において、第三者がその者が代理権を有するものと信じてした行為は、
被代理人に対して効力を有する。
```

**条件**：
- 本人が「代理権を有する」と外部に表示（誤ったHPなど）
- 相手方がそれを信頼して契約
- 相手方が善意

**例**：
- 会社が HP に「D は営業代理人です」と表示した
- 実は D に代理権がない
- 第三者がそれを信じて D と契約→ 表見代理成立

### 無権代理と表見代理の判別方法

| 判別軸 | 無権代理 | 表見代理 |
| --- | --- | --- |
| **代理権の有無** | なし | なし（しかし外観がある） |
| **本人の責任** | 本人は対抗可能 | 本人は対抗不可 |
| **相手方の地位** | 本人が拒絶可能 | 本人に対抗可能 |
| **本人の保護** | 強い（拒絶可） | 弱い（対抗不可） |
| **相手方の保護** | 弱い（拒絶リスク） | 強い（対抗可） |

## 債務不履行

契約が成立後、当事者が約定通りに履行しない場合を債務不履行といいます。2020年改正で、債務不履行時の解除要件（帰責事由）が変更され、試験対策が必須です。

### 債務不履行の3つの類型

#### 1. 履行遅滞（民法412条）

```
債務の履行について期限があるときは、
債務者がその期限経過後に履行をした場合には、
債権者は、その履行を受け取ることを拒むことができる。
```

**条件**：
- 給付期限の到来
- 期限後の履行（遅れて履行）
- 債務者の帰責事由

**相手方の救済手段**：
- 損害賠償請求（遅延損害金など）：帰責事由必須
- 契約解除：帰責事由不要（2020年改正）

#### 2. 履行不能（民法413条）

履行が不可能になった場合

**具体例**：
- A が B に「この美術品を3月1日に納品」と約束
- 2月28日に美術品が盗難で失われた

**条件**：
- 履行が客観的に不可能
- 債務者の帰責事由（過失以上）

**相手方の救済手段**：
- 損害賠償請求：帰責事由必須
- 契約解除：帰責事由不要（2020年改正）
- 代金支払い義務も免れる（双務契約）

#### 3. 不完全履行

履行はされたが、約定内容と異なる場合

**具体例**：
- 建築業者が「鉄骨造」と約束したが「軽量鉄骨」で建築
- 納品されたソフトウェアにバグがある

**相手方の救済手段**：
- 修補請求（やり直し）
- 損害賠償請求
- 代金減額請求
- 契約解除

### 帰責事由

債務不履行時の損害賠償請求と契約解除で、帰責事由の有無が重要になります。

#### 2020年改正：解除の帰責事由要件の廃止

**改正前**：契約解除には「帰責事由が必須」だった
**改正後**：契約解除には「帰責事由が不要」になった

```
新民法542条
債務者が履行期に給付をしない場合において、
債権者が相当の期間を定めて催告をし、
その期間内に履行がないときは、
債権者は、契約を解除することができる。
```

**意味**：
- 債務者の責任がなくても（不可抗力でも）、履行できなければ解除可能
- 買主が売主の病気を理由に売買契約を解除可能に（かつての判例を法制化）
- 債務者はその代わり損害賠償責任を免れる（帰責事由がないため）

### 損害賠償請求と契約解除の使い分け

| 手段 | 要件 | 帰責事由 | 試験での使い分け |
| --- | --- | --- | --- |
| **損害賠償請求** | 不履行発生、損害発生、因果関係 | **必須** | 債務者に過失がある場合に有効 |
| **契約解除** | 不履行が重大、催告と期間経過 | **不要**（改正後） | 債務者に責任がなくても解除可能 |
| **同時履行の抗弁** | 双務契約の双方が履行期同時 | 不要 | 相手方が未履行の場合に反論 |

## 契約不適合責任

2020年改正で旧「瑕疵担保責任」制度が廃止され、「契約不適合責任」に統一されました。売買に限定されず、請負などの有償契約に広く適用されます。

### 契約不適合責任の成立要件

```
新民法562条
引き渡された目的物が種類、品質又は数量に
関して契約の内容に適合しないものであるときは、
売主は、買主に対して、相当期間を定めて
当該目的物をその種類、品質又は数量に
関して契約の内容に適合するものとすること
を請求することができる。
```

**要件**：
1. 売買（または請負などの有償契約）が成立
2. 目的物が引き渡されている
3. 目的物が「契約内容と異なる」（質的または量的不適合）
4. 買主が「引き渡し後相当期間内に不適合を知った」

### 旧瑕疵担保責任との比較

| 軸 | 旧瑕疵担保責任 | 新契約不適合責任 |
| --- | --- | --- |
| **対象契約** | 売買に限定 | 請負、委任なども対象 |
| **基準** | 「隠れた瑕疵」 | 「契約内容との不適合」 |
| **売主の責任** | 無過失責任 | 契約内容責任 |
| **救済手段** | 損害賠償、解除のみ | 追完、減額、損害賠償、解除 |
| **通知期間** | 「発見」から1年 | 引き渡しから1年（「通知」が必須） |

### 契約不適合責任の4つの救済手段

#### 1. 追完請求（民法563条）

不適合を直す（修補、交換、再施工など）

```
買主は、売主に対して、相当期間を定めて、
当該目的物をその種類、品質又は数量に
関して契約の内容に適合するものとすることを
請求することができる。
```

**特徴**：
- 買主が最初に行使すべき手段
- 売主が追完可能な場合が前提
- 追完の方法（修補か交換）は売主が決定（民法563条2項）

#### 2. 代金減額請求（民法565条）

契約金額を引き下げる

```
買主は、売主に対する前条の請求権と並行して、
又はこれに代えて、その不適合の程度に応じて
代金の減額を請求することができる。
```

**特徴**：
- 追完に応じない場合に有効
- 減額額は「不適合の程度」に応じて客観的に決定（修補費用、機能低下度など）
- 売主が破産した場合にも有効

#### 3. 損害賠償請求（民法564条）

売主の責任（過失、故意）が必要

```
売主が契約の内容に適合しない目的物を
引き渡した場合において、
買主に損害を生じたときは、
売主は、その損害を賠償する責任を負う。
```

**要件**：
- 不適合の発生
- 買主の損害
- 因果関係
- 売主の帰責事由（過失以上）

#### 4. 契約解除（民法566条）

契約を白紙に戻す

```
買主は、売主が追完請求に応じないとき、
その他相当期間内の追完がないときは、
契約を解除することができる。
```

**要件**：
- 買主が売主に追完請求
- 催告と相当期間経過
- その後も追完なし

**効果**：
- 売買契約が遡って無効に
- 買主は代金返還請求、売主は物返還請求
- 買主が既に使用した場合は価値償却額を支払う可能性

### 通知義務と期間制限（重大改正）

#### 「通知」が必須

```
買主は、不適合を発見した場合、
遅滞なく売主に通知しなければならない。
```

**影響**：
- 通知がなければ、その間の損害賠償請求が制限される可能性
- ただし、売主が不適合を知っていた場合は通知不要と解釈される傾向

#### 1年以内の権利行使（民法566条）

```
買主の権利は、引き渡しから1年以内に
権利を行使しなければならない。
```

**特徴**：
- 旧法：「発見」から1年
- 新法：「引き渡し」から1年（より売主に有利）
- この期間経過後は、原則として救済手段が失われる

### 請負における契約不適合責任（民法639条）

```
請負人が仕事を成し遂げなかった場合、
または仕事が契約内容に適合しない場合は、
請負人は新たに仕事をやり直すか、
その結果を修補する義務を負う。
```

**特徴**：
- 売買同様、追完請求が第一義的手段
- 期間制限は「引き渡しから1年」（民法639条2項）
- 建築工事などで頻出

## 危険負担

売買契約成立後、目的物が滅失または毀損した場合、その「危険」（代金を支払う義務があるかどうか）を誰が負うかが問題になります。2020年改正で大きく変更されました。

### 2020年改正：債権者主義 → 債務者主義

**改正前（債権者主義）**：
- 危険は債権者（買主）が負う
- 売主が給付不能でも、買主は代金支払い義務あり

**改正後（債務者主義）**：
- 危険は債務者（売主）が負う
- 売主が給付不能なら、買主は代金支払い義務なし
- 売主は損害賠償責任を負う（帰責事由あれば）

### 新民法536条2項（危険負担の廃止）

```
債務の履行が不能となった場合において、
その不能が債務者の責めに帰すべき事由に
よるものでないときは、
債権者は、反対給付の履行を拒むことができる。
```

**意味**：
- 売主の責任がない危険でも、買主は代金支払い義務を免れる
- 売主が損害を被る
- 実務では保険で対応

### 具体例

**契約成立時点での危険負担**：

| 事実 | 改正前 | 改正後 |
| --- | --- | --- |
| 売主が物を所有、買主が代金支払い前に物が盗難 | 買主が危険負担（代金支払い義務） | 売主が危険負担（買主は代金拒否可） |
| 目的物が売主の保管中に火災で滅失 | 買主が危険負担 | 売主が危険負担 |
| 買主が既に目的物を受け取った後に毀損 | 買主が危険負担 | 買主が危険負担 |

## 債権譲渡

債権者が自分の債権を第三者に移転する場合、対抗要件（第三者に対して権利を主張できるための要件）が必要です。2020年改正で重大な変更がありました。

### 債権譲渡の要件

#### 1. 債務者への対抗要件：通知または承諾（民法467条）

```
債権の譲渡は、譲渡人から債務者に対する
通知又は債務者の承諾によって、
債務者その他の第三者に対抗することができる。
```

**要件**：
- **通知**：譲渡人が債務者に「債権を譲渡した」と一方的に通知
- **承諾**：債務者が譲渡に同意

**効果**：
- 通知・承諾があれば、債務者は譲渡人ではなく**譲受人に支払う義務**
- なければ、債務者は譲渡人に支払うと有効に消滅

#### 2. 第三者への対抗要件：確定日付ある証書（民法468条）

```
債権の譲渡は、その譲渡を示す書面に
確定日付がある場合に限り、
譲受人から譲渡人の債権者その他の者に対抗する
ことができる。
```

**要件**：
- 公正証書など「確定日付ある証書」
- 二重譲渡の場合、確定日付が早い者が優先

**具体例**：
- A が B に「C への100万円債権を譲渡」
- その後、A が D に「同じ債権を譲渡」
- B が確定日付（公正証書）で譲渡を証明 → B が優先

### 2020年改正：譲渡制限特約の廃止

#### 改正前

```
「この債権は譲渡できない」という特約がある場合、
その特約に違反して譲渡すると無効
```

#### 改正後（民法466条2項）

```
債権の譲渡は、当事者間の契約でも制限できない。
（譲渡制限特約に反しても有効）
```

**改正の意義**：
- 債権流動化を促進（企業の資金調達が容易に）
- 譲渡制限特約は「無効」ではなく、「債務者は譲受人に支払うことで免責される」との扱い
- 実務では契約内容の履行遅滞など別の手段で対応

## 相殺

債権者と債務者が互いに相互対立する債権（同じ種類の債権）を持つ場合、相手方の同意なく一方的に相殺できます。ただし禁止事由があります。

### 相殺の要件（民法505条）

```
債権者が債務者に対する債権を有する場合において、
当該債権と債務者が債権者に対する債権とが
いずれも弁済期にあるときは、
債権者は、債務者の請求があるまで相殺することができる。
```

**要件**：
1. **互いに債権を有すること**：A が B に100万円、B が A に80万円の債権
2. **同時に弁済期にあること**：両債権が履行期を迎えている
3. **異議がないこと**：債務者が反対しない

**効果**：
- 相殺可能な限度で債権は消滅
- 上記の例なら、A の100万円は80万円が相殺され、20万円のみ支払い義務残存

### 相殺禁止事由

相殺が禁止される場合があります。試験で頻出です。

#### 1. 悪意の不法行為債権（民法509条）

```
債務者が悪意で不法行為を加えた場合、
その不法行為によって生じた損害賠償請求権に対しては
相殺をもってその債権者に対抗することができない。
```

**具体例**：
- A が B に対して故意に物を壊して損害を与えた（悪意の不法行為）
- B が A に対して債権を有していても、A はその債権で B の損害賠償請求に相殺できない

#### 2. 生命または身体を害する不法行為債権（民法509条）

```
債務者が生命又は身体を害する不法行為を加えた場合、
その不法行為によって生じた損害賠償請求権に対しては
相殺をもってその債権者に対抗することができない。
```

**具体例**：
- A が B を暴行で傷つけた
- B が A に対して債権を有していても、A はその債権で B の傷害賠償請求に相殺できない

#### 3. 公租公課、一部の給料

一部の特定債権は相殺禁止の対象になることがあります（法律個別規定による）。

## 定型約款

2020年改正で新設された「定型約款」制度は、大量の利用者向け契約（保険、携帯電話、銀行窓口サービスなど）での利便性を高めるためのものです。

### 定型約款とは（民法548条の2）

```
定型約款とは、ある特定の者により提供される
商品またはサービスについて、
多数の不特定の者を相手方として、
その内容の全部または一部が画一的であることが
予定されている約款をいう。
```

**特徴**：
- 大量利用者向け（消費者、中小企業など）
- 保険約款、携帯キャリアの約款など
- 個別に内容協議しない（画一的）

### 定型約款の効力（民法548条の3）

#### 「みなし合意」の仕組み

```
定型約款は、以下の要件を満たすときは、
民法第五四八条ただし書に規定する場合を除き、
契約の内容となる。
```

**要件**：
1. 定型取引の相手方（消費者など）が、定型約款を提示する者（企業）と合意
2. 相手方が「定型約款があることを認識」
3. 相手方が「定型約款の内容を理解することが容易になっている」

**効果**：
- 相手方が全文を読まなくても、約款は契約内容に含まれる

**具体例**：
- 銀行の普通預金口座開設：「定款をご覧ください」との書面で案内 → 相手方が見なくても約款は有効
- 保険加入：「保険約款」を事前提示 → 相手方が全部読まなくても有効

### 不当条項の排除（民法548条の4）

```
定型約款であっても、相手方の利益を一方的に
害する条項は、契約の内容とならない。
```

**無効になる条項例**：
- 「保険金を絶対に支払わない」（条件なし）
- 「利用者の責任は全て利用者が負う」（過度な免責）
- 「一方的な手数料値上げ」（相手方に一切の選択肢なし）

### 定型約款の変更（民法548条の5）

```
定型約款の提供者は、契約の相手方に
不利になる変更であっても、
あらかじめ予定されている範囲内なら
個別同意なく変更できる。
```

**特徴**：
- 「今後、以下の通り約款を変更します」と告知 → 個別同意不要
- ただし、変更が「極めて不当」な場合は、裁判所が制限する可能性あり

## 法定利率（2020年改正で重大変更）

### 改正の概要

| 項目 | 改正前 | 改正後 |
| --- | --- | --- |
| **民法法定利率** | 年5% | 年3%（変動制） |
| **商事法定利率** | 年6% | 廃止（民法利率に統一） |
| **見直し周期** | なし | 3年ごと（基準利率変動時） |

### 新法定利率の仕組み

#### 基準利率と変動制（民法404条）

```
法定利率は、以下により変動する：
１　当初設定：年3%
２　３年ごとの見直し：前3年間における
　　  金融機関の貸出約定平均金利（短期）を基準
３　０．１%単位で調整
```

**意義**：
- 実勢金利（市場の金利）に合わせる
- 低金利時代に対応
- 年5%が高すぎる状況を是正

### 法定利率が適用される場面

#### 1. 利息付き金銭債務の利息

消費貸借で「利息の額」が定められていない場合

```
例：「100万円貸す、利息は後で決めよう」
→ 年3%の利息が自動的に適用される
```

#### 2. 遅延損害金

契約で「遅延損害金の利率」が定められていない場合

```
例：代金支払い遅延で「遅延損害金をいくらにするか」不明
→ 年3%が自動的に適用される
```

#### 3. 不法行為による損害賠償金の利息

被害者が損害を被った場合、加害者は損害額に年3%の利息をつけて賠償する

```
例：交通事故で500万円の損害
→ 判決確定までの間、年3%の利息つき
```

### 商事法定利率廃止の影響

**改正前**：
- 商人間の債権：年6%の利息
- 消費者等の債権：年5%の利息

**改正後**：
- すべての債権：年3%に統一
- 商人も消費者も同じレート

**実務的影響**：
- 企業間取引の利息コストが低下
- ただし、契約で「6%」と定めることは依然可能（法定利率の変更不可）

## 典型的なつまずき

### 1. 請負と委任の混同

**つまずき**：「両方とも仕事を委託する契約だから同じ」と覚える

**正解**：
- **請負**：仕事の「完成」が必須（完成できなければ報酬請求不可）
- **委任**：事務処理「そのもの」が義務（完成できなくても処理費用は請求可）
- 判例基準：「最終成果物の完成が約束されているか」で判別

### 2. 錯誤の改正内容を誤解

**つまずき**：「錯誤は無効」と覚えたまま（改正前知識）

**正解**：
- **改正後**：錯誤は「取消可能」
- 取消権行使までは契約は有効
- 第三者が登記を信頼できるようになった

### 3. 表見代理の3つの類型を区別できない

**つまずき**：「表見代理がある」で片付けてしまう

**正解**：
- **109条**：代理権の範囲外の行為（本人が「ここまで」と言ったのに超過）
- **110条**：代理権消滅後（解任したが通知しなかった）
- **112条**：代理権授与表示（本人が誤った表示で第三者を誤導）
- 各条の適用場面が異なる

### 4. 契約不適合責任と瑕疵担保責任を混同

**つまずき**：旧法の知識で「瑕疵担保責任は売買に限定」と答える

**正解**：
- **新法**：「契約不適合責任」として請負など広い契約に適用
- 基準は「隠れた瑕疵」から「契約内容との不適合」に変更
- 救済手段が4つに拡大（追完、減額、損害賠償、解除）

### 5. 危険負担の改正を見落とし

**つまずき**：「買主が危険を負う」と答える（改正前）

**正解**：
- **改正後**：売主（債務者）が危険を負う
- 売主の責任がない危険でも、買主は代金支払い義務を免れる
- 実務では保険で対応

### 6. 債権譲渡制限特約の廃止を見落とし

**つまずき**：「譲渡制限特約があれば譲渡は無効」と答える

**正解**：
- **改正後**：譲渡制限特約に反しても譲渡は「有効」
- 特約は無効ではなく、債務者の支払い義務には影響しない

### 7. 相殺禁止事由を不完全に覚える

**つまずき**：「悪意の不法行為だけ禁止」と覚える

**正解**：
- **悪意の不法行為債権**：相殺禁止（積極的に害した）
- **生命身体を害する不法行為債権**：相殺禁止（より厳格）
- 区別が試験では重要

### 8. 定型約款の「みなし合意」を過大評価

**つまずき**：「定型約款なら何でも有効」と思い込む

**正解**：
- 「みなし合意」の要件：相手方が「認識」かつ「内容理解が容易」
- 不当条項は「排除」される
- 相手方の利益を一方的に害する条項は無効

### 9. 法定利率の新制度を古い知識で答える

**つまずき**：「商人なら6%、消費者なら5%」と答える（改正前）

**正解**：
- **改正後**：全て「年3%」で統一
- 変動制（3年ごと見直し）
- 商事法定利率は廃止

## 問題を解くときの観点

### 1. 契約成立のステップを確認

```
問題文から以下を抽出：
① 申込み（提示者は誰か、提示内容は何か）
② 承諾（承諾者の意思表示は明確か）
③ 合致（申込み内容と承諾内容は一致しているか）
④ 特殊要件（要物契約なら物の交付があるか）
```

### 2. 意思表示に瑕疵がないか確認

```
問題文から以下を判別：
① 真実の意思と異なるか
   → 心裡留保？虚偽表示？錯誤？
② 相手方の責任があるか
   → 詐欺？強迫？
③ 第三者が関係しているか
   → 善意第三者に対抗可能か
```

### 3. 請負と委任を区別

```
「仕事の完成」が約束されているか
→ Yes なら請負（結果責任あり）
→ No なら委任（プロセス責任）
```

### 4. 代理の有効性を判定

```
① 代理権はあるか
   → Yes なら有効
   → No なら、表見代理に該当するか
② 表見代理なら、109/110/112条のどれか
③ 善意第三者の存在を確認
```

### 5. 債務不履行の類型を判別

```
① 履行できるが遅延しているか → 履行遅滞
② 履行が不可能になったか → 履行不能
③ 履行されたが不完全か → 不完全履行
```

**各類型で救済手段を選択**：
- **損害賠償**：帰責事由が必須
- **契約解除**：帰責事由不要（改正後）

### 6. 契約不適合責任を判定

```
① 契約内容と現物が異なるか
   → Yes なら契約不適合責任発生
② 追完可能か
   → Yes なら、まず追完請求（買主の第一選択肢）
③ 追完に応じないか
   → 代金減額請求、損害賠償、解除に進む
④ 1年以内に「通知」を要求したか
   → 通知なければ権利が制限される可能性
```

### 7. 危険負担を判定

```
① 売買契約成立後、目的物が滅失/毀損したか
② その責任は売主にあるか
   → Yes なら売主が危険負担（買主は代金拒否可）
   → No なら、2020年改正で売主が危険負担（買主は代金拒否可）
   （改正後は常に売主が危険負担と考えて無難）
```

### 8. 債権譲渡を判定

```
① 債権譲渡が発生しているか
② 債務者に対抗するには
   → 通知または承諾が必須
③ 第三者に対抗するには
   → 確定日付ある証書が必須
④ 譲渡制限特約があるか
   → 改正後は「有効」扱い（特約に反しても譲渡は有効）
```

### 9. 相殺を判定

```
① 互いに債権を有しているか
② 弁済期は同時か
③ 相殺禁止事由がないか
   → 悪意の不法行為債権？
   → 生命身体を害する不法行為債権？
④ 禁止事由がなければ相殺可能
```

### 10. 定型約款を判定

```
① 定型約款か（大量利用者向け、画一的内容）
② みなし合意の要件を満たすか
   → 相手方が「認識」しているか
   → 「内容理解が容易」か
③ 不当条項で無効部分がないか
④ 変更の提示がされているか（個別同意不要な場合も）
```

### 11. 法定利率を判定

```
① 金銭債権で利息が問題か
② 利息の約定があるか
   → Yes なら約定利率（法定利率は適用されない）
   → No なら法定利率が適用（年3%）
③ 改正前後で利率が異なるか確認
   → 改正前：5%（商人なら6%）
   → 改正後：3%（全て統一、変動制）
```

## 確認問題

<Callout title="確認問題1：契約の成立と請負・委任の区別">
    A 社がソフトウェア開発会社 B に対して、「売上管理システムの開発を依頼する。完成まで500万円、完成時に支払う」と持ちかけた。B が承諾し、契約書に署名した。

    3ヶ月経過後、B は開発の途中で重大なバグを発見したため「やり直しが必要で、契約解除したい」と申し出た。一方 A は「ここまでの開発を進めてくれたので、500万円の一部である300万円を支払う」と提案した。

    【問】この契約は請負か委任か。また、B が開発を完成させられない場合、A と B の支払い義務はどうなるか。（200字以内で答えよ）
</Callout>

<Callout title="確認問題2：意思表示の瑕疵と第三者保護">
    A が B に対して「この土地を2000万円で売却する」と言った。実は A は売却の意思がなく、単に冗談で言ったのだが、B はそれを信じて承諾した。その後、A が「冗談だから契約は無効」と主張した。

    一方、B が A に無断で、この土地を善意の第三者 C に1800万円で売却し、登記も済ませた。今 A が「契約は無効だから土地を返してほしい」と C に主張している。

    【問】
    1. A と B の契約は成立しているか、無効か、それとも取消可能か。
    2. C は土地の所有権を取得できるか。理由を述べよ。（300字以内で答えよ）
</Callout>

<Callout title="確認問題3：契約不適合責任と危険負担">
    売主 S が買主 B に「品質良好な木材1000立方メートル」の売買契約を締結した（契約価格：500万円、配送予定日：3月15日、代金支払い期限：3月31日）。

    3月10日、売主の倉庫で、S の過失でなく自然災害により、木材の50%が水没して使用不能になった。S は「不可抗力だから、残りの50%（250万円相当）を納品する」と言い、B は「契約内容と異なるので、この契約は解除したい。代金は支払わない」と主張した。

    【問】
    1. この事態は「契約不適合責任」か「危険負担」か。どちらが適用されるか。
    2. 2020年改正を踏まえ、B は代金支払い義務を免れるか。
    3. S は残りの50%を納品する義務があるか。（400字以内で答えよ）
</Callout>
