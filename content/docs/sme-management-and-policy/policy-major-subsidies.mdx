## 学習のポイント

### 補助金5つの役割を頭に入れておく

中小企業診断士試験では、補助金を「公募回や上限額」で覚えるのではなく、「どの経営課題を支援するのか」で整理することが重要です。以下の5つを軸に考えます。

- **ものづくり補助金**：革新的な製品・サービス開発、生産プロセス改善
- **IT導入補助金**：ITツール導入による業務効率化・デジタル化
- **小規模事業者持続化補助金**：販路開拓、事業の持続的発展
- **省力化投資補助金**：人手不足対応、省人化投資
- **中小企業新事業進出補助金**：新市場進出、事業転換・業種転換

### 賃上げ要件への意識

近年の補助金は「国の賃上げ政策」と連動しており、付加価値額・給与支給総額・最低賃金の引上げを要件とするものが増えています。特にものづくり補助金の「3つの要件」は典型パターンです。

### 申請段階での共通ルール

全ての補助金に共通する重要ルール：
- 交付決定前の着手は原則不可
- GビズIDプライム取得が必須
- 補助金は後払い（精算払い）
- 認定支援機関の確認が必要な場合がある

## 試験で何が問われるか

### 出題パターン

1. **補助金の目的認識**：「販路開拓を支援する」「デジタル化を支援する」などの説明ができるか
2. **対象の正確な判定**：小規模事業者向けか中小企業一般か、業種制限があるか
3. **基本要件の理解**：付加価値額、給与支給総額、最低賃金などの計算式を理解しているか
4. **補助率と補助上限**：類型による違いを区別できるか
5. **申請プロセス**：事前着手禁止、支援機関との連携、GビズID等の必須事項を知っているか

### 典型設問例

- 「小規模事業者が販路開拓を実施したい場合、どの補助金を活用すべきか」→ **持続化補助金**
- 「中小企業がITツール導入により業務効率化を図りたい場合」→ **IT導入補助金**
- 「製造業が新製品開発に取り組み、付加価値額を向上させたい」→ **ものづくり補助金**
- 「人手不足対応のため省力化設備を導入したい」→ **省力化投資補助金**
- 「既存事業とは別の新分野に進出したい」→ **新事業進出補助金**

## ものづくり補助金の詳細要件

### 制度の役割

**ものづくり補助金** は、製造業、建設業、運輸業などの中小企業が、革新的な製品やサービスを開発したり、生産プロセスを改善したりする際の投資を支援する補助金です。「革新性」と「付加価値向上」が軸です。

### 正式名称と所管

- **正式名称**：ものづくり・商業・サービス生産性向上促進補助金
- **所管**：経済産業省中小企業庁

### 申請類型（2026年度時点）

1. **省力化（オーダーメイド）枠**：人手不足対応のための汎用性の低い専用機械
2. **製品・サービス高付加価値化枠**：革新的な新製品・新サービス開発
3. **グローバル枠**：海外展開を視野に入れた生産プロセス改善

### 基本要件（3つすべてを満たす必要がある）

ものづくり補助金の申請には、以下の3つの要件をすべて満たす必要があります。

#### 1. 付加価値額の向上要件

**計算式**：`(当期営業利益 + 人件費 + 減価償却費) の前年度比 ÷ 売上高`

```
付加価値額年率向上率 = (当期営業利益 + 人件費 + 減価償却費) の増加分
                     ÷ 基準年度の売上高
                     ≧ 3% 以上
```

**重要な読み方**：補助金交付後3年間で、毎年3%以上の付加価値額向上を達成し続ける必要があります。つまり、初年度3%、2年目3%、3年目3%という「毎年の達成」が求められます。

#### 2. 給与支給総額の向上要件

```
給与支給総額年率向上率 = (当期の給与支給総額 - 前年度給与支給総額)
                       ÷ 前年度給与支給総額
                       ≧ 1.5% 以上
```

**重要な読み方**：賃上げ率が「1.5%以上」であることが条件です。継続期間は3年間です。

#### 3. 事業場内最低賃金の引上げ要件

```
事業場内最低賃金 ≧ 地域別最低賃金 + 30円
```

**重要な読み方**：補助対象経費を使用する「全ての事業場」で、この要件を満たす必要があります。地域によって地域別最低賃金が異なるため、各事業所の所在地を確認してから計算します。

### 補助率

- **中小企業**：1/2
- **小規模企業者（従業員20人以下）**：2/3

### 補助上限額

- **省力化（オーダーメイド）枠**：750万円
- **製品・サービス高付加価値化枠**：1,250万円
- **グローバル枠**：3,000万円

### 加点項目（採択に有利）

- 経営革新計画の承認を取得
- 事業継続力強化計画の認定を受けている
- 上記3要件で賃上げ要件をさらに上回る実績

### 事前着手禁止の実例

ものづくり補助金の「事前着手禁止」は厳密です。たとえば：
- 交付決定前に機械を発注した → 対象外
- 交付決定前に開発を開始した → その部分は対象外

**重要**：事業計画書には「交付決定後に着手予定」と明記する必要があります。

## IT導入補助金の詳細要件

### 制度の役割

**IT導入補助金** は、中小企業・小規模事業者がITツール（会計ソフト、POSレジ、顧客管理システムなど）を導入することで、業務効率化やデジタル化を実現するための補助金です。

### 正式名称と所管

- **正式名称**：中小企業・小規模事業者ものづくり・商業・サービス革新事業費補助金
- **所管**：経済産業省中小企業庁

### 申請手続きの特徴

**重要なポイント**：IT導入補助金は、中小企業が直接申請するのではなく、IT導入支援事業者（ベンダー）と共同で申請する仕組みです。

1. 中小企業がIT導入支援事業者を選定
2. IT導入支援事業者がカタログに登録されたツールを提案
3. 中小企業とIT導入支援事業者が共同で申請
4. 採択後、ツール導入を実施

### 適用枠（2026年度時点）

#### 1. 通常枠

- **補助上限**：5万円以上150万円未満、150万円以上450万円以下（2段階）
- **補助率**：1/2以内
- **対象**：全ての中小企業・小規模事業者

#### 2. インボイス枠

免税事業者がインボイス発行事業者に転換する場合の支援

- **小規模事業者**：補助率4/5
- **その他中小企業**：補助率2/3〜3/4

#### 3. セキュリティ対策推進枠

サイバーセキュリティ対策に特化したツール

- **補助上限**：5万〜100万円
- **補助率**：1/2

#### 4. 複数社連携IT導入枠

複数の中小企業が連携してツール導入

### 補助対象経費

- **ソフトウェア購入費**：ライセンス費、カスタマイズ費
- **クラウド利用料**：最大2年分が対象（3年目以降は自己負担）
- **導入関連費**：コンサルティング、導入サポート、テスト費

**重要な読み方**：クラウド利用料が「2年分」までしか対象にならないため、3年目以降の継続利用に向けて、採算性を検討する必要があります。

### 採択のポイント

- ツールの導入により、実現可能な「生産性向上」を説明できるか
- 導入前後での業務フロー改善が具体的か
- IT導入支援事業者の支援体制が充実しているか

## 小規模事業者持続化補助金の詳細要件

### 制度の役割

**小規模事業者持続化補助金** は、小規模事業者（従業員数が少ない企業）が、経営計画に基づいて行う販路開拓や業務効率化等の事業活動を支援する補助金です。「継続的な経営改善」が軸です。

### 対象となる小規模事業者の定義

| 業種分類 | 従業員数 |
|---------|---------|
| 商業・サービス業（旅館業を含む） | 5人以下 |
| 製造業・その他 | 20人以下 |

**重要な読み方**：従業員数が多すぎると対象外になるため、申請前に確認が必須です。

### 申請手続きの特徴

持続化補助金には、商工会や商工会議所の支援が組み込まれています。

1. 経営計画書を策定（商工会/商工会議所のサポート）
2. 補助対象事業の内容を計画書に明記
3. 商工会/商工会議所の確認を得る
4. 公募期間に申請

### 補助金の枠（類型）

#### 1. 通常枠

- **補助上限**：50万円
- **補助率**：2/3
- **対象**：全ての小規模事業者

#### 2. 特別枠

特定の経営課題に対応する場合、上限が上がります。

| 枠の種類 | 補助上限 | 補助率 | 要件 |
|---------|---------|--------|------|
| **賃金引上げ枠** | 200万円 | 2/3（赤字3/4） | 事業場内最低賃金を引き上げる |
| **卒業枠** | 200万円 | 2/3 | 小規模から中小規模への成長を目指す |
| **後継者支援枠** | 200万円 | 2/3 | 事業承継後2年以内の後継者 |
| **創業枠** | 200万円 | 2/3 | 創業から3年以内の事業者 |

### 補助対象経費

- **販路開拓**：WEB制作、広告宣伝、出展、カタログ作成等
- **業務効率化**：機器・ソフトウェア導入（一部）
- **経営サポート**：専門家コンサルティング費用

**制限事項**：補助対象経費の総額が50万円以上であることが必要です。

### インボイス特例

免税事業者がインボイス発行事業者に転換する場合、追加で50万円が上乗せされます。

```
通常の補助上限：50万円
+ インボイス特例：50万円
= 最大100万円（通常枠でも可能）
```

## 省力化投資補助金の詳細要件

### 制度の役割

**省力化投資補助金** は、人手不足に直面する中小企業が、省人化・省力化を実現するための設備投資やDX導入を支援する補助金です。2023年度に新設された比較的新しい制度です。

### 申請類型

#### 1. カタログ注文型

中小企業庁が事前に登録した省力化設備のカタログから選んで導入する簡易型

- **特徴**：審査が簡略化されている
- **補助上限**：従業員規模に応じて設定
- **対象**：汎用化された省力化設備

#### 2. 一般型（オーダーメイド）

企業の個別事情に合わせたカスタム型の省力化投資

- **特徴**：より詳細な経営診断と事業計画が必要
- **補助上限**：従業員規模に応じて設定
- **対象**：企業独自の省力化課題に対応した設備

### 従業員規模による補助上限

| 従業員数 | 補助上限（基本） | 賃上げ達成時 |
|---------|----------------|------------|
| 5人以下 | 200万円 | 300万円 |
| 6〜20人 | 500万円 | 750万円 |
| 21〜50人 | 1,000万円 | 1,500万円 |
| 51人以上 | 1,500万円 | 2,250万円 |

**重要な読み方**：賃上げを達成すれば、補助上限が1.5倍に引き上がります。

### 補助率

```
補助率 = 1/2（50%）
```

他の補助金（ものづくり補助金で2/3など）に比べると、補助率は1/2に設定されています。

### 賃上げで上限が1.5倍になる仕組み

```
基本上限：500万円（6〜20人企業の場合）
↓
以下の要件を満たすと↓
↓
上限1.5倍：750万円
```

**賃上げ要件**（どちらか一つを満たせばOK）

- 事業場内最低賃金を、地域別最低賃金＋40円以上に引き上げ
- または、給与支給総額を前年度比1.5%以上引き上げ

### 対象経費

- **汎用的な省力化設備**：ロボット、自動化機械、DX関連ツール
- **導入関連費**：設計費、工事費、テスト費
- **コンサルティング費**：省力化診断、導入支援

## 中小企業新事業進出補助金の詳細要件

### 制度の役割

**新事業進出補助金** は、2024年度に廃止された「事業再構築補助金」の後継制度として、2025年度から新設されました。既存事業では対応できない市場ニーズに対応するため、新分野進出や業種転換に取り組む中小企業を支援します。

### 背景と位置づけ

旧・事業再構築補助金は、新型コロナ対応から産業転換支援へシフトし、さらに2025年度からは「経済構造の転換」を支援する新事業進出補助金に統合されました。

### 進出タイプ別の分類

| タイプ | 説明 | 例 |
|--------|------|-----|
| **新分野展開** | 既存事業とは異なる分野に進出 | 製造業が小売業に進出 |
| **事業転換** | 事業の内容を大きく転換 | 飲食店がテイクアウト・デリバリー中心に転換 |
| **業種転換** | 既存の業種分類から別の業種に転換 | 建設業が福祉サービス業に転換 |

### 基本要件

新事業進出補助金の申請には、以下の要件を満たす必要があります。

#### 1. 付加価値額の向上要件

```
付加価値額年率向上率 ≧ 3% 以上（補助金交付後3年間）
```

計算方式はものづくり補助金と同じです。

#### 2. 給与支給総額の向上要件

```
給与支給総額年率向上率 ≧ 1.5% 以上（3年間）
```

#### 3. 経営継続力

新規事業進出後も既存事業を継続するか、新規事業が十分な採算見込みを持つことが必要です。

### 補助上限額と補助率

| 企業規模 | 補助上限 | 補助率 |
|---------|---------|--------|
| 小規模企業者 | 2,500万円/4,000万円/9,000万円 | 2/3 |
| 中小企業 | 2,500万円/4,000万円/9,000万円 | 1/2 |

**注**：上限は進出の規模や経営危機度により3段階に分かれます。

### 補助対象経費

- **建物の改装・改築費**
- **機械装置・設備購入費**
- **システム開発費**
- **マーケティング費**
- **人材採用・研修費**

## 補助金と助成金の区別

中小企業向けの支援制度を学ぶときに、「補助金」と「助成金」の違いを理解することは重要です。試験でも曖昧に答えるとミスリードされやすい分野です。

### 比較表

| 項目 | 補助金 | 助成金 |
|------|--------|--------|
| **所管省庁** | 経済産業省、中小企業庁 | 厚生労働省（主に） |
| **採択方式** | 公募・審査あり（競争的） | 条件達成で原則支給 |
| **原資** | 国の一般会計予算 | 雇用保険料 |
| **性格** | 事業活動・投資を支援 | 雇用・労働環境を支援 |
| **実績報告** | 精算払い（事業完了後） | 精算払い（実績達成後） |
| **年度変更** | 頻繁に変わる | 比較的安定 |

### 代表的な助成金

- **キャリアアップ助成金**：有期雇用労働者の正社員化を支援
- **人材育成支援助成金**：従業員訓練を支援
- **両立支援等助成金**：仕事と育児・介護の両立を支援

### 試験での区別ポイント

補助金と助成金を区別する設問では、以下のポイントで判定できます。

- **「小規模事業者向け」と明記** → 補助金（持続化など）
- **「従業員の研修」「正社員化」** → 助成金
- **「経営計画を策定」** → 補助金
- **「雇用契約の変更」「給与体系」** → 助成金

## 補助金申請の共通ルールと留意点

### 1. 事前着手禁止の原則

```
交付決定前の事業実施は補助対象外
```

**具体例**：
- 交付決定前に機械を発注した → その支払いは対象外
- 交付決定前に工事を開始した → その期間の工事費は対象外
- 交付決定前に広告を出稿した → その広告費は対象外

**申請時の対応**：事業計画書に「事業着手は交付決定後予定」と明記し、見積書や発注日が交付決定後であることを確認します。

### 2. GビズIDプライム取得

```
ほぼ全ての補助金申請に必須
```

**取得方法**：e-Gov（電子政府ポータル）で取得申請（1〜2週間）

**重要**：申請期限が迫ってからGビズIDを申請するとタイムアウトするため、事前取得が推奨されます。

### 3. 認定支援機関の確認

一部の補助金（ものづくり補助金など）では、認定支援機関（中小企業診断士、税理士、金融機関等）による「経営革新計画の承認」「事業継続力強化計画の認定」が加点項目になります。

### 4. 補助金の支払いは後払い（精算払い）

```
施工→実績報告→審査→補助金支給
```

つまり、事業者が先に資金を投入し、完工後に精算払いを受ける形式です。つなぎ融資の利用を検討する必要がある場合も多いです。

### 5. 報告義務と遡求

採択後、以下の報告が義務づけられます。

- **実績報告**：事業完了後、成果物や領収書で実績を報告
- **事業化状況報告**：補助事業終了後、最大5年間の追跡報告
- **売上高等の報告**：事業効果（売上増加、雇用創出等）を定期報告

要件未達成や不正が判明した場合、補助金の返納（遡求）を求められることがあります。

## 典型的なつまずきと回答の工夫

### つまずき① 補助金名だけで判定してしまう

**NG例**：「ものづくり補助金は製造業専用」

**改善**：目的を確認する。ものづくり補助金は「革新的な製品・サービス開発」が軸であり、製造業だけでなく飲食業やサービス業でも申請可能（ただし革新性が必要）。

### つまずき② 補助上限だけで補助金を区別する

**NG例**：「補助上限が大きい = より多く支援される」

**改善**：補助率と補助上限を組み合わせて考える。
- A補助金：上限1,000万円、補助率1/2 → 最大500万円支給
- B補助金：上限200万円、補助率2/3 → 最大133万円支給

実際に受け取る補助金額はB補助金の方が少ない場合もあります。

### つまずき③ 要件を「達成すべき理想」と勘違いする

**NG例**：「付加価値額年3%は目標値」

**改善**：これは「必須要件」です。達成できないと交付決定を受けられないか、交付決定後に返納を求められます。

### つまずき④ インボイス特例を見落とす

**NG例**：「小規模事業者持続化補助金の上限は50万円」

**改善**：免税事業者がインボイス発行事業者に転換する場合、インボイス特例で50万円上乗せされるため、最大100万円まで可能。

### つまずき⑤ 新事業進出補助金と事業再構築補助金を混同

**NG例**：「事業再構築補助金で申請しよう」

**改善**：事業再構築補助金は2024年度で廃止。2025年度以降は新事業進出補助金を使います。

## 問題を解くときの観点

### チェックリスト

補助金に関する問題が出たときは、以下の順序で判断します。

```
1. 経営課題は何か？
   ├─ 販路開拓 → 持続化補助金
   ├─ IT導入・デジタル化 → IT導入補助金
   ├─ 革新的開発・改善 → ものづくり補助金
   ├─ 人手不足対応 → 省力化投資補助金
   └─ 新分野進出・業種転換 → 新事業進出補助金

2. 対象企業の規模は？
   ├─ 従業員5人以下（商業）or 20人以下（製造） → 持続化補助金候補
   └─ それ以上 → その他の補助金

3. 必須要件を満たすか？
   ├─ 付加価値額年3%向上が可能か？
   ├─ 賃上げ（給与総額1.5%など）が可能か？
   └─ 事業計画は具体的か？

4. 補助率と補助額の計算
   ├─ 補助上限の確認
   ├─ 企業規模による補助率の確認
   └─ 実際の受取額 = 対象経費 × 補助率（ただし上限まで）
```

### 典型的な誤答パターンと正解への導き方

| 問題パターン | NG回答 | 正解への考え方 |
|---------|--------|--------------|
| 小規模事業者が販路開拓を行いたい | ものづくり補助金 | 対象が「小規模」で「販路開拓」→ 持続化補助金 |
| 中小企業が会計ソフト導入したい | ものづくり補助金 | 「IT導入」が中心 → IT導入補助金。ベンダーと共同申請が特徴 |
| 製造業が生産ラインを自動化したい | IT導入補助金 | 「省人化投資」が軸 → 省力化投資補助金。または革新性があれば → ものづくり補助金 |
| 企業が新業種に進出したい | ものづくり補助金 | 「業種転換」「新分野展開」 → 新事業進出補助金（旧・事業再構築） |
| 小規模店舗が従業員を正社員化したい | 補助金全般 | これは雇用関連 → 厚労省の「助成金」（例：キャリアアップ助成金） |

## 3つの確認問題

<Callout title="確認問題1：小規模事業者の販路開拓支援">
    ある小規模事業者（従業員8人の小売業）が、既存事業の販路を拡大するため、新しい販売チャネル（SNS広告、展示会出展、パンフレット作成）の開発に取り組もうとしています。この事業者が活用すべき補助金と、その理由を述べてください。また、この補助金の申請に必須の機関は何ですか？

    **想定される答え**：
    - **補助金**：小規模事業者持続化補助金
    - **理由**：(1) 対象が小規模事業者（5人以下の商業は対象外だが、8人は特例あり）、(2) 販路開拓が中心目的、(3) 経営計画に基づく販売促進を支援
    - **必須機関**：商工会または商工会議所（経営計画策定の支援と確認）
</Callout>

<Callout title="確認問題2：IT導入の特徴的な申請方法">
    ある中小企業が、顧客管理システムとPOSレジシステムを導入することで、業務効率化を図ろうとしています。IT導入補助金を申請する場合、この企業が気をつけるべき申請手続きは何ですか？また、補助対象経費に含まれるもの、含まれないものをそれぞれ2つずつ例示してください。

    **想定される答え**：
    - **申請手続き**：IT導入補助金は、IT導入支援事業者（ベンダー）との共同申請が必須。企業が直接申請できない。
    - **補助対象経費**：ソフトウェア購入費、クラウド利用料（最大2年分）、導入サポート費、コンサルティング費など
    - **補助対象外経費**：従業員研修費（不可）、3年目以降のクラウド利用料（不可）、既存システムの廃棄費（不可）
</Callout>

<Callout title="確認問題3：ものづくり補助金の3つの要件を計算する">
    ある製造業（従業員25人）の事業者が、新製品開発のため1,000万円の設備投資をものづくり補助金で申請しようとしています。以下の財務データをもとに：

    - 前年度：売上高10億円、営業利益1,000万円、人件費2億円、減価償却費1,000万円
    - 申請年度（予想）：売上高10.5億円、営業利益1,200万円、人件費2.05億円、減価償却費1,200万円

    (1) 付加価値額の年率向上率を計算してください。
    (2) 給与支給総額の年率向上率を計算してください。
    (3) 補助金を受けるには、これらの要件を何年間達成し続ける必要がありますか？

    **想定される答え**：

    (1) 付加価値額の年率向上率
    ```
    基準年度付加価値額 = 1,000 + 20,000 + 1,000 = 3,100万円
    申請年度付加価値額 = 1,200 + 20,500 + 1,200 = 3,900万円
    向上額 = 3,900 - 3,100 = 800万円
    向上率 = 800 ÷ (10億 × 100) = 800 ÷ 100,000 = 0.8%

    → 3% 以上の要件に達していない（不適格）
    ```

    (2) 給与支給総額の年率向上率
    ```
    前年度：2億円
    申請年度：2.05億円
    向上額：0.05億円（500万円）
    向上率 = 500万円 ÷ 2億円 = 2.5%

    → 1.5% 以上の要件を満たす（適格）
    ```

    (3) 補助金交付後、**3年間**、毎年これらの要件を達成し続ける必要があります。
</Callout>
