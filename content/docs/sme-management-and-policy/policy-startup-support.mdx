## 学習のポイント

- **産業競争力強化法**：創業支援を国が認定し、地域の支援枠組みを強化する根拠法
- **創業支援等事業計画**：市区町村が国に提出する地域全体の支援枠組み計画
- **特定創業支援等事業**：商工会議所等が実施する経営・財務・人材・販路の継続的学習
- **支援措置の4本柱**：融資（新規開業・スタートアップ支援資金）、保証（スタートアップ創出促進保証）、税制優遇（登録免許税軽減）、支援対象の拡大
- **創業支援の時間軸**：創業2か月前から対象（特定創業支援は6か月前から）
- **エンジェル税制**：個人投資家がスタートアップ出資時に所得控除or譲渡益圧縮が受けられる税優遇
- **経営者保証の廃除**：スタートアップ創出促進保証では経営者保証なしで最大3,500万円の信用保証が可能

## 試験で何が問われるか

- 産業競争力強化法が創業支援にどう活用されているか説明できるか
- 創業支援等事業計画を、市区町村が国と地域機関を調整する仕組みとして理解しているか
- 特定創業支援等事業を、継続的な学習支援と受けた創業者への優遇措置が一体になった制度として位置付けられるか
- 4つの支援措置（融資・保証・税制・対象拡大）を、それぞれ異なる役割を果たす補完的な支援として整理できるか
- 登録免許税の軽減率（1.0%→0.35%）を数値として記憶しているか
- スタートアップ創出促進保証の限度額（3,500万円）、経営者保証なしという特徴を言えるか
- エンジェル税制の2つの選択肢（所得控除 vs. 譲渡益圧縮）を区別できるか
- 創業相談の対象時期（創業前の段階からいつから対象か）を正確に述べられるか

## 産業競争力強化法に基づく創業支援の枠組み

### 産業競争力強化法とは

**産業競争力強化法** は、日本の産業競争力を高めるための基本法です。その中で創業支援は重要な施策として位置付けられています。

- **目的**：国と地域が連携し、創業環境を整備し、起業を促進する
- **手段**：市区町村による「創業支援等事業計画」を国が認定し、認定地域の創業者に優遇措置を与える
- **位置付け**：ベンチャー企業やスタートアップの生成を通じた経済成長戦略
- **支援対象時期**：創業前（準備段階）から創業後初期段階まで

### 創業支援等事業計画の構造

**創業支援等事業計画** は、市区町村が策定し国が認定する、地域全体の創業支援の基本枠組みです。

#### 主体と構成

| 要素 | 説明 |
| --- | --- |
| **策定主体** | 市区町村 |
| **認定主体** | 国（経済産業省・中小企業庁） |
| **協力機関** | 地域金融機関、商工会議所、商工会、NPOなど（複数機関の連携が前提） |
| **効果** | 認定地域の創業者に各種優遇措置が適用される根拠となる |

#### 創業支援等事業計画に記載される内容

計画書には、以下の連携体制が明記されます：

1. **相談支援体制**：どこで創業相談が受けられるか
2. **融資支援体制**：どの金融機関が創業融資を扱うか
3. **経営支援体制**：どの機関が経営指導・人材育成を行うか
4. **販路開拓支援体制**：どこが販売先確保を支援するか

## 特定創業支援等事業と支援措置

### 特定創業支援等事業の定義

**特定創業支援等事業** は、認定を受けた創業支援等事業計画に基づき、実際に創業者が受ける具体的で継続的な支援活動です。

#### 4つの支援内容

特定創業支援等事業は、以下の4つの知識習得を目的とする継続的支援で構成されます：

| 支援分野 | 対象項目 | 学習内容 |
| --- | --- | --- |
| **経営知識** | 事業計画、会社法、競争戦略 | 事業を持続可能にするための経営判断 |
| **財務知識** | 資金繰り、決算書読み方、税務 | キャッシュフロー管理と財務健全性 |
| **人材知識** | 採用、教育、労務管理 | 組織構築と人材育成 |
| **販路知識** | マーケティング、営業戦略、商品開発 | 売上拡大と顧客獲得 |

#### 「継続的」の意味

重要なのは、これが「単発の相談ではなく、一定期間にわたり複数回の支援を受ける」ことです。

- **実施期間**：通常、数か月にわたる複数セッション
- **実施機関**：商工会議所、商工会、NPO法人など（市区町村が指定した機関）
- **対象者**：新たに事業を始める者、または事業開始後5年未満の者

#### 創業支援等事業計画と特定創業支援等事業の関係

| 論点 | 創業支援等事業計画 | 特定創業支援等事業 |
| --- | --- | --- |
| **性格** | 地域全体の支援枠組み | 創業者が受ける具体的な支援活動 |
| **主体** | 市区町村が策定 | 商工会議所等が実施 |
| **内容** | 複数機関の連携体制を定める | 経営・財務・人材・販路の継続的学習 |
| **効果** | 支援措置の適用基盤となる | 受けた創業者に優遇措置が与えられる |

特定創業支援等事業を **受けた創業者** が、各種支援措置（融資枠の拡大、金利引き下げなど）の対象となります。

## 創業者向けの主な支援措置

### 金融支援：新規開業・スタートアップ支援資金

**新規開業・スタートアップ支援資金** は、日本政策金融公庫が提供する創業向け融資制度です。2025年3月より、従来の「新規開業資金」から名称が変更されました。

#### 制度の特徴

| 項目 | 内容 |
| --- | --- |
| **融資主体** | 日本政策金融公庫 |
| **対象者** | 新たに事業を始める者、または事業開始6か月以内の者 |
| **融資額** | 最大3,000万円（うち運転資金は1,500万円） |
| **融資期間** | 最大20年 |
| **担保・保証** | 原則不要（ただし対象によっては求めることもある） |
| **金利** | 政策金融公庫の基準金利（市場金利より低め） |
| **優遇措置** | 特定創業支援等事業を受けた場合、金利引き下げ |

#### 特定創業支援等事業を受けた場合の優遇

特定創業支援等事業の支援を受けた創業者は、新規開業・スタートアップ支援資金の **金利が0.2～0.3%引き下げられる** など有利な条件を得られます。

### 信用保証：スタートアップ創出促進保証

**スタートアップ創出促進保証** は、信用保証協会が提供する、経営者保証を不要とする新しい保証制度です。

#### 制度の特徴

| 項目 | 内容 |
| --- | --- |
| **保証主体** | 信用保証協会 |
| **対象者** | 新たに事業を始める者、または事業開始10年未満の者 |
| **保証限度額** | 3,500万円（通常の中小企業向け保証より大きい） |
| **経営者保証** | **不要**（重要な特徴） |
| **対象融資** | 民間金融機関からの借入 |
| **保証料** | 通常の保証より引き上げられる可能性あり |

#### 「経営者保証なし」の意味

通常、中小企業の融資では代表者が個人保証を行うのが慣例です。しかしスタートアップ創出促進保証では、信用保証協会の保証だけで融資が可能となり、個人資産を失う風険が軽減されます。これはスタートアップ創業者の資産保全を考慮した政策的な設計です。

### 税制優遇（1）：登録免許税の軽減

法人設立時の登録免許税は、通常は資本金の0.7%です。しかし特定創業支援等事業を受けた場合、軽減されます。

#### 軽減の内容

| 対象 | 通常税率 | 軽減税率 | 軽減幅 |
| --- | --- | --- | --- |
| **株式会社設立** | 0.7% | 0.35% | 50%軽減 |
| **合同会社設立** | 0.6% | 0.3% | 50%軽減 |

#### 軽減の適用条件

- 特定創業支援等事業を受けたことを証明する書類（市区町村が発行）
- 登記申請時にその証明書を添付

#### 計算例

資本金1,000万円で株式会社を設立する場合：

- **通常**：1,000万円 × 0.7% = 70,000円
- **軽減**：1,000万円 × 0.35% = 35,000円
- **節税額**：35,000円

創業初期の資金繰りが厳しい時期に、こうした節税は重要な支援となります。

### 税制優遇（2）：エンジェル税制

**エンジェル税制** は、個人投資家がスタートアップ企業に出資した際の税優遇制度です。企業からの支援ではなく、投資家側への支援です。

#### 対象企業の要件

エンジェル税制の対象となるスタートアップは、以下の要件を満たす必要があります：

- **事業開始後間もない**：設立から3年未満
- **労働人口**：常時10人以下
- **営利目的**：営利事業を行う法人
- **除外業種**：金融商品・証券、投資事業など特定業種を除く

#### 2つの優遇オプション

エンジェル税制では、投資家が2つの優遇方法から選択できます：

##### オプション1：所得控除

個人投資家が出資した年の所得から、出資金額を控除できます。

- **対象所得**：その年の給与所得や事業所得
- **控除上限**：年間で最大2,000万円（出資額がこれを超える場合）
- **効果**：その年の税負担が軽減される
- **適用時期**：出資年の翌年の確定申告で申告

##### オプション2：株式譲渡益の圧縮

将来、その企業の株式を売却して利益が出た場合、その利益を圧縮する方法です。

- **適用対象**：売却時の株式譲渡益
- **圧縮方法**：譲渡益から当初の出資額を控除（実質的に利益が減る）
- **効果**：譲渡益への課税が軽減される
- **適用時期**：株式売却時の翌年確定申告で申告

#### 2つのオプションの使い分け

| 選択肢 | こんなときに使う | メリット |
| --- | --- | --- |
| **所得控除** | 出資した年に所得がある場合 | その年の税負担をすぐ軽減 |
| **譲渡益圧縮** | 出資したスタートアップが成長して株式が値上がりした場合 | 大きな譲渡益が出たときに課税を軽減 |

## 支援措置の時間軸：創業前から創業直後まで

### 創業支援の対象時期

創業支援には明確な時間軸があります：

| 段階 | 時期 | 対象支援 | 実施主体 |
| --- | --- | --- | --- |
| **創業前準備期** | 創業2か月前～0日 | 創業相談、事業計画作成支援 | 商工会議所、商工会、金融機関 |
| **創業初期（特定支援対象）** | 創業6か月前～創業後5年未満 | 特定創業支援等事業（4分野の継続的学習） | 商工会議所等認定機関 |
| **創業直後の金融支援** | 創業6か月以内 | 新規開業・スタートアップ支援資金 | 日本政策金融公庫 |
| **信用保証の対象** | 創業2か月前～事業開始10年未満 | スタートアップ創出促進保証 | 信用保証協会 |

#### なぜこのタイミングか

- **創業2か月前から対象**：事業開始の数か月前から準備が必要（市場調査、資金調達、人材確保）
- **特定創業支援は6か月前から**：より充実した学習が必要なため、より早期から対象
- **創業直後の融資**：立ち上げにかかる初期投資（設備、運転資金）への対応
- **10年未満の信用保証**：新規企業は信用履歴がないため、保証が有効

## 支援措置の組み合わせ（シーン別）

### シーン1：事業開始3か月前の起業家

```
① 商工会議所で創業相談 → ②相談内容から特定創業支援等事業の受講を勧められる → ③3か月かけて経営・財務・人材・販路をまなぶ
↓
④ 株式会社を設立 → 登録免許税0.35%で軽減（市区町村から証明書をもらう）
↓
⑤ 日本政策金融公庫で新規開業・スタートアップ支援資金を申請
   特定創業支援等事業を受けたので、金利が0.2～0.3%引き下げられる
↓
⑥ 信用保証協会からスタートアップ創出促進保証を受ける（経営者保証なし、限度額3,500万円）
```

この場合、相談支援（①～③）と金融支援（④～⑥）、税制支援（④）が一体で機能します。

### シーン2：個人投資家がスタートアップに投資する場合

```
個人投資家 → スタートアップ企業へ1,000万円出資
↓
① 所得控除のオプション選択：その年の給与所得から1,000万円控除
   → 所得税が軽減される
↓
または
↓
② 譲渡益圧縮のオプション選択：将来、株式が1,500万円で売却できた場合
   譲渡益＝1,500万円 - 1,000万円 = 500万円 → さらに1,000万円を控除（損失繰越）
   → 譲渡益への課税が軽減 or 繰越損失で他所得から控除
```

## 創業支援政策が解く課題と限界

### 創業支援が取り組む課題

日本の起業率は欧米（10～15%程度）に比べ低いままです（約3.9%）。創業支援策は以下の課題に対応しています：

| 課題 | 支援措置による対応 |
| --- | --- |
| **事業計画の不十分さ** | 特定創業支援等事業で経営・財務知識の学習 |
| **初期資金の不足** | 新規開業・スタートアップ支援資金の融資 |
| **信用力がない** | 信用保証協会の保証（経営者保証なし） |
| **起業への踏み出しの心理的障壁** | 相談支援で不安軽減、計画の実現可能性を検証 |
| **起業後のキャッシュフロー危機** | 金利引き下げによる返済負担軽減 |

### 限界と課題

支援措置が充実しても、起業率が大幅に上昇しない理由：

- **個人の意思決定**：支援があっても起業を選ばない人が多い
- **事業継続の困難さ**：創業後の市場競争、人材確保、顧客獲得の困難さまでは支援の範囲外
- **後継者問題**：創業支援で新規企業が増えても、既存企業の廃業が多い場合は相殺される
- **成長支援との連携**：創業直後の支援は充実だが、その後の事業規模拡大への支援は異なる仕組み

## 典型的なつまずき

試験学習でよく見られる誤解：

### つまずき1：創業支援等事業計画と特定創業支援等事業を混同する

**誤り**：「創業支援等事業計画を市区町村が実施する」
**正しい理解**：市区町村は「計画」を策定し国に認定してもらう。実際の「支援活動」（特定創業支援等事業）は商工会議所などが実施する。

### つまずき2：支援措置を全て創業者が自動で受けられると思う

**誤り**：「創業支援等事業計画の認定地域なら、全ての創業者が優遇措置を受けられる」
**正しい理解**：認定は地域に与えられるが、優遇措置を受けるには、実際に「特定創業支援等事業」を受ける必要がある。

### つまずき3：スタートアップ創出促進保証の「経営者保証なし」を誤解する

**誤り**：「経営者保証なしなら、焦げ付いても企業は損をしない」
**正しい理解**：保証協会が債務を肩代わりするだけで、企業は返済義務がある。経営者保証がないのは個人資産が対象外という意味。

### つまずき4：登録免許税軽減の対象を誤解する

**誤り**：「特定創業支援等事業を受けた全ての企業が0.35%に軽減」
**正しい理解**：軽減を受けるには、市区町村から「支援を受けたことの証明書」をもらい、登記申請時に提出する手続きが必要。

### つまずき5：エンジェル税制の適用条件を狭く考える

**誤り**：「エンジェル税制は大型スタートアップのみの制度」
**正しい理解**：設立3年未満で常時10人以下の小さなスタートアップも対象。ベンチャーキャピタルではなく「個人投資家」向け。

## 問題を解くときの観点

### 問題文の読み分け

支援政策問題では、以下の3つの角度から問題を分類し、答え方を変えます：

#### 観点1：「誰が」支援を実施するのか

| 問題の指示 | 答える主体 |
| --- | --- |
| 「自治体が実施する創業支援」 | 創業支援等事業計画の策定・国への認定申請 |
| 「創業者が受ける支援」 | 特定創業支援等事業、融資、保証、税優遇 |
| 「金融機関の役割」 | 融資の実行（政策金融公庫）、保証（信用保証協会） |

#### 観点2：「どの段階」の支援か

| 段階 | 対応する支援 |
| --- | --- |
| 創業前（2か月前～） | 相談支援、事業計画作成支援 |
| 創業初期（6か月前～5年未満） | 特定創業支援等事業（経営・財務・人材・販路） |
| 創業直後の資金調達 | 融資（新規開業・スタートアップ支援資金）、保証 |
| 投資家の側 | エンジェル税制（所得控除 or 譲渡益圧縮） |

#### 観点3：「何が優遇される」のか

| 優遇対象 | 内容 |
| --- | --- |
| 登録免許税 | 0.7% → 0.35%（50%軽減） |
| 融資金利 | 政策金融公庫の金利から0.2～0.3%引き下げ |
| 信用保証 | 経営者保証不要、限度額3,500万円 |
| 投資家の税負担 | 所得控除 or 株式譲渡益圧縮 |

## 確認問題

<Callout title="確認問題1：制度の主体と役割">
次の文章の（　）内に適切な言葉を入れなさい。

「市区町村が（　①　）に創業支援等事業計画を提出し、認定を受けることで、地域の創業者は登録免許税の軽減などの優遇措置が受けられるようになる。その際、市区町村は地域の（　②　）、商工会議所、商工会などと（　③　）して、相談、融資、経営指導の体制を整える。実際に創業者が受ける継続的な経営・財務・人材・販路の学習支援を『（　④　）』と呼び、これを受けた創業者に対して各種優遇措置が適用される」

**ポイント**：①は国の機関、②は民間金融、③は連携の形、④は制度名です。
</Callout>

<Callout title="確認問題2：支援措置の計算と区別">
以下の3つのシーンで、それぞれ受けられる支援措置を述べなさい。

（1）特定創業支援等事業を受けた創業者が、資本金500万円で株式会社を設立する場合、登録免許税をいくら節税できるか。

（2）新規開業・スタートアップ支援資金と、スタートアップ創出促進保証の違いを説明しなさい（融資と保証の区別、限度額、経営者保証の有無に言及）。

（3）個人投資家がスタートアップに1,000万円出資し、その年の給与所得が800万円だった場合、所得控除を選択すると、その年の課税所得がいくらになるか。

**ポイント**：①は計算問題（軽減税率を使う）、②は制度の機能差、③は税法上の所得計算です。
</Callout>

<Callout title="確認問題3：創業支援の時間軸と条件">
「Aさんは2026年4月1日に創業予定です。その前後に利用できる支援制度と対象時期を整理しなさい」

以下について、対象時期や対象条件を述べよ。

- 創業相談と特定創業支援等事業の対象時期の違い
- スタートアップ創出促進保証を受ける場合、最遅で何日までに融資を受ける必要があるか
- エンジェル税制の対象企業の条件（設立からの経過年数、従業員数）

**ポイント**：支援ごとに対象時期が異なることが試験頻出です。特に「創業2か月前」と「6か月前」の違いを正確に押さえることが重要。
</Callout>

## 2026-03-28 時点の最新確認

- **創業支援等事業計画**：中小企業庁の公式サイトで、市区町村向けの認定申請フロー、認定地域一覧が案内されている
- **特定創業支援等事業の実施機関**：商工会議所、商工会、NPO法人など地域ごとに異なるため、創業者は市区町村に確認して利用機関を特定する必要がある
- **新規開業資金の名称変更**：2025年3月1日より「新規開業資金」から「新規開業・スタートアップ支援資金」に改称、支援対象やメニューの整理が行われた
- **スタートアップ創出促進保証**：2023年度に新設された比較的新しい保証制度。経営者保証を不要とする特徴が試験での出題ポイント
- **登録免許税の軽減**：2024年度現在も市区町村の証明書があれば適用され、登記申請時に手続きが必要
- **エンジェル税制**：2025年度税制改正で制度の拡充が議論されており、最新の要件・限度額は国税庁の年度別案内を参照すること
