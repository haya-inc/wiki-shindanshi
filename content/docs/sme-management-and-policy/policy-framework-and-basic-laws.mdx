## 中小企業基本法（最重要）

### 基本構成：目的と基本理念

**中小企業基本法**は、中小企業政策体系全体の最上位法であり、全ての個別施策の根拠となります。

**目的**：
- 国民経済の健全な発展
- 国民生活の向上

**基本理念**：「多様で活力ある成長発展」

この基本理念が、中小企業政策の全体的な方向性を規定します。

### 1999年改正：政策転換の転換点

1999年の改正は、中小企業政策における**極めて重要な転換点**です。試験でも頻繁に問われます。

**改正前の政策理念**：
- **格差是正**を重視
- 大企業との規模格差を埋めることを目標
- 二重構造論：大企業に従属する下請け・関連企業という位置づけ

**改正後の政策理念**：
- **成長発展**を重視
- 中小企業自体の多様で活力ある発展を目標
- 大企業との関係性に限定されない、自立的な発展を支援

この転換は、単に支援額の増加ではなく、**政策の根本的な考え方の変更**です。格差の解消ではなく、中小企業の自発的な成長を促すという方向へシフトしました。

### 4つの基本方針

中小企業基本法は、この基本理念を実現するため、4つの基本方針を定めています。

**①経営革新・創業の促進**
- 新事業への進出、新たな経営方法の導入を支援
- スタートアップ企業や既存企業の事業多角化を促進

**②経営基盤の強化**
- 経営管理体制の整備
- 人材育成、技術開発
- 労働環境の改善

**③環境変化への適応**
- グローバル化への対応
- デジタル化・DXへの対応
- 気候変動への対応

**④資金供給の充実**
- 融資制度の拡充
- 信用保証制度の活用
- 投資育成会社による直接投資

## 政策体系図

中小企業政策は、中小企業基本法を頂点として、複数の個別法によって構成されています。

```
中小企業基本法（最上位）
  ├─ 中小企業等経営強化法（最重要）
  │  ├─ 経営革新計画
  │  ├─ 経営力向上計画
  │  ├─ 先端設備等導入計画
  │  └─ 事業継続力強化計画
  │
  ├─ 小規模企業振興基本法（2014年制定）
  │
  ├─ 中小企業支援法
  │
  ├─ 産業競争力強化法（創業支援）
  │
  ├─ 経営承継円滑化法（事業承継）
  │
  ├─ 下請代金支払遅延等防止法
  │
  ├─ 中小企業等協同組合法
  │
  ├─ 地域未来投資促進法
  │
  └─ 中小企業地域資源活用促進法
```

**読み方**：中小企業基本法の4つの基本方針を具現化するため、それぞれの課題領域に応じた個別法が存在します。試験では、この体系の中での各法律の位置づけを理解することが重要です。

## 小規模企業振興基本法（2014年制定）

### 制定背景と基本原則

**小規模企業振興基本法**は、2014年に制定された比較的新しい法律です。小規模企業（従業員5～20人程度）の特性を踏まえた支援を規定しています。

**基本原則**：「持続的発展」

この「持続的発展」は、中小企業基本法の「成長発展」と対比されることが試験で頻出です。

- **成長発展**：売上・従業員数の拡大、事業規模の拡張を目指す
- **持続的発展**：現在の規模を保ちながら、経営の質的向上、地域への貢献を目指す

小規模企業は、必ずしも規模拡大を求めない場合が多いため、このような使い分けが必要です。

### 4つの柱と振興基本計画

小規模企業の振興施策は、以下の4つの柱で構成されます。

**①需要開拓**
- 新規市場への進出支援
- 販売力強化、マーケティング支援

**②経営資源の充実**
- 後継者育成
- 経営技術の習得
- 従業員教育

**③地域活性化への貢献**
- 地域経済の循環
- 地域文化の継承

**④支援体制の整備**
- 商工会・商工会議所の機能強化
- 認定経営革新等支援機関の拡充

**振興基本計画**は、5年ごとに見直されます。法改正や経済環境の変化に応じて、支援の重点が調整されることに注意してください。

## 中小企業等経営強化法と4つの認定計画制度

### 4つの計画制度の比較表

中小企業等経営強化法は、企業の成長段階や経営課題に応じた複数の認定計画制度を提供しています。試験では、この4つの制度の違いを正確に区別できることが必須です。

| 計画名 | 認定機関 | 根拠法 | 対象課題 | 主な支援内容 |
|---|---|---|---|---|
| **経営革新計画** | 都道府県知事 | 中小企業等経営強化法 | 経営革新（新商品・新役務など） | 低利融資、信用保証別枠、投資育成特例、補助金加点 |
| **経営力向上計画** | 主務大臣 | 中小企業等経営強化法 | 経営力の質的向上 | 固定資産税軽減、経営強化税制（即時償却 or 税額控除） |
| **先端設備等導入計画** | 市区町村 | 中小企業等経営強化法 | 生産性向上（先端設備投資） | 固定資産税軽減（ゼロ特例）、補助金 |
| **事業継続力強化計画** | 経済産業大臣 | 中小企業等経営強化法 | 事業継続力（BCP） | 特別償却（20%）、補助金加点、低利融資、保険料割引 |

**重要な区別**：
- **認定機関が異なる**：都道府県知事、主務大臣、市区町村、経済産業大臣のいずれかであることを正確に押さえてください。特に先端設備等導入計画は「市区町村」であり、「都道府県」ではありません。

## 経営革新計画

### 定義と5つの類型

**経営革新計画**は、中小企業等が新しい事業展開、経営方法の改善を計画し、都道府県知事の承認を受けて実行するものです。

**5つの類型**：

1. **新商品・新役務の開発・生産**
   - 例：既存製品の廃止と新製品開発
   - 例：新しいサービスの提供開始

2. **商品・役務の新しい生産・提供方式の導入**
   - 例：製造プロセスの改善（3次元印刷導入など）
   - 例：デジタル化による業務効率化

3. **商品・役務の新しい販売方法の導入**
   - 例：オンライン販売の開始
   - 例：直販体制への転換

4. **新しい経営管理方法の導入**
   - 例：ERP システムの導入
   - 例：管理会計体制の整備

5. **新しい事業の開始**
   - 例：既存ビジネスとは別の新規事業進出
   - 例：既存企業による創業支援事業の開始

複数の類型を組み合わせることも可能です。

### 承認基準

経営革新計画が都道府県知事に承認されるには、以下の基準を満たす必要があります。

**定量基準（必須）**：

- **付加価値額：年率3%以上の向上**
  - 計算式：`付加価値額 = 営業利益 + 人件費 + 減価償却費`
  - 例：付加価値額が1年目1,000万円 → 3年目1,093万円以上に向上

- **経常利益：年率1%以上の向上**
  - 3年計画での目標を設定
  - 例：経常利益が1年目100万円 → 3年目103万円以上に向上

これら2つの定量基準は、経営革新の実効性を示す重要な指標であり、試験で頻出です。

**定性基準**：
- 新規性（他社に比べて新しいか）
- 実行可能性（計画の現実性）
- その他、地域経済への波及効果など

### 支援内容

承認を受けた経営革新計画には、以下の支援が提供されます。

**①金融支援**
- 低利融資（政策金融公庫など）
- 利率：通常より0.5～1.0%程度低い

**②信用保証**
- 信用保証別枠（単独枠とは別に保証可能）
- 無担保・無保証人での借入が可能な場合がある

**③投資育成会社法の特例**
- 設立5年未満の企業への直接投資が可能に

**④補助金**
- 各種中小企業支援補助金の採択において加点

## 経営力向上計画

### 定義と認定の流れ

**経営力向上計画**は、中小企業が経営の質的向上（生産性向上、競争力強化）を計画し、主務大臣（経済産業大臣が中心）の認定を受けるものです。

経営革新計画との大きな違いは：
- **経営革新計画**：新しいものへの取り組み（オフェンス的）
- **経営力向上計画**：既存事業の質的向上（ディフェンス的）

### 支援内容：固定資産税軽減と経営強化税制

**固定資産税軽減**
- 対象資産：取得から3年以内の器械・装置、建物など
- 軽減率：固定資産税を2分の1に軽減（3年間）
- 金額効果：大型投資の場合、数百万円の節税になる場合がある

**経営強化税制**（2つの選択肢から1つを選択）

| 税制 | 概要 | 対象資産 | 利用企業 |
|---|---|---|---|
| **即時償却** | 取得した年度に100%償却（損金計上） | 全ての対象設備 | 利益が出ている企業向け |
| **税額控除** | 取得額の10%を法人税から直接控除（3千万円超は7%） | 生産性向上に資する設備 | 利益が出ている企業・出ていない企業両対応 |

**選択のポイント**：
- 利益が多い年は**税額控除**で直接税負担削減
- 利益が少ない年は**即時償却**で損失繰越で将来控除
- 企業の経営状況に応じて、税理士と相談して選択

### 設備類型A～D

認定計画の対象となる設備を、以下の4つの類型に分類しています。試験では、設備の特徴と類型の対応を問われることがあります。

**A類型：生産性向上設備**
- 労働生産性を向上させる設備
- 例：自動化機械、工業用ロボット、製造効率化装置
- 最新のカタログ搭載ガイドラインに基づく

**B類型：収益力強化設備**
- 売上高や営業利益を向上させる設備
- 例：販売力強化設備、マーケティング施設
- 生産性では測定困難な効果を持つ

**C類型：デジタル化対応設備**
- 企業のDX化を支援する設備
- 例：クラウドシステム、AI分析ツール、IoT センサー
- 近年、注目度が高い類型

**D類型：経営資源集約化設備**
- 中小企業の経営資源を集約化する設備
- 例：共有施設、共同利用設備
- 複数企業による共同投資を想定

## 先端設備等導入計画

### 特徴と認定機関

**先端設備等導入計画**は、先端設備への投資による生産性向上を支援する計画です。

**認定機関**：**市区町村**
- 試験で頻出の引っかけ問題：「都道府県知事」ではなく「市区町村」です。注意してください。

**根拠法**：中小企業等経営強化法

### 認定要件：認定支援機関の確認書

重要な要件として、**認定経営革新等支援機関（経営支援専門家）による確認書**が必須です。

これは、企業の計画が実現可能であることを、認定支援機関（税理士、中小企業診断士、金融機関など）が確認するプロセスです。

**手続きの流れ**：
1. 企業が初期計画案を作成
2. 認定支援機関に相談し、計画の実現可能性を検証
3. 認定支援機関が確認書を発行
4. 企業が市区町村に認定申請
5. 市区町村が認定を付与
6. **設備の取得**（認定後に購入することが重要）

### 支援内容：固定資産税ゼロ特例

**固定資産税軽減（ゼロ特例）**
- 対象資産：先端設備等の取得から3年間
- 軽減率：固定資産税を**全額免除**（ゼロ）
- 他計画との比較：経営力向上計画は「2分の1軽減」、先端設備は「全額免除」という手厚い措置

この全額免除は、固定資産税が数百万円となる大規模投資の場合、非常に効果的です。

**その他の支援**：
- 補助金の対象化
- 低利融資の対象化

## 事業継続力強化計画

### 定義と認定機関

**事業継続力強化計画**は、自然災害や感染症の大流行など、予期しない事態への事業継続対応力を高めるための計画です。

**認定機関**：**経済産業大臣**
- 他の3つの計画（都道府県知事、主務大臣、市区町村）と異なり、経済産業大臣が直接認定します。

**制定背景**：
- 2011年の東日本大震災
- 2020年の新型コロナウイルス感染症拡大
- これらの経験から、事業継続体制（BCP）の整備が政策的に重要性を増しました。

### 5つの柱と計画構成

事業継続力強化計画は、以下の5つの柱で構成されます。

**①目的の明確化**
- なぜ事業継続が必要か（経営方針との整合性）
- どの事業・機能を優先するか（優先度付け）
- 復旧目標（RTO：Recovery Time Objective）の設定

**②リスク評価：ハザードマップの確認**
- 自社が立地する地域の自然災害リスク確認
- 洪水、地震、津波、土砂災害のハザード確認
- リスク評価に基づく対応優先度の決定

**③初動対応の体制整備**
- 災害発生直後の初動チームの編成
- 連絡網の整備
- 初動判断基準の策定（いつ、誰が、何をするか）

**④事前対策**
- 設備・在庫の分散配置
- サプライチェーンの多重化
- 代替生産地の確保
- データバックアップ体制

**⑤訓練と検証**
- 定期的な机上訓練・実地訓練
- 訓練結果に基づく計画の見直し
- 従業員教育の継続

### 支援内容

**①特別償却**
- 対象資産：事業継続に必要な設備・施設
- 償却率：**20%の特別償却**が可能
- 効果：初年度の減価償却費が増加し、法人税を軽減

**②補助金加点**
- 各種中小企業支援補助金の採択時に加点
- 優遇措置により採択率向上

**③低利融資**
- 政策金融公庫などからの低利融資対象化

**④保険料割引**
- 事業継続保険の保険料が割引される場合がある

## 典型的なつまずき

### つまずき①：認定機関の混同

最も頻出の誤りです。

**よくある間違い**：
- 「先端設備等導入計画は都道府県知事の認定」← 誤り、市区町村です
- 「事業継続力強化計画は主務大臣の認定」← 誤り、経済産業大臣です
- 「経営革新計画は市区町村の認定」← 誤り、都道府県知事です

**対策**：4つの計画を表で並べて、認定機関を何度も繰り返し確認してください。

### つまずき②：経営革新計画の承認基準の誤解

**よくある間違い**：
- 「付加価値額の基準は年率5%」← 誤り、3%です
- 「経常利益の基準は年率3%」← 誤り、1%です
- 「単年度での成長を基準にする」← 誤り、3年計画での累積成長です

**対策**：計算式と数値基準を、実例で何度も解きながら確認してください。

### つまずき③：付加価値額の計算式の間違い

**よくある間違い**：
- 「付加価値額 = 売上高 - 原価」← 誤り、営業利益ベースの計算です
- 「付加価値額に減価償却費は含めない」← 誤り、含めます

**正しい計算式**：
```
付加価値額 = 営業利益 + 人件費 + 減価償却費
```

**例**：
- 営業利益：1,000万円
- 人件費：2,000万円
- 減価償却費：500万円
- 付加価値額：3,500万円

### つまずき④：経営力向上計画の税制優遇の誤解

**よくある間違い**：
- 「固定資産税は2分の1軽減」：これは経営力向上計画です
- 「固定資産税は全額免除」：これは先端設備等導入計画です
- 両者を混同しやすいため、注意が必要です

**対策**：
- **経営力向上計画** → 固定資産税「2分の1軽減」＋「経営強化税制」
- **先端設備等導入計画** → 固定資産税「全額免除（ゼロ特例）」

### つまずき⑤：小規模企業振興基本法の基本原則

**よくある間違い**：
- 「小規模企業振興基本法の基本原則は成長発展」← 誤り、持続的発展です
- 成長発展と持続的発展の違いを理解していないと、判別できません

**対策**：
- **成長発展**：中小企業基本法、規模拡張を目指す
- **持続的発展**：小規模企業振興基本法、現在の規模を保ちながら質的向上

## 問題を解くときの観点

### 観点①：4つの計画制度の認定機関を最初に確認

問題文に「市区町村が認定」という記述があったら、まずは**先端設備等導入計画**だと判定してください。認定機関は、4つの計画を区別する最初の判定基準です。

**判定フロー**：
1. 都道府県知事 → 経営革新計画
2. 主務大臣 → 経営力向上計画
3. 市区町村 → 先端設備等導入計画
4. 経済産業大臣 → 事業継続力強化計画

### 観点②：支援内容から逆算する

問題文に「固定資産税軽減」という記述があった場合：
- 「全額免除」なら先端設備等導入計画
- 「2分の1軽減」なら経営力向上計画

このように、支援内容から計画制度を特定することもできます。

### 観点③：計画の対象（課題）から推測する

問題文の「企業の課題」から計画を推測することもできます。

- 「新商品開発に取り組みたい」 → 経営革新計画
- 「既存事業の生産性を向上させたい」 → 経営力向上計画
- 「先端設備に投資して効率化したい」 → 先端設備等導入計画
- 「災害対応体制を整備したい」 → 事業継続力強化計画

### 観点④：経営革新計画の承認基準は、定量基準から検証

計算問題が出された場合、必ず以下の手順で進めてください。

**手順**：
1. 営業利益、人件費、減価償却費を抽出
2. 付加価値額を計算：`営業利益 + 人件費 + 減価償却費`
3. 年率成長率を計算：`（最終年度値 - 初年度値）/ 初年度値 / 計画期間年数`
4. 「付加価値額年率3%以上」「経常利益年率1%以上」を満たすか確認

## 確認問題

<Callout title="問題1：認定機関の区別">
    以下の4つの計画について、認定機関を正しく対応させてください。

    1. 経営革新計画 → （A）経済産業大臣 （B）主務大臣 （C）都道府県知事 （D）市区町村
    2. 経営力向上計画 → （A）経済産業大臣 （B）主務大臣 （C）都道府県知事 （D）市区町村
    3. 先端設備等導入計画 → （A）経済産業大臣 （B）主務大臣 （C）都道府県知事 （D）市区町村
    4. 事業継続力強化計画 → （A）経済産業大臣 （B）主務大臣 （C）都道府県知事 （D）市区町村

    **正答**：1-C、2-B、3-D、4-A
</Callout>

<Callout title="問題2：付加価値額の計算">
    ある中小企業が経営革新計画を立案しました。以下の財務データに基づき、付加価値額を計算し、経営革新計画の定量基準を満たすか判定してください。

    **初年度（1年目）**
    - 営業利益：1,000万円
    - 人件費：2,000万円
    - 減価償却費：500万円

    **3年目**
    - 営業利益：1,300万円
    - 人件費：2,100万円
    - 減価償却費：600万円

    付加価値額の成長率は何%でしょうか。経営革新計画の基準（年率3%以上）を満たしますか。

    **計算**：
    1. 初年度：1,000 + 2,000 + 500 = 3,500万円
    2. 3年目：1,300 + 2,100 + 600 = 4,000万円
    3. 成長率：(4,000 - 3,500) / 3,500 / 2年 = 500 / 3,500 / 2 = 約7.14% ÷ 2 = 約3.57%
    4. 判定：3.57% ≥ 3% → **基準を満たす**

    ※正確には複合年率成長率（CAGR）を使うことが推奨されます。
</Callout>

<Callout title="問題3：固定資産税軽減の比較">
    次の2つの計画の固定資産税軽減について、正しい説明を選んでください。

    **問題文**：
    A社が機械装置に1,000万円を投資しました。固定資産税年額が100万円と想定される場合、次のうち正しい説明はどれか。

    （ア）経営力向上計画では、固定資産税は全額免除される
    （イ）先端設備等導入計画では、固定資産税は2分の1軽減される
    （ウ）先端設備等導入計画では、3年間、固定資産税100万円が全額免除される
    （エ）経営力向上計画では、3年間、固定資産税は50万円となる

    **正答**：（ウ）と（エ）が正しい

    - （ウ）：先端設備等導入計画は「全額免除」（ゼロ特例）
    - （エ）：経営力向上計画は「2分の1軽減」→ 100万円 × 0.5 = 50万円
</Callout>
