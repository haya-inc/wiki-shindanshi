## このページの役割

事例Ⅱでは、`誰に / 何を / どのように / どんな効果` を一文で切れずに残すことが重要です。顧客を決めずに施策だけを並べる流れを避け、価値と接点と効果を短くつなぎます。

<Callout title="このページの役割">
  [事例Ⅱの設問型](/docs/second-stage/case-2-question-patterns) で型を見分けたあとに、`誰に、何を、どのように、どんな効果` を短文へ落とすページです。
</Callout>

## 解答を作る順

<Steps>
  <Step>
    ### 対象顧客を決める

    `誰に` を決め、設問の中心顧客を 1 つに絞ります。
  </Step>
  <Step>
    ### 価値と強みを対応させる

    その顧客に刺さる `価値` と `強み` を対応させ、与件根拠を作ります。
  </Step>
  <Step>
    ### 動かす 4P を決める

    `商品 / 価格 / 販路 / 販促` のどこを動かすか決め、接点を具体化します。
  </Step>
  <Step>
    ### 狙う効果を 1 つ置く

    `来店数`、`客単価`、`再来店`、`認知` のどの効果を狙うかを書きます。
  </Step>
</Steps>

## 設問型別の答案骨子

| 設問型 | 骨子の型 | 外せない効果 |
| --- | --- | --- |
| ターゲット設定型 | `〜を対象に、〜という強みを生かして、〜を提案する` | 新規来店、利用機会の拡大 |
| 既存顧客深耕型 | `既存顧客に対し、〜で継続接点を作り、〜を高める` | 再来店、客単価、LTV |
| 新商品・新サービス型 | `〜ニーズに対し、〜を組み合わせた商品 / 体験を提供する` | 差別化、客単価、口コミ |
| 販路・販促型 | `〜で訴求し、〜へ誘導して、〜を増やす` | 認知、来店、購買転換 |
| データ読解型 | `数値上〜が課題のため、〜を対象に〜を行い、〜を改善する` | データで示された弱点の改善 |

## そのまま使いやすい短文の型

<Cards>
  <Card title="対象と強みを置く">
    `〜を対象に、〜という強みを訴求し、〜につなげる`
  </Card>
  <Card title="接点を増やす">
    `〜を強化するため、〜で接点を増やし、〜を高める`
  </Card>
  <Card title="課題から改善へ">
    `〜という課題に対し、〜を行い、〜を改善する`
  </Card>
</Cards>

## 典型例題

| 例題の型 | 骨子の置き方 | 短文の型 |
| --- | --- | --- |
| `新規顧客向けの施策` | ターゲット、価値、接点、効果 | `観光客を対象に、地域食材の体験価値を訴求し、SNS と店頭連動で来店を増やす` |
| `既存顧客の再来店施策` | 顧客接点、継続利用、客単価 | `既存会員向けに来店後フォローと限定提案を行い、再来店と客単価を高める` |
| `新商品提案` | 強み、ニーズ、組み合わせ、差別化 | `既存人気商品と地元素材を組み合わせた新商品を投入し、差別化と口コミを促す` |
| `販路拡大` | 媒体、導線、来店 / EC 転換 | `SNS で使用場面を訴求し、EC へ誘導して新規購入を増やす` |

## 悪文例と改善例

| 想定しやすい設問 | 悪文例 | どこが弱いか | 改善例 |
| --- | --- | --- | --- |
| 価格戦略 | `安い料金プランを出して客数を増やす。` | 誰に向けるか、何を解決するかがない | `平日昼の空き時間を埋めるため、学生競技者向けの時間帯別メニューを設け、利用回数を増やす。` |
| CRM・LTV | `顧客情報を集めてフォローする。` | 蓄積項目と活用方法が曖昧 | `来店目的と競技歴、悩みを蓄積し、再来店時に最適メニューを提案して継続利用を高める。` |
| EC と店舗の併用 | `EC を強化して売上を上げる。` | 店舗との連動がなく、効果が広すぎる | `EC 購入者へ店舗体験クーポンを配信し、店舗来店と再購入をつなげて関係を深める。` |
| 新商品提案 | `若者向け新商品を出してSNSで売る。` | 与件の強みと地域性がない | `地元素材を生かした手土産商品を若年観光客向けに開発し、SNSで利用場面を訴求して購買を促す。` |

## 誤答例と直し方

| 誤答例 | どこが弱いか | 直し方 |
| --- | --- | --- |
| `若者向けに売る` | 強みと価値がない | `若年観光客向けに写真映えする地域限定商品を提案する` と価値を入れる |
| `SNS で告知する` | 内容と導線がない | `SNS で体験価値を訴求し EC へ誘導する` と接点をつなぐ |
| `新商品を販売し売上向上を図る` | 効果が広すぎる | `客単価向上`、`再来店増加` など 1 つに絞る |
| `データを見て高齢者向け施策を行う` | 数値の変化がない | `高齢者比率が上昇しているため` などの根拠を入れる |

## 年度別小問例

| 年度 | 小問の置き方 | 短文で外せないこと |
| --- | --- | --- |
| `R3` | 地域資源を使った新規顧客施策 | `対象顧客` と `地域資源の価値` を並べる |
| `R4` | 商品提案と差別化 | `強み` と `ニーズ` の対応を 1 行で書く |
| `R5` | 既存顧客の関係強化 | `継続接点` と `再来店 / 客単価` をつなぐ |
| `R6` | データ読解にもとづく施策 | 数値根拠と対象顧客を冒頭に置く |
| `R7` | オンライン連動と価格戦略 | `媒体`、`価値`、`効果` を 1 本で通す |

## 年度差の見方

| 時期 | 書き方で重くなる点 | 注意すること |
| --- | --- | --- |
| `R3〜R5` | 地域資源、協業、SNS を強みとつなぐ骨子 | `誰に` を曖昧にしない |
| `R6〜R7` | ブランド価値、LTV、オンライン連動の短文化 | `どのように` と `効果` を短く因果でつなぐ |

## 効果を絞るときの見方

| 迷う効果 | 先に見る条件 | 書き分けの例 |
| --- | --- | --- |
| 来店数か客単価か | 新規集客か既存深耕か | 新規なら `来店増`、既存なら `客単価 / 再来店` |
| 認知か購買転換か | 媒体が上流か下流か | SNS は `認知`、EC 導線は `購買転換` |
| 口コミかLTVか | 単発利用か継続利用か | 体験型は `口コミ`、会員化は `LTV` |

## 失点を避ける確認

- `誰に` が具体的か
- `何を` と `どのように` が分かれているか
- 与件の強みや地域資源を 1 つ以上使っているか
- `効果` が売上以外も含めて具体化できているか
