## このページの役割

事例Ⅳでは、`どの順で処理し / どこで切り / 何を答案に残すか` を先に決めます。完答狙いで止まるより、経営分析を先に確保し、途中式と部分点を残す進め方を固定します。

<Callout title="このページの役割">
  [事例Ⅳの設問型](/docs/second-stage/case-4-question-patterns) で論点を見分けたあとに、`どの順で処理し、どこで切り、何を答案に残すか` を整理するページです。
</Callout>

## 先に決める得点計画

<Cards>
  <Card title="取りやすい問題から先に取る">
    `取りやすい問題` から先に処理し、難問で流れを止めません。
  </Card>
  <Card title="経営分析を先に確保する">
    経営分析は短時間で確保し、難問は完答より部分点を優先します。
  </Card>
  <Card title="途中式を必ず残す">
    途中式、単位、端数処理を残して見直し可能な形で進めます。
  </Card>
</Cards>

## 設問型別の答案骨子

| 設問型 | 解く順 | 外せない確認 |
| --- | --- | --- |
| 経営分析型 | 指標候補を 3 分類で置く → 与件根拠を対応させる → 指標名、数値、改善方向を書く | `収益性 / 安全性 / 効率性` のバランス |
| CVP・利益計画型 | 求める量を確認する → 固変分解 → 式へ代入 → 制約条件で見直す | 単位、限界利益率、セールスミックス |
| CF・運転資本型 | 現金増減の方向を置く → 営業 / 投資 / 財務へ分ける → 税金と回収を入れる | 符号、初年度、最終年度、回収 |
| NPV・意思決定型 | 各年 CF を並べる → 現在価値へ割り引く → 残存価値 / 撤退価値を入れる → 比較する | 割引率、年次、追加投資、期待値 |
| 記述型 | 財務課題 → 原因 → 改善策 → 効果の順で短く書く | 数値で見えた課題と助言がつながるか |

## 時間を切る目安

| 場面 | 目安 | 意識すること |
| --- | --- | --- |
| 第1問 経営分析 | `15分前後` | 指標選定に迷いすぎない |
| 標準的な計算問題 | `15〜20分` | 途中式を残し、詰まったら次へ進む |
| 難しい投資意思決定 | `20分以内でいったん区切る` | 完答より部分点を優先する |
| 最終見直し | `5分前後` | 符号、単位、転記ミス、未記入を確認する |

## 記述で使いやすい短文の型

<Cards>
  <Card title="低下指標から改善へ">
    `〜が低下しているため、〜を見直し、〜を改善する`
  </Card>
  <Card title="過大要因から削減へ">
    `〜が過大であるため、〜を削減し、〜を高める`
  </Card>
  <Card title="圧迫要因から接続する">
    `〜により〜が圧迫されているため、〜を行い、〜につなげる`
  </Card>
</Cards>

## 失点を避ける確認

- `何を求める問題か` を式の前に言えるか
- 税率、減価償却、運転資本、残存価値を落としていないか
- 部分点狙いで途中式を残しているか
- 記述で `課題 → 改善策 → 効果` の因果が切れていないか
