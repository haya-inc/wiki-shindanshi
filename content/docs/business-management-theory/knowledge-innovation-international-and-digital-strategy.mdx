<Callout title="このページの役割">

## 学習のポイント

- `技術経営` は強みの作り方
- `国際経営` は市場の広げ方と進出形態
- `デジタル戦略` は価値提供の仕組みの変え方

## 試験で何が問われるか

- イノベーションの種類（持続的/破壊的、プロダクト/プロセス）を区別できるか
- 海外進出形態のリスク・統制・投資のトレードオフを説明できるか
- エフェクチュエーション、両利きの経営、リーンスタートアップを説明できるか

## イノベーションの分類

### 技術変化の方向性

| 分類軸 | 類型 | 内容 |
| --- | --- | --- |
| 変化の対象 | プロダクトイノベーション | 新製品・新サービスの創出 |
| 変化の対象 | プロセスイノベーション | 生産・業務プロセスの改善 |
| 変化の度合い | インクリメンタル（漸進的） | 既存技術の改良 |
| 変化の度合い | ラディカル（急進的） | 技術的に非連続な変化 |

### 持続的イノベーション vs 破壊的イノベーション（クリステンセン）

| | 持続的イノベーション | 破壊的イノベーション |
| --- | --- | --- |
| 方向性 | 既存顧客の高度なニーズに応える | ローエンドまたは新市場から参入 |
| 既存企業 | 得意（既存能力を活用） | 苦手（既存の価値基準と合わない） |
| 顧客 | 既存の優良顧客 | 非消費者やローエンド顧客 |
| リスク | 過剰品質（オーバーシュート） | 当初は性能不足に見える |

**イノベーションのジレンマ**：既存の優良顧客の声に従い持続的イノベーションに注力する優良企業が、破壊的イノベーションに対応できずに敗退する現象。

### 技術の普及（ロジャーズ）

| カテゴリ | 割合 | 特徴 |
| --- | --- | --- |
| イノベーター | 2.5% | 冒険的、最初に採用 |
| アーリーアダプター | 13.5% | **オピニオンリーダー** |
| アーリーマジョリティ | 34% | 慎重、周囲を見て追随 |
| レイトマジョリティ | 34% | 懐疑的 |
| ラガード | 16% | 最も保守的 |

**キャズム**（ムーア）：アーリーアダプターとアーリーマジョリティの間の溝。「新しさ」を求める層と「実績・安心」を求める層ではアプローチを変える必要があります。

## オープンイノベーション（チェスブロウ）

| | クローズドイノベーション | オープンイノベーション |
| --- | --- | --- |
| 知識の源泉 | 自社内のR&D | 自社＋外部（大学、スタートアップ等） |
| 考え方 | 最高の人材を社内に抱える | 社外の優秀な人材も活用する |
| 知的財産 | 守るもの | 活用するもの（ライセンスアウトも含む） |
| リスク | 開発コスト大、時間がかかる | ノウハウ流出、統合の難しさ |

CVC（コーポレート・ベンチャー・キャピタル）は、事業会社がスタートアップに出資して新技術にアクセスするオープンイノベーションの手法です。

## エフェクチュエーション（サラスバシー）

起業家の意思決定論理。コーゼーション（因果論理：目標→手段）とは逆に、**手持ちの手段から出発** して何ができるかを考えます。

| 原則 | 内容 |
| --- | --- |
| 手中の鳥 | 自分が持つ資源（知識、人脈、能力）から始める |
| 許容可能な損失 | 最大利益ではなく、許容できる損失の範囲で行動 |
| クレイジーキルト | パートナーシップで不確実性を共有 |
| レモネード | 予期せぬ事態を機会に転換 |
| 飛行機のパイロット | 予測より統制を重視 |

## 両利きの経営（オライリー＆タッシュマン）

既存事業の深化（exploitation）と新規事業の探索（exploration）を同時に行う経営のあり方です。

| | 知の深化 | 知の探索 |
| --- | --- | --- |
| 目的 | 既存の能力・知識の改善・活用 | 新しい知識・能力の探索 |
| 特徴 | 効率化、改善、確実性 | 実験、リスクテイク、不確実性 |
| 陥りやすい罠 | 能力の罠（成功体験への固執） | 失敗の罠（探索コストの浪費） |

## 国際経営 ── 海外進出形態

| 形態 | 統制 | リスク | 投資額 | スピード |
| --- | --- | --- | --- | --- |
| 間接輸出 | 低 | 低 | 小 | 速い |
| 直接輸出 | やや低 | やや低 | 小〜中 | 速い |
| ライセンシング | 低 | 低 | 小 | 速い |
| フランチャイズ | 中 | 中 | 中 | 中 |
| 合弁（JV） | 中 | 中 | 中 | 中 |
| 完全所有子会社 | **高** | **高** | **大** | 遅い |

統制とリスク・投資は **トレードオフ** の関係です。

### 国際戦略の類型（バートレット＆ゴシャール）

| 戦略 | グローバル統合 | ローカル適応 |
| --- | --- | --- |
| グローバル戦略 | 高 | 低 |
| マルチドメスティック戦略 | 低 | 高 |
| トランスナショナル戦略 | 高 | 高 |
| インターナショナル戦略 | 低 | 低 |

## プラットフォーム戦略とDX

### プラットフォーム戦略

複数の利用者グループをつなぎ、**ネットワーク効果** で価値を増大させる戦略です。

| 概念 | 内容 |
| --- | --- |
| 直接ネットワーク効果 | 同じグループの利用者が増えるほど価値が上がる |
| 間接ネットワーク効果 | 片方のグループが増えるともう片方の価値が上がる |
| マルチホーミング | 利用者が複数のプラットフォームを同時利用 |
| WTA（勝者総取り） | ネットワーク効果で1社が市場を独占する傾向 |

### DX（デジタルトランスフォーメーション）

DX は単なる IT 化・デジタル化ではなく、デジタル技術を用いて **ビジネスモデルや組織文化そのもの** を変革する取り組みです。

---

## 典型的なつまずき

- `技術開発` と `イノベーション` を同じものとして扱う → 技術開発は **手段**、イノベーションは **価値創造** の結果
- 破壊的イノベーションを「すごい技術」と理解する → ローエンドや非消費者から始まる **市場構造の変化**
- `輸出` と `現地子会社` の統制差を説明できない → 統制とリスク・投資の **トレードオフ**
- `DX` を `システム導入` だけの話だと理解する → ビジネスモデルと組織文化の **変革** まで含む
- エフェクチュエーションを「計画なし」と理解する → 手持ちの手段から出発する **別の論理**

## 問題を解くときの観点

- 問われているのは `技術で強みを作る話` か `市場を広げる話` か `仕組みを変える話` か
- 海外進出なら、重いのは `統制` か `投資負担` か `スピード` か
- イノベーションなら、`既存顧客向け改善` か `新市場創造` か

## 確認問題

<Callout title="問1：破壊的イノベーション">
    「既存製品より性能は劣るが、低価格で簡便な製品が新たな顧客層を開拓し、やがて主流市場を奪う」現象を何というか。

    **解答**：破壊的イノベーション（クリステンセン）。ローエンドまたは新市場から参入し、既存企業の対応が遅れる間に成長する。
</Callout>

<Callout title="問2：海外進出形態">
    「統制を最大限に確保しつつ海外市場に参入したい」場合、最も適した進出形態はどれか。

    **解答**：完全所有子会社（100%出資の現地法人）。統制は最も高いが、投資額とリスクも最大。
</Callout>

<Callout title="問3：両利きの経営">
    「既存事業の効率改善に注力するあまり、新規事業の探索が停滞している」状態を何の罠というか。

    **解答**：能力の罠（competency trap）。知の深化に偏り、知の探索がおろそかになる状態。両利きの経営では深化と探索のバランスが重要。
</Callout>
