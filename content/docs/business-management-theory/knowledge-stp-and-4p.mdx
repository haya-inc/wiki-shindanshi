## このページの役割

<Callout title="このページの役割">
このページは、マーケティング戦略の基本的な意思決定プロセス「STP」と、実行手段である「マーケティングミックス（4P）」の関係を整理するページです。`市場をどう分ける`（セグメンテーション）、`どこを狙う`（ターゲティング）、`どう位置づける`（ポジショニング）、そして`何をどう売るか`（4P）まで、戦略から実行へつなぐ一連の流れを押さえます。
</Callout>

<Callout title="このページの読み方">
マーケティング戦略は「3C分析→STP→4P」の階層で組み立てます。まず顧客、競合、自社を分析し、市場を分類（セグメント）してから、どのセグメントを狙うか（ターゲット）を決め、そのセグメントの中で自社をどう見せるか（ポジショニング）を定めてから、最後に製品、価格、流通、プロモーションの4Pを具体化します。この流れが営業現場や新商品開発の論理の基盤です。
</Callout>

## 学習のポイント

- マーケティングの「全体の流れ」を最初に掴む：3C → STP → 4P → 実行・管理
- STPの3つの段階（セグメント化 → ターゲット選択 → ポジショニング）を分ける
- セグメンテーション変数は「4種類の軸」として体系的に理解する
- ターゲティングの「3パターン（または4パターン）」の使い分けを整理する
- 4Pは「売り手視点」、4Cは「買い手視点」として対応関係を抑える
- ポジショニングマップで競合との相対位置を図的に把握する

## 試験で何が問われるか

- マーケティングプロセス全体（R-STP-MM-I-C）の流れを説明できるか
- 3C分析（Customer、Competitor、Company）の役割を理解しているか
- セグメンテーション変数を「4種類」に分類できるか
- 有効なセグメントの「5つの要件」を列挙できるか
- ターゲティングのパターン（無差別型、差別型、集中型、ワントゥワン）を区別できるか
- ポジショニングを「競合との相対位置」として図示・説明できるか
- 4Pと4Cの対応関係を理解できるか
- トランザクション・マーケティングとリレーションシップ・マーケティングの違いを説明できるか

## マーケティングの基本プロセス

マーケティング戦略は、一連のプロセスに従って展開されます。試験では「全体の流れ」が問われるため、各段階の目的と順序を明確に押さえることが重要です。

### R-STP-MM-I-C（マーケティングプロセス全体）

```
リサーチ（Research）
  ↓
STP：セグメンテーション→ターゲティング→ポジショニング
  ↓
マーケティングミックス（Marketing Mix）：4P
  ↓
実行（Implementation）
  ↓
管理・評価（Control & Evaluation）
```

各段階の役割：

| 段階 | 役割 | 実施内容 |
| --- | --- | --- |
| **リサーチ** | 現状把握・情報収集 | 市場規模、顧客ニーズ、競合状況、自社資源の把握 |
| **STP** | 戦略的ポジショニング | 市場分割→標的市場選定→自社位置づけ |
| **マーケティングミックス** | 実行手段の決定 | 製品、価格、流通、プロモーションの具体化 |
| **実行** | 戦略の現場展開 | 販売、営業、顧客サービスの実施 |
| **管理・評価** | 成果測定・改善 | 売上、顧客満足度、シェアの評価と改善 |

### 3C分析（マーケティングの環境分析）

STPを決める前に、企業を取り巻く3つの環境を分析します。これが戦略立案の基礎データです。

| C | 分析対象 | 主な問い | 内容 |
| --- | --- | --- | --- |
| **Customer** | 顧客・市場 | 顧客は誰か。どのようなニーズがあるか。市場規模と成長性は。 | 市場規模、顧客セグメント、ニーズの多様性、購買行動、市場トレンド |
| **Competitor** | 競合企業 | 主要競合は誰か。競合の強み・弱みは何か。 | 競合企業の製品・価格・戦略、市場シェア、差別化要因、競争力 |
| **Company** | 自社 | 自社の強み・弱みは何か。利用可能なリソースは。 | 製品ラインアップ、技術力、ブランド、流通網、財務状況、人的資源 |

**3C分析の活用**：STPを決める際には、3Cの結果から「顧客のニーズ×競合との差別化×自社の実行能力」の交点を見つけることが重要です。

## セグメンテーション（市場細分化）

セグメンテーションは、多様な顧客層を持つ市場を、共通の特性を持つグループに分割するプロセスです。

### セグメンテーション変数（4種類の分類軸）

市場を分割するために使う軸は、大きく4種類に分類されます。

| 変数タイプ | 軸 | 具体例 | 使用例 |
| --- | --- | --- | --- |
| **地理的変数（Geographic）** | 地域、気候、都市規模 | 日本、北米、欧州；東京、札幌；都市部、地方 | 飲料メーカーが地域別の味好好みに対応；コンビニ出店戦略 |
| **人口統計的変数（Demographic）** | 年齢、性別、所得、家族構成、職業、学歴 | 20代女性、高所得世帯、子育て中の親；会社員、自営業 | 化粧品の年代別ブランド展開；車の顧客層の所得別区分 |
| **心理的変数（Psychographic）** | ライフスタイル、価値観、性格、購買態度 | 健康志向、環境配慮、高級志向、時間短縮志向 | オーガニック食品の価値観共有層；プレミアム商品の顧客層 |
| **行動変数（Behavioral）** | 購買パターン、利用頻度、ブランド忠誠度、購買量 | ヘビーユーザー、初回購買層、常連客、離反層 | ポイントカード活用度による顧客分類；リピーター向けキャンペーン |

**重要**：これら4つの軸は、企業の戦略に応じて組み合わせて使われます。例えば「30代女性（人口統計）で、健康志向（心理的）、週3回以上の利用頻度（行動）」といった複合的なセグメントが実務では多く見られます。

### 有効なセグメントの5つの要件

市場を分割しただけでは不十分です。そのセグメントが実際に経営戦略の対象になるためには、次の5つの要件を満たす必要があります。

| 要件 | 内容 | 考慮ポイント |
| --- | --- | --- |
| **測定可能性（Measurable）** | セグメントの規模、購買力などが定量的に把握できるか | 市場調査で人数・金額を数値化できるか |
| **接近可能性（Accessible）** | セグメントに対して、効率的に製品やプロモーションを届けられるか | 広告媒体、販売チャネルでそのセグメントに到達できるか |
| **実質性（Substantial）** | セグメント自体に一定の規模・収益性があるか | 儲かるだけの規模と成長性があるか |
| **差別化可能性（Differentiable）** | セグメント内は同一ニーズだが、セグメント間は異なるニーズがあるか | 工場を変えたり、プロモーションを変えたり、異なる戦略で対応する価値があるか |
| **実行可能性（Actionable）** | 自社の経営資源で実際にそのセグメントに対応できるか | 製造、販売、物流の対応能力があるか |

**典型的な誤り**：「1時間に3回以上コーヒーを飲む人」というセグメントは測定困難（接近可能性が低い）；「月1回だけ使うシーズナル顧客」は実質性が低い、という具合に、要件を欠けば実行は難しくなります。

## ターゲティング（標的市場の選定）

セグメント化した複数の市場から、自社が狙う「標的市場（ターゲットセグメント）」を選定するプロセスです。

### ターゲティングのパターン（3～4パターン）

企業は、セグメント化された市場に対して、異なるアプローチを取ります。

#### 1. 無差別型マーケティング（Undifferentiated Marketing / Mass Marketing）

| 項目 | 内容 |
| --- | --- |
| **対象** | 全市場を1つとして見なし、すべての顧客に同じ製品・サービスを提供 |
| **戦略** | セグメンテーションを活用せず、市場全体に向けた単一の戦略 |
| **コスト** | 製造、流通、プロモーション コストが低い |
| **リスク** | 競合との差別化がしにくい；顧客ニーズの多様性に対応できない |
| **適する企業** | 大量生産・低価格化で競争力を持つ企業；商品が標準化されている産業（食塩、ガソリンなど） |
| **例** | 電力会社の料金設定（かつては地域一律）；ガソリンスタンド |

#### 2. 差別型マーケティング（Differentiated Marketing）

| 項目 | 内容 |
| --- | --- |
| **対象** | 複数のセグメントそれぞれに対して、異なる製品やサービスを提供 |
| **戦略** | セグメント間のニーズの違いを認識し、各セグメント向けに独立した戦略を展開 |
| **コスト** | セグメント数分の製品開発、プロモーション、流通の負担が増加 |
| **リスク** | 複数セグメント対応の複雑性；競合進出で各セグメントが侵食されるリスク |
| **適する企業** | 大企業で経営資源が充分にある企業；多品種製造能力や複数ブランドを持つ企業 |
| **例** | 自動車メーカーの価格帯別ブランド展開（トヨタなら、プリウス・カローラ・ランドクルーザー）；日用品メーカーの複数ブランド展開（P&G、花王など）；銀行の個人向け・法人向けサービスの分化 |

#### 3. 集中型マーケティング（Concentrated Marketing / Niche Marketing）

| 項目 | 内容 |
| --- | --- |
| **対象** | 複数セグメントのうち、1つまたは少数のセグメントに経営資源を集中 |
| **戦略** | 特定セグメントの深いニーズ理解と専門化を武器に、そのセグメント内での市場シェアを高める |
| **コスト** | 限定されたセグメント対応のため、相対的には低コスト |
| **リスク** | そのセグメント市場の成長停滞や衰退に脆弱；競合の参入で影響が大きい |
| **適する企業** | 中小企業、スタートアップ；特定分野に専門性を持つ企業 |
| **例** | 高級スポーツカーメーカー（フェラーリ）；有機農業の食品メーカー；B2B向け専門機械メーカー；小型漁船向けエンジン製造企業 |

#### 4. マイクロマーケティング（Micromarketing / One-to-One Marketing）

| 項目 | 内容 |
| --- | --- |
| **対象** | 顧客を個別（または超小規模グループ）単位で認識し、カスタマイズされた製品・サービスを提供 |
| **戦略** | 顧客データと分析を活用した個別対応；デジタル技術による個別ニーズ把握と提案 |
| **コスト** | 個別対応の手間とシステムコストが高い；DX（データ・テクノロジー）への投資が必要 |
| **リスク** | スケーラビリティに課題；顧客データの管理とプライバシー保護の負担 |
| **適する企業** | 高単価商品（保険、不動産、金融）；オンライン事業者（ビッグデータを活用できる企業） |
| **例** | 保険代理店の顧客別ライフプラン提案；ECサイトのレコメンデーション（あなたへのおすすめ）；プライベートバンキング |

### ターゲティングのポイント

| 観点 | 説明 |
| --- | --- |
| **市場の構造と成長性** | 選んだセグメントが成長市場か成熟市場か。長期的な展望があるか。 |
| **競合状況** | 既存競合の寡占度。新規進出の難易度。差別化の可能性。 |
| **自社の強み** | 選んだセグメントの顧客ニーズに対して、自社は本当に対応できるか。 |
| **顧客の購買力** | セグメントの顧客が十分な購買力を持つか。価格感度はどうか。 |

## ポジショニング（市場での自社の位置づけ）

ターゲットセグメントを選んだ後、そのセグメント内で「顧客の頭の中での自社の位置づけ」を決めるプロセスです。

### ポジショニングの本質

**ポジショニング** = 「競合と比べて、自社がどこに位置するか」という顧客認識

同じセグメント内でも、複数の競合が存在します。各企業は、顧客に「自社は競合と何が違うのか」を認識させることで、選ばれやすくします。

例）スマートフォン市場の「高級スマートフォン」セグメント内：
- Apple iPhone：ブランド価値、デザイン、統合体験の追求
- Samsung Galaxy：テクノロジー、大画面、多機能性
- Google Pixel：カメラ性能、AI機能

同じセグメント内でも、ポジショニングが異なる。

### ポジショニングマップ

ポジショニングを図示するツールが「ポジショニングマップ」です。2つの軸（属性や価値）を設定し、その交点に自社と競合を配置します。

```
例：スニーカー市場（20～30代、日常用）
     ↑
高価格   |
    │   ASICS（スポーツ機能×高品質）
    │         ・Nike（デザイン×パフォーマンス）
    │
    │ Adidas（ファッション×スポーツ性）
    ├──────────────────────→
    │  Uniqlo（シンプル×手頃価格）
    │
低価格 │  DAISO（質実剛健×超低価格）
    │
```

軸の設定例：
- 「価格（低～高）× 品質（低～高）」
- 「機能性（低～高）× ファッション性（低～高）」
- 「地元性（地域密着～全国展開）× 新規性（伝統的～最新）」

### 差別化の方法

ポジショニングを実現するために、企業は複数の手段で差別化します。

| 差別化の種類 | 内容 | 具体例 |
| --- | --- | --- |
| **製品差別化** | 機能、品質、デザイン、パッケージの独自性 | サムスンのスマートフォンの大画面、テスラの電動車技術 |
| **サービス差別化** | 顧客サービス、配送、アフターサービスの充実 | Amazonの翌日配送、Appleの修理保証 |
| **チャネル差別化** | 流通経路の独自性、アクセスの利便性 | セブン-イレブンの深夜営業、オンライン専売 |
| **イメージ差別化（ブランド）** | 広告、ブランドイメージ、顧客との関係性 | 高級ブランド（LVMH）の希少性と憧れ、Nikeの「Just Do It」 |

## マーケティングミックス（4P）

STPで戦略的位置づけを決めた後、実際に顧客に提供する「製品、価格、流通、プロモーション」の4要素を決定・調整するプロセスです。

### 4Pの定義と4Cとの対応

マーケティングミックスの古典的なフレームワークが「4P」ですが、顧客視点では「4C」に置き換えられます。

| 4P（売り手視点） | 4C（買い手視点） | 対応関係 | 内容 |
| --- | --- | --- | --- |
| **Product（製品）** | **Customer Value（顧客価値）** | 何を提供するか | 製品の機能、品質、デザイン、ブランド、サービス |
| **Price（価格）** | **Cost（顧客コスト）** | いくらで提供するか | 価格設定、支払い条件、ローン、割引など |
| **Place（流通）** | **Convenience（利便性）** | どこで届けるか | 販売チャネル（直販、小売店、オンライン）、配送、品揃え |
| **Promotion（プロモーション）** | **Communication（コミュニケーション）** | どう伝えるか | 広告、販売促進、PR、営業活動、顧客との対話 |

**重要な視点**：
- **4Pは売り手視点**：企業が「何をするか」を決める論理
- **4Cは買い手視点**：顧客が「何を得られるか」「どんな手間がかかるか」を考える視点
- 良いマーケティングは「4Pの決定が、顧客の4Cのニーズを満たしているか」を常に確認すること

### 各Pの詳細

#### Product（製品）戦略

製品とは、顧客に「有形の製品」だけでなく「サービス」「ブランド価値」も含みます。

| 要素 | 内容 |
| --- | --- |
| **コア製品** | 顧客が本来求める基本機能・利益 |
| **有形製品** | パッケージ、デザイン、ブランド名、品質水準 |
| **拡張製品** | アフターサービス、保証、返品対応、配送など |

**製品ライフサイクル**：新製品導入→成長→成熟→衰退の各段階で、製品戦略は変わります。

#### Price（価格）戦略

価格は、製品の価値を金銭化する重要な決定です。

| 価格戦略 | 説明 | 使用場面 |
| --- | --- | --- |
| **スキミング価格戦略** | 高価格で導入し、段階的に引き下げる | 新製品、高級品、技術革新製品 |
| **ペネトレーション価格戦略** | 低価格で市場シェアを獲得し、後に引き上げる | 市場浸透を狙う、競合との価格競争 |
| **心理的価格設定** | 9,999円など「端数」で割安感を演出 | 小売商品、日用品 |
| **バンドル価格設定** | 複数製品をセットで割引 | テレビ通信のセット割、ファミリープラン |

#### Place（流通）戦略

どのチャネルで、どのような品揃えで顧客に届けるかを決定します。

| 流通チャネル | 特徴 | 例 |
| --- | --- | --- |
| **直販** | 製造企業が直接顧客に販売 | メーカー直営店、テスラ直販モデル |
| **小売販売** | 小売店を通じた販売 | スーパー、百貨店、コンビニ |
| **卸売販売** | 卸売業者を通じた販売（B2B） | 建設資材、食品卸売 |
| **オンライン販売** | インターネットを通じた直販 | Amazon、楽天、メーカーECサイト |
| **多チャネル戦略** | 複数チャネルの並行運営 | Appleの直営店とキャリア販売の併存 |

#### Promotion（プロモーション）戦略

顧客に製品の存在や価値を伝え、購買意欲を喚起する活動です。

| 促進手段 | 説明 | 効果 |
| --- | --- | --- |
| **広告（Advertising）** | マスメディアを通じた一方向的な情報発信 | 大量の認知獲得、ブランド構築 |
| **販売促進（Sales Promotion）** | 限定セール、割引、キャンペーン、ポイント | 短期的な購買喚起 |
| **PR（Public Relations）** | メディアを通じた信頼性の高い情報発信 | ブランドイメージの向上、認知度向上 |
| **営業（Personal Selling）** | 営業担当者による顧客との対面営業 | 信頼構築、カスタマイズ対応（B2B中心） |
| **デジタルマーケティング** | SNS、メール、SEOなどを通じた双方向コミュニケーション | ターゲッティング精度、エンゲージメント |

## 関係性マーケティング（リレーションシップ・マーケティング）

従来の「トランザクション・マーケティング」と対比させて、現代的なマーケティングの考え方を理解します。

### トランザクション・マーケティング vs リレーションシップ・マーケティング

| 観点 | トランザクション・マーケティング | リレーションシップ・マーケティング |
| --- | --- | --- |
| **重点** | 1回の購買取引の成立 | 顧客との長期的な関係構築 |
| **時間軸** | 短期（購買時点） | 長期（購買前～購買後～継続） |
| **顧客との関係** | 一過性の取引相手 | パートナーシップ、継続的な関係 |
| **利益源** | 新規顧客獲得による売上増加 | 既存顧客の維持とLTV（生涯顧客価値）向上 |
| **主な施策** | 広告、販売促進、キャンペーン | ロイヤルティプログラム、カスタマーサービス、顧客データ活用 |
| **顧客コミュニケーション** | 一方向的（企業→顧客） | 双方向的（企業↔顧客） |

**重要な数字**：

| 法則 | 内容 | 意味 |
| --- | --- | --- |
| **1:5の法則** | 新規顧客を1人獲得するコストは、既存顧客を1人保持するコストの5倍 | 既存顧客への投資が効率的 |
| **5:25の法則** | 顧客離反を5%減らすと、利益は25%増加する | わずかな顧客保持の改善が大きな利益向上につながる |

---

## 典型的なつまずき

- **STPと4Pの関係を混同する** → STPは「戦略的位置づけ」、4Pは「実行手段」。STPが決まって初めて4Pが意味を持つ
- **セグメンテーション変数を「4種類」に分類できない** → 地理的、人口統計的、心理的、行動的の4つを必ず暗記すること
- **「年齢」を「心理的変数」に入れてしまう** → 年齢は人口統計的変数。心理的変数はライフスタイルや価値観
- **有効なセグメントの5要件を全部覚えていない** → 測定可能性、接近可能性、実質性、差別化可能性、実行可能性を確実に押さえる
- **ターゲティングのパターンが「3つ」か「4つ」か曖昧** → 無差別型、差別型、集中型の「3パターン」が基本。ワントゥワンは差別型の極端な形または拡張概念として扱う
- **ポジショニングを「製品の特性」だけで考える** → ポジショニングは「顧客の頭の中での位置」。図示（ポジショニングマップ）が有効
- **4Pと4Cの対応関係を覚えられない** → Product↔Customer Value、Price↔Cost、Place↔Convenience、Promotion↔Communication。売り手視点↔買い手視点と覚える
- **無差別型マーケティングを「昔の戦略」と見なす** → 今でも標準化商品や大量消費商品では有効。市場特性に応じた選択肢
- **リレーションシップ・マーケティングを「CRM」と同じと見なす** → CRMはシステム、リレーションシップ・マーケティングは経営思想。異なる概念だが関連している

---

## 問題を解くときの観点

- **STPを「戦略の前提」として読み込む** → その企業はどのセグメントを狙ったのか、どのポジショニングで競争しようとしているのかが必ず背景にある
- **セグメンテーション変数の「軸の選択」が適切か検討する** → 「なぜその軸で分割しているのか」が経営戦略と一貫しているか
- **ターゲットセグメントの「5要件」を満たすか確認する** → 測定不可、接近不可、規模小のセグメントは実行不可
- **ポジショニングと4Pが一貫しているか見る** → 「プレミアム高級品」ポジショニングなのに激安価格という矛盾はないか
- **3C分析の結果とSTPの関連を読み取る** → Customer（市場分析）の結果がセグメンテーション変数を決め、Competitor（競合分析）の結果がポジショニングを決める
- **実行段階での「制約」を考慮する** → ポジショニングが素晴らしくても、自社に実行能力がなければ意味がない（実行可能性）
- **トランザクション・マーケティング vs リレーションシップ・マーケティングの使い分けを判断する** → B2Bで関係性重視、B2Cで取引重視、という単純化は危険。市場特性に応じた選択

## 確認問題

<Callout title="問1：セグメンテーション変数の分類">
「自動車メーカーが、50代以上の高所得層を狙った新型SUVを開発する」場合、年齢と所得はそれぞれどのセグメンテーション変数に分類されるか。また、「都市部の居住者」はどの変数か。

**解答**：年齢と所得は**人口統計的変数**。都市部の居住者は**地理的変数**。セグメンテーション変数の4種類（地理的、人口統計的、心理的、行動的）を確実に分類できることが重要。
</Callout>

<Callout title="問2：ターゲティング戦略の選択">
大手飲料メーカーと地域の小規模飲料メーカーがいたとき、それぞれ最適なターゲティング戦略は何か。また、それぞれのメリット・デメリットを説明しなさい。

**解答**：
- **大手メーカー**：差別型マーケティング。複数セグメント（スポーツドリンク、エナジードリンク、ウォーター等）に対応する経営資源がある。メリット：複数セグメントで売上拡大。デメリット：複雑性とコスト。
- **小規模メーカー**：集中型マーケティング。1～2セグメント（例：有機ジュース、地元産果汁100%）に集中。メリット：深い顧客理解、対応可能。デメリット：市場縮小や競合進出に脆弱。
</Callout>

<Callout title="問3：ポジショニングと4Pの一貫性">
スマートフォン市場で、AppleのiPhoneは「高価格×高ブランド価値」、GoogleのPixelは「中価格×高カメラ性能」にポジショニングされている。この違いが4P（製品、価格、流通、プロモーション）にどのように反映されるか、各々1つずつ説明しなさい。

**解答**例：
- **Product（製品）**：iPhoneは統合的ユーザー体験とデザイン、Pixelはカメラ性能とAI機能が強調される
- **Price（価格）**：iPhoneは高価格維持、Pixelは中価格設定
- **Place（流通）**：iPhoneはキャリアと直営店、PixelはGoogle Store中心
- **Promotion（プロモーション）**：iPhoneはブランド価値を訴求、Pixelはカメラ・AI機能を技術的に訴求
</Callout>
