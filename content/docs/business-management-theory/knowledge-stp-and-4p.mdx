## このページの役割

<Callout title="このページの役割">
マーケティング戦略は「すべての顧客に同じもの」では成り立ちません。このページは、マーケティングの中核的な意思決定フロー「STP（セグメンテーション→ターゲティング→ポジショニング）」と、その後の実行手段「マーケティングミックス（4P）」をセットで理解するページです。順序と論理が重要です。
</Callout>

## 重要な流れ

マーケティング戦略の基本は、次の順序で組み立てられます。

```
3C分析（市場・競合・自社の分析）
  ↓
STP（市場の分割→狙う層→自社の位置づけ）
  ↓
4P（製品、価格、流通、プロモーション）
  ↓
実行・評価
```

**なぜこの順序か**：市場を分割せず、対象層を決めずに、いきなり「製品を安くしよう」「広告を打とう」と決めても、誰に何が刺さるのか分かりません。戦略の上流（STP）を決めてから、実行の手段（4P）を決めるのです。

---

## STPとは何か（定義と役割）

STPは、マーケティング戦略の「判断の段階」です。3つの意思決定をこの順に行います。

| 段階 | 日本語 | 英語 | 意思決定の内容 | 答え |
| --- | --- | --- | --- | --- |
| **第1段階** | セグメンテーション | Segmentation | 「市場をどう分ける？」 | 顧客グループの区分軸を決める |
| **第2段階** | ターゲティング | Targeting | 「どのグループを狙う？」 | 狙う層（セグメント）を選ぶ |
| **第3段階** | ポジショニング | Positioning | 「その層の中で自社はどう見える？」 | 競合との相対的な位置づけを決める |

この3つが決まらなければ、「何をいくらで、どこで、どう売るか」（4P）も決められません。

### STPを決める前段階：3C分析

STPを決める「前に」、企業を取り巻く外部環境と内部事情を理解しておく必要があります。これが3C分析です。

| C | 対象 | 問い | 具体的な情報 |
| --- | --- | --- | --- |
| **Customer** | 市場・顧客 | 顧客は誰か。何を求めているか。市場はどのような構造か。 | 市場規模、成長率、顧客属性の多様性、ニーズの違い、購買パターン |
| **Competitor** | 競合企業 | 競合は誰か。強い競合、弱い競合は。 | 各競合の製品・価格、市場シェア、得意分野、差別化ポイント |
| **Company** | 自社 | 自社は何ができるか。競合との差別化要因は何か。 | 製品ラインアップ、技術力、ブランド認知、流通ネットワーク、財務力 |

3C分析の結果が、セグメンテーション変数の選択やターゲット選定に直結します。例えば、Customerの分析で「年代別にニーズが大きく異なる」と分かれば、セグメンテーション変数として「年齢」を重視することになります。

## セグメンテーション（市場の分割）

### セグメンテーションの本質

市場のすべての顧客が、同じニーズを持っているわけではありません。セグメンテーションは、「似たニーズを持つ顧客グループ」を見つけ出す作業です。分割に使う軸は、業界や企業の特性によって異なります。

### セグメンテーション変数：4つの分類軸

市場を分割するとき、企業は4つの軸のいずれか（または組み合わせ）を使います。**試験では4種類をしっかり分類できることが必須**です。

| 変数タイプ | 説明 | 具体例 | マーケティング場面 |
| --- | --- | --- | --- |
| **地理的変数**（Geographic） | 地域、気候、都市規模による区別 | 関東/関西、北海道、東京都心/郊外、田舎 | 天候や文化に左右される商品（スキー用品は北国向け）；コンビニの地域別品揃え |
| **人口統計的変数**（Demographic） | 年齢、性別、所得、職業、家族構成 | 20代女性、高所得層、子育て中の親、会社員 | 化粧品の年代別ブランド分け；車の価格帯別ライン；保険の職業別設定 |
| **心理的変数**（Psychographic） | ライフスタイル、価値観、性格傾向 | 健康志向、環境配慮派、時短志向、高級志向 | オーガニック食品の販売対象；サステナビリティ重視の消費者層；デジタルネイティブ |
| **行動変数**（Behavioral） | 購買行動、利用頻度、ブランド忠誠度 | ヘビーユーザー、初回購買者、常連客、離反顧客 | ポイントカード会員の階層化；定期購入者向けキャンペーン；失客復帰プログラム |

**暗記のコツ**：「地理的」「人口（統計的）」「心理的」「行動的」の頭文字で「地人心行」と覚えるのが分かりやすいです。

**実務では組み合わせ**：例えば「30代女性（人口統計）で健康志向（心理的）、週3回以上のジム利用（行動）」という3つの軸を組み合わせたセグメントがあります。複数軸を組み合わせるほど、セグメントは「より同質」になり、同じ戦略で対応しやすくなります。

### セグメンテーション変数の選び方

市場によって「どの変数を優先するか」は異なります。

| 業界・商品 | 有効な変数 | 理由 |
| --- | --- | --- |
| **飲料・食品** | 地理的（地域の味好み） + 行動的（購買頻度） | 地域ごとに味覚が異なる。ヘビーユーザーとライトユーザーで提案が異なる |
| **化粧品・ファッション** | 人口統計的（年代・性別） + 心理的（ライフスタイル） | 年代で肌質・好みが異なる。価値観層が購買を左右する |
| **金融・保険** | 人口統計的（年収・家族構成） + 行動的（保有資産） | 経済力で必要な商品が異なる。過去の購買で顧客を層化 |
| **自動車・耐久消費財** | 人口統計的（所得） + 地理的（都市/地方） | 購買力と生活環境が購車を決める |
| **デジタルサービス** | 心理的（デジタルリテラシー） + 行動的（利用頻度） | テクノロジー親和性とユーザー粘着性 |

### 有効なセグメントの5つの要件

市場を「分割できた」ことと、「実行可能な戦略のターゲットになる」ことは別です。以下の5要件を満たすセグメントだけが、実際のマーケティング対象になります。

| 要件 | 意味 | チェックポイント |
| --- | --- | --- |
| **測定可能性**（Measurable） | セグメントの規模、購買力が数値化できるか | 「20代女性で月収30万以上」なら数値化できる；「幸せな消費者」は測定不可 |
| **接近可能性**（Accessible） | そのセグメントに、企業の広告や製品を届けられるか | テレビ広告では全年代に到達する；特定ニッチな層は到達困難 |
| **実質性**（Substantial） | セグメントに十分な規模と利益性があるか | 「月100人の購買」は実質的；「月5人」は商売にならない |
| **差別化可能性**（Differentiable） | セグメント間で異なるニーズがあり、異なる対応をする価値があるか | 「年代で好みが違う」なら別戦略を立てる価値がある；「みんな同じ」なら分割の意味がない |
| **実行可能性**（Actionable） | 自社の経営資源で、そのセグメント対応を実際にできるか | 「高級品セグメント」を狙っても、自社に品質・流通がなければ無理 |

**試験でよく見かけるトラップ**：セグメント自体が存在しても、上記5要件を満たさなければ、企業は対応できません。例えば「月1回だけ購買するシーズナル顧客」は**実質性が低い**；「1日3杯以上コーヒーを飲む人」は**測定困難**（接近可能性が低い）という判断ができることが重要です。

## ターゲティング（狙う層の選定）

セグメンテーションで市場を分割した後、「どのセグメントに経営資源を集中するか」を決めるのがターゲティングです。この選択が、その後のすべての経営判断（4P）に影響します。

### ターゲティング戦略の3大パターン（+ 拡張版）

企業は、次の3つの基本戦略（+新しい1つ）から選びます。**試験では「なぜその企業はその戦略を選んだのか」という理由付けが問われやすい**です。

#### 1. 無差別型マーケティング（Undifferentiated Marketing）

| 項目 | 内容 |
| --- | --- |
| **定義** | セグメンテーションを「せず」、市場全体を1つとして扱う。すべての顧客に同じ製品・価格・プロモーション |
| **コスト構造** | 製造・流通・プロモーション費用が最小（規模の経済） |
| **メリット** | 規模の経済が働く。製造・流通がシンプル。コスト競争力が高い |
| **デメリット** | 差別化が難しい。多様なニーズに対応できない。競合との違いが明確でない |
| **適する場面** | 顧客ニーズが「みんな同じ」か、商品が「標準化」されている業界 |
| **具体例** | 公営電力会社の料金設定（全顧客一律）；ガソリンスタンド；食塩 |

#### 2. 差別型マーケティング（Differentiated Marketing）

| 項目 | 内容 |
| --- | --- |
| **定義** | 複数セグメントそれぞれに対して、別々の製品・戦略を用意 |
| **コスト構造** | セグメント数分の製品開発・プロモーション・流通の負担増 |
| **メリット** | 各セグメントのニーズに最適に応える。複数セグメントから売上獲得。市場シェア拡大 |
| **デメリット** | 複雑性とコストが増加。管理負担が重い。セグメント間でニーズが似ていると無駄 |
| **適する企業** | 大企業。経営資源が豊富。複数ブランド・複数製造能力を持つ |
| **具体例** | トヨタのブランド戦略（プリウス・カローラ・クラウン・ランドクルーザー）；P&Gの複数ブランド（パンテーン、ヘッド&ショルダーズ、ガードなど）；銀行の個人向け・法人向け・富裕層向けの分化 |

#### 3. 集中型マーケティング（Concentrated Marketing / Niche Marketing）

| 項目 | 内容 |
| --- | --- |
| **定義** | 複数セグメントのうち、1つ（または少数）のセグメント「だけ」に経営資源を集中 |
| **コスト構造** | 対象セグメントのみの投資。他セグメントへの展開コストは不要 |
| **メリット** | セグメント内でニッチを深掘り。専門性により強い競争力。対象層への優位性が高い |
| **デメリット** | そのセグメントの衰退・縮小に脆弱。競合が同セグメントに参入するとリスク。規模の制限 |
| **適する企業** | 中小企業。スタートアップ。特定分野に特化した企業 |
| **具体例** | フェラーリ（超高級スポーツカー市場のみ）；有機・オーガニック専業メーカー；ニッチなB2B機械メーカー |

#### 4. マイクロマーケティング（One-to-One Marketing、デジタル時代の拡張）

| 項目 | 内容 |
| --- | --- |
| **定義** | 顧客「個別」（または超小規模グループ）に対応。データ・AIを活用した個別カスタマイズ |
| **コスト構造** | システム投資・データ管理は高コスト。ただし顧客単価が高い商品では回収可能 |
| **メリット** | 個別ニーズへの最高の対応。顧客満足度・ロイヤルティが高い |
| **デメリット** | スケーラビリティに限界。プライバシー懸念。DX投資が重 |
| **適する場面** | 高単価商品（保険、金融、不動産）；デジタルプラットフォーム（Amazon、Netflix）；B2B営業 |

**重要な視点**：企業が戦略を選ぶ理由は「経営資源の大きさ」と「顧客ニーズの多様性」です。大企業で資源が豊富で、市場ニーズが多様なら「差別型」。中小企業で資源が限定的なら「集中型」。顧客ニーズがほぼ同じなら「無差別型」。

### 3つの基本戦略の比較表

| 比較軸 | 無差別型 | 差別型 | 集中型 |
| --- | --- | --- | --- |
| **セグメント数** | 無視（市場全体を1つに） | 複数（通常3～10） | 1～2に限定 |
| **必要な経営資源** | 少ない | 多い | 少～中程度 |
| **製品・サービス数** | 1種類（標準製品） | 複数（セグメント数分） | 1～2種類（専門化） |
| **適する企業規模** | 大企業（規模の経済） or 小企業（選択） | 大企業が中心 | 中小企業、スタートアップ |
| **市場リスク** | 高い（競合と同質化） | 中程度 | 高い（特定市場依存） |
| **利益性** | 規模で稼ぐ | 各セグメントで利益 | ニッチで高マージン |
| **競争の激しさ** | 価格競争激化 | 各セグメントで差別化 | ニッチなため競争少ない |

## ポジショニング（顧客の頭の中での位置づけ）

### ポジショニングとは

ターゲットセグメントを選んだ後、「そのセグメントの中で、顧客は自社をどう見ているのか（または見てほしいのか）」を決めるのがポジショニングです。

**ポジショニング** = 「同じターゲット顧客の中で、競合との比較で、自社はどこに位置しているか（位置づけるか）」という認識

例を考えると分かりやすいです。スマートフォン市場の「スマートフォンが必要な会社員」というターゲットセグメント内でも：

- **Apple iPhone**：ブランド・デザイン・統合体験 → 「プレミアムで安定」
- **Samsung Galaxy**：テクノロジー・大画面・多機能 → 「高性能・拡張性」
- **Google Pixel**：カメラ・AI → 「写真とAIが得意」
- **中華メーカー（OPPO）**：価格・性能バランス → 「コスパ最高」

同じターゲットなのに、各企業の「位置づけ」が異なります。これがポジショニングです。

### ポジショニングマップ（知覚マップ）

ポジショニングは「文章」で説明するより、「図」で示すと分かりやすいです。2つの軸を設定し、その中で自社と競合がどこにいるかを示すのが「ポジショニングマップ」です。

```
例：カジュアルスニーカー市場（20～40代）

              ↑ ファッション性高
              |
            Adidas  Nike
              |  ●
 ────────────●┼●─────────→ 価格高
          Uniqlo |
              |
         ASICS ●
              |
              ↓ 実用性重視
```

軸の選び方が重要です。顧客が「本当に比較して選ぶ2つの要素」を軸にします。

- 「価格 × 品質」
- 「機能性 × ファッション性」
- 「地域密着性 × グローバル展開」
- 「伝統 × 革新」

ポイント：誰もいない空白の領域（ホワイトスペース）を見つけることが新規参入や差別化の鍵になります。

### 差別化軸の選び方

ポジショニングを実行するには、「何で差別化するか」を明確にする必要があります。

| 差別化軸 | 説明 | マーケティング例 |
| --- | --- | --- |
| **製品差別化** | 機能、品質、デザイン、特徴 | テスラの電動車技術；iPhoneの統合設計 |
| **サービス差別化** | カスタマーサービス、配送、保証 | Amazonの翌日配送；Appleの修理サービス |
| **流通差別化** | どこで・いつ・どう売るか | セブン-イレブンの24時間営業；DtoCブランド |
| **ブランド・イメージ差別化** | 企業イメージ、象徴的価値 | Nikeの「スポーツへの情熱」；Appleの「革新」 |
| **価格差別化** | 極端な価格戦略 | ユニクロの「質実剛健×低価格」；LVMHの「超高級」 |

**重要**：差別化軸は「自社ができるもの」「顧客が重視するもの」「競合との違いが明確なもの」の3つが重なるところを選びます。

## マーケティングミックス（4P）：戦略を実行に変える

STPで「誰に、どう見える位置づけで戦うか」を決めた後、実際に「何を、いくらで、どこで、どう売るか」を決めるのが4Pです。STPなしの4P決定は、羅針盤なしで船を漕ぐようなものです。

### 4Pと4Cの対応関係

「マーケティングミックス4P」は企業の視点です。しかし同じ施策を「顧客視点」で見ると「4C」に読み替えられます。**試験では4P/4C対応が頻出**です。

| 4P（企業視点）| 4C（顧客視点） | 対応の意味 | 具体例 |
| --- | --- | --- | --- |
| **Product** | **Customer Value** | 「企業が何を売るか」 vs 「顧客が何の価値を得るか」 | iPhoneは「スマートフォン」ではなく「シンプルな体験」を売っている |
| **Price** | **Cost** | 「企業が価格を決める」 vs 「顧客が払う総コストを考える」 | 定価1万円でも、買いに行く時間・手間が高い場合、顧客のコスト全体は高い |
| **Place** | **Convenience** | 「企業が流通チャネルを設定」 vs 「顧客が便利に買える場所か」 | オンライン専売は企業にはコスト削減だが、顧客には「店で触れない」という不便さ |
| **Promotion** | **Communication** | 「企業が一方的に宣伝」 vs 「顧客と対話的に情報交換」 | TV広告は「企業→顧客」の一方向；SNSは「顧客↔企業」の双方向 |

**マーケティングの本質**：4Pを決めるときに、「顧客の4Cのニーズを満たしているか」を常に問い直すことが、成功するマーケティングの鍵です。

### 4Pと4Cの対応例：スターバックスの場合

スターバックスのマーケティングを例に、4Pと4Cの対応を見てみましょう。

| P / C | 企業の決定（4P） | 顧客の視点（4C） | 実行例 |
| --- | --- | --- | --- |
| **Product / Value** | 高品質コーヒー＋洗練な空間 | 「美味しく、落ち着いて過ごせる体験」を求めている | 「コーヒー」ではなく「第三の場所」を提供 |
| **Price / Cost** | 高価格（300～500円） | 「その体験に払う価値がある」と認識 | 単なる飲料代ではなく「環境代＋時間代」 |
| **Place / Convenience** | 駅前・商業施設・オフィス周辺出店 | 「仕事中に立ち寄れる」「待ち合わせに使える」 | 利便性の高い立地選定 |
| **Promotion / Communication** | SNS・アプリで双方向対話 | 「自分のニーズに合った情報がほしい」 | パーソナライズされた提案：学生向けキャンペーン、ファミリー向けイベント |

このように、4Pを決める際に「顧客の4C」が満たされているかを確認することが、効果的なマーケティングの条件です。

### 4Pの全体像

各Pの詳細な戦略論は別ページで扱います。ここでは「4つがどのようなカテゴリか」を押さえます。

| P | 日本語 | マーケティング要素 | 主な決定項目 |
| --- | --- | --- | --- |
| **Product** | 製品 | 何を提供するか | 機能・品質・デザイン・ブランド・サービス・パッケージ |
| **Price** | 価格 | いくらで提供するか | 定価・割引・支払条件・ローンの有無 |
| **Place** | 流通 | どこで届けるか | 販売チャネル（直販・小売・オンライン）・配送・在庫 |
| **Promotion** | プロモーション | どう伝えるか | 広告・販売促進・PR・営業・SNS |

**重要なポイント**：4Pは「独立」ではなく「連動」しています。例えば、「超高級品」にポジショニングしたのに、激安価格では矛盾しています。STPで決めた戦略と、4Pが一貫しているかを常に確認することが成功の鍵です。

**詳細は他ページで**：各P（製品戦略、価格戦略、流通戦略、プロモーション戦略）の具体的な手法、ライフサイクル、セグメント別の展開については、別々のページで詳しく扱います。

## トランザクション・マーケティング vs リレーションシップ・マーケティング

### 2つの経営思想

マーケティングの実行方法は、大きく2つの考え方に分かれます。**試験では「この企業はどちらを採用しているのか」という判断が問われます**。

| 観点 | トランザクション・マーケティング | リレーションシップ・マーケティング |
| --- | --- | --- |
| **目的** | 1回1回の購買取引成立 | 顧客との長期的な関係構築 |
| **時間軸** | 短期（購買時点） | 長期（購買前～購買後～ずっと） |
| **利益の源** | 新規顧客を何人獲得するか | 既存顧客をどれだけ保持・拡大購買させるか |
| **顧客との関係** | 「売り手と買い手」の一過性 | 「パートナー」の継続的な信頼関係 |
| **主な施策** | 広告・販売促進・キャンペーン | ロイヤリティプログラム・カスタマーサービス・顧客データ活用 |
| **コミュニケーション方向** | 一方向（企業→顧客） | 双方向（企業↔顧客） |
| **適する業界** | 日用品・ファッション・食品 | 保険・銀行・通信・高級品 |

### 実行施策の違い

| トランザクション・マーケティング | リレーションシップ・マーケティング |
| --- | --- |
| テレビCM、新聞広告で「広く浅く」リーチ | メールマガジン、会員アプリで「狭く深く」対話 |
| キャンペーン・期間限定セール | ポイントカード・ロイヤルティプログラム |
| 成約後はフォローアップなし | 購買後のフォローアップ（サンクスメール、使い方サポート） |
| 顧客評価：新規獲得数 | 顧客評価：LTV（生涯顧客価値）、リピート率 |

### 関連する「法則」

| 法則 | 内容 | マーケティングの教訓 |
| --- | --- | --- |
| **1:5の法則** | 新規顧客1人の獲得コスト = 既存顧客1人の維持コスト × 5 | 既存顧客を守る方が、新規獲得より効率的 |
| **5:25の法則** | 顧客離反を5%減らす → 利益は25%増加 | わずかな保持率向上が、大きな利益改善につながる |

**重要な教訓**：リレーションシップ・マーケティングが重視される理由は、**既存顧客のコストが安く、利益貢献が大きいから**です。高度な競争環境では、新規獲得よりも既存顧客との関係を深める方が、より経営効率的です。

## よくあるつまずき

<Callout title="重要：試験で落とすパターン">

1. **STPと4Pの論理を逆に理解**
   - 誤り：「まず4Pを決めてから、セグメンテーション変数を選ぶ」
   - 正解：STP（市場分割→ターゲット→位置づけ）が決まってから、4Pが意味を持つ。順序が大事

2. **セグメンテーション変数の分類間違え**
   - 誤り：「年齢は心理的変数」「職業は心理的変数」
   - 正解：年齢・所得・職業はすべて「人口統計的変数」。心理的変数は「ライフスタイル」「価値観」など心の状態

3. **「心理的」と「行動的」を混同**
   - 誤り：「買う頻度が高い」→ 心理的変数
   - 正解：「買う頻度」は「行動変数」。心理的変数は「なぜ買うのか」という価値観や信念

4. **有効なセグメント5要件をうろ覚え**
   - よく忘れるもの：「差別化可能性」「実行可能性」。単に「規模がある」だけではダメ
   - トラップ問題：「月1回の購買」セグメント → 「実質性が低い」と判断できるか

5. **ターゲティング戦略の判断ミス**
   - 誤り：「複数セグメントに対応 = 差別型」「1セグメント = 集中型」の単純化
   - 正解：「経営資源の配分」と「セグメント数」の両方を見る必要がある

6. **ポジショニングを製品特性だけで説明**
   - 誤り：「iPhoneは高性能だからプレミアム」
   - 正解：ポジショニングは「顧客の認識」。iPhoneが「デザイン×ブランド価値」でポジショニングされているのが重要

7. **4Pと4Cの対応を忘れる**
   - 試験では「Productは何に対応するか」という問いかけがある
   - Product ↔ Customer Value、Price ↔ Cost、Place ↔ Convenience、Promotion ↔ Communication

8. **無差別型を「古い戦略」と見なす**
   - 誤り：「今は全員が差別化戦略をしている」
   - 正解：食塩、水、ガソリンなど「標準化商品」では無差別型が理に適っている

</Callout>

## 問題を解く流れ（診断士試験での読み方）

<Callout title="STP分析の出題パターン">

試験問題では、通常以下のような問い方をされます：

1. **「この企業のセグメンテーション変数は何か」**
   - 読むべき点：市場をどう分割しているか
   - 答え方：4種類のどれか（地理的・人口統計的・心理的・行動的）を明記

2. **「狙うべきターゲットセグメントは何か」**
   - 読むべき点：経営資源の大きさ、市場規模、競合状況
   - 答え方：セグメントの定義 + ターゲティング戦略の理由

3. **「このセグメントは有効か」**
   - 読むべき点：5要件（測定・接近・実質・差別化・実行）を満たすか
   - 答え方：どの要件が欠けているか、または「満たしている」の論拠

4. **「ポジショニングと4Pが一貫しているか」**
   - 読むべき点：企業が宣言しているポジションと、実際の施策が矛盾していないか
   - 答え方：例「高級品ポジション（ポジショニング）なのに激安価格（4P）→ 矛盾」

</Callout>

## 確認問題

<Callout title="問1：セグメンテーション変数の分類（暗記確認）">

「スポーツジムが、30～40代の高所得会社員で健康志向の男性」をターゲットにしている場合、以下をそれぞれ何変数に分類するか。

- 年齢（30～40代）
- 所得（高所得）
- 性別（男性）
- ライフスタイル（健康志向）
- 利用頻度（週3回以上）

**解答**：
- 年齢・所得・性別 → **人口統計的変数**
- ライフスタイル（健康志向） → **心理的変数**
- 利用頻度 → **行動変数**

※ 地理的変数が問題に含まれていなくても、「都市部の会員」なら「地理的変数」の例になることを認識しておく。

</Callout>

<Callout title="問2：有効なセグメントの判定">

ある企業が「月1回だけランニングシューズを購買するシーズナル顧客」をセグメンテーションの対象にしたいと考えている。このセグメントが企業のターゲッティング対象として有効か、5要件に照らして判定せよ。

**解答例**：
- **測定可能性**：〇（購買頻度は計測可能）
- **接近可能性**：△（特定の季節にしか広告効果がない）
- **実質性**：✗（月1回では収益性が低い。年間の利益貢献が限定的）
- **差別化可能性**：△（シーズナル層は他セグメントと異なるニーズを持つが、顧客が少ないため施策が限定的）
- **実行可能性**：✗（少数の顧客のために専門的な製品・流通を用意するのは非効率）

**判定**：**有効性が低い**。特に「実質性」と「実行可能性」の要件を満たさないため、経営戦略の対象として適切ではない。

</Callout>

<Callout title="問3：ターゲティング戦略の選択根拠">

以下の3社について、それぞれ最適なターゲティング戦略を選び、その理由を述べよ。

- A社：大手自動車メーカー（資本金500億円、複数製造工場）
- B社：地元の工務店（資本金5000万円、特定地域で営業）
- C社：オーガニック食品専業企業（資本金2000万円、全国販売）

**解答例**：
- **A社**：**差別型マーケティング**。理由：経営資源が豊富であり、複数セグメント（高級車・ファミリーカー・商用車など）に向けて異なる製品・戦略を展開できる。規模の経済を活用できる。
- **B社**：**集中型マーケティング**。理由：経営資源が限定的。特定地域（例：〇〇県）かつ特定ニーズ（例：環境配慮系リフォーム）に集中することで、競争力を発揮できる。
- **C社**：**集中型またはニッチ型**。理由：オーガニック市場は限定的だが、その市場での専門性を武器に、全国の価値観共有層にアプローチする。ただし経営資源の制約から、複数セグメント展開は困難。

</Callout>
