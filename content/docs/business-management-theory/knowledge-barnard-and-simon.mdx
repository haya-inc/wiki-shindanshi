## このページの役割

<Callout title="このページの役割">
    このページは、`バーナード` と `サイモン` を整理する解説ページです。どちらも組織論の定番ですが、`組織がどう成り立つか` と `人がどう意思決定するか` で問いの立て方が違います。
</Callout>

<Callout title="このページの読み方">
  バーナードの問いは「なぜ人は組織に参加し、命令に従うのか」、サイモンの問いは「なぜ人は最適解を選べないのか」です。一見似ていますが、バーナードは組織の **成立と存続** を、サイモンは個人の **意思決定の限界** を扱います。古典的管理論・人間関係論との位置づけも合わせて押さえましょう。
</Callout>

## 学習のポイント

- `バーナード` は組織の成立条件と権限受容説
- `サイモン` は限定合理性と意思決定の分類
- 古典的管理論と人間関係論の位置づけも押さえる

## 試験で何が問われるか

- バーナードの組織の3要素を列挙できるか
- 権限受容説と無関心圏を説明できるか
- 限定合理性と満足化基準の違いを言えるか
- 誘因と貢献の均衡を説明できるか

## バーナードの組織論

### 組織の3要素

公式組織が成立するための3つの条件です。

| 要素 | 内容 |
| --- | --- |
| 共通目的 | 組織の目的（メンバーが共有する目標） |
| 貢献意欲（協働意思） | メンバーが組織に貢献しようとする意志 |
| コミュニケーション | 意思疎通（目的と意思をつなぐ手段） |

3つが揃って初めて公式組織は成立します。

### 公式組織と非公式組織

| | 公式組織 | 非公式組織 |
| --- | --- | --- |
| 形成 | 意識的・計画的 | 自然発生的 |
| 構造 | 明確な目的・規則・権限 | 人間関係のネットワーク |
| 役割 | 組織の目的達成 | コミュニケーション促進、貢献意欲の維持 |

### 誘因と貢献の理論（組織均衡論）

```
組織が提供する誘因 ≧ 構成員の貢献 → 参加し続ける（組織均衡）
誘因 < 貢献 → 離脱する
```

誘因には物質的誘因（給与）だけでなく、非物質的誘因（やりがい、地位、仲間との関係）も含まれます。

### 権限受容説

命令が権限を持つかどうかは、**受け手が受容するか否か** によって決まるという考え方です。

**無関心圏（受容圏）**：この範囲内の命令は無条件で受け入れられます。無関心圏が広いほど、組織の統制は容易です。

| 対比 | 公式権限説 | 権限受容説（バーナード） |
| --- | --- | --- |
| 権限の源泉 | 地位・役職（上から） | 受け手の受容（下から） |
| 命令の有効性 | 地位があれば有効 | 受け入れられて初めて有効 |

### 有効性と能率

| 概念 | 内容 |
| --- | --- |
| 有効性 | 組織目的の達成度 |
| 能率 | 個人の動機の満足度 |

組織は有効性と能率の両方を満たす必要があります。

## サイモンの意思決定論

### 限定合理性と満足化基準

| 概念 | 内容 |
| --- | --- |
| 限定合理性 | 人間の情報処理能力には限界があり、完全に合理的な意思決定はできない |
| 満足化基準（satisficing） | 最適解ではなく、「満足できる水準」の解を選ぶ |

「限定合理性」は「非合理」ではなく、合理的であろうとするが **能力に限界がある** という意味です。

### 意思決定の前提

| 前提 | 内容 |
| --- | --- |
| 価値前提 | 目的の選択（何を目指すか）── 上位の意思決定 |
| 事実前提 | 手段の選択（どう実現するか）── 下位の意思決定 |

### 意思決定の分類

| 種類 | 内容 | 対処法 |
| --- | --- | --- |
| プログラム化された意思決定 | 定型的・反復的 | ルール、手続き、組織構造 |
| プログラム化されていない意思決定 | 非定型的・新規 | 判断力、創造性、訓練 |

### 組織と意思決定

組織は個人の意思決定の **前提を統制** することで、間接的に行動を方向づけます。直接命令するだけでなく、情報や評価基準を通じて意思決定に影響を与えます。

## 組織間関係論

| 理論 | 提唱者 | 内容 |
| --- | --- | --- |
| 資源依存理論 | フェファー＆サランシク | 組織は外部資源に依存し、依存度を下げるように行動する |
| 組織エコロジー | ハナン＆フリーマン | 環境が組織を選択する（淘汰）。構造的慣性により変化しにくい |
| 制度理論（同型化） | ディマジオ＆パウエル | 制度的圧力で組織は同型化する |

### 同型化の3つのメカニズム

| メカニズム | 内容 | 例 |
| --- | --- | --- |
| 強制的同型化 | 法規制等による強制 | 法令遵守 |
| 模倣的同型化 | 不確実性下での成功企業の模倣 | ベストプラクティスの導入 |
| 規範的同型化 | 専門家教育等による規範共有 | MBA教育、業界団体の基準 |

## 典型的なつまずき

- バーナードとサイモンを理論家名だけで覚える → 問いの違い（**組織の成立** vs **意思決定の限界**）を押さえる
- 権限受容説を上意下達の話だと誤解する → 受け手が受容して初めて権限が成立（**下から上**）
- 限定合理性を「非合理」と混同する → 合理的であろうとするが **能力に限界** がある
- 最適化と満足化の違いを言えない → 最適化は全選択肢の中の最善、満足化は **許容水準を超える解** で十分
- 組織均衡論を給与だけの話と理解する → 非物質的誘因（やりがい、仲間）も重要

## 問題を解くときの観点

- 問われているのは `組織の成立` か `個人の意思決定` か
- 論点は `命令が受け入れられる条件` か `情報処理の限界` か
- 組織全体の協働を見るのか、個人の選択を見るのか

## 確認問題

<Callout title="問1：組織の3要素">
    バーナードが示した公式組織の成立に必要な3つの要素を挙げよ。

    **解答**：①共通目的、②貢献意欲（協働意思）、③コミュニケーション。3つが揃って初めて公式組織が成立する。
</Callout>

<Callout title="問2：権限受容説">
    「上司が部下に命令したが、部下が不合理だと感じて従わなかった」。バーナードの権限受容説ではこの状況をどう説明するか。

    **解答**：命令が部下の無関心圏（受容圏）の外にあったため、権限として成立しなかった。権限受容説では、命令の有効性は受け手の受容に依存する。
</Callout>

<Callout title="問3：限定合理性">
    サイモンの「満足化基準」とはどのような意思決定の方法か。完全合理性との違いを説明せよ。

    **解答**：満足化基準は、全ての選択肢を検討して最善を選ぶのではなく、許容できる水準を超える解が見つかった時点で意思決定する方法。人間の情報処理能力に限界がある（限定合理性）ため、最適化ではなく満足化が現実的な意思決定となる。
</Callout>
