## 学習のポイント

- `製品の3層` を理解してから、`ブランド戦略` へ進む

- `価格設定の3アプローチ` を、それぞれ「誰の視点か」で区別する
- `新製品の価格戦略`（スキミング vs ペネトレーション）と `心理的価格設定` を分けて扱う

## 試験で何が問われるか

- 製品の3層モデル（コトラー）を説明できるか
- 製品ライフサイクルの各段階で、売上・利益・顧客数がどう変わるか
- ブランド拡張の4象限を使って、戦略の妥当性を判断できるか
- 価格設定の3アプローチを、状況に応じて使い分けられるか
- スキミング価格とペネトレーション価格の条件を区別できるか

## 製品の3層構造（コトラー）

製品は、顧客視点で3つの層で考えます。マーケティング戦略を立てるときは、表面の「実体」だけでなく、中核の「ベネフィット」と周囲の「付随機能」も含めて設計する必要があります。

```
┌─────────────────────────────────┐
│      付随機能（周囲）            │
│  ├─ アフターサービス            │
│  ├─ 保証・返品対応              │
│  ├─ 配送・インストール          │
│  └─ カスタマー・サポート        │
│  ┌─────────────────────────┐   │
│  │    実体（見えるもの）    │   │
│  │  ├─ 品質・形状・色      │   │
│  │  ├─ デザイン           │   │
│  │  ├─ パッケージング     │   │
│  │  └─ ブランド名         │   │
│  │  ┌───────────────────┐ │   │
│  │  │  中核（ベネフィット）│ │   │
│  │  │ 顧客が得られる価値 │ │   │
│  │  │ (快適さ、安心、    │ │   │
│  │  │  時間節約など)    │ │   │
│  │  └───────────────────┘ │   │
│  └─────────────────────────┘   │
└─────────────────────────────────┘
```

### 3層の説明

| 層 | 内容 | 例 |
| --- | --- | --- |
| **中核（Core Benefit）** | 顧客が求める本質的な価値・ベネフィット | ドリルを買う客が本当に欲しいのは「穴」（機能）。さらに言えば「棚を設置する自由」（ベネフィット） |
| **実体（Product）** | 製品の物理的・視覚的な姿 | ドリルの仕様、デザイン、色、ブランド名、パッケージング |
| **付随機能（Augmented）** | 購入後のサービス・保証・配送など | 返品対応、修理保証、配送、使い方の相談、カスタマーサポート |

**実践例：「ホテルの部屋」**
- **中核**：快適な睡眠、安心できる空間
- **実体**：ベッドのサイズ・質感、内装・色、眺望、部屋の広さ、設備（シャワー・トイレ等）
- **付随機能**：チェックイン対応、荷物運び、朝食配信、トラブル時の対応、ロイヤルティプログラム

## 製品ミックス

企業が提供する複数の製品・サービス全体を、次の4つの視点から分析します。

### 製品ミックスの4つの次元

| 次元 | 定義 | 説明 |
| --- | --- | --- |
| **幅（Width）** | 製品ラインの数 | 企業が提供する異なるカテゴリー・ジャンルの数 |
| **長さ（Length）** | 全製品アイテムの総数 | 全ラインに含まれるアイテム数の合計 |
| **深さ（Depth）** | 各ラインのバリエーション数 | 同じラインの中での色・サイズ・モデルの種類 |
| **整合性（Consistency）** | ラインどうしの関連性の強さ | ラインが原材料、流通チャネル、ユーザー層で結びついているか |

**例：家電メーカーA社**

```
┌─────────────────────────────────────────────┐
│ 製品ミックスの形状                            │
├─────────────────────────────────────────────┤
│ 幅：冷蔵庫、洗濯機、エアコン、テレビ（4ライン）│
│ 長さ：冷蔵庫3種類 + 洗濯機2種類 + ...計12品  │
│ 深さ：冷蔵庫ラインで色4種×容量3種=12バリエ  │
│ 整合性：全て家庭用、同じ流通チャネル、重複顧客│
└─────────────────────────────────────────────┘
```

製品ミックスを広げれば市場対応力が高まる一方、複雑性やコストが増す。スリム化する（リポジショニング）なら、確実に勝てるセグメントに集中する。

## 製品ライフサイクル（PLC）

製品は市場投入後、導入期 → 成長期 → 成熟期 → 衰退期 の4段階で売上・利益が推移します。各段階で市場の状況、競合数、顧客ニーズが異なるため、戦略も柔軟に変える必要があります。

### 4段階の比較表

| 段階 | 売上 | 利益 | 顧客数 | 競合 | 特徴 |
| --- | --- | --- | --- | --- | --- |
| **導入期** | 低 | 負または小 | 革新的層 | 少ない | 認知不足、高価格設定、投資が大きい |
| **成長期** | 急増 | 増加・最大化 | 採用者層・多数派層へ拡大 | 増加 | 需要が急増、参入企業増える、品質・機能で差別化 |
| **成熟期** | 横ばい～減少 | 減少 | 市場の大部分 | 多い | 競争激化、供給過剰、価格低下、セグメント化進む |
| **衰退期** | 減少 | 減少 | 保守的層のみ | 減少 | 代替製品登場、市場縮小、終了か低コスト維持かの判断 |

### 各段階での4P戦略

| | **導入期** | **成長期** | **成熟期** | **衰退期** |
| --- | --- | --- | --- | --- |
| **製品** | 基本機能に絞る | 機能・品質向上、ラインアップ拡大 | バリエーション増加、差別化深化 | ライン整理、高機能版継続 |
| **価格** | 高価格（スキミング）または低価格（ペネトレーション）| 競争激化で下げ始める | 低価格・値下げ圧力 | さらに低価格化または撤退 |
| **流通** | 限定的 | 流通チャネル拡大 | 広く利用可能 | 絞り込み |
| **プロモーション** | 認知獲得に投資大 | 差別化訴求、ブランド構築 | 防御的、維持的 | 削減 |

## ブランド戦略

### ブランドエクイティ（D.アーカー）

ブランドエクイティは、ブランド名が製品に付加する価値です。強いブランドは、利益率向上、新製品投入時の成功率向上、顧客離反の防止をもたらします。

アーカーは、ブランドエクイティを次の5要素から構成するとしました。

#### 5つの構成要素

| 要素 | 説明 | 企業への効果 |
| --- | --- | --- |
| **ブランド認知** | 市場での知名度、想起度 | 認知が高いほど選択肢に入りやすい |
| **知覚品質** | 顧客が感じる品質の程度 | 品質が高いと感じさせられれば価格プレミアムが可能 |
| **ブランド連想** | ブランドと結び付く独特の属性・価値観・イメージ | 「安心」「高級」「革新的」など、差別化の源泉 |
| **ブランドロイヤルティ** | 繰り返し購買、推奨行動 | 既存顧客の維持コスト低減、安定売上 |
| **その他の資産** | 特許、商標、流通チャネル上の優位性 | 法的保護、競合参入抑制 |

**例：Apple社**
- **ブランド認知**：世界的に高い
- **知覚品質**：「プレミアム・高品質」のイメージ
- **ブランド連想**：「革新」「シンプル」「ステータス」
- **ロイヤルティ**：継続購買率が高い、推奨度が高い
- **その他資産**：独自のOS、デザイン特許、App Store生態系

### ブランド体系：NB vs PB

| | **ナショナルブランド（NB）** | **プライベートブランド（PB）** |
| --- | --- | --- |
| **提供者** | メーカー/ブランド企業 | 流通企業（スーパー、コンビニ等） |
| **特徴** | メーカー独自、全国で統一 | 流通企業の名前で展開、地域差あり |
| **価格** | 一般に高い | 一般に低い（コスト削減、高利益率）|
| **品質認知** | 長年の積み重ねで確立 | 「安い」で選ばれることが多い |
| **顧客心理** | ブランド価値への信頼 | コストパフォーマンス重視 |
| **事例** | コカ・コーラ、ソニー等 | セブン-イレブン「セブンプレミアム」等 |

**PB戦略の狙い**：流通企業にとって、NBより高い粗利益率が得られ、メーカーとの交渉力も強化できます。ただし、ブランド構築に時間がかかり、品質維持の責任も重くなります。

### ブランド拡張戦略（4象限）

既存のブランド資産を活かして、新しい製品カテゴリーに進出するか、既存カテゴリーに新ブランドを立てるかの判断。

| | **既存ブランド名** | **新ブランド名** |
| --- | --- | --- |
| **既存カテゴリー** | **ライン拡張** | **マルチブランド戦略** |
| **新カテゴリー** | **ブランド拡張** | **新ブランド** |

#### 各象限の説明と条件

**1. ライン拡張（Line Extension）**
- 既存ブランドで、同じカテゴリー内の新しい製品を追加
- **条件**：既存ブランドが強く、顧客が自然に期待できる範囲
- **メリット**：ブランド認知を活用、開発投資削減
- **リスク**：オリジナルのブランドイメージが希薄化（ブランド稀釈化）
- **事例**：コカ・コーラ → コカ・コーラゼロシュガー

**2. ブランド拡張（Brand Extension / Category Extension）**
- 既存ブランドを、異なるカテゴリーに拡張
- **条件**：拡張先カテゴリーと既存ブランドのイメージ親和性が高い
- **メリット**：強力なブランド資産を新市場で活用、開発リスク低減
- **リスク**：親ブランドのイメージ損傷、説得力の欠如
- **事例**：トヨタ（自動車） → トヨタホーム（住宅）

**3. マルチブランド戦略（Multi-brand Strategy）**
- 既存カテゴリーで、異なるブランド名を複数立てる
- **条件**：異なるセグメントをターゲット、ポジショニング・価格帯が異なる
- **メリット**：棚割で複数ポジション占有、市場カバレッジ向上
- **リスク**：ブランド間の共食い、管理コスト増加
- **事例**：P&G → 同じ洗剤市場で「Tide」「Ariel」などの複数ブランド

**4. 新ブランド（New Brand）**
- 新カテゴリーで、新しいブランドを一から立ち上げ
- **条件**：既存ブランドとのシナジーがない、または害になる
- **メリット**：新市場で自由な位置づけ、既存ブランドへの影響なし
- **リスク**：ブランド構築に時間・投資が大きい、認知獲得困難
- **事例**：ソニー（電子機器） → Vaio（パソコン）※後に撤退

#### ブランド拡張の成功・失敗を判断する視点

| 評価観点 | 拡張が成功しやすい場合 | 失敗しやすい場合 |
| --- | --- | --- |
| **親ブランドのイメージ** | イメージが汎用的・広い | イメージが狭い・限定的 |
| **拡張先との親和性** | 顧客が「自然」と思う | 「なぜ？」と疑問に思う |
| **親ブランドの強さ** | 強く、信頼度が高い | 弱い、認知度が低い |
| **拡張先市場の難易度** | 参入障壁が低い | 参入障壁が高い（特殊技術等） |

### 地域ブランド

地方の資源（伝統、特産物、観光地）を活用して、地域の認知度・ブランド価値を高める戦略。

**要素**：
- 地域資源の発掘・活用（農産物、工芸品、観光地等）
- 統一的なブランドメッセージの発信
- 品質保証・認証制度（GI・地理的表示等）
- 流通・販売チャネルの開発

**2次試験との接続**：事例Ⅱ（流通・マーケティング）で地方企業の地域ブランド展開が出題されやすい。

## 新製品開発プロセス

新製品開発は、単なる技術開発ではなく、市場調査 → 検証 → テスト → 本格化という段階的プロセスです。各段階で失敗判定し、不採算案を早期に除外することが重要です。

### 7段階のプロセス

```
┌───────────┐
│ アイデア創出 │ 社員提案、顧客ニーズ調査、技術シーズから着想
└─────┬─────┘
      ↓
┌───────────┐
│スクリーニング│ 企業戦略との整合性、技術実現性、市場規模で判定
└─────┬─────┘
      ↓
┌──────────────────┐
│コンセプト開発     │ 顧客ニーズに対応する製品コンセプト
│テスト         │ ターゲット層へのコンセプトテスト
└─────┬─────────────┘
      ↓
┌──────────────────┐
│事業性分析        │ 市場規模、コスト、損益予測の検討
│（ビジネス評価）  │
└─────┬─────────────┘
      ↓
┌──────────────────┐
│試作品開発        │ プロトタイプ製造、機能・品質検証
└─────┬─────────────┘
      ↓
┌──────────────────┐
│テストマーケティング│ 限定市場での試販、反応確認
└─────┬─────────────┘
      ↓
┌───────────┐
│商品化      │ 本格生産、全国展開、プロモーション実施
└───────────┘
```

### 各段階の詳細

| 段階 | 目的 | 実施内容 | 主な意思決定 |
| --- | --- | --- | --- |
| **アイデア創出** | 新規案を多数発掘 | ブレストレーミング、顧客ヒアリング、R&D提案 | どの案を進めるか |
| **スクリーニング** | 不可能案を早期排除 | 企業戦略との適合、技術性、市場規模で判定 | ゴーノーゴー判定 |
| **コンセプト開発** | 製品像を明確化 | 顧客ニーズ、機能、ターゲット、ポジショニング | 顧客受け入れ可能か |
| **事業性分析** | 採算性を検証 | 市場規模、参入コスト、想定売上、損益分岐点 | 投資に値するか |
| **試作品開発** | 機能を検証 | プロトタイプ製造、技術テスト、品質確保 | 製造可能か |
| **テストマーケティング** | 市場反応を検証 | 限定エリア・期間での試販、消費者調査 | 修正必要か、展開するか |
| **商品化** | 市場投入 | 本格生産設備、流通構築、プロモーション | 全国展開のタイミング |

**重要**：新製品開発は、段階を進むにつれ投資額が増加します。早期段階での厳しい判定が、後段での失敗を防ぎ、限られた資源を有効活用するカギになります。

## 価格戦略

価格は、製品の価値を金銭で表現する最後の決定です。同じ製品でも、価格設定の論理が異なると、利益構造や競合対応が大きく変わります。

### 価格設定の3つのアプローチ

企業は、以下の3つの視点から価格を決定します。

#### 1. コスト志向（Cost-Oriented Pricing）

**原理**：`製品のコストを基準に、一定の利益率を上乗せして価格を決める`

| 方法 | 計算式 | 活用場面 | 注意点 |
| --- | --- | --- | --- |
| **コストプラス法** | 売価 = 原価 ÷（1 − 目標利益率） | 一般的な価格決定 | 市場需要を無視しがち |
| **マークアップ法** | 売価 = 原価 + （原価 × マークアップ率） | 流通業、小売店 | コストと売価の関係が単純 |
| **損益分岐点法** | BEP = 固定費 ÷（売価 − 変動費） | 新製品、新事業の最低価格設定 | 売上目標を先に決める必要あり |

**メリット**：わかりやすい、計算が簡単、原価割れを防げる
**デメリット**：顧客の支払い意思度を無視、市場競争を考慮しない

#### 2. 需要志向（Demand-Oriented Pricing）

**原理**：`顧客の知覚価値や購買欲求を基準に、いくらなら買ってくれるかを想定して価格を決める`

| 方法 | 説明 | 適用例 |
| --- | --- | --- |
| **知覚価値価格設定** | 顧客が感じる製品価値（機能、ブランド、利便性）に基づいて価格を設定 | プレミアムブランド、高機能製品 |
| **需要差別価格** | 顧客セグメント・使用場面・時期により価格を変え、需要パターンに対応 | 映画館の時間帯別料金、航空運賃の季節変動 |

**メリット**：顧客満足度が高い、利益率向上、セグメント戦略と結合可能
**デメリット**：顧客の支払い意思度を正確に把握が難しい、計算が複雑

#### 3. 競争志向（Competition-Oriented Pricing）

**原理**：`競合製品の価格を参考にして、自社製品をどのポジションに置くかを決める`

| 方法 | 説明 | 活用場面 |
| --- | --- | --- |
| **実勢価格法** | 競合品の価格に合わせる、またはそれより高い/低いを設定 | 差別化が難しい商品、多数参入市場 |
| **入札価格法** | 競争入札で、採算点を考慮しつつ競争力ある価格を提示 | 建設、公共入札、B2B取引 |

**メリット**：市場反応が予測しやすい、過度な競争回避
**デメリット**：自社の利益構造が軽視されがち、価格競争に巻き込まれやすい

#### 3アプローチの比較表

| 視点 | コスト志向 | 需要志向 | 競争志向 |
| --- | --- | --- | --- |
| **基準** | 内部（原価） | 外部（顧客） | 外部（競合） |
| **利点** | シンプル、原価割れ防止 | 顧客満足、利益最大化 | 市場適応、予測しやすい |
| **欠点** | 需要無視 | 把握困難 | 利益無視 |
| **実務** | 基本方針、最低価格の決定 | セグメント・プレミアム化 | 参考値、調整 |

### 新製品の価格戦略：スキミング vs ペネトレーション

新製品を市場投入するとき、高い価格から始めるか、低い価格から始めるかで、初期段階の利益構造と市場カバレッジが大きく変わります。

#### スキミング価格（Skimming Price）

**定義**：新製品投入時に、高めの価格を設定し、革新的層や上流層から利益を「搾り取る」戦略

```
売価（高）
  │     ┌─────┐
  │     │ ス │ 利益率高
  │     │ キ │ 販売数少
  │     │ ミ │
  │──────┘   └─────→時間経過
  │
  └──市場浸透(価格低下)
```

| 側面 | 説明 |
| --- | --- |
| **目的** | 初期投資の早期回収、利益最大化 |
| **適用条件** | 新製品の差別化が明確、顧客の支払い意思度が高い、競合参入が遅い |
| **メリット** | 高利益率、ブランド価値維持、開発投資を急速回収 |
| **デメリット** | 市場規模が限定的、競合参入を招きやすい、顧客不満の可能性 |
| **事例** | 新型スマートフォン、高級家電、医療機器 |

#### ペネトレーション価格（Penetration Price）

**定義**：新製品投入時に、低めの価格を設定し、大量の顧客を獲得して素早く市場を占有する戦略

```
売価（低）
  │
  │ ┌─────┐  販売数多い
  │ │ペ  │ 利益率低い
  │ │ネ  │ 競合参入が遅れる
  │─┘   └──────→時間経過
  │
  └──市場独占後、価格上昇
```

| 側面 | 説明 |
| --- | --- |
| **目的** | 素早い市場占有、競合の参入阻止、スケールメリット獲得 |
| **適用条件** | 価格に敏感な顧客層、大量生産による低コスト実現可能、競合参入が予想される |
| **メリット** | 大量販売で規模の経済実現、市場シェア獲得、競合参入の抑止 |
| **デメリット** | 初期段階の利益率が低い、ブランド価値構築に時間、後の価格上げが困難 |
| **事例** | Nintendoゲーム機の初期投入、通信キャリアの新規サービス |

#### 選択の判断基準

| 要因 | スキミング有利 | ペネトレーション有利 |
| --- | --- | --- |
| **顧客層** | 上流層・アーリーアダプター | 大衆層・価格敏感層 |
| **製品** | 明確な差別化・高品質 | 汎用的・コモディティ化傾向 |
| **競合** | 参入が遅い、技術障壁高い | 参入が容易、模倣容易 |
| **コスト** | 高いが、販売数少で許容 | 低くできる、大量生産で実現 |
| **市場** | 小さいが利益率重視 | 大きく、占有率重視 |

### 心理的価格設定

実際の価格設定では、単なる計算式ではなく、消費者の心理が大きく働きます。

#### 端数価格（Odd Pricing）

**定義**：`9` や `98` で終わる価格（例：198円、1,980円）を設定することで、顧客に「安い」という心理的錯覚を起こす

**原理**：
- `200円` よりも `198円` のほうが、視覚的・心理的に「安い」と感じさせる
- 左側の最初の数字に注意が集まり、下一桁が目に入りにくい

**用途**：小売店、流通チャネル全般

**注意**：高級品や差別化製品には逆効果。「安っぽい」と見なされることもある。

#### 名声価格（Prestige Pricing / Premium Pricing）

**定義**：商品の高い価格そのものが、品質や希少性の象徴となり、顧客の購買欲求を刺激する価格設定

**原理**：
- 高い価格 = 品質が高い、ステータスシンボル
- 低い価格では「出来が悪い」と思われる可能性
- 富裕層や地位重視層にとって、高価格が購買動機

**例**：ルイヴィトン、フェラーリ、高級シャンパン

**注意**：価格を下げると、かえって販売が減る「ヴェブレン効果」も存在

#### 慣習価格（Customary Pricing）

**定義**：過去の慣習や期待から、「この商品はこのくらいの価格」という消費者の心理的基準を尊重する価格設定

**例**：
- コンビニのコーヒー = 100円
- スーパーの牛乳パック = 180～200円
- 映画館 = 1,800円

**特徴**：
- 価格を変えるとき、説明や価値訴求が不可欠
- 据え置きが長いほど、顧客の「当たり前価格」が固定化

#### 均一価格と段階価格

| タイプ | 定義 | 例 |
| --- | --- | --- |
| **均一価格** | どこで、いつ買っても同じ価格 | チェーン店の商品、ブランド品 |
| **段階価格** | 購買量・季節・時間帯で価格が異なる | スーパーのセール、飲食店のランチセット |

---

### 特殊な価格戦略

#### キャプティブ価格（Captive Pricing）

**定義**：主製品（本体）は低価格・低利益で販売し、付属品・消耗品・メンテナンスで利益を回収する戦略

**例**：
- インクジェットプリンタ：本体安い → インクカートリッジで高利益
- ゲーム機：本体低価格 → ゲームソフトで利益
- シェーバー：本体安い → 替え刃で利益
- コーヒーマシン：本体安い → 専用カプセルで利益

**メリット**：
- 初期購買のハードルが低い（本体安い）
- 継続的な収益源（消耗品・サービス）
- 顧客ロック・イン効果

**デメリット**：
- 消耗品の高さが顧客不満になるリスク
- 競合のサードパーティー製品で迂回されるリスク

#### ロスリーダー価格（Loss Leader Pricing）

**定義**：限定的に商品を原価以下で販売（赤字）して集客し、他の商品の売上で全体利益をカバーする戦略

**例**：
- スーパーの目玉商品：卵、牛乳を安く → 店舗来店客数増加 → 他商品購買
- 家電量販店の型落ちモデル

**メリット**：
- 来店客数・トラフィック大幅増加
- 同点購買の他商品で利益補填
- ブランド認知・好感度向上

**デメリット**：
- 赤字商品が多すぎると全体利益悪化
- 目玉商品の品薄・入手困難で顧客不満
- 顧客が目玉だけ買って帰るリスク（チェリーピッキング）

#### バンドル価格（Bundle Pricing）

**定義**：複数の商品をセット販売し、単品購買より安い価格を提供することで、顧客価値を高める戦略

**例**：
- Office Suite：Word + Excel + PowerPoint のセット
- 携帯電話：本体 + 通話プラン + データプランのセット
- 映画館：ポップコーン + ドリンク + チケット割引セット

**メリット**：
- 複数商品の売上向上
- 顧客が感じる総価値増加（セット割引）
- 低採算商品の売上向上（高採算商品に抱き合わせ）

**デメリット**：
- セット内容が顧客ニーズと一致しないリスク
- 単品の価格低下圧力

#### ダイナミックプライシング（Dynamic Pricing）

**定義**：需要と供給、タイミング、顧客属性などの条件に応じて、リアルタイムに価格を変動させる戦略

**例**：
- 航空運賃：時間帯、曜日、搭乗日までの日数で変動
- ホテル宿泊料：繁忙期・閑散期で変動
- ウーバーイーツ：天候、時間帯、需要で配達料金変動
- コンサート・イベントチケット：発売タイミング、残席数で変動

**メリット**：
- 供給を最大限活用（空席の利益化）
- 需要・供給バランスの柔軟な調整
- 全体収益の最大化

**デメリット**：
- 顧客に不公正感（同じ商品で異なる価格）
- システム構築・管理の複雑性・コスト
- 顧客情報と価格を結びつけると、倫理的な批判も

---

## 典型的なつまずき

- **製品の「実体」だけを見る**
  → 中核のベネフィット（顧客が本当に欲しい価値）を無視すると、競合との差別化ができない

- **製品ライフサイクルを単なる「売上曲線」と思う**
  → 各段階で市場構造、競合状況、顧客ニーズが根本的に変わる。それぞれに合わせた4P戦略を切り替える必要がある

- **ブランド拡張を「ブランド名を流用すること」と誤解**
  → 拡張先カテゴリーとの親和性が低いと、親ブランドまで傷つく。4象限で事前検証が必須

- **「価格 = コスト + 利益」と固く考える**
  → 顧客の支払い意思度、競合価格、市場需要を無視すれば、稼げない。むしろ需要志向や心理的価格設定が売上に直結

- **スキミング vs ペネトレーションを「高い・低い」だけで判定**
  → 新製品の差別化度、競合参入速度、顧客層、製造コストなどの複合要因で選ぶ。単一で決まらない

- **心理的価格設定を「小売店の細かい工夫」と軽視**
  → 実務では、端数価格、名声価格、慣習価格がブランド戦略と結びつき、利益率に大きく影響

---

## 問題を解くときの観点

- **「何が問われているのか」を分類する**：製品戦略か、価格戦略か、それともブランド戦略か
- **製品ライフサイクルの段階を確認する**：導入期と成熟期では施策が全く異なる
- **顧客視点を忘れない**：製品の3層構造で、「顧客が何を求めているか」を先に整理
- **ブランド拡張の妥当性を判定**：4象限で、拡張先カテゴリーとの親和性を論拠で説明
- **価格設定の論理を追う**：コスト・需要・競争のどの視点が優先されているか、その理由は何か
- **スキミングとペネトレーションの選択理由を考える**：単に「高い・低い」でなく、市場状況・競合・製品特性で判定

## 確認問題

<Callout title="問1：製品ライフサイクルと4P戦略">
あるメーカーが、数年前に発売した高機能スマートフォンが、現在、多くの競合製品に取り囲まれている状況です。

（1）このスマートフォンはPLCのどの段階にあるか。
（2）この段階で想定される特徴（売上、利益、競合数、顧客ニーズ）を簡潔に述べよ。
（3）採るべき価格戦略と、その理由を述べよ。

**解答例**：
（1）成熟期
（2）売上が横ばい～減少傾向、利益率が低下、競合が多い、差別化が難しい段階
（3）低価格戦略、またはセグメント別の差別化価格設定。理由は、競争激化による値下げ圧力が強く、コスト競争力か機能の絞り込みで対抗する必要があるため。
</Callout>

<Callout title="問2：ブランド拡張の妥当性判定">
化粧品メーカーAが、フェイスケア（スキンケア）で確立された強いブランド「BeautyX」を、日用雑貨・家庭用洗剤市場に拡張しようとしています。

（1）この拡張は、4象限のどれに該当するか。
（2）この拡張の成功可能性を、ブランドイメージの親和性から判定せよ。
（3）もし拡張を実行する場合、親ブランドへのリスクを最小化する方策を1つ提示せよ。

**解答例**：
（1）ブランド拡張（既存ブランド名 × 新カテゴリー）
（2）成功可能性は低い。理由は、フェイスケア＝「肌の美しさ」のイメージが強く、洗剤＝「汚れ落とし」という機能的・実利的なイメージとの親和性が薄い。消費者は「BeautyXなら洗剤も美容成分がある」と期待するが、その期待を満たす差別化が難しい。
（3）新ブランド「BeautyX Clean」など、親ブランドと区別しつつ連想させるサブブランドを立てることで、親ブランドイメージの毀損を防ぐ。
</Callout>

<Callout title="問3：新製品の価格戦略の選択">
テクノロジー企業Bが、革新的な音声AI搭載イヤフォンを開発しました。競合品は高機能品でも3万円程度ですが、Bの製品は業界初の機能を3つ搭載しており、明らかに差別化されています。ただし、大手メーカーも2年以内に類似品投入を予定しているという情報があります。

（1）スキミング価格とペネトレーション価格のどちらを採るべきか、判定理由を述べよ。
（2）選択した価格戦略における、時間経過に伴う価格調整の方向性を述べよ。

**解答例**：
（1）スキミング価格を採るべき。理由は、①業界初の明確な差別化があり、顧客の支払い意思度が高い、②初期段階での高利益率で開発投資を急速回収する必要がある（競合参入前）、③高級・プレミアム路線で顧客ロイヤルティを構築できるため。
（2）導入期は4～5万円程度で設定し、成長期から競合参入に応じて段階的に値下げ。ただし、独自機能の価値訴求を続けることで、無制限な値下げは避け、セグメント別価格設定（高機能版は高価格、基本版は低価格）へ移行。
</Callout>
