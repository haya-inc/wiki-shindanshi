## イメージをつかむ

駅前のラーメン激戦区で生き残るには、何が必要でしょうか？

- **敵対関係が激しい** → 多くのライバル店が密集していて、すぐに値下げ競争になる
- **新規参入が容易** → 資本があれば誰でも店を出せる（参入障壁が低い）
- **代替品が多い** → ラーメンの代わりに、うどん、つけ麺、カレーがある
- **買い手に交渉力がある** → 客は「この店じゃなくてもいい」と思うから、値下げに応じる

このとき、業界全体の収益性は低い。では、生き残るには？

- **差別化で専門化する** →「濃厚とんこつ専門」「鶏白湯スープ」など、他店と違う特色を持つ
- **特定客層に集中する** → 朝は労働者向け、昼はOL向けなど、ターゲットを絞る
- **活動の連鎖を整える** → 仕入れ→調理→提供までのすべての活動が「差別化」に向いている

これが **5フォース × 3基本戦略** の関係です。

## 学習のポイント

- `5フォース` は業界構造（外部環境）
- `3つの基本戦略` は競争優位の源泉（勝ち筋の選択）
- `バリューチェーン` は活動の連鎖（内部の強み）
- `VRIO` は資源の持続的競争優位（内部資源の評価）

## 試験で何が問われるか

- 5フォースの各力を強める要因を列挙できるか
- コストリーダーシップ・差別化・集中の使い分けができるか
- VRIOフレームワークで持続的競争優位を **判定** できるか
- 先発優位と後発優位の源泉を説明できるか

## ポーターの5フォース分析

5つの力が **強い** ほど、業界の収益性は **低い** 。言い換えれば、5フォース分析は「業界がどれだけ儲けにくいか」を測定するツールです。

### 「なぜ」5フォース分析が重要か

業界の収益性が低い → 企業が得られるマージンが小さい → 競争優位がないと生き残れない。つまり、5フォースの結果によって「自社がどんな戦略を取るべきか」が方向づけられるのです。

- **5フォースが弱い業界** （医療機器、特殊素材など）→ 大手も中小も利益率が高い → コストリーダーシップより差別化戦略で十分
- **5フォースが強い業界** （コンビニ、サプライチェーン下流など）→ 全体的に利益率が低い → 低コスト体質が必須

### 各力と強まる要因

| 力 | 強まる要因 | 業界例 |
| --- | --- | --- |
| **既存業者間の敵対関係** | 競争業者が多数/同規模、成長率低い、固定費高い、差別化困難、撤退障壁高い | コンビニ、ガソリンスタンド、ファストフード |
| **新規参入の脅威** | 参入障壁が低い（規模の経済なし、ブランドなし、必要投資額小、スイッチングコスト低い） | 飲食店、小売店、ネット販売 |
| **代替品の脅威** | 代替品のコストパフォーマンス向上、スイッチングコスト低い | CD → ストリーミング、タクシー → ライドシェア |
| **買い手の交渉力** | 大口購入、標準品、スイッチングコスト低い、後方統合の脅威 | 自動車メーカーと部品製造、大型チェーン小売と仕入先 |
| **売り手の交渉力** | 供給者が少数、代替品なし、前方統合の脅威 | 液晶パネルメーカー対テレビメーカー、石油企業 |

### HHI（ハーフィンダール・ハーシュマン指数）

市場の集中度を測る定量指標。各企業の市場シェア（%）を2乗して合計します。HHI が低いほど「既存業者間の敵対関係」が強く、業界全体の収益性が低くなります。

```
【計算例】
  A社40%、B社30%、C社20%、D社10%
  HHI = 40² + 30² + 20² + 10² = 1600+900+400+100 = 3,000

【判定基準】
  1,500未満 → 非集中的（競争的市場、既存業者間競争が激烈）
  1,500〜2,500 → やや集中的（寡占へ向かう段階）
  2,500超 → 高度に集中的（大手2〜3社が支配）
```

### コンビニ業界の5フォース分析例

**既存業者間の敵対関係：強い**
- セブン-イレブン、ファミリーマート、ローソンなど大手が競争
- 固定費が高い（店舗賃料、オーナー給与、システム利用料）
- 成長率が鈍化している（人口減、少子化）
→ 低価格、便利さ、サービスで競争が激化

**新規参入の脅威：低い**
- フランチャイズシステムの構築が必須（高い投資）
- 大手のブランド・物流ネットワークが強力
- スイッチングコストが高い（オーナーはシステムに依存）
→ 新興プレイヤーの参入が困難

**代替品の脅威：中～強い**
- Amazon等オンラインストアへのシフト
- ドラッグストア（コンビニと同じ立地で便利さを提供）
- スーパー、デパ地下の便利商品充実
→ 来客数が徐々に減少

**買い手の交渉力：中程度**
- 個人顧客が主体（スイッチングコストは低い）
- ただし通勤ルート沿いの立地では固定客がいる
→ 定価販売を原則としながらも、キャンペーン競争は活発

**売り手の交渉力：強い（大手チェーン側）**
- 仕入先は多数（弱い立場）
- ただしプライベートブランド（PB）拡大でコスト改善を進行中
→ 仕入先に価格低下を強要する力がある

## ポーターの3つの基本戦略

| 戦略 | 競争優位の源泉 | ターゲット | 成功条件 |
| --- | --- | --- | --- |
| **コストリーダーシップ** | 低コスト | 業界全体 | 規模の経済、経験曲線、垂直統合、効率的物流 |
| **差別化** | 独自性 | 業界全体 | ブランド、技術、サービス、品質差別化 |
| **集中** | 低コスト or 独自性 | **特定セグメント** | セグメント内のニーズ深掘り、参入障壁構築 |

### コストリーダーシップの実装パターン

- **規模の経済** → 生産量が増えると、単位あたりコストが低下（固定費の按分効果）
- **経験曲線** → 製造回数を重ねると、作業効率が上がり、コストが低下（80%の学習効果：売上が2倍になると、単位コストが20%低下）
- **垂直統合** → 仕入先や流通を自社化して、中間マージンを排除
- **効率的な物流・調達網** → サプライチェーン全体を最適化

**例**：ユニクロはアジア生産、垂直統合、大量仕入れで低コスト化を実現。

### 差別化の実装パターン

- **品質差別化** → 耐久性、精度、安全性で競争
- **ブランド差別化** → ロゴ、イメージ、顧客心理（ルイ・ヴィトン、アップル）
- **サービス差別化** → 配送、カスタマイズ、保証、アフターサービス
- **技術差別化** → 特許技術、革新的機能

**例**：BMWはエンジン技術とブランドで差別化。スターバックスは店舗体験とブランドで差別化。

### 集中戦略

特定のセグメント（顧客層、地域、製品カテゴリ）に経営資源を集中し、そのセグメント内でコストリーダーか差別化かのいずれかを実現します。中小企業は通常、この戦略が現実的です。

- **コスト集中** → 低価格ニッチ（例：業務用スーパー）
- **差別化集中** → プレミアムニッチ（例：オーダーメイドスーツ専門店）

### スタック・イン・ザ・ミドルと具体例

コストリーダーシップと差別化のどちらも中途半端な状態。収益性が最も低下します。

**例**：
- かつての日本の家電メーカーの一部 → 中国勢より安くはない、でも高級ブランドでもない → 市場競争力喪失
- 中価格帯の衣料品メーカー → ユニクロより安くない、でもプラダより高級感もない → 存在意義が不明確

**脱出方法**：
1. 徹底的なコスト削減に舵を切る（コストリーダーシップへ）
2. または高級化・プレミアム化する（差別化へ）
3. または特定セグメントに集中（集中戦略へ）

中小企業は経営資源が限られるため、**差別化集中戦略** が基本です。業界全体との競争は避け、「自社が圧倒的に強いセグメント」を作ります。

### 競争地位別戦略（コトラー）

各競争地位によって、取るべき戦略は異なります。「自社がどの立場にいるのか」を正確に把握することが重要です。

| 地位 | 市場シェア | 目標 | 主な戦略（具体的な打ち手） |
| --- | --- | --- | --- |
| **リーダー** | 最大シェア（トップ） | 地位の維持・拡大 | **同質化戦略**（チャレンジャーの差別化商品を素早く模倣する）、周辺需要拡大（新たな用途開発）、非価格対応（品質・サービス向上で価格競争を避ける） |
| **チャレンジャー** | 2〜3番手 | リーダー攻略 | **差別化**（リーダーと異なる独自商品・サービス）、フランキング攻撃（リーダーの隙間を突く）、コスト優位（コストリーダーシップ）、市場セグメント再定義（新しい顧客層開拓） |
| **フォロワー** | 下位（3番手以下） | 安定経営 | **模倣戦略**（上位企業の成功ノウハウを後追い）、地域限定展開（全国ではなく特定地域で強化）、コスト低減（上位より安く提供） |
| **ニッチャー** | 小さいが独特なセグメント | セグメント内支配 | **集中・専門化**（特定顧客層・製品分野で圧倒的強み）、参入障壁構築（模倣困難な技術・ブランド・顧客関係を作る）、ミニリーダー化（そのセグメントではリーダー的地位） |

#### 戦略の具体例

**リーダー戦略の例**：
- セブン-イレブンが「24時間営業」の標準を作り、他社も追随 → 同質化
- 新しい商品カテゴリ（おでん、チキン）を開発して市場を拡大 → 周辺需要拡大
- 他社より高い品質・サービス水準を保つ → 非価格対応

**チャレンジャー戦略の例**：
- ファミリーマートが「プライベートブランド強化」でセブンと差別化 → 差別化
- ローソンが「地方展開」で隙間を突く → フランキング攻撃

**フォロワー戦略の例**：
- 大手スーパーのノウハウを地方で展開
- 他社より数円安くすることで客を引きつける

**ニッチャー戦略の例**：
- カフェチェーンが「高級志向の20代向け」に特化（スターバックス）
- 医療用医器メーカーが「特定の手術用ロボット」に専門化

### 先発優位と後発優位

| | 源泉 | 内容 |
| --- | --- | --- |
| 先発優位 | 技術的リーダーシップ | 経験曲線、特許による優位 |
| 先発優位 | 希少資源の先制確保 | 立地、チャネル、棚スペース |
| 先発優位 | スイッチングコスト | ブランドの結びつき、学習コスト |
| 後発優位 | フリーライダー効果 | 先発者の市場教育コストへのただ乗り |
| 後発優位 | 先発者の失敗からの学習 | 低リスクでの参入 |
| 後発優位 | 技術的陳腐化への対応 | 最新技術での参入 |

## バリューチェーン（価値連鎖）分析

自社の活動全体を「連鎖」として捉え、どの活動で強み・弱みを持つか、どの活動が差別化に貢献しているかを明らかにします。

```
【支援活動】（全主活動を支える）
  企業インフラストラクチャー（経営企画、財務、法務）
  人事・労務管理（採用、育成、給与体系）
  技術開発（R&D、プロセス改善、製品開発）
  調達活動（仕入れ、サプライチェーン管理）

【主活動】（顧客に直接価値を提供）
  購買物流 → 製造 → 出荷物流 → マーケティング・販売 → サービス → マージン
```

### 各活動の意味と強み・弱みの例

**購買物流**：原材料・部品の調達・管理
- 強み例：グローバルな調達ネットワーク、長期契約による低コスト仕入れ
- 弱み例：納期遅延、不良率が高い

**製造**：製品の製造・加工
- 強み例：自動化投資、熟練技能者の蓄積、品質管理体制
- 弱み例：コストが高い、新技術への対応が遅い

**出荷物流**：製品の配送・在庫管理
- 強み例：独自の物流網、配送速度が速い、在庫管理が効率的
- 弱み例：配送費が高い、納期対応が遅い

**マーケティング・販売**：営業、広告、チャネル管理
- 強み例：ブランド力、営業力、流通チャネル数が多い
- 弱み例：広告費が高い、チャネル支配力が弱い

**サービス**：アフターサービス、保証、メンテナンス
- 強み例：迅速な対応、顧客満足度が高い、保証期間が長い
- 弱み例：対応が遅い、費用がかかる

**支援活動の強み・弱み例**：
- 技術開発が強い → 新製品開発が早い → 差別化源泉
- 人事管理が強い → 優秀な人材が定着 → 組織の継続性
- 調達力が強い → コストが低い → コストリーダーシップ源泉

### 「フィット」（活動間の整合性）の重要性

各活動がそれぞれ独立しているのではなく、**全体として一つの目標に統一されている** ことが競争優位を強化します。

**例1：差別化戦略の場合**
- マーケティング：高級イメージを訴求
- 製造：手作業、高品質基準を維持
- サービス：丁寧なアフターサービス
- 価格設定：プレミアム価格
→ すべての活動が「高級感」に向いている（フィットしている）

**例2：コストリーダーシップの場合**
- 購買物流：大量仕入れで単価を低下
- 製造：自動化、効率化
- 出荷物流：低コスト物流業者の活用
- マーケティング：最小限の広告費
- 価格設定：最安値で販売
→ すべての活動が「低コスト」に向いている

**フィットが欠ければ**：
- 高品質製品を安く売ろうとする → 採算が合わない
- 大量生産で高級感を演出しようとする → 信頼性がない

## 戦略グループと移動障壁

同一業界内でも、企業は異なる戦略を取っています。**戦略グループ** はこれを整理するフレームワークです。

### 戦略グループとは

業界内で **類似した戦略を取っている企業の集団** 。

**例：自動車業界**
- グループA：トヨタ、ホンダ → 品質・信頼性で差別化、中価格帯
- グループB：ポルシェ、フェラーリ → 高性能・高級感で差別化、高価格帯
- グループC：シナ・インドの低価格メーカー → 低コスト、低価格帯

### 移動障壁とは

企業が **ある戦略グループから別のグループへ移行することが難しい理由** 。

**移動障壁の例**：

| 障壁 | 具体例 |
| --- | --- |
| **投資要件** | 高級ブランドへの転換には、ブランド刷新に膨大な投資が必要（例：トヨタ→レクサス） |
| **ブランド資産** | 「安い」イメージが定着すると、高級化は困難（例：ユニクロが高級線を作りにくい） |
| **技術・スキル** | 低コスト製造と高品質製造は全く異なる技術体系→転換に時間がかかる |
| **流通チャネル** | 安売り流通（ドンキホーテ）と高級流通（百貨店）は全く異なる→チャネル再構築が必要 |
| **顧客心理** | かつてのイメージを払拭するのに数年～10年かかる |

### コンビニ↔ドラッグストア間の移動障壁の例

**コンビニ→ドラッグストアへの転換を考える**：
- 従来の「24時間営業、夜間客多数」→ 「日中の医薬品・健康商品販売」への転換
- **移動障壁**：
  - 調達ネットワーク全て変わる（医薬品卸との新規契約、医薬品管理システム構築）
  - 店員の教育・資格（薬剤師、医薬品知識が必須）
  - ブランド転換（「夜間の便利さ」から「昼間の健康商品」への認識転換）
  - 立地戦略の変更（駅前24時間型→商住地域日中型）
→ 実際には転換事例が少ない（参入障壁が高い）

## RBV と VRIO フレームワーク

### RBV（資源ベース理論）vs ポジショニング学派

**ポーター（ポジショニング学派）**：
- 「競争優位は業界構造で決まる」
- 5フォースが弱い業界に参入せよ
- 外部環境を重視

**バーニー（RBV・資源ベース理論）**：
- 「競争優位は内部資源で決まる」
- 同じ業界でも、優れた資源を持つ企業が勝つ
- 内部資源を重視

**実際には両者は補完的**：
- 5フォースが強い業界でも、独自な資源があれば利益が出せる（例：ユニクロ、スターバックス）
- 5フォースが弱い業界でも、資源が弱いと利益が出ない

つまり、**業界を選んだ後、その中で独自資源を作る** ——これが実務の姿勢です。

### ダイナミック・ケイパビリティ（ティース）

環境変化に対応する能力。単なる資源所有ではなく、**資源を組み替え・進化させる能力** が競争優位の源泉だという考え方。

- 例：スマートフォン登場時に、各メーカーの対応が異なった
  - 成功企業（Apple）：既存技術を組み替え、新製品開発プロセスを作り直した → ダイナミック・ケイパビリティあり
  - 失敗企業（ノキア）：従来のプロセスを変えられなかった → ケイパビリティなし

### VRIO フレームワーク

| 基準 | 問い |
| --- | --- |
| V（Value） | 経営資源は価値があるか？（機会活用 or 脅威中和） |
| R（Rareness） | その資源は希少か？ |
| I（Imitability） | 模倣が困難か？ |
| O（Organization） | 組織的に活用できているか？ |

### VRIO の評価結果

| V | R | I | O | 競争上の帰結 |
| --- | --- | --- | --- | --- |
| × | − | − | − | 競争劣位 |
| ○ | × | − | − | 競争均衡 |
| ○ | ○ | × | − | 一時的競争優位 |
| ○ | ○ | ○ | × | 優位を活用できていない |
| ○ | ○ | ○ | ○ | **持続的競争優位** |

### 模倣困難性の4つの要因

| 要因 | 内容 |
| --- | --- |
| 独自の歴史的条件 | 長年の経験の蓄積 |
| 因果関係の曖昧性 | 競争優位の源泉が外部から分かりにくい |
| 社会的複雑性 | 組織文化、人間関係等の複雑な要因 |
| 特許 | 法的に保護された技術 |

## 典型的なつまずき

- `5フォース` を自社分析だと思ってしまう → 5フォースは **業界構造**（外部環境）の分析
- `差別化` と `集中` を同じものとして扱う → 差別化は業界全体、集中は **特定セグメント**
- `バリューチェーン` と `VRIO` の分析対象が曖昧 → バリューチェーンは **活動の連鎖**、VRIO は **経営資源**
- `RBV` を単に `強み` の言い換えとして覚える → VRIO を満たす資源のみが **持続的競争優位** になる
- リーダーの戦略を「差別化」と答えてしまう → リーダーの戦略は **同質化**（チャレンジャーの差別化を模倣して無効化）

## 問題を解くときの観点

- いま見ているのは `業界全体` か `自社内部` か
- 問われているのは `競争の厳しさ` か `勝ち筋` か `強みの持続性` か
- 強みを確認したいなら、`活動` を見るのか `資源` を見るのか
- 競争地位は リーダー/チャレンジャー/フォロワー/ニッチャー のどれか

## 誤答パターン

<Callout title="パターン1：5フォースを自社分析と間違える">
    「わが社は営業力が弱い」を「5フォースが弱い」と言ってしまう。

    **正しくは**：5フォースは業界全体の構造。営業力は自社の資源（VRIO で評価）。両者は別の枠組み。
</Callout>

<Callout title="パターン2：「差別化」と「集中」を同じもの扱い">
    「ニッチ市場で独自商品を売っている」→ 「それは集中戦略？差別化戦略？」と混同。

    **正しくは**：
    - **集中戦略** = ターゲットが「特定セグメント」か「業界全体」かの区別
    - **差別化/コストリーダーシップ** = 競争優位の「源泉」の区別
    - ニッチで独自商品なら「差別化集中」（テーゲット：特定セグメント、源泉：独自性）
</Callout>

<Callout title="パターン3：「コストが低い＝コストリーダーシップ」と誤解">
    「うちは原価が安い」≠ コストリーダーシップ戦略

    **正しくは**：コストリーダーシップは「業界全体で最も低いコスト」を継続的に維持する戦略。
    原価が安いだけでは不十分。販売・物流・マーケティング全体で同業最低コスト体質を作らねばならない。
</Callout>

<Callout title="パターン4：「スタック・イン・ザ・ミドル」の理解不足">
    「品質も良く、価格も安い」と聞くと「理想的な戦略」と思ってしまう。

    **正しくは**：品質向上にはコストがかかる。「品質も良く、価格も安い」を両立させるには、
    コストリーダーシップ+差別化の両方を実現する必要がある。通常は片方に集中すべき。
</Callout>

## 確認問題

<Callout title="問1：5フォース分析の目的">
    「業界の固定費が高く、差別化が困難で、撤退障壁も高い」場合、5フォースのどの力が強まるか。また、このとき企業が取るべき戦略は何か。

    **解答**：既存業者間の敵対関係が強まる。固定費が高いと稼働率を維持するため値下げ競争が起き、差別化困難で撤退もできないため競争が激化する。

    このような業界では、業界全体での利益率が低いため、中小企業は「差別化集中戦略」で特定セグメントに集中すべき。
</Callout>

<Callout title="問2：バリューチェーン分析">
    「自社の製造技術は競合より優れているが、流通チャネルが弱い」場合、バリューチェーン全体の「フィット」はどのような状態か。

    **解答**：フィット（整合性）が欠けている。製造で差別化しても、流通チャネルが弱いと消費者に届かない。
    差別化戦略を実行するなら、製造+流通+マーケティング全体を「品質差別化」に向ける必要がある。
</Callout>

<Callout title="問3：VRIOの判定">
    「自社の製品開発技術は価値があり希少だが、競合が2〜3年で追いつける」場合、VRIO の評価結果はどれか。

    **解答**：一時的競争優位（V=○、R=○、I=×）。模倣困難性を欠くため、競争優位は一時的にとどまる。
    持続的競争優位にするには、特許取得、継続的な研究開発投資、社会的複雑性（組織文化）を構築する必要がある。
</Callout>

<Callout title="問4：競争地位別戦略">
    「業界最大シェアの企業が、2番手企業の新しい差別化商品と同等の商品を素早く投入した」。この行動は何戦略か。

    **解答**：同質化戦略（リーダーの定石戦略）。チャレンジャーの差別化を模倣して無効化し、シェア優位を維持する。
</Callout>

<Callout title="問5：RBV vs ポーター">
    「同じ飲食業界でも、スターバックスは高利益、他のカフェは薄利」。この差は何か。

    **解答**：ポーター的には「同じ業界だから5フォースは同じはず」だが、RBV的には「スターバックスが優れた資源（ブランド、店舗体験、サプライチェーン）を持つ」と説明する。
    実務的には「競争優位は業界構造（5フォース）と内部資源（VRIO）の両方で決まる」と理解すべき。
</Callout>
