## このページの役割

<Callout title="このページの役割">
    このページは、`企業会計原則` と `連結会計の基礎` を独立に整理する解説ページです。制度や理論を暗記項目としてではなく、`何を防ぎたい仕組みか` の観点で固めます。
</Callout>

<Callout title="このページの読み方">
  親会社が子会社に商品を 100万円で売って利益を計上──しかしグループ全体で見ると、お金は右のポケットから左のポケットに移動しただけです。こうした「身内取引で利益を膨らませる」ことを防ぐのが連結会計です。このページでは、まず企業会計原則の枠組みを理解し、次に会社法と金商法の制度的な違いを確認し、最後に連結会計の消去仕訳と計算方法を学びます。
</Callout>

## 学習のポイント

- 会計原則は `信頼できる財務情報を作るためのルール`
- 会社法と金商法は `何のために作る書類か` を分ける
- 連結会計は `企業集団を1つの会社のように見る` 発想から入る

## 試験で何が問われるか

- 主要な会計原則の意味と目的を説明できるか
- 会社法と金融商品取引法の作成書類の違いを言えるか
- 連結会計の消去仕訳を **計算つき** で処理できるか

## 企業会計原則

企業会計原則は、恣意的な利益操作や不正確な表示を防ぐための基準です。

| 原則 | 何を守るか | 具体例 |
| --- | --- | --- |
| 真実性の原則 | 財務諸表が企業の状況を真実に表示する | 最上位の原則（他のすべての基礎） |
| 正規の簿記の原則 | 正確な帳簿記録に基づく | 網羅性・検証可能性・秩序性 |
| 資本取引・損益取引区分の原則 | 資本と利益を混同しない | 増資は利益に含めない |
| 明瞭性の原則 | 利害関係者に分かりやすく表示する | 注記、区分表示 |
| 継続性の原則 | 正当な理由なく会計処理を変更しない | 期間比較の確保 |
| 保守主義の原則 | 利益は控えめに、損失は早めに認識 | 引当金の計上、低価法 |
| 重要性の原則 | 重要でない項目は簡便な処理を許容 | 消耗品の即時費用化 |

## 会社法と金融商品取引法

| 比較軸 | 会社法 | 金融商品取引法 |
| --- | --- | --- |
| 主な目的 | 債権者保護・配当規制 | 投資家への情報開示 |
| 対象企業 | すべての株式会社 | 上場企業等 |
| 作成書類 | 計算書類（B/S, P/L, 株主資本等変動計算書, 個別注記表） | 財務諸表（B/S, P/L, CF計算書, 株主資本等変動計算書 等） |
| 監査 | 会計監査人（大会社等） | 公認会計士・監査法人 |

大きな違いは、会社法には C/F 計算書の作成義務がないのに対し、金商法では必須である点です。

## 連結会計 ── 企業集団を1つの会社のように見る

### なぜ連結が必要か

親会社と子会社の個別財務諸表をそのまま足し合わせると、内部取引が二重計上されます。連結会計では、これらの内部取引を消去して、企業集団全体の実態を示します。

### 連結の範囲

| 関係 | 基準 | 連結の扱い |
| --- | --- | --- |
| 子会社 | 議決権 50%超、または実質支配 | 全部連結 |
| 関連会社 | 議決権 20%以上、または重要な影響力 | 持分法 |

### 主要な消去仕訳

#### ① 投資と資本の消去

親会社の「子会社株式」と子会社の「資本」を相殺します。差額が **のれん** です。

```
【計算例】
  親会社の子会社株式 800万円
  子会社の純資産 600万円（資本金400＋利益剰余金200）
  持株比率 100%

  のれん = 800 − 600 = 200万円

  （借方）資本金      400 ／ （貸方）子会社株式  800
        利益剰余金  200
        のれん      200
```

のれんは 20年以内で均等償却します。毎年の償却額は P/L の費用になります。

#### ② 内部取引の消去

親会社と子会社の間の売上・仕入を消去します。

```
【計算例】
  親会社が子会社に商品 300万円を販売

  （借方）売上  300 ／ （貸方）売上原価  300
```

#### ③ 未実現利益の消去

グループ内で取引した商品が期末に在庫として残っている場合、その利益はまだ外部に実現していないため消去します。

**ダウンストリーム（親→子への販売）**:

```
【計算例】
  親会社が子会社に原価 100万円の商品を 130万円で販売
  期末に子会社の在庫に残っている

  未実現利益 = 130 − 100 = 30万円
  （借方）売上原価  30 ／ （貸方）商品  30
  → 利益を全額消去（親会社負担）
```

**アップストリーム（子→親への販売）**:

```
【計算例】
  子会社が親会社に原価 80万円の商品を 100万円で販売
  期末に親会社の在庫に残っている。非支配株主持分 20%

  未実現利益 = 100 − 80 = 20万円
  （借方）売上原価  20 ／ （貸方）商品  20
  → 非支配株主持分にも按分
  非支配株主負担 = 20 × 20% = 4万円
```

#### ④ 債権債務の消去

```
親会社の売掛金 50万円 と 子会社の買掛金 50万円
  （借方）買掛金  50 ／ （貸方）売掛金  50
```

<Callout title="ここまでの整理 ── 連結会計のポイント">
  連結会計の消去仕訳は 4 種類（投資と資本、内部取引、未実現利益、債権債務）を押さえれば試験に対応できます。特にのれんの計算と未実現利益の消去は頻出です。ダウンストリーム（親→子）とアップストリーム（子→親）で非支配株主持分の按分が異なる点に注意してください。
</Callout>

## 典型的なつまずき

- 会計原則を名称だけで覚え、何を防ぐかが分からない → 各原則の **目的** を理解する
- 会社法と金商法を `作成書類の暗記` だけで処理する → 「誰を守るための制度か」で分ける
- 連結会計を個別会計の延長として見てしまう → 企業集団 **全体** を1つの実体として見る発想
- のれんの計算で純資産の持分比率を掛け忘れる → 100%子会社でなければ持分比率を乗じる
- ダウンストリームとアップストリームで非支配株主の扱いを混同する → ダウンストリームは全額親会社負担

## 問題を解くときの観点

- その制度は `誰を守るためのものか`
- 比較可能性、信頼性、期間対応のどれを確保したいか
- 単体の会社を見るのか、企業集団全体を見るのか
- 消去仕訳の 4 類型のどれに該当するか

## 確認問題

<Callout title="問1：のれんの計算">
    親会社が子会社株式を 1,200万円で取得。子会社の純資産は 1,000万円。持株比率 100%。のれんの金額と、のれん償却期間が 10年の場合の年間償却額を求めよ。

    **解答**：のれん = 1,200−1,000 = 200万円。年間償却額 = 200÷10 = 20万円。
</Callout>

<Callout title="問2：未実現利益の消去">
    親会社が子会社に原価 200万円の商品を 250万円で販売した。期末に子会社の在庫に全額残っている。消去すべき未実現利益はいくらか。

    **解答**：250−200 = 50万円。ダウンストリームなので全額を消去。
</Callout>

<Callout title="問3：会計原則の判定">
    企業が正当な理由なく減価償却方法を定額法から定率法に変更した。これはどの会計原則に反するか。

    **解答**：継続性の原則。正当な理由のない会計処理の変更は、期間比較可能性を損なうため認められない。
</Callout>
