## このページの役割

このページは、`デリバティブ` を整理する解説ページです。`先物`、`オプション`、`スワップ` を、`どのリスクをどう固定したいか` という観点で切り分けます。

<Callout title="このページの読み方">
  輸入企業が 3 か月後にドルで支払いをする場面を想像してください。今のレートなら 1 ドル = 150 円ですが、3 か月後に 160 円になるかもしれません。この「将来の価格が読めないリスク」に対処するための金融商品がデリバティブです。このページでは、まず「何のリスクを、どういう形で管理したいか」を明確にしてから、先物・オプション・スワップの仕組みと計算方法を順番に見ていきます。
</Callout>

## 学習のポイント

- まず `何のリスクを避けたいか` を先に決める
- 次に `将来価格を固定したいのか`、`不利な方向だけ避けたいのか` を分ける
- そのうえで `先物`、`オプション`、`スワップ` の機能差を整理する

## 試験で何が問われるか

- `先物`、`オプション`、`スワップ` の違いを説明できるか
- 価格変動、為替変動、金利変動のどれをヘッジしたいかで手段を選べるか
- `権利` と `義務` の違いを切って言えるか

---

## まず区別すべきこと ── 義務か権利か

デリバティブの根本的な分かれ目は「取引する **義務** を負うか」「取引する **権利** を得るか」です。

先物・先渡は双方が義務を負います。将来の決められた条件で必ず取引しなければなりません。有利に動いても不利に動いても実行する義務があります。

オプションは買い手に権利を与えます。不利に動けば権利を放棄でき、有利に動けば権利を行使できます。そのかわり、買い手はこの権利を得るために `プレミアム`（オプション料）を支払います。

---

## 先物と先渡 ── 将来の価格を「固定」する

先物と先渡はどちらも「将来の特定日に、あらかじめ決めた価格で売買する」契約です。違いは取引の仕組みにあります。

| 項目 | 先物（Futures） | 先渡（Forward） |
| --- | --- | --- |
| 取引場所 | 取引所（標準化された契約） | 店頭（当事者間の相対取引） |
| 信用リスク | 清算機関が介在するため低い | カウンターパーティリスクあり |
| 決済 | 日次で値洗い（差金決済） | 満期時に決済 |
| 義務 | 売り手も買い手も **義務** を負う | 同左 |

先物は標準化されているため流動性が高く、清算機関が間に入るため相手方の倒産リスクも低いのが特徴です。一方、先渡は条件を自由にカスタマイズできる反面、相手方の信用リスクを負います。

---

## オプション ── 「権利」を買う

オプションは 2 種類あります。

| 種類 | 権利の内容 | 買い手 | 売り手 |
| --- | --- | --- | --- |
| コール | 原資産を **買う権利** | 行使 or 放棄を選択 | 義務（行使に応じる） |
| プット | 原資産を **売る権利** | 行使 or 放棄を選択 | 義務（行使に応じる） |

ヨーロピアン型は満期日のみ行使可能、アメリカン型は満期日までいつでも行使可能です。

### オプションの損益 ── いくら儲かるか、いくら損するか

オプションの損益計算には、行使価格 `K`、原資産価格 `S`、プレミアムの 3 つが登場します。

**コールの買い手** は原資産価格が行使価格を上回ったときに行使して利益を得ます。原資産価格が行使価格を下回れば権利を放棄し、損失はプレミアムだけです。

$$
\text{コール買いの利益} = \max(S - K,\ 0) - \text{プレミアム}
$$

$$
\text{損益分岐点} = K + \text{プレミアム}
$$

**プットの買い手** は原資産価格が行使価格を下回ったときに行使します。

$$
\text{プット買いの利益} = \max(K - S,\ 0) - \text{プレミアム}
$$

$$
\text{損益分岐点} = K - \text{プレミアム}
$$

買い手の最大損失はプレミアム（権利料）に限定されます。一方、コールの売り手の最大損失は理論上無限大です。

### 計算例

```
【データ】コールオプション。行使価格 1,000円、プレミアム 80円。

ケース1: 原資産価格が 1,200円
  利益 = (1,200−1,000) − 80 = 120円

ケース2: 原資産価格が 1,050円
  利益 = (1,050−1,000) − 80 = −30円（行使するが、プレミアムの方が大きい）

ケース3: 原資産価格が 900円
  権利行使しない。損失 = プレミアム 80円（最大損失）

損益分岐点 = 1,000 + 80 = 1,080円
```

### オプションの価値 ── 本質的価値と時間的価値

$$
\text{オプション価値} = \text{本質的価値} + \text{時間的価値}
$$

本質的価値は「今すぐ行使したら得られる金額」、時間的価値は「満期までに状況が好転する可能性への対価」です。

| 状態 | コール | プット |
| --- | --- | --- |
| ITM（イン・ザ・マネー） | `S > K` | `S < K` |
| ATM（アット・ザ・マネー） | `S = K` | `S = K` |
| OTM（アウト・オブ・ザ・マネー） | `S < K` | `S > K` |

### オプション価格に影響する要因

| 要因 | コール | プット |
| --- | --- | --- |
| 原資産価格↑ | ↑ | ↓ |
| 行使価格↑ | ↓ | ↑ |
| 残存期間が長い | ↑ | ↑ |
| ボラティリティ↑ | ↑ | ↑ |
| 金利↑ | ↑ | ↓ |

ボラティリティ（価格変動の大きさ）と残存期間は **コール・プット両方を上昇** させます。不確実性が高いほど、大きく有利に動く可能性も高まるからです。

### プット＝コール・パリティ

同じ原資産・同じ行使価格・同じ満期のコールとプットの間には、次の関係が常に成り立ちます。

$$
C + PV(K) = P + S
$$

$$
PV(K) = \frac{K}{(1+r)^t}
$$

この関係を使えば、3 つの値が分かれば残り 1 つを求められます。

```
【計算例】S = 1,000円、K = 1,050円、r = 5%、t = 1年、C = 80円

PV(K) = 1,050 / 1.05 = 1,000円
P = C + PV(K) − S = 80 + 1,000 − 1,000 = 80円
```

<Callout title="ここまでの整理 ── 先物とオプションの使い分け">
  将来価格を「確実に固定」したいなら先物・先渡（ただし有利な方向の利益も放棄）。「不利な方向だけ避けて、有利な方向の利益は残したい」ならオプション（ただしプレミアムを支払う）。次はオプション価格の理論的な求め方と、金利リスクに対処するスワップを見ていきます。
</Callout>

## 二項モデル ── オプション価格を理論的に求める

二項モデルは、原資産価格が次の期間に「上昇」か「下降」の 2 パターンのみに動くと仮定し、リスク中立確率を使ってオプション価格を求める方法です。

$$
p = \frac{(1+r) - d}{u - d}
$$

$$
C = \frac{p \times C_u + (1-p) \times C_d}{1+r}
$$

`u` は上昇倍率、`d` は下降倍率、`Cu` と `Cd` はそれぞれ上昇時・下降時のオプション価値です。

```
【計算例】S = 100、u = 1.2、d = 0.9、K = 105、r = 5%

手順1: 上昇・下降時の原資産価格
  Su = 100 × 1.2 = 120、Sd = 100 × 0.9 = 90

手順2: 上昇・下降時のコール価値
  Cu = max(120−105, 0) = 15
  Cd = max(90−105, 0) = 0

手順3: リスク中立確率
  p = (1.05−0.9) / (1.2−0.9) = 0.15/0.3 = 0.5

手順4: コール価格
  C = (0.5×15 + 0.5×0) / 1.05 = 7.5/1.05 ≒ 7.14円
```

## スワップ ── キャッシュフローを「交換」する

スワップは、異なる条件のキャッシュフローを当事者間で交換する契約です。元本そのものは動かさず、利息などの支払いだけを交換します。

| 種類 | 内容 | 活用場面 |
| --- | --- | --- |
| 金利スワップ | 固定金利と変動金利の交換 | 金利変動リスクのヘッジ |
| 通貨スワップ | 異なる通貨の元本・利息の交換 | 為替リスクのヘッジ |

### 金利スワップの具体例

```
A社：変動金利で借入中（TIBOR+1%）→ 金利上昇リスクあり
B社：固定金利で借入中（3%）→ 金利低下の恩恵なし

スワップ契約：A社がB社に固定3%を支払い、B社がA社にTIBOR+1%を支払う
→ A社は実質的に固定金利3%の借入に変換（金利変動リスクを回避）
→ B社は実質的に変動金利の借入に変換（金利低下時の恩恵を得る）
```

スワップのポイントは、既存の借入を変更するのではなく、キャッシュフローの交換によって **実質的に** 条件を変えることです。元本は移動しません。

## リアルオプション ── 投資判断にオプションの考え方を使う

NPV がマイナスでも、状況を見てから判断できる「柔軟性」に価値があるという考え方です。

$$
\text{拡張NPV} = \text{従来のNPV} + \text{リアルオプションの価値}
$$

| 種類 | 内容 | 対応する金融オプション |
| --- | --- | --- |
| 延期オプション | 投資実行を遅らせる権利 | コール |
| 拡張オプション | 成功時に追加投資する権利 | コール |
| 撤退オプション | 失敗時に中止する権利 | プット |

---

## 為替リスクのヘッジ手法

| 手法 | 内容 |
| --- | --- |
| 為替予約 | 将来の特定日に一定レートで外貨を売買する契約 |
| 通貨オプション | 一定レートで外貨を売買する **権利** を取得 |
| リーズ・アンド・ラグズ | 為替見通しに応じて決済時期を前倒し・後ろ倒し |
| マリー | 外貨建て債権と債務を同額にして相殺 |
| ネッティング | グループ内の外貨建て債権債務を相殺 |

## 問題を解くときの観点

- まず `何が上がると困るか`、`何が下がると困るか` を確認する
- 有利な方向の利益を `残したい` のか `固定したい` のかを分ける
- `権利` を買うのか `契約義務` を負うのかを切る
- 問われているのが `価格リスク`、`為替リスク`、`金利リスク` のどれかを確認する

## 典型的なつまずき

- `先物` と `オプション` の違いを `義務` と `権利` で切れない → 先物は **双方に義務**、オプションは買い手に **権利**
- 価格上昇リスクを避けたいのに `売りヘッジ` を選んでしまう → 上昇が怖い＝ **買い** ヘッジ
- `スワップ` を `新しい借入` と誤解する → 既存のCFを交換するだけで元本は動かない（金利スワップ）
- オプションの損益で「プレミアム」を忘れる → 利益は行使価値からプレミアムを引いた額
- ボラティリティが高いとコールだけ上がると思う → プットも上がる（両方に有利）
- 二項モデルでリスク中立確率の分子を間違える → `(1+r-d)` が分子

## 確認問題

<Callout title="問1：オプションの損益分岐点">
    プットオプション。行使価格 500円、プレミアム 30円。損益分岐点と、原資産価格 450円のときの利益を求めよ。

    **解答**：損益分岐点 = 500−30 = 470円。利益 = (500−450)−30 = 20円。
</Callout>

<Callout title="問2：二項モデル">
    S = 200、u = 1.3、d = 0.8、K = 220、r = 10%。コールオプション価格を求めよ。

    **解答**：Su = 260、Sd = 160。Cu = max(260−220,0) = 40。Cd = 0。p = (1.1−0.8)/(1.3−0.8) = 0.3/0.5 = 0.6。C = (0.6×40+0.4×0)/1.1 = 24/1.1 ≒ 21.8円。
</Callout>

<Callout title="問3：ヘッジ手段の選択">
    輸入企業が3か月後にドル建て支払いを控えている。円安リスクを回避しつつ、円高時の利益は残したい。最も適切なヘッジ手段はどれか。(a) ドル買い先物 (b) ドルコール・オプションの購入 (c) 金利スワップ。

    **解答**：(b)。ドルコール・オプションを購入すれば、円安時は行使価格でドルを買え（損失を限定）、円高時は権利放棄して市場で安く買える（利益を残せる）。(a) は円高の利益を捨てる。(c) は金利リスクのヘッジ手段。
</Callout>
