## このページの役割

このページは、`経営分析` のうち `収益性` を固めるページです。`売上高利益率`、`ROA`、`ROE` を、式の暗記ではなく `何がどれだけもうかっているか` の言葉で説明できる状態を目指します。

## 学習のポイント

- まず `売上高総利益率`、`売上高営業利益率`、`売上高経常利益率` の違い
- 次に `ROA` と `ROE` の分母の違い
- そのうえで `デュポン分解` で `利益率` と `回転率` と `財務レバレッジ` を切り分ける

## まず固める主要指標

| 指標 | 何を見るか | 高いとどう読みやすいか |
| --- | --- | --- |
| 売上高総利益率 | 売上から仕入や製造原価を引いた粗利の厚さ | 商品力や仕入力が比較的強い |
| 売上高営業利益率 | 本業のもうけの厚さ | 本業の収益性が高い |
| 売上高経常利益率 | 本業と金融収支を含む経常的なもうけ | 財務負担まで含めた収益力が高い |
| ROA | 総資本をどれだけ利益へ変えたか | 資産全体を効率よく使っている |
| ROE | 自己資本をどれだけ利益へ変えたか | 株主資本に対する収益力が高い |

## ROA と ROE の見方

`ROA` は `総資本` を分母に取るため、資産効率も含めて見ます。`ROE` は `自己資本` を分母に取るため、借入を含む資本構成の影響も受けます。

同じ利益率でも、資産が重い企業は `ROA` が下がりやすく、借入依存が高い企業は `ROE` が押し上がることがあります。したがって、`ROE が高い = 無条件によい` ではなく、`何で高くなっているか` を分けて見ることが重要です。

## デュポン分解

`ROE = 売上高当期純利益率 × 総資本回転率 × 財務レバレッジ`

この分解を使うと、同じ `ROE` でも `利益率が高い企業`、`資産回転が速い企業`、`借入を使って押し上げている企業` を区別できます。

| 軸 | 主に見ること | 崩れたときに疑いやすいこと |
| --- | --- | --- |
| 収益性 | 利益率 | 値引き、原価上昇、固定費負担 |
| 効率性 | 回転率 | 在庫滞留、設備遊休、売掛金回収遅れ |
| 財務レバレッジ | 自己資本に対する総資本の厚さ | 借入依存の上昇 |

## 公式一覧

### 売上高利益率

| 指標 | 計算式 |
| --- | --- |
| 売上高総利益率 | 売上総利益 ÷ 売上高 × 100 |
| 売上高営業利益率 | 営業利益 ÷ 売上高 × 100 |
| 売上高経常利益率 | 経常利益 ÷ 売上高 × 100 |
| 売上高当期純利益率 | 当期純利益 ÷ 売上高 × 100 |
| 売上高販管費率 | 販管費 ÷ 売上高 × 100 |

通常の企業では：売上高総利益率 ＞ 営業利益率 ＞ 経常利益率 ＞ 純利益率。

営業利益率 = 総利益率 − 販管費率 → 営業利益率の低下原因を粗利と販管費に切り分けられる。

### 資本利益率

| 指標 | 計算式 | 分子の注意 |
| --- | --- | --- |
| ROA | 利益 ÷ 総資産 × 100 | 問題文の指示優先。指定なしなら経常利益 |
| ROE | 当期純利益 ÷ 自己資本 × 100 | — |
| ROIC | NOPAT ÷ 投下資本 × 100 | NOPAT = 営業利益 ×(1−税率) |

EBITDA = 営業利益 + 減価償却費。設備投資・償却方法・資本構成の差を除いた比較に使う。

### ROA の分解

```
ROA = 売上高利益率 × 総資産回転率
    = (利益 / 売上高) × (売上高 / 総資産)
```

薄利多売型（小売）は利益率↓回転率↑、高付加価値型（製薬・不動産）は利益率↑回転率↓。

### ROE の分解（デュポンシステム）

```
ROE = 売上高純利益率 × 総資産回転率 × 財務レバレッジ
    = (当期純利益/売上高) × (売上高/総資産) × (総資産/自己資本)
        収益性             効率性            財務構造
```

## 計算手順

### デュポン分解の計算例

```
【データ】
売上高: 1,000百万円、当期純利益: 30百万円
総資産: 500百万円、自己資本: 200百万円

手順1: 各要素を求める
  売上高純利益率 = 30 ÷ 1,000 = 3%
  総資産回転率 = 1,000 ÷ 500 = 2.0回
  財務レバレッジ = 500 ÷ 200 = 2.5倍

手順2: 掛け合わせて検算
  ROE = 3% × 2.0 × 2.5 = 15%
  直接計算：30 ÷ 200 × 100 = 15% ✓
```

### 同じ ROE でも中身が違う例

```
A社：ROE = 5% × 2.0回 × 2.0倍 = 20%（効率性型）
B社：ROE = 10% × 1.0回 × 2.0倍 = 20%（収益性型）
C社：ROE = 5% × 1.0回 × 4.0倍 = 20%（レバレッジ型）

→ C社は借入依存が高く、安全性にリスクあり
```

### ROE を改善する3つの方向

- **収益性の向上**：売上高純利益率を上げる（コスト削減、高付加価値化）
- **効率性の向上**：総資産回転率を上げる（不要資産の圧縮、売上拡大）
- **レバレッジの活用**：負債を増やす（ただし過度な借入はリスク増大）

## 問題を解くときの観点

- いま見たいのは `粗利`、`営業利益`、`経常利益`、`最終利益` のどこか
- 分母は `売上高`、`総資本`、`自己資本` のどれか
- 数値差の原因は `利益率` 側か `資産効率` 側か

## 典型的なつまずき

- `ROA` と `ROE` の分母を混同する → ROA は**総資産**、ROE は**自己資本**
- 利益率の高低だけで判断し、資産回転率を見ない → 必ずデュポン分解で分ける
- `ROE` の高さを借入依存の影響まで分けずに評価する → 財務レバレッジが 3 倍超ならリスクを疑う
- ROA の分子で問題文の指定を読み飛ばし、常に経常利益を使ってしまう
- EBITDA を「利益」と混同する → 減価償却を足し戻した**キャッシュフロー近似値**

## 確認問題

<Callout title="問1：デュポン分解">
    売上高 800百万円、当期純利益 24百万円、総資産 400百万円、自己資本 160百万円。ROE をデュポン分解せよ。

    **解答**：純利益率 = 24÷800 = 3%、回転率 = 800÷400 = 2.0回、レバレッジ = 400÷160 = 2.5倍。ROE = 3% × 2.0 × 2.5 = 15%。検算：24÷160 = 15% ✓
</Callout>

<Callout title="問2：ROE の改善要因">
    前期 ROE 12%（純利益率 4%、回転率 1.5回、レバレッジ 2.0倍）→ 当期 ROE 15%（純利益率 5%、回転率 1.5回、レバレッジ 2.0倍）。ROE 上昇の主因はどれか。

    **解答**：回転率・レバレッジは不変。純利益率が 4%→5% に改善したことが主因。収益性の向上（コスト削減や高付加価値化）により ROE が改善。
</Callout>

<Callout title="問3：2社比較">
    X社：ROE=18%（純利益率 6%、回転率 1.0回、レバレッジ 3.0倍）。Y社：ROE=18%（純利益率 9%、回転率 1.0回、レバレッジ 2.0倍）。安全性の観点でどちらが望ましいか。

    **解答**：Y社。X社はレバレッジ 3.0 倍で ROE を押し上げており、負債依存度が高い。Y社は収益性が高くレバレッジも低いため、安全性面で望ましい。
</Callout>
