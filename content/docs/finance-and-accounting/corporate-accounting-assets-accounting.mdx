## 学習のポイント

- 棚卸資産は `取得原価` と `期末評価` の両方を見る
- 固定資産は `取得`、`配分`、`減損 / 売却` を分けて考える
- 低価法、減価償却、減損はすべて `期末で見直す処理` として並べる

## 試験で何が問われるか

- `先入先出法` と `移動平均法` の違いを言えるか
- `減価償却` がなぜ必要かを説明できるか
- `減損`、`売却`、`除却` の違いを区別できるか

## 定義と仕組み

棚卸資産は、まず取得原価で記録し、期末には `実地棚卸` や `低価法` によって評価を見直します。固定資産は、取得原価を使用期間へ配分するために減価償却を行い、価値が大きく落ちたときは減損を検討します。

## 典型例

| 論点 | 見るべきこと | 処理の軸 |
| --- | --- | --- |
| 棚卸資産の評価 | 評価法と低価法 | 期末評価の見直し |
| 減価償却 | 使用期間への配分 | 費用配分と累計額 |
| 固定資産売却 | 帳簿価額との差 | 売却益 / 売却損の判定 |
| 減損 | 将来収益力の低下 | 簿価の切下げ |

## 棚卸資産の評価法と公式

| 方法 | 計算方法 | 物価上昇時の影響 |
| --- | --- | --- |
| 個別法 | 個々の取得原価で評価 | 個別に対応 |
| 先入先出法（FIFO） | 先に取得した在庫から払い出す | 期末在庫↑、売上原価↓、**利益↑** |
| 移動平均法 | 仕入の都度、加重平均単価を再計算 | 中間的な結果 |
| 総平均法 | 期間全体の加重平均単価で計算 | 中間的な結果 |
| 最終仕入原価法 | 期末に最も近い仕入価格で評価 | 実務で簡便 |

後入先出法（LIFO）は 2010 年に日本基準で廃止済み。

**低価法**（強制適用）：

```
期末評価額 = min（取得原価, 正味売却価額）
正味売却価額 = 売却見込額 − 見積追加製造原価 − 見積販売直接経費
```

正味売却価額が取得原価を下回る → **棚卸資産評価損** を計上。

**棚卸減耗損**：帳簿棚卸数量 ＞ 実地棚卸数量のとき発生。

```
棚卸減耗損 = (帳簿数量 − 実地数量) × 帳簿単価
```

### 売上原価の算定公式

```
売上原価 = 期首商品棚卸高 + 当期仕入高 − 期末商品棚卸高
```

期末商品棚卸高が小さいほど売上原価が大きく、利益が小さくなる。

## 棚卸資産の計算手順

### 先入先出法の計算例

```
期首在庫：100個 × @100円 = 10,000円
仕入①  ：200個 × @110円 = 22,000円
仕入②  ：150個 × @120円 = 18,000円
期末実地棚卸数量：130個
正味売却価額    ：@105円

手順1: 払出の仮定を決める
  FIFO → 古い順に払い出す → 残るのは新しいもの

手順2: 期末在庫の取得原価を求める
  帳簿上の在庫数量 = 100 + 200 + 150 − 払出 = ?
  ここでは期末130個が残る想定
  仕入②から 130個 × @120円 = 15,600円

手順3: 低価法を適用する
  正味売却価額 = 130 × 105 = 13,650円
  min(15,600, 13,650) = 13,650円
  → 棚卸資産評価損 = 15,600 − 13,650 = 1,950円
```

### 移動平均法の計算例

```
期首在庫：100個 × @100円
仕入①  ：200個 × @110円

仕入①時点の平均単価：
  (100×100 + 200×110) ÷ (100+200)
  = (10,000 + 22,000) ÷ 300
  = @106.67円

仕入②  ：150個 × @120円（仕入①後に150個払出済みとする）
残数 = 300 − 150 = 150個
  平均単価 = (150×106.67 + 150×120) ÷ (150+150)
  = (16,000 + 18,000) ÷ 300
  = @113.33円
```

ポイント：移動平均法は仕入の**都度**再計算するため、問題文の時系列を追う。

## 減価償却の公式と計算手順

| 方法 | 公式 | 特徴 |
| --- | --- | --- |
| 定額法 | `(取得原価 − 残存価額) ÷ 耐用年数` | 毎期均等 |
| 200%定率法 | `期首帳簿価額 × 償却率` | 初期に多く計上。償却率 = 定額法償却率 × 200% |

**期中取得の場合**：月割計算

```
減価償却費 = 年間償却費 × 使用月数 ÷ 12
```

### 減価償却の計算例

```
取得原価 1,200万円、耐用年数 6年、残存価額 0、定額法
取得日：X1年10月1日、決算日：X2年3月31日

年間償却費 = 1,200 ÷ 6 = 200万円
X1年度の使用月数 = 10月〜3月 = 6か月
X1年度の減価償却費 = 200 × 6/12 = 100万円
```

## 減損会計の4段階

| 段階 | 内容 | 判定基準 |
| --- | --- | --- |
| ①グルーピング | CF を生む最小単位を識別 | 工場・店舗単位等 |
| ②兆候の把握 | 減損の可能性を示す事象の有無 | 継続赤字、市場価格の著しい下落 |
| ③認識の判定 | 減損を計上すべきか | **割引前**将来 CF ＜ 帳簿価額 → 認識 |
| ④測定 | 金額の算定 | 減損損失 = 帳簿価額 − 回収可能価額 |

**回収可能価額** = max（正味売却価額, 使用価値）

- 正味売却価額 = 時価 − 処分費用見込額
- 使用価値 = 将来 CF の**割引**現在価値

減損損失は**戻入れしない**（日本基準）。IFRS では戻入れ可能。

### 減損の計算例

```
工場の帳簿価額：5,000万円
割引前将来CF合計：4,500万円 → 帳簿価額を下回る → 減損を認識

正味売却価額 = 4,000 − 500 = 3,500万円
使用価値（割引後CF） = 3,800万円
回収可能価額 = max(3,500, 3,800) = 3,800万円

減損損失 = 5,000 − 3,800 = 1,200万円
仕訳：（借）減損損失 1,200 ／（貸）建物等 1,200
→ P/L の特別損失に計上
```

## 無形固定資産と繰延資産

| 項目 | 償却方法 | 償却期間 |
| --- | --- | --- |
| のれん | 定額法 | 20年以内 |
| 特許権 | 定額法 | 8年 |
| ソフトウェア（自社利用） | 定額法 | 5年以内 |
| ソフトウェア（市場販売目的） | 見込販売数量基準等 | 3年以内 |

繰延資産は 5 項目に限定：創立費・開業費（5年以内）、株式交付費（3年以内）、社債発行費（償還期限内）、開発費（5年以内）。

## 有価証券の4分類

| 分類 | B/S 評価 | 評価差額の処理先 |
| --- | --- | --- |
| 売買目的有価証券 | 時価 | 当期損益（P/L 営業外） |
| 満期保有目的の債券 | 償却原価 | 利息として各期配分 |
| 子会社・関連会社株式 | 取得原価 | なし |
| その他有価証券 | 時価 | 純資産の部（その他有価証券評価差額金） |

## 典型的なつまずき

- `低価法` を棚卸資産の評価法（FIFO 等）そのものと混同する → 低価法は評価法で算出した取得原価に**追加で**適用する切下げルール
- `減価償却費`（P/L 費用）と `減価償却累計額`（B/S 控除）を取り違える
- 売却損益を `取得原価` ベースで計算してしまう → 正しくは `帳簿価額`（取得原価 − 減価償却累計額）との差
- 減損の認識判定を**割引後** CF で行ってしまう → 認識は**割引前**、測定で**割引後**
- 棚卸減耗損と棚卸資産評価損を混同する → 減耗損は**数量差**、評価損は**単価差**
- 移動平均法で仕入時点ごとの再計算を忘れ、総平均法と同じ計算をしてしまう

## 確認問題

<Callout title="問1：棚卸資産の評価">
    以下のデータから、先入先出法による期末商品棚卸高と売上原価を求めよ。
    期首商品：50個×@200円、当期仕入：第1回 100個×@220円、第2回 80個×@250円。
    当期販売数量：160個。

    **解答**：期末在庫 = 50 + 100 + 80 − 160 = 70個。FIFO では新しいものが残る → 第2回仕入分80個のうち70個 → 70×250 = 17,500円。売上原価 = (50×200 + 100×220 + 80×250) − 17,500 = 52,000 − 17,500 = 34,500円。
</Callout>

<Callout title="問2：減価償却の月割計算">
    取得原価 900万円、耐用年数 5年、残存価額 0、定額法。X1年7月1日取得、決算日X2年3月31日。X1年度とX2年度の減価償却費をそれぞれ求めよ。

    **解答**：年間償却費 = 900 ÷ 5 = 180万円。X1年度：180 × 9/12 = 135万円（7月〜3月）。X2年度：180万円（12か月分フル）。
</Callout>

<Callout title="問3：減損の認識と測定">
    店舗の帳簿価額 8,000万円。割引前将来CF合計 7,200万円、正味売却価額 5,500万円、使用価値 6,300万円。減損損失の額を求めよ。

    **解答**：割引前CF 7,200 ＜ 帳簿価額 8,000 → 減損を認識。回収可能価額 = max(5,500, 6,300) = 6,300。減損損失 = 8,000 − 6,300 = 1,700万円。
</Callout>

## 問題を解くときの観点

- この論点は `取得時`、`期末`、`処分時` のどこか
- 比較しているのは `取得原価` か `帳簿価額` か
- P/L の費用 / 損益と B/S の残高をどう結ぶか
