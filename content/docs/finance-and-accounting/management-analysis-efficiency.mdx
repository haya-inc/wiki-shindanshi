## このページの役割

このページは、`経営分析` のうち `効率性` を固める知識ノードです。売上債権、棚卸資産、固定資産、総資産が `どれだけ速く回っているか` を見て、資産の重さを読める状態を目指します。

## 学習のポイント

- まず `何が分子で何が分母か` を固定する
- 次に `回転率` と `回転期間` を行き来できるようにする
- そのうえで `高い方がよい` を機械的に当てはめず、業種や戦略との関係で読む

## 主要指標の比較

| 指標 | 何を見るか | 低いと疑いやすいこと |
| --- | --- | --- |
| 総資本回転率 | 総資産全体がどれだけ売上へ結びついているか | 資産全体が重い |
| 売上債権回転率 | 売掛金などの回収の速さ | 回収遅れ、与信負担 |
| 棚卸資産回転率 | 在庫の回り方 | 在庫滞留、需要予測のずれ |
| 有形固定資産回転率 | 設備が売上へどれだけ結びついているか | 過大投資、稼働不足 |

## 回転率を見るときの基本姿勢

回転率は、`同じ資産でどれだけ売上をつくれたか` を見る指標です。したがって、低いときは `分母の資産が重すぎる`、高いときは `資産を絞っている`、`稼働が高い` などを疑います。

ただし、在庫を意図的に厚く持つ業種や、先行投資直後の企業では、一時的に回転率が低くても不自然ではありません。数字だけでなく、`なぜその資産が積み上がっているか` を読むことが重要です。

## よくある読み取り

| 指標差 | どう読むか | 次に見たいこと |
| --- | --- | --- |
| 売上債権回転率が低い | 回収が遅い、与信管理が甘い | 売上債権回転期間 |
| 棚卸資産回転率が低い | 在庫が滞留している | 当座比率、在庫戦略 |
| 有形固定資産回転率が低い | 設備投資に売上が追いついていない | 操業度、利益率 |

## 公式一覧

### 回転率

| 指標 | 計算式 | 意味 |
| --- | --- | --- |
| 総資産回転率 | 売上高 ÷ 総資産（回） | 資産全体の回転速度 |
| 売上債権回転率 | 売上高 ÷ 売上債権（回） | 売上債権の回収速度 |
| 棚卸資産回転率 | 売上高（or 売上原価） ÷ 棚卸資産（回） | 在庫の回転速度 |
| 有形固定資産回転率 | 売上高 ÷ 有形固定資産（回） | 固定資産の活用度 |
| 仕入債務回転率 | 売上原価（or 仕入高） ÷ 仕入債務（回） | 仕入代金の支払速度 |

売上債権 = 売掛金 + 受取手形。仕入債務 = 買掛金 + 支払手形。

### 回転期間

```
売上債権回転期間 = 売上債権 ÷ (売上高 ÷ 365)（日）
               = 365 ÷ 売上債権回転率（日）
棚卸資産回転期間 = 棚卸資産 ÷ (売上原価 ÷ 365)（日）
仕入債務回転期間 = 仕入債務 ÷ (売上原価 ÷ 365)（日）
```

回転率と回転期間の関係：`回転率 × 回転期間 = 365日`（月数なら 12）。回転率が高い ⟺ 回転期間が短い。回転期間は短いほど良い。ただし仕入債務回転期間は長い方が資金繰りに有利。

### CCC（キャッシュ・コンバージョン・サイクル）

仕入代金を支払ってから売上代金を回収するまでの期間。

```
CCC = 売上債権回転日数 + 棚卸資産回転日数 − 仕入債務回転日数
```

| CCCの値 | 意味 |
| --- | --- |
| CCC > 0 | 支払いが回収より先（運転資金が必要） |
| CCC = 0 | 支払いと回収が同時 |
| CCC < 0 | 回収が支払いより先（資金に余裕がある） |

## 計算手順

### 回転率・回転期間・CCCの一連の計算

```
【データ】
  売上高 3,650百万円、売上原価 2,920百万円
  売上債権 500百万円、棚卸資産 200百万円、仕入債務 400百万円

手順1: 回転率を求める
  売上債権回転率 = 3,650 ÷ 500 = 7.3回
  棚卸資産回転率 = 2,920 ÷ 200 = 14.6回（売上原価ベース）
  仕入債務回転率 = 2,920 ÷ 400 = 7.3回

手順2: 回転日数を求める
  売上債権回転日数 = 500 ÷ (3,650 ÷ 365) = 500 ÷ 10 = 50日
  棚卸資産回転日数 = 200 ÷ (2,920 ÷ 365) = 200 ÷ 8 = 25日
  仕入債務回転日数 = 400 ÷ (2,920 ÷ 365) = 400 ÷ 8 = 50日

手順3: CCCを求める
  CCC = 50 + 25 − 50 = 25日
  → 仕入代金を支払ってから売上代金を回収するまでに25日の資金が必要

検算: 365 ÷ 7.3 ≒ 50日（売上債権回転日数）✓
```

棚卸資産回転日数は売上原価ベースで計算する（棚卸資産は原価で評価されるため）。問題文に指定がなければ売上原価を用いる。

### CCCの改善策

| 改善方向 | 施策 |
| --- | --- |
| 売上債権回転日数の短縮 | 回収の早期化、ファクタリング活用 |
| 棚卸資産回転日数の短縮 | 在庫管理の効率化、JIT導入 |
| 仕入債務回転日数の延長 | 支払サイトの交渉 |

## 問題を解くときの観点

- 分母の資産が `なぜ増えたか` を与件から拾う
- 回転率の低下が `在庫`、`売掛金`、`設備` のどれで起きているか切り分ける
- 回転率の悪化を `収益性` や `安全性` の指標とつなげる
- 求めたいのは `回転率`（回）か `回転期間`（日）か

## 典型的なつまずき

- `回転率は高いほどよい` と機械的に決める → 業種や戦略を見ずに判断すると間違える
- `回転率` と `回転期間` を逆向きに読んでしまう → 回転率が高い＝回転期間が短い
- 分母の資産が何を表しているかを意識せずに解く → 売上債権に受取手形を含め忘れる
- 棚卸資産回転率の分子に売上高を使ってしまう → 原則は**売上原価**（問題指定がない限り）
- CCCで仕入債務回転日数を**足して**しまう → 正しくは**引く**（支払いを遅らせるほど有利）
- 回転期間の計算で 365 と 12 を混ぜる → 日数なら 365、月数なら 12 で統一

## 確認問題

<Callout title="問1：回転率と回転期間">
    売上高 1,460百万円、売上債権 200百万円。売上債権回転率と回転期間（日数）を求めよ。

    **解答**：回転率 = 1,460÷200 = 7.3回。回転期間 = 365÷7.3 = 50日。または 200÷(1,460÷365) = 200÷4 = 50日。
</Callout>

<Callout title="問2：CCCの計算と改善">
    売上高 7,300百万円、売上原価 5,840百万円。売上債権 1,000百万円、棚卸資産 320百万円、仕入債務 800百万円。CCCを求め、改善策を1つ挙げよ。

    **解答**：売上債権回転日数 = 1,000÷(7,300÷365) = 1,000÷20 = 50日。棚卸資産回転日数 = 320÷(5,840÷365) = 320÷16 = 20日。仕入債務回転日数 = 800÷(5,840÷365) = 800÷16 = 50日。CCC = 50+20−50 = 20日。改善策：在庫管理の効率化で棚卸資産回転日数を短縮する（例：JIT導入）。
</Callout>

<Callout title="問3：回転率の悪化原因の判定">
    前期と当期で総資産回転率が 1.5回→1.2回に低下。売上高は 900百万→960百万に増加。総資産は 600百万→800百万に増加。主な増加は棚卸資産（100百万→250百万）。原因を指摘せよ。

    **解答**：売上高は 6.7%増だが総資産は 33%増。総資産増加の主因は棚卸資産（150百万増）。棚卸資産回転率も悪化（9.0回→3.84回）しており、在庫滞留が総資産回転率低下の原因。在庫管理の改善が必要。
</Callout>
