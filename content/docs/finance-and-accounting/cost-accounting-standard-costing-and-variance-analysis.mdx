## このページの役割

このページは、`標準原価計算` と `差異分析` を固めるページです。式を丸暗記するのではなく、`何が標準からずれたのか` を言葉で説明できる状態を目指します。

## 学習のポイント

- まず `標準` と `実際` のどちらを比べているかを切る
- 次に差異を `価格` と `数量` の軸で分ける
- `有利差異 / 不利差異` を式ではなく意味で判定する

## 標準原価計算の考え方

標準原価計算は、あらかじめ `あるべき原価` を設定し、実際原価との差を管理する方法です。原価差異は、原価管理の入口として使われます。

## 差異分析の基本

| 差異 | 何のずれを見るか | 代表的な見方 |
| --- | --- | --- |
| 価格差異 | 単価や賃率のずれ | `標準単価 - 実際単価` |
| 数量差異 | 消費量や作業時間のずれ | `標準数量 - 実際数量` |
| 能率差異 | 労働時間や操業効率のずれ | `標準時間 - 実際時間` |

## 原価標準（標準原価カード）

製品 1 単位あたりの「あるべき原価」を設定したもの。

```
【標準原価カード 例】製品X（1単位）
直接材料費  標準単価@500円 × 標準消費量 2kg = 1,000円
直接労務費  標準賃率@1,200円 × 標準時間 0.5h = 600円
製造間接費  標準配賦率@800円 × 標準時間 0.5h = 400円
                               標準製造原価  2,000円
```

標準の種類：理想標準原価（達成不能）、正常原価（複数年平均）、**現実的標準原価**（原価管理に最適、試験で最重要）。

## 差異分析の公式

### 直接材料費差異

```
価格差異 = (標準価格 − 実際価格) × 実際消費量
数量差異 = (標準消費量 − 実際消費量) × 標準価格
```

### 直接労務費差異

```
賃率差異 = (標準賃率 − 実際賃率) × 実際作業時間
作業時間差異 = (標準作業時間 − 実際作業時間) × 標準賃率
```

### 製造間接費差異（3分法）

```
予算差異(支出差異) = 予算許容額 − 実際発生額
  予算許容額 = 変動費率 × 実際操業度 + 固定費予算額
能率差異 = (標準操業度 − 実際操業度) × 標準配賦率
操業度差異 = (実際操業度 − 基準操業度) × 固定費率
```

差異がプラス → **有利差異**（標準より良い）。マイナス → **不利差異**（標準より悪い）。

### 2分法・4分法との関係

```
2分法: 予算差異 = 3分法の予算差異 + 能率差異
       操業度差異 = 同じ
4分法: 3分法の予算差異を変動費予算差異と固定費予算差異に分解
       3分法の能率差異を変動費能率差異と固定費能率差異に分解
```

## 計算手順

### 総合計算例（3分法）

```
【標準原価カード】（1個あたり）
  材料費: @500円 × 2kg = 1,000円
  労務費: @1,200円 × 0.5h = 600円
  間接費: @800円 × 0.5h = 400円（変動@300 + 固定@500）

【当月実績】生産量 100個
  材料: 210kg × @480円 = 100,800円
  労務: 55h × @1,250円 = 68,750円
  間接費: 42,000円（変動17,000 + 固定25,000）
  基準操業度 = 50h

手順1: 標準原価を計算する
  標準材料費 = 1,000 × 100 = 100,000円
  標準消費量 = 2 × 100 = 200kg
  標準労務費 = 600 × 100 = 60,000円
  標準時間 = 0.5 × 100 = 50h

手順2: 直接材料費の差異分析
  総差異 = 100,000 − 100,800 = △800（不利）
  価格差異 = (500 − 480) × 210 = +4,200（有利：安く買えた）
  数量差異 = (200 − 210) × 500 = △5,000（不利：多く使った）
  検算: 4,200 + (△5,000) = △800 ✓

手順3: 直接労務費の差異分析
  総差異 = 60,000 − 68,750 = △8,750（不利）
  賃率差異 = (1,200 − 1,250) × 55 = △2,750（不利）
  時間差異 = (50 − 55) × 1,200 = △6,000（不利）
  検算: △2,750 + △6,000 = △8,750 ✓

手順4: 製造間接費の差異分析（3分法）
  予算許容額 = 300 × 55 + 25,000 = 41,500
  予算差異 = 41,500 − 42,000 = △500（不利）
  能率差異 = (50 − 55) × 800 = △4,000（不利）
  操業度差異 = (55 − 50) × 500 = +2,500（有利）
  検算: △500 + △4,000 + 2,500 = △2,000
  標準配賦額 40,000 − 実際 42,000 = △2,000 ✓
```

## 問題を解くときの観点

- 標準原価を**先に**計算してから分解に入る
- 差異の符号だけでなく、**なぜ**有利/不利かを言葉にする
- 総差異と内訳差異の合計が一致するか必ず検算する
- 製造間接費は問題文で 2分法/3分法/4分法のどれを要求しているか確認する

## 典型的なつまずき

- 価格差異に**標準消費量**を掛けてしまう → 正しくは**実際消費量**
- 数量差異に**実際単価**を掛けてしまう → 正しくは**標準単価**
- 操業度差異の符号を逆にする → `(実際 − 基準) × 固定費率`。実際が基準を上回れば有利
- 能率差異と操業度差異を混同する → 能率差異は「標準より遅いか速いか」、操業度差異は「基準操業度に対して実際がどうか」
- 3分法の予算差異を「固定費予算 − 実際固定費」だけで計算する → 変動費部分も含む

## 確認問題

<Callout title="問1：直接材料費の差異分析">
    標準：@400円 × 3kg/個。当月生産 200個。実績：消費量 640kg、@410円。価格差異と数量差異を求めよ。

    **解答**：標準消費量 = 3×200 = 600kg。価格差異 = (400−410)×640 = △6,400（不利）。数量差異 = (600−640)×400 = △16,000（不利）。総差異 = △22,400。検算：標準 240,000 − 実際 262,400 = △22,400 ✓。
</Callout>

<Callout title="問2：製造間接費の3分法">
    標準配賦率@600/h（変動@250、固定@350）。基準操業度 100h。当月生産で標準操業度 90h、実際操業度 95h。実際間接費 60,000円。予算差異・能率差異・操業度差異を求めよ。

    **解答**：予算許容額 = 250×95 + 350×100 = 23,750 + 35,000 = 58,750。予算差異 = 58,750 − 60,000 = △1,250（不利）。能率差異 = (90−95)×600 = △3,000（不利）。操業度差異 = (95−100)×350 = △1,750（不利）。検算：標準配賦額 600×90 = 54,000。54,000 − 60,000 = △6,000 = △1,250 + △3,000 + △1,750 ✓。
</Callout>
