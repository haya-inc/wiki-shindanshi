## このページの役割

このページは、`収益認識に関する会計基準`（収益認識基準）を整理する解説ページです。2021年4月から強制適用された新しい収益認識のルールを、5ステップモデルを軸に理解します。

<Callout title="このページの読み方">
  スマートフォンを「本体 + 2年間の保守サービス」のセットで 120,000円で販売したとします。この 120,000円を「いつ」「いくら」売上に計上するかを体系的に決めるのが収益認識基準です。従来の「出荷したら売上」という単純なルールでは対応できない取引が増えたため、5つのステップで判断する枠組みが導入されました。このページでは、5ステップを順に理解し、計算例で処理方法を身につけます。
</Callout>

## 学習のポイント

- まず `5ステップモデル` の全体像を押さえる
- 次に `履行義務` の識別と `取引価格の配分` を理解する
- そのうえで `一時点` と `一定期間` の認識の違いを区別する

## 試験で何が問われるか

- 5ステップの順序と内容を説明できるか
- 複数の履行義務がある場合の **取引価格の配分計算** ができるか
- 一時点認識と一定期間認識の **区別** ができるか

## 5ステップモデル

収益認識基準では、次の 5 つのステップで「いつ」「いくら」収益を認識するかを決めます。

### ステップ 1: 契約の識別

顧客との契約を識別します。契約とは、法的に強制力のある権利・義務を生じさせる合意です。

### ステップ 2: 履行義務の識別

契約の中に含まれる `履行義務`（顧客に財やサービスを移転する約束）を個別に識別します。1つの契約に複数の履行義務が含まれることがあります。

識別のポイント: その財やサービスが **単独で** 顧客に便益を提供できるか。

### ステップ 3: 取引価格の算定

契約全体の取引価格を決定します。変動対価（値引き、リベート、ペナルティ等）がある場合は、その見積りを含めます。

### ステップ 4: 取引価格の配分

取引価格を、各履行義務に **独立販売価格の比率** で配分します。

$$
\text{配分額} = \text{取引価格} \times \frac{\text{当該履行義務の独立販売価格}}{\text{全履行義務の独立販売価格の合計}}
$$

### ステップ 5: 収益の認識

各履行義務が充足された時点（または充足されるにつれて）収益を認識します。

| 認識のタイミング | 条件 | 具体例 |
| --- | --- | --- |
| 一時点で認識 | 支配が一時点で移転する | 商品の引渡し、物品の販売 |
| 一定期間にわたり認識 | 支配が徐々に移転する | 工事契約、保守サービス |

## 計算例 ── 複数履行義務の取引価格配分

```
【データ】
  契約: スマートフォン本体＋2年間の保守サービスで 120,000円
  独立販売価格: 本体 100,000円、保守サービス 50,000円（合計 150,000円）

ステップ2: 履行義務の識別
  → ①本体の引渡し ②保守サービスの提供（2つの履行義務）

ステップ4: 取引価格の配分
  本体への配分 = 120,000 × (100,000 / 150,000) = 120,000 × 2/3 = 80,000円
  保守への配分 = 120,000 × (50,000 / 150,000) = 120,000 × 1/3 = 40,000円
  合計 = 80,000 + 40,000 = 120,000 ✓

ステップ5: 収益認識
  本体: 引渡時に 80,000円を一時点で認識
  保守: 2年間にわたり 40,000円を均等に認識（年20,000円）

→ 1年目の収益 = 80,000 + 20,000 = 100,000円
→ 2年目の収益 = 20,000円
```

## 計算例 ── 一定期間にわたる収益認識

工事契約のように、一定期間にわたって履行義務を充足する場合、進捗度に応じて収益を認識します（旧・工事進行基準に相当）。

```
【データ】
  工事契約: 請負金額 1,000万円、見積総原価 800万円
  1年目の発生原価 320万円

進捗度 = 320 / 800 = 40%
1年目の収益 = 1,000 × 40% = 400万円
1年目の原価 = 800 × 40% = 320万円
1年目の利益 = 400 − 320 = 80万円
```

進捗度の測定には、**インプット法**（発生原価の割合）と**アウトプット法**（成果物の完成割合）があります。

## 変動対価

取引価格に不確実性がある場合（値引き、リベート、返品権付き販売など）、**期待値法** または **最も可能性の高い金額法** で見積ります。

```
【計算例】返品権付き販売
  販売: 1,000個 × @500円 = 500,000円
  過去実績から返品率 5%と見積り

  収益計上額 = 500,000 × (1−0.05) = 475,000円
  返金負債 = 500,000 × 0.05 = 25,000円
```

## 従来の基準との違い

| 比較軸 | 従来（実現主義） | 収益認識基準 |
| --- | --- | --- |
| 認識の基準 | 出荷・引渡が中心 | 履行義務の充足（支配の移転） |
| 複数要素取引 | 明確なルールなし | 5ステップで配分 |
| 変動対価 | 確定時に認識 | 見積りを取引価格に含める |
| 工事契約 | 工事進行基準/完成基準 | 一定期間にわたる認識（進捗度） |

---

## 典型的なつまずき

- 5ステップの順序を覚えず、いきなり収益認識の判断をする → 必ず **ステップ1から順に** 処理
- 複数の履行義務があるのに取引価格を **まとめて** 認識する → 独立販売価格の比率で配分が必要
- 一時点認識と一定期間認識の区別ができない → 「支配がいつ移転するか」で判断
- 独立販売価格の合計と取引価格の違いを見落とす → 差額（値引き等）は比率で配分
- 変動対価を無視して全額を収益計上する → 返品や値引きの見積りを取引価格に反映

## 問題を解くときの観点

- 契約に `複数の履行義務` が含まれるか
- 取引価格に `変動要素` はあるか
- 収益は `一時点` で認識するか `一定期間` で認識するか
- 独立販売価格の **比率** で配分しているか

## 確認問題

<Callout title="問1：取引価格の配分">
    契約金額 90,000円。履行義務: 製品A（独立販売価格 60,000円）と製品B（独立販売価格 40,000円）。各履行義務への配分額を求めよ。

    **解答**：製品A = 90,000 × 60,000/(60,000+40,000) = 90,000 × 0.6 = 54,000円。製品B = 90,000 × 0.4 = 36,000円。
</Callout>

<Callout title="問2：進捗度による収益認識">
    工事契約: 請負金額 2,000万円、見積総原価 1,600万円。当期の発生原価 640万円。当期に認識すべき収益と利益を求めよ。

    **解答**：進捗度 = 640/1,600 = 40%。収益 = 2,000×40% = 800万円。原価 = 1,600×40% = 640万円。利益 = 800−640 = 160万円。
</Callout>

<Callout title="問3：変動対価（返品権）">
    100個を @1,000円で販売。返品率を 8%と見積る。収益計上額と返金負債を求めよ。

    **解答**：収益 = 100×1,000×(1−0.08) = 92,000円。返金負債 = 100×1,000×0.08 = 8,000円。
</Callout>
