## このページの役割

このページは、`個別原価計算` と `総合原価計算` の違いを固める知識ノードです。問題文を見た瞬間に `受注生産` か `大量生産` かを切り、`何をどこへ集計するか` を決められるようにします。

<Callout title="このページの読み方">
  原価計算の世界には、オーダーメイドのスーツを仕立てるような原価計算（個別）と、工場で同じペットボトル飲料を何万本も作る原価計算（総合）の 2 種類があります。まず「何が違うのか」をイメージで掴み、そのあと計算の仕組みを見ていきます。
</Callout>

## 学習のポイント

- まず `受注生産` と `大量生産` を切る
- 次に `製造指図書` と `完成品換算量` のどちらが出てくるかを見る
- そのうえで `予定配賦`、`平均法`、`先入先出法` の使いどころを整理する

## 個別原価計算 ── 注文ごとに原価を追いかける

個別原価計算は、製造指図書ごとに原価を集計する考え方です。受注生産やロットごとの差が大きい場面（建設業、造船業、印刷業など）で使います。

イメージとしては、「この注文にいくらかかったか」を 1 件ずつ追いかける方法です。A 社向けの橋と B 社向けのビルでは使う材料も工期も全く違うので、それぞれ別々に原価を記録します。この記録簿が `製造指図書` です。

### 3 つの原価要素を指図書に集める

原価計算では、どの方式でも原価を 3 つに分けます。

| 原価要素 | 何のことか | 具体例 |
| --- | --- | --- |
| 直接材料費 | その製品に直接使った材料 | 木材、鉄鋼、部品 |
| 直接労務費 | その製品を作った作業者の賃金 | 作業時間 × 単価 |
| 製造間接費 | 特定の製品に紐づかない費用 | 工場の家賃、電気代、管理者の給料 |

直接材料費と直接労務費は「この指図書に○○円」とすぐ分かりますが、製造間接費は困ります。工場の電気代は A 社の注文にも B 社の注文にも使われていて、直接には紐づけられません。そこで登場するのが `配賦` という仕組みです。

### 製造間接費の配賦 ── 「直接紐づかない費用」をどう割り振るか

配賦とは、共通にかかった費用を何らかの基準で各製品に割り振ることです。事前に決めた配賦率（予定配賦率）を使うのが一般的です。

$$
\text{予定配賦率} = \frac{\text{製造間接費予算額}}{\text{基準操業度}}
$$

$$
\text{予定配賦額} = \text{予定配賦率} \times \text{実際配賦基準量}
$$

配賦基準は、労働集約的な製造なら `直接作業時間`、資本集約的なら `機械運転時間` というように、間接費の発生パターンに近い基準を選びます。

### 予定と実績がずれる ── 配賦差異

予定配賦率で計算した配賦額と、実際に発生した間接費は一致しないことがほとんどです。この差が `配賦差異` です。

$$
\text{配賦差異} = \text{予定配賦額} - \text{実際発生額}
$$

プラスなら `有利差異`（予定より少なく済んだ）、マイナスなら `不利差異`（予定より多くかかった）。原則として売上原価に賦課し、差異が大きい場合は売上原価と棚卸資産に按分します。

### 個別原価計算の計算例

ここで一連の流れを追いかけてみましょう。

```
【設定】
  指図書No.201（当月完成）：材料費 400,000、労務費 200,000
  指図書No.202（月末仕掛中）：材料費 300,000、労務費 150,000
  製造間接費の実際発生額：280,000円
  予定配賦率：直接労務費の 80%

ステップ1: 各指図書への間接費配賦
  No.201：200,000 × 80% = 160,000
  No.202：150,000 × 80% = 120,000
  予定配賦額合計 = 280,000

ステップ2: 各指図書の製造原価
  No.201 = 400,000 + 200,000 + 160,000 = 760,000
    → 完成したので「製品」勘定へ振り替え
  No.202 = 300,000 + 150,000 + 120,000 = 570,000
    → まだ作っている途中なので「月末仕掛品」として残る

ステップ3: 配賦差異の確認
  予定配賦額 280,000 − 実際発生額 280,000 = 0（差異なし）
```

この例ではたまたま差異がゼロですが、実際には予定と実績のずれが生じるのが普通です。

<Callout title="ここまでのまとめ">
  個別原価計算では、製造指図書ごとに直接費を集め、間接費は予定配賦率で割り振ります。注文が完成したら製品勘定へ、まだ途中なら月末仕掛品として残します。次に見る総合原価計算は、考え方がかなり違います。
</Callout>

## 総合原価計算 ── 大量生産の原価をどう分けるか

総合原価計算は、同じ製品を大量に作る工場で使います。食品、化学製品、繊維など、1 個 1 個を追いかけるのが現実的でない場合に、「当月にかかった原価の合計を、完成品と月末仕掛品にどう配分するか」を考えます。

個別原価計算との最大の違いは、`完成品換算量` という概念が入ることです。

### 完成品換算量 ── 作りかけの製品をどう数えるか

月末にまだ完成していない製品（仕掛品）がある場合、その仕掛品に「原価の何割分を割り当てるか」を考える必要があります。

たとえば、ある月末の仕掛品が 100 個あり、加工の進み具合が 40% だとします。この 100 個は、完成品に換算すると 40 個分の加工しかしていない、と考えます。これが完成品換算量です。

$$
\text{月末仕掛品の完成品換算量} = \text{仕掛品数量} \times \text{加工進捗度}
$$

ただし、材料の投入タイミングによって扱いが変わります。材料を工程の最初（始点）で全量投入する場合、仕掛品の進捗度が何%でも、材料費については 100% 投入済みです。加工費だけが進捗度に比例します。

### 平均法と先入先出法 ── 単価の求め方が違う

総合原価計算では、月初仕掛品（先月から作りかけだった分）と当月投入分を合わせて原価を配分します。このとき、単価の求め方が 2 通りあります。

**平均法** は、月初仕掛品と当月投入を混ぜて平均単価を出す方法です。計算が簡便で、試験での出題頻度が高いです。

**先入先出法** は、月初仕掛品を先に完成させたと考え、当月投入分だけの単価を計算する方法です。当月の原価管理に適しています。

### 平均法の計算例 ── 手順を一つずつ追う

```
【データ】
  月初仕掛品 200個（進捗度50%）：材料費 100,000、加工費 60,000
  当月投入 800個：材料費 400,000、加工費 330,000
  完成品 900個、月末仕掛品 100個（進捗度40%）
  ※材料は始点で全量投入

ステップ1: 完成品換算量を求める
  材料費の分母：900（完成品）+ 100（月末仕掛品）= 1,000個
    → 材料は始点投入なので、仕掛品も 100% 投入済み
  加工費の分母：900 + 100 × 40% = 940個
    → 仕掛品は 40% しか加工していないので、40個分と数える

ステップ2: 平均単価を求める
  材料費の総額 = 100,000（月初）+ 400,000（当月）= 500,000
  材料費単価 = 500,000 ÷ 1,000 = @500
  加工費の総額 = 60,000 + 330,000 = 390,000
  加工費単価 = 390,000 ÷ 940 ≒ @414.9

ステップ3: 月末仕掛品と完成品に配分する
  月末仕掛品 = @500 × 100個 + @414.9 × 40個 = 50,000 + 16,596 = 66,596円
  完成品原価 = 総額 890,000 − 月末 66,596 = 823,404円
```

ポイントは、材料費と加工費で「分母が違う」ことです。材料費は仕掛品もフル投入済みなので 1,000個で割りますが、加工費は仕掛品が40%しか進んでいないので 940個で割ります。

### 先入先出法の計算例 ── 平均法との違い

先入先出法では、月初仕掛品を先に完成させたと考えます。そのため、当月投入分だけの換算量を求める必要があります。

```
【データ】同上

ステップ1: 当月投入の完成品換算量を求める
  材料費：900（完成品）+ 100（月末仕掛品）− 200（月初仕掛品）= 800個
  加工費：900 + 100×40% − 200×50% = 840個
  → 月初仕掛品が「すでに投入していた分」を引くのがポイント

ステップ2: 当月投入の単価を求める
  材料費単価 = 400,000 ÷ 800 = @500
  加工費単価 = 330,000 ÷ 840 ≒ @392.9

ステップ3: 配分
  月末仕掛品 = @500×100 + @392.9×40 = 50,000 + 15,714 = 65,714円
  完成品原価 = (100,000+60,000) + (400,000+330,000) − 65,714 = 824,286円
```

この例では月末仕掛品原価が平均法 66,596 vs 先入先出法 65,714 と僅差ですが、原価水準が大きく変動した月には差が開きます。

<Callout title="ここまでのまとめ">
  総合原価計算の核心は「完成品換算量」です。仕掛品の進捗度に応じて加工費の分母を調整し、材料費と加工費を別々に配分します。次は、もう一つの頻出論点「仕損・減損」を見ます。
</Callout>

## 仕損と減損 ── 不良品が出たらどうするか

製造過程では、失敗作（仕損品）が出たり、材料が蒸発・粉散して減ったり（減損）することがあります。

| 概念 | 何が起きるか | 具体例 |
| --- | --- | --- |
| 仕損 | 加工に失敗して合格品にならない | 焼き物のひび割れ、部品の不良 |
| 減損 | 材料が物理的に減る | 化学反応での蒸発、切削くず |

### 正常仕損と異常仕損

通常の製造過程で避けられない仕損は `正常仕損` と呼び、良品（完成品や仕掛品）の原価に含めます。一方、設備故障などで生じる想定外の仕損は `異常仕損` と呼び、特別損失として処理します。

### 誰が仕損のコストを負担するか ── 発生点と進捗度の比較

正常仕損のコストを「完成品だけが負担するのか」「月末仕掛品も負担するのか」は、仕損が発生した地点と月末仕掛品の進捗度を比べて決めます。

考え方はこうです。月末仕掛品がまだ仕損発生点を通過していないなら、仕掛品は「仕損に出会っていない」ので負担しません。通過済みなら負担します。

```
仕損発生点が月末仕掛品の進捗度より後 → 完成品のみが負担
仕損発生点が月末仕掛品の進捗度より前 → 完成品と月末仕掛品の両方が負担
仕損が終点で発生 → 完成品のみが負担
仕損が始点で発生 → 両方が負担
```

### 度外視法の計算例 ── 仕損を「なかったこと」にする

度外視法は、仕損品を計算から外して無視する方法です。仕損分の原価が自動的に良品に上乗せされます。

```
【データ】
  月初仕掛品 50個（進捗度40%）：材料費 25,000、加工費 8,000
  当月投入 150個：材料費 75,000、加工費 58,800
  完成品 160個、正常仕損 10個（終点発生）、月末仕掛品 30個（進捗度40%）
  ※材料は始点投入、仕損品の評価額なし

判断: 仕損は終点（100%の地点）で発生。
      月末仕掛品の進捗度は40%。
      → 仕掛品は仕損発生点を通過していない → 完成品のみ負担
      → 仕損10個を無視して計算する

ステップ1: 完成品換算量（仕損を除く）
  材料費：160 + 30 = 190個
  加工費：160 + 30×40% = 172個

ステップ2: 平均単価
  材料費 = (25,000+75,000) ÷ 190 ≒ @526.3
  加工費 = (8,000+58,800) ÷ 172 ≒ @388.4

ステップ3: 配分
  月末仕掛品 = @526.3×30 + @388.4×12 ≒ 15,789 + 4,660 = 20,449円
  完成品 = 166,800 − 20,449 = 146,351円
  → 仕損10個分の原価が完成品に自動的に上乗せされている
```

## 2 方式の比較

| 比較軸 | 個別原価計算 | 総合原価計算 |
| --- | --- | --- |
| 前提 | 受注生産、ロットごとの差が大きい | 大量生産、同種製品を連続生産する |
| 原価の集計単位 | 製造指図書、仕事単位 | 工程全体、当月生産分 |
| 頻出論点 | 予定配賦、配賦差異、勘定連絡 | 完成品換算量、平均法、先入先出法、仕損・減損 |

問題文に `製造指図書` が出てきたら個別原価計算、`月末仕掛品` と `完成品換算量` が出てきたら総合原価計算を疑ってください。

## 典型的なつまずき

- 個別原価計算なのに完成品換算量を探し始める → 指図書単位で集計するだけ
- 総合原価計算なのに指図書単位で原価を追おうとする → 工程全体で配分する
- 加工費の完成品換算量に月末仕掛品を**数量のまま**入れてしまう → 進捗度を掛ける
- 先入先出法で月初仕掛品の**既投入分**を引き忘れる → 当月投入の換算量 = 完成品 + 月末 − 月初
- 仕損の負担先を発生点と進捗度の比較なしに決めてしまう → 度外視法でも負担先の判定は必要

## 確認問題

<Callout title="問1：個別原価計算の配賦差異">
    製造間接費の予定配賦率 @600円/時間。指図書A（完成）の実際作業時間 80h、指図書B（仕掛中）の実際作業時間 40h。実際間接費 68,000円。配賦差異を求め、有利・不利を判定せよ。

    **解答**：予定配賦額 = 600×(80+40) = 72,000円。配賦差異 = 72,000−68,000 = +4,000円（有利差異）。予定より実際が少なく済んだ。
</Callout>

<Callout title="問2：総合原価計算（平均法）">
    月初仕掛品 100個（進捗度60%）：材料費 50,000円、加工費 24,000円。当月投入 400個：材料費 200,000円、加工費 156,000円。完成品 450個、月末仕掛品 50個（進捗度40%）。材料は始点投入。完成品原価を求めよ。

    **解答**：材料費換算量 = 450+50 = 500。単価 = 250,000÷500 = @500。月末材料費 = 500×50 = 25,000。加工費換算量 = 450+50×40% = 470。単価 = 180,000÷470 ≒ @383。月末加工費 ≒ 383×20 = 7,660。月末仕掛品 = 32,660円。完成品 = 430,000−32,660 = 397,340円。
</Callout>

<Callout title="問3：仕損の負担先判定">
    正常仕損が加工進捗度 60% の地点で発生。月末仕掛品の進捗度は 50%。仕損費の負担先はどうなるか。

    **解答**：仕損発生点（60%）> 月末仕掛品の進捗度（50%）。月末仕掛品は仕損発生点をまだ通過していないため、**完成品のみ**が仕損費を負担する。
</Callout>
