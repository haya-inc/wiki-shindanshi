## このページの役割

ファイナンスは、資金調達と投資判断の分野です。診断士試験では、`WACC`、`CAPM`、`NPV`、`IRR`、`企業価値評価` が最重要で、事例Ⅳにも直結します。

## 最優先で固める論点

### 資本コストと WACC

投資判断の起点になるのが資本コストです。`株主資本コスト` と `負債コスト` を区別し、企業全体の資金調達コストとして `WACC` を理解します。

特に重要なのは次の点です。

- 負債コストには `節税効果` を反映する
- WACC は `投資判断のハードルレート`
- 原則として `時価ベース` で考える

### CAPM と β値

CAPM は株主資本コストを求める基本モデルです。`β値` は市場全体に対する感応度を表し、個別企業の株主資本コストを考えるときの中心になります。

### NPV と IRR

投資判断では、まず `NPV` を中心に考えるのが基本です。IRR は重要ですが、複数案比較や前提条件によっては NPV の方が扱いやすいことを理解しておく必要があります。

### 企業価値評価

DCF 法では、将来の FCF を現在価値に割り引いて企業価値を求めます。したがって、`FCF` と `割引率` をどう置くかが核心です。

## 独立で押さえる後半論点

### 収益性指数法

`収益性指数法` は、投資 1 単位あたりどれだけ現在価値を生むかを見る指標です。資本制約がある場面では補助指標として役立ちますが、相互排他的な比較では `NPV` を優先しやすい点を区別する必要があります。

### 効率的市場仮説

証券投資論では、`弱度`、`準強度`、`強度` の違いを切って理解します。`どの情報がすでに価格へ反映済みか` を問う理論だと押さえると崩れにくくなります。

### マルチプル法

企業価値評価では `DCF法` だけでなく `PER`、`PBR`、`EV/EBITDA` を使う `マルチプル法` も比較対象になります。`株式価値` と `事業価値` のどちらを見ているかを切ることが重要です。

### MM理論と配当政策

`MM理論` と `配当政策` は、`企業価値を決めるのは何か` を考える理論です。`完全市場`、`法人税`、`節税効果` といった前提を切らずに読むと崩れやすいため、独立して押さえます。

### デリバティブとリスク管理

`デリバティブ` は、価格、為替、金利の変動リスクをどう抑えるかを考える論点です。`先物`、`オプション`、`スワップ` を、`固定したいのか`、`不利な方向だけ避けたいのか` で分けると整理しやすくなります。

## 後半で伸ばす論点

### ポートフォリオ理論

複数資産を組み合わせると、相関関係によってリスクが下がるという考え方です。計算問題として出ることがあるため、期待収益率と分散の基本は押さえます。

## 公式一覧：CAPM → WACC → NPV の計算チェーン

### CAPM（株主資本コストの算出）

```
Re = Rf + β × (Rm − Rf)

Rf: 無リスク利子率、Rm: 市場期待収益率
β: 市場に対する感応度、(Rm−Rf): 市場リスクプレミアム
```

| βの値 | 意味 |
| --- | --- |
| β = 1 | 市場と同じリスク |
| β > 1 | 市場より高リスク（景気敏感株） |
| 0 < β < 1 | 市場より低リスク（ディフェンシブ株） |

### WACC（加重平均資本コスト）

```
WACC = E/(D+E) × Re + D/(D+E) × Rd × (1−t)

E: 株主資本の時価、D: 負債の時価
Re: 株主資本コスト、Rd: 負債コスト（借入金利）
t: 法人税率
```

負債コストに `(1−t)` を掛ける → **タックスシールド**（利息の節税効果）を反映。資本構成は原則**時価ベース**。

### NPV（正味現在価値）

```
NPV = Σ{CFt ÷ (1+r)^t} − 初期投資

r: 割引率（通常 WACC）
CFt: t期のキャッシュフロー
```

NPV > 0 → 投資実行。複数案があるときは NPV が最大の案を選ぶ。

### IRR（内部収益率）

```
Σ{CFt ÷ (1+IRR)^t} − 初期投資 = 0
→ NPV がゼロになる割引率
```

IRR > WACC → 投資実行。ただし相互排他案では NPV を優先する。

### 企業価値（DCF法）

```
企業価値 = Σ{FCFt ÷ (1+WACC)^t} + TV ÷ (1+WACC)^n
TV（ターミナルバリュー） = FCFn+1 ÷ (WACC − g)
株式価値 = 企業価値 − 有利子負債
```

### ポートフォリオ理論（2資産）

```
期待収益率: E(Rp) = w₁E(R₁) + w₂E(R₂)
分散: σp² = w₁²σ₁² + w₂²σ₂² + 2w₁w₂ρ₁₂σ₁σ₂
```

相関係数 ρ が 1 未満なら、分散投資でリスクが低減される。

## 計算手順

### CAPM → WACC → NPV の一連の計算

```
【データ】
  無リスク利子率 2%、市場リスクプレミアム 6%、β = 1.5
  負債コスト 4%、法人税率 30%
  株主資本の時価 800万円、負債の時価 200万円
  投資案: 初期投資 500万円、5年間のCF 各150万円

手順1: CAPMで株主資本コストを求める
  Re = 2% + 1.5 × 6% = 11%

手順2: WACCを求める
  WACC = 800/1000 × 11% + 200/1000 × 4% × (1−0.3)
       = 8.8% + 0.56% = 9.36%

手順3: NPVを求める
  年金現価係数(5年, 9.36%) ≒ 3.847
  NPV = 150 × 3.847 − 500 = 577 − 500 = +77万円

判定: NPV > 0 → 投資実行
```

### 企業価値の計算例

```
【データ】
  FCF 年間 100万円（永続）、WACC 8%、成長率 2%
  有利子負債 300万円

企業価値 = 100 ÷ (0.08 − 0.02) = 100 ÷ 0.06 ≒ 1,667万円
株式価値 = 1,667 − 300 = 1,367万円
```

### ポートフォリオの計算例

```
資産A: 期待収益率 12%、σ=20%
資産B: 期待収益率 6%、σ=10%
ρ = 0.3、投資比率 A:B = 60:40

E(Rp) = 0.6×12 + 0.4×6 = 9.6%
σp² = 0.36×0.04 + 0.16×0.01 + 2×0.6×0.4×0.2×0.1×0.3
     = 0.0144 + 0.0016 + 0.00288 = 0.01888
σp = √0.01888 ≒ 13.7%

加重平均σ = 0.6×20+0.4×10 = 16% → 13.7%に低下 = 分散投資効果
```

## 典型的なつまずき

- CAPM、WACC、NPV を別々の式として覚えてしまう → CAPM→Re→WACC→NPV の**チェーン**で理解する
- WACC の資本構成を**簿価**で考えてしまう → 原則**時価ベース**（株式時価 = 株価×株式数）
- 負債コストに `(1−t)` を掛け忘れる → タックスシールドを反映しないと WACC が過大になる
- IRR が高ければ常によいと単純化する → 相互排他案では NPV 優先。IRR は再投資収益率の暗黙の仮定がある
- FCF と企業価値評価のつながりを見失う → 企業価値 = FCF の現在価値合計
- ポートフォリオの標準偏差を加重平均で計算してしまう → 共分散項を含む分散公式を使う

## 確認問題

<Callout title="問1：CAPM と WACC">
    無リスク利子率 3%、市場期待収益率 9%、β = 1.2。負債コスト 5%、法人税率 30%。株主資本時価 600万円、負債時価 400万円。WACC を求めよ。

    **解答**：Re = 3 + 1.2×(9−3) = 10.2%。WACC = 600/1000×10.2 + 400/1000×5×(1−0.3) = 6.12 + 1.40 = 7.52%。
</Callout>

<Callout title="問2：NPV 判定">
    初期投資 1,000万円。3年間のCF：1年目 400万、2年目 500万、3年目 300万。割引率 10%。NPV を求め、投資の可否を判定せよ。

    **解答**：PV = 400/1.1 + 500/1.21 + 300/1.331 = 364 + 413 + 225 = 1,002万円。NPV = 1,002 − 1,000 = +2万円。NPV > 0 なので投資実行。
</Callout>

<Callout title="問3：企業価値（DCF法）">
    FCF: 1年目 80万、2年目以降 毎年 100万（永続）。WACC 10%。有利子負債 500万円。株式価値を求めよ。

    **解答**：2年目以降の事業価値（1年末時点）= 100÷0.10 = 1,000万。企業価値 = 80/1.1 + 1,000/1.1 = 72.7 + 909.1 = 981.8万円。株式価値 = 981.8 − 500 = 481.8万円。
</Callout>

## 問題を解くときの観点

- いま求めたいのは `要求収益率` か、`投資案の価値` か
- この式の分子と分母は何を意味するか
- 資金調達コストと投資収益率を比較しているのか
- 前提は単独資産か、企業全体か
