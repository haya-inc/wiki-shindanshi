## このページの役割

このページは、`経営分析` のうち `安全性` を固める知識ノードです。短期の支払い余力と長期の財務基盤を分けて見て、`どこが危ないか` を指標ごとに説明できる状態を目指します。

## 学習のポイント

- まず `流動比率` と `当座比率` の違い
- 次に `自己資本比率` と `固定比率 / 固定長期適合率`
- そのうえで `高い / 低い` の意味を `資金繰り` と結びつける

## 主要指標の比較

| 指標 | 何を見るか | 典型的な読み方 |
| --- | --- | --- |
| 流動比率 | 流動負債に対する流動資産の厚さ | 高いほど短期安全性は高い |
| 当座比率 | すぐ現金化しやすい資産で短期負債を賄えるか | 在庫を除いた安全性を見る |
| 自己資本比率 | 総資本のうち自己資本がどれだけあるか | 高いほど借入依存が低い |
| 固定比率 | 固定資産を自己資本でどこまで賄えているか | 低いほど無理が少ない |
| 固定長期適合率 | 固定資産を自己資本と固定負債で賄えているか | `100%以下` が基本形 |

## 短期安全性と長期安全性を分ける

`流動比率` と `当座比率` は短期の支払い余力を見る指標です。`流動比率` が高くても `当座比率` が低いときは、在庫に資金が滞留している可能性があります。

一方で、`自己資本比率`、`固定比率`、`固定長期適合率` は長期の財務基盤を見る指標です。固定資産を短期資金で賄っていると、設備投資が資金繰りを圧迫しやすくなります。

## 危険信号として見たい型

| 状況 | 疑いやすいこと | 追加で見たい指標 |
| --- | --- | --- |
| 流動比率は高いが当座比率は低い | 在庫が多い | 棚卸資産回転率 |
| 自己資本比率が低い | 借入依存が高い | 固定長期適合率、利息負担 |
| 固定長期適合率が高い | 固定資産を短期資金で賄っている | 流動比率、短期借入金の動き |

## 公式一覧

### 短期安全性（流動性）

| 指標 | 計算式 | 目安 |
| --- | --- | --- |
| 流動比率 | 流動資産 ÷ 流動負債 × 100 | 200%以上が理想、120%以上が目安 |
| 当座比率 | 当座資産 ÷ 流動負債 × 100 | 100%以上が理想 |
| 手元流動性比率 | (現金預金 + 有価証券) ÷ 月商 | 1か月分以上 |

当座資産 = 現金預金 + 売掛金 + 受取手形 + 有価証券（棚卸資産を含まない）。

### 長期安全性

| 指標 | 計算式 | 目安 |
| --- | --- | --- |
| 固定比率 | 固定資産 ÷ 自己資本 × 100 | 100%以下が理想 |
| 固定長期適合率 | 固定資産 ÷ (自己資本 + 固定負債) × 100 | 100%以下が**必須** |

固定比率 ≧ 固定長期適合率（常に成立。分母が大きいほど比率は小さい）。固定長期適合率 100%超 → 固定資産を短期借入で賄っている → **危険信号**。

### 資本構成

| 指標 | 計算式 | 見方 |
| --- | --- | --- |
| 自己資本比率 | 自己資本 ÷ 総資産 × 100 | 40%以上が望ましい |
| 負債比率 | 負債 ÷ 自己資本 × 100 | 100%以下が望ましい |
| D/Eレシオ | 有利子負債 ÷ 自己資本 | 低いほど安全 |
| ICR | (営業利益 + 受取利息配当金) ÷ 支払利息 | 1.0以下→利息を払えない |
| 債務償還年数 | 有利子負債 ÷ 営業CF | 短いほど返済能力が高い |

```
自己資本比率 + 他人資本比率 = 100%
自己資本比率 50% → 負債比率 100%
自己資本比率 40% → 負債比率 150%
自己資本比率 25% → 負債比率 300%
```

## 計算手順

### 安全性指標の計算例

```
【B/S データ】
流動資産 600（うち現金200、売掛金200、在庫200）
固定資産 400
資産合計 1,000
流動負債 300、固定負債 200、自己資本 500

手順1: 短期安全性
  流動比率 = 600 ÷ 300 × 100 = 200% → 理想的
  当座資産 = 200 + 200 = 400（在庫除く）
  当座比率 = 400 ÷ 300 × 100 = 133% → 良好

手順2: 長期安全性
  固定比率 = 400 ÷ 500 × 100 = 80% → 100%以下で良好
  固定長期適合率 = 400 ÷ (500+200) × 100 = 57% → 良好

手順3: 資本構成
  自己資本比率 = 500 ÷ 1,000 × 100 = 50% → 良好
```

### 安全性指標の改善策

| 改善したい指標 | 施策 |
| --- | --- |
| 流動比率 | 短期借入→長期借入への借換、売掛金の早期回収、不要在庫の削減 |
| 当座比率 | 売掛金の早期回収、手形割引・ファクタリング |
| 自己資本比率 | 利益の内部留保、増資、不要資産売却→負債返済 |
| 固定長期適合率 | 遊休固定資産の売却、長期資金への借換、設備投資の抑制 |

## 問題を解くときの観点

- いま見ているのは `短期資金繰り` か `長期の財務基盤` か
- 流動資産の中身は `現預金 / 売掛金 / 在庫` のどれに寄っているか
- 固定資産を賄う資金は `自己資本` と `固定負債` で足りているか

## 典型的なつまずき

- `流動比率` が高ければ十分だと判断して `当座比率` を見ない → 在庫が膨らんでいるだけの場合あり
- `固定比率` と `固定長期適合率` の分母の違いを混同する → 固定比率は**自己資本のみ**、固定長期適合率は**自己資本＋固定負債**
- 借入依存の高さを `自己資本比率` だけで終わらせる → ICR や D/E レシオも確認
- 当座資産に棚卸資産を含めてしまう → 当座資産は現金・売掛金・受取手形・有価証券のみ
- 固定長期適合率が 100%超の深刻さを軽視する → 短期資金で固定資産を賄う状態は危険

## 確認問題

<Callout title="問1：指標の計算">
    流動資産 450百万円（うち棚卸資産 150）、流動負債 300百万円。流動比率と当座比率を求めよ。

    **解答**：流動比率 = 450÷300×100 = 150%。当座資産 = 450−150 = 300。当座比率 = 300÷300×100 = 100%。流動比率は目安を満たすが、当座比率がぎりぎりなので在庫への依存度が高い。
</Callout>

<Callout title="問2：固定長期適合率の判定">
    固定資産 800百万円、自己資本 400百万円、固定負債 300百万円。固定比率と固定長期適合率を求め、問題点を指摘せよ。

    **解答**：固定比率 = 800÷400×100 = 200%（自己資本だけでは賄えない）。固定長期適合率 = 800÷(400+300)×100 = 114%（100%超のため、固定資産の一部を短期借入で賄っている状態。資金繰りリスクあり）。
</Callout>

<Callout title="問3：改善策の判断">
    自己資本比率が 20% の企業が安全性を高めるための施策として適切なものはどれか。(a) 配当金の増額 (b) 自己株式の取得 (c) 利益の内部留保と増資 (d) 短期借入金の増加。

    **解答**：(c)。利益の内部留保と増資は自己資本を増やし、自己資本比率を改善する。(a)(b) は自己資本を減らす方向、(d) は負債を増やす方向で、いずれも悪化要因。
</Callout>
