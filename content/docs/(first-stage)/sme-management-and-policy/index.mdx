中小企業経営・中小企業政策は、`中小企業の実態` と `政策・支援施策` を扱う科目です。白書、統計、補助金、取引適正化、支援制度など、年度ごとに更新される情報が多く、他科目よりも `いつ時点の情報か` を意識する必要があります。

<Callout title="この科目の位置づけ">
  白書や統計は `何を示す数字か`、補助金や支援策は `対象者`、`目的`、`支援手段` で整理します。安定情報と年度更新を分けることが前提です。
</Callout>

## 学習のポイント

<Cards>
  <Card title="中小企業経営">
    中小企業・小規模事業者の定義、企業数、開廃業、経営課題を、`何を示す数字か` で整理します。
  </Card>
  <Card title="中小企業政策">
    補助金や支援策は、名称ではなく `対象者`、`目的`、`支援手段` で整理すると崩れにくくなります。
  </Card>
  <Card title="更新論点">
    定義や政策体系は安定して覚え、補助金や制度の細目は年度ごとに更新確認する前提で整理します。
  </Card>
</Cards>

## この科目で問われること

<Steps>
  <Step>
    ### 定義や実態を理解しているか

    中小企業・小規模事業者の定義、企業数、従業者数、開廃業の流れを理解しているかが問われます。
  </Step>
  <Step>
    ### 白書や統計を読めるか

    数字を丸暗記するのではなく、白書が示すテーマや環境変化を読み取る力が求められます。
  </Step>
  <Step>
    ### 政策体系を区別できるか

    法律、金融、補助金、伴走支援、取引適正化のような支援体系を、目的別に整理できることが重要です。
  </Step>
</Steps>

## 2026-03-27 時点の更新確認メモ

<Tabs items={["白書", "制度", "補助金", "人材・海外展開"]}>
  <Tab value="白書">
    - 2026-03-26 時点で確認できる最新の中小企業白書・小規模企業白書は `2025年版` です。
    - 2025年版では、円安・物価高、`金利のある世界`、構造的な人手不足の下で、経営者の `経営力` を主題に据えています。
  </Tab>
  <Tab value="制度">
    - `小規模企業振興基本計画（第Ⅲ期）` が `2025-03-25` に閣議決定されています。
    - 旧 `下請法` は `2026-01-01` から `取適法` に変わっており、旧称 / 新称の両方で押さえる必要があります。
  </Tab>
  <Tab value="補助金">
    - 持続化、IT導入、ものづくり、省力化、事業承継・M&A は年度や公募回で条件が動きます。
    - 本文では役割を整理し、詳細は公式公募要領を見る前提にします。
  </Tab>
  <Tab value="人材・海外展開">
    - `人材確保等支援助成金` と `中小企業大学校` は、人材不足対応を `定着支援` と `育成支援` で分けて理解する入口になります。
    - `新規輸出1万者支援プログラム` と `JETRO / 中小機構` の連携支援は、海外展開を `相談`、`伴走`、`金融・保険` の流れで捉える方が現行制度に合います。
  </Tab>
</Tabs>

## 典型的なつまずき

- 白書の具体的な数値だけを覚えて、意味や方向感を押さえない
- 中小企業の定義と、小規模企業者の定義、制度ごとの定義の違いを混同する
- 補助金の名称ばかりを覚えて、対象者や目的を区別できない
- 毎年変わる情報と、安定して問われる枠組みを分けずに学習する
- 旧称と新称が混ざる移行期の制度を取り違える
