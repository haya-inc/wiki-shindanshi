## このページの役割

このページは、`中小企業とは何か` を定義から固める知識ノードです。`中小企業者`、`小規模企業者`、`業種ごとの基準`、`制度ごとに定義がずれる場面` を、試験で混ざりやすい順に整理します。

## 学習のポイント

- まず `中小企業者` と `小規模企業者` を分ける
- 次に `業種ごとの基準` を、資本金と従業員数のどちらで見るか整理する
- そのうえで `法律や制度ごとに定義が異なることがある` と押さえる

## 試験で何が問われるか

- `中小企業者` の定義を業種別に区別できるか
- `小規模企業者` の定義を別枠で言えるか
- `又は` の条件を正しく読めるか
- 制度の対象者を読むときに、`一般的な定義` と `その制度固有の定義` を混同しないか

## 定義の見方

### 中小企業者の定義

診断士試験では、まず `業種ごとに基準が違う` ことを押さえます。`製造業その他`、`卸売業`、`小売業`、`サービス業` は、資本金と従業員数の上限が異なります。

| 業種 | 資本金 | 従業員数 |
| --- | --- | --- |
| `製造業その他` | 3億円以下 | 300人以下 |
| `卸売業` | 1億円以下 | 100人以下 |
| `小売業` | 5,000万円以下 | 50人以下 |
| `サービス業` | 5,000万円以下 | 100人以下 |

試験では、表全体を丸暗記するより、`小売業は従業員数 50 人`、`卸売業は 100 人`、`製造業その他は 300 人` のように、差が出やすい箇所から固める方が崩れにくくなります。

### 小規模企業者の定義

`小規模企業者` は、中小企業者とは別の定義です。ここは `製造業その他` と `商業・サービス業` で切るのが基本です。したがって、`中小企業者の中でも、さらに小さい事業者を政策上どう捉えるか` という階層で理解します。

| 区分 | 従業員数 |
| --- | --- |
| `製造業その他` | 20人以下 |
| `商業・サービス業` | 5人以下 |

ここでいう `商業` は、卸売業、小売業、飲食店を含む整理で読むと混ざりにくくなります。

### OR 条件の読み方

定義で頻出なのは、`資本金又は従業員数` です。これは、両方を満たす必要があるわけではありません。したがって、片方が基準内なら中小企業者に当たる、という読み方を固定します。

## 制度ごとにずれる場面

中小企業の定義は、白書や基本法で使う一般的な定義と、補助金、税制、金融支援で使う定義が完全には一致しないことがあります。したがって、制度問題では `この制度の対象者は誰か` を本文で確認する必要があります。

試験で迷ったら、次の順で見ます。

1. 一般的な中小企業者の定義か
2. 小規模企業者の定義か
3. 制度固有の対象要件か

## 典型例

| 論点 | 典型的な見分け方 |
| --- | --- |
| `中小企業者` と `小規模企業者` | 一般的な区分か、より小さい層の政策区分か |
| `卸売業` と `小売業` | 流通段階の違いと従業員数基準の違いで分ける |
| `OR 条件` と `AND 条件` | どちらか一方で足りるか、両方必要か |
| `一般定義` と `制度要件` | 基本法上の定義か、制度本文の対象要件か |

## 典型的なつまずき

- `中小企業者` と `小規模企業者` を同じ意味で覚える
- `又は` を `かつ` のように読んでしまう
- 小売業とサービス業の基準を混同する
- 制度問題でも、一般的な定義だけで判断してしまう

## 問題を解くときの観点

- 聞かれているのは `一般定義` か `制度対象` か
- 業種は `製造業その他 / 卸売業 / 小売業 / サービス業` のどれか
- 条件は `又は` か
- `小規模企業者` まで聞かれているか
