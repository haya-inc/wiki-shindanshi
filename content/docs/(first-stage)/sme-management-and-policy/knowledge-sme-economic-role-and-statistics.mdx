## このページの役割

このページは、`中小企業が日本経済でどんな位置を占めるか` を読む知識ノードです。`企業数`、`従業者数`、`付加価値`、`地域雇用` を、数字の暗記ではなく `何を示す指標か` で整理します。

## 学習のポイント

- まず `企業数が多い` ことと `経済への影響が大きい` ことを区別する
- 次に `従業者数` と `付加価値` を別の意味として読む
- そのうえで `地域経済` と `サプライチェーン` への役割を押さえる

## 試験で何が問われるか

- 中小企業が `企業数` で大きな割合を占めることを説明できるか
- `従業者数` や `雇用` の面で重要な役割を果たすと読めるか
- `付加価値額` や `労働生産性` を、企業数とは別の観点として区別できるか
- 白書の統計表から `水準`、`増減`、`比較` を読み取れるか

## 主要統計の読み方

### 企業数

企業数は、`経済の裾野の広さ` を見る指標です。中小企業は企業数ベースでは非常に大きな割合を占めます。したがって、試験では `数として多数派である` という構図を外さないことが重要です。

### 従業者数

従業者数は、`雇用をどれだけ支えているか` を見る指標です。企業数の多さと合わせて、地域の雇用基盤としての役割を読めるようにします。

### 付加価値

付加価値は、`どれだけ価値を生み出しているか` を見る指標です。ここは企業数や従業者数と別です。企業数が多いことと、一社当たりの付加価値が高いことは同じではありません。

### 地域経済への役割

中小企業は、地域の生活、雇用、取引先ネットワークを支える存在です。したがって、白書では全国平均だけでなく、`地方での雇用維持`、`地域資源の活用`、`取引網の維持` の文脈で読むことが多くなります。

## 白書統計を読む順番

統計表を見たら、次の順に整理します。

1. `何を数えている表か`
2. `企業数 / 従業者数 / 付加価値 / 生産性` のどれか
3. `水準` を聞いているのか `増減` を聞いているのか
4. `規模別`、`業種別`、`地域別` のどの切り方か

## 典型例

| 論点 | 典型的な見分け方 |
| --- | --- |
| `企業数` と `従業者数` | 事業者の数を見る話か、雇用を見る話か |
| `従業者数` と `付加価値` | 人の多さを見る話か、価値創出を見る話か |
| `付加価値` と `労働生産性` | 総量を見る話か、一人当たりを見る話か |
| `全国の傾向` と `地域の役割` | マクロの構図か、地域経済の支え方か |

## 典型的なつまずき

- `企業数が多い` ことと `一社当たりの生産性が高い` ことを混同する
- 従業者数と付加価値を同じ意味として読む
- 統計表の数字だけを追って、何の指標かを見失う
- 地域経済での役割を、全国の総量だけで考えてしまう

## 問題を解くときの観点

- その統計は `数`、`雇用`、`価値創出` のどれを示すか
- `総量` を見ているか、`一人当たり` を見ているか
- `規模別`、`業種別`、`地域別` のどの比較か
- 数字の暗記より `何を示す表か` を先に言えるか
