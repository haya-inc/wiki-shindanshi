## このページの役割

このページは、`中小企業の新陳代謝をどう読むか` を整理する知識ノードです。`開業率`、`廃業率`、`休廃業・解散`、`倒産` を、意味の違いと試験での読み方から整理します。

## 学習のポイント

- まず `開業率` と `廃業率` を分ける
- 次に `休廃業・解散` と `倒産` を区別する
- そのうえで `率そのもの` より `トレンド` を読む

## 試験で何が問われるか

- `開業率` と `廃業率` の意味を説明できるか
- `休廃業・解散` と `倒産` を別概念として言えるか
- 新陳代謝の活発さを、単年の数字だけでなく傾向で読めるか
- 事業承継、高齢化、人手不足とのつながりを読めるか

## 主要指標の違い

### 開業率

開業率は、`新しく事業が始まる動き` を見る指標です。起業のしやすさ、成長期待、地域の活力などと結び付きます。

### 廃業率

廃業率は、`事業が閉じる動き` を見る指標です。新陳代謝として自然に起きる部分もありますが、後継者不在や採算悪化と結び付くこともあります。

### 休廃業・解散

`休廃業・解散` は、必ずしも法的な倒産ではありません。経営破綻ではなくても、事業をやめる場合が含まれます。試験では `倒産と同じではない` ことが重要です。

### 倒産

倒産は、資金繰り悪化や債務不履行など、経営破綻の色が強い概念です。したがって、`休廃業・解散` と `倒産` は別に整理します。

## 読み方の軸

| 指標 | 何を見るか |
| --- | --- |
| `開業率` | 新しい事業が生まれているか |
| `廃業率` | 既存事業がどれだけ退出しているか |
| `休廃業・解散` | 破綻以外も含めた退出の動き |
| `倒産` | 資金繰り悪化などによる経営破綻 |

## 背景としてつながる論点

開業率と廃業率は、起業環境だけでなく、後継者不在、経営者高齢化、人手不足、収益力不足ともつながります。したがって、数字だけでなく `なぜ増えたか / 減ったか` を読む必要があります。

## 典型例

| 論点 | 典型的な見分け方 |
| --- | --- |
| `開業率` と `廃業率` | 入る動きか、出る動きか |
| `休廃業・解散` と `倒産` | 破綻を含むか、含まない退出もあるか |
| `廃業` と `事業承継` | やめる話か、引き継いで続ける話か |

## 典型的なつまずき

- 開業率と廃業率を逆に覚える
- `休廃業・解散` をすべて倒産として扱う
- 単年の数値だけで良し悪しを判断する
- 事業承継問題とのつながりを見落とす

## 問題を解くときの観点

- この指標は `参入` と `退出` のどちらを見るか
- `破綻` を含む話か
- 増減の背景に `高齢化`、`後継者不在`、`起業環境` のどれがあるか
- 数値暗記ではなく、`構造変化` として説明できるか
