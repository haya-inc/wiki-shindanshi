## このページの役割

このページは、`近年の白書で繰り返し問われる経営課題` を整理する知識ノードです。`人手不足`、`賃上げ`、`価格転嫁`、`DX`、`事業承継`、`生産性向上` を、単語の列挙ではなく `どうつながる課題か` で読めるようにします。

## 学習のポイント

- まず `人手不足` と `賃上げ` を分ける
- 次に `価格転嫁` と `収益力` を結び付ける
- そのうえで `DX`、`事業承継`、`生産性向上` を成長課題として見る

## 試験で何が問われるか

- 中小企業の課題を `人材`、`収益`、`事業継続`、`成長投資` に分けて説明できるか
- `賃上げ` と `価格転嫁` の関係を言えるか
- `DX` を単なる IT 導入ではなく、付加価値向上や省力化と結び付けて読めるか
- `事業承継` を高齢化だけでなく、雇用や地域経済の維持と結び付けて読めるか

## 主要な経営課題

### 人手不足と賃上げ

人手不足は、採用難、離職、防衛的な賃上げと結び付きやすい論点です。したがって、`人が足りない` と `賃金を上げる必要がある` を一体で読む必要があります。

### 価格転嫁と収益力

賃上げや原材料高への対応には、収益力の確保が必要です。ここで頻出なのが `価格転嫁` です。適切に価格へ反映できないと、利益が圧迫され、賃上げ余力も弱くなります。

### 生産性向上と DX

中小企業では、人数を増やしにくい中で `一人当たりの付加価値` を高める必要があります。そのため、設備投資、業務改善、デジタル化、見える化を通じた生産性向上が重要になります。

### 事業承継

事業承継は、単に経営者が交代する話ではありません。廃業回避、技術や雇用の維持、地域経済の継続とも結び付きます。したがって、`承継できないと何が失われるか` まで含めて読むと、政策論点にもつながります。

## 課題どうしのつながり

| 出発点 | つながる課題 |
| --- | --- |
| `人手不足` | 採用難、離職防止、賃上げ、省力化 |
| `原材料高 / コスト上昇` | 価格転嫁、利益圧迫、資金繰り |
| `収益力不足` | 賃上げ余力不足、投資不足、生産性停滞 |
| `経営者高齢化` | 事業承継、廃業、地域雇用の維持 |
| `成長投資の遅れ` | DX、生産性向上、付加価値向上 |

## 典型例

| 論点 | 典型的な見分け方 |
| --- | --- |
| `人手不足` と `賃上げ` | 人の確保の問題か、賃金水準の問題か |
| `価格転嫁` と `値上げ` | コスト反映か、より広い価格戦略か |
| `DX` と `IT化` | 省力化や付加価値向上まで含むか |
| `事業承継` と `廃業` | 引き継いで続ける話か、やめる話か |

## 典型的なつまずき

- 課題を単語ごとにばらばらに覚える
- `人手不足` を採用問題だけで終わらせる
- `価格転嫁` を政策論点だけと見て、収益力との関係を見落とす
- `DX` を機械導入と同義にしてしまう
- `事業承継` を高齢化の話だけとして読む

## 問題を解くときの観点

- この設問は `人材`、`収益`、`成長投資`、`事業継続` のどれを聞いているか
- 課題の背景は `コスト上昇`、`競争環境`、`後継者不在` のどれか
- 企業側の対応と政策側の対応を分けて説明できるか
- 個別論点の背後にある `付加価値向上` の軸を見失っていないか
