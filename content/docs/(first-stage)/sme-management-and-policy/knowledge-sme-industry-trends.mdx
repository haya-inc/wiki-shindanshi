## このページの役割

このページは、`業種ごとに中小企業の見え方がどう変わるか` を整理する知識ノードです。`製造業`、`建設業`、`卸売業`、`小売業`、`サービス業` を、雇用、人手不足、価格転嫁、生産性、取引構造の違いで読み分けます。

## 学習のポイント

- まず `業種によって課題の出方が違う` と押さえる
- 次に `価格転嫁しやすさ` と `人手不足の強さ` を見る
- そのうえで `生産性` と `取引構造` の違いを読む

## 試験で何が問われるか

- 製造業とサービス業の違いを、付加価値や生産性の観点で説明できるか
- 建設業、小売業、サービス業の人手不足の強さを読めるか
- 卸売業や下請構造で `価格転嫁` や `取引関係` が論点になると分かるか
- 白書の業種別グラフから、どの業種で課題が目立つかを読み取れるか

## 業種別の見方

### 製造業

製造業は、設備投資、取引先との関係、生産性向上が論点になりやすい業種です。価格転嫁やサプライチェーンとの関係も読みやすく、白書では基準になることが多くあります。

### 建設業

建設業は、人手不足と高齢化が強く出やすい業種です。技能継承や現場人材の確保、労務費や資材費の転嫁が重要な読みどころになります。

### 卸売業

卸売業は、川上と川下の間に立つため、価格転嫁や取引条件の変化が論点になりやすい業種です。製造業、小売業の間にあるため、景況やコスト上昇の影響を受けやすい点も押さえます。

### 小売業

小売業は、消費動向、人手不足、店舗運営、販売価格への転嫁が主要論点です。来店客数、購買単価、物価動向の影響も読みやすい業種です。

### サービス業

サービス業は、人手依存度が高く、賃上げや採用難の影響を受けやすい業種です。業務の標準化やデジタル化が進みにくい分、労働生産性が論点になりやすくなります。

## 比較するときの軸

| 比較軸 | 何を見るか |
| --- | --- |
| `人手不足` | 現場人材の不足が強いか |
| `価格転嫁` | コスト上昇を価格へ反映しやすいか |
| `生産性` | 一人当たり付加価値を高めやすいか |
| `取引構造` | 下請、顧客接点、地域密着のどれが強いか |

## 典型例

| 論点 | 典型的な見分け方 |
| --- | --- |
| `製造業` と `サービス業` | 設備や物の流れが中心か、人への依存が強いか |
| `卸売業` と `小売業` | 中間流通か、最終消費者との接点か |
| `建設業` と `製造業` | 現場施工が中心か、工場生産が中心か |
| `小売業` と `サービス業` | 商品販売が中心か、役務提供が中心か |

## 典型的なつまずき

- 全業種を同じ課題で一括りにしてしまう
- サービス業の課題を小売業と同じように読む
- 価格転嫁を製造業だけの話と思い込む
- 業種別グラフで `どの業種が目立つか` を見ずに平均だけを見る

## 問題を解くときの観点

- この設問は `どの業種` の話か
- `人手不足`、`価格転嫁`、`生産性` のどれが主軸か
- その業種の特徴は `現場依存`、`設備依存`、`顧客接点`、`流通段階` のどれか
- 他業種との比較で、どこが違いとして使われているか
