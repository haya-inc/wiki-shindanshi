## 第1層：BCPと事業継続力強化計画の概念

### BCP（事業継続計画）とは

BCP（Business Continuity Plan）は、企業が自然災害、感染症、テロ、サイバー攻撃などの緊急事態に直面した場合、その影響を最小限に抑え、重要な事業機能を継続、またはできるだけ早期に復旧させるための計画です。

BCPの基本的な要素は以下の通りです。

| 要素 | 説明 |
| --- | --- |
| **リスク分析** | 企業に影響を与える可能性のある緊急事態を特定・分析 |
| **中核事業の特定** | 存続させなければならない重要な事業を明確化 |
| **目標復旧時間（RTO）の設定** | 事業を復旧させるまでの許容時間を決定 |
| **代替手段の確保** | 被災時の拠点替え、要員確保、外部委託などの対策を準備 |
| **訓練・検証** | 定期的にシミュレーションを行い、計画の実効性を確認 |
| **見直し** | 事業環境の変化に応じて計画を更新 |

### 事業継続力強化計画（中小企業向けBCP）

事業継続力強化計画は、中小企業等経営強化法（2014年制定）に基づく認定制度です。一般的なBCPより策定のハードルを低く設定し、中小企業が取り組みやすい形に整理されています。

| 項目 | 内容 |
| --- | --- |
| **法的根拠** | 中小企業等経営強化法 |
| **認定機関** | 経済産業大臣（実務は経済産業局） |
| **対象企業** | 中小企業者（単独型）、複数の中小企業者が連携（連携型） |
| **計画期間** | 3年間 |

計画の内容は、以下の4つの段階で構成されます。

1. **リスク認識段階**：自然災害、感染症、サプライチェーン寸断などのリスク認識
2. **事前対策段階**：ヒト（従業員の確保・訓練）、モノ（代替生産拠点、設備等）、カネ（資金確保）、情報（データバックアップ）の対策
3. **初動対応段階**：被災直後の重要業務の継続、従業員・顧客への対応、情報発信
4. **復旧段階**：事業の段階的復旧計画

## 第2層：BCP と事業継続力強化計画の比較

両者の位置づけの違いを表でまとめます。

| 項目 | BCP（一般的） | 事業継続力強化計画 |
| --- | --- | --- |
| **性格** | 包括的な事業継続計画（業種・企業規模を問わない） | 中小企業向けの簡易版・入口 |
| **法的根拠** | 特になし（任意策定） | 中小企業等経営強化法 |
| **認定制度** | なし | 経済産業大臣認定あり |
| **策定主体** | 企業独自（外部コンサルタント活用も多い） | 簡易なフォーマットで自社策定可能 |
| **対象リスク** | 企業が特定したあらゆる緊急事態 | 主に自然災害、感染症、サプライチェーン寸断 |
| **計画期間** | 企業独自に設定（1年～5年程度が多い） | 3年間 |
| **メリット** | 自社のリスク管理体制を強化できる | 認定による各種支援措置が受けられる |
| **策定のハードル** | 高い（経営層の判断、専門知識必要、コスト負担） | 比較的低い（簡易なフォーマット、指定様式） |

## 第3層：事業継続力強化計画の認定による支援措置

事業継続力強化計画の認定を受けると、以下の5つの支援措置が活用できます。

### 支援措置1：税制措置（特別償却）

防災・減災設備等の特別償却（20%）が適用されます。

| 項目 | 内容 |
| --- | --- |
| **対象企業** | 事業継続力強化計画の認定を受けた中小企業者 |
| **対象設備** | 防災・減災設備（認定後に取得した設備） |
| **特別償却率** | 20%（当該資産の基礎価額の20%を初年度に特別償却可能） |
| **適用時期** | 認定後に取得した設備が対象（認定前に取得した設備は対象外） |
| **効果** | 初年度の減価償却費が増加し、課税所得が減少 |

**重要な留意点**：特別償却は「償却資産」に限定されます。土地や無形資産は対象外です。

### 支援措置2：金融支援（低利融資）

日本政策金融公庫（JFC）による低利融資が利用できます。

| 項目 | 内容 |
| --- | --- |
| **融資主体** | 日本政策金融公庫 |
| **対象企業** | 事業継続力強化計画の認定を受けた中小企業者、小規模事業者 |
| **融資対象** | 防災・減災に関連する設備投資、施設整備等 |
| **金利** | 通常の融資より低利 |
| **融資期間** | 最大15年程度（設備により異なる） |

### 支援措置3：補助金加点

ものづくり補助金や事業革新補助金などの審査において、事業継続力強化計画の認定企業が加点対象となります。

| 項目 | 内容 |
| --- | --- |
| **対象補助金** | ものづくり補助金、事業革新補助金ほか複数 |
| **加点内容** | 評価点に対する加算（加点率は補助金ごとに異なる） |
| **効果** | 採択可能性が高まる |

### 支援措置4：信用保証枠の追加

信用保証協会の別枠保証が利用できます。

| 項目 | 内容 |
| --- | --- |
| **保証主体** | 都道府県信用保証協会 |
| **枠組み** | 一般保証と別枠（上限額が独立） |
| **保証率** | 100%保証が多い |
| **対象融資** | 運転資金、設備資金 |

### 支援措置5：損害保険料の割引

一部の損害保険会社が認定企業向けの割引を提供します。

| 項目 | 内容 |
| --- | --- |
| **対象保険** | 火災保険、地震保険など |
| **割引率** | 保険会社ごとに異なる（通常5～10%程度） |
| **条件** | 事業継続力強化計画の認定を受けていることを確認 |

## 第4層：実際の適用要件と誤答パターン

### 誤答パターン1：認定前に取得した設備も特別償却の対象になる

**誤り**：認定を受ければ、それまでに取得した防災設備も特別償却できると思うこと

**正解**：特別償却が適用されるのは、認定**後**に取得した設備です。認定前に取得した設備は、標準的な減価償却で処理されます。

### 誤答パターン2：事業継続力強化計画の認定自体が補助金である

**誤り**：認定を受けることで現金の補助金が直接支給されると思うこと

**正解**：認定そのものは補助金ではありません。認定を受けることで、税制優遇（特別償却）、金融支援（低利融資）、補助金の加点といった各種支援措置への**入口**になります。

### 誤答パターン3：セーフティネット保証4号は事業継続力強化計画の支援制度である

**誤り**：セーフティネット保証4号を、事業継続力強化計画の認定による支援の一部だと思うこと

**正解**：セーフティネット保証4号は、事業継続力強化計画とは別の制度です。被災後の企業が利用する緊急保証制度であり、認定計画とは無関係に利用可能です。

### 誤答パターン4：BCP と事業継続力強化計画は同じもの

**誤り**：一般的なBCPと事業継続力強化計画を完全に同一視すること

**正解**：事業継続力強化計画は、中小企業等経営強化法に基づく認定制度であり、一般的なBCPより策定のハードルが低く、法的根拠と認定による支援措置が異なります。

## 第5層：災害時の支援制度（被災後の復旧支援）

実際に被災した場合に利用できる支援制度を、平時の事前対策とは別の層として整理します。

### セーフティネット保証4号（自然災害等対応）

突発的な自然災害の影響を受けた中小企業者向けの信用保証制度です。

| 項目 | 内容 |
| --- | --- |
| **根拠** | 中小企業信用保険法 |
| **対象企業** | 中小企業者（売上高が前年同月比20%以上減少） |
| **保証対象** | 運転資金・設備資金 |
| **保証率** | 100%保証（別枠） |
| **保証限度額** | 1社当たり2億8,000万円 |
| **手続き** | 市区町村長の認定が必要 |
| **利用期間** | 災害発生から原則1年以内の申込 |

**事業継続力強化計画との関係**：セーフティネット保証4号は、認定計画の有無に関わらず、被災したすべての中小企業が利用できます。

### 災害復旧貸付

日本政策金融公庫による被災企業向けの融資です。

| 項目 | 内容 |
| --- | --- |
| **融資主体** | 日本政策金融公庫 |
| **対象企業** | 被災した中小企業、小規模事業者 |
| **融資目的** | 施設・設備の復旧、災害に伴う負債の返済 |
| **金利** | 特別利率（通常より低い） |
| **融資期間** | 最大15年程度（条件により異なる） |
| **申込方法** | 市区町村の罹災証明書が必要な場合が多い |

### なりわい再建補助金（被災事業者向け）

自然災害で被災した中小企業・小規模事業者の施設等の復旧を支援する補助金です。

| 項目 | 内容 |
| --- | --- |
| **対象企業** | 中小企業者、小規模事業者 |
| **対象経費** | 施設・設備の復旧、再建に必要な経費 |
| **補助率** | 3/4（中小企業者）、4/5（小規模事業者） |
| **補助上限** | 大規模災害ごとに異なる（通常1,000万円～3,000万円程度） |
| **実施時期** | 大規模災害時のみ（法律で定められた指定災害対象） |

### 小規模事業者持続化補助金＜災害支援枠＞

被災した小規模事業者の事業再建を支援する補助金です。

| 項目 | 内容 |
| --- | --- |
| **対象企業** | 小規模事業者（従業員20人以下） |
| **対象経費** | 再建に必要な経費（機械装置、備品、テレワーク機器など） |
| **補助率** | 2/3～4/5（経費の種類により異なる） |
| **補助上限** | 100万円～200万円程度（時期により異なる） |
| **実施時期** | 大規模災害の都度、補正予算で措置 |

## 事業継続力強化計画と他の認定計画制度の位置づけ

中小企業等経営強化法に基づく認定制度は複数あります。以下は、それぞれの計画の目的と認定機関を整理した表です。

| 計画制度 | 法的根拠 | 目的 | 認定機関 |
| --- | --- | --- | --- |
| **経営革新計画** | 中小企業等経営強化法 | 新事業活動による経営の向上 | 都道府県知事 |
| **経営力向上計画** | 中小企業等経営強化法 | 生産性向上、競争力強化 | 主務大臣 |
| **先端設備等導入計画** | 生産性向上特別措置法 | 先端設備・ソフトウェアの導入 | 市区町村 |
| **事業継続力強化計画** | 中小企業等経営強化法 | 防災・減災の事前対策、事業継続体制の構築 | 経済産業大臣 |

## 典型的なつまずき

### つまずき1：認定のメリットと補助金を混同する

学習者は「認定を受けると補助金がもらえる」と理解しがちですが、正確には「認定により、税制優遇、金融支援、補助金の加点などの入口が開かれる」です。補助金そのものの給付は、別途、補助金制度ごとの公募に応募して採択されることが前提になります。

### つまずき2：事業継続力強化計画とセーフティネット保証4号を同じ制度として記憶する

事業継続力強化計画は「平時の事前対策の認定制度」、セーフティネット保証4号は「被災後の緊急保証」という異なるレイヤーの制度です。被災企業がセーフティネット保証4号を利用するために、事業継続力強化計画の認定は不要です。

### つまずき3：BCP と事業継続力強化計画を完全に同一視する

一般的なBCPは任意策定であり法的強制力がありませんが、事業継続力強化計画は、法律に基づく認定制度として、計画の要件が指定様式で明確に定義されています。

### つまずき4：特別償却の適用時期を誤る

特別償却（20%）は、認定**後**に取得した防災設備に限定されます。認定前に購入した設備は対象外です。また、償却資産（機械装置、建物付属設備）が対象であり、土地や無形資産は対象外です。

## 問題を解くときの観点

### 観点1：事前対策か、被災後支援か

設問で言及されている企業の状態を確認してください。
- **「防災対策を進めている」「今から事業継続計画を作ろうとしている」** → 事業継続力強化計画、BCP、認定による支援措置の層
- **「先月の台風で被災した」「売上が20%減少した」** → セーフティネット保証、災害復旧貸付、復興補助金の層

### 観点2：認定のメリットが何か

事業継続力強化計画の認定を受けた場合の5つの支援措置を個別に指摘できるか確認しましょう。
- 税制（特別償却20%、ただし認定後取得が条件）
- 金融（JFCの低利融資）
- 補助金（各種補助金の加点）
- 保証（信用保証協会の別枠保証）
- 保険（損害保険料割引）

### 観点3：要件を正確に理解しているか

特に以下の要件は頻出です。
- セーフティネット保証4号：売上高が前年同月比**20%以上減少**（20%未満は対象外）
- 市区町村長の認定が別途必要
- 特別償却：認定**後**に取得した設備が対象

### 観点4：計画制度の使い分け

経営革新計画、経営力向上計画、先端設備等導入計画、事業継続力強化計画のうち、どの計画が設問で求められているのかを区別してください。

## 確認問題

<Callout title="確認問題1">
    **問題**：事業継続力強化計画の認定を受けた中小製造業が、防災設備（防水パネル、自家発電機など）を認定の翌月に500万円で取得しました。この場合、防災設備に対して適用できる税制優遇の内容を説明してください。また、認定の**1年前**に取得していた場合はどうなるかも述べてください。

    **解答例**：
    認定後に取得した防災設備（500万円）に対しては、特別償却（20%）が適用されます。基礎価額500万円の20%である100万円を初年度に特別償却でき、初年度の減価償却費が100万円増加して課税所得が減少します。

    一方、認定の1年前に取得した設備については、特別償却の適用対象外です。なぜなら、特別償却は「認定後に取得した設備」に限定されているからです。その設備は通常の減価償却により処理されます。
</Callout>

<Callout title="確認問題2">
    **問題**：自然災害により売上高が前年同月比25%減少した小売業が、資金繰り対策として融資を希望しています。利用できる信用保証制度と、その要件、保証率をそれぞれ説明してください。また、この企業が事業継続力強化計画の認定を持っていない場合、この保証制度は利用できるか答えてください。

    **解答例**：
    セーフティネット保証4号が利用できます。要件は、売上高が前年同月比20%以上減少していることです。この企業は25%の減少であるため要件を満たしています。保証率は100%保証（別枠）です。また、利用には市区町村長の認定が別途必要です。

    事業継続力強化計画の認定がなくても、セーフティネット保証4号は利用できます。この制度は、被災したすべての中小企業が対象であり、事前計画の認定状況に関わらず適用されます。
</Callout>

<Callout title="確認問題3">
    **問題**：以下の記述のうち、誤りを指摘し、正しい内容を述べてください。

    **記述**：「事業継続力強化計画の認定を受けると、国から直接、事業継続準備金として1,000万円の補助金が支給される。」

    **解答例**：
    記述は誤りです。事業継続力強化計画の認定そのものは、補助金の支給制度ではありません。認定により、以下の支援措置への入口が開かれます。
    - 防災設備の特別償却（20%）による税制優遇
    - 日本政策金融公庫の低利融資
    - ものづくり補助金等の各種補助金での加点
    - 信用保証協会の別枠保証
    - 損害保険料の割引

    各種補助金の給付は、別途、補助金ごとの公募に応募して採択されることが必要です。認定計画があることは採択可能性を高める要因になりますが、補助金そのものの給付を保証するものではありません。
</Callout>
