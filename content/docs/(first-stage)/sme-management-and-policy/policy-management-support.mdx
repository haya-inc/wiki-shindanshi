## このページの役割

<Callout title="このページの役割">
    このページは、`計画認定と伴走支援` を整理する更新ページです。試験では、補助金名より先に、`計画を認定する制度` と `支援する主体` を分けられることが重要です。
</Callout>

## 学習のポイント

- まず `計画認定` と `相談支援` を分ける
- 次に `経営革新計画` と `経営力向上計画` を区別する
- そのうえで `認定経営革新等支援機関`、`商工会`、`商工会議所` の役割差を押さえる

## 試験で何が問われるか

- `経営革新計画` と `経営力向上計画` を目的の違いで説明できるか
- `認定経営革新等支援機関` を、補助金や融資そのものではなく `専門支援の主体` として位置付けられるか
- 商工会、商工会議所などの支援機関を `相談 / 計画 / 実行支援` の流れで読めるか

## 支援の切り方

### 経営革新計画

`経営革新計画` は、新事業活動や経営の革新に取り組む事業者の計画を認定する枠組みです。したがって、`新しい取組に挑戦する計画` と読むと整理しやすくなります。

### 経営力向上計画

`経営力向上計画` は、人材育成、コスト管理、設備投資などを通じて自社の経営力を上げる計画です。こちらは `既存事業を含む経営基盤の強化` として読むと、経営革新計画と混ざりにくくなります。

### 認定経営革新等支援機関

`認定経営革新等支援機関` は、税務、金融、財務などに専門性を持つ支援主体です。制度の主役ではなく、事業者が計画策定や実行の支援を受ける `伴走役` として位置付けます。

### 商工会・商工会議所などの地域支援

商工会、商工会議所、よろず支援拠点などは、事業者の相談窓口として機能します。試験では、`どの制度があるか` だけでなく、`誰が支援するか` を区別すると解きやすくなります。

## 比較するときの軸

| 論点 | 典型的な見分け方 |
| --- | --- |
| `経営革新計画` と `経営力向上計画` | 新しい事業活動への挑戦か、経営基盤の強化か |
| `計画認定` と `支援機関` | 計画そのものか、支える主体か |
| `認定支援機関` と `商工会 / 商工会議所` | 専門性の高い認定主体か、地域の相談窓口か |

## 2026-03-28 時点の最新確認メモ

- 中小企業庁の `経営力向上支援` では、`経営力向上計画` を、人材育成、コスト管理、設備投資などを通じて経営力を高める計画として案内しています。
- 中小企業庁の `経営革新支援` は、`経営革新計画` を新たな事業活動への挑戦を後押しする枠組みとして整理しています。
- `認定経営革新等支援機関` の制度は現在も継続しており、中小企業に対する専門性の高い支援体制の整備として位置付けられています。

## 典型的なつまずき

- `経営革新計画` と `経営力向上計画` を同じ認定制度として覚える
- `認定経営革新等支援機関` を補助金制度の一種だと誤解する
- 支援主体の違いを見ずに、制度名だけを暗記する

## 問題を解くときの観点

- 問われているのは `挑戦支援` か `基盤強化` か
- `計画認定` の話か `支援主体` の話か
- 支援内容は `相談`、`計画策定`、`実行支援` のどこか
