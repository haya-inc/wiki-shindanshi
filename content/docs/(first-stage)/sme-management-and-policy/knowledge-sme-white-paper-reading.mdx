## このページの役割

このページは、`白書をどう読めば試験に使えるか` を整理する知識ノードです。`中小企業白書` と `小規模企業白書` を、`環境変化`、`経営課題`、`企業の対応`、`政策的示唆` の 4 つに分けて読む型を固定します。

## 学習のポイント

- まず `何が起きているか` を読む
- 次に `その結果どんな経営課題が出ているか` を整理する
- そのうえで `企業の対応` と `政策の方向` を分ける

## 試験で何が問われるか

- 白書の重点テーマを、単年の数字ではなく `今年の主題` として説明できるか
- `環境変化` と `政策名` を混同しないか
- `価格転嫁`、`賃上げ`、`人手不足`、`生産性向上`、`DX`、`事業承継` を頻出テーマとして読めるか
- 白書の表や図から、`何が問題として強調されているか` を言えるか

## 白書を読む 4 つの箱

### 1. 環境変化

まず見るのは、外部環境の変化です。物価、金利、人手不足、需要の変化、地域経済の変化など、`企業を取り巻く状況` がここに入ります。

### 2. 経営課題

次に、その環境変化によって何が課題になっているかを見ます。典型的には、人材確保、価格転嫁、収益力、生産性向上、事業承継です。

### 3. 企業の対応

白書では、課題に対して企業がどんな行動を取っているかも問われます。設備投資、業務改善、デジタル化、販路開拓、賃上げ、承継準備などがここに入ります。

### 4. 政策的示唆

最後に、政策は何を後押ししようとしているかを見ます。ここで個別施策の名前が出てきても、まずは `何を支える政策か` を先に整理します。

## 頻出テーマ

| テーマ | 読むときの軸 |
| --- | --- |
| `人手不足` | 採用難、離職防止、賃上げとの関係 |
| `価格転嫁` | コスト上昇を利益圧迫で終わらせないか |
| `賃上げ` | 防衛的か、業績改善を伴うか |
| `生産性向上` | 一人当たり付加価値をどう高めるか |
| `DX` | 単なる IT 導入ではなく、経営改善につながっているか |
| `事業承継` | 廃業回避、雇用維持、地域経済の継続にどうつながるか |

## 白書の表を読む順番

1. `何の表か`
2. `規模別`、`業種別`、`地域別` のどれか
3. `水準` を見せたいのか、`増減` を見せたいのか
4. その表は `何を課題として示しているか`

## 典型例

| 論点 | 典型的な見分け方 |
| --- | --- |
| `環境変化` と `経営課題` | 外から起きた変化か、企業が直面する問題か |
| `企業の対応` と `政策` | 企業自身の行動か、公的な後押しか |
| `価格転嫁` と `賃上げ` | 収益確保の話か、人材確保の話か |

## 典型的なつまずき

- 白書の数字だけを覚えて主題を読まない
- 環境変化と政策名を同じ箱に入れてしまう
- `価格転嫁` や `DX` をその年だけの流行語として扱う
- 図表から `何が問題か` を言えない

## 問題を解くときの観点

- 今年の白書が強調しているのは `環境`、`課題`、`対応`、`政策` のどこか
- そのテーマは `人材`、`収益`、`成長投資`、`事業継続` のどこに属するか
- 図表の数字ではなく、`そこから言いたいこと` を説明できるか
