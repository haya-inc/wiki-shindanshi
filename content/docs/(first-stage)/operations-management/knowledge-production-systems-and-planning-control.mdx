<Callout title="このページの役割">
    このページは、`どの生産方式で作るか` と `どう計画し、どう統制するか` をまとめて整理する解説ページです。
    `生産の前提` と `現場の管理` を切り分ける戻り先です。

## 学習のポイント

- `受注生産` と `見込生産` を分ける
- `個別`、`ロット`、`連続` の違いを流れで押さえる
- `大日程`、`中日程`、`小日程` と `生産統制` を混ぜない

## 試験で何が問われるか

- 生産方式と生産類型の違いを説明できるか
- 大日程、中日程、小日程の役割を言えるか
- 生産統制の `進捗管理`、`余力管理`、`現品管理` を区別できるか

## 定義と仕組み

### 生産管理の基本

生産管理は、`QCD` を満たしながら現場を回す考え方です。計画だけでも、改善だけでも足りず、`どう作るか` と `どう維持するか` を両方見ます。

### 生産方式と生産類型

`受注生産` は注文を受けてから作り、`見込生産` は需要を見込んで先に作ります。`個別生産`、`ロット生産`、`連続生産` は、品種、量、流れの安定度で分かれます。

### 生産計画と生産統制

生産計画は `何をいつどれだけ作るか` を定める活動で、`大日程`、`中日程`、`小日程` に分かれます。生産統制は、計画どおりに進んでいるかを現場で管理する活動です。

## 典型例

| 論点 | 見ること | 典型的な判断 |
| --- | --- | --- |
| 受注 / 見込 | 作るタイミング | 注文後に作るか、先に作るか |
| 個別 / ロット / 連続 | 生産の流れ | 多品種少量か、少品種大量か |
| 大 / 中 / 小日程 | 計画の粒度 | 長期計画か、現場直前か |
| 生産統制 | 計画との差 | 遅れ、余力、現品の詰まり |

## 典型的なつまずき

- 生産方式と生産類型を同じ軸で覚えてしまう
- `大日程` と `進捗管理` を同じものとして扱う
- 受注生産なのに見込生産向けの在庫発想で考えてしまう

## 問題を解くときの観点

- いま問われているのは `作り方` か `管理の仕方` か
- 前提は `多品種少量` か `少品種大量` か
- 計画の話か、計画との差を埋める統制の話か
