## このページの役割

<Callout title="このページの役割">
    このページは、`外からどう安定調達するか` を整理する解説ページです。購買管理と外注管理を、
    コストだけでなく品質、納期、関係管理の観点で見ます。
</Callout>

<Callout title="このページの読み方">
  「自社で作るべきか、外注すべきか」──この判断を内製・外注の意思決定と呼びます。単なるコスト比較ではなく、コア技術の保持、品質管理の難易度、サプライチェーンのリスクまで考慮する必要があります。
</Callout>

## 学習のポイント

- `購買管理` は資材調達の管理
- `外注管理` は工程・作業の委託管理
- `内製 or 外注` の判断基準を押さえる
- `集中購買` と `分散購買` の使い分け

## 試験で何が問われるか

- 購買管理と外注管理の違いを説明できるか
- 内製・外注の判断基準を言えるか
- 集中購買と分散購買のメリット・デメリットを区別できるか
- 外注管理の品質・納期管理の方法を説明できるか

## 購買管理 ── 資材の調達

購買管理は、必要な資材を適切な品質・価格・納期で調達する活動です。

### 購買管理の 5 原則

| 原則 | 内容 |
| --- | --- |
| 適正な品質 | 要求仕様を満たす品質 |
| 適正な数量 | 不足も過剰もない量 |
| 適正な時期 | 生産計画に合ったタイミング |
| 適正な価格 | 品質・納期を踏まえた妥当な価格 |
| 適正な供給者 | 信頼できる取引先 |

### 集中購買 vs 分散購買

| 比較軸 | 集中購買 | 分散購買 |
| --- | --- | --- |
| 発注の主体 | 本部が一括 | 各工場・拠点が個別 |
| メリット | 大量発注で価格交渉力向上 | 現場ニーズに即応 |
| デメリット | 現場の個別対応が遅い | 重複発注、管理コスト増 |
| 向く場面 | 共通資材、標準品 | 特殊資材、緊急調達 |

### 調達先選定の評価基準

| 基準 | 見るポイント |
| --- | --- |
| 品質 | 不良率、品質管理体制、認証 |
| コスト | 単価、輸送費、トータルコスト |
| 納期 | リードタイム、遵守率 |
| 柔軟性 | 急な増減への対応力 |
| 経営安定性 | 財務状況、事業継続リスク |

---

## 外注管理 ── 工程・作業の委託

外注管理は、自社の工程や作業の一部を外部に委託し、品質・納期・コストを管理する活動です。

### 外注の種類

| 種類 | 内容 | 例 |
| --- | --- | --- |
| 能力外注 | 自社の能力不足を補う | 繁忙期の増産対応 |
| 専門外注 | 自社にない技術を活用 | 特殊加工、めっき処理 |

---

## 内製 vs 外注の判断

### 判断基準

| 判断軸 | 内製が有利 | 外注が有利 |
| --- | --- | --- |
| コスト | 固定費の活用（余力あり） | 変動費化（需要変動対応） |
| コア技術 | 自社のコア技術 → 内製維持 | 非コア工程 → 外注可 |
| 品質管理 | 厳密な管理が必要 | 標準的な品質で足りる |
| 需要変動 | 安定需要 | 変動が大きい |
| 設備投資 | 既に設備がある | 新規投資が必要 |

### 内製・外注の損益分岐点

内製と外注のどちらが安いかは、生産量によって変わります。

```
【考え方】
  内製コスト = 固定費 + 変動費 × 数量
  外注コスト = 外注単価 × 数量

  損益分岐点 = 固定費 ÷ (外注単価 − 変動費)

【計算例】
  内製: 固定費 500,000円、変動費 300円/個
  外注: 800円/個

  損益分岐点 = 500,000 ÷ (800−300) = 500,000 ÷ 500 = 1,000個

→ 1,000個未満なら外注が安い、1,000個以上なら内製が安い
→ ただしコスト以外の判断軸（コア技術、品質、リードタイム）も考慮
```

## 外注管理のポイント

### 外注先の管理項目

| 管理項目 | 内容 |
| --- | --- |
| 品質管理 | 受入検査、工程監査、品質データの共有 |
| 納期管理 | 進捗確認、リードタイム管理 |
| コスト管理 | 見積もり精度、コスト低減活動 |
| 技術管理 | 図面・仕様書の明確化、技術指導 |

### 下請法の留意点

親事業者が下請事業者に対して守るべきルールとして、発注書面の交付義務、下請代金の支払遅延禁止、買いたたきの禁止などがあります（詳細は [経営法務](/docs/business-law) を参照）。

## 典型的なつまずき

- `購買` と `外注` を同じ調達活動として一括で覚える → 購買は **モノ**（資材）、外注は **コト**（工程・作業）
- 最安値の調達先が常に最適だと考える → **品質、納期、安定供給** を含むトータル評価が必要
- 外注先管理を価格交渉だけだと思ってしまう → 品質、納期、技術の **継続的な管理** が重要
- 内製・外注の判断をコストだけで行う → **コア技術の保持** は長期的に重要
- 損益分岐点を超えれば必ず内製が良いと判断する → 設備投資の回収期間や需要の不確実性も考慮

## 問題を解くときの観点

- いま問われているのは資材（購買）か、工程（外注）か
- 重いのは `価格` か `品質` か `納期` か
- 内製維持と委託活用のどちらが自然か
- コスト比較だけでなく戦略的判断を求められていないか

## 確認問題

<Callout title="問1：内製・外注の損益分岐点">
    内製の固定費 600,000円、変動費 200円/個。外注単価 500円/個。損益分岐点を求め、月産 1,500個の場合どちらが有利か答えよ。

    **解答**：損益分岐点 = 600,000÷(500−200) = 2,000個。1,500個 < 2,000個なので外注が有利（外注750,000円 vs 内製900,000円）。
</Callout>

<Callout title="問2：集中購買と分散購買">
    「複数の工場で共通に使用する標準ボルトの調達コストを下げたい」場合、集中購買と分散購買のどちらが適切か。

    **解答**：集中購買。共通資材・標準品は本部で一括発注することで、大量発注による価格交渉力を発揮できる。
</Callout>

<Callout title="問3：外注の種類">
    「自社では対応できない特殊な表面処理を外部の専門業者に委託する」のは、能力外注・専門外注のどちらか。

    **解答**：専門外注。自社にない技術を活用するための外注。能力外注は自社の処理能力不足を補うもの（繁忙期の増産対応など）。
</Callout>
