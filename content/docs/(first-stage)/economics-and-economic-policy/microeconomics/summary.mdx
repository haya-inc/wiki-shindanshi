## このページの役割

このページは、`ミクロ経済学の直前確認` に使う要約ページです。細かな説明を増やすのではなく、比較軸、基本公式、見落としやすい点を短く確認できるようにします。

## 学習のポイント

- 試験直前は `図の意味` と `比較軸` の確認を優先する
- 公式は単独で覚えず、`どの場面で使うか` とセットで見る
- 解けない論点があれば、要約で止まらず元の解説ページへ戻る

## 最低限の確認リスト

- 価格変化は `曲線上の移動`、外部条件変化は `曲線のシフト`
- 需要が弾力的なら値下げで売上高が増えやすい
- 最適消費点は `無差別曲線` と `予算制約線` の接点
- `MC` は `AVC` と `AC` の最低点を通る
- 完全競争では `価格 = 限界収入 = 限界費用`
- 公共財は `非競合性` と `非排除性`
- ゲーム理論では `支配戦略` と `ナッシュ均衡` を区別する
- 比較優位は `絶対優位` と違う

## 比較軸の要約

| 論点 | 比較軸 |
| --- | --- |
| 需要曲線の移動とシフト | 価格変化か、価格以外の変化か |
| 弾力性 | 数量の反応が大きいか小さいか |
| 消費者行動 | 接点か、補助線の意味か |
| 企業行動 | `MC`、`AVC`、`AC` の関係 |
| 市場構造 | 企業数、参入障壁、差別化、価格支配力 |
| 市場の失敗 | どこに効率性の崩れがあるか |
| ゲーム理論 | 支配戦略があるか、ナッシュ均衡か |
| 国際貿易理論 | 絶対優位ではなく機会費用で見る |

## 典型的なつまずき

- 公式だけを暗記し、図の読み方を後回しにする
- 市場構造の表を覚えても、政策問題とつながらない
- 総復習の段階で、知識不足と設問慣れ不足を混同する
