## このページの役割

このページは、`マクロ経済学` という章全体のハブです。ここで章の全体像と学習順を確認し、詳細は `解説ページ`、`解き方ページ`、`演習ページ`、`要約` に進む構成にします。

## 学習のポイント

- 最初に `GDP`、`GNI`、`物価指数`、`失業率` を固める
- 次に `消費理論`、`投資理論`、`乗数` を経由して `IS-LM` へ進む
- そのうえで `AD-AS`、`フィリップス曲線`、`国際マクロ` を位置付ける
- 最後に `主要経済理論の系譜` を比較で整理する
- 最後に `どう問われるか` を出題と演習で確認する

## この章で解けるようにしたいこと

1. `名目` と `実質`、`GDP` と `GNI`、`CPI` と `GDP デフレーター` を区別できる
2. 財政政策と金融政策が `IS`、`LM`、`AD`、`AS` にどう効くか説明できる
3. 短期と長期、閉鎖経済と開放経済を切り分けて読める
4. 図表問題と政策比較問題で、何を見ればよいか判断できる

## 対応する過去問テーマ

- GDP、GNI、三面等価
- CPI と GDP デフレーター
- 消費関数、ライフサイクル仮説、恒常所得仮説
- 加速度原理、トービンの q
- 限界消費性向と乗数
- IS 曲線、LM 曲線のシフト要因
- 財政政策と金融政策の比較
- AD-AS とフィリップス曲線
- 古典派、ケインズ派、マネタリズム、合理的期待
- 為替、国際収支、マンデル=フレミング・モデル

問題文そのものではなく、まずは `どのテーマがどのページで解けるか` を固定します。

## 典型的なつまずき

- `名目` と `実質` を区別しないまま計算する
- `CPI` と `GDP デフレーター` を同じものとして扱う
- `現在所得` と `将来所得の見通し` を混ぜる
- `IS` と `LM` の意味を取り違える
- 経済理論の名前を暗記だけで処理し、何を重視する理論かを言えない
- 財政政策と金融政策の効果を、曲線のシフトなしに丸暗記する
- 閉鎖経済の感覚のまま国際マクロを読む
