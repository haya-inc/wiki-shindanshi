## このページの役割

このページは、`統計学の基礎` を整理する解説ページです。式を細かく暗記するより、`何を測る指標か` と `結果をどう読むか` を固定します。

## 学習のポイント

- まず `代表値` と `ばらつき` を分ける
- 次に `相関` と `回帰` の違いを切る
- そのうえで `有意差` と `因果関係` を混同しない

## 試験で何が問われるか

- `平均`、`中央値`、`最頻値` の違いを説明できるか
- `分散`、`標準偏差` が何を表すか言えるか
- `相関係数` と `回帰分析` の違いを説明できるか
- `検定` の結果を、`有意`、`有意でない` の意味で読めるか

## 定義と仕組み

### 代表値

`平均` は全体の中心を 1 つの数字で表す考え方です。`中央値` は並べたときの真ん中、`最頻値` は最も多く現れる値です。外れ値に引っ張られやすいかどうかで、向いている使い方が変わります。

### ばらつき

`分散` と `標準偏差` は、データが平均からどの程度散らばっているかを見る指標です。平均が同じでも、標準偏差が大きい方が値の振れ幅が大きいと読みます。

### 相関と回帰

`相関` は、2 つの変数がどの程度一緒に動くかを見るものです。`回帰` は、ある変数を使って別の変数を説明、予測する考え方です。したがって、`関係がある` と `予測式を置く` は同じではありません。

### 検定

`検定` は、観測された差や関係が偶然かどうかを確率的に判断する考え方です。`有意` は `差があるとみなす根拠がある` に近く、`重要` や `因果関係が確定した` とは違います。

## 典型例

- 売上の中心的な水準を見たいときに `平均`
- 成績や所要時間の散らばりを見たいときに `標準偏差`
- 広告費と売上の動きを見るときに `相関`
- 広告費から売上を予測したいときに `回帰`

## 典型的なつまずき

- `平均` だけ見て、外れ値の影響を無視する
- `相関が高い` を `因果関係がある` と読み替える
- `回帰` と `相関` を同じ分析だと考える
- `有意差がある` を `実務的に重要` と短絡する

## 問題を解くときの観点

- 求めたいのは `中心` か `ばらつき` か
- 見たいのは `一緒に動くか` か `説明 / 予測したいか` か
- 結果を `因果` に飛ばしていないか
- `有意` の意味を `偶然ではなさそう` の範囲で止めているか
