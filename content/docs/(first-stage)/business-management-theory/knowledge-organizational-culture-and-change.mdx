## このページの役割

<Callout title="このページの役割">
    このページは、`組織の空気と変化の進め方` を整理する解説ページです。`シャイン`、`レビン`、`コッター`、
    `学習する組織`、`SECIモデル` を、`文化の層`、`変革の段階`、`知識の回り方` で分けて理解します。
</Callout>

## 学習のポイント

- まず `文化の深さ` を理解する
- 次に `変革の進め方` を押さえる
- 最後に `学習` と `知識創造` をつなぐ

## 試験で何が問われるか

- `シャインの3層モデル` の違いを説明できるか
- `レビン` と `コッター` の変革プロセスを区別できるか
- `シングルループ` と `ダブルループ` の違いを判断できるか
- `SECIモデル` の 4 プロセスを押さえているか

## 定義と仕組み

### 組織文化

| 層 | 内容 | ポイント |
| --- | --- | --- |
| 人工物 | 制度、言葉、シンボル、見える行動 | 観察しやすいが意味は深掘りが必要 |
| 標榜された価値観 | 理念、戦略、方針 | 言っていることと実態がずれる場合がある |
| 基本的仮定 | 無意識の前提 | 最も深く、変えにくい |

`シャイン` の整理では、変革が難しいのは表面の制度より `基本的仮定` です。

### 組織変革

| 理論 | 見方 | 核心 |
| --- | --- | --- |
| レビン | 3段階 | `解凍 → 変革 → 再凍結` |
| コッター | 8段階 | 危機意識、推進チーム、ビジョン、短期成果、定着までを細かく分ける |

変革は `新制度を入れること` だけではなく、`必要性を受け入れさせること` と `定着させること` まで含みます。

### 学習と知識創造

- `センゲ`: `システム思考`、`自己マスタリー`、`メンタルモデル`、`共有ビジョン`、`チーム学習`
- `アージリス`: `シングルループ学習` は前提を変えずに修正し、`ダブルループ学習` は前提そのものを見直す
- `野中のSECIモデル`: `共同化`、`表出化`、`連結化`、`内面化` の 4 段階で知識を回す

## 典型例

| 課題 | 見るべき論点 |
| --- | --- |
| 理念はあるが現場行動が変わらない | シャインの層のずれ |
| 変革の必要性は言うが現場が動かない | レビンの `解凍` 不足 |
| 一度改善しても元に戻る | `再凍結` 不足 |
| 個人の知見が組織に残らない | SECI モデルの `表出化` と `連結化` 不足 |

## 典型的なつまずき

- `組織文化` を単なる社風の話で終わらせる
- `変革` を新制度導入だけだと考える
- `SECIモデル` の順番を混同する
- `ダブルループ学習` を単なる改善活動と同じにしてしまう

## 問題を解くときの観点

- 問題は `見える制度` か `深い価値観` か
- 変革の段階は `解凍`、`変革`、`定着` のどこで止まっているか
- 個人知を組織知へ変える仕組みがあるか
- 課題は `構造` ではなく `文化` や `学習` の不足ではないか
