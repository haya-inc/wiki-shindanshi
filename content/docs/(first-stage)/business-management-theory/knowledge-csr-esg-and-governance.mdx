## CSR（企業の社会的責任）

### CSRの定義

CSR（Corporate Social Responsibility）は、企業が営利追求だけでなく、従業員、顧客、取引先、地域社会、環境など幅広いステークホルダーに対して果たすべき責任のことです。法令順守（コンプライアンス）にとどまらず、倫理的・社会的な行動を求める広い概念です。

### キャロルのCSRピラミッド

アーチー・キャロルが提唱した、企業責任の層構造です。下層から順に積み上げられ、上層ほど追加的な責任とされます。

| 層 | 責任の種類 | 内容 | 試験での位置づけ |
| --- | --- | --- | --- |
| **1 層目（基礎）** | 経済的責任 | 利益を生み出し、雇用を創出し、持続的に事業を営む。株主に配当を還元する | CSR以前の基本的な役割 |
| **2 層目** | 法的責任 | 法令を遵守し、規制を守る。強制的に求められる最低限の責任 | コンプライアンス |
| **3 層目** | 倫理的責任 | 社会的規範を尊重し、法律には書かれていない道徳的・倫理的行動をとる | 自発的な配慮 |
| **4 層目（頂点）** | 社会貢献的責任（フィランソロピー） | 教育支援、寄附、ボランティア、地域活動。企業に直接的な見返りを求めない活動 | 最上位の自発的責任 |

**キャロルのピラミッドの読み方**：
- 下層から上層へ積み上げるため、まず経済的責任と法的責任がなければ話にならない
- 倫理的責任と社会貢献的責任は、経営層や事業環境、社会的期待によって実現度が変わる
- 「すべての企業が頂点まで達成する」わけではなく、財務状況や業界特性によって優先順位が決まる

### ステークホルダー理論

#### フリーマンのステークホルダー理論

R. エドワード・フリーマンが提唱した理論で、企業は **株主だけでなく、複数の利害関係者（ステークホルダー）に対して責任を負う** という考え方です。

**ステークホルダーとは**：企業の経営決定に影響を与える、またはその決定から影響を受ける、すべての個人・グループです。

### 株主中心主義 vs ステークホルダー主義

| 項目 | 株主中心主義 | ステークホルダー主義 |
| --- | --- | --- |
| **基本原則** | 企業の最終的な目標は株主価値最大化 | 企業は複数の利害関係者のために存在 |
| **意思決定の優先順位** | 株主利益が最優先 | すべてのステークホルダーの利益を考慮 |
| **経営層の責務** | 株主利益の最大化 | 各ステークホルダーとの関係管理 |
| **例：配当と従業員給与** | 利益最大化のため給与を抑制、配当を優先 | 従業員の生活水準や雇用安定と配当のバランス |
| **短期 vs 長期** | 短期利益を重視しやすい | 長期的な信頼・持続可能性を重視 |
| **日本での位置づけ** | 1990年代以降の経営改革で採用が進む | 従来の日本企業経営に近い（長期雇用、地域貢献） |

#### 主要ステークホルダーとその関心事

| ステークホルダー | 関心事 | CSRとの関連 |
| --- | --- | --- |
| **株主** | 株価上昇、配当 | 長期的な企業価値と信頼性 |
| **従業員** | 給与、雇用安定、キャリア、職場環境 | 人権尊重、安全衛生、研修機会 |
| **顧客** | 製品品質、価格、安全性、情報開示 | 製品品質保証、消費者保護 |
| **取引先・サプライヤー** | 公正な取引、適正な支払い | フェアトレード、下請負保護 |
| **地域社会** | 雇用、環境保全、インフラ整備 | 環境負荷低減、地域貢献活動 |
| **政府・行政機関** | 法令遵守、税納付、社会貢献 | コンプライアンス、透明性 |
| **環境・将来世代** | 気候変動対策、資源保全 | サステナビリティ、脱炭素 |

## CSV（共通価値の創造）

### CSR vs CSV

マイケル・ポーター＆マーク・クラマーが提唱した概念で、**社会的課題と企業の競争優位が同時に実現される価値創造** を目指します。

| 観点 | CSR（企業の社会的責任） | CSV（共通価値の創造） |
| --- | --- | --- |
| **起点** | 社会課題に企業が責任を果たす | 社会課題の解決が企業利益につながる |
| **目的** | 社会貢献（利他的） | 社会価値と事業価値の同時実現（相互作用） |
| **関係** | 費用・負担（慈善活動に近い） | 投資・機会（競争優位の源泉） |
| **例：環境対策** | CO2削減のための設備投資（コスト） | 環境技術の開発→新市場開拓→利益増 |
| **スケール** | 個別の社会貢献活動が中心 | 事業モデル全体を見直す |
| **試験での使われ方** | CSRの実践例として問われる | 「社会課題と事業の関係」として戦略論と結びつく |

### CSVの3つのアプローチ

1. **製品・市場の再定義**：社会課題に対応する新製品・サービス開発（例：低所得層向け医療サービス、環境配慮型製品）

2. **バリューチェーンの再定義**：調達・製造・流通の過程で社会的価値を組み込む（例：フェアトレード、廃棄物削減）

3. **地域クラスター支援**：企業の事業環境が依存する地域の競争力を高める（例：サプライチェーン全体の技術向上、職業訓練）

## ESG（環境・社会・ガバナンス）

### ESGの定義

ESG（Environmental, Social, Governance）は、企業の **非財務情報** を3つの観点から評価する枠組みです。特に投資家やアナリストが企業の **長期的持続可能性と競争力** を測定する際に用いられます。

### E：環境（Environmental）

企業が環境へ与える影響と、環境リスクへの対応を見ます。

| 観点 | 評価項目 | 試験での出題パターン |
| --- | --- | --- |
| **気候変動対策** | GHG（温室効果ガス）排出削減、脱炭素への取り組み、シナリオ分析 | RE100（再生可能エネルギー100%化）、カーボンニュートラル宣言 |
| **資源効率** | 水資源利用の削減、廃棄物削減・リサイクル | サーキュラーエコノミー、拡大生産者責任 |
| **汚染・化学物質管理** | 大気汚染・水汚濁防止、有害物質管理 | RoHS指令対応、REACH規制 |
| **生物多様性** | 生態系への配慮、農業・漁業への影響 | 生物多様性インパクト評価 |
| **エネルギー管理** | 省エネ、エネルギー効率改善 | ISO 50001認証 |

### S：社会（Social）

従業員、顧客、サプライチェーン、コミュニティとの関係を見ます。

| 観点 | 評価項目 | 試験での出題パターン |
| --- | --- | --- |
| **労働慣行** | 賃金の公正性、労働時間、安全衛生、児童労働・強制労働の排除 | ディーセント・ワーク（ILO）、ハラスメント対策 |
| **人権** | 差別禁止、多様性・包括性、先住民権の尊重 | ジェンダー平等、LGBTQ+対応 |
| **顧客対応** | 製品安全、情報セキュリティ、データプライバシー | 個人情報保護、AI倫理 |
| **コミュニティ関係** | 地域社会への貢献、事業展開地域への影響 | 地域雇用、教育支援、スキルアップ研修 |
| **サプライチェーン管理** | 取引先への強要的慣行排除、人権デューディリジェンス | 下請負企業の労働環境改善 |

### G：ガバナンス（Governance）

企業経営の意思決定プロセス、透明性、監督機能を見ます。

| 観点 | 評価項目 | 試験での出題パターン |
| --- | --- | --- |
| **取締役会構成** | 社外取締役比率、多様性、独立性、専門性 | 監査等委員会設置会社の仕組み |
| **経営陣の報酬** | 報酬の透明性、短期業績との連動、長期インセンティブ | ストックオプション、業績連動給与 |
| **利益相反管理** | 関連当事者取引の透明性、内部取引の監視 | 監査役会の独立性確保 |
| **情報開示** | 経営方針の透明性、リスク情報の開示、IR活動 | サステナビリティレポート、統合報告書 |
| **コンプライアンス・内部統制** | 規制遵守、内部監査、不正防止 | COSO フレームワーク、内部通報制度 |
| **経営支配構造** | 株主と経営者の関係、監督機構 | エージェンシー問題の対応 |

### ESG投資の類型

| 投資アプローチ | 説明 | 特徴 |
| --- | --- | --- |
| **ネガティブ・スクリーニング** | ESG課題が深刻な企業を投資対象から除外 | リスク回避型 |
| **ポジティブ・スクリーニング** | 同業他社と比較してESG対応が優秀な企業に投資 | 相対比較 |
| **ESG統合** | 財務分析とESG評価を組み合わせ、長期リターンを追求 | 標準的アプローチ |
| **サステナビリティ・テーマ投資** | 再生可能エネルギー、水処理など特定テーマに投資 | テーマ型 |
| **インパクト投資** | 社会的・環境的成果を目指し、ファイナンシャルリターンも求める | 成果測定重視 |
| **エンゲージメント** | 株主として投資先企業のESG改善を働きかける | 対話型・アクティビスト |

### ESGとSDGs（持続可能開発目標）との関係

**SDGs**（Sustainable Development Goals）は、国連が2030年までに達成すべき17のグローバル目標です。ESGは企業がSDGs達成に貢献する枠組みとも見なされます。

| SDGs 例 | ESGの対応要素 |
| --- | --- |
| SDG 5：ジェンダー平等を実現しよう | S（社会）：ジェンダー多様性、同一賃金 |
| SDG 7：エネルギーをみんなに そしてクリーンに | E（環境）：再生可能エネルギー利用 |
| SDG 13：気候変動に具体的な対策を | E（環境）：GHG削減、脱炭素 |
| SDG 16：平和と公正をすべての人に | G（ガバナンス）：コンプライアンス、人権尊重 |

## コーポレートガバナンス

### ガバナンスの定義

コーポレートガバナンスは、企業経営を適切に監督し、規律付ける仕組みです。経営者が企業資産を適切に管理・運用し、長期的な企業価値向上を実現させるための制度・機構・プロセスの総称です。

### 所有と経営の分離

近代企業では、株主（所有者）と経営者が分離しています。これにより、経営の専門性は高まりますが、新たな問題が生じます。

| 影響 | 内容 |
| --- | --- |
| **メリット** | 経営の専門性、分業と効率性 |
| **問題** | 経営者が株主利益を無視する可能性、透明性欠如 |

### エージェンシー理論

**エージェンシー理論** は、経営者（エージェント）が株主（プリンシパル）の利益を代理して実現する関係を分析します。

#### 主要概念

| 概念 | 説明 |
| --- | --- |
| **プリンシパル** | 企業の所有者である株主 |
| **エージェント** | 株主に代わって企業を経営する経営層 |
| **情報の非対称性** | 経営者は企業内部情報を多く持つが、株主は持たない。その差を利用した行動リスク |
| **エージェンシーコスト** | 情報非対称性から生じる追加的な監視・統制コスト |
| **モラルハザード** | 経営者が個人利益を優先し、株主利益を損なう自己中心的行動 |

#### モラルハザード（逆選択）の具体例

| 行動 | リスク |
| --- | --- |
| 過度な経営者報酬・ストックオプション | 長期的な企業価値より短期的な株価上昇を求める |
| 帝国建設（empire building） | 企業価値を損なう無理な買収・多角化 |
| 利益操作・不正会計 | 粉飾決算で株価を意図的に操作 |
| 過度なリスクテイク | ハイリスク・ハイリターン事業に集中 |
| 情報開示の不透明性 | 都合の悪い情報を隠蔽 |

#### エージェンシー問題への対策

ガバナンス機構・制度は、エージェンシーコストを低減させるために設計されます。

| 対策 | 仕組み |
| --- | --- |
| **監視・監督** | 取締役会、監査役会、監査委員会 |
| **インセンティブ調整** | 業績連動報酬、ストックオプション（ただし短期志向のリスクあり） |
| **透明性・開示** | 定期的な財務報告、監査報告書、経営方針の明示 |
| **規制・コンプライアンス** | コーポレートガバナンス・コード、金融商品取引法 |
| **社外性の確保** | 社外取締役、社外監査役の配置 |

## 日本のコーポレートガバナンス機構

日本では、監査役制度の伝統があり、会社法で3つの異なるガバナンス機構が認められています。

### 3つのガバナンス機構の比較

| 項目 | 監査役会設置会社 | 指名委員会等設置会社 | 監査等委員会設置会社 |
| --- | --- | --- | --- |
| **導入時期** | 昔からの伝統的制度 | 2002年に会社法で導入（米国型） | 2015年の会社法改正で導入（日本型） |
| **採用企業** | 大多数（約99%） | 少数派（外資系企業など） | 導入進行中 |
| **監査体制** | 監査役（社員から登用可）と監査役会 | 監査委員会（社外取締役で構成） | 監査等委員会（取締役で構成、社外取締役が過半数） |
| **意思決定機構** | 取締役会（経営） | 指名・報酬・監査委員会（監督） | 取締役会（経営と監督を統合） |
| **社外取締役の位置づけ** | 任意（多くの企業が増員傾向） | 必須（委員会に参画） | 必須（監査等委員には過半数が必須） |
| **経営陣との独立性** | 比較的高い（監査役は別組織） | 高い（委員会が経営層から独立） | 中程度（同じ取締役会内） |
| **経営者選任プロセス** | 取締役会の提案、株主総会で承認 | 指名委員会（社外取締役中心）が提案 | 取締役会で決定、株主総会で承認 |
| **長期経営との関係** | 監査役の業務経験を生かし長期視点 | 委員会型で短期的になりやすいリスク | 双方の要素を持つ |
| **日本的経営との親和性** | 高い（既存組織の延長） | 低い（米国式） | 中程度（日本的改良型） |

### 各機構の詳細

#### 監査役会設置会社（最多数派）

**特徴**：
- 取締役会が経営を担当し、監査役が経営を監視する
- 監査役は監査役会（複数の監査役で構成）を形成
- 監査役には常勤・非常勤、社内出身・社外登用がある

**メリット**：
- 日本企業の伝統的制度で組織文化との親和性が高い
- 監査役が長年の経営経験を持つ場合、実質的な助言が可能
- 移行の手続き負荷が低い

**デメリット**：
- 監査役が社内出身だと独立性が低くなる可能性
- 経営と監視の分離が曖昧になるリスク

#### 指名委員会等設置会社（外資系企業など）

**特徴**：
- 3つの委員会（指名・報酬・監査）を設置し、各委員会が経営を監視
- 委員会のメンバーは社外取締役が中心
- 米国企業統治制度（米国のSarbanes-Oxley法）に近い

**メリット**：
- 経営層から完全に独立した監視
- グローバル投資家への信頼性が高い
- 経営の透明性と説明責任が強化される

**デメリット**：
- 日本企業文化（長期雇用、経営の連続性）と不親和性
- 短期的な業績圧力が強まるリスク
- 経営陣の多様性導入が急速で経営の安定性が損なわれる可能性

#### 監査等委員会設置会社（最近の推奨型）

**特徴**：
- 取締役の中から監査等委員を選任（社外取締役が過半数）
- 取締役会内に監査等委員会を組織
- 日本の監査役制度とアングロサクソン的経営の折衷型

**メリット**：
- 日本企業の文化的背景を保ちながら、社外性を強化
- 経営と監視の分離が明確になり、透明性が向上
- 2015年改正以降、公開企業の「好ましい形」として推奨

**デメリット**：
- 移行に伴う制度設計が複雑
- 社外取締役の確保・育成が必要

### 社外取締役の役割

| 役割 | 説明 |
| --- | --- |
| **監視・チェック機能** | 経営層の不正・不適切な行為を早期発見・是正 |
| **多角的視点** | 企業外部の視点から経営判断を指摘、多様性確保 |
| **利益相反監視** | 特定株主や経営陣による私利私欲を防止 |
| **説明責任** | ステークホルダーへの説明責任を強化 |
| **アドバイス機能** | 経営経験や専門知識を活かし、経営課題の解決に貢献 |

## コーポレートガバナンス・コード

### 定義と基本原則

**コーポレートガバナンス・コード** は、東京証券取引所が上場企業に対して示したガイドラインで、企業統治のベストプラクティスを示しています。

**原則主義（プリンシプルベース）**：「何をすべきか」を具体的に指定するのではなく、「どのような原則に基づいて判断すべきか」を示し、企業の状況に応じた柔軟な対応を許容します。

### Comply or Explain（遵守せよ、さもなくば説明せよ）

企業は以下の3つのいずれかの対応を取ります。

| 対応 | 説明 |
| --- | --- |
| **Comply（遵守）** | コードに定められた慣行に従う |
| **Explain（説明）** | コードに従わない場合、なぜ従わないか、その理由と代替策を説明する |
| **Change（変更検討）** | 説明では不十分と判断したら、従来の慣行を変更する |

この原則により、企業規模や業界の特性を考慮した柔軟な対応が可能になります。

### コーポレートガバナンス・コードの主要原則（簡略版）

| 原則 | 説明 |
| --- | --- |
| **原則1：株主の権利と平等性** | 株主権の保護、株主平等の確保 |
| **原則2：株主以外のステークホルダー** | 従業員、顧客、サプライヤー、地域社会などの利益を適切に考慮 |
| **原則3：情報開示と透明性** | 経営方針、リスク、経営成績の透明な開示 |
| **原則4：取締役会・監査役等の機能** | 経営監視、適切なガバナンス機構の設置 |
| **原則5：株主との対話** | 建設的かつ継続的な対話 |

## 内部統制（Internal Control）

### 内部統制の定義

内部統制は、企業が **経営目標を達成し、リスクを適切に管理し、信頼性のある財務報告を実現する** ために、経営層が構築・運営する組織的な仕組みです。

**日本での枠組み**：金融商品取引法（J-SOX）で、上場企業の内部統制の評価・開示が義務づけられています。

### COSO フレームワーク

**COSO**（Committee of Sponsoring Organizations of the Treadway Commission）が示した、内部統制の国際的スタンダードです。

#### 内部統制の4つの目的

| 目的 | 説明 | 具体例 |
| --- | --- | --- |
| **有効性と効率性** | 経営目標を達成し、資源を効率的に活用する | 不良品低減、コスト削減 |
| **財務報告の信頼性** | 正確で適時な財務情報を提供する | 不正会計防止、監査対応 |
| **法令遵守** | 適用法規を遵守する | コンプライアンス研修、規制監視 |
| **資産の安全保護** | 企業資産の盗難・不正使用を防止する | アクセス制限、監査ログ、保管管理 |

#### 内部統制の6つの基本的要素

| 要素 | 説明 | 実装例 |
| --- | --- | --- |
| **1. 統制環境** | 誠実性、倫理観、コンプライアンス意識の組織風土 | 行動憲章、企業倫理教育、トップの姿勢 |
| **2. リスク評価** | 経営目標達成を阻害するリスクの識別・評価 | リスクマップ作成、監視体制 |
| **3. 統制活動** | リスク低減のための具体的な施策・手続き | 承認ルール、チェック機能、分離（牽制） |
| **4. 情報と通信** | 必要な情報の収集・伝達・処理 | 報告制度、内部通報制度、情報システム |
| **5. 監視活動** | 内部統制が有効に機能しているかの評価 | 内部監査、経営層による定期確認 |
| **6. IT一般統制** | 情報システムの安全性・信頼性確保 | アクセス管理、バックアップ、セキュリティ |

### J-SOXと実務対応

**金融商品取引法（J-SOX）** では、上場企業の経営者が内部統制の評価報告書を提出する義務があります。

| 項目 | 内容 |
| --- | --- |
| **対象企業** | 上場企業、大会社（資本金5億円以上など） |
| **評価対象** | 財務報告の信頼性に関わる内部統制 |
| **報告時期** | 年1回（決算後） |
| **外部監査** | 有限責任監査法人による監査を受ける |
| **非遵守時** | 課徴金、公示、上場廃止のリスク |

## 典型的なつまずき

1. **CSRと ESGの違いが曖昧**
   - CSRは「企業が社会に対して何を責任として果たすべきか」という経営行動の観点
   - ESGは「投資家が企業の持続可能性をどう評価するか」という投資判断の観点
   - 同じテーマ（例：環境対策）でも、CSR論では「社会への責任」、ESG投資では「リスク・機会要因」として見方が異なる

2. **ステークホルダー理論を「従業員第一主義」と誤解**
   - ステークホルダー理論は「複数の利害関係者のバランスを取る」もので、従業員を優先するわけではない
   - 株主、顧客、取引先、地域社会など、文脈に応じて優先順位が変わる

3. **CSV を CSR の延長と見なす**
   - CSVは「社会課題解決が事業価値になる」という逆転の発想
   - 単なる「社会への配慮+事業」ではなく、「社会課題と競争優位が同時実現される状態」を指す

4. **モラルハザードを「経営者の悪意」だけと理解**
   - モラルハザードは、悪意がなくても情報非対称性と誘因構造により発生する組織的問題
   - 例：業績連動報酬が短期的な粉飾決算を誘発する

5. **日本のガバナンス機構を暗記だけで理解した気になる**
   - 3つの機構（監査役会、指名委員会等、監査等委員会）の特徴を、それぞれ「なぜそう設計されたのか」から理解する
   - 「99%が監査役会設置会社」という現状と「監査等委員会設置会社が推奨される理由」の両方を見る

6. **内部統制を「不正防止」だけと理解**
   - 内部統制は「有効性・効率性」「財務報告の信頼性」「法令遵守」「資産保護」の4つの目的を持つ
   - 規制対応だけでなく、経営目標達成の仕組みとしての側面を押さえる

## 問題を解くときの観点

### 問われているのは何か

| 問題のパターン | 見分けるポイント | 対応方法 |
| --- | --- | --- |
| **CSRについて** | 「企業の責任」「社会への関わり」「キャロル」という用語 | ピラミッドの層構造と各層の内容を整理 |
| **ステークホルダー理論** | 「株主中心」「複数の利益」「取引先」「地域社会」という記述 | フリーマン、株主中心 vs ステークホルダー主義の違い |
| **CSVについて** | 「社会課題と事業価値」「同時実現」「競争優位」 | ポーター＆クラマー、CSR との比較 |
| **ESG** | 「投資」「評価」「持続可能性」「非財務情報」 | E/S/G各要素の定義、ESG投資の類型 |
| **エージェンシー理論** | 「経営者」「株主」「情報の非対称性」「モラルハザード」 | プリンシパル＝株主、エージェント＝経営者 |
| **ガバナンス機構** | 「監査役」「社外取締役」「委員会」「独立性」 | 3つの機構の特徴、日本の現状 |
| **内部統制** | 「統制環境」「リスク評価」「J-SOX」「不正防止」 | COSO フレームワーク、4つの目的、6つの要素 |

### 解答のポイント

- **比較を求める問題**：必ず「対比軸」（観点）を明示して、A と B の違いを整理してから記述する
- **企業価値との関係**：CSR・ESG・ガバナンス → 信頼向上 → リスク低減 → 長期的企業価値向上、という因果チェーンを意識する
- **日本の特殊性**：監査役制度、長期雇用、ステークホルダー志向など、日本企業の伝統的特性と欧米型ガバナンスの違いを理解
- **最新動向**：ESG投資の拡大、監査等委員会設置会社の推奨、サステナビリティの重要性の高まり

## 確認問題

<Callout title="問1：キャロルのCSRピラミッド">
  キャロルが示した CSR ピラミッドの4層を下から順に述べよ。また、各層の特徴を説明できるか。

  **解答**：
  1. 経済的責任（基礎）：利益と雇用を生み出す基本的責任
  2. 法的責任：法令順守、規制遵守
  3. 倫理的責任：法では求められないが社会的に期待される行動
  4. 社会貢献的責任（フィランソロピー）：寄附やボランティア、企業に直接見返りがない活動

  下層から上層へ積み上げられ、すべての企業が頂点に達するわけではなく、経営資源や業界特性によって実現度が異なる。
</Callout>

<Callout title="問2：ステークホルダー理論と株主中心主義の違い">
  フリーマンのステークホルダー理論と株主中心主義の根本的な違いを説明せよ。また、CSR との関係を述べよ。

  **解答**：
  - **株主中心主義**：企業の目的は株主価値最大化。経営判断の優先順位は株主利益が最高位
  - **ステークホルダー理論**：企業は株主だけでなく、従業員・顧客・取引先・地域社会など複数の利害関係者のために存在。各ステークホルダーの利益をバランス良く考慮

  CSR は、ステークホルダー理論に基づき、企業が従業員・顧客・地域社会などに対して責任を果たす実践的な考え方。短期利益より長期的信頼と持続可能性を重視。
</Callout>

<Callout title="問3：CSRと CSVの違い">
  CSR（企業の社会的責任）と CSV（共通価値の創造）の根本的な違いを説明せよ。具体例を挙げよ。

  **解答**：
  - **CSR**：社会課題に対して企業が責任を果たすという考え方。社会貢献が主な目的で、企業にとってはコスト・負担（慈善活動）となる傾向
  - **CSV**：社会課題の解決が企業の競争優位（利益）につながるという考え方。社会価値と事業価値の同時実現を目指す

  **具体例**：
  - CSR：低所得層への寄附、環境教育支援→社会に良い、企業は費用負担
  - CSV：低所得層向けの廉価医療サービス開発→社会課題解決×新市場開拓→企業利益向上、社会課題解決の両立
</Callout>

<Callout title="問4：ESGの3つの要素">
  ESG（環境・社会・ガバナンス）の各要素について、企業が評価される代表的な観点を述べよ。

  **解答**：
  - **E（環境）**：GHG排出削減、脱炭素化（RE100）、廃棄物削減・リサイクル、水資源効率
  - **S（社会）**：労働環境改善（適正賃金、安全衛生）、人権尊重（児童労働・強制労働排除）、ジェンダー平等、地域社会への貢献、サプライチェーン管理（下請負企業の労働環境）
  - **G（ガバナンス）**：社外取締役比率・独立性、経営の透明性・情報開示、コンプライアンス体制、内部統制、内部通報制度

  ESG投資家は、これらの観点から企業の長期的持続可能性と競争力を評価する。
</Callout>

<Callout title="問5：エージェンシー理論とモラルハザード">
  エージェンシー理論におけるプリンシパル、エージェント、モラルハザードの定義を述べよ。また、モラルハザードの具体例を挙げよ。

  **解答**：
  - **プリンシパル**：企業の所有者である株主。企業の資産管理と価値向上を経営者に委託
  - **エージェント**：株主に代わって企業を経営する経営層。委託された職務を遂行する責任を持つ
  - **モラルハザード**：経営者が個人利益を優先し、株主利益を損なう自己中心的行動。情報の非対称性が存在するため、株主は経営者の行動を完全に監視できない

  **具体例**：
  - 過度な経営者報酬：個人報酬を増やす一方で配当を削減
  - 帝国建設：企業価値を損なう無理な買収や多角化で個人の権力を拡大
  - 利益操作・粉飾決算：株価上昇から短期利益
  - 情報隠蔽：都合の悪い経営情報を開示しない
</Callout>

<Callout title="問6：日本のガバナンス機構の比較">
  日本で認められている3つのガバナンス機構（監査役会設置会社、指名委員会等設置会社、監査等委員会設置会社）を比較せよ。それぞれの採用企業の特徴は何か。

  **解答**：
  | 項目 | 監査役会設置会社 | 指名委員会等設置会社 | 監査等委員会設置会社 |
  | --- | --- | --- | --- |
  | **採用企業** | 大多数（約99%、日本企業の伝統） | 少数派（外資系企業中心） | 増加傾向（2015年改正後、推奨型） |
  | **独立性** | 監査役が企業出身だと低い | 高い（社外取締役が主体） | 中程度（社外取締役が過半数） |
  | **日本文化との親和性** | 高い（長期経営、協調的） | 低い（米国式、短期志向） | 中程度（日本的改良版） |
  | **特徴** | 取締役会が経営、監査役が監視（分離） | 経営と監視を完全分離（委員会型） | 取締役会内に監査等委員会（統合型） |

  採用企業の特徴：
  - 監査役会：日本的経営（長期雇用、安定性重視）の企業が多数
  - 指名委員会等：グローバル企業、外資系、透明性・説明責任を強く求める環境
  - 監査等委員会：上場企業、ESG・ガバナンス強化を経営課題と見なす企業が増加中
</Callout>

<Callout title="問7：内部統制の目的と要素">
  COSO フレームワークにおける内部統制の4つの目的と6つの基本的要素を述べよ。

  **解答**：

  **4つの目的**：
  1. 有効性と効率性：経営目標達成と資源の効率的活用
  2. 財務報告の信頼性：正確で適時な財務情報の提供
  3. 法令遵守：適用法規の遵守
  4. 資産の安全保護：企業資産の盗難・不正使用防止

  **6つの基本的要素**：
  1. 統制環境：誠実性・倫理観・コンプライアンス意識が根付いた組織風土
  2. リスク評価：経営目標達成を阻害するリスクの識別・評価
  3. 統制活動：リスク低減のための具体的施策・手続き（承認ルール、分離牽制）
  4. 情報と通信：必要な情報の収集・伝達・処理（内部通報制度を含む）
  5. 監視活動：内部統制の有効性の継続的評価（内部監査）
  6. IT一般統制：情報システムの安全性・信頼性確保（アクセス管理、バックアップ）

  これら6つ要素が相互に機能することで、内部統制システムが有効に機能する。
</Callout>
