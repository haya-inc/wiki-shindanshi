## このページの役割

<Callout title="このページの役割">
    このページは、`何が人を動かすか` を整理する解説ページです。`マズロー`、`ハーズバーグ`、
    `期待理論`、`公平理論`、`職務特性モデル` を、`欲求の内容`、`判断の過程`、`仕事の設計` で切り分けます。
</Callout>

## 学習のポイント

- `内容理論` と `過程理論` を先に分ける
- `満足` と `不満` を同じ軸で見ない
- 最後に `職務設計` へつなぐ

## 試験で何が問われるか

- `マズロー`、`ハーズバーグ`、`アルダファー` の違いを説明できるか
- `期待理論` と `公平理論` を区別できるか
- `内発的動機付け` と `外発的動機付け` の違いを判断できるか
- `職務特性モデル` の 5 要素と `MPS` の考え方を押さえているか

## 定義と仕組み

### 内容理論

| 理論 | 何を見るか | 核心 |
| --- | --- | --- |
| マズロー | 欲求の階層 | 低次欲求が満たされると上位欲求が顕在化しやすい |
| ハーズバーグ | 満足と不満の要因 | `動機付け要因` は満足を高め、`衛生要因` は不満を防ぐ |
| アルダファー ERG | 欲求のまとまり | 存在、関係、成長の 3 区分で、複数欲求が同時に動く |
| デシ | 動機の源泉 | 外的報酬で内発的動機付けが下がる `アンダーマイニング効果` に注意する |

### 過程理論

| 理論 | 何を見るか | 核心 |
| --- | --- | --- |
| 期待理論 | 努力から報酬までの見通し | `期待 × 手段性 × 誘意性` のどれかがゼロなら動機づけも弱い |
| 公平理論 | 他者との比較 | `報酬 / 投入` の比率の不公平感が行動を変える |
| 目標設定理論 | 目標の置き方 | 明確で困難な目標とフィードバックが成果を高める |

### 職務設計

`職務特性モデル` は、`技能多様性`、`タスク完結性`、`タスク重要性`、`自律性`、`フィードバック` の
5 要素で仕事を設計し、`有意味感`、`責任感`、`結果認知` を通じて動機づけを高めます。

`職務充実` は責任や裁量を広げる `垂直的拡大`、`職務拡大` は類似作業を増やす `水平的拡大` です。

## 典型例

| 論点 | 典型的な読み方 |
| --- | --- |
| 給与を上げてもやる気が上がらない | ハーズバーグの `衛生要因` を疑う |
| 努力しても評価されないと思っている | 期待理論の `手段性` を疑う |
| 同僚と比べて不公平感が強い | 公平理論で読む |
| 仕事が単調で裁量もない | 職務特性モデルや職務充実で読む |

## 典型的なつまずき

- `満足が高い = 不満がない` と考えてしまう
- `期待理論` を単なる性格論として覚えてしまう
- `公平理論` と `期待理論` の比較軸を持てない
- `職務拡大` と `職務充実` を取り違える
- `内発的動機付け` に対する外的報酬の副作用を落とす

## 問題を解くときの観点

- 問題は `欲求の内容`、`判断の過程`、`仕事の設計` のどこか
- 不満を消したいのか、満足を高めたいのか
- 公平感、達成感、裁量、フィードバックのどれが不足しているか
- 人を変える話か、仕事の設計を変える話か
