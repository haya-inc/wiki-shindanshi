## このページの役割

<Callout title="このページの役割">
    このページは、`外部環境` を見る競争戦略と、`内部資源` を見る経営資源戦略を切り分ける解説ページです。
    似たフレームワークを `分析対象` の違いで使い分けられる状態を目指します。
</Callout>

<Callout title="このページの読み方">
  ポーターの5フォースは「業界の収益性を決めるのは何か」、VRIO は「自社の強みは持続するか」という問いに答えます。外部を見るか内部を見るかで使い分ける──この軸が最も大切です。
</Callout>

## 学習のポイント

- `5フォース` は業界構造（外部環境）
- `3つの基本戦略` は競争優位の源泉（勝ち筋の選択）
- `バリューチェーン` は活動の連鎖（内部の強み）
- `VRIO` は資源の持続的競争優位（内部資源の評価）

## 試験で何が問われるか

- 5フォースの各力を強める要因を列挙できるか
- コストリーダーシップ・差別化・集中の使い分けができるか
- VRIOフレームワークで持続的競争優位を **判定** できるか
- 先発優位と後発優位の源泉を説明できるか

## ポーターの5フォース分析

5つの力が **強い** ほど、業界の収益性は **低い** 。

### 各力と強まる要因

| 力 | 強まる要因 |
| --- | --- |
| 既存業者間の敵対関係 | 競争業者が多数/同規模、成長率低い、固定費高い、差別化困難、撤退障壁高い |
| 新規参入の脅威 | 参入障壁が低い（規模の経済なし、ブランドなし、必要投資額小、スイッチングコスト低い） |
| 代替品の脅威 | 代替品のコストパフォーマンス向上、スイッチングコスト低い |
| 買い手の交渉力 | 大口購入、標準品、スイッチングコスト低い、後方統合の脅威 |
| 売り手の交渉力 | 供給者が少数、代替品なし、前方統合の脅威 |

### HHI（ハーフィンダール・ハーシュマン指数）

市場の集中度を測る定量指標。各企業の市場シェア（%）を2乗して合計します。

```
【計算例】
  A社40%、B社30%、C社20%、D社10%
  HHI = 40² + 30² + 20² + 10² = 1600+900+400+100 = 3,000

【判定基準】
  1,500未満 → 非集中的（競争的市場）
  1,500〜2,500 → やや集中的
  2,500超 → 高度に集中的
```

## ポーターの3つの基本戦略

| 戦略 | 競争優位の源泉 | ターゲット | リスク |
| --- | --- | --- | --- |
| コストリーダーシップ | 低コスト | 業界全体 | 技術革新でコスト優位喪失 |
| 差別化 | 独自性 | 業界全体 | コスト増大、模倣 |
| 集中（コスト集中/差別化集中） | 低コスト or 独自性 | **特定セグメント** | セグメントの縮小、大手参入 |

**スタック・イン・ザ・ミドル**：コストリーダーシップと差別化のどちらも中途半端な状態。収益性が低下します。

中小企業は経営資源が限られるため、**差別化集中戦略** が基本です。

### 競争地位別戦略（コトラー）

| 地位 | 特徴 | 主な戦略 |
| --- | --- | --- |
| リーダー | 最大シェア | 同質化戦略、周辺需要拡大、非価格対応 |
| チャレンジャー | 2〜3番手 | リーダーとの **差別化** |
| フォロワー | 上位に追随 | 模倣戦略 |
| ニッチャー | 小さな市場に特化 | 集中・専門化（ミニリーダー） |

### 先発優位と後発優位

| | 源泉 | 内容 |
| --- | --- | --- |
| 先発優位 | 技術的リーダーシップ | 経験曲線、特許による優位 |
| 先発優位 | 希少資源の先制確保 | 立地、チャネル、棚スペース |
| 先発優位 | スイッチングコスト | ブランドの結びつき、学習コスト |
| 後発優位 | フリーライダー効果 | 先発者の市場教育コストへのただ乗り |
| 後発優位 | 先発者の失敗からの学習 | 低リスクでの参入 |
| 後発優位 | 技術的陳腐化への対応 | 最新技術での参入 |

## バリューチェーン（価値連鎖）

```
【支援活動】
  全般管理（インフラストラクチャー）
  人事・労務管理
  技術開発
  調達活動

【主活動】
  購買物流 → 製造 → 出荷物流 → マーケティング・販売 → サービス → マージン
```

各活動のコスト構造と差別化の源泉を把握し、競合との比較で強み・弱みを特定します。活動間の **フィット**（整合性）が持続的競争優位を生むとポーターは主張しています。

## RBV と VRIO フレームワーク

### RBV（資源ベース理論）

バーニーが提唱。競争優位の源泉は企業の **内部資源** にあるとする理論。ポーターの外部環境重視の視点と対をなします。

### VRIO フレームワーク

| 基準 | 問い |
| --- | --- |
| V（Value） | 経営資源は価値があるか？（機会活用 or 脅威中和） |
| R（Rareness） | その資源は希少か？ |
| I（Imitability） | 模倣が困難か？ |
| O（Organization） | 組織的に活用できているか？ |

### VRIO の評価結果

| V | R | I | O | 競争上の帰結 |
| --- | --- | --- | --- | --- |
| × | − | − | − | 競争劣位 |
| ○ | × | − | − | 競争均衡 |
| ○ | ○ | × | − | 一時的競争優位 |
| ○ | ○ | ○ | × | 優位を活用できていない |
| ○ | ○ | ○ | ○ | **持続的競争優位** |

### 模倣困難性の4つの要因

| 要因 | 内容 |
| --- | --- |
| 独自の歴史的条件 | 長年の経験の蓄積 |
| 因果関係の曖昧性 | 競争優位の源泉が外部から分かりにくい |
| 社会的複雑性 | 組織文化、人間関係等の複雑な要因 |
| 特許 | 法的に保護された技術 |

## 典型的なつまずき

- `5フォース` を自社分析だと思ってしまう → 5フォースは **業界構造**（外部環境）の分析
- `差別化` と `集中` を同じものとして扱う → 差別化は業界全体、集中は **特定セグメント**
- `バリューチェーン` と `VRIO` の分析対象が曖昧 → バリューチェーンは **活動の連鎖**、VRIO は **経営資源**
- `RBV` を単に `強み` の言い換えとして覚える → VRIO を満たす資源のみが **持続的競争優位** になる
- リーダーの戦略を「差別化」と答えてしまう → リーダーの戦略は **同質化**（チャレンジャーの差別化を模倣して無効化）

## 問題を解くときの観点

- いま見ているのは `業界全体` か `自社内部` か
- 問われているのは `競争の厳しさ` か `勝ち筋` か `強みの持続性` か
- 強みを確認したいなら、`活動` を見るのか `資源` を見るのか
- 競争地位は リーダー/チャレンジャー/フォロワー/ニッチャー のどれか

## 確認問題

<Callout title="問1：5フォースの判断">
    「業界の固定費が高く、差別化が困難で、撤退障壁も高い」場合、5フォースのどの力が強まるか。

    **解答**：既存業者間の敵対関係。固定費が高いと稼働率を維持するため値下げ競争が起き、差別化困難で撤退もできないため競争が激化する。
</Callout>

<Callout title="問2：VRIOの判定">
    「自社の製品開発技術は価値があり希少だが、競合が2〜3年で追いつける」場合、VRIO の評価結果はどれか。

    **解答**：一時的競争優位（V=○、R=○、I=×）。模倣困難性を欠くため、競争優位は一時的にとどまる。
</Callout>

<Callout title="問3：競争地位別戦略">
    「業界最大シェアの企業が、2番手企業の新しい差別化商品と同等の商品を素早く投入した」。この行動は何戦略か。

    **解答**：同質化戦略（リーダーの定石戦略）。チャレンジャーの差別化を模倣して無効化し、シェア優位を維持する。
</Callout>
