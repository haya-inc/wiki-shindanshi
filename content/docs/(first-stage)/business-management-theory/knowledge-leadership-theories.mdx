## このページの役割

<Callout title="このページの役割">
    このページは、`人をどう導くか` を整理する解説ページです。`特性`、`行動`、`状況との適合`、
    `変革の推進` という見方の違いを押さえ、`SL理論`、`パス=ゴール理論`、`変革型リーダーシップ`
    を混同しないようにします。
</Callout>

## 学習のポイント

- まず `何を見ている理論か` を分ける
- 次に `フォロワー` と `状況` の扱いを見る
- 最後に `交流型` と `変革型` を比較する

## 試験で何が問われるか

- `特性理論`、`行動理論`、`状況適応理論` の違いを説明できるか
- `SL理論` の成熟度とスタイルの対応を押さえているか
- `パス=ゴール理論` の 4 スタイルを使い分けられるか
- `交流型` と `変革型` の違いを判断できるか

## 定義と仕組み

### 理論の流れ

| 理論群 | 何を見るか | 代表論点 |
| --- | --- | --- |
| 特性理論 | リーダー本人の資質 | 知能、決断力、社交性など |
| 行動理論 | リーダーの振る舞い | 専制型 / 民主型、P機能 / M機能 |
| 状況適応理論 | 状況や部下との適合 | フィードラー、SL理論、パス=ゴール理論 |
| 変革型リーダーシップ | 変化を起こす力 | ビジョン、知的刺激、個別配慮 |

### 行動理論で押さえる比較

- `レビン`: `専制型`、`民主型`、`放任型`
- `オハイオ研究`: `構造づくり` と `配慮`
- `PM理論`: `P機能` と `M機能`
- `マネジリアル・グリッド`: `人への関心` と `業績への関心`

どれも `行動をどう組み合わせるか` を見る理論です。

### 状況適応理論で押さえる比較

| 理論 | 見る条件 | 核心 |
| --- | --- | --- |
| フィードラー | 関係、タスク、地位権限 | 状況の有利さに応じて有効なスタイルが変わる |
| SL理論 | フォロワーの成熟度 | `M1` から `M4` に応じて `教示`、`説得`、`参加`、`委任` を使い分ける |
| パス=ゴール理論 | 目標達成の障害 | 指示型、支援型、参加型、達成志向型で道筋を示す |

### 変革型リーダーシップ

`変革型リーダーシップ` は、`理想化された影響`、`鼓舞する動機付け`、`知的刺激`、`個別配慮` の 4 要素で、
短期的な報酬管理よりも `ビジョンを示して変化を起こすこと` を重視します。`交流型` は報酬と罰で管理する点が中心です。

## 典型例

| 状況 | 合わせやすい理論 |
| --- | --- |
| 未熟なメンバーに手順を明確に示したい | SL理論の `教示型` |
| 既存の前提を疑わせ、新しい発想を促したい | 変革型リーダーシップの `知的刺激` |
| 仕事の進め方は分かっているが負荷が高い | パス=ゴール理論の `支援型` |
| 目標達成と集団維持を両立したい | PM 理論の `PM型` |

## 典型的なつまずき

- `モチベーション理論` と `リーダーシップ論` を一つの話として覚える
- `SL理論` と `パス=ゴール理論` の条件を混同する
- `変革型` を単なるカリスマ性だけで理解してしまう
- `民主型` を常に最善と固定してしまう

## 問題を解くときの観点

- 問われているのは `資質`、`行動`、`状況適合`、`変革` のどれか
- 主語は `リーダー本人` か `フォロワー` か `組織の状況` か
- 目標達成と人間関係のどちらをどう補う話か
- 日常管理の話か、組織変革の話か
