## このページの役割

<Callout title="このページの役割">
    このページは、`全社戦略` と `成長戦略` をまとめて整理する解説ページです。`どの事業へ資源配分するか` と
    `どう成長するか` を分けて考えられる状態を目指します。
</Callout>

## 学習のポイント

- `全社戦略` は事業ポートフォリオを見る
- `成長戦略` は既存 / 新規、市場 / 製品の組み合わせで見る
- `M&A` と `アライアンス` は統合の深さで分ける

## 試験で何が問われるか

- PPM の軸と位置付けを説明できるか
- 多角化の種類を区別できるか
- アンゾフの成長マトリクスを、事例文の施策に当てられるか
- M&A とアライアンスの違いを言えるか

## 定義と仕組み

### 全社戦略

`全社戦略` は、複数事業を持つ企業が `どの事業へ投資するか`、`どの事業から撤退するか` を決める考え方です。`PPM` や `多角化` はここで使います。

### PPM と多角化

`PPM` は `市場成長率` と `相対的市場シェア` で事業を分類し、資源配分を考える道具です。`多角化` は、既存事業との関連の強さで、`水平`、`垂直`、`集中`、`集成` などに分けます。

### 成長戦略

`アンゾフの成長マトリクス` は、`既存市場 / 新市場` と `既存製品 / 新製品` で成長の方向を整理します。市場浸透、市場開拓、製品開発、多角化の 4 つに分かれます。

### M&A とアライアンス

`M&A` は資本を伴う統合、`アライアンス` は資本を伴わない連携として整理すると分かりやすいです。統合の強さ、意思決定の一体化、リスク分担のあり方が違います。

## 典型例

| 論点 | 見ること | よくある出題の形 |
| --- | --- | --- |
| PPM | 資源配分 | どの事業へ投資するか |
| 多角化 | 関連性と範囲 | 関連多角化か集成多角化か |
| アンゾフ | 成長の方向 | 既存 / 新規の組み合わせ判断 |
| M&A / アライアンス | 統合の深さ | 買収か提携かの選択 |

## 典型的なつまずき

- `市場浸透` と `製品開発` を逆に覚える
- `関連多角化` と `垂直統合` の違いが曖昧
- `PPM` を競争戦略の道具のように扱う
- `M&A` と `アライアンス` をどちらも `提携` として一括で覚える

## 問題を解くときの観点

- いま問われているのは `どの事業を持つか` か `どう成長するか` か
- 新しいのは `市場` か `製品` か
- 統合の深さは `資本` を伴うかどうかで切れるか
