## このページの役割

<Callout title="このページの役割">
    このページは、`人をどう採り、育て、評価し、報いるか` を整理する解説ページです。`メンバーシップ型`
    と `ジョブ型` の違いを軸に、`育成`、`評価`、`報酬`、`ダイバーシティ` を一つの人事制度として見ます。
</Callout>

## 学習のポイント

- まず `雇用の考え方` を分ける
- 次に `育成` と `評価` を分ける
- 最後に `報酬` と `多様性活用` へ広げる

## 試験で何が問われるか

- `メンバーシップ型` と `ジョブ型` の違いを説明できるか
- `OJT`、`Off-JT`、`ジョブローテーション` の役割を区別できるか
- `MBO`、`コンピテンシー評価`、`360度評価` の違いを押さえているか
- `職能給`、`職務給`、`役割給`、`成果給` を制度目的で使い分けられるか

## 定義と仕組み

### 雇用と人事制度の型

| 観点 | メンバーシップ型 | ジョブ型 |
| --- | --- | --- |
| 雇用 | 長期雇用を前提に人で採る | 職務で採る |
| 配置 | 異動、ローテーションが多い | 職務範囲が比較的明確 |
| 評価 | 年功や能力蓄積を見やすい | 職務内容や成果を見やすい |
| 育成 | 企業内育成と OJT が中心 | 専門性と自己投資を重視しやすい |

### 人材育成

- `OJT`: 現場で仕事をしながら学ぶ
- `Off-JT`: 研修やセミナーで体系的に学ぶ
- `ジョブローテーション`: 幅を広げる
- `メンタリング / コーチング`: 長期支援と自発行動の支援を分ける
- `CDP`: 中長期のキャリア形成を支える

### 人事評価と評価エラー

| 手法 | 見るもの |
| --- | --- |
| MBO | 目標達成度 |
| コンピテンシー評価 | 高業績者の行動特性 |
| 360度評価 | 多面的な見え方 |
| BSC | 財務、顧客、業務、学習の 4 視点 |

代表的な評価エラーとして、`ハロー効果`、`中心化傾向`、`寛大化傾向`、`近接誤差` があります。

### 報酬制度

| 制度 | 向いている考え方 |
| --- | --- |
| 年功給 | 長期雇用、安定運用 |
| 職能給 | 能力蓄積の評価 |
| 職務給 | ジョブ型、職務の重さ |
| 役割給 | 担当役割の大きさ |
| 成果給 | 結果責任の明確化 |
| ストックオプション | 中長期インセンティブ |

### ダイバーシティとインクルージョン

`ダイバーシティ` は属性の多様性、`インクルージョン` は多様な人材が実際に意思決定へ参加し、能力を発揮できる状態です。
多様性だけで終わらず、活かせる環境まで整える必要があります。

## 典型例

| 課題 | 見るべき論点 |
| --- | --- |
| 現場で人が育たない | OJT、メンタリング、ジョブローテーション |
| 評価に納得感がない | 評価基準、評価エラー、360度評価 |
| 専門職の定着が弱い | ジョブ型、職務給、キャリア開発 |
| 多様な人材が活躍しにくい | ダイバーシティとインクルージョン |

## 典型的なつまずき

- `育成制度` と `評価制度` を一緒に覚えてしまう
- `職能給` と `職務給` を混同する
- `MBO` を単なる目標設定だと思ってしまう
- `ダイバーシティ` を人数比の話だけで終わらせる

## 問題を解くときの観点

- 問題は `採用 / 配置 / 育成 / 評価 / 報酬` のどこにあるか
- `能力の蓄積` を見たいのか、`職務や成果` を見たいのか
- 制度を変えるのか、運用を変えるのか
- 個人の納得感と組織の整合のどちらが崩れているか
