企業経営理論は、`経営戦略論`、`組織論`、`マーケティング論` をまとめて扱う科目です。理論名やフレームワークを暗記するだけでは足りず、`何を分析する理論か`、`似た概念と何が違うか`、`どの場面で使うか` を見抜く力が求められます。

<Callout title="この科目の位置づけ">
  全社戦略と競争戦略、組織構造と人材マネジメント、STP と 4P の前後関係を崩さず、`分析対象` と `使う場面` の違いで整理します。
</Callout>

## 学習のポイント

<Cards>
  <Card title="戦略">
    SWOT、PPM、5フォース、バリューチェーン、VRIO を、分析対象の違いで整理します。
  </Card>
  <Card title="組織">
    組織構造、バーナード、サイモン、モチベーション理論、リーダーシップ論を比較で押さえます。
  </Card>
  <Card title="マーケティング">
    顧客理解から施策設計までの流れを維持し、理論を `使う場面` とセットで整理します。
  </Card>
</Cards>

## この科目で問われること

<Steps>
  <Step>
    ### どの理論を当てるべきか

    企業の状況に対して、どの理論やフレームワークを使うべきかを判断する力が問われます。
  </Step>
  <Step>
    ### 似た概念の違いを区別できるか

    全社戦略と競争戦略、モチベーションとリーダーシップ、STP と 4P のような近い概念を切り分けます。
  </Step>
  <Step>
    ### 2次試験へ言い換えられるか

    戦略、組織、人事、顧客、販路の論点を、事例Ⅰと事例Ⅱで使う言葉へ変換できることが重要です。
  </Step>
</Steps>

## 学習の優先順位

<Cards>
  <Card title="経営戦略論">
    - 最優先: SWOT、PPM、5フォース、3つの基本戦略、バリューチェーン、VRIO
    - 次に固める: 多角化、アンゾフ、M&A、アライアンス、イノベーション
    - 後半: 国際経営、CSR・ESG、プラットフォーム、デジタル戦略
  </Card>
  <Card title="組織論">
    - 最優先: 組織構造、バーナード、サイモン、モチベーション理論、リーダーシップ論
    - 次に固める: 人的資源管理、組織文化、組織変革、コンフリクト
    - 後半: 労働関連法規、心理的安全性、ダイバーシティ
  </Card>
  <Card title="マーケティング論">
    - 最優先: STP、4P、消費者行動、新製品開発、流通チャネル、プロモーション
    - 次に固める: ブランド、価格戦略、サービスマーケティング、CRM
    - 後半: デジタルマーケティング、共創、BtoB マーケティング
  </Card>
</Cards>

## 典型的なつまずき

- `全社戦略` と `競争戦略` を混同する
- 5フォース、バリューチェーン、VRIO の `分析対象` を取り違える
- 組織構造の違いを、権限や調整の観点で説明できない
- モチベーション理論とリーダーシップ理論を一緒に覚えてしまう
- STP と 4P の前後関係が曖昧なまま、施策だけを覚えてしまう
