## このページの役割

<Callout title="このページの役割">
    このページは、`いつ権利が消え、誰がどこまで責任を負うか` を整理する解説ページです。`消滅時効`、
    `完成猶予 / 更新`、`通常保証 / 連帯保証`、`個人根保証` を比較で押さえます。
</Callout>

## 学習のポイント

- まず `時効の二重基準`
- 次に `完成猶予` と `更新`
- 最後に `通常保証` と `連帯保証`

## 試験で何が問われるか

- 消滅時効の `5年 / 10年` の考え方を説明できるか
- `完成猶予` と `更新` の違いを区別できるか
- `通常保証` と `連帯保証` の差を押さえているか
- `個人根保証` の `極度額` を押さえているか

## 定義と仕組み

### 消滅時効

`消滅時効` は、権利を行使できることを `知った時から5年`、または `行使できる時から10年` の
いずれか早い方で完成します。試験では、この `主観的起算点` と `客観的起算点` の二重基準を問われます。

### 完成猶予と更新

| 類型 | 効果 | 典型例 |
| --- | --- | --- |
| 完成猶予 | 完成を一時的に止める | 裁判上の請求、催告、協議を行う旨の合意 |
| 更新 | 時効期間を最初からやり直す | 権利の承認、確定判決、和解 |

`催告` は `6か月` だけ猶予する点、`権利の承認` は更新事由になる点が頻出です。

### 保証の基本

保証は、主たる債務者が履行しないときに保証人が責任を負う制度です。契約は `書面または電磁的記録`
が必要です。

| 比較軸 | 通常保証 | 連帯保証 |
| --- | --- | --- |
| 催告の抗弁権 | あり | なし |
| 検索の抗弁権 | あり | なし |
| 分別の利益 | あり | なし |

### 個人根保証と情報提供

`個人根保証` は `極度額` の定めが必須です。事業に係る債務の保証では、保証意思確認や情報提供義務が
問われることがあります。ここは `保証人保護` の方向で理解します。

## 典型例

| 論点 | 典型的な見分け方 |
| --- | --- |
| 完成猶予と更新 | `一時停止` か `リセット` か |
| 通常保証と連帯保証 | `抗弁権が残るか` で区別する |
| 個人根保証 | `極度額` があるかを見る |

## 典型的なつまずき

- `時効の完成猶予` と `更新` を同じ扱いにしてしまう
- `主観的起算点` と `客観的起算点` を混同する
- 連帯保証でも催告の抗弁権があると誤解する
- `個人根保証` の `極度額` を落とす

## 問題を解くときの観点

- まず `時効期間` の話か `保証責任` の話かを切る
- 時効なら `止まるだけ` か `最初からやり直し` かを見る
- 保証なら `通常保証` か `連帯保証` かを先に判断する
- 保証人保護の仕組みとして `極度額` や `情報提供` があるかを確認する
