## このページの役割

<Callout title="このページの役割">
    このページは、`契約で何が発生し、履行が崩れたときにどう処理するか` を整理する解説ページです。
    `契約の成立`、`債務不履行`、`契約不適合責任`、`物権変動`、`担保` を一連でつなぎます。
</Callout>

## 学習のポイント

- まず `契約の入口`
- 次に `履行が崩れたとき`
- 最後に `権利の対抗要件` と `担保`

## 試験で何が問われるか

- 契約がどう成立するかを説明できるか
- `請負` と `委任` の違いを区別できるか
- `契約不適合責任` の救済手段を押さえているか
- 不動産と動産の `対抗要件` を使い分けられるか
- `抵当権` と `質権` の違いを説明できるか

## 定義と仕組み

### 契約の成立

契約は、原則として `申込み` と `承諾` の意思表示が合致すると成立します。試験では、`締結の自由`、
`相手方選択の自由`、`内容の自由`、`方式の自由` をまとめて `契約自由の原則` として整理します。

### 契約類型の比較

| 契約 | 何を約束するか | 比較で押さえること |
| --- | --- | --- |
| 売買 | 財産権の移転と代金支払い | 双務、有償、諾成 |
| 請負 | 仕事の完成 | `結果責任` がある |
| 委任 / 準委任 | 事務処理の委託 | `仕事の完成` までは約束しない |
| 賃貸借 | 使用収益と賃料支払い | 継続利用を中心に見る |
| 贈与 | 無償の財産移転 | 書面によらない贈与の撤回に注意する |

### 債務不履行と契約不適合責任

`履行遅滞`、`履行不能`、`不完全履行` が債務不履行の基本です。`契約不適合責任` は、旧 `瑕疵担保責任`
に代わる考え方で、`追完請求`、`代金減額請求`、`損害賠償請求`、`契約解除` の 4 つの救済手段を持ちます。

### 物権と担保

| 論点 | 比較軸 | 押さえること |
| --- | --- | --- |
| 物権変動の対抗要件 | 不動産 / 動産 | 不動産は `登記`、動産は `引渡し` |
| 抵当権 | 非占有担保 | 占有を移さずに使い続けられる |
| 質権 | 占有担保 | 占有を移す |
| 留置権 | 法定担保 | 優先弁済権はない |

## 典型例

| 論点 | 典型的な見分け方 |
| --- | --- |
| 請負と委任 | `仕事の完成` を約束しているか |
| 契約不適合責任 | 契約内容に `適合しているか` で見る |
| 抵当権と質権 | `占有を移すか` で区別する |
| 不動産と動産の対抗要件 | `登記` か `引渡し` かで分ける |

## 典型的なつまずき

- `請負` と `委任` を業務委託という言葉だけでまとめてしまう
- `契約不適合責任` を旧 `瑕疵担保責任` の感覚で覚える
- `抵当権` と `質権` を担保という言葉だけで一括りにする
- `登記` と `引渡し` の対抗要件を混同する

## 問題を解くときの観点

- 問われているのは `契約の成立` か `履行後の処理` か
- 救済手段は `追完`、`減額`、`損害賠償`、`解除` のどれか
- 権利を第三者に主張するには `登記` か `引渡し` か
- 担保は `占有を移すか`、`法定か約定か` のどちらで切るか
