## このページの役割

このページは、`損益分岐点分析` を固める知識ノードです。`限界利益`、`固定費`、`変動費率` の関係を押さえ、`損益分岐点売上高`、`安全余裕率`、`目標利益売上高` を一連の流れで使えるようにします。

## 学習のポイント

- まず `変動費` と `固定費` を分ける
- 次に `限界利益` と `限界利益率` を固める
- そのうえで `損益分岐点` と `安全余裕率` を計算できるようにする

## 試験で何が問われるか

- 損益分岐点売上高や安全余裕率を計算できるか
- 固定費削減、単価改定、変動費率改善の効果を比較できるか
- CVP の結果を、事例Ⅳの助言や利益計画へつなげられるか

## まず固める式の意味

| 項目 | 意味 | 使いどころ |
| --- | --- | --- |
| 限界利益 | 売上高から変動費を引いた利益 | 固定費回収力を見る |
| 限界利益率 | 売上高のうち限界利益が占める割合 | 損益分岐点売上高を求める |
| 損益分岐点売上高 | 利益がゼロになる売上高 | 赤字化ラインを読む |
| 安全余裕率 | 現在売上高と損益分岐点の距離 | どこまで売上減に耐えられるかを見る |

## 何が利益を動かすか

CVP では、利益を動かす要因を `単価`、`販売数量`、`変動費率`、`固定費` に切れます。式を当てるだけでなく、`どこを動かすと利益が改善しやすいか` まで言えることが重要です。

## 公式一覧

### 基本公式

```
限界利益 = 売上高 − 変動費
限界利益率 = 限界利益 ÷ 売上高 = 1 − 変動費率
変動費率 = 変動費 ÷ 売上高

損益分岐点売上高 = 固定費 ÷ 限界利益率
                 = 固定費 ÷ (1 − 変動費率)
損益分岐点数量 = 固定費 ÷ 単位限界利益
               = 固定費 ÷ (販売単価 − 単位変動費)
```

### 安全余裕率と損益分岐点比率

```
安全余裕率 = (実際売上高 − BEP売上高) ÷ 実際売上高 × 100
損益分岐点比率 = BEP売上高 ÷ 実際売上高 × 100
安全余裕率 + 損益分岐点比率 = 100%
```

安全余裕率は 20%以上が望ましい。

### 目標利益達成売上高

```
目標利益達成売上高 = (固定費 + 目標利益) ÷ 限界利益率
目標利益達成数量 = (固定費 + 目標利益) ÷ 単位限界利益
```

**税引後目標利益の場合**（2次試験で頻出）：

```
税引前目標利益 = 税引後目標利益 ÷ (1 − 税率)
必要売上高 = (固定費 + 税引前目標利益) ÷ 限界利益率
```

### 経営レバレッジ係数（DOL）

```
DOL = 限界利益 ÷ 営業利益 = 1 ÷ 安全余裕率
```

売上高が 1%変化 → 営業利益は DOL%変化。固定費比率が高いほど DOL が大きい（ハイリスク・ハイリターン）。

### 高低点法（費用の固変分解）

```
変動費率 = (最高点の総費用 − 最低点の総費用) ÷ (最高点の売上高 − 最低点の売上高)
固定費 = 総費用 − 変動費率 × 売上高
```

## 計算手順

### 基本の CVP 計算

```
【データ】売上高 1,000万円、変動費 600万円、固定費 300万円

手順1: 変動費率と限界利益率
  変動費率 = 600 ÷ 1,000 = 0.6（60%）
  限界利益率 = 1 − 0.6 = 0.4（40%）

手順2: 損益分岐点売上高
  BEP売上高 = 300 ÷ 0.4 = 750万円

手順3: 安全余裕率
  安全余裕率 = (1,000 − 750) ÷ 1,000 × 100 = 25%

手順4: 利益の検算
  利益 = 1,000 − 600 − 300 = 100万円
  BEP検算: 750 × 0.6 + 300 = 750 ✓（売上高＝総費用）
```

### 税引後目標利益の計算

```
【データ】固定費 300万円、変動費率 60%、法人税率 30%
【条件】税引後利益 140万円を達成する売上高は？

手順1: 税引前目標利益に変換
  税引前目標利益 = 140 ÷ (1 − 0.3) = 200万円

手順2: 必要売上高
  必要売上高 = (300 + 200) ÷ 0.4 = 1,250万円

検算: 1,250 − 750 − 300 = 200（税引前）→ 200 × 0.7 = 140 ✓
```

### 高低点法の計算

```
【データ】
  6月: 売上高 400万円、総費用 400万円（最低点）
  7月: 売上高 1,000万円、総費用 700万円（最高点）

手順1: 変動費率 = (700 − 400) ÷ (1,000 − 400) = 0.5（50%）
手順2: 固定費 = 700 − 0.5 × 1,000 = 200万円
検算: 400 − 0.5 × 400 = 200 ✓
```

### 経営レバレッジの効果

```
DOL = 限界利益 ÷ 営業利益 = 400 ÷ 100 = 4.0
→ 売上高が10%増加すると、営業利益は40%増加
→ 売上高が10%減少すると、営業利益は40%減少
```

## 問題を解くときの観点

- 求めたいのは `売上高` か `販売数量` か
- 分母に使うのは `限界利益率` か `単位当たり限界利益` か
- 利益改善策として `単価`、`数量`、`変動費率`、`固定費` のどこを動かすか
- 税引後目標利益は必ず税引前に変換してから計算する

## 典型的なつまずき

- `固定費 ÷ 限界利益` として、**率**と**額**を混ぜる → BEP 売上高なら分母は**率**、BEP 数量なら分母は**額**
- 安全余裕率の分母を `損益分岐点売上高` にしてしまう → 正しくは `実際売上高` が分母
- 税引後目標利益をそのまま使ってしまう → まず `÷(1−税率)` で税引前に変換
- 高低点法で操業度（売上高）の最高・最低ではなく総費用の最高・最低を使ってしまう
- 経営レバレッジ係数の意味を理解せず計算だけで終わる → 固定費が高い企業のリスク構造を説明できるようにする
- CVP を計算問題だけで終わらせ、改善助言へつなげない

## 確認問題

<Callout title="問1：損益分岐点売上高と安全余裕率">
    売上高 2,000万円、変動費 1,200万円、固定費 500万円。損益分岐点売上高と安全余裕率を求めよ。

    **解答**：変動費率 = 1,200÷2,000 = 0.6。限界利益率 = 0.4。BEP = 500÷0.4 = 1,250万円。安全余裕率 = (2,000−1,250)÷2,000×100 = 37.5%。
</Callout>

<Callout title="問2：税引後目標利益">
    固定費 400万円、変動費率 50%、法人税率 40%。税引後利益 180万円を達成する売上高を求めよ。

    **解答**：税引前目標利益 = 180÷(1−0.4) = 300万円。必要売上高 = (400+300)÷0.5 = 1,400万円。検算：1,400−700−400 = 300 → 300×0.6 = 180 ✓。
</Callout>

<Callout title="問3：経営レバレッジ係数">
    限界利益 800万円、営業利益 200万円。売上高が 5%減少したとき、営業利益は何%変化するか。

    **解答**：DOL = 800÷200 = 4.0。営業利益の変化率 = 5% × 4.0 = 20%減少。営業利益は 200×0.8 = 160万円になる。
</Callout>
