## このページの役割

<Callout title="このページの役割">
    このページは、`勘定科目の5分類` と `仕訳の基本処理` を最初に固めるための解説ページです。何が増え、何が減るかを言葉で説明できる状態を目指します。
</Callout>

<Callout title="このページの読み方">
  「商品を売ったのに損益計算書に何も載らない」──そんな事態を防ぐのが仕訳です。仕訳は取引を「借方」と「貸方」に同額で分ける作業ですが、まず必要なのは「何が増えて何が減ったか」を 5 分類で言えることです。このページでは、5 分類のルールを覚え、典型的な取引の仕訳を数値つきで練習します。
</Callout>

## 学習のポイント

- `資産` と `費用` は借方で増える
- `負債`、`純資産`、`収益` は貸方で増える
- 形ではなく、`何がどう動いたか` を先に言えるようにする

## 試験で何が問われるか

- 勘定科目を 5 分類へ正しく置けるか
- 取引を借方と貸方へ **同額** で分けられるか
- 典型仕訳を **金額つき** で正確に処理できるか

---

## 5 分類のルール

| 分類 | 性質 | 増加する側 | 代表的な勘定科目 |
| --- | --- | --- | --- |
| 資産 | 会社が持つ財産 | **借方** | 現金、売掛金、商品、建物、土地 |
| 負債 | 返す義務のあるお金 | **貸方** | 買掛金、借入金、未払金、社債 |
| 純資産 | 資産 − 負債の差額 | **貸方** | 資本金、利益剰余金 |
| 収益 | 利益を増やす項目 | **貸方** | 売上、受取利息、固定資産売却益 |
| 費用 | 利益を減らす項目 | **借方** | 仕入、給料、減価償却費、支払利息 |

覚え方: B/S の左側にある `資産` は借方で増え、右側にある `負債` と `純資産` は貸方で増えます。P/L の `費用` は借方（B/S 左側と同じ）、`収益` は貸方（B/S 右側と同じ）です。

---

## 仕訳の手順

仕訳は 3 ステップで処理します。

1. **取引を言葉にする**: 何が増えて、何が減ったか
2. **5 分類に当てはめる**: 増えた・減ったものはどの分類か
3. **借方・貸方を決める**: 資産・費用の増加は借方、負債・純資産・収益の増加は貸方

---

## 典型仕訳と金額例

### 商品を現金で販売（売上取引）

```
取引: 商品を 50,000円で現金販売した
  → 現金（資産）が増える → 借方
  → 売上（収益）が増える → 貸方

  （借方）現金    50,000 ／ （貸方）売上    50,000
```

### 商品を掛けで仕入（仕入取引）

```
取引: 商品 30,000円を掛けで仕入れた
  → 仕入（費用）が増える → 借方
  → 買掛金（負債）が増える → 貸方

  （借方）仕入    30,000 ／ （貸方）買掛金  30,000
```

### 備品を現金で購入（資産同士の振替）

```
取引: 備品 80,000円を現金で購入した
  → 備品（資産）が増える → 借方
  → 現金（資産）が減る → 貸方

  （借方）備品    80,000 ／ （貸方）現金    80,000
```

### 銀行から借入（資産と負債が同時に増える）

```
取引: 銀行から 200,000円を借り入れ、普通預金に入金された
  → 普通預金（資産）が増える → 借方
  → 借入金（負債）が増える → 貸方

  （借方）普通預金 200,000 ／ （貸方）借入金 200,000
```

### 買掛金の支払い

```
取引: 買掛金 30,000円を現金で支払った
  → 買掛金（負債）が減る → 借方
  → 現金（資産）が減る → 貸方

  （借方）買掛金  30,000 ／ （貸方）現金    30,000
```

### 給料の支払い

```
取引: 従業員に給料 150,000円を現金で支払った
  → 給料（費用）が増える → 借方
  → 現金（資産）が減る → 貸方

  （借方）給料   150,000 ／ （貸方）現金   150,000
```

---

## 典型的なつまずき

- `仕入` を資産のように扱ってしまう → 仕入は **費用** であり借方で増加
- `借方が増える / 貸方が増える` を丸暗記して、勘定科目の分類と結び付かない → 5 分類に必ず戻る
- `備品購入` のような資産同士の振替で、収益や費用を誤って動かす → 資産 ↑ と資産 ↓ の組み合わせもある
- 貸借が同額にならない → 必ず借方合計 = 貸方合計を確認
- `減少` の処理で反対側に記入することを忘れる → 資産の減少は **貸方** に記入

## 問題を解くときの観点

- まず `何が増え、何が減るか`
- 次に `その勘定科目は5分類のどこか`
- 最後に `借方で増えるか、貸方で増えるか`
- 金額が `借方合計 = 貸方合計` になっているか

## 確認問題

<Callout title="問1：仕訳の作成">
    商品 120,000円を掛けで販売した。仕訳を示せ。

    **解答**：（借方）売掛金 120,000 ／ （貸方）売上 120,000。売掛金（資産）の増加は借方、売上（収益）の増加は貸方。
</Callout>

<Callout title="問2：複合仕訳">
    商品 50,000円を仕入れ、代金のうち 20,000円は現金で支払い、残りは掛けとした。仕訳を示せ。

    **解答**：（借方）仕入 50,000 ／ （貸方）現金 20,000、買掛金 30,000。仕入（費用）の増加は借方、現金（資産）の減少は貸方、買掛金（負債）の増加は貸方。
</Callout>

<Callout title="問3：5分類の判定">
    次の勘定科目を5分類せよ。(a) 未払金 (b) 土地 (c) 受取利息 (d) 減価償却費 (e) 資本金

    **解答**：(a) 負債 (b) 資産 (c) 収益 (d) 費用 (e) 純資産。
</Callout>
