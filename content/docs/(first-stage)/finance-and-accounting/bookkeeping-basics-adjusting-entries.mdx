## このページの役割

<Callout title="このページの役割">
    このページは、`決算整理仕訳` を独立に整理する解説ページです。日常仕訳と決算整理を分け、`時間のズレ` と `見積り` をどう直すかに絞って扱います。
</Callout>

## 学習のポイント

- 最初に `日常取引` と `決算整理` を分ける
- 経過勘定は `時間の前後関係` で判断する
- 減価償却、引当金、棚卸資産の評価は `期末で見積り直す処理` として覚える

## 試験で何が問われるか

- `前払費用`、`未払費用`、`前受収益`、`未収収益` を時間軸で区別できるか
- `減価償却費`、`貸倒引当金`、`商品評価損` を期末処理として説明できるか
- 決算整理後に B/S と P/L のどちらへ出るかを判断できるか

## 決算整理の型

| 型 | 何を直すか | 見るべきこと |
| --- | --- | --- |
| 経過勘定 | 期間対応のズレ | `もう使ったか / まだ使っていないか` |
| 減価償却 | 固定資産の価値配分 | 取得原価を期間へ配分する |
| 引当金 | 将来発生見込みの費用 | 見積り額を当期へ対応させる |
| 棚卸整理 | 商品の実地残高と評価 | 期末棚卸高と低価法の影響を見る |

## 典型例

- `前払保険料`: まだ使っていない部分を資産へ残す
- `未払家賃`: 既に使ったが未払いの部分を費用と負債へ置く
- `減価償却`: 当期分の費用を P/L に、累計額を B/S 側で控除する
- `貸倒引当金`: 将来の貸倒見込みを当期費用へ配分する

## 典型的なつまずき

- 経過勘定を名称で丸暗記して、時間の前後で判断できない
- `減価償却費` と `減価償却累計額` の位置を混同する
- `引当金` をそのまま費用だと思い込み、B/S 側の表示を崩す

## 問題を解くときの観点

- この処理は `時間対応` か `見積り` か
- 期末に直すのは `費用 / 収益` か、`資産 / 負債` か
- 決算整理後に B/S と P/L のどちらへ残るか
