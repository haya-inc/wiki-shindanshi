## このページの役割

<Callout title="このページの役割">
    このページは、`決算整理仕訳` を独立に整理する解説ページです。日常仕訳と決算整理を分け、`時間のズレ` と `見積り` をどう直すかに絞って扱います。
</Callout>

## 学習のポイント

- 最初に `日常取引` と `決算整理` を分ける
- 経過勘定は `時間の前後関係` で判断する
- 減価償却、引当金、棚卸資産の評価は `期末で見積り直す処理` として覚える

## 試験で何が問われるか

- `前払費用`、`未払費用`、`前受収益`、`未収収益` を時間軸で区別できるか
- `減価償却費`、`貸倒引当金`、`商品評価損` を期末処理として説明できるか
- 決算整理後に B/S と P/L のどちらへ出るかを判断できるか

## 決算整理の型

| 型 | 何を直すか | 見るべきこと |
| --- | --- | --- |
| 経過勘定 | 期間対応のズレ | `もう使ったか / まだ使っていないか` |
| 減価償却 | 固定資産の価値配分 | 取得原価を期間へ配分する |
| 引当金 | 将来発生見込みの費用 | 見積り額を当期へ対応させる |
| 棚卸整理 | 商品の実地残高と評価 | 期末棚卸高と低価法の影響を見る |

## 決算整理仕訳の主要パターン一覧

| # | 項目 | 借方 | 貸方 |
| --- | --- | --- | --- |
| ① | 売上原価の算定 | 仕入 → 繰越商品 | 繰越商品 → 仕入 |
| ② | 棚卸減耗損・商品評価損 | 棚卸減耗損・商品評価損 | 繰越商品 |
| ③ | 減価償却 | 減価償却費 | 減価償却累計額 |
| ④ | 貸倒引当金の設定 | 貸倒引当金繰入 | 貸倒引当金 |
| ⑤ | 経過勘定の計上 | （項目による） | （項目による） |
| ⑥ | 有価証券の評価替え | 有価証券評価損 or 有価証券 | 有価証券 or 有価証券評価益 |

## 経過勘定項目の公式

| 経過勘定 | 内容 | B/S 分類 | 仕訳 |
| --- | --- | --- | --- |
| 前払費用 | 支払済みだが次期以降の費用 | **資産** | (借)前払○○ /(貸)○○費 |
| 未払費用 | 未支払だが当期分の費用 | **負債** | (借)○○費 /(貸)未払○○ |
| 前受収益 | 受取済みだが次期以降の収益 | **負債** | (借)受取○○ /(貸)前受○○ |
| 未収収益 | 未受取だが当期分の収益 | **資産** | (借)未収○○ /(貸)受取○○ |

覚え方：「前払・未収」→ 資産（将来の権利）。「前受・未払」→ 負債（将来の義務）。お金が動いている → 「前」。お金が動いていない → 「未」。

### 売上原価の算定（しーくりくりしー）

```
(借) 仕入     ×× / (貸) 繰越商品 ××  ← 期首在庫を仕入に加算
(借) 繰越商品 ×× / (貸) 仕入     ××  ← 期末在庫を仕入から控除

売上原価 = 期首商品棚卸高 + 当期仕入高 − 期末商品棚卸高
```

### 棚卸減耗損と商品評価損

```
棚卸減耗損 = @原価 × (帳簿棚卸数量 − 実地棚卸数量)  ← 数量差
商品評価損 = (@原価 − @正味売却価額) × 実地棚卸数量  ← 単価差
```

## 計算手順

### 経過勘定の計算例

```
【条件】12月決算。10月1日に1年分の保険料 12万円を支払い済み。

手順1: 当期帰属分を月割りで求める
  当期分 = 10月〜12月 = 3か月 → 12万 × 3/12 = 3万円（費用）
  次期分 = 1月〜9月 = 9か月 → 12万 × 9/12 = 9万円（資産）

手順2: 仕訳
  (借) 前払保険料 90,000 / (貸) 保険料 90,000
  → P/L には 3万円だけ費用計上、B/S に 9万円を資産計上
```

### 棚卸減耗損・商品評価損の計算例

```
@原価 100円、帳簿数量 500個、実地数量 450個、@正味売却価額 80円

棚卸減耗損 = 100 × (500 − 450) = 5,000円
商品評価損 = (100 − 80) × 450 = 9,000円
期末商品(B/S) = 80 × 450 = 36,000円

仕訳:
(借) 棚卸減耗損 5,000 / (貸) 繰越商品 5,000
(借) 商品評価損 9,000 / (貸) 繰越商品 9,000
→ 売上原価に算入する場合はさらに仕入へ振替
```

### 貸倒引当金の計算例

```
売掛金残高 1,000万円、前期末の貸倒引当金残高 15万円
設定率 2%（差額補充法）

当期末の必要額 = 1,000 × 2% = 20万円
繰入額 = 20 − 15 = 5万円

(借) 貸倒引当金繰入 50,000 / (貸) 貸倒引当金 50,000
→ P/L の販管費に繰入額、B/S で売掛金から控除表示
```

## 典型的なつまずき

- 経過勘定を名称で丸暗記して、時間の前後で判断できない → 「当期に使った/稼いだか？」で判断する
- `減価償却費`（P/L 費用）と `減価償却累計額`（B/S 控除）を混同する → 費用と累計額は別勘定
- `引当金` をそのまま費用だと思い込む → 引当金は B/S（負債または資産の控除）、繰入額が P/L 費用
- 棚卸減耗損と商品評価損を区別できない → 減耗損は**数量差**、評価損は**単価差**
- 経過勘定の月割計算で決算月を含めるか迷う → 「○月1日から」なら○月を含む

## 確認問題

<Callout title="問1：経過勘定の判定と仕訳">
    3月決算の会社。11月1日に1年分の家賃 24万円を受け取った。決算整理仕訳を示せ。

    **解答**：当期帰属分 = 11月〜3月 = 5か月 → 24万×5/12 = 10万円（収益）。次期分 = 4月〜10月 = 7か月 → 24万×7/12 = 14万円。仕訳：(借)受取家賃 140,000 /(貸)前受家賃 140,000。前受家賃は負債。
</Callout>

<Callout title="問2：棚卸減耗損と商品評価損">
    @原価 200円、帳簿棚卸数量 300個、実地棚卸数量 280個、@正味売却価額 180円。棚卸減耗損・商品評価損・期末商品 B/S 計上額を求めよ。

    **解答**：棚卸減耗損 = 200×(300−280) = 4,000円。商品評価損 = (200−180)×280 = 5,600円。期末商品 = 180×280 = 50,400円。
</Callout>

<Callout title="問3：貸倒引当金（差額補充法）">
    期末売掛金 800万円、受取手形 200万円。設定率 3%。貸倒引当金の前期末残高 20万円。繰入額を求めよ。

    **解答**：対象 = 800+200 = 1,000万。必要額 = 1,000×3% = 30万。繰入額 = 30−20 = 10万円。
</Callout>

## 問題を解くときの観点

- この処理は `時間対応` か `見積り` か
- 期末に直すのは `費用 / 収益` か、`資産 / 負債` か
- 決算整理後に B/S と P/L のどちらへ残るか
