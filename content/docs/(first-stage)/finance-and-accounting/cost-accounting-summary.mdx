## このページの役割

このページは、`原価計算の直前確認` に使う要約ページです。細かな説明ではなく、`どの原価計算か` を切る基準、`何をどこへ配るか`、`利益差がなぜ出るか` を短く見直します。

## 学習のポイント

- `分類軸` と `計算手順` を混ぜずに見る
- 解けない論点があれば、要約で止まらず元ページへ戻る
- `在庫増減` と `固定製造間接費` の関係は必ず言葉で確認する

## 最低限の確認リスト

- `直接費 / 間接費` は `製品との関係`
- `変動費 / 固定費` は `操業度との関係`
- `製造指図書` が出たら個別原価計算を疑う
- `完成品換算量` が出たら総合原価計算を疑う
- 差異分析は `価格` と `数量`、`賃率` と `能率` を分ける
- 在庫が増えると、全部原価計算の利益が高くなりやすい
- 埋没原価は意思決定から外す

## 比較軸の要約

| 論点 | 比較軸 |
| --- | --- |
| 直接費と間接費 | 製品へ直接集計できるか |
| 変動費と固定費 | 生産量で増減するか |
| 個別原価計算と総合原価計算 | 製造指図書単位か、完成品換算量か |
| 標準原価計算の差異 | 価格のずれか、数量のずれか |
| 直接原価計算と全部原価計算 | 固定製造間接費を製品原価へ入れるか |

## 典型的なつまずき

- 分類軸を混同したまま問題を解く
- 予定配賦と実際発生額の違いを追わない
- 利益差の方向だけを暗記し、理由を説明できない
