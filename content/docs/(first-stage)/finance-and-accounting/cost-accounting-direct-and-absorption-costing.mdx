## このページの役割

このページは、`直接原価計算` と `全部原価計算` の違いを固める知識ノードです。`固定製造間接費` をどこで処理するかと、`在庫増減で利益差が出る理由` を説明できるようにします。

<Callout title="このページの読み方">
  工場で 1,000 個作って 800 個売ったとき、売れ残った 200 個の中には工場の家賃が含まれるでしょうか？ この問いへの答えが「含める（全部原価計算）」と「含めない（直接原価計算）」で分かれ、営業利益の金額まで変わります。このページでは、まずその仕組みを理解し、次に利益差の計算方法を身につけ、最後に意思決定に使う特殊原価の考え方までつなげます。
</Callout>

## 学習のポイント

- まず `固定製造間接費` を製品原価へ入れるかどうかを見る
- 次に `在庫が増えるか減るか` で利益差の向きを判断する
- そのうえで `差額原価`、`埋没原価`、`機会原価` を意思決定論点として整理する

## 試験で何が問われるか

- 2 方式の P/L を書けるか
- 在庫変動がある場合の利益差を計算できるか
- 特殊原価概念を使って意思決定問題を解けるか

---

## 2 つの方式は何が違うのか

両者の違いは一点、**固定製造間接費を製品原価に入れるか、期間費用にするか** です。

全部原価計算は、すべての製造コスト（変動費＋固定費）を製品原価に含めます。制度上の正式な方法で、外部への財務報告に使います。一方、直接原価計算は変動費だけを製品原価とし、固定製造間接費は発生した期に全額費用処理します。CVP 分析（損益分岐点分析）と整合性が高いため、内部の経営管理に使います。

| 比較軸 | 直接原価計算 | 全部原価計算 |
| --- | --- | --- |
| 製品原価に含める範囲 | **変動費のみ** | 変動費＋固定費 |
| 固定製造間接費 | 期間原価として全額費用処理 | 製品原価に配賦する |
| 外部報告 | ×（内部管理用） | ○（制度上の正式な方法） |
| CVP分析との整合性 | **高い** | 低い |
| 在庫変動の利益影響 | なし | あり |

---

## P/L のかたちが違う

2 つの方式では損益計算書の構造が異なります。

### 直接原価計算の P/L ── 「貢献利益」を経由する

直接原価計算では、まず売上から変動費だけを引いて `貢献利益`（限界利益）を出し、そこから固定費をまとめて引きます。「あと何個売れば固定費を回収できるか」が見えやすい形です。

```
売上高                        ×××
 − 変動売上原価                ×××
──────────────────
変動製造マージン               ×××
 − 変動販売費                  ×××
──────────────────
貢献利益（限界利益）           ×××
 − 固定費
    固定製造原価    ×××
    固定販管費      ×××        ×××
──────────────────
営業利益                       ×××
```

### 全部原価計算の P/L ── 通常の「売上総利益」型

全部原価計算は一般的な財務諸表と同じ形で、変動・固定を区別しません。

```
売上高                        ×××
 − 売上原価                    ×××
──────────────────
売上総利益                     ×××
 − 販売費及び一般管理費         ×××
──────────────────
営業利益                       ×××
```

---

## なぜ利益差が出るのか ── 在庫に「寝る」固定費

全部原価計算では、固定製造間接費が製品原価に含まれます。売れ残った製品は「在庫」として B/S に残るため、在庫に含まれる固定費は当期の費用にならずに次期へ繰り延べられます。

たとえば 1,000 個生産して 800 個販売し、固定製造間接費の単価が @100 円なら、200 個 × @100 = 20,000 円が期末在庫の中に「寝て」います。全部原価計算ではこの 20,000 円は当期の費用にならないので、その分だけ利益が大きくなります。

一方、直接原価計算では固定製造間接費は在庫に入りません。生産量にかかわらず固定費全額が当期の費用になります。

$$
\text{全部の営業利益} - \text{直接の営業利益} = \text{固定製造間接費単価} \times (\text{期末在庫量} - \text{期首在庫量})
$$

| 状況 | 利益の関係 | 理由 |
| --- | --- | --- |
| 生産量 > 販売量（在庫増加） | 全部 > 直接 | 固定費の一部が期末在庫に繰り延べ |
| 生産量 < 販売量（在庫減少） | 全部 < 直接 | 前期の固定費が期首在庫から費用化 |
| 生産量 = 販売量 | 全部 = 直接 | 在庫変動がないため一致 |

### 固定費調整

直接原価計算の営業利益を全部原価計算の営業利益に合わせる手続きです。

$$
\text{全部の営業利益} = \text{直接の営業利益} + \text{期末在庫の固定製造間接費} - \text{期首在庫の固定製造間接費}
$$

### 計算例 ── 利益差を確かめる

```
【データ】
  直接材料費 @240（変動費）、直接労務費 @160（変動費）
  製造間接費 @200（変動@100 + 固定@100）
  変動販売費 @100、固定販管費 100,000円
  販売価格 @1,000、生産量 1,000個、販売量 800個（期首在庫なし）

直接原価計算:
  売上高 = 1,000 × 800 = 800,000
  変動売上原価 = (240+160+100) × 800 = 400,000
  変動製造マージン = 400,000
  変動販売費 = 100 × 800 = 80,000
  貢献利益 = 320,000
  固定費 = 100×1,000 + 100,000 = 200,000 ← 固定製造間接費は全額当期費用
  営業利益 = 120,000円

全部原価計算:
  売上高 = 800,000
  売上原価 = (240+160+200) × 800 = 480,000
  売上総利益 = 320,000
  販管費 = 80,000 + 100,000 = 180,000
  営業利益 = 140,000円

利益差の検証:
  140,000 − 120,000 = 20,000円
  = 固定製造間接費@100 × 期末在庫200個 = 20,000 ✓
  生産量 > 販売量 → 全部 > 直接 ✓
```

<Callout title="ここまでの整理 ── 利益差のしくみ">
  利益差の原因は「固定製造間接費が在庫に入るかどうか」の一点です。在庫が増えれば全部原価計算の利益が大きくなり、在庫が減れば逆転します。固定販管費は両方式で同額の期間費用なので利益差には影響しません。
</Callout>

## 特殊原価概念 ── 意思決定で「何を数えるか」

ここまでは「期間損益をどう計算するか」の話でしたが、原価計算にはもう一つ大事な役割があります。「自社で作るか外注するか」「この追加注文を受けるか断るか」といった **意思決定** です。意思決定では、過去にいくらかかったかではなく、「これから選ぶ案によって変わる金額」だけを比較します。

| 概念 | 何を意味するか | 意思決定での扱い |
| --- | --- | --- |
| 差額原価 | 案によって変わる原価 | 比較対象に **入れる** |
| 埋没原価（サンクコスト） | すでに発生し、どの案でも変わらない原価 | 意思決定から **除外する** |
| 機会原価 | ある案を選ぶことで諦めなければならない利益 | 選んだ案の **コストに加える** |

### 計算例 ── 自社製造か外注か

```
【データ】自社製造か外注かの選択
  自社製造：変動費 @300、固定費 200,000円（既存設備で追加なし）
  外注：@500/個
  数量 1,000個

判断：
  自社製造の関連原価 = 変動費 300×1,000 = 300,000円
  （固定費 200,000円は自社製造でも外注でも発生する → 埋没原価 → 除外）
  外注の関連原価 = 500×1,000 = 500,000円
  差額 = 500,000 − 300,000 = 200,000円 → 自社製造が有利

注意：固定費が追加発生する場合は埋没原価ではなく、差額原価として考慮する
```

## 問題を解くときの観点

- 問われているのは `利益差` か `意思決定` か
- 在庫が `増えた` のか `減った` のか
- その原価は `今後変わる` のか、すでに発生済みか
- P/L を書くとき、直接原価計算では `貢献利益` を経由する

## 典型的なつまずき

- 直接原価計算を `変動費だけ計算する方法` とだけ覚え、固定費の処理先を説明できない → 固定費は **期間原価** として全額当期費用
- 在庫増減と利益差の向きを逆に覚える → 在庫 **増** なら全部 > 直接
- 埋没原価を意思決定へ入れてしまう → どの案でも変わらない原価は除外する
- 固定製造間接費を「固定費全体」と混同する → 利益差に関係するのは **固定製造間接費** のみ（固定販管費は両方式で同額）
- 固定費調整の加減の向きを逆にする → 期末在庫分を **加算**、期首在庫分を **減算**

## 確認問題

<Callout title="問1：利益差の計算">
    固定製造間接費の単価 @150円。期首在庫 100個、期末在庫 300個。全部原価計算の営業利益と直接原価計算の営業利益の差額を求めよ。

    **解答**：差額 = 150 × (300−100) = 150 × 200 = 30,000円。在庫が増えているので全部原価計算の方が 30,000円大きい。
</Callout>

<Callout title="問2：直接原価計算の P/L 作成">
    売上高 500,000円。変動製造原価 200,000円、変動販売費 50,000円、固定製造原価 100,000円、固定販管費 80,000円。直接原価計算の営業利益を求めよ。

    **解答**：変動製造マージン = 500,000−200,000 = 300,000。貢献利益 = 300,000−50,000 = 250,000。固定費 = 100,000+80,000 = 180,000。営業利益 = 250,000−180,000 = 70,000円。
</Callout>

<Callout title="問3：特殊原価概念">
    ある設備の帳簿価額 200万円。売却すれば 80万円。この設備を使って製品Aを作ると追加利益 150万円が見込める。製品A を作るべきか。なお、帳簿価額 200万円のうち埋没原価はいくらか。

    **解答**：帳簿価額 200万円は過去の支出であり全額が埋没原価。意思決定に関係するのは、製品Aの追加利益 150万円と、売却による 80万円（機会原価）。追加利益 150万−機会原価 80万 = 70万円のプラス。製品Aを作るべき。
</Callout>
