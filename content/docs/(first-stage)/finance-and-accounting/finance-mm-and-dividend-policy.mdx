## このページの役割

このページは、`MM理論` と `配当政策` を整理する解説ページです。`負債を増やすと価値が必ず上がるのか`、`配当を増やすと株価が必ず上がるのか` を、前提条件ごとに切り分けます。

<Callout title="このページの読み方">
  企業がお金を調達する方法は大きく 2 つ──株式（自己資本）と借入・社債（負債）です。「負債と自己資本の比率を変えると企業の価値は変わるのか？」という問いに対して、理論的な出発点を与えたのが MM 理論です。このページでは、まず完全市場という理想世界で MM 理論が何を言っているかを理解し、次に法人税を導入して結論がどう変わるかを見ます。後半では、配当政策が企業価値に影響するかという問いと、株価を理論的に求める配当割引モデルを扱います。
</Callout>

## 学習のポイント

- まず `完全市場かどうか` で結論が変わると押さえる
- 次に `法人税なし` と `法人税あり` の MM理論を分ける
- そのうえで `配当政策そのもの` と `投資政策` を混同しない

## 試験で何が問われるか

- `MM理論` の基本結論を説明できるか
- `負債の節税効果` が企業価値へどう効くか言えるか
- `配当政策の無関連命題` を前提条件つきで説明できるか

## MM理論 ── 資本構成は企業価値に影響するか

### 法人税なし（完全市場）── 出発点の理想世界

MM 理論の出発点は、税金もなく、倒産コストもなく、情報の非対称性もない「完全市場」です。この理想世界では 2 つの命題が成り立ちます。

**命題Ⅰ**: 企業価値は資本構成に依存しない。

$$
V_L = V_U
$$

つまり、負債がある企業（VL）もない企業（VU）も価値は同じです。これは「同じ営業キャッシュフローを生む企業は、お金の調達方法がどうあれ同じ価値を持つ」という考え方です。ピザを何等分に切ろうとピザ全体の大きさは変わらないのと同じ発想です。

**命題Ⅱ**: 負債比率が高まると株主資本コストが上昇し、WACC は一定のまま。

$$
R_e = R_u + (R_u - R_d) \times \frac{D}{E}
$$

`Re` は株主資本コスト、`Ru` は無負債企業の資本コスト、`Rd` は負債コスト、`D/E` は負債・自己資本比率です。負債を増やすと金融リスクが高まるため株主はより高いリターンを要求し、負債の安さを打ち消します。結果として WACC（加重平均資本コスト）は変わりません。

### 法人税あり ── 節税効果が生まれる

現実には法人税があります。支払利息は税務上の費用（損金）になるため、負債を使うと税金が減ります。この `節税効果`（タックスシールド）の分だけ企業価値が増大します。

**命題Ⅰ（税あり）**:

$$
V_L = V_U + \tau \times D
$$

`τ`（タウ）は法人税率、`D` は負債の時価です。節税効果の現在価値が `τ × D` として企業価値に上乗せされます。

**命題Ⅱ（税あり）**:

$$
R_e = R_u + (R_u - R_d) \times (1 - \tau) \times \frac{D}{E}
$$

法人税がない場合と比べて `(1−τ)` が掛かるため、株主資本コストの上昇が **緩やか** になります。節税効果があるぶん、負債を増やすメリットが残るということです。

理論上は 100% 負債が最適ということになりますが、現実にはそうなりません。負債が増えすぎると倒産リスクが高まるからです。

### 計算例

```
【データ】
  無負債企業の価値 VU = 1,000万円
  法人税率 30%、負債 D = 400万円

手順1: 負債あり企業の価値
  VL = 1,000 + 0.3 × 400 = 1,120万円

手順2: 株主資本の価値
  E = 1,120 − 400 = 720万円

→ 節税効果 120万円の分だけ企業価値が増大
```

### 最適資本構成の理論 ── 完全市場を超えて

MM 理論の完全市場の仮定を緩めると、最適な資本構成が存在するかどうかについていくつかの理論が提案されています。

| 理論 | 考え方 |
| --- | --- |
| トレードオフ理論 | 節税効果と倒産コストのバランスで最適資本構成が決まる |
| ペッキングオーダー理論 | 内部留保 → 負債 → 新株発行の順で調達する（情報の非対称性による） |
| エージェンシー理論 | 負債にはフリーCFの浪費を抑制する規律効果がある |

<Callout title="ここまでの整理 ── MM理論のポイント">
  完全市場では資本構成は企業価値に無関係（VL=VU）。法人税を導入すると節税効果で VL=VU+τD となり、負債にメリットが生まれます。ただし現実には倒産コストが存在するため、最適な負債比率がどこかにある（トレードオフ理論）。試験では「法人税なし」と「法人税あり」の結論を混ぜないことが重要です。
</Callout>

## 配当政策 ── 配当を増やすと株価は上がるか

### MM 配当無関連命題

MM 理論は資本構成だけでなく配当についても命題を出しています。完全市場では、配当政策は企業価値に **無関係** です。配当を増やしても、その分だけ内部留保が減って将来の再投資に使えるお金が減るか、新株発行で補わなければならないため、トータルでは変わらないという論理です。

企業価値を決めるのは `投資政策`（どんなプロジェクトに投資するか）であり、投資によって得られるお金をどう分配するか（配当か内部留保か）は価値に影響しません。

### 現実の配当政策

完全市場の仮定を外すと、配当にはさまざまな効果が認められています。

| 理論・政策 | 内容 |
| --- | --- |
| シグナリング効果 | 増配は経営者の自信の表れとして株価にプラス |
| クライアンテル効果 | 配当政策に応じて株主が自然に集まる |
| 残余配当政策 | 投資に必要な資金を確保した残りを配当に回す |
| 安定配当政策 | 業績変動にかかわらず安定的な配当を維持 |
| 自社株買い | 配当の代替手段。発行済株式数が減るため EPS が向上する |

## 株価の理論値 ── 配当割引モデル

配当政策の議論を計算に落とし込むのが配当割引モデル（DDM）です。株式の理論価格は、将来受け取る配当の現在価値の合計です。

### ゼロ成長モデル

配当が永久に一定の場合、株価は単純な割り算です。

$$
P_0 = \frac{D}{R_e}
$$

### 定率成長モデル（ゴードンモデル）

配当が毎年一定の成長率 `g` で増える場合のモデルです。

$$
P_0 = \frac{D_1}{R_e - g}
$$

`D₁ = D₀ × (1+g)` で、条件は `Re > g` です。`Re ≤ g` のときはこのモデルは使えません。

逆に株価から株主資本コストを逆算することもできます。

$$
R_e = \frac{D_1}{P_0} + g
$$

### サステナブル成長率

外部資金調達なしで達成できる最大の成長率です。

$$
g = ROE \times (1 - \text{配当性向}) = ROE \times \text{内部留保率}
$$

利益のうち配当に回さなかった分（内部留保）を再投資して ROE で回すことで、企業は成長します。この成長率をゴードンモデルの `g` に使うことで、理論株価を求めることができます。

### 計算例 ── ゴードンモデル

```
【データ】ROE 10%、配当性向 40%、今期の配当 100円、株主資本コスト 8%

手順1: サステナブル成長率
  g = 10% × (1−0.4) = 10% × 0.6 = 6%
  条件確認: Re(8%) > g(6%) ✓

手順2: 理論株価（ゴードンモデル）
  D₁ = 100 × (1+0.06) = 106円
  P₀ = 106 / (0.08 − 0.06) = 106 / 0.02 = 5,300円
```

### 計算例 ── 2 段階成長モデル

企業の成長率が永久に一定とは限りません。高成長期のあとに安定期に入るケースを扱うのが 2 段階成長モデルです。

```
【データ】D₀ = 100円、高成長期3年(g₁=10%)、安定期(g₂=2%)、Re = 8%

手順1: 高成長期の配当の現在価値
  D₁ = 110 → PV = 110/1.08 = 101.9
  D₂ = 121 → PV = 121/1.08² = 103.7
  D₃ = 133.1 → PV = 133.1/1.08³ = 105.6
  小計 = 311.2

手順2: 安定期の株式価値
  D₄ = 133.1×1.02 = 135.8
  安定期の価値（3年目末時点）= 135.8/(0.08−0.02) = 2,263
  現在価値 = 2,263/1.08³ = 1,796

手順3: 理論株価
  P₀ = 311 + 1,796 ≒ 2,107円
```

## 問題を解くときの観点

- 問題文の前提は `完全市場` か
- `法人税` や `節税効果` を考える設定か
- 問われているのは `資本構成の効果` か `配当政策の効果` か
- `企業価値` と `株主への現金分配` を混同していないか
- DDM で `Re > g` の条件を満たしているか

## 典型的なつまずき

- `負債コストが低い` ので、負債を増やすほど常によいと考える → 倒産コストも考慮する
- `法人税なし` と `法人税あり` の結論を混ぜる → 法人税なし: `VL=VU`、法人税あり: `VL=VU+τD`
- `配当が高い会社は必ず価値が高い` と短絡する → 配当無関連命題の前提を確認
- `配当政策` と `投資政策` の違いを切れない → 価値を作るのは投資機会
- ゴードンモデルで `Re ≤ g` のまま計算してしまう → `Re > g` が成立条件
- MM命題Ⅱの法人税あり版で `(1−τ)` を掛け忘れる → 税ありでは株主資本コストの上昇が緩やか

## 確認問題

<Callout title="問1：MM理論（法人税あり）">
    無負債企業の価値 2,000万円。法人税率 40%。負債 600万円を利用した場合の企業価値と株主資本の価値を求めよ。

    **解答**：VL = 2,000 + 0.4×600 = 2,240万円。E = 2,240−600 = 1,640万円。
</Callout>

<Callout title="問2：サステナブル成長率">
    ROE 12%、配当性向 50%。外部資金調達なしで達成できる最大成長率を求めよ。

    **解答**：g = 12%×(1−0.5) = 12%×0.5 = 6%。
</Callout>

<Callout title="問3：ゴードンの成長モデル">
    今期の配当 80円、成長率 3%、株主資本コスト 7%。理論株価を求めよ。

    **解答**：D₁ = 80×1.03 = 82.4円。P₀ = 82.4/(0.07−0.03) = 82.4/0.04 = 2,060円。
</Callout>
