## このページの役割

このページは、`経営分析` のうち `安全性` を固める知識ノードです。短期の支払い余力と長期の財務基盤を分けて見て、`どこが危ないか` を指標ごとに説明できる状態を目指します。

## 学習のポイント

- まず `流動比率` と `当座比率` の違い
- 次に `自己資本比率` と `固定比率 / 固定長期適合率`
- そのうえで `高い / 低い` の意味を `資金繰り` と結びつける

## 主要指標の比較

| 指標 | 何を見るか | 典型的な読み方 |
| --- | --- | --- |
| 流動比率 | 流動負債に対する流動資産の厚さ | 高いほど短期安全性は高い |
| 当座比率 | すぐ現金化しやすい資産で短期負債を賄えるか | 在庫を除いた安全性を見る |
| 自己資本比率 | 総資本のうち自己資本がどれだけあるか | 高いほど借入依存が低い |
| 固定比率 | 固定資産を自己資本でどこまで賄えているか | 低いほど無理が少ない |
| 固定長期適合率 | 固定資産を自己資本と固定負債で賄えているか | `100%以下` が基本形 |

## 短期安全性と長期安全性を分ける

`流動比率` と `当座比率` は短期の支払い余力を見る指標です。`流動比率` が高くても `当座比率` が低いときは、在庫に資金が滞留している可能性があります。

一方で、`自己資本比率`、`固定比率`、`固定長期適合率` は長期の財務基盤を見る指標です。固定資産を短期資金で賄っていると、設備投資が資金繰りを圧迫しやすくなります。

## 危険信号として見たい型

| 状況 | 疑いやすいこと | 追加で見たい指標 |
| --- | --- | --- |
| 流動比率は高いが当座比率は低い | 在庫が多い | 棚卸資産回転率 |
| 自己資本比率が低い | 借入依存が高い | 固定長期適合率、利息負担 |
| 固定長期適合率が高い | 固定資産を短期資金で賄っている | 流動比率、短期借入金の動き |

## 問題を解くときの観点

- いま見ているのは `短期資金繰り` か `長期の財務基盤` か
- 流動資産の中身は `現預金 / 売掛金 / 在庫` のどれに寄っているか
- 固定資産を賄う資金は `自己資本` と `固定負債` で足りているか

## 典型的なつまずき

- `流動比率` が高ければ十分だと判断して `当座比率` を見ない
- `固定比率` と `固定長期適合率` の分母の違いを混同する
- 借入依存の高さを `自己資本比率` だけで終わらせる
