## このページの役割

このページは、`キャッシュ・フロー管理` を固める知識ノードです。`利益は出ているのに現金が足りない` 状態を説明できるようにし、`営業 CF`、`投資 CF`、`財務 CF`、`FCF`、`運転資本` を一つの流れで整理します。

## 学習のポイント

- まず `利益` と `現金` は同じでないと押さえる
- 次に `営業 / 投資 / 財務` の 3 区分を切る
- そのうえで `間接法` と `FCF` を使えるようにする

## 試験で何が問われるか

- 間接法による営業 CF を計算できるか
- 営業 CF、投資 CF、財務 CF の符号から企業の局面を読めるか
- `FCF` や `運転資本` を企業価値評価や資金繰りへつなげられるか

## 3 区分の役割

| 区分 | 何を見るか | 典型例 |
| --- | --- | --- |
| 営業 CF | 本業でどれだけ現金を生んだか | 税引前利益、減価償却費、売上債権増減 |
| 投資 CF | 将来のためにどれだけ投資したか | 設備投資、資産売却 |
| 財務 CF | 資金調達や返済をどうしたか | 借入、返済、配当 |

## 間接法の基本

間接法では、`税引前利益` から始めて、`非資金項目`、`営業外損益`、`運転資本増減` を調整します。`現金が動いたか` を基準に加算 / 減算を判断すると崩れにくくなります。

## FCF と運転資本

`FCF` は、事業が生んだ現金から設備投資や運転資本負担を差し引いた後に、企業が自由に使える現金です。売上成長に伴って `売上債権` や `棚卸資産` が増えると、利益が出ていても現金が寝て資金繰りが苦しくなることがあります。

## 符号パターンの読み方

`営業 CF プラス`、`投資 CF マイナス`、`財務 CF プラス` は投資先行の成長局面、`営業 CF プラス`、`投資 CF マイナス`、`財務 CF マイナス` は成熟局面と読みやすいです。暗記だけでなく、`本業`、`投資`、`資金調達` のどれが起きているかを言葉で追います。

## 問題を解くときの観点

- 見たいのは `営業 CF` か `FCF` か
- 現金が寝ている原因は `売上債権`、`在庫`、`設備投資` のどこか
- 資金不足を `本業悪化` と `成長投資` のどちらで説明するか

## 典型的なつまずき

- 減価償却費の加算理由を説明できない
- `売上債権増加` と `仕入債務増加` の向きを逆にする
- `営業 CF` と `FCF` を同じ意味で扱う
