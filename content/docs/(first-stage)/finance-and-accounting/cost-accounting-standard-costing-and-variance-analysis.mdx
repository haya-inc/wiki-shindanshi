## このページの役割

このページは、`標準原価計算` と `差異分析` を固める知識ノードです。式を丸暗記するのではなく、`何が標準からずれたのか` を言葉で説明できる状態を目指します。

## 学習のポイント

- まず `標準` と `実際` のどちらを比べているかを切る
- 次に差異を `価格` と `数量` の軸で分ける
- `有利差異 / 不利差異` を式ではなく意味で判定する

## 標準原価計算の考え方

標準原価計算は、あらかじめ `あるべき原価` を設定し、実際原価との差を管理する方法です。原価差異は、原価管理の入口として使われます。

## 差異分析の基本

| 差異 | 何のずれを見るか | 代表的な見方 |
| --- | --- | --- |
| 価格差異 | 単価や賃率のずれ | `標準単価 - 実際単価` |
| 数量差異 | 消費量や作業時間のずれ | `標準数量 - 実際数量` |
| 能率差異 | 労働時間や操業効率のずれ | `標準時間 - 実際時間` |

## 材料費差異と労務費差異

材料費差異では、`単価がずれたのか`、`使った量がずれたのか` を分けます。労務費差異では、`賃率` と `能率` を切り分けるのが基本です。

## 解く順

1. 標準数量か標準時間を先に出す
2. 実際原価と標準原価の総差異を確認する
3. `価格` と `数量`、`賃率` と `能率` に分解する
4. 実際が多いのか少ないのかを言葉で確認して、有利差異か不利差異かを決める

## 問題を解くときの観点

- 単価の問題か、使用量の問題か
- `実際が多い` ときに不利になりやすいのは何か
- 総差異と内訳差異がつながっているか

## 典型的なつまずき

- 差異の符号だけ見て有利差異 / 不利差異を判定する
- `価格差異` と `数量差異` の基準数量を取り違える
- 標準原価そのものを先に計算せず、分解式だけ当てはめる
