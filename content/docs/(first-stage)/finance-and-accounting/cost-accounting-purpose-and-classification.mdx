## このページの役割

このページは、`原価計算の入口` を固める知識ノードです。`何のために原価を計算するか`、`どの分類軸で原価を分けるか` を整理し、後続の `個別原価計算`、`総合原価計算`、`標準原価計算` を読む前提をそろえます。

## 学習のポイント

- まず `原価計算の目的` を `財務報告`、`原価管理`、`意思決定` に分ける
- 次に `材料費 / 労務費 / 経費` と `直接費 / 間接費` を分ける
- そのうえで `変動費 / 固定費` を `操業度との関係` で切る

## 原価計算の目的

原価計算は、単に製品原価を出すだけの作業ではありません。診断士試験で押さえるべき目的は、少なくとも次の 3 つです。

| 目的 | 何を見るか | 代表的な使いどころ |
| --- | --- | --- |
| 財務報告 | 製品原価、売上原価、棚卸資産評価 | 全部原価計算、月次決算 |
| 原価管理 | 標準と実際の差、非効率の発見 | 標準原価計算、差異分析 |
| 意思決定 | どの費用を意思決定に入れるか | 直接原価計算、差額原価 |

## まず固める分類軸

### 1. 要素別分類

最も基本なのが `材料費`、`労務費`、`経費` の 3 つです。ここでは `何にお金がかかったか` を見ています。

### 2. 製品との関係による分類

`直接費 / 間接費` は、`どの製品へ直接ひも付くか` の軸です。材料費でも直接材料費と間接材料費があり、費目と分類軸を混ぜないことが重要です。

### 3. 操業度との関係による分類

`変動費 / 固定費` は、`操業度が変わると増減するか` の軸です。`製品との関係` を見ているのではなく、`生産量との関係` を見ています。

## 分類軸を混同しないための比較表

| 比較 | 何を見ているか | 代表例 |
| --- | --- | --- |
| 直接費 / 間接費 | 製品へ直接集計できるか | 直接材料費、製造間接費 |
| 変動費 / 固定費 | 生産量で増減するか | 材料費、減価償却費 |
| 製造原価 / 販売費及び一般管理費 | 製造に入るか、期間費用か | 製造間接費、広告宣伝費 |

## 問題を解くときの観点

- いま見ている分類軸は `製品との関係` か `操業度との関係` か
- その原価は `製造原価` か `期間費用` か
- 原価計算の目的は `財務報告`、`原価管理`、`意思決定` のどれか

## 典型的なつまずき

- `直接費` と `変動費` を同じ意味で覚える
- `材料費`、`労務費`、`経費` と `直接費 / 間接費` を同じ表で処理して混乱する
- `製造原価` と `販売費及び一般管理費` を同じ原価概念として扱う
