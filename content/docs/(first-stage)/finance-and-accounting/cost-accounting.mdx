## このページの役割

原価計算は、`製品やサービスにどれだけ原価がかかったか`、`その原価をどう管理や意思決定に使うか` を整理する章ハブです。診断士試験では、`原価の目的と分類`、`個別原価計算と総合原価計算`、`標準原価計算と差異分析`、`直接原価計算と全部原価計算` がまとまりとして問われます。

## 学習のポイント

- まず `原価計算の目的` と `分類軸`
- 次に `個別原価計算` と `総合原価計算`
- そのうえで `標準原価計算` と `差異分析`
- 最後に `直接原価計算` と `全部原価計算` を `利益差` と `意思決定` までつなげる

## 最優先で固める論点

### 原価計算の目的と分類

入口で迷わないように、まず `原価を何のために計算するか` と `どの分類軸で切るか` を固めます。ここが曖昧だと、`直接費 / 間接費` と `変動費 / 固定費` を混同しやすくなります。

### 個別原価計算と総合原価計算

試験では `受注生産` か `大量生産` かを切るのが最初です。`製造指図書ごとに集計する` のか、`完成品と月末仕掛品へ配分する` のかで、計算の見方が変わります。

### 標準原価計算と差異分析

`価格差異`、`数量差異`、`賃率差異`、`能率差異` を、式ではなく `何が標準からずれたか` で説明できる状態にします。原価差異の方向を言葉で判定できるかが重要です。

### 直接原価計算と全部原価計算

`固定製造間接費` を製品原価へ入れるか、期間費用にするかで利益差が出ます。CVP や意思決定につなげる前提として、在庫増減と利益差の関係を先に見ます。

## 典型的なつまずき

- 直接費と変動費を同じ意味だと思ってしまう
- 個別原価計算と総合原価計算の前提を混同する
- 差異分析の式だけ覚えて、有利差異 / 不利差異の意味を取り違える
- 直接原価計算と全部原価計算の利益差の理由が説明できない

## 問題を解くときの観点

- いま求めたいのは `製品別の原価` か、`期間の利益差` か
- 前提は `受注生産` か、`大量生産` か
- 実際原価を見ているのか、標準との差異を見ているのか
- 固定費をどう扱うか

## 近年の出題の寄り方

| 区分 | 出やすい型 | 特に重く見たい論点 |
| --- | --- | --- |
| `R3〜R5` で繰り返し出る型 | 個別原価計算、総合原価計算、全部原価計算と直接原価計算の利益差 | 配賦の流れ、平均法、固定費調整の意味 |
| `R6〜R7` で比重が高い型 | 勘定連絡、材料消費価格差異、仕損・減損を含む総合原価計算 | 手順の正確さ、先入先出法や度外視法の使い分け |

## よくある誤答パターン

| 誤答パターン | 起こりやすい原因 | 解く前の確認 |
| --- | --- | --- |
| 直接費と変動費を同じものとして処理する | 分類軸を混同している | `製品との関係` と `操業度との関係` を分ける |
| 個別原価計算なのに総合原価計算の発想で解く | 生産形態の前提を見ていない | `受注生産` か `大量生産` かを先に確認する |
| 差異分析で有利差異 / 不利差異の向きを逆にする | 式だけで判定している | `実際が多いのか、標準が多いのか` を言葉で確認する |
| 全部原価計算と直接原価計算の利益差を説明できない | 固定製造間接費の扱いを追えていない | 在庫が増減したときの固定費の残り方を見る |
