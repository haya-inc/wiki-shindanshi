## 簿記基礎

### 第1問　帳簿組織と特殊仕訳帳

**問題要旨**: 特殊仕訳帳導入時の帳簿処理における、重複記録、転記方法などの正誤判定。

`K5 制度・データ` `T1 正誤判定` `L2` `二重記録`

**正解**: （設問による。標準的：a と d が正、b と c が誤）

**必要知識**:
- **普通仕訳帳のみの場合**:
  - 全取引を普通仕訳帳に記録
  - 総勘定元帳に転記（月次まとめて転記可能）
- **特殊仕訳帳導入時**:
  - 売掛金取引 → 売上仕訳帳（取引ごとに記録）
  - 仕入取引 → 仕入仕訳帳（取引ごとに記録）
  - その他取引 → 普通仕訳帳
  - 転記：月次で各仕訳帳から総勘定元帳へ（売掛金元帳への転記も並行）

**解法の思考プロセス**:
1. **特殊仕訳帳の設置目的**:
   - 仕訳と転記の手間を削減
   - 取引が多い項目（売上・仕入）を重点管理
2. **各記述の正誤判定**:
   - a「取引内容に応じて、合計額を一括またはごとに転記」→ 正（柔軟性）
   - b「1つの取引が2つの帳簿に記録」→ 誤（各帳簿には1回のみ）
   - c「普通仕訳帳のみで合計額を一括転記」→ 正（効率化）
   - d「普通仕訳帳のみで個別転記」→ 正（小規模取引向け）

**誤答の落とし穴**:
- **「複数帳簿への同時記録は二重計上」という誤解**:
  - 特殊仕訳帳と普通仕訳帳は適用範囲が異なるので二重計上ではない
  - ただし同一帳簿内での同一取引の2重記録は誤り
- **転記のタイミング**:
  - 「毎日転記」vs「月末まとめて転記」の選択肢を混同
  - 両者とも可能（ただし試算表作成までに完了が条件）

**学習アドバイス**:
- 帳簿の系統図（仕訳帳→元帳・補助簿）を手書きして可視化
- 実例：小規模店舗（普通仕訳帳のみ）vs 大規模商社（特殊仕訳帳導入）の違い
- 現代システム：Excel やクラウド会計ソフトでの自動振り分け

### 第2問　試算表と勘定閉鎖

**問題要旨**: 試算表の作成、または不正確な記帳による試算表の不一致を検出する問題。

`K4 因果メカニズム` `T2 グラフ読解` `L2` `記入漏れ`

**正解**: （設問による。試算表の左右差異から誤差額を特定）

**必要知識**:
- **試算表の種類**:
  - 残高試算表（balance trial balance）：各勘定の期末残高
  - 合計試算表（total trial balance）：各勘定の合計（期首+取引額）
  - 合計残高試算表（total balance trial balance）：左右の合計と残高の両方
- **試算表が成立する条件**:
  - 借方合計 = 貸方合計（複式簿記の基本）
  - 不一致 → 記帳誤り あり

**解法の思考プロセス**:
1. **記帳内容の確認**:
   - 借方・貸方の金額が正確か
   - 摘要欄の記載が正確か
2. **各勘定を集計**:
   - 借方残高（資産・費用）
   - 貸方残高（負債・資本・収益）
3. **合計額の比較**:
   - 左右が一致 → 記帳は正確（ただし計上誤りは検出不可）
   - 左右が不一致 → 記帳誤りあり

**誤答の落とし穴**:
- **試算表の一致 = 完全正確性ではない**:
  - 記帳誤り（摘要の誤字、金額の間違い）でも試算表は一致することがある
  - 記帳漏れ（忘れた取引）は試算表の不一致を招かない（両方忘れれば一致）
- **帳簿的誤り vs 概念的誤り**:
  - 帳簿的誤り：左右不一致（試算表で検出）
  - 概念的誤り：左右一致だが内容誤り（決算で発見）

**学習アドバイス**:
- 試算表の左右不一致 = 誤差額を2で割った値で誤差個所が特定できることが多い
  - 例：不一致額 1,000 → 500円の記帳誤り、または 10倍誤り（100→1000）など
- 試算表は「デバッグツール」：不一致 → 原因特定 → 修正

## 財務会計

### 第3問　資産の分類と評価

**問題要旨**: 流動資産・固定資産の分類、または資産の評価基準（原価評価・時価評価・低価法など）に関する判断問題。

`K2 グラフ形状` `T1 正誤判定` `L2` `-`

**正解**: ウ

**必要知識**:
- **資産分類**:
  - 流動資産：1年以内に現金化可能（現金、売掛金、棚卸資産）
  - 固定資産：1年超で利用（土地、建物、機械、ソフトウェア）
- **資産評価基準**:
  - 原価評価：取得時の支払額で記録（会計の基本）
  - 時価評価：期末の市場価格で評価（有価証券、外貨資産）
  - 低価法（LCM）：原価 vs 市場価額の低い方（在庫資産）

**解法の思考プロセス**:
1. **資産の性質を確認**:
   - 棚卸資産？→ 低価法の適用対象
   - 土地・建物？→ 原価評価（減価償却なし or あり）
   - 有価証券？→ 時価評価の可能性
2. **評価方法の選択**:
   - 原価評価が基本（保守的）
   - 時価評価は限定的（金融資産など）

**誤答の落とし穴**:
- **「資産は常に市場価格で評価すべき」という誤解**:
  - 財務会計では原価主義が基本
  - 時価評価は例外（有価証券、外貨など特定資産のみ）
- **低価法の適用範囲**:
  - 在庫資産（商品・原材料・仕掛品）に限定
  - 固定資産には通常適用しない
- **土地の減価償却**:
  - 土地は減価償却しない（永遠の資産）
  - 建物は減価償却（劣化あり）

**学習アドバイス**:
- 貸借対照表の左側（資産）を「1年基準」で流動・固定に分類する習慣
- 在庫評価方法（先入先出・移動平均・個別法）との関連確認
- 国際会計基準（IFRS）での時価評価の拡大を知識として把握

### 第4問　負債と利益認識

**問題要旨**: 負債の定義と分類、または収益・費用の認識時点（実現主義）に関する判断問題。

`K2 グラフ形状` `T1 正誤判定` `L2` `-`

**正解**: ウ

**必要知識**:
- **負債の定義**:
  - 過去の取引により発生
  - 将来の経済資源流出の可能性
  - 企業支配下にない
- **負債分類**:
  - 流動負債：1年以内に決済（買掛金、短期借入金）
  - 固定負債：1年超で決済（長期借入金、予約金）
- **収益認識**:
  - 実現主義：商品引渡または役務提供時に認識
  - 発生主義：契約時ではなく実現時に認識

**解法の思考プロセス**:
1. **負債の発生要件**:
   - 商品未納 → 買掛金（負債あり）
   - 金銭未貸付 → 未払利息（負債あり）
2. **収益認識時点**:
   - 販売契約時：認識なし（見積）
   - 商品発送時：認識あり（実現）
   - 金銭回収時：確認（ただし既に認識済み）

**誤答の落とし穴**:
- **予約金と売上の混同**:
  - 予約金を受け取った時点では売上に計上しない（負債として計上）
  - 商品納品時に売上へ振り替え
- **引き渡し条件による影響**:
  - FOB送出地：発送時に所有権移転
  - FOB到着地：到着時に所有権移転
  - 売上計上のタイミングが異なる

**学習アドバイス**:
- 「実現の原則」：商品・サービスの引渡がない限り売上認識なし
- 返品予定商品の扱い：一旦売上計上後、返品処理で相殺
- 特殊な収益認識（継続的役務、完工基準 vs 成果基準）の違い

### 第5問　財務諸表分析

**問題要旨**: 複数年度の財務データから、経営状況の変化を読み取り、原因分析を行う問題。

`K3 数式・公式` `T2 グラフ読解` `L3` `時系列混同`

**正解**: イ

**必要知識**:
- **収益性指標**:
  - ROA（総資産利益率）= 当期利益 / 総資産
  - ROE（自己資本利益率）= 当期利益 / 自己資本
  - 営業利益率 = 営業利益 / 売上高
- **効率性指標**:
  - 資産回転率 = 売上高 / 総資産
  - 在庫回転率 = 売上原価 / 平均在庫
- **安全性指標**:
  - 流動比率 = 流動資産 / 流動負債
  - 自己資本比率 = 自己資本 / 総資産

**解法の思考プロセス**:
1. **複数指標を計算**:
   - 利益率の変化 → 価格・原価圧力を検知
   - 回転率の変化 → 効率悪化 vs 品質向上か判断
   - 流動比率の変化 → 資金繰り悪化の可能性
2. **相互関係を読む**:
   - ROA = 営業利益率 × 資産回転率（デュポン分析）
   - 利益率低下だが回転率上昇 → 薄利多売戦略
3. **原因特定**:
   - 産業全体の影響か企業固有か
   - 経営戦略の意図的変更か悪化か

**誤答の落とし穴**:
- **単一指標の偏重**:
  - ROA が低下 → 原因は何か？（利益率 vs 回転率）
  - 両者を分解分析して原因を特定
- **時系列混同**:
  - 前年度比 vs 前年度数値そのものの混同
  - 例：「当期 ROA が 10%」（水準）vs「当期 ROA が前年比 +2%」（変化率）
- **季節性・一時的要因の見落とし**:
  - 決算期が異なれば、季節的な在庫変動が影響

**学習アドバイス**:
- 指標計算を必ず手書き（電卓不可試験）
- デュポン分析図を手描きして、複数指標の関連性を可視化
- 実例：景気後退期（2009）での各企業の対応戦略

## 原価計算

### 第6問　製造原価の計算

**問題要旨**: 原材料費・直接労務費・製造間接費から、製品原価を計算する。仕掛品の期首・期末金額を考慮した計算。

`K4 因果メカニズム` `T4 因果推論` `L2` `-`

**正解**: （設問による。標準的：当期製造原価の計算）

**必要知識**:
- **製造原価の構成**:
  - 直接材料費（raw materials）：特定製品に直接割り当て可能
  - 直接労務費（direct labor）：作業員の賃金
  - 製造間接費（manufacturing overhead）：工場費用（役員給与、減価償却など）
- **当期製造原価の公式**:
  $$\text{当期製造原価} = \text{期首仕掛品} + \text{当期投入費用} - \text{期末仕掛品}$$
- **完成品原価と当期製造原価の違い**:
  - 当期製造原価：当期に投入した費用
  - 完成品原価：当期に完成した製品の原価（期首仕掛品含む）

**解法の思考プロセス**:
1. **費用項目を分類**:
   - 材料費 → 直接材料費（製品直接関連）
   - 工場作業員給与 → 直接労務費
   - 工場経営費 → 製造間接費
   - 営業費（販売・管理）は含まない
2. **公式に当てはめ**:
   - 期首仕掛品 + (材料費 + 労務費 + 間接費) - 期末仕掛品
3. **検証**:
   - 完成品の数量が妥当か
   - 1製品あたりの原価が業界標準と比較して妥当か

**誤答の落とし穴**:
- **販売費・一般管理費の誤含**:
  - 営業費は製造原価に含めない
  - 完成品販売手数料、本社給与は別途処理
- **期首仕掛品と期末仕掛品の足し引き誤り**:
  - 期首：加算（前期から繰越）
  - 期末：減算（当期に完成していない）
- **配分基準の誤解**:
  - 機械的に按分（例：労務時間比）vs 実際個別集計

**学習アドバイス**:
- 「投入」「変換」「完成」の3段階フロー図を描く
- 実習：3カ月の仕掛品計算で、累積効果を体験
- 標準原価制度（第7問）との関連確認

### 第7問　標準原価と差異分析

**問題要旨**: 実際原価と標準原価の差異（材料費差異、労務費差異、製造間接費差異）を計算し、原因を分析する問題。

`K4 因果メカニズム` `T4 因果推論` `L3` `符号誤り`

**正解**: （設問による。複数差異の同時計算と符号確認が必要）

**必要知識**:
- **材料費差異**:
  - 数量差異：標準数量 vs 実際数量（物量の効率性）
  - 価格差異：標準価格 vs 実際価格（仕入価格の有利・不利）
  - 公式：
    - 数量差異 = (実際数量 - 標準数量) × 標準価格
    - 価格差異 = (実際価格 - 標準価格) × 実際数量
- **労務費差異**:
  - 効率差異：実際時間 vs 標準時間（作業効率）
  - 賃率差異：実際賃率 vs 標準賃率（人件費コスト）
- **製造間接費差異**:
  - 支出差異：実際支出 vs 予算額
  - 操業度差異：予算額 vs 標準配賦額（生産量と固定費の関係）

**解法の思考プロセス**:
1. **標準値と実績値を整理**:
   - 標準：事前計画値
   - 実績：実際に発生した数値
2. **各差異を順に計算**:
   - 材料費：数量差異 + 価格差異
   - 労務費：効率差異 + 賃率差異
   - 間接費：支出差異 + 操業度差異
3. **符号の確認**:
   - 有利差異（F）：コスト削減
   - 不利差異（U）：コスト増加
   - 計算結果がプラス or マイナスか慎重に判定

**誤答の落とし穴**:
- **符号誤りが頻発**:
  - 実績 > 標準（数量）→ 不利差異（負）
  - 実績 < 標準（数量）→ 有利差異（正）
  - 符号を逆にする受験生が多い
- **標準と予算の混同**:
  - 標準：単位当たりの目標値（製品1個あたり）
  - 予算：生産計画全体での目標額
  - 製造間接費差異で両者を混同しやすい
- **固定費と変動費の分離不足**:
  - 製造間接費 = 変動費 + 固定費
  - 操業度差異 = 固定費 × (稼働度の低下)

**学習アドバイス**:
- 差異分析表（variance analysis table）を手作成
- 符号ルール：(実績 - 標準) × 単価 の順で統一
- 実例：製造業での原価管理（材料の品質低下が数量増 → 不利差異など）

## 管理会計

### 第8問　吸収原価法と直接原価法

**問題要旨**: 吸収原価法（全部原価法）vs 直接原価法（変動原価法）による利益計算の相違を問う問題。

`K3 数式・公式` `T2 グラフ読解` `L3` `損益分岐点`

**正解**: ウ

**必要知識**:
- **吸収原価法（absorption costing）**:
  - 完成品原価 = 変動費 + 固定費（全部配分）
  - 在庫が増える → 固定費の一部が繰越 → 当期利益が増加
  - 管理会計では評価が低い（変動原価法優先）
  - 但し、財務会計では吸収原価法が標準
- **直接原価法（direct costing）**:
  - 完成品原価 = 変動費のみ
  - 固定費は当期費用化
  - 在庫増減による利益操作なし（より透明）
- **利益差**:
  - 吸収原価利益 = 直接原価利益 + (期末在庫 - 期首在庫) × 単位固定費

**解法の思考プロセス**:
1. **単位固定費を計算**:
   - 固定費 / 標準生産量 = 単位固定費
2. **在庫変化を把握**:
   - 生産量 > 販売量 → 在庫増加 → 吸収原価利益が高い
   - 生産量 < 販売量 → 在庫減少 → 吸収原価利益が低い
   - 生産量 = 販売量 → 両者は同じ
3. **利益差を計算**:
   - (期末在庫数量 - 期首在庫数量) × 単位固定費 = 利益差

**誤答の落とし穴**:
- **「在庫があると利益が増える」は管理会計では悪い話**:
  - 吸収原価法の欠点：在庫を増やすと見かけの利益が増える
  - 直接原価法でコントロール：実際の販売成果を反映
- **固定費の配分基準の誤解**:
  - 標準生産量で配分（実績生産量ではない）
  - 操業度差異が生じる
- **在庫評価方法（先入先出・移動平均）の影響**:
  - 個別商品の流れを正確に追跡する必要

**学習アドバイス**:
- 吸収原価法と直接原価法の利益格差を、在庫数量と単位固定費で説明
- CVP分析で直接原価法を前提とすることを確認
- 現代マネジメント：「利益と現金の乖離」を認識することの重要性

### 第9問　CVP分析（本質原価法）

**問題要旨**: 損益分岐点の計算、または利益目標達成に必要な売上高を計算する問題。

`K3 数式・公式` `T4 因果推論` `L3` `数量 vs 金額`

**正解**: ウ

**必要知識**:
- **貢献利益（contribution margin）**:
  - CM = 売上高 - 変動費
  - 貢献利益率 = CM / 売上高 = (P - V) / P
  - 1製品あたりの貢献利益 = P - V（単位貢献利益）
- **損益分岐点（BEP）**:
  - 利益 = 0 となる売上高
  - BEP（金額） = 固定費 / 貢献利益率
  - BEP（数量） = 固定費 / 単位貢献利益
- **利益目標達成の売上高**:
  - 必要売上高 = (固定費 + 目標利益) / 貢献利益率

**解法の思考プロセス**:
1. **費用を固定費と変動費に分離**:
   - 固定費：生産量に関わらず発生（給与、減価償却）
   - 変動費：売上に応じて増減（材料費、販売手数料）
2. **単価と変動費率を確認**:
   - 単価 = 売上 / 販売数量
   - 変動費率 = 変動費 / 売上高
3. **BEPを計算**:
   - 固定費 ÷ (1 - 変動費率) = BEP売上高
   - または：固定費 ÷ (単価 - 変動費単価) = BEP数量
4. **検証**:
   - BEP時点で利益 = 0 か確認

**誤答の落とし穴**:
- **数量 vs 金額の混同**:
  - BEP数量：個数
  - BEP売上高：円・ドル
  - 設問を慎重に読み、何を求めているか確認
- **限界利益（貢献利益）率の計算誤り**:
  - 分子分母を逆にする（費用／売上高ではなく、利益／売上高）
- **固定費と変動費の分類誤り**:
  - 販売手数料が変動費なのか固定費なのか不明確な場合がある
  - 問題文から慎重に判定

**学習アドバイス**:
- CVP図（Cost-Volume-Profit Chart）を手描きして、BEPを視覚化
- 安全余裕度 = (実績売上高 - BEP) / 実績売上高 × 100（%）
- 製品ミックスが複数ある場合のBEP（加重平均貢献利益率）

### 第10問　キャッシュフロー分析

**問題要旨**: 営業キャッシュフロー（OCF）、投資キャッシュフロー（ICF）、財務キャッシュフロー（FCF）から、企業の資金繰り状況を分析する問題。

`K3 数式・公式` `T2 グラフ読解` `L3` `現金化タイミング`

**正解**: ウ

**必要知識**:
- **キャッシュフロー計算書の構造**:
  - 営業CF = 利益 + 減価償却 - 売上債権増加 - 在庫増加 + 買掛金増加
  - 投資CF = -設備投資 - 投資有価証券取得 + 売却益
  - 財務CF = 借入金増加 - 借入金返済 + 配当金支払い
- **フリーキャッシュフロー（FCF）**:
  - FCF = 営業CF - 資本支出
  - 企業が実際に自由に使える現金
- **各CFの意味**:
  - 営業CF > 0：本業で現金が入ってくる（健全）
  - 投資CF < 0：設備投資に支出（成長投資）
  - 財務CF：借入・配当による調整

**解法の思考プロセス**:
1. **各CFの計算**:
   - 営業：利益から現金化のズレを調整
   - 投資：固定資産の増減を反映
   - 財務：資本構成の変化を反映
2. **パターン分析**:
   - 営業CF > 0 + 投資CF < 0 + 財務CF > 0：成長期（借入で投資）
   - 営業CF > 0 + 投資CF < 0 + 財務CF < 0：成熟期（内部資金で投資）
   - 営業CF < 0 + 投資CF < 0 + 財務CF > 0：苦しい時期（借入で持つ）
3. **キャッシュ創出力を判断**:
   - FCF = 営業CF - 資本支出 > 0 か確認

**誤答の落とし穴**:
- **現金化タイミングの見落とし**:
  - 売上高が増加しても、売掛金増加で現金入金は遅延
  - 在庫増加で現金流出
- **減価償却の扱い**:
  - 減価償却は非現金費用（現金が出ていない）
  - CF計算で利益に加算して調整
- **営業CFと利益の乖離原因**:
  - 企業の健全性を判断する上で重要
  - 粉飾会計検出の手がかり（利益大だが営業CF小など）

**学習アドバイス**:
- 損益計算書（利益）vs キャッシュフロー計算書（現金）の違いを整理
- 利益と現金の乖離例：赤字でも現金黒字（売却益など）
- 企業評価では CF ベースの分析が重要

### 第11問　投資意思決定

**問題要旨**: 複数の投資案を評価し、正味現在価値（NPV）、内部収益率（IRR）、回収期間などの指標で比較する問題。

`K3 数式・公式` `T4 因果推論` `L3` `NPV vs IRR`

**正解**: ウ

**必要知識**:
- **NPV（正味現在価値）**:
  - $NPV = \sum \frac{CF_t}{(1+r)^t} - I_0$
  - r = 割引率（資本コスト）
  - NPV > 0 なら投資を実施
  - 経営学的には最も理論的
- **IRR（内部収益率）**:
  - NPV = 0 となる割引率
  - IRR > 資本コストなら投資を実施
  - 計算が煩雑（試行錯誤が必要）
- **回収期間（payback period）**:
  - 初期投資を回収するのに必要な年数
  - 流動性重視（リスク回避的）
  - 理論的には劣るが現実的判断で有効
- **プロファイタビリティ・インデックス（PI）**:
  - PI = PV（キャッシュフロー）/ 初期投資
  - PI > 1 なら投資を実施
  - 投資効率性の指標

**解法の思考プロセス**:
1. **各案の NPV を計算**:
   - 割引率は問題で与えられる（例：5%）
   - 各年度の CF を割引現在価値に変換
   - 合計から初期投資を引く
2. **複数案がある場合**:
   - NPV が大きい案を選択
   - ただし、相互排他的な案と独立案で判定が異なる
3. **補助指標を確認**:
   - IRR で再検証（理論的確認）
   - 回収期間で流動性確認（リスク管理）

**誤答の落とし穴**:
- **相互排他的 vs 独立案の誤解**:
  - 相互排他的：1案のみ選択可（例：設備A か設備B か）→ NPV 最大の案を選ぶ
  - 独立案：複数案同時実施可能 → NPV > 0 の案を全て採択
- **IRR の落とし穴**（NPV との乖離）**:
  - 規模の異なる投資案で IRR が高くても、NPV は低い場合あり
  - 例：小規模案の IRR 30% vs 大規模案の IRR 15% で、NPV は大規模案が大きい
  - → NPV 優先が理論的に正しい
- **割引率の変更による NPV の感応度**:
  - 割引率が高まると NPV は低下
  - 遠い将来の CF は割引率に敏感

**学習アドバイス**:
- 計算機の使用（試験では電卓使用不可）：手計算での NPV 導出は避ける傾向
- 複数案の意思決定フロー図：初期投資規模、回収期間、リスク等を総合判定
- ファイナンス理論との連結：資本コスト（WACC）の理解
