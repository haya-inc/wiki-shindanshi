## 簿記基礎・仕訳

### 第1問　基本仕訳（資産・負債・純資産）

**問題要旨**: 会社設立時の出資金と基本的な取引（商品仕入、売上、費用支払い）の仕訳を判定する問題。

`K4 因果メカニズム` `T1 正誤判定` `L1` `Trap-B 条件すり替え`

**正解**: （複数選択肢の正誤判定）

**必要知識**: [簿記基礎](/finance-and-accounting/bookkeeping-basics-account-categories-and-journal-entries) — 仕訳の基本規則

**解法の思考プロセス**:
1. 仕訳の基本：資産↑ = 負債↑ + 純資産↑（複式簿記）
2. 各取引を分類：
   - 出資金：資産（現金）↑、純資産（資本金）↑
   - 商品仕入（掛け）：資産（商品）↑、負債（買掛金）↑
   - 売上（現金）：資産（現金）↑、純資産（売上）↑
   - 費用支払い：費用↑、資産（現金）↓
3. 勘定科目と左右（借方・貸方）を正確に判定

**誤答の落とし穴** `Trap-B 条件すり替え`:
- 資産と純資産の混同：出資金は「資本金」（純資産）であり、資産ではない
- 買掛金と売掛金の逆：顧客からの買掛金は自社の負債

**学習アドバイス**: 仕訳は簿記の最小単位です。「Aが増えたら、Bも変わる」という複式簿記の本質を理解することが、すべての基礎となります。

### 第2問　複合仕訳（複数取引）

**問題要旨**: 1つの経済事象に複数の会計処理（例：商品販売＋返品処理＋値引き）が含まれる複合仕訳。

`K4 因果メカニズム` `T1 正誤判定` `L2` `Trap-C 部分正解`

**正解**: （複合仕訳の正確な記録）

**必要知識**: [簿記基礎](/finance-and-accounting/bookkeeping-basics-account-categories-and-journal-entries) — 複合取引の記録

**解法の思考プロセス**:
1. 取引を段階的に分解：
   - 販売取引：売上↑
   - 返品処理：売上↓（返品販売控除）
   - 値引き処理：売上↓（値引き）
2. 複数の仕訳をまとめるか、個別に記録するかの判断
3. 最終的な資産・負債・純資産への影響を確認

**誤答の落とし穴** `Trap-C 部分正解`:
- 返品と割引の混同：返品は契約キャンセル、割引は価格値下げ
- 部分的な認識：すべての処理を含めない誤り

**学習アドバイス**: 複合仕訳は、現実ビジネスの複雑な取引を記録する際に必須です。大きな取引は小分けして、段階的に処理することが重要です。

### 第3問　決算整理仕訳

**問題要旨**: 減価償却費、未払い費用、前払い保険料など、期間に対応させるための決算整理仕訳を作成する問題。

`K4 因果メカニズム` `T3 計算実行` `L3` `Trap-E 計算ミス`

**正解**: （決算整理仕訳の正確性）

**必要知識**: [簿記基礎](/finance-and-accounting/bookkeeping-basics-adjusting-entries) — 決算整理仕訳

**解法の思考プロセス**:
1. 発生主義への調整：取引発生時期と支払い時期のズレを調整
2. 各項目の決算整理：
   - 減価償却：固定資産↓、減価償却費↑
   - 未払い費用：費用↑、未払い金↑
   - 前払い費用：資産（前払い金）↓、費用↑
3. 金額の計算：日数按分など

**誤答の落とし穴** `Trap-E 計算ミス`:
- 減価償却率の計算誤り：定額法の場合、(取得原価 - 残存価額) / 耐用年数
- 期中取引の重複計上：決算整理により既に記録された取引を重複させる

**学習アドバイス**: 決算整理仕訳は、会計の最も「技術的」な部分です。公式を覚え、パターン認識することで、対応速度を高めることができます。

### 第4問　精算表作成・振替仕訳

**問題要旨**: 試算表から精算表（修正前残高試算表 → 修正後残高試算表 → 損益計算書 → 貸借対照表）を完成させ、決算仕訳から振替仕訳までを実行する問題。

`K4 因果メカニズム` `T3 計算実行` `L4` `Trap-E 計算ミス`

**正解**: （精算表の完全作成）

**必要知識**: [簿記基礎](/finance-and-accounting/bookkeeping-basics-account-categories-and-journal-entries) — 精算表と決算処理フロー

**解法の思考プロセス**:
1. 修正前試算表の各科目に修正仕訳を加える
2. 修正後残高を計算
3. 損益計算書欄に「収益」「費用」を、貸借対照表欄に「資産」「負債」「純資産」を振り分け
4. 差額（純利益）を計算し、両欄でバランスを確認
5. 振替仕訳：損益計算書の各科目を「損益」勘定に振替

**誤答の落とし穴** `Trap-E 計算ミス`:
- 修正仕訳の重複適用：既に修正済みの項目を再度調整
- 振替仕訳の誤り：「損益」勘定への振替を誤る
- 計算ミス：合計値の不一致で全体がバラバラに

**学習アドバイス**: 精算表は「決算処理の総合問題」です。各ステップの小さなミスが最終的な不一致につながるため、丁寧さが最重要です。

## 財務諸表・5つの利益

### 第5問　損益計算書と5つの利益

**問題要旨**: 売上総利益、営業利益、経常利益、税金前利益、当期純利益の定義と、各利益を計算する問題。

`K2 グラフ形状` `T2 グラフ読解` `L2` `Trap-D 混同誘発`

**正解**: （正確な利益計算）

**必要知識**: [財務諸表と5つの利益](/finance-and-accounting/corporate-accounting-financial-statements-and-five-profits) — 損益計算書の構造

**解法の思考プロセス**:
1. 売上総利益 = 売上高 - 売上原価
2. 営業利益 = 売上総利益 - 販売費及び一般管理費
3. 経常利益 = 営業利益 + 営業外収益 - 営業外費用
4. 税金前利益 = 経常利益 ± 特別利益・損失
5. 当期純利益 = 税金前利益 - 法人税等

各層での意味：
- 売上総利益：営業活動の直接的収益性
- 営業利益：本業の利益
- 経常利益：通常の事業活動からの利益
- 当期純利益：最終的な株主への帰属利益

**誤答の落とし穴** `Trap-D 混同誘発`:
- 営業外損益と特別損益の混同：金利・配当は営業外、固定資産売却益は特別
- 原価と費用の区別：売上原価は販売費ではない

**学習アドバイス**: 5つの利益の階層構造は、企業の経営的な「良し悪し」を診断する枠組みです。営業利益が高いが経常利益が低い場合は「本業は好調だが、財務（借金）が負担」という状況が読み取れます。

### 第6問　営業外損益・特別損益

**問題要旨**: 営業外収益（利息収入、配当金）と営業外費用（支払利息、手数料）、特別利益（固定資産売却益）と特別損失（社債償却損失、災害損失）の分類と影響分析。

`K2 グラフ形状` `T4 因果推論` `L3` `Trap-B 条件すり替え`

**正解**: （正確な分類と影響判定）

**必要知識**: [財務諸表と5つの利益](/finance-and-accounting/corporate-accounting-financial-statements-and-five-profits) — 営業外損益と特別損益の定義

**解法の思考プロセス**:
1. 営業外損益：日常的だが本業以外
   - 収益：金利、配当金、投資回収による利益
   - 費用：支払利息、有価証券売却損
2. 特別損益：非日常的、臨時的
   - 利益：固定資産売却益、保険金受け取り
   - 損失：固定資産除却損、自然災害による損失

3. 経常利益への含否の判定：営業外は含む、特別は含まない

**誤答の落とし穴** `Trap-B 条件すり替え`:
- 営業外と特別の判定基準の曖昧さ：「本業ではないこと」（営業外）と「非日常的」（特別）の区別
- 例外的取扱い：正常販売ではない商品売却（営業外）など

**学習アドバイス**: 営業外損益と特別損益の区分は、企業の「通常の利益性」と「臨時的事象」を分離する目的です。分析者が「本当の稼ぎ力」を判断する際に必要な分類です。

### 第7問　PL・BS の読み取り

**問題要旨**: 損益計算書と貸借対照表から、売上高・営業利益・総資産・自己資本など主要科目の金額を読み取り、経営状況を分析する問題。

`K2 グラフ形状` `T2 グラフ読解` `L2` `Trap-D 混同誘発`

**正解**: （正確な数値読取と分析）

**必要知識**: [財務諸表と5つの利益](/finance-and-accounting/corporate-accounting-financial-statements-and-five-profits) — 財務諸表の構造と関係

**解法の思考プロセス**:
1. PL から：売上高、各利益層、費用率
2. BS から：資産構成（流動・固定）、負債・資本構成
3. 関係式の確認：PL の当期純利益 = BS の純資産変化（配当を除く）
4. 効率指標の計算：ROA = 純利益 / 総資産、など

**誤答の落とし穴** `Trap-D 混同誘発`:
- 単年度数値との比較誤り：前年度との比較なく現状判定
- 相対指標の計算誤り：分子分母を反対にするなど

**学習アドバイス**: 財務諸表の「読み方」は、単なる数値抽出ではなく、その背後にある経営現象を理解することです。利益は出ているが現金が不足している状況など、PL と BS の乖離を見ることが重要です。

### 第8問　修正仕訳の影響分析

**問題要旨**: 決算整理仕訳が、損益計算書と貸借対照表のどの科目にどの金額で影響するかを分析する問題。複数の修正仕訳の総合的影響。

`K4 因果メカニズム` `T4 因果推論` `L3` `Trap-C 部分正解`

**正解**: （修正仕訳の全体影響の正確な把握）

**必要知識**: [簿記基礎](/finance-and-accounting/bookkeeping-basics-account-categories-and-journal-entries) — 決算整理の会計処理全般

**解法の思考プロセス**:
1. 各修正仕訳を記録し、その影響を追跡
2. 修正前の試算表から修正後の各科目値を計算
3. PL への影響：各費用・収益の増減
4. BS への影響：各資産・負債・資本の増減

**誤答の落とし穴** `Trap-C 部分正解`:
- 修正仕訳の相互関係の見落とし：1つの修正が他に波及
- 部分的修正：すべての修正を反映していない

**学習アドバイス**: この問題は「修正仕訳の意味」を深く理解しているか確認するものです。単なる仕訳ではなく、その経済的意味（何が変わったのか）を把握することが重要です。

## 資産会計

### 第9問　在庫資産評価（先入先出法等）

**問題要旨**: 売上原価の計算方法（先入先出法FIFO、後入先出法LIFO、加重平均法）の選択と、期末在庫評価額の計算。

`K3 数式・公式` `T3 計算実行` `L2` `Trap-E 計算ミス`

**正解**: （正確な在庫計算）

**必要知識**: [資産会計](/finance-and-accounting/corporate-accounting-assets-accounting) — 在庫資産評価方法

**解法の思考プロセス**:
1. 各方法の原理：
   - FIFO：先に仕入れた商品から売却
   - LIFO：後に仕入れた商品から売却（日本では廃止傾向）
   - 加重平均：すべての仕入れ商品の加重平均で評価
2. インフレ環境での影響：
   - FIFO：売上原価低い → 利益高い → 納税多い
   - LIFO：売上原価高い → 利益低い → 納税少ない（節税）
3. 期末在庫の計算

**誤答の落とし穴** `Trap-E 計算ミス`:
- 方法の計算誤り：加重平均では各取引の日数を反映
- 金額と数量の混同：単価×数量を正確に計算

**学習アドバイス**: 在庫評価方法は、企業の利益操作の可能性を示唆する重要な論点です。同じ販売量でも、在庫評価方法の選択により利益が大きく変わることは、会計の相対性を理解する上で重要です。

### 第10問　建設仮勘定・固定資産減価償却

**問題要旨**: 建設中の固定資産を「建設仮勘定」で管理し、完成後に本資産に振り替え、その後減価償却を実行する一連の会計処理。

`K4 因果メカニズム` `T3 計算実行` `L3` `Trap-B 条件すり替え`

**正解**: （建設仮勘定から固定資産への振替と減価償却計算）

**必要知識**: [資産会計](/finance-and-accounting/corporate-accounting-assets-accounting) — 固定資産の取得と減価償却

**解法の思考プロセス**:
1. 建設中：支払額を「建設仮勘定」で集計
2. 完成時：建設仮勘定の残高を「建物」「機械」等に振替
3. 振替後：完成日からの減価償却開始

例：
- 建設開始時支払い 1000 万円 → 建設仮勘定 1000 万円
- 完成時 → 建物 1000 万円、建設仮勘定 0 万円
- 翌年減価償却費計算：1000 万円 / 20 年 = 50 万円

**誤答の落とし穴** `Trap-B 条件すり替え`:
- 完成前の減価償却：建設中は減価償却しない
- 振替のタイミング：完成時に正確に振替（期中完成の場合は日数按分）

**学習アドバイス**: 建設仮勘定は「期間帰属」の原則を示す重要な概念です。費用が発生した時期と、その価値が生成される期間を一致させるために使用されます。

## 原価計算・管理会計

### 第13問　原価要素の分類

**問題要旨**: 製造に関わる支出を、原価要素（材料費、労務費、経費）と、直接費・間接費に分類する問題。

`K1 定義・用語` `T1 正誤判定` `L1` `Trap-A 逆方向誘発`

**正解**: （正確な原価分類）

**必要知識**: [原価計算概論](/finance-and-accounting/cost-accounting-purpose-and-classification) — 原価分類の基本

**解法の思考プロセス**:
1. 原価要素の 3 分類：
   - 材料費：商品に直接組み込まれる支出
   - 労務費：製造に従事する労働者の給与
   - 経費：その他製造関連支出（光熱費、設備リース料など）

2. 直接費・間接費の分類：
   - 直接費：製品に直接帰属可能（直接材料費、直接労務費）
   - 間接費：複数製品に帰属（製造間接費）

3. 分類の実例：
   - 直接材料費：鋼材（自動車製造）
   - 直接労務費：組立工の給与
   - 製造間接費：工場家賃、機械減価償却費

**誤答の落とし穴** `Trap-A 逆方向誘発`:
- 直接性の過度な狭定義：追跡可能だが手間のため簿記計上されない費用
- 製造・販売の境界：販売費は製造間接費ではなく、期間費用

**学習アドバイス**: 原価分類の基本は「製品のコストか、期間のコストか」という判定です。これが原価計算システムの第一歩となります。

### 第14問　原価計算制度の選択（全部原価計算 vs 直接原価計算）

**問題要旨**: 全部原価計算（製造間接費をすべて製品に配賦）と直接原価計算（固定製造間接費は期間費用）の特性と、期末在庫評価による利益差。

`K2 グラフ形状` `T4 因果推論` `L2` `Trap-B 条件すり替え`

**正解**: （計算制度の選択と影響判定）

**必要知識**: [原価計算概論](/finance-and-accounting/cost-accounting-purpose-and-classification) — 原価計算制度の比較

**解法の思考プロセス**:
1. 全部原価計算：
   - 製品単価 = 直接費 + 変動製造間接費 + 固定製造間接費（按分）
   - 利益 = 売上 - 売上原価（完成品ベース）

2. 直接原価計算：
   - 製品単価 = 直接費 + 変動製造間接費
   - 固定製造間接費は全額期間費用

3. 利益差の発生：在庫増加時に全部原価計算が利益大となる
   - 理由：固定費の一部が期末在庫に含まれ、販売されていないコストが遅延される

**誤答の落とし穴** `Trap-B 条件すり替え`:
- 販売量と生産量の混同：在庫変動時に利益差が発生
- 固定費と変動費の定義の曖昧性：「固定」の判定基準

**学習アドバイス**: 2つの原価計算制度は、「本当の利益は何か」という問いに異なる答えを提供します。全部は外部報告向け、直接は経営判断向けという使い分けが実務です。

### 第15問　標準原価・差異分析

**問題要旨**: 標準原価（予め設定した原価）と実際原価の差を、数量差異と価格差異に分解し、原因分析する問題。

`K3 数式・公式` `T3 計算実行` `L4` `Trap-E 計算ミス`

**正解**: （差異の正確な計算と判定）

**必要知識**: [原価計算（標準原価・差異分析）](/finance-and-accounting/cost-accounting-standard-costing-and-variance-analysis) — 標準原価と差異分析

**解法の思考プロセス**:
1. 原価差異の 2 層分解：
   - 数量差異（Labor Quantity Variance）= (実際数量 - 標準数量) × 標準価格
   - 価格差異（Price Variance）= (実際価格 - 標準価格) × 実際数量

2. 例（労務費）：
   - 標準：時給 1,000 円 × 100 時間 = 100,000 円
   - 実際：時給 1,100 円 × 110 時間 = 121,000 円
   - 数量差異 = (110 - 100) × 1,000 = 10,000 円（不利）
   - 価格差異 = (1,100 - 1,000) × 110 = 11,000 円（不利）
   - 合計差異 = 21,000 円（不利）

3. 原因分析：数量差異は工程管理、価格差異は調達・給与政策の責任

**誤答の落とし穴** `Trap-E 計算ミス`:
- 差異の計算順序の誤り：先に固定される値を正確に把握
- 有利・不利の判定：完全な方向確認が必須

**学習アドバイス**: 差異分析は、管理会計の「診断ツール」です。数値の差を見つけるだけでなく、その原因を特定することが重要です。

### 第16問　CVP分析・損益分岐点

**問題要旨**: Cost-Volume-Profit 分析により、売上高・変動費・固定費から、損益分岐点（Breakeven Point）を計算し、利益目標達成に必要な売上高を求める問題。

`K3 数式・公式` `T3 計算実行` `L4` `Trap-E 計算ミス`

**正解**: （損益分岐点と利益目標達成売上の計算）

**必要知識**: [管理分析（CVP分析）](/finance-and-accounting/management-analysis-cvp-and-break-even) — 損益分岐点分析

**解法の思考プロセス**:
1. 変動費率 = 変動費 / 売上高
2. 貢献利益 = 売上高 - 変動費 = 売上高 × (1 - 変動費率)
3. 損益分岐点（売上高） = 固定費 / (1 - 変動費率)

例：
- 固定費：5,000 万円
- 変動費率：60%（貢献利益率 40%）
- 損益分岐点 = 5,000 / 0.4 = 12,500 万円

4. 利益目標達成売上：(固定費 + 利益目標) / (1 - 変動費率)

**誤答の落とし穴** `Trap-E 計算ミス`:
- 固定費と変動費の混同：どちらが分子に来るかを正確に把握
- 利益率と変動費率の混同：(1 - 変動費率) = 貢献利益率

**学習アドバイス**: CVP 分析は、企業経営の「実行可能性」を判断する最も実用的な道具です。起業時の「どれだけ売ればいいか」に直接答えられる分析です。

### 第17問　部門別計算・配賦

**問題要旨**: 複数の部門（製造部門・補助部門）がある企業で、各製品への原価配賦を行う問題。補助部門費の製造部門への配賦。

`K4 因果メカニズム` `T3 計算実行` `L3` `Trap-E 計算ミス`

**正解**: （部門別原価配賦の正確な実行）

**必要知識**: [原価計算（部門別計算）](/finance-and-accounting/cost-accounting) — 部門費配賦

**解法の思考プロセス**:
1. 部門分類：
   - 製造部門：直接製品に関わる（組立、仕上げ）

2. 配賦手順：
   - 補助部門の費用を、製造部門に配賦（用役利用時間などを基準）
   - 各製造部門の費用を、製品に配賦（直接労務時間などを基準）

   - 段階別配賦法：補助部門間の相互関係を考慮
   - 相互配賦法：連立方程式で解く（複雑）

**誤答の落とし穴** `Trap-E 計算ミス`:
- 配賦順序の誤り：補助部門間の関連性を見落とし
- 配賦基準の不適切な選択：用役利用量と実績が乖離

**学習アドバイス**: 部門別計算は、多くの企業で実務的に実行されている方法です。配賦基準（分配キー）の選択が、結果に大きく影響するため、各部門の責任会計と関連します。

## 経営分析・財務管理

### 第18問　利益率分析・ROI

**問題要旨**: 営業利益率、総資本利益率（ROA）、自己資本利益率（ROE）等の計算と、企業の収益性を診断する問題。

`K3 数式・公式` `T3 計算実行` `L2` `Trap-E 計算ミス`

**正解**: （各利益率の正確な計算と診断）

**必要知識**: [管理分析（収益性）](/finance-and-accounting/management-analysis-profitability) — 収益性指標

**解法の思考プロセス**:
1. 営業利益率 = 営業利益 / 売上高
   - 本業の収益性を示す（営業外費用の影響なし）

2. ROA = 純利益 / 総資産
   - 資産全体の利用効率

3. ROE = 純利益 / 自己資本
   - 株主投資の利回り

4. DuPont 分析：
   - ROE = (営業利益率) × (総資産回転率) × (レバレッジ)
   - 利益率、効率、財務戦略の 3 要素を分解

例：
- 売上 100 億円、営業利益 10 億円、純利益 6 億円
- 総資産 120 億円、自己資本 40 億円
- 営業利益率 = 10 / 100 = 10%
- ROA = 6 / 120 = 5%
- ROE = 6 / 40 = 15%

**誤答の落とし穴** `Trap-E 計算ミス`:
- 分子・分母の誤認：営業利益と純利益、総資産と自己資本を混同
- 分子の時点的相違：期首と期末の平均を使用すべき場合

**学習アドバイス**: ROI 系指標は、企業の「儲ける力」を測定する基本です。ROE が ROA より高いのは、負債による「レバレッジ効果」を示しています。

### 第19問　流動性分析・支払能力

**問題要旨**: 流動比率、当座比率、現金比率など短期支払能力指標を計算し、企業の短期的な財務安全性を診断する問題。

`K2 グラフ形状` `T2 グラフ読解` `L2` `Trap-D 混同誘発`

**正解**: （流動性指標の正確な計算と判定）

**必要知識**: [管理分析（安全性）](/finance-and-accounting/management-analysis-safety) — 安全性指標

**解法の思考プロセス**:
1. 流動比率 = 流動資産 / 流動負債（目安 100～150%）
   - 短期債務償還能力の基本指標
   - 100% 未満 = 即座の危機

2. 当座比率 = 当座資産 / 流動負債（目安 100%）
   - 流動資産から在庫を除く（回収不確実性考慮）
   - より保守的な評価

3. 現金比率 = 現金・預金 / 流動負債（目安 20～30%）
   - 最も厳格な判定

4. 固定比率 = 固定資産 / 自己資本（目安 100%以下）
   - 長期的な資金バランス

例：
- 流動資産 100 万円、在庫 30 万円、流動負債 80 万円
- 流動比率 = 100 / 80 = 125%（良好）
- 当座比率 = 70 / 80 = 87.5%（要注意）
- 現金比率 = 40 / 80 = 50%（仮定）

**誤答の落とし穴** `Trap-D 混同誘発`:
- 業種別基準の無視：小売業と製造業で基準値が異なる
- 季節変動の見落とし：期末の一時的な状況で判定

**学習アドバイス**: 流動性分析は「企業が倒産しないか」を判定する最前線です。いくら利益があっても、現金不足で支払いできなければ倒産です。

### 第20問　キャッシュフロー分析

**問題要旨**: 営業活動によるキャッシュフロー（OCF）、投資活動によるキャッシュフロー（ICF）、財務活動によるキャッシュフロー（FCF）を分析し、企業のキャッシュ創出能力を診断する問題。

`K4 因果メカニズム` `T4 因果推論` `L3` `Trap-B 条件すり替え`

**正解**: （キャッシュフロー計算と分析）

**必要知識**: [キャッシュフロー・資金管理](/finance-and-accounting/profit-and-cash-management-cash-flow-and-fcf) — CFO・CFI・CFF の関係

**解法の思考プロセス**:
1. OCF = 営業利益 + 減価償却費 - 税金 + 運転資本変動
   - 本業からのキャッシュ生成力

2. ICF = -(資本支出 + 投資活動)
   - 成長投資への支出（通常マイナス）

3. CFF = 社債発行 - 返済 + 配当・自社株買い
   - 株主・債権者からの資金調達と返済

4. フリーキャッシュフロー = OCF - 資本支出
   - 自由に使えるキャッシュ

各パターンの含意：
- OCF > 0, ICF < 0, CFF > 0：成長企業（投資資金を借入で補っている）
- OCF > 0, ICF < 0, CFF < 0：成熟企業（営業CFで配当・返済）
- OCF < 0：経営危機

**誤答の落とし穴** `Trap-B 条件すり替え`:
- 利益とキャッシュの混同：利益 > 0 でも、OCF < 0 の場合がある（売上債権増加など）
- 投資活動と財務活動の混同：M&A は投資、借入は財務

**学習アドバイス**: キャッシュフロー分析は、利益分析より「現実的」です。企業の真の健全性は、営業CFが持続的にプラスであることです。

### 第21問　予算制御・予算差異分析

**問題要旨**: 予め設定した予算（Budget）と実績を比較し、差異原因を特定する予算管理の問題。部門別・項目別の差異分析。

`K4 因果メカニズム` `T3 計算実行` `L4` `Trap-E 計算ミス`

**正解**: （予算差異の正確な計算と原因判定）

**必要知識**: [利益・資金管理（予算管理）](/finance-and-accounting/profit-and-cash-management-profit-planning-and-budget-control) — 予算編成と差異分析

**解法の思考プロセス**:
1. 差異の定義：
   - 好適差異（Favorable Variance）：実績が予算より良好
   - 不利差異（Unfavorable Variance）：実績が予算より悪化

2. 固定予算 vs. フレキシブル予算：
   - 固定予算：予定売上に基づく予算で固定
   - フレキシブル予算：実績売上に応じて調整

3. 差異分析の 2 層：
   - 販売量差異：実績売上数 × 標準単価 - 予算
   - 価格差異：(実績単価 - 標準単価) × 実績数量

例：
- 予算：単価 1,000 円 × 100 個 = 100,000 円
- 実績：単価 900 円 × 120 個 = 108,000 円
- 販売量差異 = (120 - 100) × 1,000 = 20,000 円（好適）
- 価格差異 = (900 - 1,000) × 120 = -12,000 円（不利）
- 合計差異 = 8,000 円（好適）

**誤答の落とし穴** `Trap-E 計算ミス`:
- 固定予算ベースでの差異判定：実績量が予算量と異なる場合、調整が必要
- 差異の有利・不利の論理：売上は高いほど好適、費用は低いほど好適

**学習アドバイス**: 予算差異分析は、経営層への「実績報告」の基礎となります。単なる数値比較ではなく、経営判断に資する分析が求められます。

---

### 第22問　経営分析総合問題

**問題要旨**: 複数年の PL・BS データから、各種指標を計算し、企業の経営状況の推移と問題点を総合的に診断する問題。

`K4 因果メカニズム` `T4 因果推論` `L3` `Trap-D 混同誘発`

**正解**: （複合的な分析と判断）

**必要知識**: 全章の統合的理解が必要

**解法の思考プロセス**:
1. 時系列データの整理：年度別 PL・BS を表に整理
2. 各種指標の時系列計算：
   - 売上成長率、営業利益率の推移
   - ROA、ROE の推移
   - 流動比率、固定比率の推移
3. 複合的な診断：
   - 売上は成長しているが、利益率が低下？ → 原価率上昇を確認
   - ROE が低下？ → ROA 低下か、レバレッジ低下か判定
4. 問題点の特定と対策案の提示

**誤答の落とし穴** `Trap-D 混同誘発`:
- 単一指標での判断：全体像を見ずに一部指標で結論
- トレンドの見落とし：1 年度の数値だけでなく、推移を観察

**学習アドバイス**: この問題は「診断士試験の最終関門」とも言えます。複数の分析手法を統合し、現実的な経営判断を下す能力を問うものです。
