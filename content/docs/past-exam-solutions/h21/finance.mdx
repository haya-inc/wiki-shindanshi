## 概要

## 簿記・決算基礎

### 第1問　決算整理と純損益計算

**問題要旨**: 決算整理前試算表から、商品棚卸、有価証券評価、減価償却、貸倒引当金、前払・未払項目を処理し、当期純損益を求める複合問題

`K3 計算手順` `T3 計算実行` `L3` `Trap-E 計算ミス`

**正解**: エ

**必要知識**: [決算整理仕訳](/finance-and-accounting/bookkeeping-basics-adjusting-entries) — 8項目の決算整理手続、[有価証券評価](/finance-and-accounting/corporate-accounting-assets-accounting) — その他有価証券の時価評価

**解法の思考プロセス**:

1. **売上原価の計算**: 期首商品 + 当期仕入 - 期末商品
2. **有価証券評価差額**: 時価 - 帳簿価額の合計
3. **有価証券評価損益**: 評価差額の発生・取崩
4. **減価償却費**: (取得原価 - 残存価額) / 耐用年数 × 経過年数
5. **貸倒引当金繰入**: 売掛金 × 引当率 - 既設残高（差額が繰入）
6. **前払い家賃**: 当期費用 = 支払家賃 - 前払い増加額
7. **未払い利息**: 当期費用 = 支払利息 + 未払い増加額
8. **純損益**: 売上 - 売上原価 - 営業費用（調整後）

各項目の相互関連を正確に把握し、試算表形式で整理することがポイント。

**誤答の落とし穴** `Trap-E 計算ミス`:

1. **前払・未払処理の方向誤り**: 「前払い減少 = 費用増加」「未払い増加 = 費用増加」の関係を逆向きに処理
2. **有価証券**: 取得原価と時価を混淆、既往評価差額と当期差異を区別できない
3. **貸倒引当金**: 繰入前残高との差引を忘れ、新規設定額をそのまま費用化する
4. **複数項目同時処理で計算カスケード誤り**: 1項目のミスが後続項目に波及

**学習アドバイス**:

決算整理は「発生主義」と「マッチング原則」の具体化です。各処理の**背景にある会計原則を理解**したうえで、系統的な手順（①売上原価 → ②資産評価 → ③引当金 → ④経過勘定 → ⑤集計）で進めましょう。複数設問の場合、先に1項目の解説全体を読んでから計算に入ると、論理が明確になります。

### 第2問　損益科目の分類

**問題要旨**: 売上値引・売上割引・売上割戻の3つの概念を区別し、損益計算書での適切な分類箇所を判定する問題

`K1 定義・用語` `T2 分類判断` `L1` `Trap-D 類似混同`

**正解**: イ

**必要知識**: [損益科目の分類](/finance-and-accounting/corporate-accounting-financial-statements-and-five-profits) — 売上高の控除項目と販売費の区分

**解法の思考プロセス**:

- **売上値引**: 商品の不良・破損に基づく値引 → **売上高から控除**（総売上高－売上値引＝売上高）
- **売上割引**: 現金払い奨励、支払期限短縮への割引 → **販売費及び一般管理費**（金融的コスト）
- **売上割戻**: 大量購入奨励、販売促進目的の割戻 → **販売費及び一般管理費**（販売促進費）

3つを並べたとき、売上値引だけが「売上高の修正項目」であり、割引・割戻は「営業外の付随費用」という構造を理解することが鍵。

**誤答の落とし穴** `Trap-D 混同誘発`:

- **売上割引と売上割戻の混同**: どちらも「割」という字を含むため、機械的に同じ分類と誤認識する
- **売上値引を販売費に分類**: 値引も割引も「売上を減らす」という共通点だけで判断する
- **後発的処理と事後的調整の混淆**: 割引は「支払時点で判明」する後発事象、割戻は「契約時点で合意」する先行条件という時点の違いを見落とす

**学習アドバイス**:

この3つの違いは**テキストで厳密な定義**として提示されています。単語の字面ではなく「損益計算書の構造：売上高 → 売上原価 → 粗利 → 販売費」に照らして、各項目がどこに属するか**図式化して確認**する習慣をつけましょう。過去問では毎年のように出題されるテーマです。

### 第3問　その他有価証券の評価差額

**問題要旨**: 複数銘柄のその他有価証券について、期末時価と帳簿価額から評価差額を計算し、純資産の部に計上する額を求める

`K2 分類・表示` `T3 計算実行` `L2` `Trap-E 計算ミス`

**正解**: ウ

**必要知識**: [有価証券の評価](/finance-and-accounting/corporate-accounting-assets-accounting) — その他有価証券の時価評価と評価差額の処理

**解法の思考プロセス**:

1. **各銘柄の評価差額を計算**: 期末時価 - 帳簿価額（取得原価または既往評価額）
2. **複数銘柄の合計**: 各銘柄の差額を代数和（有利差と不利差を相殺）
3. **既往評価差額との差分**: 期末評価差額 - 期首評価差額 = 当期評価変動（P/L計上分）
4. **純資産計上額**: 期末評価差額の全額を「その他有価証券評価差額」として純資産に計上

ポイントは「直接法」で処理し、評価差額の累積額（ストック）をB/S純資産に表示することです。

**誤答の落とし穴** `Trap-E 計算ミス`:

1. **売却済銘柄の除外漏れ**: 期中売却した銘柄を含めてしまい、過度に計算される
2. **帳簿価額の誤認識**: 取得原価と既往評価額のどちらを「帳簿価額」とするか混淆
3. **当期評価変動と累積評価差額の混同**: P/L計上（当期）と純資産表示（累積）の区別が不明確
4. **相殺の誤り**: 有利差と不利差を相殺せず、絶対値で合計してしまう

**学習アドバイス**:

有価証券の評価は「4分類（売買目的・満期保有・子会社・その他）ごとに評価方法が異なる」という原則を前提に、その他有価証券は「時価評価で評価差額は純資産計上」というルールを確実に押さえてください。複数銘柄の場合は表を作って、銘柄ごと・項目ごとに整理する癖をつけましょう。

## 財務諸表分析

### 第4問　五期利益の定義と計算

**問題要旨**: 損益計算書の五期利益（売上総利益、営業利益、経常利益、税引前当期利益、当期利益）の定義と計算段階を正確に理解しているかを問う

`K1 定義・用語` `T1 正誤判定` `L1` `Trap-D 類似混同`

**正解**: エ

**必要知識**: [五期利益](/finance-and-accounting/corporate-accounting-financial-statements-and-five-profits) — 各段階の定義と計算構造

**解法の思考プロセス**:

- **売上総利益** = 売上高 - 売上原価 → 本業の粗利率（製造業は重要指標）
- **営業利益** = 売上総利益 - 販売費及び一般管理費 → 本業の成績
- **経常利益** = 営業利益 ± 営業外収益・費用 → 全社的な経常成績
- **税引前当期利益** = 経常利益 ± 特別利益・損失 → 税務上の益金
- **当期利益** = 税引前当期利益 - 法人税等 → 株主帰属利益

各段階の経済的意味：営業利益は「経営効率」、経常利益は「全社成績」、当期利益は「納税後成績」と区別することが重要。

**誤答の落とし穴** `Trap-D 混同誘発`:

1. **営業利益と経常利益の混同**: 営業外収益・費用（金利、為替差益、賃借料など）を営業利益に含める誤り
2. **経常利益と税引前当期利益の混同**: 特別項目（固定資産売却益、災害損失など）を経常利益に含める誤り
3. **売上原価と販売費の混淆**: 配送費、委託販売手数料など分類が曖昧な項目の扱い

**学習アドバイス**:

五期利益は単なる計算ではなく、**各段階が経営分析のどの側面を反映するか**を理解することが診断士試験の本質です。例えば、営業利益の増減は「本業の競争力変化」、経常利益と営業利益の差は「金融コストと投資効果」を示します。P/Lの行ごとに「この段階は何を見ているのか」と問い直す習慣をつけましょう。

### 第5問　包括利益

**問題要旨**: 当期利益とその他包括利益を合算した包括利益の概念と、IFRSでの導入背景を理解しているかを問う

`K1 定義・用語` `T1 正誤判定` `L1` `Trap-C 部分正解`

**正解**: ウ

**必要知識**: [包括利益](/finance-and-accounting/corporate-accounting-financial-statements-and-five-profits) — IFRS導入時の会計基準改定、その他包括利益の内容

**解法の思考プロセス**:

**包括利益** = 当期純利益 + その他包括利益

その他包括利益に含まれるもの:
- その他有価証券評価差額の発生・取崩
- 為替換算差額（外貨子会社のB/S翻訳）
- 確定給付年金の再測定差異
- キャッシュフロー・ヘッジの有効部分

**背景**: 従来は「評価差額は損益外（直接純資産計上）」としていたが、IFRSでは「すべての利益変動を一括開示（包括利益計算書）」に統一。これにより、当期純利益だけでなく、その他の経済変動も含めた**全体的な成績**が可視化される。

**誤答の落とし穴** `Trap-C 部分正解`:

1. **当期純利益 = 包括利益と誤認識**: 部分正解（当期純利益は包括利益に含まれるが、全部ではない）
2. **その他包括利益の範囲が曖昧**: 通常の営業外収益・費用（金利、為替取引損益）とその他包括利益を混淆
3. **IFRS導入前後の基準差**: 日本基準での従来的な利益計算と包括利益の違いを見落とす

**学習アドバイス**:

包括利益は「会計基準国際化（IFRS）の結果」として出題されるテーマです。単に「包括利益 = 当期利益 + その他包括利益」という公式だけでなく、**「なぜこの項目がその他包括利益に含まれるのか」という会計的意義**を理解しておくと、応用問題への対応力が高まります。セグメント情報との関連も参考にしてください。

### 第6問　ROA・ROE分析

**問題要旨**: 総資産利益率（ROA）と自己資本利益率（ROE）を複数年度で計算し、レバレッジ効果を含めた経営効率の変動を分析する

`K2 計算・分析` `T3 計算実行` `L2` `Trap-E 計算ミス`

**正解**: イ

**必要知識**: [ROA・ROE分析](/finance-and-accounting/management-analysis-profitability) — 利益率分析と資本構成の影響

**解法の思考プロセス**:

- **ROA** = 当期純利益 / 総資産平均 → 経営効率（資産活用度）
- **ROE** = 当期純利益 / 自己資本平均 → 株主利益率
- **関係式**: ROE = ROA × 金融レバレッジ（総資産 / 自己資本）

複数年度のデータから:
1. 各年度のROA、ROEを計算
2. ROA低下の要因（利益減か資産増加か）を分解
3. 金融レバレッジの変化（負債増加・減少）がROEに与えた影響を分析

ポイント: **利益が下降してもROEが変わらない（または上昇する）のは、金融レバレッジが上昇したから**という因果を理解すること。

**誤答の落とし穴** `Trap-E 計算ミス`:

1. **分子分母の誤反転**: ROA = 総資産 / 利益など逆算してしまう
2. **年度平均の計算漏れ**: 期末数値だけを用いて年間平均資産を無視
3. **複数年の比較誤り**: 当期のみの計算で前年度との変動を見落とす
4. **金融レバレッジの無視**: ROEが改善した原因を「利益増」だけで説明し、「負債増加」による底上げを見落とす

**学習アドバイス**:

ROA・ROEは経営診断の基本指標です。**DuPont分析（ROE = 利益率 × 資産回転 × 金融レバレッジ）による分解**を習慣化しておくと、複合的な経営変動を論理的に説明できます。また、「レバレッジが高い = 好況期は株主利益が大きく、不況期は赤字が大きくなる」というリスクとリターンの関係も理解しておきましょう。

### 第7問　流動比率・当座比率

**問題要旨**: 短期支払能力を示す流動比率と当座比率を計算し、企業の短期流動性を評価する

`K2 計算・分析` `T3 計算実行` `L2` `Trap-A 逆方向`

**正解**: ウ

**必要知識**: [安全性分析](/finance-and-accounting/management-analysis-safety) — 流動資産・当座資産・流動負債の定義と分類

**解法の思考プロセス**:

- **流動比率** = 流動資産 / 流動負債 → 100%以上が目安。企業が流動負債で短期的に支払う必要がある額に対し、流動資産（1年以内に現金化可能）がどれだけあるか
- **当座比率** = 当座資産 / 流動負債 → より厳格な指標。商品など売却に時間がかかる棚卸資産を除いた、より流動性の高い資産だけを分子に用いる

計算:
1. 流動資産 = 現金 + 売掛金 + 商品 + その他流動資産
2. 当座資産 = 流動資産 - 商品（棚卸資産）= 現金 + 売掛金 + その他当座資産
3. 流動比率と当座比率の双方で判定（当座比率 > 100%なら、商品なしでも支払可能で より安全）

**誤答の落とし穴** `Trap-A 逆方向誘発`:

1. **分子分母の逆転**: 流動負債 / 流動資産などと計算してしまい、100%以上が悪いと誤認識
2. **流動負債への長期負債混入**: 1年以上の返済期限がある長期借入金の一部を誤って流動負債に含める
3. **当座資産の範囲誤認**: 受け取手形、短期投資等をカウント漏れする
4. **時点混淆**: 決算日の残高と年度平均のどちらを使うか不明確

**学習アドバイス**:

流動比率・当座比率は**業種によって基準値が異なる**ことが重要です。小売業（在庫が多い）は流動比率が150%程度でも健全ですが、金融機関（在庫がない）は高い比率を維持します。データを見たら「この企業の業種は何か」を意識し、業種別基準と照らし合わせて判定する習慣をつけましょう。

### 第8問　資産回転率と在庫回転率

**問題要旨**: 資産活用効率を示す総資産回転率と棚卸資産回転率を計算し、経営効率を分析する

`K2 計算・分析` `T3 計算実行` `L2` `Trap-E 計算ミス`

**正解**: ウ

**必要知識**: [効率性分析](/finance-and-accounting/management-analysis-efficiency) — 回転率の定義と経営意義

**解法の思考プロセス**:

- **総資産回転率** = 売上高 / 総資産平均 → 資産がその期間にど何回転売上を生み出したか。高いほど効率的
- **棚卸資産回転率** = 売上原価 / 平均棚卸資産 → 在庫が何回転したか。高い = 在庫を効率的に販売
- **回転日数** = 365 / 回転率 → 現金化までの日数に転換

計算のポイント:
1. 回転率は「年度平均資産」を分母に用いる（期末値だけでは年間平均を反映しない）
2. 在庫回転率の分子は「売上原価」（売上高ではなく）であることに注意
3. 複数年の推移で、回転率の改善・悪化傾向を把握

**誤答の落とし穴** `Trap-E 計算ミス`:

1. **分子の誤選択**: 資産回転率に売上原価を、在庫回転率に売上高を用いる
2. **平均値の計算漏れ**: 期末値だけで計算
3. **回転日数の逆算ミス**: 365 / 回転率の計算で小数点を誤る
4. **複数指標を混淆**: 流動比率（比率）と回転率（回数）の概念を混同

**学習アドバイス**:

回転率が「高い ⟹ 効率的」という単純な判定に陥りがちです。実際には、業種（卸売業は回転率が低い傾向、小売業は高い傾向）や経営戦略（品質重視で在庫多保有 vs 薄利多売）によって基準値が異なります。また、回転日数に転換すると「現金化までの平均日数」として経営管理上の意義（資金繰り圧力）が一層明確になります。

### 第9問　キャッシュフロー四象限分析

**問題要旨**: 営業CF、投資CF、財務CFの符号パターンから企業の経営ステージと財務状況を判定する

`K1 概念` `T1 分類判定` `L2` `Trap-D 類似混同`

**正解**: ウ

**必要知識**: [キャッシュフロー分析](/finance-and-accounting/profit-and-cash-management-cash-flow-and-fcf) — CF三区分の経済的意味と四象限パターン

**解法の思考プロセス**:

CF四象限の典型パターン:

| 象限 | 営業CF | 投資CF | 財務CF | 経営ステージ | 解釈 |
| --- | --- | --- | --- | --- | --- |
| ① | + | - | - | 成長期後期〜成熟期 | 本業で稼ぎ、設備投資し、負債返済（理想形） |
| ② | + | + | - | 衰退期・縮小期 | 本業が低迷、資産売却で現金化、負債返済 |
| ③ | - | - | + | 急速成長期 | 本業未熟、多額投資、財務調達で補填（ベンチャー） |
| ④ | - | + | - | 危機段階 | 本業赤字、資産売却も追いつかず、縮小局面 |

問題文のデータから、各CFの符号を読み取り、上表のいずれかのパターンに該当するかを判定。

**誤答の落とし穴** `Trap-D 混同誘発`:

1. **営業CF = 利益と混淆**: 発生主義利益（P/L）と現金主義キャッシュフロー（CF）の概念混淆
2. **CF符号の誤認識**: 「営業CF＜0 = 赤字」と過度に悲観的に解釈（設備投資財源不足を示すだけかも）
3. **投資CFと財務CFの混淆**: 設備投資（投資CF）と株主配当・負債返済（財務CF）の区別が不明確
4. **象限別の経営評価を誤る**: 象限③（成長投資期）を「危機」と誤認識

**学習アドバイス**:

CF四象限分析は**企業のライフサイクル（創業期 → 成長期 → 成熟期 → 衰退期）と連動**しています。象限だけで「良い・悪い」と判定するのではなく、「その企業はいまどのステージにあるのか、その中で健全なCF構造か」という相対的な評価が重要です。年度経過による象限の遷移を追うと、企業診断の奥行きが深まります。

### 第10問　標準原価と差異分析

**問題要旨**: 標準原価と実際原価の差異を、数量差異と価格差異に分解し、経営上の意義を理解する

`K3 計算手順` `T3 計算実行` `L3` `Trap-E 計算ミス`

**正解**: ウ

**必要知識**: [標準原価と差異分析](/finance-and-accounting/cost-accounting-standard-costing-and-variance-analysis) — 差異の計算手順と原因分析

**解法の思考プロセス**:

**材料差異の分解**:
- **数量差異** = (実際使用量 - 標準使用量) × 標準価格
- **価格差異** = (実際価格 - 標準価格) × 実際使用量

**労務費差異の分解**:
- **時間差異** = (実際時間 - 標準時間) × 標準賃率
- **賃率差異** = (実際賃率 - 標準賃率) × 実際時間

差異の符号:
- 実際が標準を上回る（使用量超過、価格上昇）→ **不利差異**（-）
- 実際が標準を下回る（使用量削減、価格抑制）→ **有利差異**（+）

**誤答の落とし穴** `Trap-E 計算ミス`:

1. **差異の符号を逆**：不利差異を有利と計算する
2. **複数段階差異の足し忘れ**：材料差異の合計を忘れ、数量差異だけで判定
3. **標準・実際の混淆**：分子に標準、分母に実際を入れるなど公式を誤適用
4. **単位の混淆**：kg、個、時間など単位系の統一漏れ

**学習アドバイス**:

差異分析は**「なぜ差異が発生したのか」という原因追究が本質**です。単に計算結果の数字だけを比較するのではなく、「数量差異は作業効率の低下か、価格差異は市場変動か」という経営判断に結びつけることが重要です。差異の相互関連（数量が増えると価格が下がるなど）を考慮した総合的な経営分析力を養いましょう。

### 第11問　製造原価計算

**問題要旨**: 直接材料費、直接労務費、製造間接費から製造原価を算定し、期首期末在庫との関連で売上原価を計算する

`K3 計算手順` `T3 計算実行` `L3` `Trap-E 計算ミス`

**正解**: ウ

**必要知識**: [製造原価計算](/finance-and-accounting/cost-accounting-job-order-and-process-costing) — 原価要素の分類と原価計算書の構造

**解法の思考プロセス**:

**製造原価の構成**:
製造原価 = 直接材料費 + 直接労務費 + 製造間接費

**売上原価への橋渡し**:
売上原価 = 期首仕掛品 + 製造原価 - 期末仕掛品
         = 期首半製品 + 期首仕掛品相当 + 製造原価 - 期末仕掛品

各原価要素の特定:
1. 直接材料費：仕入・消費の追跡可能な材料
2. 直接労務費：製造部門の賃金・給与
3. 製造間接費：配賦対象（配賦率 = 製造間接費 / 配賦基礎）

**誤答の落とし穴** `Trap-E 計算ミス`:

1. **製造間接費の配賦漏れ**：配賦基礎（直接労務費、機械時間など）の選択誤りや計算漏れ
2. **販売費を製造原価に混入**：配送費、営業手当など販売部門費を誤分類
3. **期末評価**：先入先出法と移動平均法の適用基準誤り
4. **在庫勘定の混淆**：仕掛品と製品、販売用と製造用の区別不明確

**学習アドバイス**:

製造原価計算は「どこまでが製造部門費で、どこからが販売部門費か」という**部門別経営管理の基礎**です。小規模企業（簡便法適用）と大規模企業（詳細配賦）の相違も理解しておくと、現実の企業診断に即応できます。原価計算書の行ごとに「この項目は何を示しているのか」と問い直す習慣が重要です。

### 第12問　損益分岐点売上高

**問題要旨**: 変動費、固定費、売上高から損益分岐点（BEP）売上高を計算し、利益計画の基礎を理解する

`K3 計算手順` `T3 計算実行` `L3` `Trap-E 計算ミス`

**正解**: ウ

**必要知識**: [損益分岐点分析](/finance-and-accounting/management-analysis-cvp-and-break-even) — CVP分析の基本と応用

**解法の思考プロセス**:

**利益ゼロの分岐点**:
営業利益 = 売上高 - 変動費 - 固定費 = 0
⟹ 売上高 = 変動費 + 固定費

**限界利益を用いた算定**:
限界利益 = 売上高 - 変動費
限界利益 = 固定費（利益ゼロ時）

BEP売上高 = 固定費 / 限界利益率
          = 固定費 / (限界利益 / 売上高)

**計算の流れ**:
1. 変動費率 = 変動費 / 売上高
2. 限界利益率 = 1 - 変動費率
3. BEP売上高 = 固定費 / 限界利益率

**誤答の落とし穴** `Trap-E 計算ミス`:

1. **変動販売費の漏れ**：製造原価の変動費のみを計算し、販売部門の変動費を除外
2. **限界利益率計算の誤り**：限界利益「額」と「率」の混淆
3. **固定費の過少計上**：製造間接費の固定部分を忘れる
4. **BEP販売量との混同**：売上高ベースと個数ベースの計算式を誤適用

**学習アドバイス**:

CVP分析はマネジメント会計の最重要ツールです。「売上が増えると利益が加速度的に増える」という仕組み（限界利益の威力）を数字で実感することが重要です。BEP売上高だけでなく、「目標利益達成の売上高」や「販売量」への応用も習慣化させましょう。

### 第13問　限界利益率と目標利益

**問題要旨**: 目標営業利益を達成するための販売量・売上高を計算する応用問題

`K3 計算手順` `T3 計算実行` `L3` `Trap-E 計算ミス`

**正解**: エ

**必要知識**: [損益分岐点分析](/finance-and-accounting/management-analysis-cvp-and-break-even) — 目標利益達成の販売計画

**解法の思考プロセス**:

**目標営業利益達成式**:
営業利益 = (売上高 - 変動費) - 固定費
目標営業利益 = 限界利益 - 固定費

⟹ 限界利益 = 固定費 + 目標営業利益

**販売量による表現**:
限界利益単価 × Q = 固定費 + 目標営業利益
Q = (固定費 + 目標営業利益) / 限界利益単価

**売上高による表現**:
売上高 = (固定費 + 目標営業利益) / 限界利益率

例：限界利益単価400円、固定費9,000,000円、目標営業利益1,500,000円
Q = (9,000,000 + 1,500,000) / 400 = 26,250個

**誤答の落とし穴** `Trap-E 計算ミス`:

1. **目標営業利益の誤認識**：「1個当たり150円」を総額150円と読み違える
2. **単価計算の誤り**：限界利益率で計算すべき場面で単価を用いる
3. **BEP販売量との混同**：BEP（9,000,000 / 400 = 22,500個）と目標販売量を区別できない
4. **複数年度の累積**：当年度目標と複数年度目標の区別不明確

**学習アドバイス**:

目標利益の設定（いくら稼ぐのか）は経営計画の核心です。BEP分析は「最低水準」を示すに過ぎず、実際の経営目標はBEPを上回る利益達成が必要です。「固定費をいくら圧縮するか」「限界利益率をいかに改善するか」といった経営改善のシミュレーションにも活用できます。

### 第15問　自由キャッシュフロー（FCF）計算

**問題要旨**: 営業CF、投資CFから、企業の自由な資金使途可能額（配当・借入返済の原資）となるFCFを計算する

`K3 計算手順` `T3 計算実行` `L3` `Trap-E 計算ミス`

**正解**: ウ

**必要知識**: [キャッシュフロー管理と FCF](/finance-and-accounting/profit-and-cash-management-cash-flow-and-fcf) — FCFの定義と企業価値評価への応用

**解法の思考プロセス**:

**FCFの計算式**:
FCF = 営業キャッシュフロー - 投資キャッシュフロー
   = 営業CF - (資本支出 + 減額投資)

**経済的意義**:
- FCF > 0：本業の稼ぎで投資をまかなえ、余剰があり配当・返済可能
- FCF = 0：本業収入 = 投資支出（維持的投資＝成長投資ゼロ）
- FCF < 0：本業収入では投資不足、外部調達が必要（成長投資フェーズ）

**計算の詳細**:
営業CF（間接法）: 税引前利益 + 減価償却 - 運転資本増加 - 法人税支払 = X
資本支出：機械装置・建物購入 = Y
FCF = X - Y

**誤答の落とし穴** `Trap-E 計算ミス`:

1. **CF計算の誤り**：営業CFと当期純利益を混淆
2. **投資CFの範囲誤認**：配当支払い（財務CF）を投資CFに含める
3. **資本支出の過少計上**：定期的な更新投資（維持的支出）を見落とす
4. **フリーの意味誤認**：「いくら使ってもよい」と無制限に解釈

**学習アドバイス**:

FCFは**企業価値評価（DCF法）の要素**として、ファイナンス理論では極めて重要です。配当政策（いくら株主に還元するか）や債務返済能力（借入余力）を判定する際の核となる指標です。FCFから各ステークホルダー（株主、債権者）への配分額を逆算する思考プロセスを習慣化させましょう。

### 第17問　営業CF と現金営業利益

**問題要旨**: CF計算書の間接法による営業CF計算の詳細手順を理解し、利益と現金の隔差を説明する

`K3 計算手順` `T3 計算実行` `L3` `Trap-E 計算ミス`

**正解**: ウ

**必要知識**: [キャッシュフロー計算書](/finance-and-accounting/profit-and-cash-management-cash-flow-and-fcf) — 間接法による営業CF算定

**解法の思考プロセス**:

**営業CFの間接法手順**:

1. **スタート**: 税引前当期純利益
2. **非資金項目の加算**: 減価償却費、貸倒引当金繰入、減損損失、有価証券評価損
3. **非資金項目の減算**: 有価証券評価益、有形固定資産売却益
4. **運転資本の調整**:
   - 売掛金増加 ⟹ CF減少（販売しても現金未回収）
   - 買掛金増加 ⟹ CF増加（仕入但支払い遅延）
   - 棚卸資産増加 ⟹ CF減少（現金を在庫に転化）
5. **法人税等支払い**: 税引前利益から減算（実際支払い額）

営業CF = 税引前利益 + 減価償却 - 有価証券評価益 + 売掛金減少 - 棚卸増加 + 買掛金増加 - 法人税支払

**誤答の落とし穴** `Trap-E 計算ミス`:

1. **運転資本調整の方向誤り**：売掛金増加を加算、減少を減算してしまう（逆向き）
2. **税金計算の誤り**：当期計上税金ではなく実際支払い額で計算すべき
3. **複数段階の誤り蓄積**：1項目の計算ミスが後続に波及
4. **利益と現金の混淆**：PL計上利益の増減とCF増減を同一視する

**学習アドバイス**:

CF計算書は「利益 ≠ 現金」という会計の本質を示しています。黒字企業でもCF危機に陥る（黒字倒産）メカニズムを、運転資本の変動を通じて理解することが経営診断の要点です。決算分析で「営業利益は好調だが営業CF悪化 ⟹ 売掛金回収困難か在庫過剰」といった読解力を養いましょう。

### 第18問　CF三区分による経営判定

**問題要旨**: CF計算書の営業・投資・財務の三区分から、企業の経営ステージと財務健全性を総合判定する

`K1 概念` `T1 分類判定` `L2` `Trap-D 類似混同`

**正解**: ウ

**必要知識**: [キャッシュフロー分析](/finance-and-accounting/profit-and-cash-management-cash-flow-and-fcf) — CF四象限と経営診断

**解法の思考プロセス**:

**健全な経営パターン（成熟期）**:
- 営業CF > 0（本業で稼ぐ）
- 投資CF < 0（設備投資を実施）
- 財務CF ≤ 0（負債返済・配当支払い）
⟹ 営業CF > |投資CF|で、配当・返済余力あり

**懸念パターン**:
- 営業CF < 0（本業赤字） ⟹ リストラ必要
- 投資CF > 0（資産売却）⟹ 資金繰り逼迫
- 財務CF > 0（増資・借入増加）⟹ 外部依存度上昇

**段階別判定**:
1. 営業CFの符号から本業の健全性を判定
2. 投資CFで成長投資か維持投資かを判定
3. 財務CFで資金調達戦略を判定
4. 三者の相互関係から経営ステージを推定

**誤答の落とし穴** `Trap-D 混同誘発`:

1. **営業CF < 0 = 破綻と誤認識**：ベンチャー（外部調達で投資継続）なら許容される
2. **投資CF符号の誤認識**：投資CFはマイナスが通常（資産購入で支出）
3. **一時的な変動を過度に評価**：1年度のみで長期トレンドを見落とす
4. **業種別基準値の無視**：資本集約的産業（土木）と軽資産産業（IT）のCF構造は異なる

**学習アドバイス**:

CF三区分は企業のライフサイクル観を提供します。「いま、この企業はどのステージにあり、その中で健全なCF構造か」という相対的評価が重要です。また、年度ごとのCFの推移（営業CFの改善傾向、投資CF規模の変化）を追うことで、経営改革の兆候や危機の前触れを察知できます。

### 第19問　標準原価差異の実務適用

**問題要旨**: 標準原価制度を運用する企業で、当月の材料・労務・製造間接費の差異を総合的に計算・分析する

`K3 計算手順` `T3 計算実行` `L3` `Trap-E 計算ミス`

**正解**: ウ

**必要知識**: [標準原価と差異分析](/finance-and-accounting/cost-accounting-standard-costing-and-variance-analysis) — 差異の合成と経営改善への活用

**解法の思考プロセス**:

**材料差異の詳細分解**:
- 数量差異：(実使用量 - 標準使用量) × 標準価格
- 価格差異：(実際価格 - 標準価格) × 実使用量
- 購入価格差異（別途）：(購入価格 - 標準価格) × 購入量

**労務費差異の詳細分解**:
- 時間差異（能率差異）：(実作業時間 - 標準時間) × 標準賃率
- 賃率差異：(実賃率 - 標準賃率) × 実作業時間

**製造間接費差異**:
- 能率差異：(実配賦基礎量 - 標準配賦基礎量) × 標準配賦率
- 予算差異：実際発生額 - (標準配賦基礎量 × 標準配賦率)

**総差異の合算**:
全差異 = 材料差異 + 労務費差異 + 製造間接費差異

**誤答の落とし穴** `Trap-E 計算ミス`:

1. **複数層差異の漏れ**：材料差異のみを計算し、労務費・間接費を見落とす
2. **単位換算誤り**：kg、時間、個の混淆
3. **既往差異と当月差異の混淆**：累積と当月を区別できない
4. **配賦基礎の誤選択**：直接労務費時間ベースと機械時間ベースの混淆

**学習アドバイス**:

実務の標準原価制度では、差異を「原因ごとに分解し、責任部門に帰属させ、改善を促す」プロセスが重要です。単に数字を計算するのではなく、「この差異はなぜ発生したのか（作業効率か市場変動か）」を常に問い直す思考が診断士には求められます。

### 第20問　製造間接費の配賦と実績集計

**問題要旨**: 予定配賦率による製造間接費を実際費用と照合し、配賦差額を計算・処理する

`K3 計算手順` `T3 計算実行` `L3` `Trap-E 計算ミス`

**正解**: ウ

**必要知識**: [原価計算-製造間接費](/finance-and-accounting/cost-accounting-job-order-and-process-costing) — 予定配賦と実績差異の処理

**解法の思考プロセス**:

**予定配賦率の算定**:
予定配賦率 = 予定製造間接費 / 予定配賦基礎量（直接労務費、機械時間等）

**配賦額の計算**:
配賦製造間接費 = 実績配賦基礎量 × 予定配賦率

**配賦差額の計算**:
配賦差額 = 実際発生額 - 配賦額

例：
- 予定製造間接費：1,200万円
- 予定配賦基礎量（直接労務費）：3,000万円
- 予定配賦率：1,200 / 3,000 = 0.4（40%）
- 実績直接労務費：2,800万円
- 配賦額：2,800 × 0.4 = 1,120万円
- 実際発生額：1,160万円
- 配賦差額：1,160 - 1,120 = 40万円（過少配賦）

**誤答の落とし穴** `Trap-E 計算ミス`:

1. **配賦率の誤計算**：予定数値と実績数値を混淆
2. **配賦基礎の誤選択**：直接労務費、機械時間、直接材料費の選択誤り
3. **差額の処理誤り**：配賦差額を製品に配分し直す際の計算誤り
4. **複数部門の合算誤り**：部門別配賦で部門ごとの差異を集計し忘れ

**学習アドバイス**:

製造間接費は「実績が予定と異なるため、一定の配賦差が必ず生じる」という現実を反映しています。差額をどう処理するか（製品に配分、営業外費用化など）は会計政策であり、企業によって異なります。小規模企業では簡便法（実績値を即座に適用）を採用することも多いです。

### 第21問　原価分類と意思決定

**問題要旨**: 変動費・固定費、直接費・間接費の分類を正確に行い、意思決定会計（関連原価）に適用する

`K2 分類・表示` `T2 分類判定` `L2` `Trap-D 類似混同`

**正解**: ウ

**必要知識**: [原価分類と管理会計](/finance-and-accounting/cost-accounting-purpose-and-classification) — 意思決定に関連する原価の定義

**解法の思考プロセス**:

**原価の二つの分類軸**:

1. **変動性による分類**:
   - 変動費：売上高・生産量に比例（直接材料費、直接労務費、変動販売費）
   - 固定費：短期的に不変（賃借料、減価償却、本社費、固定製造間接費）

2. **直接性による分類**:
   - 直接費：特定製品・部門に直接帰属（直接材料、直接労務）
   - 間接費：複数製品に共通（製造間接費、本社費）

**意思決定への応用**:
- **単価決定**：変動費単価＋限界利益で価格設定
- **受託製造判定**：関連原価（直接材料＋直接労務）のみ考慮
- **受注・拒否判定**：関連原価 > 受託価格なら拒否
- **事業廃止判定**：固定費が廃止後も発生するなら継続

**誤答の落とし穴** `Trap-D 混同誘発`:

1. **変動費と直接費の混淆**：直接材料は直接費かつ変動費だが、「直接費 = 変動費」ではない
2. **関連原価の誤定義**：過去に投じた埋没費用（サンク・コスト）を意思決定に含める
3. **機会原価の無視**：設備稼働の代替案（他製品生産の利益）を考慮漏れ
4. **短期意思決定と長期戦略の混淆**：短期は変動費のみ、長期は固定費も変化する

**学習アドバイス**:

原価分類は「目的によって異なる」という柔軟性が重要です。同じ「間接費」でも、短期の受託判定では除外（固定値）し、長期の事業評価では検討（変動可能）する、という相対的な思考が診断士には求められます。

### 第22問　NPV（正味現在価値）計算

**問題要旨**: キャッシュフロー割引現在価値合計から初期投資を差し引き、投資プロジェクトのNPVを計算する

`K3 計算手順` `T3 計算実行` `L3` `Trap-E 計算ミス`

**正解**: イ

**必要知識**: [NPV分析](/finance-and-accounting/finance-profitability-index) — 割引CF法による企業価値評価

**解法の思考プロセス**:

**NPV計算式**:
NPV = Σ[CF_t / (1 + r)^t] - 初期投資
   = 年1CF / (1+r) + 年2CF / (1+r)^2 + ... + 終了時価値 / (1+r)^n - 初期投資

**割引率（r）の決定**:
- WACC（加重平均資本コスト）：一般的
- コスト・オブ・キャピタル：リスク調整済み
- 企業の目標収益率

**NPV判定**:
- NPV > 0：採択（企業価値を増加させる）
- NPV < 0：拒却（企業価値を減少させる）
- NPV = 0：基準点（IRRと同じ利率）

**計算例**:
初期投資：1,000万円、割引率：10%、期待CF：年500万円×5年
PV = 500 / 1.1 + 500 / 1.1^2 + ... + 500 / 1.1^5
   = 500 × 3.791 = 1,895.5万円
NPV = 1,895.5 - 1,000 = 895.5万円（採択）

**誤答の落とし穴** `Trap-E 計算ミス`:

1. **割引率の誤適用**：事前に与えられた割引率を見落とし、デフォルト（0%）で計算
2. **CF計算の誤り**：税引き前CFと税引き後CFを混淆
3. **複利計算ミス**：(1 + r)^t の計算間違い、特に累乗の誤り
4. **初期投資のタイミング**：年0か年1かの時点で差違が生じる

**学習アドバイス**:

NPVは「絶対額」（いくら儲かるか）を示し、複数プロジェクトの規模が異なるときに有効です。ただし、資本が制約（投資できる総額が限定）される場合は、NPVの他にIRRや収益性指数（PI = PV / 初期投資）も並行して検討する必要があります。

### 第23問　投資判定：NPV vs IRR

**問題要旨**: NPVとIRRの定義の違い、および複数案比較での使い分けを理解する

`K1 定義・用語` `T1 正誤判定` `L2` `Trap-C 部分正解`

**正解**: ウ

**必要知識**: [投資意思決定](/finance-and-accounting/finance-profitability-index) — NPV、IRR、回収期間法の使い分け

**解法の思考プロセス**:

**NPVの特徴**:
- 定義：将来CFを現在価値に割引き、初期投資を差し引いた値
- 判定：NPV > 0 なら採択
- 複数案：NPV が大きい順に採択（資本無制限）
- メリット：絶対額で比較可能
- デメリット：プロジェクト規模の大小を反映しない

**IRRの特徴**:
- 定義：NPV = 0 となる割引率
- 計算：試行錯誤または内挿法
- 判定：IRR > 目標収益率（割引率）なら採択
- 複数案：IRR が高い順に採択（相対効率）
- メリット：規模に依存しない相対指標
- デメリット：複数の収益逆転（異なるIRR複数個）の可能性

**使い分け**:
- 資本無制約：NPV優先
- 資本制約下（予算上限あり）：IRR または 収益性指数（PI）で選別
- リスク調整：割引率をプロジェクトの リスク水準に応じて変動

**誤答の落とし穴** `Trap-C 部分正解`:

1. **部分正解**：「IRR が高い = 常に優位」と誤認識（資本制約が前提）
2. **複数IRRの無視**：CF が単調減少でなく、途中逆転時には複数IRR が存在することを見落とす
3. **割引率と IRR の混淆**：WACC（割引率）と IRR（NPV=0の利率）を同一視
4. **会計上利益とCFの混淆**：利益率が高い＝ CFも多いという誤認

**学習アドバイス**:

NPV と IRR の選択は「企業の資本制約状況」に依存します。大企業（外部調達可能）と中小企業（自己資金限定）では、投資判定のプロセスが異なります。診断士として企業診断する際は、「その企業の経営資源を前提に、どのプロジェクトを優先すべきか」という現実的な判断が求められます。

### 第24問　回収期間法

**問題要旨**: 初期投資が回収される期間（Payback Period）を計算し、投資の流動性リスクを評価する

`K2 計算・分析` `T3 計算実行` `L2` `Trap-E 計算ミス`

**正解**: ウ

**必要知識**: [投資意思決定](/finance-and-accounting/finance-profitability-index) — 回収期間法の計算と限界

**解法の思考プロセス**:

**回収期間の計算**:
初期投資が回収される期間（通常は年数）を、年度別累積CFから算定。

例：初期投資1,000万円、年CF：300、350、300、300、250万円
- 1年末累積：300
- 2年末累積：300 + 350 = 650
- 3年末累積：650 + 300 = 950
- 4年末累積：950 + 300 = 1,250 ⟹ 回収完了

回収期間：3年 + (1,000 - 950) / 300 = 3.17年

**判定基準**:
- 回収期間が短い ⟹ リスク低（現金化早い）
- 回収期間が長い ⟹ リスク高（長期間の不確実性）

**メリット・デメリット**:
- メリット：計算が単純、流動性リスクを直感的に反映
- デメリット：回収後のCF を無視（長期利益性を見落とす）、割引を考慮しない（時間価値無視）

**誤答の落とし穴** `Trap-E 計算ミス`:

1. **累積CF計算の誤り**：年度順序を逆にする、負のCFを加算するなど
2. **期間計算の誤り**：整数年で区切り、年度内の日割計算を漏らす
3. **割引回収期間との混淆**：通常の回収期間と割引回収期間（割引率適用）の区別不明確

**学習アドバイス**:

回収期間法は「NPV や IRR の補助判定基準」として用いられます。例えば「NPV は同程度だが、回収期間が短い方を選ぶ（リスク回避）」といった実務的な判断に活用されます。ただし、投資の本質的価値（長期利益）を見落とすため、単独判定は避けるべきです。

### 第25問　複数投資案の比較

**問題要旨**: 複数の投資案を NPV、IRR、回収期間など複合的な指標で比較し、意思決定を行う

`K3 計算手順` `T3 計算実行` `L3` `Trap-E 計算ミス`

**正解**: エ

**必要知識**: [投資意思決定](/finance-and-accounting/finance-profitability-index) — 複数案の総合比較

**解法の思考プロセス**:

**複数案比較の手順**:

1. **各案のNPVを計算** → 「資本無制約なら NPV最大案」
2. **各案のIRRを計算** → 「資本制約下なら IRR高順 または PI（収益性指数）高順」
3. **回収期間を参考** → 「流動性リスク の考慮」
4. **定性的要因を加味** → 「戦略適合性、技術リスク、市場リスク等」

**比較表例**:

| 案 | 初期投資 | NPV | IRR | 回収期間 | 判定 |
|---|---|---|---|---|---|
| A | 1,000 | 500 | 15% | 3.0年 | NPV最大 |
| B | 800 | 350 | 18% | 2.5年 | IRR高、回収早い |
| C | 1,200 | 480 | 12% | 3.5年 | 総合判定で劣位 |

資本無制約 ⟹ A採択（NPV最大）
資本800万円制約 ⟹ B採択（限られた資本で収益率高）

**誤答の落とし穴** `Trap-E 計算ミス`:

1. **単一指標のみ判定**：NPV 最大だけで選択し、IRR や回収期間を無視
2. **資本制約の見落とし**：問題文で資本上限が設定されているのに気づかない
3. **複数案相互の影響**：代替案選択の相互排他性（どれか一つのみ選択可能）を理解していない
4. **時系列の錯綜**：複数期間のプロジェクトで初期投資時点の不統一

**学習アドバイス**:

実務では、経営陣が投資判定を行う際に、定量指標（NPV、IRR）だけでなく、戦略的適合性や市場シェア拡大、技術革新、組織能力構築など、定性的要因を総合的に勘案します。診断士は「数字だけで判定するのではなく、経営全体の文脈の中で投資の意義を評価する」能力が求められます。

## 年度総括

### 思考法別の出題構造

**T1（正誤判定）**：4問（16%）
- 定義・概念の正確な理解を問う
- 五期利益、包括利益、基本概念（第4, 5, 23問など）

**T2（分類判断）**：5問（20%）
- 原価・費用などの適切な分類判定
- 損益科目、セグメント、原価分類（第2, 14, 16, 21問など）

**T3（計算実行）**：16問（64%）
- 計算ステップを正確に遂行する力が最重要
- 決算整理、有価証券評価、比率分析、CF計算、NPV・IRR（大多数）

**診断**: 平成21年度は**計算科目**であり、4選択肢から正解を判定する力（T1）よりも、複数ステップの計算を正確に遂行する力（T3）が合否を分けます。

### 形式層の難度分布

**L1 基礎知識（12%）**：
- 3問のみ（損益科目分類、五期利益、包括利益）
- 基本的な定義の暗記でカバー可能

**L2 応用理解（32%）**：
- 8問
- 基礎知識の組み合わせと部分的計算
- 流動比率・当座比率、ROA・ROE など

**L3 計算応用（56%）**：
- 14問（半数以上）
- 複数ステップの計算、複合的な判断が必要
- 決算整理、標準原価、CF計算、投資判定など

**診断**: **L1 のみで合格は不可能**（最大12点）。合格ライン60点には、L2（応用理解）と L3（計算応用）で48点以上の得点が必須です。

### 罠パターンの分布

| 罠パターン | マーク数 | 割合 | 該当問 |
| --- | --- | --- | --- |
| Trap-A 逆方向 | 1 | 4% | 7 |
| Trap-B 条件見落とし | 0 | 0% | — |
| Trap-C 部分正解 | 2 | 8% | 5, 23 |
| Trap-D 類似混同 | 6 | 24% | 2, 4, 9, 14, 18, 21 |
| Trap-E 計算ミス | 16 | 64% | 1, 3, 6, 7, 8, 10, 11, 12, 13, 15, 17, 19, 20, 22, 24, 25 |

**Trap-E（計算ミス）が圧倒的**：64%が計算ミスに起因する誤答。単なる計算力ではなく、**「何を計算するのか、どの公式を適用すべきか」という判定ミス**（売上原価の公式誤適用など）も含まれます。

### 本番セルフチェック10項目

1. **決算整理の8項目**: 商品棚卸、有価証券評価、減価償却、貸倒引当金、前払い・未払い、その他
2. **五期利益の段階**: 売上総利益 → 営業利益 → 経常利益 → 税引前当期利益 → 当期利益
3. **有価証券の4分類**: 売買目的、満期保有、子会社、その他（評価方法・開示が異なる）
4. **比率計算の分子分母**: 年度平均か期末値か、時点混淆の確認
5. **変動費と固定費の分離**: CVP分析での正確な分類
6. **CF三区分の符号**: 営業CF > 0 が基本、投資CFはマイナスが通常
7. **運転資本調整の方向**: 売掛金増加 = CF減少（逆向きに処理しない）
8. **標準原価の差異符号**: 実際 > 標準なら不利差異（マイナス）
9. **NPV vs IRR**: 資本無制約 ⟹ NPV、制約 ⟹ IRR/PI
10. **関連原価の定義**: 意思決定に「関連する」原価のみ（サンク・コスト除外）
