## 概要

令和2年度の企業経営理論は全37問（各4点、148点満点）で出題されました。経営戦略（問1～10）、組織論（問11～28）、マーケティング（問29～37）という3領域を中心に、基本的なフレームワークの理解と使用場面の判断力が問われます。

問題文は [J-SMECA 公式サイト（令和2年度 企業経営理論）](https://www.jf-cmca.jp/attach/test/shikenmondai/1ji2020/C1ji2020.pdf) から入手できます。手元に PDF を用意したうえでお読みください。全年度の問題は [J-SMECA 試験問題ページ](https://www.jf-cmca.jp/contents/010_c_/shikenmondai.html) で公開されています。

<Callout title="解説の読み方">
  各問について「問題要旨 → 分類タグ → 正解 → 必要知識 → 解法の思考プロセス → 誤答の落とし穴 → 学習アドバイス」の順で解説しています。分類タグの意味は[本ページ末尾の凡例](#分類タグの凡例)を参照してください。
</Callout>

## 経営戦略

### 第1問　VRIOフレームワークと競争優位

**問題要旨**: VRIOフレームワークを用いた競争優位の分析について、各要素の役割と関係性を正確に理解する問題。

`K1 定義・用語` `T1 正誤判定` `L1` `Trap-D 混同誘発`

**正解**: （公式発表を確認してください）

**必要知識**: [競争戦略とリソース戦略](/business-management-theory/knowledge-competitive-and-resource-strategy)

**解法の思考プロセス**: VRIOは経営資源の戦略的価値を判定するフレームワークです。(1) **Value** = 市場機会を活かし脅威に対抗する価値があるか、(2) **Rarity** = 競争相手が保有していない希少性があるか、(3) **Imitability** = 模倣されにくいか、(4) **Organization** = 当社は経営資源を活用できる体制か。4要素すべてが揃って初めて持続的競争優位が生まれます。

**誤答の落とし穴** `Trap-D 混同誘発`: VRIOの要素を混同すること。特に「Imitability = 経営資源の物理的な希少性」と誤認し、「Rarity」と混同する誤りが多い。

**学習アドバイス**: VRIO は競争優位を判定する最重要フレームワークで、2次試験でも事例企業の強みを分析する際に使われます。

### 第2問　経営戦略の段階

**問題要旨**: H. I. アンゾフの経営戦略の考え方について、意思決定のカテゴリーと内容を正確に理解する問題。

`K1 定義・用語` `T1 正誤判定` `L1` `Trap-D 混同誘発`

**正解**: （公式発表を確認してください）

**必要知識**: [企業戦略と成長戦略](/business-management-theory/knowledge-corporate-and-growth-strategy) — アンゾフの意思決定の階層、目標・戦略・組織の違い

**解法の思考プロセス**: アンゾフの戦略フレームワークは3段階：(1) **企業目標** = 企業が達成すべき定量的・定性的目標、(2) **戦略的方針** = 目標達成のための事業分野・競争方法の選択、(3) **組織的方針** = 内部組織を目標達成のために整備すること。

**誤答の落とし穴** `Trap-D 混同誘発`: 意思決定のカテゴリーを混同すること。特に「経営資源の配分」と「戦略的方針」を同一視する誤りが多い。

**学習アドバイス**: アンゾフの戦略論は経営戦略の基礎となるフレームワークです。各段階の意思決定主体と時間軸を対応させて整理することが重要です。

### 第3問　川上産業と部品企業の交渉力

**問題要旨**: 業界の構造分析と企業間の交渉力について、部品企業が川上産業に対してどのような立場にあるかを判定する問題。

`K4 因果メカニズム` `T4 因果推論` `L2` `Trap-A 逆方向`

**正解**: （公式発表を確認してください）

**必要知識**: [戦略的計画とSWOT](/business-management-theory/knowledge-strategic-planning-and-swot) — 業界分析と競争構造

**解法の思考プロセス**: M. ポーターの5フォース分析における「サプライヤーの交渉力」を問う問題です。部品企業が川上産業に対して強い交渉力を持つ条件：(1) 部品企業が寡占状態である、(2) 川上産業の製造コストに大きな影響を与える部品である、(3) 部品企業の転換コストが低い場合。

**誤答の落とし穴** `Trap-A 逆方向誘発`: 「供給者が少ない → 供給者に有利」という単純な論理で選択肢を選ぶこと。実際には「その部品がどれだけ重要か」「代替品があるか」という因果関係を見落とすこと。

**学習アドバイス**: 5フォースの各要素は独立していなく、相互に影響しあっています。1つの企業視点ではなく「業界全体の力学」を把握することが重要です。

### 第4問　PIMSと経営成績

**問題要旨**: PIMS研究における市場シェアと経営成績の関係について、因果メカニズムを正確に理解する問題。

`K4 因果メカニズム` `T4 因果推論` `L2` `Trap-D 混同誘発`

**正解**: （公式発表を確認してください）

**必要知識**: [企業戦略と成長戦略](/business-management-theory/knowledge-corporate-and-growth-strategy) — PIMS、市場シェアと収益性の関係

**解法の思考プロセス**: PIMS研究では市場シェアと経営成績（ROI）の正の相関が報告されています。因果メカニズム：(1) 高い市場シェア → スケール・メリットによる単位当たりコストが低下、(2) 高い市場シェア → ブランド認知が向上 → プレミアム価格を設定可能、(3) 高い市場シェア → 顧客ロイヤルティが高まり新規営業費が削減される。

**誤答の落とし穴** `Trap-D 混同誘発`: 「市場シェアが高い → 絶対的に経営成績が良い」と単純に誤認すること。実際には「シェアが高いから利益率が低い場合」（デフレ業界での過度な価格競争）も存在します。

**学習アドバイス**: PIMS研究は経営戦略の実証的な知見を提供する重要なフレームワークです。「相関関係」と「因果関係」の違いを理解することが重要です。

### 第5問　多角化とM&A

**問題要旨**: 企業の多角化とM&Aの関係について、経営戦略上の位置づけと実装方法の違いを理解する問題。

`K1 定義・用語` `T1 正誤判定` `L1` `Trap-D 混同誘発`

**正解**: （公式発表を確認してください）

**必要知識**: [企業戦略と成長戦略](/business-management-theory/knowledge-corporate-and-growth-strategy)

**解法の思考プロセス**: 多角化戦略の実装方法：(1) **内部開発** = 既存事業の経営資源を新規事業に配分、(2) **M&A** = 既存企業を買収・合併、(3) **提携・JV** = 他企業との協業。各方法には一長一短があり、時間軸、必要資本、リスク許容度に応じて選択されます。

**誤答の落とし穴** `Trap-D 混同誘発`: 「多角化 = M&A」と同義と誤認すること。実際には多角化の一部の手段がM&Aであり、内部開発による多角化もあります。

**学習アドバイス**: 多角化戦略は企業成長の重要な選択肢です。各実装方法の「メリット・デメリット」を整理することが重要です。

### 第6問　バリューチェーンと付加価値

**問題要旨**: M. ポーターのバリューチェーン分析について、各段階における付加価値の創出を理解する問題。

`K1 定義・用語` `T1 正誤判定` `L1` `Trap-D 混同誘発`

**正解**: （公式発表を確認してください）

**必要知識**: [競争戦略とリソース戦略](/business-management-theory/knowledge-competitive-and-resource-strategy) — バリューチェーン分析

**解法の思考プロセス**: バリューチェーンは主活動（研究開発 → 製造 → 販売 → サービス）と支援活動（人事・組織 → 管理・財務 → 調達）から構成されます。各段階での差別化と効率化が企業の競争優位を決めます。重要な点は「業界全体のバリューチェーン」を把握し、当社がどの段階に参加するか、どこで価値を創出するかを判定することです。

**誤答の落とし穴** `Trap-D 混同誘発`: バリューチェーンの各段階を「順序的に進める直線」と誤認すること。実際には並列的に最適化される側面があります。

**学習アドバイス**: バリューチェーン分析は2次試験の事例分析でも頻出テーマです。自分の業界に当てはめて、各段階がどのように価値を創出するかを具体的に考える習慣をつけましょう。

## 技術経営・イノベーション

### 第7問　製品開発プロセスと取り組み

**問題要旨**: 企業における新製品開発のプロセスについて、開発段階の特性と各段階での課題を理解する問題。

`K1 定義・用語` `T1 正誤判定` `L1` `Trap-D 混同誘発`

**正解**: （公式発表を確認してください）

**必要知識**: [技術経営・イノベーション戦略](/business-management-theory/knowledge-innovation-international-and-digital-strategy)

**解法の思考プロセス**: 新製品開発は複数の段階で構成：(1) アイデア創出 → (2) コンセプト開発 → (3) 製品設計 → (4) 試作・試験 → (5) 製造化 → (6) 市場投入。各段階での失敗リスク、必要投資額、継続判断の方法が異なります。

**誤答の落とし穴** `Trap-D 混同誘発`: 「製品開発 = 研究開発部門の責任」と単純に誤認すること。実際には営業・企画・製造・品質保証など複数部門が関与します。

**学習アドバイス**: 製品開発プロセスは業界によって大きく異なります。試験で出題される「実例企業の開発アプローチ」を丁寧に読むことが重要です。

### 第8問　大規模・小規模バッチ技術の選択

**問題要旨**: 製造業における生産技術の選択について、生産規模と技術方式の関係を理解する問題。

`K1 定義・用語` `T1 正誤判定` `L1` `Trap-D 混同誘発`

**正解**: （公式発表を確認してください）

**必要知識**: [技術経営・イノベーション戦略](/business-management-theory/knowledge-innovation-international-and-digital-strategy)

**解法の思考プロセス**: 製品の生産規模と技術方式の適合性：(1) **大規模バッチ生産** = 流れ作業・自動化が有効。標準化製品の大量生産に最適。固定費が大きく変更コストが高い、(2) **小規模バッチ生産** = 柔軟な設備・手作業が必要。多品種少量生産に向きます。

**誤答の落とし穴** `Trap-D 混同誘発`: 「大規模バッチ生産 = 最も効率的」という固定観念で選択肢を選ぶこと。実際には「顧客ニーズの多様性」によって、小規模バッチ生産の方が総合的に効率的な場合もあります。

**学習アドバイス**: 製造業の競争優位は「適切な生産技術の選択」にあります。具体的な企業事例と対応させて学習することが重要です。

### 第9問　ベンチャー企業のモデル

**問題要旨**: J. A. ディモンズによるベンチャー企業の成功モデルについて、起業家精神と企業成長要件の関係を理解する問題。

`K1 定義・用語` `T1 正誤判定` `L1` `Trap-D 混同誘発`

**正解**: （公式発表を確認してください）

**必要知識**: [技術経営・イノベーション戦略](/business-management-theory/knowledge-innovation-international-and-digital-strategy)

**解法の思考プロセス**: ディモンズのベンチャー企業モデルの主要要素：(1) **事業機会** = 市場で必要とされているが、既存企業が供給していないニーズ、(2) **起業家チーム** = 機会をビジネスに変える能力、(3) **資金調達能力** = 成長資金を確保する力、(4) **経営資源の効率的配分** = 限られた資源の活用。4要素すべてが必要です。

**誤答の落とし穴** `Trap-D 混同誘発`: 「起業家個人の資質」を重視しすぎ、「市場機会の存在」「資金調達能力」などの環境要因を過小評価すること。

**学習アドバイス**: ベンチャー企業は経済全体のイノベーション源泉です。「市場・組織・資金の三要素が揃うことで初めて成長が実現される」というシステム的な理解を深めましょう。

## 国際経営

### 第10問　国際経営の組織形態

**問題要旨**: 企業の国際化に伴う組織構造の発展段階について、事業規模と組織形態の関係を理解する問題。

**解法の思考プロセス**: ブランド戦略の選択肢：(1) **ブランド拡張** = 既存ブランドで新商品カテゴリーに進出（メリット：認知・信頼活用、デメリット：悪影響リスク）、(2) **新規ブランド** = 新しいブランドで展開（メリット：既存ブランドへの悪影響を回避、デメリット：投資が大きい）。選択は「対象カテゴリーと既存ブランドの関連性」を考慮します。

**誤答の落とし穴** `Trap-D 混同誘発`: 「ブランド拡張 = 常に効率的」と固定的に判定し、「類似度が低い商品への拡張は失敗リスクが高い」という条件付きの判定が必要であることを見落とすこと。

**学習アドバイス**: ブランド戦略は企業の長期的価値創造の基礎です。「短期的な売上拡大」と「長期的なブランド資産」のバランスを考慮した選択が重要です。

## 分類タグの凡例

### 知識種類（K）

- **K1 定義・用語**: 概念・理論の定義理解
- **K2 グラフ形状**: 図表・グラフの構造理解
- **K3 数式・公式**: 計算式の理解
- **K4 因果メカニズム**: 要因間の因果関係理解

### 思考法（T）

- **T1 正誤判定**: 複数選択肢の正誤判定
- **T2 構造理解**: 図表構造の理解に基づいて判定
- **T3 計算実行**: 計算結果に基づいて判定
- **T4 因果推論**: 複数要因の因果関係から推論

### 形式層（L）

- **L1 定義暗記**: 理論・用語の定義を当てはめるレベル
- **L2 グラフ構造理解・因果推論**: フレームワーク活用、複数要因で判定
- **L3 複合推論**: 複数領域を統合して判定

### 罠パターン

- **Trap-A 逆方向**: 因果関係を逆に読む誤り
- **Trap-B 条件すり替え**: 条件付き判定を無視する誤り
- **Trap-C 部分正解**: 正解の一部だけで判定する誤り
- **Trap-D 混同誘発**: 類似概念を混同する誤り（最頻出）
