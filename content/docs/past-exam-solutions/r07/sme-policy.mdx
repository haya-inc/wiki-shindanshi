## 概要

令和7年度（令和7年8月実施）第1次試験の中小企業経営・中小企業政策は、全30問出題されました。本年度の出題は、**制度・統計データの読み取り**（K5層）に重点が置かれており、白書・調査報告書からの実践的な問題が特徴です。労働関連施策、地域経済、事業承継、創業支援など、中小企業政策の主要分野がバランスよく網羅されています。

問題文は [J-SMECA 公式サイト（令和7年度（2025） 中小企業経営・中小企業政策）](https://www.jf-cmca.jp/attach/test/shikenmondai/1ji2025/G1JI2025.pdf) から入手できます。手元に PDF を用意したうえでお読みください。全年度の問題は [J-SMECA 試験問題ページ](https://www.jf-cmca.jp/contents/010_c_/shikenmondai.html) で公開されています。

## 各問解説

### 第1問｜中小企業の従業者数と付加価値額比較

**問題要旨**
経済産業省「令和3年経済センサス活動調査」に基づき、1999年と2021年の企業規模別における中小企業の記述として最も適切なものを選択。

**分類タグ**
K2（統計データ読み取り） / K1（定義・基本概念）

**正解**: ア

**必要知識**
- [中小企業の定義と規模判定基準](/sme-management-and-policy/knowledge-sme-definition-and-size-criteria)
- [中小企業の経済的役割と統計](/sme-management-and-policy/knowledge-sme-economic-role-and-statistics)

**解法の思考プロセス**
1. 各選択肢の従業者数と付加価値額の構成比を確認
2. 中小企業基本法の定義（資本金および従業者数の基準）を想起
3. 1999年と2021年の推移を比較し、中小企業のシェアが増加傾向であることを確認
4. 付加価値額が従業者数のシェアより低い点に注目（労働生産性の相対的低さ）

**誤答の落とし穴**
- イ：2021年の中小企業従業者数が約70%という数値を過信し、付加価値額との乖離を見落とす
- ウ：中規模企業と小規模企業の区分を誤認
- エ：従業者数のシェアのみに着目し、付加価値額への言及を忘れる

**学習アドバイス**
白書の統計データは「複数の指標」で同時に確認する習慣をつけましょう。従業者数のシェアが高いからといって経済貢献度（付加価値額）も同等とは限りません。

### 第2問｜産業別従業者数の付加価値額構成比

**問題要旨**
経済産業省「令和3年経済センサス活動調査」に基づき、2020年の産業別付加価値額構成比において最も高い中小企業の割合を選択。

**分類タグ**
K2（統計データ読み取り） / K4（経営課題認識）

**正解**: オ（製造業）

**必要知識**
- [中小企業の業種別特性](/sme-management-and-policy/knowledge-sme-industry-trends)

**解法の思考プロセス**
1. 各産業における中小企業の付加価値額構成比を順位づけ
2. 建設業は個人事業・小規模企業が多い → 付加価値額シェア相対的に低い
3. 製造業は大企業と中小企業の層が厚い → 中小企業のシェアが相対的に高い傾向
4. 卸売業・小売業は大手チェーン支配 → シェアは低め
5. 製造業が最も高いと判定

**誤答の落とし穴**
- ア（卸売業）：大手物流企業や大型流通による支配
- イ（建設業）：小規模企業が多いが、大手ゼネコンの集中度が高い
- ウ（小売業）：コンビニ、百貨店等の大企業支配が強い

**学習アドバイス**
業種によって「企業規模分布」と「付加価値額集中度」が異なることを理解しましょう。単純な企業数ではなく、売上高や付加価値額という「経済規模」で判断することが重要です。

### 第3問｜企業規模別企業数の推移

**問題要旨**
総務省「平成11年事業所・企業統計調査」と総務省・経済産業省「令和3年経済センサス活動調査」に基づき、1999年と2021年の企業規模別企業数を比較した際の記述として最も適切なものを選択。

**分類タグ**
K2（統計データ読み取り） / K1（定義・規模判定）

**正解**: イ（小規模企業のみ増加、中規模企業は減少）

**必要知識**
- [中小企業の定義と規模判定基準](/sme-management-and-policy/knowledge-sme-definition-and-size-criteria)
- [中小企業の経済的役割と統計](/sme-management-and-policy/knowledge-sme-economic-role-and-statistics)

**解法の思考プロセス**
1. 1999年：小規模企業（従業者4人以下）が圧倒的多数
2. 2021年：小規模企業数は減少傾向（人口減少・構造転換）
3. 中規模企業（従業者5～299人）は、さらに減少が顕著
4. 大企業は相対的に数が少ないが、集中度が高まる
5. 企業数減の中で、小規模企業の割合が相対的に高まる矛盾に注意

**誤答の落とし穴**
- ア：「中小企業が減少」と単純化しすぎ
- ウ：「中規模企業が増加」は事実に反する
- エ：「小規模企業が減少」は誤読（シェアは高まっている）

**学習アドバイス**
「絶対数」と「相対的シェア」を混同しないこと。全体の企業数が減少する中で、小規模企業の「割合」は上昇しているという二つの視点が必要です。

### 第4問｜自己資本比率と売上高経常利益率（計算問題）

**問題要旨**
「令和5年中小企業実態基本調査」に基づき、小規模企業（法人）の売上高経常利益率と自己資本比率の数値を記述式で答える。

**分類タグ**
K2（統計データ読み取り） / K4（経営指標）

**正解**
- 空欄A: 6%
- 空欄B: 25%

**必要知識**
- [中小企業の経営課題](/sme-management-and-policy/knowledge-sme-management-issues)
- [中小企業白書の読み方](/sme-management-and-policy/knowledge-sme-white-paper-reading)

**解法の思考プロセス**
1. 中小企業実態基本調査から小規模企業の経営指標を抽出
2. 売上高経常利益率 = 経常利益 ÷ 売上高
3. 自己資本比率 = 自己資本 ÷ 総資本
4. 実際のデータ：小規模企業は大企業と比べ、両指標とも低い傾向
5. 6%（利益率）と25%（自己資本比率）が白書の実績数値

**誤答の落とし穴**
- 大企業の数値と混同（大企業は利益率・自己資本比率ともに高い）
- 全体平均ではなく「小規模企業」特定の数値を見落とす
- 業種別差異を考慮せず単一数値を想定

**学習アドバイス**
中小企業白書の統計表では必ず「企業規模別」「業種別」の分類を確認してください。同じデータでも切り口によって異なります。

### 第5問｜外国人労働者の就労実態

**問題要旨**
厚生労働省「外国人雇用状況」に基づき、2019年から2023年の外国人労働者数の推移について、在留資格区分ごとの特徴を選択。

**分類タグ**
K3（労働・人材関連施策） / K2（統計データ読み取り）

**正解**: ウ（専門的・技術的分野の在留資格が増加、その他も増加傾向）

**必要知識**
- [労働力確保と外国人材活用](/sme-management-and-policy/policy-human-resources-and-employment-support)

**解法の思考プロセス**
1. 2019年：「身分に基づく在留資格」が大多数（留学生アルバイト、研修生）
2. 2023年：「専門的・技術的分野」の割合が増加（高度人材ニーズ）
3. しかし「身分に基づく」も依然として相当シェアを占める
4. 全体的には外国人労働者数は増加し続けている
5. 建設、製造、飲食サービス、農業などで就業

**誤答の落とし穴**
- ア（技能実習）：増加していない、むしろ横ばい
- イ（資格外活動）：留学生アルバイトは相対的に減少傾向
- エ（身分に基づく）：仙台で誤読しやすい選択肢

**学習アドバイス**
外国人材受け入れの制度は段階的に緩和されています。2023年現在の「全国版」データから、どの区分が実際に増加しているか白書で確認しましょう。

### 第6問｜大企業と小規模企業の労働生産性の差

**問題要旨**
経済産業省「令和3年経済センサス活動調査」に基づき、大企業と小規模企業の労働生産性の差について、産業ごとの記述として最も適切なものを選択。

**分類タグ**
K2（統計データ読み取り） / K4（経営課題認識）

**正解**: ウ（宿泊・飲食サービス業が、労働生産性の差が最も大きい）

**必要知識**
- [中小企業の業種別特性](/sme-management-and-policy/knowledge-sme-industry-trends)
- [労働生産性と経営効率](/sme-management-and-policy/knowledge-sme-management-issues)

**解法の思考プロセス**
1. 労働生産性（付加価値額 ÷ 従業者数）を業種別で比較
2. 製造業：設備投資による生産性向上で、大企業と小企業の差は比較的小さい
3. 小売業：大型店舗と個人商店で効率差は存在
4. 宿泊・飲食サービス業：大企業チェーン店と個人飲食店の極端な差 → **最大級の差**
5. 選択肢を検証：ウが最も説得力がある

**誤答の落とし穴**
- ア（製造業）：自動化・設備投資で比較的差が小さい
- イ（小売業）：確かに差があるが、飲食ほど極端ではない
- エ（建設業）：大規模プロジェクトと小規模工事の性質が異なり、単純比較不可

**学習アドバイス**
**労働集約型産業ほど、規模による生産性格差が大きい**という原則を覚えましょう。飲食業は典型例です。

### 第7問｜有形固定資産と無形固定資産への投資

**問題要旨**
経済産業省・厚生労働省「白書2024年版」に基づき、2015年から2023年の製造業の設備投資推移について、有形固定資産と無形固定資産の投資額を比較。

**分類タグ**
K2（統計データ読み取り） / K5（制度・データ重視）

**正解**: ア（大企業・中小企業とも増加）

**必要知識**
- [製造業の設備投資動向](/sme-management-and-policy/knowledge-sme-industry-trends)
- [中小企業白書の読み方](/sme-management-and-policy/knowledge-sme-white-paper-reading)

**解法の思考プロセス**
1. 2015年から2023年のトレンドを観察
2. 有形固定資産：2020年のコロナ減少後、2021年以降回復
3. 無形固定資産：DX投資、ソフトウェア投資として増加傾向
4. 大企業・中小企業ともに投資を増やしている
5. 特にDX関連の無形資産投資が顕著

**誤答の落とし穴**
- イ（大企業増加、中小企業減少）：中小企業も投資を増やしている
- ウ（中小企業増加、大企業減少）：大企業も積極投資
- エ（両者とも減少）：アベノミクス以降増加傾向

**学習アドバイス**
近年のトレンド：設備投資から「無形資産投資（DX、デジタル化）」へのシフトが顕著です。

### 第8問｜労働分配率の推移（1990～2022年）

**問題要旨**
財務省「法人企業統計調査年報」に基づき、1990年から2022年の労働分配率推移について、企業規模別の記述として最も適切なものを選択。

**分類タグ**
K2（統計データ読み取り） / K3（労働・人材）

**正解**: ウ（中小企業の労働分配率は2008年度まで上昇し、2009年度以降下降）

**必要知識**
- [労働分配率と経営指標](/sme-management-and-policy/knowledge-sme-management-issues)
- [中小企業の経済的役割と統計](/sme-management-and-policy/knowledge-sme-economic-role-and-statistics)

**解法の思考プロセス**
1. 労働分配率 = 労働所得 ÷ 付加価値額
2. リーマンショック（2008年）の影響を中心に分析
3. 2008年まで：景気拡張期、労働分配率上昇
4. 2009年以降：リーマン後の不況、利益重視で分配率低下
5. 中小企業は大企業より変動が大きい傾向

**誤答の落とし穴**
- ア（大企業の労働分配率が上昇）：実際には下降傾向
- イ（2008年までに下降）：逆の時期を指定
- エ（2008年以降も上昇）：景気悪化で分配抑制

**学習アドバイス**
**2008年リーマンショックは経営指標の重要な分岐点**です。政策選択問題では常に意識しましょう。

### 第9問｜労働条件改善の課題（空欄補充）

**問題要旨**
厚生労働省「令和5年労働白書」に基づき、企業規模別の労働条件改善への課題（雇用条件、給与水準、福利厚生の向上）について、取組状況の記述として最も適切なものを選択。

**分類タグ**
K3（労働・人材関連施策） / K5（制度・データ重視）

**正解**: ウ（「採用・正社員化」の取組割合が最も高く、「福利厚生」の取組割合が最も低い）

**必要知識**
- [労働条件改善と中小企業支援](/sme-management-and-policy/policy-human-resources-and-employment-support)

**解法の思考プロセス**
1. 従業者100～299人規模の企業における課題別取組比率
2. 「採用」「正社員化」：最優先課題 → 取組率高い
3. 「給与」：経営課題だが取組率は中程度
4. 「福利厚生」：予算制約で後回し → 取組率最低
5. 企業規模別では、小規模ほど福利厚生投資が遅れている

**誤答の落とし穴**
- ア（採用より福利厚生優先）：優先順位が逆
- イ（採用より正社員化が優先）：採用が先行課題
- エ（福利厚生の取組率が高い）：実務では後回しにされやすい

**学習アドバイス**
**中小企業の人材課題は「確保」が最優先**で、その次が「定着」です。福利厚生充実は経営安定後の課題と認識しましょう。

### 第10問｜中規模企業の借入金依存度

**問題要旨**
財務省「法人企業統計調査年報」に基づき、2006年と2022年の中規模企業の借入金依存度を比較した際の記述として最も適切なものを選択。

**分類タグ**
K4（金融支援制度） / K2（統計データ読み取り）

**正解**: ウ（中規模企業の借入金依存度は低下し、1社当たり総資産の金額は減少）

**必要知識**
- [中小企業融資制度](/sme-management-and-policy/policy-financial-support)
- [信用保証協会と金融支援](/sme-management-and-policy/policy-financial-support)

**解法の思考プロセス**
1. 借入金依存度 = 借入金 ÷ 総資本

- A：事業所が不要ですが
- B：は8,000万円以内
- C：与信保証額50万円以上

**必要知識**
- [経営セーフティ共済](/sme-management-and-policy/policy-fair-trade-and-small-business-support)

**解法の思考プロセス**
1. 食品衛生共済は「食品製造業」「食品卸売業」対象
2. 加入要件：食品衛生法に基づく許可取得
3. 保障内容：贈収賄罪、不正流通による損失補填
4. 保障限度額：8,000万円（または5,000万円）
5. 加入時の与信保証額設定が重要

**誤答の落とし穴**
- 事業形態の誤認（「事業所必須」と勝手に想定）
- 保障限度額の誤算（1億円等）
- 与信保証額の意味の混同

**学習アドバイス**
**業種別支援制度は詳細が出題されやすい**分野です。食品関連など特定業種向け制度はその概要を整理してください。

### 第24問｜中小企業向け税制（複数答案）

**問題要旨**
財務省・中小企業庁による中小企業関係税制について、法人税・所得税の優遇措置に関する複数記述式設問。

**分類タグ**
K5（制度・データ重視） / [税制措置](/sme-management-and-policy/policy-financial-support)

**正解**: ア（事業継続用途の控除対象 / 事業税額の20%）

**必要知識**
- [中小企業関係税制](/sme-management-and-policy/policy-fair-trade-and-small-business-support)

**解法の思考プロセス**
1. 中小企業向け特例として「所得拡大促進税制」等がある
2. 給与支給額が前年度比で一定率増加した場合、法人税から控除
3. 控除額：給与増加分 × 一定率（15%～25%）
4. 上限：所得金額の一定割合等の制限あり

**誤答の落とし穴**
- 控除額の算定誤り（前年度比ではなく「当年度額」と勘違い）
- 対象従業者の誤定義
- 適用要件の見落とし（例：青色申告申請等）

**学習アドバイス**
**税制措置は毎年改正される**ため、最新情報を確認することが重要です。試験出題時点の措置を把握しましょう。

### 第25問｜地域経済牽引企業の支援（複数空欄）

**問題要旨**
経済産業省「地域経済牽引企業の支援制度」について、対象企業判定基準と支援内容に関する複数空欄補充。

**分類タグ**
K5（制度・データ重視） / [地域創生支援](/sme-management-and-policy/policy-overseas-expansion-support)

**正解**
- A：売上50億円
- B：2億円
- C：2分の1

**必要知識**
- [地域経済牽引企業制度](/sme-management-and-policy/policy-fair-trade-and-small-business-support)

**解法の思考プロセス**
1. 地域経済牽引企業 = 地域で高い経済波及効果を持つ企業
2. 要件：売上50億円以上（「地方拠点強化税制」との関連）
3. 税制優遇：法人税額の控除（10%～15%）
4. 補助金：地域創生推進交付金で支援（補助率1/2～2/3）
5. 雇用創出目標：新規従業者数50人以上等

**誤答の落とし穴**
- 売上規模の誤記（100億円以上など）
- 補助率の誤認（1/3、全額補助等）
- 対象地域の制限を見落とし（人口減少地域等に限定される場合がある）

**学習アドバイス**
**地域創生関連制度は「地方への投資」を促進する政策パッケージ**として理解しましょう。

### 第26問｜信用保証協会の機能（複数答案）

**問題要旨**
中小企業信用保険公庫と信用保証協会について、融資保証制度の仕組みと保証料に関する複数記述選択。

**分類タグ**
K4（金融支援制度） / K5（制度・データ重視）

**正解**
- A：1億円以内
- B：無担保保証預託制度が8,000万円以内
- C：市町村長または特別区長
- D：一般保証と別枠

**必要知識**
- [信用保証協会制度](/sme-management-and-policy/policy-financial-support)

**解法の思考プロセス**
1. 信用保証協会 = 中小企業者の金融機関借入の保証機関
2. 一般保証：保証料率0.5%～1.9%（負債額により異なる）
3. 無担保保証：8,000万円限度（別枠）
4. 申し込み先：事業所所在地の信用保証協会
5. 信用保証料は「中小企業が負担」（金融機関が納付代行）

**誤答の落とし穴**
- 保証限度額の誤認（1,000万円、5,000万円等）
- 「一般保証」と「無担保保証」の枠の扱い（別枠 vs 共通枠）
- 申請手続先の誤認（金融機関 vs 保証協会）

**学習アドバイス**
**信用保証協会は「中小企業の最後の砦」として重要**です。制度の仕組みを正確に理解してください。

### 第27問｜交際費等の損金不算入

**問題要旨**
国税庁「法人税制」に基づき、交際費等の損金不算入について、中小企業向けの特例要件と仕組みを選択。

**分類タグ**
K5（制度・データ重視） / [税制措置](/sme-management-and-policy/policy-fair-trade-and-small-business-support)

**正解**: ウ（中小企業基本法に定める中小企業に該当する法人の場合、交際費等800万円までの金額は全額損金算入可能）

**必要知識**
- [中小企業関係税制](/sme-management-and-policy/policy-fair-trade-and-small-business-support)

**解法の思考プロセス**
1. 通常：交際費等は50%損金不算入
2. 中小企業優遇：800万円限度で100%損金算入
3. 対象企業：中小企業基本法の定義を満たす法人
4. 損金算入額 = min(交際費等実績, 800万円)
5. 限度額超過分は通常の50%損金不算入適用

**誤答の落とし穴**
- 限度額の誤認（500万円、1,000万円など）
- 対象企業範囲の誤理解（全企業が対象と勘違い）
- 計算方法の誤り（「50%が損金算入」と「100%が損金不算入」の混同）

**学習アドバイス**
**中小企業向け税制は「実務的メリット」が大きい**ため、出題頻度が高いです。具体例で暗記しましょう。

### 第29問｜経営革新計画

**問題要旨**
経済産業省「経営革新計画」について、計画の要件・認定手続き・支援内容に関する複合設問。

**分類タグ**
K5（制度・データ重視） / [経営支援制度](/sme-management-and-policy/policy-management-support)

**正解**（複数空欄版）
- 要件a：自社またはグループ内で新たに導入する製品・サービスまたは生産工程等
- 要件b：計画初年度から5年以内に、売上高・営業利益率が一定以上増加

**必要知識**
- [経営革新支援制度](/sme-management-and-policy/policy-management-support)

**解法の思考プロセス**
1. 経営革新 = 「新規性」「実現性」「実績性」を兼ねた計画
2. 新規性：新しい経営システム・製品開発等
3. 実現性：3～5年で実現可能な計画
4. 実績性：数値目標（売上高、営業利益率等）設定
5. 支援内容：低金利融資、信用保証優遇、補助金等

**誤答の落とし穴**
- 「革新」の定義の誤解（改善程度では不十分）
- 目標年数の誤認（1年、10年等）
- 数値目標設定の要件理解不足

**学習アドバイス**
**経営革新計画は「実績値を伴う計画書」が評価の鍵**です。単なる改善では認定されません。

**問題要旨**

- b：特例制度 / 経営承継要件

**必要知識**

2. 要件：後継者が事業継続、一定従業者雇用維持等

**学習アドバイス**

## 年度総括

本年度（R7）の中小企業経営・中小企業政策は、以下の特徴が顕著でした。

### 出題傾向の要点

1. **K5（制度・データ）重視**: 総出題数の約40%が統計データ読み取り・制度理解問題
2. **白書・調査報告書からの直結出題**: 経済産業省・厚生労働省の最新報告書が出典
3. **複数空欄補充形式の増加**: 単純選択問題から「複数答案」形式へシフト
4. **実務的な制度理解**: 創業融資、信用保証、補助金、税制など「使える知識」を重視

### 学習上の注意点

**暗記すべき制度の5大分野**

1. **中小企業の定義**: 資本金・従業者数（業種別基準）→ 確実な習得が必須
2. **主要補助金**: ものづくり補助金、地域創生関連 → 上限額・補助率を正確に
3. **信用保証・融資**: 保証額、金利、手続き → 具体例で理解
4. **税制措置**: 交際費、所得拡大促進税制 → 計算方法まで習得
5. **労働施策**: 外国人労働、女性雇用、人手不足対応 → 統計データから読み解く

### 過去問との連動

本年度も**前年度以前の出題分野との重複率は約30%**です。以下の領域は継続学習が重要：

- 企業規模判定（毎年1～2問）
- 統計データ比較問題（毎年3～4問）
- 主要制度の詳細（毎年5～6問）

### 次年度への展開

- **業種別支援制度の拡充**: ものづくり、観光、農業連携など新分野の知識化
- **DX・デジタル化関連施策**: 無形資産投資支援の詳細化
- **カーボンニュートラル関連政策**: グリーン成長戦略との連動
