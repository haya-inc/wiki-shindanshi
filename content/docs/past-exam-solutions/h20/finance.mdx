## 簿記・会計基礎

### 第1問　支店勘定による本支店取引の整理

**問題要旨**: 本店と支店の取引を支店勘定で調整し、期末の支店勘定残高（現金・売掛金・立替金等）を計算する問題。

`K4 手続・手順` `T3 計算実行` `L3` `Trap-E 計算ミス`

**正解**: エ

**必要知識**: [支店独立会計と本支店統一会計](/finance-and-accounting/corporate-accounting-principles-and-consolidation) — 支店勘定の理論、本支店間の現金・債権・債務の相殺処理

**解法の思考プロセス**: 支店勘定は本店側で「支店の資産・負債・利益」をまとめて追跡する勘定です。支店が報告してきた各種取引（売上、仕入、経費、本店への送金など）を集計し、期末の支店勘定残高を求めます。

取引を時系列で追いながら、支店側の現金・売掛金・買掛金などの動きを把握し、本店との相殺項目を除外します。「立替金」（本店が支店の代わりに支払った額）や「発送」（本店から支店への商品送付）は支店勘定の増加要因となります。最終的に支店勘定 = 期末支店現金 + 期末支店売掛金 + その他期末残高の合計として確認できます。

**誤答の落とし穴** `Trap-E 計算ミス`: (1) 本店と支店の現金を単純加算してしまい、相殺調整を忘れる。(2) 立替金の方向（本店が負債か、支店が負債か）を取り違える。(3) 発送額の取扱い（本店の仕入としての控除、支店の在庫への加算）を誤る。(4) 送金額の二重計上（既に支店現金に含まれているのに支店勘定から控除してしまう）。

**学習アドバイス**: 支店勘定は「本店が支店の全取引を一括掌握する」仕組みです。「支店 = 企業内の独立採算部門」と考え、本店との相互取引（送金、商品発送、手形引受など）のたびに支店勘定で記帳します。繰越・当期の増減・振替をていねいに追うことが重要です。

### 第2問　先入先出法による売上原価計算

**問題要旨**: 先入先出法に基づき、複数の仕入日における異なる単価の商品から、指定された売上日の売上原価を計算する問題。

`K3 数式・公式` `T3 計算実行` `L3` `Trap-E 計算ミス`

**正解**: イ

**必要知識**: [棚卸資産評価方法](/finance-and-accounting/bookkeeping-basics-account-categories-and-journal-entries) — 先入先出法（FIFO）の計算手順

**解法の思考プロセス**: 先入先出法では、古い仕入から順に払出（売上原価）に充当します。仕入帳簿から「仕入日・数量・単価」を読み取り、月別・日別の売上数量に対して、最も古い仕入単価から充当していきます。

例えば、7月に90個売上があり、在庫が「6月仕入50個@400円」「7月仕入40個@400円」であれば、6月仕入50個が全て売上原価に充当され、残り40個は7月仕入から使用します。月後半の売上でも同様に古い在庫から充当していくため、期末在庫は最新の仕入商品で構成されます。

**誤答の落とし穴** `Trap-E 計算ミス`: (1) 後入先出法（LIFO）と混淆し、最新の仕入単価から払出す。(2) 移動平均法を意識せず、単純平均単価を使用する。(3) 月ごとの繰越を誤り、通期計算時に月間の在庫推移を反映しない。(4) 売上原価計算式（期首 + 仕入 - 期末 = 売上原価）の確認を忘れ、各売上日の合計と齟齬する。

**学習アドバイス**: 棚卸資産評価方法は「払出順序」の定義が異なるだけです。先入先出法は「古い順」、後入先出法は「新しい順」と単純に分けて考え、在庫の物理的流れと論理的流れを別問題として整理しましょう。

### 第3問　繰延資産の分類と会計処理

**問題要旨**: 繰延資産と無形資産、研究開発費、企業債務といった各項目が繰延資産に該当するかを判定する問題。

`K1 定義・用語` `T1 正誤判定` `L1` `Trap-D 類似混同`

**正解**: エ

**必要知識**: [資産会計](/finance-and-accounting/corporate-accounting-assets-accounting) — 繰延資産の定義、繰延資産と無形資産の区別

**解法の思考プロセス**: 繰延資産は「支出は完了したが、支出額の全部または一部が次期以降に配分される資産」です。代表例は「社債発行差金」「創立費」「開業費」「企業債務繰延処理」です。

一方、無形資産は「特許権、商標権、のれん、ソフトウェア」など、非物質的な企業資産です。研究開発費は原則費用化（例外的に開発段階の支出のみ無形資産化）されます。会計基準の変更により、繰延資産の廃止論もありますが、試験ではまだ出題対象です。

**誤答の落とし穴** `Trap-D 混同誘発`: (1) 繰延資産と無形資産を混淆し、特許権取得費を繰延資産と誤認識する。(2) 研究開発費全体を繰延資産と見なし、費用化の判断を誤る。(3) 企業債務繰延処理（社債発行差金）の必要性（「将来の利息との相殺」「期間配分」）を理解していない。

**学習アドバイス**: 繰延資産の判断基準は「一時的か継続的か」「配分可能性があるか」です。企業債務繰延処理は「社債発行時の手数料や割引を、社債の償還期間にわたって償却する」という背景を理解すると、繰延資産扱いの理由が明確になります。

### 第4問　公債（社債）取得時の経過利息と取得原価

**問題要旨**: 公債を額面より低い価格で購入した際の取得原価（経過利息を含めるか否か）を判定する問題。

`K4 手続・手順` `T3 計算実行` `L2` `Trap-E 計算ミス`

**正解**: エ

**必要知識**: [有価証券の取得と評価](/finance-and-accounting/corporate-accounting-assets-accounting) — 債券取得時の経過利息処理、取得原価の算定

**解法の思考プロセス**: 債券をクーポン日（利息支払日）以外に購入する場合、売却者に対して経過利息を支払う必要があります。この経過利息は「費用」ではなく「取得原価に含める」処理が一般的です。

取得原価 = 購入代金 + 経過利息 + 仲介手数料等。この取得原価をベースに、償却原価法（割引債の場合）または原価法（満期保有目的の場合）で評価されます。利息支払日に受け取る利息（全額）から経過利息分を差し引いた額が当期の利息収入となります。

**誤答の落とし穴** `Trap-E 計算ミス`: (1) 経過利息を「売上益」として当期収入に含める誤り。(2) 経過利息を別途「利息受取」として処理し、取得原価に含めない。(3) 取得原価を購入代金のみとし、後日経過利息分を「雑費用」計上する。(4) 利息支払日に受け取った利息全額を利息収入とし、経過利息の調整を忘れる。

**学習アドバイス**: 債券売買時の経過利息処理は「取得原価に含める → 次回の利息支払日に全額受け取る → 当期利息 = 全額 - 経過利息」という一貫した流れで理解しましょう。

### 第5問　引当金の計上要件と貸借対照表表示

**問題要旨**: 特定の偶発事象（債務の発生可能性、見積金額の確実性など）に基づき、引当金を計上すべきか否かを判定する問題。

`K2 分類・表示` `T1 正誤判定` `L1` `Trap-B 条件見落とし`

**正解**: エ

**必要知識**: [負債と引当金](/finance-and-accounting/corporate-accounting-liabilities-equity-and-tax-effect) — 引当金の計上要件（確実性・見積可能性）、種類別の取扱い

**解法の思考プロセス**: 引当金の計上要件は以下の通りです。

1. **確実性**: 決算日現在で、特定の債務が存在するか、または高い蓋然性をもって発生が予想される
2. **見積可能性**: 債務額を合理的に見積ることができる
3. **過去の実績**: 同種の事象について、過去のデータに基づく実績がある

例えば、「返品の可能性がある売上」「訴訟に敗訴する可能性がある」などは、確実性は低いがいずれ実現する可能性があります。見積額の根拠が「過去の実績」に基づいているかが重要です。

**誤答の落とし穴** `Trap-B 条件すり替え`: (1) 偶発事象の発生確率は高いが、金額が「見積不可能」な場合、引当金を計上できない（注記のみ）。(2) 確実性は高いが、「過去の実績がない」初めての事象の場合、見積額の妥当性が問われる。(3) 過度な引当金を計上し、「隠れた利益」を蓄積するケースを識別できない。

**学習アドバイス**: 引当金は「発生の蓋然性」と「金額の確実性」のバランスで判断します。「発生可能性 70%以上 + 見積可能」が目安ですが、試験では「引当金を計上しない場合の注記義務」も出題されるので、計上線引きを理解しましょう。

### 第6問　キャッシュ・フロー計算書（間接法）

**問題要旨**: 損益計算書と貸借対照表の増減から、営業活動によるキャッシュ・フローを間接法で計算する問題。

`K4 手続・手順` `T3 計算実行` `L3` `Trap-E 計算ミス`

**正解**: エ

**必要知識**: [キャッシュ・フロー計算書](/finance-and-accounting/profit-and-cash-management-cash-flow-and-fcf) — 間接法による営業CF計算（利益調整 + 運転資本調整 + 税金調整）

**解法の思考プロセス**: 間接法の計算順序は次の通りです。

```
税引前当期純利益
+ 減価償却費（利益に含まれるが現金支出なし）
+ 貸倒引当金（同上）
- 売却益（利益に含まれるが営業外）
± 運転資本の変動（売掛金・棚卸資産・買掛金の増減）
= 営業CF（税引前）
- 法人税等の支払額
= 営業CF（税引後）
```

売掛金が減少すれば現金入金、棚卸資産が増加すれば現金支出、買掛金が増加すれば現金支出減（負債の増加）となります。未払法人税等の変動にも注意が必要です。

**誤答の落とし穴** `Trap-E 計算ミス`: (1) 売掛金・棚卸資産の増減方向を逆にする（減少なのに減算、増加なのに加算）。(2) 減価償却費を加算し忘れる。(3) 未払法人税等の増減を誤り、税金支払額を過大または過小に算出する。(4) 当期純利益からスタートして、法人税をもう一度調整する二重計上。

**学習アドバイス**: CF計算書の間接法は「利益 ≠ キャッシュ」を正確に理解することが鍵です。「非資金項目」（減価償却費、貸倒引当金の変動）と「運転資本の変動」の2つの調整を分けて考えると間違いが減ります。

### 第7問　企業結合（取得法）とのれん・少数株主持分

**問題要旨**: 子会社を買収した場合、取得企業が支払対価と子会社の公正価値から、のれんと少数株主持分を計算する問題。

`K4 手続・手順` `T3 計算実行` `L3` `Trap-E 計算ミス`

**正解**: エ

**必要知識**: [企業結合と連結会計](/finance-and-accounting/corporate-accounting-principles-and-consolidation) — 取得法による企業結合会計、のれんと少数株主持分の計算

**解法の思考プロセス**: 取得法では以下のステップで計算します。

```
のれん = 支払対価 - (子会社資産 - 子会社負債)の公正価値
```

例えば、取得企業が子会社に8,000百万円支払い、子会社の総資産の公正価値が12,000百万円、総負債が2,000百万円の場合：

```
のれん = 8,000 - (12,000 - 2,000) = 8,000 - 10,000 = -2,000 → 負のれん
（または：支払額が簿価より低い場合）
```

少数株主持分（取得企業の持分外）は、子会社の公正価値に未取得持分比率を乗じます。全額のれん法と部分のれん法で計算が異なります。

**誤答の落とし穴** `Trap-E 計算ミス`: (1) のれん計算時に「子会社の簿価」と「公正価値」を混淆する。(2) 少数株主持分の計算で、「公正価値」か「簿価」かを誤り、または持分比率を逆にする（持有比率70% = 少数株主30%）。(3) 全額のれん法と部分のれん法を意識せず、不正確な計算をする。(4) 被取得企業側の「固有資産」（特許、ライセンス）の公正価値を見落とし、単純に簿価での計算となってしまう。

**学習アドバイス**: 企業結合会計は「①取得企業と被取得企業の特定」「②対価の計算」「③公正価値測定」の3ステップで整理します。試験では②と③の計算問題が頻出です。

### 第8問　税効果会計（繰延税金資産・負債）

**問題要旨**: 会計利益と税務利益のズレから繰延税金資産・負債を計算し、法人税等調整額を求める問題。

`K4 手続・手順` `T3 計算実行` `L2` `Trap-E 計算ミス`

**正解**: イ

**必要知識**: [税効果会計](/finance-and-accounting/corporate-accounting-liabilities-equity-and-tax-effect) — 繰延税金資産・負債の計算、オフセット原則

**解法の思考プロセス**: 税効果会計は「会計利益」と「税務利益」の差を調整し、適切な法人税負担を反映します。

```
会計上の利益（税引前）> 税務上の利益 → 繰延税金負債（将来、税金多く払う）
会計上の利益（税引前）< 税務上の利益 → 繰延税金資産（将来、税金少なく払う）
```

例えば、減価償却費の計上方法が会計と税務で異なる場合、会計では加速償却で利益が低い年度に、繰延税金資産（将来の税軽減）が発生します。

繰延税金資産と負債は相殺可能（オフセット原則）であり、同じB/S区分に属する場合に限り相殺表示できます。

**誤答の落とし穴** `Trap-E 計算ミス`: (1) 繰延税金資産と負債の相殺ルール（オフセット原則）を知らず、個別に記載する。(2) 一時差異と永久差異を混淆し、永久差異（例：交際費の定額控除）に対しても繰延税金を計上する。(3) 税率改正のタイミングで、新旧税率の適用を誤る。(4) 繰延税金の回収可能性（特に資産について）を評価し忘れる。

**学習アドバイス**: 税効果会計は「会計と税務の差 = 一時差異」「一時差異 × 税率 = 繰延税金額」という単純な構造です。各業種別（減価償却、引当金、未払い）の一般的な差異パターンを整理しておくと、計算が速くなります。

### 第9問　労務費の発生額計算

**問題要旨**: 給与支払や未払金の増減から、当期に発生した労務費の総額を逆算する問題。

`K4 手続・手順` `T3 計算実行` `L2` `Trap-B 条件見落とし`

**正解**: イ

**必要知識**: [費用・収益の認識](/finance-and-accounting/bookkeeping-basics-adjusting-entries) — 発生主義に基づく労務費計算、未払い・前払いの調整

**解法の思考プロセス**: 発生主義では、現金支払ベースではなく、当期に**発生した**労務費を計上します。

```
当期発生額 = 期末未払金 + 支払額 - 期首未払金
```

例えば、期首未払金が3,600千円、当期支払額が11,100千円、期末未払金が2,500千円の場合：

```
発生額 = 2,500 + 11,100 - 3,600 = 10,000千円
```

この10,000千円が当期の労務費として損益計算書に計上されます。

**誤答の落とし穴** `Trap-B 条件すり替え`: (1) 支払額をそのまま当期費用とし、未払金の増減を調整しない（当期費用 = 11,100と誤認識）。(2) 期末未払金を「費用の追加」と理解せず、期首のみを調整する。(3) 給与・賞与・福利厚生費など複数の労務費項目を統合すべきなのに、一部のみで計算する。(4) 給与は労務費だが、役員賞与は別途処理（経常利益控除の可能性）する基準を曖昧にする。

**学習アドバイス**: 未払金の調整は「当期の義務」を反映させるためです。「支払った ≠ 費用化」を常に意識し、毎月の給与発生時に未払金を計上する仕訳パターンを習得しましょう。

## 経営分析・指標

### 第10問　経営指標の定義と読解

**問題要旨**: 経営分析指標（ROA、流動比率、当座比率、固定比率、自己資本比率など）の定義と改善方向を正確に理解しているかを問う問題。

`K1 定義・用語` `T1 正誤判定` `L1` `Trap-D 類似混同`

**正解**: ア

**必要知識**: [経営分析の指標体系](/finance-and-accounting/management-analysis-profitability) — 収益性・効率性・安全性指標の定義、改善方向

**解法の思考プロセス**: 経営指標は「定義（分子・分母）」と「改善方向（上昇/下降のどちらが改善か）」の2点がセットです。

- **ROA（総資産利益率）** = 当期純利益 / 総資産 → 上昇で改善
- **流動比率** = 流動資産 / 流動負債 → 上昇で改善
- **当座比率** = 当座資産 / 流動負債 → 上昇で改善
- **固定比率** = 固定資産 / 自己資本 → 下降で改善（低いほど良い）
- **売上債権回転期間** = 売掛金 / (売上 / 365) → 下降で改善（資金化速度向上）

**誤答の落とし穴** `Trap-D 混同誘発`: (1) 分子・分母を逆にする（例：固定比率 = 自己資本 / 固定資産と誤認識）。(2) 改善方向を逆に覚える（例：固定比率は「上昇で改善」と誤解）。(3) 類似指標を混淆（例：流動比率と当座比率の違いが分からない）。(4) 業界・時期によって「改善」の評価が異なることを見落とす（例：成長期は高いROAが必ずしも効率的とは限らない）。

**学習アドバイス**: 各指標の定義は**試験3週間前に暗唱できるレベル**に仕上げましょう。改善方向は「分子が増える = どのような経営改善か」を文字で追うと、正確に判断できます。

### 第11問　指標の時系列改善と経営診断

**問題要旨**: 過去3年間の指標推移から、企業の財務健全性の改善・悪化を判定し、改善の原因を特定する問題。

`K3 数式・公式` `T2 分類判断` `L2` `Trap-A 逆方向`

**正解**: イ

**必要知識**: [安全性分析](/finance-and-accounting/management-analysis-safety) — 流動性指標、健全性指標の改善メカニズム

**解法の思考プロセス**: 複数年度の指標を比較する際は、「各指標の計算式」と「分子・分母の変動」を並行して追跡します。

例えば、「流動比率が2年連続で上昇している」原因として、①流動資産の増加、②流動負債の減少、③その組み合わせが考えられます。その原因が「売上回復による流動資産増」か「借入返済による負債減」かで、経営の質が異なります。

**誤答の落とし穴** `Trap-A 逆方向誘発`: (1) 「固定比率の改善」と「悪化」の方向を逆に判定する（固定比率は「下降で改善」なのに「上昇を改善」と誤解）。(2) 「売上債権回転期間の短縮」を「売上減少」と混淆する（短縮 = 資金化速度向上で、通常は売上増加か回収効率向上の結果）。(3) 指標が改善しても「経営の実態が改善」とは限ないことを見落とす（例：売却益で一時的に利益増、手持ちの固定資産売却で流動資産増）。

**学習アドバイス**: 指標の時系列分析では「数値の改善 = 経営改善」とは限りません。**根底にある現金フロー、事業の質的変化**を合わせて診断することが実務的です。

### 第12問　CVP分析と目標利益の販売量計算

**問題要旨**: 固定費、変動費率、目標利益から、達成に必要な販売量またはマージンを計算する問題。

`K3 数式・公式` `T3 計算実行` `L3` `Trap-E 計算ミス`

**正解**: イ

**必要知識**: [CVP分析](/finance-and-accounting/management-analysis-cvp-and-break-even) — 損益分岐点計算、目標利益達成販売量、限界利益率

**解法の思考プロセス**: CVP分析の基本公式は：

```
営業利益 = 売上高 × 限界利益率 - 固定費
目標利益達成売上 = (固定費 + 目標利益) / 限界利益率
目標利益達成販売量 = (固定費 + 目標利益) / 限界利益単価
```

まず売上・変動費から単価あたりの限界利益を求め、限界利益率を算出します。その後、固定費と目標利益の合計を限界利益率で割ることで、必要な売上高が得られます。

**誤答の落とし穴** `Trap-E 計算ミス`: (1) 限界利益率と変動費率を混淆し、計算式を逆にする（例：目標利益 / 変動費率で計算）。(2) 固定費と変動費の分類を誤り、不要な項目を含める。(3) 「目標利益1個当たり150円」を「総額150円」と読み違える。(4) 販売量と売上高を取り違える（単価が変わる場合）。(5) 損益分岐点と目標利益達成点の計算方法を混淆する。

**学習アドバイス**: CVP分析は「費用の固定/変動分離 → 限界利益算出 → 目標設定」の3ステップです。特に、限界利益率の計算（営業利益ではなく、売上 - 変動費）に誤りやすいので、毎回確認しましょう。

### 第13問　株式評価指標（PER、PBR）

**問題要旨**: 株価、1株当たり利益（EPS）、1株当たり純資産（BPS）から、PER、PBRを計算し、株式の相対的評価を判定する問題。

`K3 数式・公式` `T3 計算実行` `L2` `Trap-D 類似混同`

**正解**: エ

**必要知識**: [株式価値評価](/finance-and-accounting/finance-multiples-valuation) — PER、PBRの定義と解釈

**解法の思考プロセス**:

```
PER（株価収益率） = 株価 / EPS
PBR（株価純資産倍率） = 株価 / BPS
```

PERが低い（例：15倍）企業は、1円の利益を生むのに15円の株価が付いている（割安の可能性）。PERが高い（例：30倍）企業は、成長期待が高い、または過熱気味。

PBRが1倍未満（例：0.8倍）は簿価より割安（解散価値より安い可能性や、低収益性の警告）。PBRが1倍以上（例：1.5倍）は簿価より割高（収益性への評価）。

**誤答の落とし穴** `Trap-D 混同誘発`: (1) PERとPBRを逆に定義（例：PER = 株価 / BPS）。(2) 「PER低い = 常に割安」と単純判定し、低迷企業との区別ができない。(3) 業界平均との比較を忘れ、単一年度の数値のみで評価する。(4) 株価の時点（期首・期末・平均）とEPS・BPSの時点（期首・期末自己資本）のミスマッチ。

**学習アドバイス**: 株式評価は「相対的評価（PER・PBR）」と「絶対的評価（DCF法）」の2つの方法があります。試験では相対評価（マルチプル法）の計算が頻出ですが、成長率や配当政策を反映できない限界を理解しておきましょう。

### 第14問　資本コスト（WACC）の計算

**問題要旨**: 自己資本コスト、負債コスト、資本構成から、加重平均資本コスト（WACC）を計算する問題。

`K3 数式・公式` `T3 計算実行` `L2` `Trap-E 計算ミス`

**正解**: イ

**必要知識**: [資本構成](/finance-and-accounting/finance-mm-and-dividend-policy) — WACC = (E/(D+E)) × Re + (D/(D+E)) × Rd × (1-t)

**解法の思考プロセス**:

```
WACC = ウェイト(自己資本) × 自己資本コスト + ウェイト(負債) × 負債コスト（税引後）
```

例えば、自己資本200、負債300の場合：

```
E/(D+E) = 200/500 = 0.4
D/(D+E) = 300/500 = 0.6
WACC = 0.4 × 10% + 0.6 × 5% × (1-0.3)
     = 4% + 0.6 × 3.5%
     = 4% + 2.1%
     = 6.1%
```

**誤答の落とし穴** `Trap-E 計算ミス`: (1) 負債コストの税調整を忘れ、税引前のまま使用する（Rd × (1-t) を計算しない）。(2) D/E比率から加重への変換を誤る（D/E = 0.6 のとき、D/(D+E) = 0.6 ではなく、0.6 / 1.6 ≒ 0.375）。(3) 自己資本と負債の単位（円 vs 百万円）の統一忘れ。(4) 名目コストと実質コスト、インフレーション調整の混淆。

**学習アドバイス**: WACC計算は「D/E比から加重への変換」が最大の引っかかりです。公式「D/(D+E) = D/E / (1 + D/E)」を覚えておくと、計算が速くなります。

### 第15問　社債の資本コスト（利回り）

**問題要旨**: 社債の発行条件（表面利率、発行価格、満期年数）から、社債の利回り（YTM）を推定する問題。

`K3 数式・公式` `T3 計算実行` `L2` `Trap-E 計算ミス`

**正解**: ア

**必要知識**: [社債の評価](/finance-and-accounting/finance-mm-and-dividend-policy) — 社債利回り、イールドスプレッド

**解法の思考プロセス**: 社債の理論価格は、クーポン（利息）と満期時の返済を割引いた現在価値です。

```
社債価格 = Σ(クーポン × 複利現価係数) + 額面 × 複利現価係数(満期)
```

発行価格と表面利率から、実際の利回り（YTM）を逆算するには、試行錯誤または利回り関数を使用します。

例：表面利率3%、発行価格98円、満期5年の場合、YTMは3%より高くなる（価格が額面より低いため）。

**誤答の落とし穴** `Trap-E 計算ミス`: (1) 表面利率（クーポンレート）と実際の利回り（YTM）を混淆する。(2) 発行価格の割増・割引による利回りへの影響を誤る（価格 < 額面 ⟹ 利回り > 表面利率）。(3) 期間中のスプレッド変動（信用リスク変動）を無視する。(4) 複利現価係数の適用を誤り、等比数列の合計公式を正確に使用しない。

**学習アドバイス**: 社債の資本コストは「無リスク資産の利回り + 信用スプレッド」で構成されます。企業の格付けが低下すれば、スプレッドが拡大し、利回りが上昇します。このメカニズムを理解するとYTM計算の背景が明確になります。

### 第16問　NPVとIRRによる投資意思決定

**問題要旨**: 初期投資、毎期キャッシュフロー、割引率から、NPV法とIRR法で投資案を評価し、採否判定の組み合わせを求める問題。

`K3 数式・公式` `T3 計算実行` `L3` `Trap-E 計算ミス`

**正解**: ウ

**必要知識**: [投資意思決定基準](/finance-and-accounting/finance-profitability-index) — NPV法（正味現在価値）、IRR法（内部収益率）、資本コストとの比較

**解法の思考プロセス**:

```
NPV = Σ(CF / (1+r)^t) - I₀
NPVが正 ⟹ 採択、NPVが負 ⟹ 不採択

IRR: Σ(CF / (1+IRR)^t) = I₀ を満たす IRR を求める
IRR > 資本コスト ⟹ 採択、IRR < 資本コスト ⟹ 不採択
```

例：初期投資1,000、毎期CF300、5年間、資本コスト10%の場合：

```
NPV = 300/(1.1) + 300/(1.1)^2 + ... + 300/(1.1)^5 - 1,000
    = 300 × 3.791 - 1,000
    = 1,137 - 1,000
    = 137 > 0 ⟹ 採択
```

**誤答の落とし穴** `Trap-E 計算ミス`: (1) 複利現価係数の適用を誤る（年金現価係数との混淆）。(2) 初期投資の時点（年0 vs 年1）を誤る。(3) NPVが正なのにIRR < 資本コストという矛盾を選んでしまう（両指標は通常一致）。(4) 永続年金（CF / r）とIRR（CF / IRR）の公式を取り違える。

**学習アドバイス**: NPV法とIRR法は**独立投資案の採否判定では常に同じ結論**になります。矛盾した結論の組み合わせを選んでいないか、最終確認をしましょう。

### 第17問　投資プロジェクトの経済性評価（機会原価）

**問題要旨**: 投資案の評価に際し、埋没原価（過去の支出）、機会原価（代替案での利益）、カニバリゼーション（既存事業への悪影響）の考慮要否を正誤判定する問題。

`K4 手続・手順` `T1 正誤判定` `L2` `Trap-B 条件見落とし`

**正解**: ア

**必要知識**: [投資意思決定の関連CF](/finance-and-accounting/finance-profitability-index) — 意思決定に関連するCF、埋没原価、機会原価の定義

**解法の思考プロセス**: 投資評価には「意思決定に関連するCFのみ」を含めます。

- **埋没原価**（サンクコスト）：過去に支出済み → **含めない**
- **機会原価**：代替案を選ばなかったことによる逸失利益 → **含める**
- **カニバリゼーション**：新事業導入による既存事業への負の影響 → **含める**

例えば、「既に購入済みの土地をプロジェクトに利用」する場合、土地購入費は埋没原価（除外）ですが、その土地を売却すれば得られたはずの売却益は機会原価（含める）です。

**誤答の落とし穴** `Trap-B 条件すり替え`: (1) 機会原価の概念を見落とし、「実際に支払っていないもの」を意思決定から除外する。(2) 埋没原価と機会原価を混淆し、「今後支出しない = 含めない」と誤解する。(3) 「事業の乗り換え効果」（例：新製品で既存製品の売上が減る）を無視する。(4) 既存設備の残存簿価と売却可能額を区別しない。

**学習アドバイス**: 投資評価の鉄則は「**過去の支出は除外、将来の差分を含める**」です。「新事業を始めるなら、既存設備を売ったら」という選択肢があれば、売却益を含めるべきです。

### 第18問　リスク・リターンの関係と資産分類

**問題要旨**: 複数の資産（株式、社債、国債）について、期待リターンと標準偏差（リスク）の関係から、リスク回避的投資家の選択肢を判定する問題。

`K1 定義・用語` `T2 分類判断` `L1` `Trap-D 類似混同`

**正解**: ア

**必要知識**: [ポートフォリオ理論](/finance-and-accounting/finance-mm-and-dividend-policy) — リスク選好、無差別曲線、効率的フロンティア

**解法の思考プロセス**: 投資家の資産選択は「期待リターン」と「リスク（標準偏差）」のバランスで決まります。

- **リスク回避的投資家**：同じリターンなら、リスクが低い資産を選ぶ
- **リスク中立的投資家**：リターンだけで判断、リスクは無視
- **リスク愛好的投資家**：同じリターンなら、リスクが高い資産を選ぶ

例：資産A（リターン10%、標準偏差5%）、資産B（リターン10%、標準偏差8%）がある場合、リスク回避的投資家はAを選びます。

**誤答の落とし穴** `Trap-D 混同誘発`: (1) 「リスク回避」を「リスク嫌悪」と混淆し、行動を誤る。(2) 「分散投資 = リスク回避」と単純化し、個別資産の選択基準を見落とす。(3) 効率的フロンティア（最適なリスク・リターン組み合わせ）を理解していない。

**学習アドバイス**: ポートフォリオ理論は「リスクとリターンのトレードオフ」を表現しています。投資家の無差別曲線と効率的フロンティアの接点が最適ポートフォリオです。

### 第19問　CAPM（資本資産価格モデル）による自己資本コスト

**問題要旨**: リスクフリーレート、市場リスクプレミアム、ベータ係数から、企業の自己資本コストをCAPMで計算する問題。

`K3 数式・公式` `T3 計算実行` `L3` `Trap-E 計算ミス`

**正解**: イ

**必要知識**: [資本コスト計算](/finance-and-accounting/finance-mm-and-dividend-policy) — CAPM：Re = Rf + β(Rm - Rf)

**解法の思考プロセス**:

```
Re = Rf + β × (Rm - Rf)

Re：自己資本コスト
Rf：リスクフリーレート（例：国債利回り3%）
Rm：市場ポートフォリオの期待収益率
β：当該企業のベータ値（市場全体の変動に対する感応度）
```

例：Rf = 3%、Rm = 8%、β = 1.2 の場合：

```
Re = 3% + 1.2 × (8% - 3%)
   = 3% + 1.2 × 5%
   = 3% + 6%
   = 9%
```

**誤答の落とし穴** `Trap-E 計算ミス`: (1) ベータの意味を誤解し、単位を無視する（β > 1 なら市場より変動性大）。(2) 市場リスクプレミアム（Rm - Rf）を Rm と混淆する。(3) ベータを負と見なすケース（値下がり期待の企業）の処理。(4) インフレーション調整の必要性を見落とす。

**学習アドバイス**: CAPMは「リスクフリーレート + リスクプレミアム」という直感的な構造です。ベータ値が1.0なら市場平均と同じ変動性、1.0より大きければ市場より変動性が高い（ハイリスク・ハイリターン企業）、と理解しましょう。

### 第20問　配当政策と株主価値

**問題要旨**: 配当方針（一定額 vs 業績連動型）の違いが、配当性向、1株当たり配当額、株主価値に与える影響を正誤判定する問題。

`K1 定義・用語` `T1 正誤判定` `L1` `Trap-A 逆方向`

**正解**: エ

**必要知識**: [配当政策](/finance-and-accounting/finance-mm-and-dividend-policy) — 配当政策の類型と特徴

**解法の思考プロセス**: 配当方針の効果を整理します。

| 方針 | 配当額 | 配当性向 | 株価への影響 |
| --- | --- | --- | --- |
| **一定額配当** | 安定 | 変動大 | 安心感、業績悪化で配当カット可能性 |
| **業績連動型** | 変動大 | 安定 | 利益成長を株主に還元、変動大 |
| **残余配当** | 不安定 | 安定（目標値） | リスク大 |

**誤答の落とし穴** `Trap-A 逆方向誘発`: (1) 「配当性向が安定」と「配当額が安定」を逆に捉える。一定額配当では配当額が安定し配当性向が変動、業績連動型では配当性向が安定し配当額が変動です。(2) 「配当 > 利益」の赤字企業における配当政策の評価（減配すべき）。(3) MM理論（配当政策は企業価値に中立）と実務的な企業価値への影響の混淆。

**学習アドバイス**: 配当政策は「企業キャッシュフロー → 配当 → 内部留保」の資金配分問題です。成長期は留保、成熟期は配当という選択も重要です。

### 第21問　企業価値評価（DCF法と配当割引モデル）

**問題要旨**: 複数年の予測CF、継続価値、割引率から、企業価値（株主価値）を計算する問題。

`K3 数式・公式` `T3 計算実行` `L3` `Trap-E 計算ミス`

**正解**: ウ

**必要知識**: [企業価値評価](/finance-and-accounting/finance-multiples-valuation) — DCF法（フリーキャッシュフロー）、継続価値、割引現在価値

**解法の思考プロセス**:

```
企業価値 = Σ(予測年度のFCF / (1+WACC)^t) + (継続価値 / (1+WACC)^n)

継続価値 = 最終年度のFCF × (1+永続成長率) / (WACC - 永続成長率)
```

例：年1～3のFCFが100・120・150、年4以降永続成長率3%、WACCが8%の場合：

```
継続価値（年3末） = 150 × 1.03 / (0.08 - 0.03) = 154.5 / 0.05 = 3,090

企業価値 = 100/1.08 + 120/1.08^2 + 150/1.08^3 + 3,090/1.08^3
```

**誤答の落とし穴** `Trap-E 計算ミス`: (1) 継続価値を「最終年度FCF / (WACC - g)」で計算し、成長を反映しない。(2) 割引年数を誤える（「年3以降」と「年4以降」の違い）。(3) FCFと配当を混淆（配当割引モデルではなくFCF法を使う）。(4) WACCの計算誤り、または税効果の反映忘れ。

**学習アドバイス**: DCF法は「2段階モデル（予測期間 + 継続価値）」が基本です。「継続価値は予測期間の末尾に立ち、そこからさらに現在へ割り引く」という2段階を確実に踏むことが鍵です。

### 第22問　資本構成（負債・株式）とMM理論

**問題要旨**: 企業が全額株式で調達する場合と、負債と株式の組み合わせで調達する場合の企業価値を比較し、MM理論（租税なし）の適用可否を判定する問題。

`K4 手続・手順` `T1 正誤判定` `L2` `Trap-C 部分正解`

**正解**: イ

**必要知識**: [MM理論と資本構成](/finance-and-accounting/finance-mm-and-dividend-policy) — MM理論（命題I）：企業価値は資本構成に無関係

**解法の思考プロセス**: MM理論（完全市場を仮定）では：

```
企業価値（負債あり） = 企業価値（全株式）

つまり：V_L = V_U

ただし、この等式が成立するのは「法人税がない場合」に限ります。
```

実務では法人税が存在するため、負債の利息が損金算入可能（税盾）となり、負債調達が有利になります。

**誤答の落とし穴** `Trap-C 部分正解`: (1) 「法人税なしのMM理論」と「法人税考慮のMM理論」を混淆する。(2) 「資本構成は企業価値に無関係」を過度に単純化し、実務における負債の利点を見落とす。(3) 「負債が増えると財務リスク（破綻リスク）が増加」という現実を無視する。(4) 株主の要求リターンが資本構成に応じて変化すること（レバレッジ効果）を見落とす。

**学習アドバイス**: MM理論は「完全で効率的な市場」という強い仮定のもとで成り立つ理論です。現実では摩擦（税、取引コスト、破綻リスク）があり、最適な資本構成が存在します。

### 第23問　MM理論と法人税（税盾効果）

**問題要旨**: 法人税が存在する場合、負債による利息の税盾効果を考慮した企業価値を計算する問題。

`K4 手続・手順` `T1 正誤判定` `L2` `Trap-C 部分正解`

**正解**: ア

**必要知識**: [税盾効果](/finance-and-accounting/finance-mm-and-dividend-policy) — MM命題II（税考慮）：企業価値 = 無負債企業価値 + 法人税率 × 負債額

**解法の思考プロセス**:

```
V_L = V_U + t × D

V_U：負債がない場合の企業価値
t：法人税率
D：負債額
t × D：税盾の価値
```

例：無負債企業価値1,000、負債300、法人税率30%の場合：

```
税盾価値 = 0.3 × 300 = 90
企業価値（負債あり） = 1,000 + 90 = 1,090
```

負債の利息は税務上の損金となり、法人税が減少するため、税盾効果が生じます。

**誤答の落とし穴** `Trap-C 部分正解`: (1) 法人税率を誤ったレートで適用する。(2) 税盾価値を「負債額そのもの」と見なし、t を乗じない。(3) 「税盾があるなら負債が多いほど良い」と単純化し、破綻リスクを無視する。(4) 税盾の現在価値計算に誤りがある（成長率がある場合）。

**学習アドバイス**: 税盾効果は「負債 → 利息支払 → 税務控除 → 税金減少」という仕組みです。法人税率が高い国ほど税盾効果は大きくなり、負債による資金調達が相対的に有利になります。

### 第24問　最適資本構造と企業価値最大化

**問題要旨**: 異なる負債比率における企業価値、株主価値、加重平均資本コストを計算し、企業価値を最大化する資本構成を判定する問題。

`K3 数式・公式` `T3 計算実行` `L2` `Trap-E 計算ミス`

**正解**: エ

**必要知識**: [最適資本構造](/finance-and-accounting/finance-mm-and-dividend-policy) — 負債比率とWACC、企業価値の関係

**解法の思考プロセス**: 最適資本構造を求めるため、複数の負債比率でWACCを計算し、最小のWACCをもたらす負債比率を特定します。

```
WACC = (E/(D+E)) × Re(負債比率) + (D/(D+E)) × Rd × (1-t)
```

負債比率が低い場合、WACCは自己資本コスト（Re）に近い。負債比率が高くなると、負債コストは低いが、株主のリスクが増加（Re上昇）。これら2つの効果がバランスする負債比率でWACCが最小となり、企業価値が最大になります。

**誤答の落とし穴** `Trap-E 計算ミス`: (1) 異なる負債比率での自己資本コストの変化を無視し、Re を固定と見なす。(2) WACC計算時に複合効果（負債増加 → Re上昇 + 負債コスト低下）を正確に反映しない。(3) 企業価値（PV）が最大のとき、WACC（割引率）が最小であることを理解していない。(4) 破綻リスクが高まる負債比率での計算エラー。

**学習アドバイス**: 最適資本構造の問題は「WACCを複数の負債比率で計算 → グラフを描く → 最小点を特定」という手順で解きます。試験では計算量が多いため、複利現価係数表を活用し、計算ミスを減らしましょう。

### 第25問　ファイナンス・リース会計

**問題要旨**: ファイナンス・リースの会計処理（固定資産の計上、減価償却、利息費用の計上）と適切な表示方法を判定する問題。

`K2 分類・表示` `T1 正誤判定` `L1` `Trap-D 類似混同`

**正解**: イ

**必要知識**: [リース会計](/finance-and-accounting/corporate-accounting-assets-accounting) — ファイナンス・リース、オペレーティング・リースの定義と会計処理

**解法の思考プロセス**: ファイナンス・リース（本質的には購入と同じ）とオペレーティング・リース（利用権のレンタル）の会計処理は異なります。

**ファイナンス・リース**:
- 固定資産として計上（リース資産）
- 長期借入金として負債計上（リース債務）
- 毎期、減価償却費と利息費用を計上
- B/S：資産 = 負債 + 純資産

**オペレーティング・リース**:
- 資産・負債として計上しない（オフバランス）
- リース料を賃借料として費用計上

**誤答の落とし穴** `Trap-D 混同誘発`: (1) ファイナンス・リースとオペレーティング・リースを混淆し、オペレーティングリースを固定資産として計上する。(2) リース資産の減価償却とリース債務の利息計上の方法を誤る。(3) 「リースは必ず固定資産」と単純化し、オペレーティング・リースの定義を見落とす。(4) リース開始日と減価償却期間の計算誤り。

**学習アドバイス**: ファイナンス・リースは「本質は購入」という原則で判断します。リース開始時に固定資産と負債を同額計上し、その後の減価償却と利息計上で両者の差が生じていきます。

| **L1 基礎知識** | 定義や基本公式の理解で解ける |
| **L2 応用理解** | 複数の概念を組み合わせた理解が必要 |
| **L3 計算応用** | 複数ステップの計算手順と条件管理が必要 |
| **Trap-A 逆方向** | 因果関係の向きが逆の選択肢で誘導 |
| **Trap-B 条件見落とし** | 重要な前提条件を見落とさせる |
| **Trap-C 部分正解** | 一部正しいが全体として誤った選択肢 |
| **Trap-D 類似混同** | 類似概念を混淆させる選択肢 |
| **Trap-E 計算ミス** | 計算過程での誤りを誘発する選択肢 |
