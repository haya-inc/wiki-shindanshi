## 経営戦略・計画

### 第1問　経営計画技法
**問題要旨**: 経営計画を策定する際の技法に関する説明で、最も不適切なものを選択

`K3 数式・公式` `T1 正誤判定` `L2` `Trap-D 混同誘発`

**正解**: ウ

**必要知識**: 経営計画技法（ABM、DCF、ガントチャート、線形計画法、待ち行列理論）

**解法の思考プロセス**:
1. ABM = 活動基準原価計算 ✓（間接費管理）
2. DCF = 割引キャッシュフロー法 ✓（投資評価）
3. ガントチャート → 問題文ではコンピュータを活用してクリティカル・パスを明らかにすると説明。しかし実際にはガントチャートは時系列で工程を表示するだけで、CPMは別の手法。コンピュータを活用するのはCPMではなく、ガントチャートの欠点を補うため ×
4. 線形計画法 = 制約条件下で目的関数を最適化 ✓
5. 待ち行列理論 → 輸送と在庫管理ではなく、サービスシステムの待ち行列を分析 ✓

**誤答の落とし穴**: ガントチャートはPERT/CPMと混同しやすい。ガントチャート自体は単なる視覚化ツール。

**学習アドバイス**: 各経営技法の正確な用途と適用場面を整理。特にガントチャートとCPM/PERTの違いを明確に。

### 第2問　リソース・ベースド・ビュー（RBV）
**問題要旨**: 経営資源と企業戦略に関する記述で、最も不適切なものを選択

`K2 グラフ形状` `T1 正誤判定` `L2` `Trap-A 逆方向誘発`

**正解**: エ

**必要知識**: RBV、競争優位の源泉、経営資源の特性（希少性、模倣困難性、代替困難性、価値）

**解法の思考プロセス**:
1. 経営資源を保有しない企業は複製困難 → 逆。保有していないなら自社で開発可能 ✓理解が必要
2. 経営資源の能力は歴史的経緯に依存 → 経路依存性。先行企業優位 ✓
3. 経営資源の関係が不明確 → tacit knowledge、複雑性。個別に分離困難 ✓
4. **外部調達可能性が高く、調達コストが低い → 競争優位が失われる** ← これが正しい。エは逆を述べている ×
5. 希少性だけでは競争優位 ≠ 顧客価値との合致が必要 ✓

**誤答の落とし穴**: エは「調達コストが低い = コスト優位」という表面的な論理。実際には容易に入手できるリソースは競争優位の源泉にならない。

**学習アドバイス**: RBVの4要件（VRIO）。競争優位持続性は模倣困難性と代替困難性が重要。

### 第3問　戦略グループ
**問題要旨**: 戦略グループの形成と特性に関する説明で、最も不適切なものを選択

`K2 グラフ形状` `T1 正誤判定` `L2` `Trap-D 混同誘発`

**正解**: イ

**必要知識**: 戦略グループの定義、参入障壁、グループ内外の競争構造

**解法の思考プロセス**:
1. 垂直統合強化 → 生産体制・製品ラインが似通う → 戦略グループ発生 ✓
2. **戦略グループ形成後、グループ間移動は難しい。グループ内競争は緩和** ← グループ内は高い競争 ×
3. 顧客層・製品ライン最適化 → 一貫した戦略 → 似通った企業 ✓
4. 複数グループ存在 = 市場広がり、製品ラインが異なる ✓
5. グループ間収益差 = 脅威・機会が異なる ✓

**誤答の落とし穴**: 戦略グループ内は最も激しい競争。グループ間では参入障壁があり競争が弱い（イとウの逆）。

**学習アドバイス**: 戦略グループの定義：「同一産業内で似通った戦略を採る企業群」。グループ内競争 > グループ間競争の構造。

### 第4問　成長戦略（Ansoff）
**問題要旨**: 企業の成長戦略に関する記述で、最も適切なものを選択

`K2 グラフ形状` `T2 グラフ読解` `L2` `Trap-多角化誤解`

**正解**: ア

**必要知識**: 成長戦略の類型（市場浸透、新製品開発、新市場開発、多角化）、リストラクチャリング

**解法の思考プロセス**:
1. **自社優位分野への参入防止 → 市場支配力強化** ← 防衛的戦略で成長に直結しないが、収益性向上に有効 → 比較判断が必要
2. R&D不足 → 技術導入 → 低価格戦略のみ ← 多様な戦略選択肢あり。技術導入後も差別化可能 ×
3. 多角化 → 総花的 → 競争優位喪失 → 収益悪化 ← 適切な多角化なら収益向上も可能 ×
4. リストラクチャリング → 強み喪失 ← むしろ事業構造最適化で強み強化。競争激化時にも実施 ×

**誤答の落とし穴**: ウは多角化を一概に悪と評価しているが、戦略的多角化は有効。エはリストラクチャリングを悪と決めつけている。

**学習アドバイス**: 成長戦略の選択は市場環境と企業資源のマッチング。多角化も適切な範囲なら有効。リストラクチャリングは競争激化時に必須。

---

### 第5問　技術イノベーションと戦略
**問題要旨**: 技術イノベーションと戦略の関係で、最も不適切なものを選択

`K3 数式・公式` `T2 グラフ読解` `L2` `Trap-因果関係誤認`

**正解**: ア

**必要知識**: 技術進化パス、顧客ニーズとの適合、disruptive technology、技術的優位 ≠ 市場優位

**解法の思考プロセス**:
1. **開発技術が支持を受けるほど、その後の技術発展方向が制約** ← 経路依存性で優位喪失リスク低下。実際には逆。制約が強いと技術固定化し衰退 ×
2. 技術優位 ≠ 市場ニーズ → ダーウィンの海。先端技術製品 < 劣る製品 ✓（実例：VHS vs Betamax）
3. 技術拡散速い → 優位性失いやすい。ただし先発者利得・累積生産で維持可能 ✓
4. 市場ニーズ適合技術 → 成長に貢献 → 革新的技術が新市場形成で急速に市場喪失 ✓
5. 暗黙知は革新的技術に適用困難 → 問題文後半を確認

**誤答の落とし穴**: アの「制約されやすく...優位失われる」という説は、むしろ優位が強化される方向（lock-in効果）を誤解している。

**学習アドバイス**: 技術進化の経路依存性。一度技術路線が定着すると、それ以外の方向への転換が困難（技術的ロック・イン）。これにより競争優位が維持される側面と、新技術への対応が遅れるリスク両立。

## 組織設計・実行

### 第6問　セル生産方式
**問題要旨**: セル生産方式の特性に関する説明で、最も不適切なものを選択

`K2 グラフ形状` `T1 正誤判定` `L2` `Trap-部分的正確性`

**正解**: オ

**必要知識**: セル生産、多能工化、柔軟性、熟練労働の必要性

**解法の思考プロセス**:
1. 多能工化 → 手待ち無駄排除 ✓
2. ワークセル単位のチーム生産 ✓
3. 製品多様化・変化速い → ライン切り替えコスト削減 ✓
4. 単調作業排除 → モチベーション向上 → 生産性改善 ✓
5. **ベルトコンベア撤去した熟練労働のみ → 海外展開困難** ← セル生産方式の利点は多能工化で汎用性向上。むしろ半熟練労働でも対応可能。むしろ柔軟に対応できるため海外展開可能 ×

**誤答の落とし穴**: オは「セル生産 = 熟練労働専用」という誤解。実際には標準化された多能工プログラムで海外でも展開可能。

**学習アドバイス**: セル生産の利点は柔軟性と作業者のモチベーション向上。技能要件の明確化により国際展開も可能。

### 第7問　組織設計
**問題要旨**: 機能別部門組織の特性で、最も適切なものを選択

`K2 グラフ形状` `T1 正誤判定` `L2` `Trap-組織形態の混同`

**正解**: ア

**必要知識**: 機能別組織、事業部制組織の特性、集権・分権、規模の経済vs対応性

**解法の思考プロセス**:
1. **機能別組織 → 専門機能基礎 → 次世代トップマネジメント養成困難** ← 機能特化で全社的視点が育ちにくい → 正解候補 ✓
2. 分権化進展 → トップマネジメントへの集権度低い ← 機能別は通常集権的 ×
3. 部門間コンフリクト発生可能性低い ← 機能の相互依存で競争関係発生 ×
4. 専門化利益 → 規模経済は犠牲 ← 機能別は規模経済を活用 ×

**誤答の落とし穴**: 機能別組織は効率的だが、全社的経営判断経験が不足し、次世代リーダー育成が課題。

**学習アドバイス**: 組織形態による特性（機能別 = 効率・専門性高いが対応性低い。事業部制 = 対応性高いが機能重複）。

### 第8問　職務設計
**問題要旨**: 分業と職務設計に関する説明で、最も適切なものを選択

`K2 グラフ形状` `T1 正誤判定` `L2` `Trap-概念順序誤り`

**正解**: イ

**必要知識**: 職務拡大（job enlargement）、職務充実（job enrichment）、エンパワーメント

**解法の思考プロセス**:
1. エンパワーメント = 職務拡大の進化 + プランニング権限 ← 正確には職務充実の発展形。記述は理解できるが定義として若干異なる
2. **単調な単位に分解 → 単純作業 → 職務充実で職務幅広げ** ← 正確な定義と対応 ✓
3. 多能工化 + 品質管理一体化 → 柔軟対応・低コスト・品質維持 ✓（但し最も適切は要確認）
4. プロセス標準化 → 専門能力向上、アウトプット分散大 → 検査コスト増 ← 分散は減少傾向 ×
5. 目標・評価基準標準化 → モジュール化 → 管理者調整負担増 ← むしろ減少 ×

**誤答の落とし穴**: イが最も正確。職務充実は単調性解消と自律性向上の両面が特徴。

**学習アドバイス**: 職務拡大 = 仕事量増（横方向）。職務充実 = 権限・責任増（縦方向）。エンパワーメント = 両者の統合版。

### 第9問　管理の幅（Span of Control）
**問題要旨**: 管理の幅に影響する要因で、最も適切なものを選択

`K2 グラフ形状` `T1 正誤判定` `L2` `Trap-因果倒置`

**正解**: オ

**必要知識**: 管理の幅、組織階層、部下管理能力、業務の標準化

**解法の思考プロセス**:
1. マニュアル化進む → 例外事項増加 ← 例外は減少。標準化で処理明確化 ×
2. 職務同期取る必要高い → 調整負担増 → 幅狭くなる ← 妥当だが「最も適切」か検証必要
3. 部下熟練 → 相互依存度高い → 幅狭い ← 逆。熟練は幅広げる要因 ×
4. 相互依存度高い、環境不規則 → 幅広くできる ← 調整必要で幅狭くなる ×
5. **標準化された業績評価指標で統一管理可能 → 幅広くできる** ← 管理システム整備で効率的に複数部下管理可能 ✓

**誤答の落とし穴**: 管理の幅は部下の自律性と管理システムの整備度で決まる。評価基準標準化は効率化ツール。

**学習アドバイス**: 管理の幅を広げるには（1）部下の能力・訓練、（2）業務の標準化、（3）情報システム整備、（4）明確な評価基準の4要素が重要。

### 第10問　コミュニケーション・危機管理
**問題要旨**: 工場でのうわさ発生時に工場管理者がとるべき対応で、最も適切なものを選択

`K1 定義・用語` `T2 グラフ読解` `L2` `Trap-過剰反応`

**正解**: イ

**必要知識**: 組織内コミュニケーション、危機管理、情報真空、デマ対策

**解法の思考プロセス**:
1. うわさが自然におさまるまで待つ ← 情報真空を放置するとデマ拡大 ×
2. **直ちに工場内放送で事実発表** ← 情報真空を埋める。正確な情報を即座に提供。モラール回復 ✓
3. 後日、社内報で報告 ← 遅すぎ。うわさ拡大期間を放置 ×
4. 張本人探して処分 ← 報復的で信頼喪失。根本解決にならない ×
5. 役員会→記者会見 ← 外部発表で逆に大事化。内部問題を内部で早期解決すべき ×

**誤答の落とし穴**: 組織内危機では「情報の迅速性と正確性」が最優先。沈黙は問題を悪化させる。

**学習アドバイス**: デマ・うわさ対策の鉄則：（1）即時対応、（2）正確情報の発信、（3）複数チャネル活用、（4）継続的な説明。

## 年度総括

### 出題傾向
- **戦略系の出題が約40%**：経営計画、成長戦略、イノベーション、国際展開
- **組織・人事が約40%**：組織設計、職務設計、モチベーション、HRM
- **マーケティング・その他が約20%**：STP、ブランド、価格

### 難易度分布
- 易問：第4問、第11問、第14問（基礎知識で対応可能）
- 中問：第1問～3問、第6～10問、第16～19問（定義や特性の正確理解が必須）
- 難問：第5問、第23問（因果関係の理解と応用的思考が必要）

### 学習優先度（Tier別）
**Tier 1（必須）**: 1～5問（経営戦略基礎）、11～15問（マーケティング基礎）
**Tier 2（重要）**: 6～10問（組織設計）、16～20問（人的資源管理）
**Tier 3（応用）**: 21～25問（経営課題統合）

## 分類タグの凡例

### 知識種類（K）
- K1: 定義・基本用語
- K2: 理論・フレームワーク
- K3: 因果関係・因果メカニズム
- K4: 計算・数値判断
- K5: 質的判断・トレードオフ

### 思考法（T）
- T1: 定義確認型（正確な定義の理解）
- T2: 比較判断型（複数選択肢の相対比較）
- T3: 因果推論型（原因と結果の因果関係把握）
- T4: 計算型
- T5: 統合判断型（複合的条件の整理）

### 形式層（L）
- L1: 一文定義（単純な正誤判断）
- L2: 短段落説明（一段落で説明可能）
- L3: 事例分析（事例の分析が必要）
- L4: 複合問題（複数概念の組み合わせ）

### 罠パターン（Trap）
- Trap-定義誤り: 概念の定義を誤って理解させる
- Trap-因果逆転: 原因と結果を反対に述べる
- Trap-部分的正確性: 一部は正確だが全体として不適切
- Trap-概念混同: 類似概念を混同させる
- Trap-因果倒置: 条件と結果の関係を逆にする
