## 簿記・基礎会計

### 第1問　商品売買益の計算

**問題要旨**: 商品の購入・販売取引から、売上原価法または売上高控除法を用いて商品売買益を計算する。

`K3 数式・公式` `T3 計算実行` `L3` `Trap-E 計算ミス`

**正解**: 数値計算による判定（PDFで確認）

**必要知識**: [簿記基礎](https://www.example.com/wiki/bookkeeping-basics) — 売上原価計算と期末商品棚卸

**解法の思考プロセス**:
1. 売上原価 = 期首商品棚卸 + 当期商品仕入 − 期末商品棚卸
2. または、商品売買益 = 売上高 − 売上原価
3. 各月の仕入数量・単価、販売数量・販売価格を整理
4. 先入先出法（FIFO）または加重平均法で単価を決定

**誤答の落とし穴** `Trap-E 計算ミス`:
- 期末在庫数量の読み誤り（月末 vs 期末の区別）
- 単価計算時に、加重平均と FIFOの結果の違いを無視
- 返品・値引きを売上高から直接控除するケースの誤認

**学習アドバイス**: 商品売買益計算は、工業簿記（製造原価）への橋渡し。各月の在庫推移を「流量図」で視覚化すると誤り減少。

### 第2問　補助金の会計処理

**問題要旨**: 政府からの補助金受領時の会計処理（貸借対照表科目の判定）と税務上の扱いの相違を理解する。

`K5 制度・データ` `T4 因果推論` `L2` `Trap-B 条件すり替え`

**正解**: 補助金関連科目の分類（PDFで確認）

**必要知識**: [企業会計基礎](https://www.example.com/wiki/corporate-accounting-basics) — 負債性補助金と非負債性補助金

**解法の思考プロセス**:
1. 返還不要な補助金 → 現金受領時は「補助金受贈益」（収益）として計上
2. または、「その他の包括利益」経由で「資本剰余金」に計上する場合も
3. 返還条件付き（工事期間内に工事完成義務など） → 「前受金」（負債）として処理
4. 補助対象資産の減価償却時に、補助金相当額を益金化する方法も存在

**誤答の落とし穴** `Trap-B 条件すり替え`:
- 補助金をすべて「営業外収益」と誤認（実は条件次第）
- 返還条件の有無を問題文から正確に読取れない
- 法人税上の益金算入時期が会計と異なることを間過

**学習アドバイス**: 補助金は会計基準により処理方法が複数存在。問題文で「条件の有無」「資産か費用か」を最初に仕分する習慣が重要。

---

### 第3問　連結会計の基礎

**問題要旨**: 親会社と子会社の企業集団における、連結決算の基本原理（内部取引消去、のれん等）を判定する。

`K1 定義・用語` `T1 正誤判定` `L2` `Trap-D 混同誘発`

**正解**: 連結会計の原則説明（PDFで確認）

**必要知識**: [企業会計基本](https://www.example.com/wiki/corporate-accounting-principles-and-consolidation) — 連結決算の手順と消去仕訳

**解法の思考プロセス**:
1. 連結貸借対照表：親子の貸借対照表を合算後、内部取引を消去
2. 内部取引消去の例：親会社が子会社に商品を売却した場合、その売上・仕入を消去
3. のれん = 子会社取得時の買収価格 − 子会社の純資産
4. 投資と資本金の相殺消去：親会社の「子会社株式」と子会社の「資本金」を消去

**誤答の落とし穴** `Trap-D 混同誘発`:
- のれんの負数（負のれん）が発生する場合の処理方法
- 連結子会社と関連会社（持分法適用会社）の処理方法の相違
- 期中新規取得子会社の連結開始月の判定

**学習アドバイス**: 連結会計は「企業グループ全体を１つの企業と見なす」考え方。内部取引の「二重計上排除」が核となる。

## 企業会計（財務諸表）

### 第4問　収益認識の基準

**問題要旨**: 新しい会計基準（IFRS15 または国際会計基準）に基づく収益認識のポイント（履行義務、重要な支払期限など）を判定する。

`K5 制度・データ` `T2 グラフ読解` `L2` `Trap-B 条件すり替え`

**正解**: 基準適用の判定（PDFで確認）

**必要知識**: [企業会計：財務諸表と5つの利益](https://www.example.com/wiki/corporate-accounting-financial-statements-and-five-profits) — 収益認識基準の変更

**解法の思考プロセス**:
1. 従来基準：「商品引渡し時」に売上計上
2. 新基準（IFRS15）：「顧客に支配が移転するタイミング」を重視
3. 例：予約販売、定期配送、ライセンス料など、「支配移転タイミング」が明確に必要
4. 重要な支払期限：報告期間の終了後12ヶ月以内 → 流動負債、それ以外 → 非流動負債

**誤答の落とし穴** `Trap-B 条件すり替え`:
- 「商品の物理的引渡し」と「リスク・報酬の移転」が異なるケースの判定誤り
- クレジット売上の場合、いつ収益認識するか（引渡し時 vs 回収確定時）の混淆
- 複数の履行義務を持つ契約（バンドル販売など）の分割処理

**学習アドバイス**: 収益認識基準は 2021年4月以降の企業会計で必須。日本の上場企業も IFRS移行が進行中。

### 第5問　負債の評価基準

**問題要旨**: 貸借対照表上の負債科目（流動負債、固定負債）の定義と分類基準を確認する。

`K1 定義・用語` `T1 正誤判定` `L1` `Trap-A 逆方向誘発`

**正解**: 定義説明問題（PDFで確認）

**必要知識**: [企業会計基本](https://www.example.com/wiki/corporate-accounting-basics) — 負債の評価と分類

**解法の思考プロセス**:
1. 流動負債：報告期間末から12ヶ月以内に決済すべき負債
2. 固定負債：12ヶ月を超える期間で決済する負債
3. 支払利息、購買債務、預り金、引当金など各科目の分類
4. 長期借入金の返済期限が「翌年度」の部分 → 流動負債（1年以内返済額）に振替

**誤答の落とし穴** `Trap-A 逆方向誘発`:
- 「固定負債＝長期借入金」と単純化（固定資産購入時の関連負債など多様）
- 支払期限が決まらない負債（例：返還不要の補助金）の分類
- 偶発債務（訴訟中の損害賠償請求など）の開示と計上の相違

**学習アドバイス**: 負債分類の基準は「報告期間末から12ヶ月」という時間軸。この一線で流動 vs 非流動が分かれることを習慣化。

### 第6問　純利益と収益

**問題要旨**: 損益計算書上の複数の利益概念（営業利益、経常利益、税前当期純利益、当期純利益）の定義と相違を判定する。

`K2 グラフ形状` `T1 正誤判定` `L1` `Trap-A 逆方向誘発`

**正解**: 利益概念の説明（PDFで確認）

**必要知識**: [企業会計：財務諸表と5つの利益](https://www.example.com/wiki/corporate-accounting-financial-statements-and-five-profits) — 段階別利益の定義

**解法の思考プロセス**:
1. 売上総利益（粗利益）= 売上高 − 売上原価
2. 営業利益 = 売上総利益 − 販管費
3. 経常利益 = 営業利益 + 営業外収益 − 営業外費用
4. 当期純利益 = 経常利益 − 法人税等 + 特別利益 − 特別損失

**誤答の落とし穴** `Trap-A 逆方向誘発`:
- 「営業外」の定義がぼやけている（金融業の場合、利息受取が営業収益）
- 特別利益・特別損失（固定資産売却益・損失など）の分類
- 事業分離損（セグメント廃止など）の科目分類

**学習アドバイス**: 5つの利益概念は、「利益の質」を示す。営業利益が継続的であるのに対し、特別利益は一時的という視点が経営分析の鍵。

### 第7問　キャッシュフロー計算

**問題要旨**: キャッシュフロー計算書から、営業活動・投資活動・財務活動のキャッシュフローを計算し、企業の資金繰り状況を判断する。

`K4 因果メカニズム` `T3 計算実行` `L3` `Trap-E 計算ミス`

**正解**: キャッシュフロー計算（PDFで確認）

**必要知識**: [利益とキャッシュ管理：キャッシュフロー](https://www.example.com/wiki/profit-and-cash-management-cash-flow-and-fcf) — 直接法と間接法

**解法の思考プロセス**:
1. 営業活動CF = 当期純利益 + 減価償却費 + 税金支払い − 運転資本の増減
2. 投資活動CF = −資本支出 + 固定資産売却益
3. 財務活動CF = 借入増減 + 配当金支払い
4. フリーキャッシュフロー（FCF）= 営業CF − 資本支出

**誤答の落とし穴** `Trap-E 計算ミス`:
- 「減価償却はキャッシュ支出ではない」という理屈での過度な修正
- 売上債権増加 vs 減少の方向誤り（運転資本として負の影響）
- 税金支払額と税金費用（損益計算書）の相違

**学習アドバイス**: キャッシュフローは「現金の出入り」が核。利益と異なり、信用販売や減価償却を厳密に処理する習慣が重要。

### 第8問　減価償却方法

**問題要旨**: 定額法・定率法など複数の減価償却方法の特性と計算、および方法選択による利益への影響を理解する。

`K2 グラフ形状` `T2 グラフ読解` `L2` `Trap-C 部分正解`

**正解**: 減価償却額の計算と影響分析（PDFで確認）

**必要知識**: [企業会計：資産の会計](https://www.example.com/wiki/corporate-accounting-assets-accounting) — 減価償却の方法と計算

**解法の思考プロセス**:
1. 定額法：毎年同額（年間償却額 = 取得原価 ÷ 耐用年数）
2. 定率法：毎年一定率で残簿価を減価（最初の年が最大）
3. ファンド償却法：あらかじめ修繕費用を見積もり基金化
4. 生産高比例法：実際の使用量に応じた償却

**誤答の落とし穴** `Trap-C 部分正解`:
- 定率法での「残簿価」と「取得原価」の混淆（毎年の記帳の変化）
- 減価償却が資産価値の客観的評価ではなく「費用配分」であることの誤解
- 耐用年数の変更・建物の償却終了後の継続使用の処理

**学習アドバイス**: 減価償却方法の選択は、経営方針を反映。初期段階で多く償却したい（利益抑制）なら定率法、安定的利益が目標なら定額法。

### 第9問　貸借対照表の読解

**問題要旨**: 企業の貸借対照表から、資産・負債・純資産の構成、および資産の流動性や負債の支払期限を読み取る。

`K1 定義・用語` `T1 正誤判定` `L1` `Trap-A 逆方向誘発`

**正解**: 貸借対照表の科目・金額判定（PDFで確認）

**必要知識**: [企業会計基本](https://www.example.com/wiki/corporate-accounting-basics) — 貸借対照表の構成

**解法の思考プロセス**:
1. 資産 = 負債 + 純資産（基本等式）
2. 流動資産（1年以内に現金化可能）：現金、売上債権、商品在庫
3. 固定資産：建物・機械（有形）、無形資産、投資
4. 流動負債 vs 固定負債の12ヶ月基準で分類

**誤答の落とし穴** `Trap-A 逆方向誘発`:
- 「資産が大きい＝優良企業」という短絡的判断（負債も大きい場合）
- 土地などの時価と簿価の乖離を見落とし
- 連結企業での「のれん」や「営業権」の開示内容の理解不足

**学習アドバイス**: 貸借対照表は「静止画」。経営分析には損益計算書（動き）やキャッシュフロー（現金の流れ）と合わせた読解が必須。

## 原価計算

### 第10問　材料費の計算

**問題要旨**: 製造工程で消費された材料費を、先入先出法や加重平均法で計算し、当期材料費と期末在庫を算定する。

`K3 数式・公式` `T3 計算実行` `L2` `Trap-E 計算ミス`

**正解**: 材料費の計算（PDFで確認）

**必要知識**: [原価計算](https://www.example.com/wiki/cost-accounting) — 材料費の計算方法

**解法の思考プロセス**:
1. 期首在庫 + 当期購入 − 期末在庫 = 消費額
2. 先入先出法：最初に購入した在庫から順に消費
3. 加重平均法：全体の平均単価で計算
4. 移動平均法：受入れのたびに単価を更新

**誤答の落とし穴** `Trap-E 計算ミス`:
- FIFO と 移動平均法の計算結果が異なることを意識しないまま計算
- 在庫数量の読み誤り（摘要欄の記号や表記を誤解）
- 単価計算時に小数点以下の端数処理の影響

**学習アドバイス**: 材料費計算は原価計算の基礎。「流量図」（期首 → 購入 → 消費 → 期末）で在庫推移を把握する習慣が重要。

### 第11問　製造間接費の配賦

**問題要旨**: 製造間接費（工場家賃、管理者給与など）を、生産数量や機械時間などの配賦基準で各製品に配分し、製造原価を計算する。

`K4 因果メカニズム` `T4 因果推論` `L3` `Trap-B 条件すり替え`

**正解**: 配賦額の計算と適用（PDFで確認）

**必要知識**: [原価計算：目的と分類](https://www.example.com/wiki/cost-accounting-purpose-and-classification) — 配賦の基準と方法

**解法の思考プロセス**:
1. 製造間接費 = 固定費（設備減価償却など）+ 変動費（補助材料など）
2. 配賦基準の選択：直接労働時間、機械時間、生産数量など
3. 配賦率 = 予定製造間接費 ÷ 予定配賦基準数量
4. 各製品への配賦 = 配賦率 × 実際配賦基準数量

**誤答の落とし穴** `Trap-B 条件すり替え`:
- 予定配賦 vs 実際配賦の概念混淆（標準原価では予定配賦が基本）
- 複数の製品がある場合、配賦基準を誤認（例：全員が同じ基準ではなく、部門別かもしれない）

**学習アドバイス**: 配賦は「恣意性」が入りやすい。配賦基準の「合理性」を常に問う思考が、原価管理の質を高める。

### 第12問　損益分岐点分析

**問題要旨**: CVP分析（Cost-Volume-Profit）から、損益分岐点売上高や利益目標達成に必要な販売数量を計算する。

`K3 数式・公式` `T3 計算実行` `L2` `Trap-E 計算ミス`

**正解**: 分岐点分析の計算（PDFで確認）

**必要知識**: [財務分析：CVP分析と損益分岐点](https://www.example.com/wiki/management-analysis-cvp-and-break-even) — 分岐点計算

**解法の思考プロセス**:
1. 変動費率（変動費÷売上高）と固定費を整理
2. 損益分岐点売上高 = 固定費 ÷ 貢献利益率（1 − 変動費率）
3. 損益分岐点数量 = 固定費 ÷ 単位貢献利益

**誤答の落とし穴** `Trap-E 計算ミス`:

**学習アドバイス**: CVP分析は経営意思決定の基本ツール。「あとどのくらい売上が減少できるか」（安全余裕率）の計算も併せて習得すると、経営判断に直結。

### 第13問　標準原価と差異分析

**問題要旨**: 標準原価（あるべき原価）と実際原価の差を、材料費・労務費・製造間接費ごとに分析し、差異の原因を特定する。

`K4 因果メカニズム` `T3 計算実行` `L3` `Trap-E 計算ミス`

**正解**: 差異の計算と分解（PDFで確認）

**必要知識**: [原価計算：標準原価と差異分析](https://www.example.com/wiki/cost-accounting-standard-costing-and-variance-analysis) — 数量差異と価格差異

**解法の思考プロセス**:
1. 材料費差異 = 価格差異 + 数量差異
   - 価格差異 = (標準価格 − 実際価格) × 実際数量
   - 数量差異 = (標準数量 − 実際数量) × 標準価格
2. 労務費差異 = 賃率差異 + 効率差異（同様の分解）
3. 製造間接費差異 = 予算差異 + 操業度差異 + 能率差異

**誤答の落とし穴** `Trap-E 計算ミス`:

- 製造間接費の固定費と変動費の区分に基づく差異分解の複雑性
- 有利差異が必ずしも「経営成績が良い」とは限ないことの見落とし（例：低品質で単価が安い）

**学習アドバイス**: 差異分析は「原因究明」の出発点。数値だけでなく、その背景（市場価格変動、生産効率改善 vs 品質低下など）を読む力が重要。

### 第14問　全部原価と直接原価

**問題要旨**: 全部原価計算（製造原価に固定費を含める）と直接原価計算（変動費のみ）の特性と、利益への影響の相違を理解する。

`K2 グラフ形状` `T1 正誤判定` `L2` `Trap-D 混同誘発`

**正解**: 計算方法と利益の相違説明（PDFで確認）

**必要知識**: [原価計算：直接原価と吸収原価](https://www.example.com/wiki/cost-accounting-direct-and-absorption-costing) — 計算方法と経営判断への影響

**解法の思考プロセス**:
1. 全部原価計算：製品原価 = 直接材料 + 直接労務 + 製造間接費（固定含む）
2. 直接原価計算：製品原価 = 直接材料 + 直接労務 + 変動製造費
3. 販売量 > 生産量：全部原価計算は利益大きい（固定費が在庫に含まれる）
4. 販売量 < 生産量：全部原価計算は利益小さい（余剰生産分の固定費が在庫へ）

**誤答の落とし穴** `Trap-D 混同誘発`:
- 固定費を「全て」変動費化すると勘違い（実は固定費は存在し続ける）
- 直接原価計算の「貢献利益」と全部原価の「営業利益」の混同
- 期末在庫の金額評価の相違（全部では固定費含有、直接では変動費のみ）

**学習アドバイス**: 両者の選択は会計目的による。外部報告（財務諸表）は全部原価、内部経営判断（利益計画）は直接原価が適切。実務では併行運用が一般的。

### 第15問　経営分析：採算性

**問題要旨**: ROA（総資産利益率）や ROE（自己資本利益率）など、収益性指標を計算し、企業の採算性を評価する。

**解法の思考プロセス**:

2. ROE = 当期純利益 ÷ 自己資本 × 100（資本効率）

**学習アドバイス**: ROA と ROE の相違は「資本構成（レバレッジ）」。自己資本が少ないほど ROE は大きくなるが、財務リスクも高まる（借入過多）。

### 第16問　経営分析：安全性

**問題要旨**: 流動比率・当座比率など、短期支払能力を示す指標や、自己資本比率など長期的財務安定性を評価する。

`K1 定義・用語` `T1 正誤判定` `L1` `Trap-A 逆方向誘発`

**正解**: 安全性指標の計算と評価（PDFで確認）

**必要知識**: [管理会計分析：安全性](https://www.example.com/wiki/management-analysis-safety) — 流動性と自己資本の評価

**解法の思考プロセス**:
1. 流動比率 = 流動資産 ÷ 流動負債 × 100（短期支払能力）
2. 当座比率 = 当座資産 ÷ 流動負債 × 100（即時支払能力、在庫除外）
3. 自己資本比率 = 自己資本 ÷ 総資産 × 100（財務安定性）
4. 負債比率 = 総負債 ÷ 自己資本（借金に対する資本の厚さ）

**誤答の落とし穴** `Trap-A 逆方向誘発`:

- 自己資本比率が低い＝危険企業という単純判定（事業特性に応じた最適水準あり）
- 負債が多い＝悪い という短絡（成長企業の積極投資は負債増加が正常）

**学習アドバイス**: 安全性は「相対比較」が重要。同業他社とのベンチマーク、業界平均、数年間の時系列比較で初めて判定可能。

### 第17問　経営分析：効率性

**問題要旨**: 資産回転率・在庫回転率・売上債権回転期間など、資産利用効率を示す指標を計算・評価する。

`K3 数式・公式` `T3 計算実行` `L2` `Trap-E 計算ミス`

**正解**: 効率性指標の計算（PDFで確認）

**必要知識**: [管理会計分析：効率性](https://www.example.com/wiki/management-analysis-efficiency) — 回転速度の評価

**解法の思考プロセス**:

2. 在庫回転率 = 売上原価 ÷ 在庫（在庫の滞留度）
3. 売上債権回転期間 = 365日 ÷ 売上債権回転率 = 365日 × 売上債権 ÷ 売上高（回収までの日数）

**学習アドバイス**: 効率性指標は「作業資本管理」に直結。短期的キャッシュフロー改善には、売上債権の早期回収と在庫削減が鍵。

### 第18問　ROA と ROE

**問題要旨**: DuPont分析を用いて、ROA を売上利益率 × 資産回転率に分解し、ROE へのレバレッジ効果を理解する。

`K3 数式・公式` `T3 計算実行` `L2` `Trap-E 計算ミス`

**正解**: 比率分解と解析（PDFで確認）

**必要知識**: [管理会計分析：採算性](https://www.example.com/wiki/management-analysis-profitability) — DuPont分析

**解法の思考プロセス**:
1. ROA = 純利益率 × 資産回転率 = (純利益 ÷ 売上) × (売上 ÷ 総資産)
2. ROE = ROA × 財務レバレッジ = ROA × (総資産 ÷ 自己資本)
3. ROE改善の経路：① 純利益率↑ ② 資産回転率↑ ③ レバレッジ↑
4. ③は借入増加を意味し、財務リスク増加と紙一重

**誤答の落とし穴** `Trap-E 計算ミス`:
- 「ROEが高い＝優良企業」という単純判定（レバレッジの過度な活用）
- 分解式の計算誤り（消約（キャンセル）が正確か確認）
- 競争力 vs 財務構造を区別していない分析（①②は競争力、③は財務構造）

**学習アドバイス**: DuPont分析は企業の「強み」と「課題」を見える化する。3つの改善経路を並列で検討し、最適な戦略を選択する経営判断に活用。

### 第19問　予算統制と分散分析

**解法の思考プロセス**:

### 第20問　利益計画と CVP分析応用

**問題要旨**: 複数製品の利益計画を立案する際、製品ミックスの最適化や増減による損益の影響を分析する。

`K3 数式・公式` `T3 計算実行` `L3` `Trap-E 計算ミス`

**正解**: 利益計画シナリオの計算（PDFで確認）

**必要知識**: [財務分析：CVP分析と損益分岐点](https://www.example.com/wiki/management-analysis-cvp-and-break-even) — 複数製品の分析

**解法の思考プロセス**:
1. 各製品の単位貢献利益と貢献利益率を計算
2. 加重平均貢献利益率 = Σ（各製品の貢献利益率 × ミックス比率）
3. 全社損益分岐点 = 固定費 ÷ 加重平均貢献利益率

**学習アドバイス**: 利益計画は「シナリオプランニング」。複数の選択肢を財務的に比較し、経営リスクを踏まえた意思決定が実務的な応用。

### 第21問　資本構成と資本費用

**問題要旨**: 企業の資本構成（自己資本 vs 他人資本）と加重平均資本費用（WACC）、および企業価値評価との関連を理解する。

`K4 因果メカニズム` `T4 因果推論` `L2` `Trap-B 条件すり替え`

**正解**: WACC計算と経営判断（PDFで確認）

**必要知識**: [財務：MM理論と配当政策](https://www.example.com/wiki/finance-mm-and-dividend-policy) — 資本構成と企業価値

**解法の思考プロセス**:
1. WACC = 自己資本コスト × (自己資本 ÷ 総資本) + 負債コスト × (1 − 法人税率) × (負債 ÷ 総資本)
2. 自己資本コスト：投資家が要求する期待収益率（リスク考慮）
3. 負債コスト：借入金利（税シールド効果により割引）

- 資本構成の最適化が企業価値最大化に寄与することの理論と実務の乖離

**学習アドバイス**: 資本構成は経営の根本的意思決定。借入増加は ROEを高める一方、破産リスクも増加。両立点（最適資本構成）を見極める力が経営学の真髄。

### 第23問　経営指標の関係式

**問題要旨**: ROA、ROE、利益率、回転率など複数の指標を相互に関連付け、企業経営の因果関係を読み解く。

`K3 数式・公式` `T3 計算実行` `L2` `Trap-E 計算ミス`

**正解**: 指標関係式の活用（PDFで確認）

**必要知識**: [管理会計分析：採算性](https://www.example.com/wiki/management-analysis-profitability) — 統合的指標分析

**解法の思考プロセス**:
1. ROE = ROA × 財務レバレッジ = (利益率 × 回転率) × 財務レバレッジ
2. 利益 = 売上 × 利益率
3. 売上 = 総資産 × 回転率
4. 各要素の変化が ROEにどう影響するか、矢印図（sensitivity analysis）で把握

**誤答の落とし穴** `Trap-E 計算ミス`:
- 関係式に代入する分子・分母（売上高 vs 売上原価など）の不正確さ
- 指標の「相関」と「因果」の混淆（相関が高い＝因果ではない）
- 異業種間で指標比較するときの「特性」（卸売業 vs 小売業など）の見落とし

**学習アドバイス**: 経営指標は「総合診断ツール」。複数の切り口から読むことで、企業の強みと課題が見えてくる。診断力が実務的価値。

### Tier別学習優先度

- **Tier 1（確実に取りたい）**: 問3,5,6,9,14,16（L1層）= 6問 = 24点
  - 定義と基本概念。教科書の太字を確実に習得

- **Tier 2（合格ラインの鍵）**: 問2,4,8,10,12,15,17,18,21,23（L2層）= 10問 = 40点
  - 構造理解に基づく計算。公式の意味を深く理解すれば対応可能

- **Tier 3（差をつける問題）**: 問1,7,11,13,19,20,22（L3層）= 7問 = 28点
  - 複雑な計算と推論。受験生全体で正答率低い。1-2問正解でリード獲得

### 本番セルフチェック5項目

1. **公式の適用前に定義確認**: 「この数値は何の分子？分母？」と問題を読むたびに問い直す
2. **複数の計算方法の比較**: 先入先出法と加重平均法など、別の方法で検証
3. **長期 vs 短期の区分**（負債の12ヶ月基準など）: 問題文にマーカー引く
4. **計算の検算**: 逆算（答え → 元の値に戻す）で必ず確認
5. **指標の「好ましい方向」の確認**: 「高い方が良いのか？低い方が良いのか？」を明確に
