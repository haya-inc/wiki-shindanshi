## 問題別解説

### 第1問

**問題要旨:** 備品を仕入れた在庫品のうち、400,000円分の売却時に最も適切な仕訳を選ぶ。在庫の原価は同じ金額で、売却時に掛けで行われた。

`K4 手続・手順` `T3 計算実行` `L1 基礎`

**正解**: ウ

**必要知識**:
- [商品売却時の仕訳処理](/finance-and-accounting/bookkeeping-basics-account-categories-and-journal-entries)
- [在庫の原価と売却](/finance-and-accounting/cost-accounting)

**解法の思考プロセス**:
- 問題から備品が在庫として保有され、400,000円で売却されたことを確認
- 売却価格と原価の関係を整理（備品入 400,000 = 売却価格）
- 掛けで売却されたため、売掛金として計上
- 仕訳形式: 売掛金 / 商品売却益か商品売上か、原価側のどちらを使うか確認

**誤答の落とし穴**:
- アの選択肢: 先入先出法でなく、間単純に原価のみを計上する誤り
- イの選択肢: 仕訳の左右が逆になっている（売上と原価の処理順序の誤解）
- エの選択肢: 売却時に既に損益計上されている在庫備品の仕訳として不適切

**学習アドバイス**: 仕訳は取引の事実を正確に表現します。売却時の仕訳では必ず①資産の増減②負債の増減③利益計上の3つを確認しましょう。

### 第2問

**問題要旨:** 先入先出法を用いた在庫評価において、月日ごとの仕入数量・単価・売上が与えられており、売上原価の計算最終的な仕訳を選ぶ。

`K1 定義・用語` `T2 分類判断` `L2 応用`

**正解**: イ

**必要知識**:
- [先入先出法の計算手順](/finance-and-accounting/cost-accounting-job-order-and-process-costing)
- [在庫台帳の記録](/finance-and-accounting/cost-accounting)

**解法の思考プロセス**:
- 月別・品目別に仕入と売上を整理
- 各売上時点で、先に仕入れた商品から順に原価を適用
- 7月1日：20@410 = 8,200円（前月繰越）
- 7月中の売上：受取手形100単位と売掛金の分離が必要
- 期末在庫の評価（先入先出法では新しい仕入単価を適用）
- 計算：売上原価 = 期首在庫 + 仕入 - 期末在庫

**誤答の落とし穴**:
- アの選択肢: 平均原価法の考え方を混入させた計算
- ウの選択肢: 後入先出法を適用した計算値
- エの選択肢: 移動平均法による計算（異なる在庫評価法の混同）

**学習アドバイス**: 先入先出法では「古い仕入を先に使う」という単純な原則を常に念頭に置きます。取引順序を時系列で整理する表（在庫台帳）を作成すれば、計算ミスが減ります。

### 第3問

**問題要旨:** 仕入期限と支払期限の関係から、支払利息の計上要否と金額を判定する。条件：支払額400,000円で既定利息が定められている場合。

`K4 因果メカニズム` `T1 正誤判定` `L1 基礎`

**正解**: ウ

**必要知識**:
- [支払利息の計上基準](/finance-and-accounting/corporate-accounting-basics)
- [仕入期限と支払条件](/finance-and-accounting/corporate-accounting-basics)

**解法の思考プロセス**:
- 取引時点での明示的な利息条件を確認
- 仕入時に既に利息が含まれているかどうかを判定
- 場合①：利息が最初から仕入額に含まれている → 支払利息計上なし
- 場合②：支払時に別途利息が発生する → 支払利息計上あり
- 問題では400,000円の仕入があり、それが支払時に変わるか否かを問うている

**誤答の落とし穴**:
- アの選択肢: 支払利息を過大に計上（支払条件をすべて利息と見なす誤り）
- イの選択肢: 条件すり替え（仕入額と支払額が異なる場合と同じと考える）
- エの選択肢: 支払利息を過小に計上（利息の一部のみを認識）

**学習アドバイス**: 「利息」と「割引」は異なります。仕入時点と支払時点での金額差をよく確認し、その差が利息なのか割引なのかを正確に判定することが重要です。

### 第4問

**問題要旨:** キャッシュ・フロー計算書における「支払利息及び受取配当金」と「当期税金」に関する記述から、複合的に支払う金額の内訳を判定する。

`K3 数式・公式` `T3 計算実行` `L3 高度`

**正解**: ウ

**必要知識**:
- [キャッシュフロー計算書の構造](/finance-and-accounting/profit-and-cash-management-cash-flow-and-fcf)
- [営業活動・投資活動・財務活動の分類](/finance-and-accounting/profit-and-cash-management-cash-flow-and-fcf)

**解法の思考プロセス**:
- キャッシュ・フロー計算書の営業活動CF、投資活動CF、財務活動CFを整理
- 「支払利息及び受取配当金」は営業活動CF上の支出科目
- 「当期税金」も営業活動CF上の支出科目
- 与えられた数字（配当金受取3,200万円、税金支払1,000万円など）を分類
- 法人税等支払額の計算：当期税金と支払配当金の複合計算

**誤答の落とし穴**:
- アの選択肢: 配当金受取額を除外（営業CF内分類の誤り）
- イの選択肢: 配当金と税金を混同（支出・収入の向きの誤り）
- エの選択肢: 計算順序の誤り（複数要素の統合計算失敗）

**学習アドバイス**: キャッシュフロー計算書は「実際の現金の流れ」を追跡します。損益計算書の利益とは異なることに注意。営業CF・投資CF・財務CFの区分を常に意識してください。

### 第5問

**問題要旨:** 複数の営業キャッシュフロー要素が与えられ、当期純利益の計算に使用する公式と各要素の加減算処理を判定する。

`K3 数式・公式` `T3 計算実行` `L3 高度`

**正解**: ウ

**必要知識**:
- [当期純利益の定義](/finance-and-accounting/corporate-accounting-financial-statements-and-five-profits)
- [直接法による営業CF計算](/finance-and-accounting/profit-and-cash-management-cash-flow-and-fcf)

**解法の思考プロセス**:
- 営業CFのデータから逆算して当期純利益を求める
- 直接法での計算式: 営業CF = 当期純利益 + 非現金費用 - 非現金収益 + 運転資本変化
- 逆に：当期純利益 = 営業CF - 非現金費用 + 非現金収益 - 運転資本変化
- 与えられた要素：減価償却費、売掛金増減、在庫増減など
- それぞれを正しく加減算する

**誤答の落とし穴**:
- アの選択肢: 売掛金と在庫増減の方向を逆に計上
- イの選択肢: 減価償却費を減算している（逆方向）
- エの選択肢: 複数要素の計算順序が誤っている

**学習アドバイス**: 運転資本の増減は「売掛金増加 = 現金流出遅延」と考えると、加減算の方向が明確になります。図解すると理解しやすいです。

### 第6問

**問題要旨:** 原価計算の基本用語である「原価」「製造原価」の定義として最も適切な記述を選ぶ。仕入期限や代金支払いの時点も関わる。

`K1 定義・用語` `T1 正誤判定` `L1 基礎`

**正解**: ア

**必要知識**:
- [原価計算基準の定義](/finance-and-accounting/cost-accounting)
- [製造原価と製品原価の区別](/finance-and-accounting/cost-accounting)

**解法の思考プロセス**:
- 原価計算基準に基づく正確な定義を確認
- 仕入期限の概念：仕入商品の中で当期製造に使用されたもの
- 代金支払いの有無は関係なし（発生主義）
- 各選択肢で「定期限」「品質不良」「処理基準」などの用語を検討

**誤答の落とし穴**:
- イの選択肢: 仕入期限を超えたもの（破損品・品質不良品）を誤って含める
- ウの選択肢: 代金支払いの有無を基準にしている（実現主義の誤解）
- エの選択肢: 代金支払い後のみを「原価」と見なす誤り

**学習アドバイス**: 「発生主義」の理解が重要です。原価は「現金支出」ではなく「経済的犠牲の発生」で判定されます。

### 第7問

**問題要旨:** 製造原価の計算表から、以下の値を算出して記載する空欄を埋める。当期の仕入、労働時間、製造間接費などから総製造原価を算出。

`K3 数式・公式` `T3 計算実行` `L3 高度`

**正解**: ウ

**必要知識**:
- [製造原価の計算手順](/finance-and-accounting/cost-accounting)
- [月別・製品別・費目別の製造原価](/finance-and-accounting/cost-accounting-purpose-and-classification)

**解法の思考プロセス**:
- 製造原価計算表の構造を理解：期首仕掛品 + 当期製造原価 - 期末仕掛品 = 当期製品製造原価
- 当期製造原価の計算：直接材料費 + 直接労働費 + 製造間接費
- 直接材料費：期首材料在庫 + 仕入 - 期末材料在庫
- 直接労働費：製造部門の給与
- 製造間接費：固定費+変動費（配賦ベースで計算）
- 各要素を合算して最終値を算出

**誤答の落とし穴**:
- アの選択肢: 期末在庫を加算している（減算が正しい）
- イの選択肢: 仕掛品の変化を誤った方向に計上
- エの選択肢: 製造間接費の配賦率を誤用

**学習アドバイス**: T字型のフロー図を書いて、各要素の流れを視覚化すれば、加減算の誤りを防げます。段階的に計算を進めましょう。

### 第8問

**問題要旨:** 売上原価分析において、前期と当期の売上高・売上原価・在庫を比較し、その差異を分析する。原価差異の要因分解。

`K3 数式・公式` `T3 計算実行` `L3 高度`

**正解**: ウ

**必要知識**:
- [売上原価の計算](/finance-and-accounting/cost-accounting)
- [売上原価分析（増減要因）](/finance-and-accounting/cost-accounting)

**解法の思考プロセス**:
- 売上原価 = 期首在庫 + 仕入 - 期末在庫
- 前期と当期の売上高：311,600 → 320,000円（増加）
- 売上原価：190,000 → 196,000円（増加）
- 原価率の変化を確認：前期 = 190,000/311,600 ≈ 61%、当期 = 196,000/320,000 ≈ 61.25%
- 原価率が上昇 → 売上高原価率が上昇（効率が低下）
- 差異額をしたがって、売上高増に伴う原価の増加要因を分析

**誤答の落とし穴**:
- アの選択肢: 売上原価の増加額を過大評価
- イの選択肢: 在庫変化の影響を見落とし
- エの選択肢: 原価率の計算を誤った（逆向き）

**学習アドバイス**: 金額変化だけでなく「率」の変化に注目することが重要です。原価率が上昇 = 経営効率低下を意味します。

### 第9問

**問題要旨:** セグメント別の利益計算において、固定費の配賦方法を選択する。限界利益とセグメント利益の計算から、最適な配賦ルールを判定。

`K4 因果メカニズム` `T3 計算実行` `L3 高度`

**正解**: ウ

**必要知識**:
- [セグメント利益の定義](/finance-and-accounting/cost-accounting)
- [固定費の配賦（段階法・相互配賦）](/finance-and-accounting/cost-accounting-purpose-and-classification)

**解法の思考プロセス**:
- 営業利益 = 限界利益 - セグメント固定費
- 限界利益 = 売上 - 変動費（原価・販売費の変動部分）
- セグメント固定費の配賦基準を決定：直接配賦か共通配賦か
- 与えられた固定費（給与・減価償却など）をセグメントに配分
- 最終的なセグメント利益を算出
- 複数セグメントがある場合、各セグメントの貢献度を評価

**誤答の落とし穴**:
- アの選択肢: 変動費を固定費として扱った誤り
- イの選択肢: セグメント共通の固定費を過度に配賦
- エの選択肢: 限界利益ベースの判定をしていない

**学習アドバイス**: セグメント分析では「直接配賦できるコストは配賦、共通コストは削減対象」という思想が基本です。

### 第10問

**問題要旨:** 比較損益計算書（前期・当期）から、各年度の主要数値を計算し、成長率や効率指標の変化を分析する。

`K3 数式・公式` `T3 計算実行` `L3 高度`

**正解**: イ

**必要知識**:
- [損益計算書の読み方](/finance-and-accounting/corporate-accounting-financial-statements-and-five-profits)
- [前期比較分析](/finance-and-accounting/management-analysis-profitability)

**解法の思考プロセス**:
- 前期第22期と当期第23期の損益計算書を並べて比較
- 売上高：1,000 → 1,200（増加20%）
- 売上原価：450 → 530（増加17.8%）
- 売上総利益：550 → 670（増加21.8%）

**誤答の落とし穴**:
- ア、ウ、エの選択肢: 計算過程の誤りや四捨五入の不正確さ
- 特に営業利益の減少に気づかず、売上増加のみで判定する誤り

**学習アドバイス**: 「売上が増えた」からといって「利益が増えた」とは限りません。構成比の変化（原価率、販管費率）に注目する癖をつけましょう。

### 第11問

**問題要旨:** 損益分岐点分析の基本公式における、空欄 A（売上）・B（固定費率か変動費率か）・C（利益）の用語を選択する。

`K3 数式・公式` `T4 因果推論` `L2 応用`

**正解**: ア

**必要知識**:
- [損益分岐点の定義](/finance-and-accounting/cost-accounting-exam-patterns)
- [損益分岐点分析の公式](/finance-and-accounting/cost-accounting-exam-patterns)

**解法の思考プロセス**:
- 損益分岐点の定義：利益 = 0 となる売上高
- 基本公式：売上 - 変動費 - 固定費 = 利益
- または：売上 × (1 - 変動費率) - 固定費 = 利益
- 損益分岐点では利益 = 0：売上 × (1 - 変動費率) = 固定費
- したがって：損益分岐点売上 = 固定費 / (1 - 変動費率) = 固定費 / 貢献利益率
- 空欄Aは「売上」、Bは「変動費率」か「1-変動費率」、Cは「利益」

**誤答の落とし穴**:
- イ、ウ、エの選択肢: 用語の混同（貢献利益率と変動費率、限界利益と総利益など）
- 因果関係を逆に理解する誤り

**問題要旨:** 加重平均資本コスト（WACC）を計算する。時価ベースの自己資本と借入金、それぞれのコスト率を用いて算出。配当割引モデルで自己資本コストを求める。
