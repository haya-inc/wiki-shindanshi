## 簿記基礎・財務諸表

### 第1問　仕訳の基礎

**問題要旨**: 典型的な取引（売上、仕入、給与支払い等）に対する仕訳を、複数の選択肢から判定する。

`K1 定義・用語` `T1 正誤判定` `L1 定義暗記` `Trap-D 混同誘発`

**正解**: （実際の仕訳内容に基づく）

**必要知識**: [簿記基礎](/finance-and-accounting/bookkeeping-basics-account-categories-and-journal-entries) — 仕訳の基本原則と勘定科目

**解法の思考プロセス**:
1. 仕訳の基本ルール確認：
   - 借方 = 資産↑、費用↑、収益↓
   - 貸方 = 負債↑、資本↑、収益↑、費用↓
   - 左側（借方）と右側（貸方）の金額が常に等しい（複式簿記）
2. 取引の分類：
   - 資産取引：現金、預金、売掛金、有形固定資産
   - 負債取引：買掛金、短期借入金、社債
   - 資本取引：資本金、利益剰余金
   - 損益取引：売上、仕入、給与、地代家賃
3. 取引例と仕訳：
   - 商品を現金で販売：[借]現金 / [貸]売上
   - 商品を掛けで仕入：[借]仕入 / [貸]買掛金
   - 従業員に給与を支払い：[借]給与 / [貸]現金
   - 銀行から借入：[借]現金（または預金） / [貸]短期借入金
4. 決算仕訳の初期段階：
   - 期中仕訳をすべて記録後、決算整理仕訳を実施
   - 本問は期中仕訳が中心

**誤答の落とし穴** `Trap-D 混同誘発`:
- 勘定科目の混同（例：「給与」と「給与引当」の区別）
- 借方・貸方を反対に書く
- 複数の取引を1つの仕訳で処理しようとする

**学習アドバイス**: 仕訳は簿記学習の最初の関門。「誰が」「何を」「どのように」という取引の実体を掴むことが重要。暗記ではなく「なぜこの勘定科目なのか」という背景理由を理解すると、応用問題にも対応できます。

### 第2問　決算整理仕訳と月次決算

**問題要旨**: 期末の決算整理（減価償却、未払費用、前払費用、引当金等）に関する複合問題。複数の取引から、適切な決算仕訳と決算後の勘定残高を判定する。

`K4 因果メカニズム` `T4 因果推論` `L2 構造理解` `Trap-B 条件すり替え`

**正解**: （決算整理の複合処理結果に基づく）

**必要知識**: [企業会計の基本](/finance-and-accounting/corporate-accounting-basics) — 決算整理の手順と各科目の処理

**解法の思考プロセス**:
1. 決算整理の主要項目：
   - **減価償却**：有形固定資産の価値減少を配分
     - 仕訳：[借]減価償却費 / [貸]累積減価償却
     - 例：機械装置100万円、耐用年数10年 → 年間減価償却費10万円
   - **未払費用**：月末時点で発生したが、支払いはまだの費用
     - 仕訳：[借]費用 / [貸]未払費用
     - 例：月末の給与、まだ支払っていない電気代
   - **前払費用**：すでに支払ったが、まだサービス提供を受けていない費用
     - 仕訳：[借]前払費用 / [貸]現金
     - 期末調整：[借]費用 / [貸]前払費用
   - **売上原価の計算**：
     - 期首棚卸資産 + 当期仕入 - 期末棚卸資産 = 売上原価
     - 棚卸法または総額法を使い分け
   - **引当金**：将来の費用・損失に備える
     - 貸倒引当金：売掛金の一部が回収不能と予想される
     - 従業員賞与引当金：期末時点での対従業員債務
2. 決算整理の順序：
   - 手形・債権の確認と修正
   - 在庫棚卸
   - 減価償却
   - 未払費用・前払費用の計上
   - 引当金の計上
3. 月次決算との違い：
   - 月次：より簡略化、見積もり多用
   - 期末（年次）：完全性、精度を優先

**誤答の落とし穴** `Trap-B 条件すり替え`:
- 減価償却を固定資産から直接減額する処理（実際は「累積減価償却」として対応）
- 未払費用と前払費用を逆に処理
- 棚卸資産の期末額を損益計算書に直接記載（実際は「売上原価」計算を通じて記載）

**学習アドバイス**: 決算整理は「期末の実態把握」という会計の本質を示す処理です。発生主義会計の理解が深まる重要なセクション。

### 第3問　財務諸表の相互関連

**問題要旨**: 損益計算書、貸借対照表、キャッシュフロー計算書の3つの財務諸表がどのようにリンクしているかを理解し、特定の数値や関係式を判定する。

`K2 グラフ形状` `T2 グラフ読解` `L2 構造理解` `Trap-D 混同誘発`

**正解**: （財務諸表の連動構造に基づく）

**必要知識**: [財務諸表と5つの利益](/finance-and-accounting/corporate-accounting-financial-statements-and-five-profits) — 3つの財務諸表の構造と相互関係

**解法の思考プロセス**:
1. 3つの財務諸表の定義：
   - **損益計算書（PL）**：売上から利益までの「フロー」を示す
     - 売上 - 売上原価 = 粗利益
     - 粗利益 - 販管費 = 営業利益
     - 営業利益 ± 営業外損益 = 経常利益
     - 経常利益 ± 特別損益 = 税引前利益
     - 税引前利益 - 法人税 = 当期利益（純利益）
   - **貸借対照表（BS）**：資産、負債、資本の「ストック」を示す
     - 資産 = 負債 + 資本
     - 資産 = 流動資産 + 固定資産
     - 負債 = 流動負債 + 固定負債
     - 資本 = 資本金 + 利益剰余金
   - **キャッシュフロー計算書（CF）**：現金の「流れ」を示す
     - CF = 営業CF + 投資CF + 財務CF
2. 3表の連動：
   - PL上の「当期利益」→ BS上の「利益剰余金」に加算
   - BS上の「現金」 = CF計算書の最終残高（流動資産）
   - CF上の「営業CF」≠ PL上の「営業利益」（理由：減価償却、売掛金の増減等）
3. 数値チェック表：
   | 要素 | PL | BS | CF |
   |---|---|---|---|
   | 売上 | ✓（上位行） | × | ✓（営業CF） |
   | 当期利益 | ✓（最下行） | → 利益剰余金 | ✓（営業CF起点） |
   | 現金 | × | ✓（流動資産） | ✓（最終残高） |
   | 減価償却 | ✓（費用） | ✓（対資産） | ✓（加算項） |

**誤答の落とし穴** `Trap-D 混同誘発`:
- 「営業CF = 営業利益」という誤解（減価償却、売掛金増減等を見落とし）
- BS上の「利益剰余金」が PL上の「当期利益」と等しいと思い込む（配当や前期分を見落とし）
- CFと実現主義（PL）の違いを理解できず、「当期利益＝キャッシュ」と誤認

**学習アドバイス**: 3つの財務諸表を一体として理解することが、経営分析の基礎になります。各表の縦軸（行の流れ）と表間の横軸（相互リンク）の両方を掴むことが重要。

### 第4問　損益計算書の詳細分析

**問題要旨**: 複数年の損益計算書から、売上増減率、利益率、原価率、経費率等の指標を計算し、経営状況の変化を判定する。

`K2 グラフ形状` `T2 グラフ読解` `L2 構造理解` `Trap-C 部分正解`

**正解**: （指標計算の実行結果に基づく）

**必要知識**: [経営分析](/finance-and-accounting/management-analysis) — PL上の各利益と比率

**解法の思考プロセス**:
1. PL上の5つの利益：
   - **粗利益** = 売上 - 売上原価
   - **営業利益** = 粗利益 - 販管費
   - **経常利益** = 営業利益 ± 営業外損益
   - **税引前利益** = 経常利益 ± 特別損益
   - **当期利益** = 税引前利益 - 法人税等
2. 主要な PL比率：
   - **原価率** = 売上原価 / 売上 × 100%（低いほど良い）
   - **粗利益率** = 粗利益 / 売上 × 100% = 100% - 原価率
   - **販管費率** = 販管費 / 売上 × 100%（低いほど良い）
   - **営業利益率** = 営業利益 / 売上 × 100%
   - **当期利益率** = 当期利益 / 売上 × 100%
3. 複数年比較の分析：
   - 売上が 1000万円（前年）→ 1200万円（当年）→ 増加率20%
   - 売上原価が 600万円 → 660万円 → 増加率10%（売上より低い増加率：利幅改善）
   - または、680万円 → 増加率13.3%（売上と同等：利幅維持）
   - または、700万円 → 増加率16.7%（売上より高い増加率：利幅悪化）
4. 実例（当期利益率が上昇する場合）：
   - 売上増で原価圧縮（スケール効果）
   - 原価低減施策（設備投資、仕入先交渉）
   - 販管費の抑制（人員削減、効率化）
   - 営業外損益の改善（金利低下）

**誤答の落とし穴** `Trap-C 部分正解`:
- 金額の増減と比率の増減を混同（例：売上↑でも原価率が↑なら利益率は↓）
- 分母となる「売上」の値を誤認識
- 複数年の値を「足す・引く」べきか「比較」すべきかの判断ミス

**学習アドバイス**: PL比率分析は「経営の効率性」を測る最初の指標。売上、原価、経費の三角形の関係を常に把握しながら、各比率の意味を理解することが重要です。

### 第5問　貸借対照表の構成と勘定科目

**問題要旨**: 特定の取引（融資申込、設備購入、配当支払い等）が貸借対照表のどの項目を変化させるかを判定する。

`K1 定義・用語` `T1 正誤判定` `L1 定義暗記` `Trap-B 条件すり替え`

**正解**: （BS勘定科目の移動に基づく）

**必要知識**: [貸借対照表と5つの利益](/finance-and-accounting/corporate-accounting-financial-statements-and-five-profits) — BS の構成要素と流動性分類

**解法の思考プロセス**:
1. BS の基本構成：
   ```
   【資産の部】
   流動資産：現金、預金、売掛金、有価証券、在庫
   固定資産：有形固定資産（機械、建物）、無形固定資産（特許）、投資資産

   【負債の部】
   流動負債：買掛金、短期借入金、未払費用
   固定負債：長期借入金、社債、退職給与引当金

   【資本の部】
   資本金、利益剰余金（=過去の累計利益 - 配当）
   ```
2. 取引と BS 変化の例：
   - 銀行から 1000万円を長期借入：[資産]現金 ↑1000万、[負債]長期借入金 ↑1000万
   - 機械装置を 500万円で購入（現金）：[資産]機械装置 ↑500万、現金 ↓500万（資産内部の移動）
   - 配当 100万円を支払い：[資産]現金 ↓100万、[資本]利益剰余金 ↓100万
   - 売上 200万円（未回収）：[資産]売掛金 ↑200万、[資本]利益剰余金 ↑200万（PL経由）
3. 流動・固定の区別：
   - 流動資産：1年以内に現金化されるもの
   - 流動負債：1年以内に支払われるもの
   - 固定資産・負債：1年超の期間に関するもの

**誤答の落とし穴** `Trap-B 条件すり替え`:
- 配当支払いを「費用」と誤解（実際は資本の減少、費用ではない）
- 機械購入による「資産の減少」を無視（現金減だが、資産総額は変わらない）
- 売上計上が「現金を必要としない」ことを理解しない

**学習アドバイス**: BSは「企業の財政状態」のスナップショット。各勘定科目が何を表し、どの取引で変化するかを 1:1対応で理解することが重要。

### 第6問　キャッシュフロー計算と現金の流れ

**問題要旨**: CF計算書の営業CF、投資CF、財務CFを計算し、期首現金と期末現金の差を検証する複合問題。

`K4 因果メカニズム` `T3 計算実行` `L3 因果連鎖推論` `Trap-E 計算ミス`

**正解**: （CFの計算と合計値に基づく）

**必要知識**: [利益とキャッシュフロー管理](/finance-and-accounting/profit-and-cash-management-cash-flow-and-fcf) — 現金流計算書の構成と計算方法

**解法の思考プロセス**:
1. CF計算書の3つの区分：
   - **営業CF**：通常の営業活動から生じる現金流
     - 間接法：税引前利益 + 減価償却 - 売上債権増 + 買入債務増 ...
     - 直接法：営業による現金入出（実務的だが複雑）
   - **投資CF**：固定資産投資、有価証券投資による現金流
     - マイナスが通常（投資は現金流出）
     - 売却益はプラス
   - **財務CF**：資金調達・返済による現金流
     - 借入：プラス（現金流入）
     - 返済・配当：マイナス（現金流出）
2. 間接法の計算（最も一般的）：
   ```
   営業CF = 税引前利益
        + 減価償却費（非現金費用を加算）
        - 売掛金増（営業資産の増加）
        + 買掛金増（営業負債の増加）
        - 在庫増
        - 法人税支払い
   = 営業CF
   ```
3. 数値例：
   - 税引前利益 1000万円
   - 減価償却費 200万円（非現金費用なので +）
   - 売掛金が 300万円増加（現金回収遅延なので -）
   - 買掛金が 150万円増加（支払い遅延なので +）
   - 営業CF = 1000 + 200 - 300 + 150 = 1050万円
4. CF総体の検証：
   - 期末現金残高 = 期首現金残高 + 営業CF + 投資CF + 財務CF
   - BS上の「現金」と整合確認

**誤答の落とし穴** `Trap-E 計算ミス`:
- 減価償却を「引く」（実際は「足す」。非現金費用だから）
- 売掛金増と買掛金増の方向を逆に処理
- 投資CFの符号を誤る（設備購入はマイナス）
- 税引前利益と当期利益を混同

**学習アドバイス**: CF分析は現代経営分析の必須要素。利益≠現金という発見から、「健全な成長」の条件を考察することが重要です。

### 第7問　収益認識基準（IFRS 対応）

**問題要旨**: 新しい収益認識基準（IFRS15）に基づいて、複雑な取引の収益認識タイミングと金額を判定する。

`K5 制度・データ` `T1 正誤判定` `L2 構造理解` `Trap-B 条件すり替え`

**正解**: （IFRS15に基づく収益認識に準拠）

**必要知識**: [企業会計の基本](/finance-and-accounting/corporate-accounting-basics) — 新収益基準の 5ステップモデル

**解法の思考プロセス**:
1. 従来の基準（実現主義）：
   - 商品の引渡し時点で売上を計上
   - 単純だが、複雑取引に対応不足
2. 新基準（IFRS15）の 5ステップ：
   - Step 1：顧客との契約を識別（契約存在の確認）
   - Step 2：履行義務を識別（商品・サービスの引渡し）
   - Step 3：取引価格を決定（対価額の確定）
   - Step 4：履行義務に取引価格を配分（複数商品の場合、按分）
   - Step 5：履行義務が充足された時点で収益を認識（引渡し、サービス提供等）
3. 複雑取引の例：
   - **リース取引**：リース料の全体を収益認識（従来は分割）
   - **複数履行義務**：商品 + アフターサービス → 販売価格 + サービス対価で按分
   - **割戻し・値引き**：当初認識時に見積控除（返金見積額を売上から控除）
   - **納期未定の受注**：単なる引き合いか、確定した契約か判定
4. 実務上の判定基準：
   - 「顧客が商品・サービスをコントロール」した時点 = 収益認識時点
   - 例：物流企業が商品を顧客倉庫に納入 → 顧客がコントロール取得 → 売上認識

**誤答の落とし穴** `Trap-B 条件すり替え`:
- 「商品の引き渡し＝常に売上認識」という旧基準の遺産
- 複数履行義務がある場合、各々を分離して処理する判定ミス
- 条件付き対価（返品リスク等）を完全に無視

**学習アドバイス**: IFRS基準への適合は中小企業診断士試験の現代的課題。従来の基準との相違点を 5ステップで整理することが重要です。

### 第8問　連結財務諸表

**問題要旨**: 親会社・子会社間の内部取引を排除し、連結 PL・BS を作成する問題。のれん、少数株主持分等の複合概念が問われる。

`K2 グラフ形状` `T2 グラフ読解` `L3 因果連鎖推論` `Trap-D 混同誘発`

**正解**: （連結処理の正確な実行に基づく）

**必要知識**: [企業会計の基本](/finance-and-accounting/corporate-accounting-basics) — 連結財務諸表の作成と内部取引排除

**解法の思考プロセス**:
1. 連結の基本：
   - 親会社が子会社の50%超を保有 → 連結対象
   - 親会社と子会社を「1つの経済単位」として分析
   - 内部取引（親←→子）は完全に排除
2. 連結処理の主要項目：
   - **親子間の売上・仕入**：相殺排除
     - 親の「売上」 = 子の「仕入」 → 両者を相殺（外部への取引のみ残す）
   - **親子間の売上債権・買入債務**：相殺排除
     - 親の「売掛金」 = 子の「買掛金」 → 相殺排除
   - **親子間の配当金**：相殺排除
     - 親の「投資利益」 = 子の「配当金支払い」 → 相殺排除
   - **のれん（goodwill）**：計上
     - のれん = (子会社取得時の親の支払額) - (子会社の時価純資産)
     - 毎年償却または減損テストを実施
   - **少数株主持分（非支配持分）**：計上
     - 親が100%保有しない場合、少数株主の持分を計上
     - 例：親が80%、少数株主が20%保有 → 少数株主持分 = 子会社純資産 × 20%
3. 連結 PL の例：
   - 親の売上：1000万円
   - 子の売上：500万円
   - 親子間の取引：200万円（親が子から仕入）
   - 連結売上 = 1000 + 500 - 200 = 1300万円（外部のみ）
4. 連結 BS の例：
   - 親の総資産：3000万円
   - 子の総資産：1500万円
   - 親が子に投資：800万円（親の投資資産）
   - 子の株式純資産：1200万円
   - のれん = 800 - (1200 × 親の持分%) = ...
   - 連結総資産 = 3000 + 1500 - 親の投資800 - 相殺額 = ...

**誤答の落とし穴** `Trap-D 混同誘発`:
- 内部取引の相殺を忘れる（特に複数取引がある場合）
- のれんの計算ミス（子会社の何の価値が基準か）
- 少数株主持分を「負債」と誤分類（実際は「資本」の一部）

**学習アドバイス**: 連結処理は複雑ですが、「グループ全体で外部との取引のみ見る」という基本原則を常に思い出せば、処理の判定が容易になります。

### 第9問　原価の分類（変動・固定・直接・間接）

**問題要旨**: 特定の費用（材料費、労務費、製造間接費、販管費等）を、変動費・固定費、直接費・間接費に分類する。

`K1 定義・用語` `T1 正誤判定` `L1 定義暗記` `Trap-D 混同誘発`

**正解**: （原価分類の定義に基づく判定）

**必要知識**: [原価会計の基本](/finance-and-accounting/cost-accounting-purpose-and-classification) — 原価分類の体系

**解法の思考プロセス**:
1. 変動費 vs 固定費（生産量との関係）：
   - **変動費**：生産量に比例して増減
     - 直接材料費（kg当たり）、直接労務費（時間当たり）
     - 例：生産1個 = 材料100円、生産 100個 = 材料 10,000円
   - **固定費**：生産量に関わらず一定
     - 工場の地代家賃、管理人給与、設備の減価償却
     - 例：月額地代 300万円は、生産100個でも10,000個でも同じ
   - **準変動費（半変動費）**：固定部分 + 変動部分
     - 例：電気代 = 基本料金（固定） + 使用量料金（変動）
2. 直接費 vs 間接費（製品との関連性）：
   - **直接費**：特定製品に直結するもの
     - 直接材料費、直接労務費（具体的な対象製品が明確）
   - **間接費**：複数製品に配分する必要がある
     - 製造間接費（工場の共通費）、販管費（会社全体の費用）
3. 分類マトリックス：
   | | 直接費 | 間接費 |
   |---|---|---|
   | 変動費 | 直接材料費、直接加工費 | 製造消耗品 |
   | 固定費 | 直接労務費（月給でも対象製品が明確な場合） | 工場地代、減価償却、管理人給与 |
4. 販管費の扱い：
   - 原価計算：製造原価に含めない（販売・一般管理に分類）
   - 損益計算では「販管費」として独立した区分

**誤答の落とし穴** `Trap-D 混同誘発`:
- 「燃料費＝絶対に変動費」という誤解（固定部分がある場合も多い）
- 「給与＝固定費」という思い込み（出来高給は変動費の側面）
- 製造間接費を「製造に関連＝直接費」と誤認（実際は間接費が多い）

**学習アドバイス**: 原価分類は後続の原価計算・損益分析の基礎。定義を正確に理解し、「なぜこの費用がこの分類なのか」を説明できるレベルまで習得することが重要。

### 第10問　標準原価と標準差異分析

**問題要旨**: 標準原価（理想的・予想される原価）と実際原価の差（差異）を計算し、その原因（数量差、価格差）を分析する複合問題。

`K4 因果メカニズム` `T3 計算実行` `L3 因果連鎖推論` `Trap-E 計算ミス`

**正解**: （差異分析の計算と分類に基づく）

**必要知識**: [原価会計：標準原価と分散分析](/finance-and-accounting/cost-accounting-standard-costing-and-variance-analysis) — 標準差異の計算と解釈

**解法の思考プロセス**:
1. 標準原価の定義：
   - 「正常な条件下での、予想される原価」
   - 例：製品1個 = 材料2kg × 500円/kg = 1000円（標準）
2. 差異分析の 2層構造：
   - **総差異** = 実際原価 - 標準原価
   - **総差異の分解**：
     a. 数量差異（使用量の差）= (実際数量 - 標準数量) × 標準単価
     b. 価格差異（単価の差）= 実際数量 × (実際単価 - 標準単価)
3. 具体例（直接材料費）：
   - 標準：製品1個 = 材料2kg × 500円/kg = 1000円
   - 実績：100個生産時、材料210kg × 480円/kg = 100,800円
   - 標準原価（100個）= 1000円 × 100個 = 100,000円
   - 総差異 = 100,800 - 100,000 = 800円（不利差異）
   - 数量差異 = (210kg - 200kg) × 500円 = +5,000円（不利：使用量過剰）
   - 価格差異 = 210kg × (480円 - 500円) = -4,200円（有利：単価低下）
   - 検証：-4,200 + 5,000 = 800円（合致）
4. 労務費の差異分析：
   - 時間差異 = (実際時間 - 標準時間) × 標準賃率
   - 賃率差異 = 実際時間 × (実際賃率 - 標準賃率)
5. 製造間接費の差異分析：
   - 3分差（支出差異、能力差異、操業度差異）が一般的
   - または 2分差（予算差異、操業度差異）

**誤答の落とし穴** `Trap-E 計算ミス`:
- 「差異＝悪い」という誤解（有利差異もあり、改善方向）
- 数量差異と価格差異を計算後、「足す」べきか「かけ合わせる」べきか迷う
- 「実績が標準より高い = 全て不利」という思い込み（実は価格低下なら有利）

**学習アドバイス**: 差異分析は「経営改善の指標」。好ましくない差異の原因を追究し、次期の改善に活かすというサイクルが重要です。

### 第11問　予定価格法（予定配賦）

**問題要旨**: 製造間接費を予定配賦率に基づいて配賦し、製造原価を計算する問題。実際配賦との差を理解する。

`K3 数式・公式` `T3 計算実行` `L2 構造理解` `Trap-E 計算ミス`

**正解**: （予定配賦率の計算と配賦額に基づく）

**必要知識**: [原価会計：標準原価と分散分析](/finance-and-accounting/cost-accounting-standard-costing-and-variance-analysis) — 製造間接費の配賦方法

**解法の思考プロセス**:
1. 製造間接費配賦の 3つの方法：
   - **実際配賦**：当期実績の総費用 ÷ 当期実績の操業度
     - 正確だが、期末月までデータが揃わない、月次決算が複雑
   - **予定配賦**（予定配賦率法）：予定費用 ÷ 予定操業度
     - 月次決算が簡単、在庫原価が安定
   - **標準配賦**：標準費用 ÷ 標準操業度
     - 標準原価制度との統合、差異分析が詳細
2. 予定配賦率の計算：
   - 予定配賦率 = 予定製造間接費 / 予定操業度（機械時間または直接労務時間）
   - 例：月当たり予定製造間接費 300万円、予定機械時間 1,000h
   - 予定配賦率 = 300万円 / 1,000h = 3,000円/h
3. 月次の配賦：
   - 実際機械時間 950h → 配賦額 = 950h × 3,000円/h = 2,850,000円
   - 実際製造間接費 2,900,000円 → 配賦差異 = 2,900,000 - 2,850,000 = 50,000円（不利）
4. 配賦差異の処理：
   - 月次：配賦差異を間接費として繰り越し
   - 期末：累計配賦差異を「売上原価」「在庫」に按分（月次簡便法では全額売上原価に計上）
5. 操業度の選択：
   - 直接労務時間：労務費が主要因の場合
   - 機械時間：機械費が主要因の場合
   - 単位時間（生産数量）：シンプルだが正確性に欠ける

**誤答の落とし穴** `Trap-E 計算ミス`:
- 予定配賦率を「年間配賦率」として月次に適用（季節変動を考慮すべき場合がある）
- 実際費用と配賦額の差（配賦差異）を「誤り」と思う（実際は正常な差異）
- 配賦差異の処理時期を間違える（月次記録 vs 期末処理）

**学習アドバイス**: 予定配賦は製造間接費の「民主的な」配分方法。月次決算の効率性と精度のバランスを取った実務的なアプローチです。

### 第19問 効率性分析（資産回転率）

- **売掛金回転率** = 売上 / 売掛金
- **在庫回転率** = 売上原価 / 平均在庫

### 第20問 安全性分析（負債返済能力）

`K2 グラフ形状` `T2 グラフ読解` `L2 構造理解` `Trap-D 混同誘発`

- **負債比率** = 負債 / 資産
- **自己資本比率** = 自己資本 / 資産（50%以上が健全）
- **利息カバレッジレシオ** = EBIT / 支払利息

### 第21問 DuPont分析（ROEの分解）

`K4 因果メカニズム` `T4 因果推論` `L3 因果連鎖推論` `Trap-E 計算ミス`

- ROE = 売上高利益率 × 資産回転率 × 財務レバレッジ
- 各要素の改善が ROE にどう寄与するかを分析

### 第22問 多期比較分析（トレンド分析）

`K2 グラフ形状` `T2 グラフ読解` `L2 構造理解` `Trap-D 混同誘発`

- 複数年の指標を比較、傾向を把握
- 改善傾向 vs 悪化傾向の判定

## CVP分析・予算管理

### 第23問 CVP分析基礎（損益分岐点）

`K3 数式・公式` `T3 計算実行` `L2 構造理解` `Trap-E 計算ミス`

- 損益分岐点 = 固定費 / (売上高 - 変動費) × 売上高 = 固定費 / 貢献利益率
- 例：固定費 300万円、販売価格 1,000円/個、変動費 600円/個
- 損益分岐点 = 300万円 / (1,000 - 600) = 7,500個

### 第24問 安全余裕度と利益予測

`K3 数式・公式` `T3 計算実行` `L2 構造理解` `Trap-E 計算ミス`

- 安全余裕度 = (予想売上 - 損益分岐点売上) / 予想売上 × 100%
- 50%なら、売上が50%まで低下しても利益ゼロ

### 第25問 予算差異分析と業績評価

`K4 因果メカニズム` `T4 因果推論` `L3 因果連鎖推論` `Trap-E 計算ミス`

- 予算差異 = 実績 - 予算
- 有利差異（実績が予算を上回る）と不利差異を分析
