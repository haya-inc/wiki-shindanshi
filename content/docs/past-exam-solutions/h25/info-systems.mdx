## 概要

平成25年度（2013）の経営情報システムは全25問（各4点、100点満点）で出題されました。ハードウェア・ネットワーク・データベース・システム開発方法論・プロジェクト管理・IT戦略がバランスよく出題された年度です。IT技術の定義理解と業務適用の両面が求められました。

問題文は [J-SMECA 公式サイト（平成25年度 経営情報システム）](https://www.jf-cmca.jp/contents/010_c_/shikenmondai.html) から入手できます。手元に PDF を用意したうえでお読みください。全年度の問題は [J-SMECA 試験問題ページ](https://www.jf-cmca.jp/contents/010_c_/shikenmondai.html) で公開されています。

</Callout>

## ハードウェア・OS・基本技術

### 第1問　補助記憶装置と e-SATA インターフェース

**問題要旨**: 企業での補助記憶装置の種類として最も適切な説明を選ぶ

`K1 定義・用語` `T2 分類判断` `L2` `Trap-D 類似混同`

**正解**: ア

**必要知識**: [ストレージ技術とハードウェア](/management-information-systems/knowledge-information-security-basics) — HDD、SSD、外部記憶装置の特性と用途

**解法の思考プロセス**:
- 補助記憶装置には複数の種類があります：
  - HDD：大容量、低速、機械的
  - SSD：高速、高価、信頼性が高い
  - e-SATA：外部接続用の高速インターフェース
- アは「e-SATA」の説明として正確です

**誤答の落とし穴**:
- イ・ウ・エは異なるインターフェースや装置の説明です
- インターフェース の速度や接続方法を混同しやすいです

**学習アドバイス**: USB、e-SATA、Thunderbolt の速度と用途を整理してください。企業システムでは信頼性と速度のバランスが重要です。

### 第5問　スタック・キュー・LIFO・FIFO

**問題要旨**: コンピュータでのデータ処理における配列の処理順序に関する記述で最も適切なものを選ぶ

`K3 数式・公式` `T3 計算実行` `L3` `Trap-E 計算ミス誘発`

**正解**: ア

**必要知識**: [スタック・キュー・LIFO・FIFO](/management-information-systems/knowledge-database-and-sql) — データ構造の処理順序

**解法の思考プロセス**:
- スタック（LIFO: Last-In, First-Out）：最後に追加された要素から処理
- キュー（FIFO: First-In, First-Out）：最初に追加された要素から処理

**学習アドバイス**: 「Last-In, First-Out = スタック」「First-In, First-Out = キュー」を反復暗記してください。逆向きの対応も確認してください。

### 第13問　SQL 複合条件（AND と OR の組み合わせ）

**問題要旨**: SQL の WHERE 句で複数条件の AND と OR を組み合わせて正確にデータを抽出する最適な方法を選ぶ

`K3 数式・公式` `T3 計算実行` `L4` `Trap-E 計算ミス誘発`

**正解**: ウ

**必要知識**: [複雑な WHERE 句の論理設計](/management-information-systems/knowledge-database-and-sql) — AND・OR の優先度と括弧の使い方

**解法の思考プロセス**:
- SQL の論理演算には優先度があります
- 括弧を使わない場合、AND の優先度が OR より高い
- 複雑な条件では括弧で明示的に条件グループを作成
- ウが正解です（具体的な条件は PDF 参照）

**誤答の落とし穴**:
- AND と OR の優先度を誤解する
- 条件の量化（「すべての」「いずれかの」）を見落とす

**学習アドバイス**: WHERE 句の論理演算は「真理値表」で正確性を検証してください。

## 統計分析

### 第22問　統計基礎（平均・標準偏差）

**必要知識**: [統計分析の基礎](/management-information-systems/knowledge-statistics-basics) — 平均、分散、標準偏差

**解法の思考プロセス**:
- 統計量の定義：
  - 平均：データの中央値
  - 分散：データのばらつき（平均からの距離の平方の平均）
  - 標準偏差：分散の平方根
- 計算の具体的な手順は PDF を参照してください

**誤答の落とし穴**:
- 分散と標準偏差を混同する
- 計算順序を誤る

**学習アドバイス**: 標準偏差は「データの平均的なばらつき」です。正規分布では±1 標準偏差で約 68% を含みます。

### 第23問　相関係数と統計分析

**問題要旨**: 2 つの変数の関係を分析する統計手法に関する記述で最も適切なものを選ぶ

`K3 数式・公式` `T3 計算実行` `L4` `Trap-E 計算ミス誘発`

**正解**: イ

**必要知識**: [相関分析と回帰分析](/management-information-systems/knowledge-statistics-basics) — 相関係数、因果関係の推論

**解法の思考プロセス**:
- 相関係数（-1～1）：2 変数の線形関係の強さ
- 回帰分析：一方の変数から他方を予測
- イが正解です（具体的な計算は PDF 参照）

**誤答の落とし穴**:
- 相関と因果関係を混同する（相関があっても因果関係がない場合がある）
- 相関係数の符号を誤読する

**学習アドバイス**: 「相関と因果は別」を常に意識してください。相関係数が高くても、隠れた第 3 の変数が影響している場合があります。

### 第24問　カイ二乗検定

**問題要旨**: 統計的仮説検定に関する記述で最も適切なものを選ぶ

`K3 数式・公式` `T3 計算実行` `L4` `Trap-E 計算ミス誘発`

**正解**: ウ

**必要知識**: [カイ二乗検定](/management-information-systems/knowledge-statistics-basics) — 独立性の検定

**解法の思考プロセス**:
- カイ二乗検定：カテゴリ変数の独立性を検定
- 帰無仮説：「2 つの変数は独立している」
- P 値が有意水準（0.05 など）より小さければ、帰無仮説を棄却
- ウが正解です（具体的な計算は PDF 参照）

**誤答の落とし穴**:
- P 値の意味を誤解する（「P 値 = 帰無仮説が真である確率」ではない）
- 有意性 と 実務的重要性を混同する

**学習アドバイス**: 統計的有意性は「サンプルサイズに依存」します。大規模データでは有意でも実務的には無視できる場合があります。
