## 概要

平成25年度（2013）の経済学・経済政策は全25問（各4点、100点満点）で出題されました。マクロ経済学とミクロ経済学が出題範囲で、金融政策・財政政策、国民所得、消費・投資理論、市場構造、国際経済などが主なテーマです。

問題文は [J-SMECA 公式サイト](https://www.jf-cmca.jp/contents/010_c_/shikenmondai.html) から入手できます。手元に PDF を用意したうえでお読みください。全年度の問題は [J-SMECA 試験問題ページ](https://www.jf-cmca.jp/contents/010_c_/shikenmondai.html) で公開されています。

## マクロ経済学：統計指標と基本概念

### 第1問　完全失業率の統計読み取り

**問題要旨**: 総務省労働力調査のグラフから、年齢階級別の完全失業率の推移を読み取り、「15〜24歳」の系列を特定する統計問題。

`K5 制度・データ` `T2 グラフ読解` `L2` `Trap-D 混同誘発`

**正解**: エ

**必要知識**: [国民所得計算と主要指標](/economics-and-economic-policy/macroeconomics-knowledge-national-income-and-indicators) — 完全失業率の定義と統計的特性

**解法の思考プロセス**: 年齢別失業率の特性から推論 → 若年層（15〜24歳）は失業率が高く、かつ景気変動に敏感 → グラフの振幅の大きさと絶対値から「最も上のライン（D）」に該当。

**誤答の落とし穴** `Trap-D 混同誘発`: 「失業率が高い＝グラフが上にある」と認識しても、複数のグラフ系列が存在する場合、曲線の形状（波動のパターン）を見落とすと誤選択。例えば「中高年層（55〜64歳）の下降トレンド」と「若年層の周期的変動」を混同。

**学習アドバイス**: 統計問題は毎年2〜3問出題されており、8〜12点分を占めます。「グラフの形状」よりも「相対的な大小関係と変化の方向」を重視する読み方が重要です。

---

### 第2問　需給ギャップの定義

**問題要旨**: 内閣府の「需給ギャップ」という経済指標で、総需要・総供給に用いられる具体的な変数の組み合わせを判定する知識問題。

`K1 定義・用語` `T1 正誤判定` `L1` `Trap-B 条件すり替え`

**正解**: イ

**必要知識**: [国民所得と需給ギャップ](/economics-and-economic-policy/macroeconomics-knowledge-national-income-and-indicators) — 潜在GDP（供給サイド）と実際GDP（需要サイド）の定義

**解法の思考プロセス**: 需給ギャップ = 実際GDP - 潜在GDP → 「総需要」＝実際のGDP → 「総供給」＝民間設備と労働力を使った潜在GDP → 選択肢を照合。

**誤答の落とし穴** `Trap-B 条件すり替え`: 「完全雇用GDP」と「潜在GDP」を混同。完全雇用GDPは古い概念で、現在は「潜在GDP」を用います。また「国内総支出」は需要の代理変数ですが、正確には「実際のGDP」との区別が必要。

**学習アドバイス**: マクロ指標の定義は試験出題の頻出エリア。用語の「英語名」と「日本語名」をセットで覚え、各指標の計測方法（支出面vs生産面）を理解することが基本です。

---

### 第3問　均衡GDP・デフレギャップの計算

**問題要旨**: 与えられた総需要関数（$D = 50 + 0.8Y$）から均衡GDPを計算し、完全雇用GDP（300）との乖離（デフレギャップ）を判定する計算問題。

`K3 数式・公式` `T3 計算実行` `L3` `Trap-E 計算ミス`

**正解**: ウ

**必要知識**: [ケインズ型マクロモデル](/economics-and-economic-policy/macroeconomics-knowledge-is-lm-and-policies) — 有効需要原理と均衡条件（$Y = D$）

**解法の思考プロセス**:
均衡 → $Y = D$ → $Y = 50 + 0.8Y$ → $0.2Y = 50$ → $Y = 250$

デフレギャップ = 完全雇用GDP - 均衡GDP = 300 - 250 = 50

**誤答の落とし穴** `Trap-E 計算ミス`: 計算過程で「乗数（1/(1-0.8) = 5）」を求める際、分母の計算誤りが多発。また「インフレギャップ」と「デフレギャップ」の符号を逆に選ぶ（均衡GDPが完全雇用より小さい＝需要不足＝デフレギャップ）ケースが頻出。

**学習アドバイス**: この型の問題は毎年1〜2問出題。計算ミス対策として「単位の確認」「逆算検証」を習慣化することで確実に4点を獲得できます。

### 第4問　総供給曲線の形状と政府支出

**問題要旨**: GDP = $C + I + G$、消費関数 $C = C_0 + c(Y - T)$ の条件下で、政府が労働課税を採用したときの均衡GDPの変化を判定する多段階推論問題。

`K2 グラフ形状` `T2 グラフ読解` `L2` `Trap-A 逆方向`

**正解**: ウ

**必要知識**: [IS-LMモデルと政策効果](/economics-and-economic-policy/macroeconomics-knowledge-is-lm-and-policies) — 政府支出乗数と税乗数の関係

**解法の思考プロセス**: 各選択肢の条件を解析 → 政府支出増加 $\Rightarrow$ 均衡GDP低下？との矛盾を検出 → 「投資が限界的に減少する」という条件をくわえると逆方向が発生するケースを判別。

**誤答の落とし穴** `Trap-A 逆方向誘発`: 直感的に「政府支出↑ $\Rightarrow$ GDP↑」と予想するが、その前提条件（投資の外生性、完全雇用仮定など）を見落とすと逆の結論に。例えば「貨幣供給が固定」の場合、金利上昇による投資減少効果が支出増加を打ち消す可能性がある。

**学習アドバイス**: マクロの「政策効果」は単純な矢印推論では不十分。常に「IS-LMメカニズム全体」と「制約条件」を同時にチェックする習慣が必要です。

---

### 第5問　均衡予算乗数

**問題要旨**: 政府支出と税収を同時に増加させた場合の均衡GDPの変化を計算する問題。（$\Delta G = \Delta T$の条件）

`K3 数式・公式` `T3 計算実行` `L2` `Trap-E 計算ミス`

**正解**: ア

**必要知識**: [財政政策と乗数](/economics-and-economic-policy/macroeconomics-knowledge-is-lm-and-policies) — 支出乗数と税乗数の差分

**解法の思考プロセス**: 均衡予算乗数 = 支出乗数 - 税乗数 = $\frac{1}{1-c} - \frac{c}{1-c} = \frac{1-c}{1-c} = 1$ → $\Delta G = \Delta T$ のとき $\Delta Y = \Delta G = \Delta T$

**誤答の落とし穴** `Trap-E 計算ミス`: 「$\Delta G$と$\Delta T$が等しい」という条件から「効果がゼロ」と誤推論。実際には支出乗数と税乗数の大きさが異なるため、均衡予算乗数は 1 になります（公式を覚えていないと計算誤り）。

**学習アドバイス**: 乗数公式は「公式を暗記する」ではなく「なぜそうなるのか」を式変形で理解することが本番での対応力を生みます。消費関数の傾き $c$ の値が変わってもパターン対応できます。

## マクロ経済学：IS-LM分析と金融政策

### 第6問　IS-LM分析：貨幣供給と均衡

**問題要旨**: IS-LMモデルで、貨幣供給量を増加させたときの金利・GDPへの影響を因果連鎖で説明する問題（「最適な（あるいは不適切な）組み合わせ」の選択）。

`K2 グラフ形状` `T2 グラフ読解` `L3` `Trap-B 条件すり替え`

**正解**: イ（またはウ）

**必要知識**: [IS-LMモデルと金融政策](/economics-and-economic-policy/macroeconomics-knowledge-is-lm-and-policies) — LM曲線のシフトと金利への影響

**解法の思考プロセス**:
1. 貨幣供給 ↑ → LM曲線が右シフト
2. 金利 ↓（金銭市場が緩和）
3. 投資 ↑（低金利で投資有利）
4. GDP ↑

各選択肢を照合し、この因果連鎖に矛盾しない組み合わせを選択。

**誤答の落とし穴** `Trap-B 条件すり替え`: 「貨幣供給増加」と「金利の方向」を直結せず、「どの市場で」「どの条件下で」という前提を見落とすエラー。例えば「完全雇用下」では名目金利は上昇する可能性もあり、単純な方向推論が通用しない。

**学習アドバイス**: IS-LM問題は毎年2〜3問。「グラフの左手系」（IS曲線）と「右手系」（LM曲線）をそれぞれ独立に分析してから統合するプロセスが鍵です。

### 第7問　政府支出と需要の有効性

**問題要旨**: 政府支出増加の効果について、「短期には有効か、長期には無効か」「貨幣政策との協調」などの条件下での判定問題。

`K4 因果メカニズム` `T4 因果推論` `L2` `Trap-C 部分正解`

**正解**: ア と b （選択肢形式に応じて）

**必要知識**: [財政政策と金融政策の相互作用](/economics-and-economic-policy/macroeconomics-knowledge-is-lm-and-policies) — クラウディングアウト効果

**解法の思考プロセス**: 政府支出 ↑ → 金利 ↑ → 投資 ↓ の圧縮効果（クラウディングアウト）が存在するか否かを判定 → 「在庫が存在」「政府支出増加で取引額増加」等の条件から最適な説明を探す。

**誤答の落とし穴** `Trap-C 部分正解`: 「政府支出は有効である」という結論に至ったが、その条件を読み落とす。例えば「投資が影響受けない場合にのみ有効」という限定条件があると、一般的な答えは「最適ではない」になる。

**学習アドバイス**: 政策効果の問題は「無条件に有効/無効」ではなく、「どの条件下で」という修飾語が常に付く。選択肢の最後の文まで丁寧に読む癖が必須です。

### 第8問　リカードの等価命題

**問題要旨**: 政府がいま赤字国債を発行するか、いま増税するか。将来世代の負担に変わりはない（リカードの等価命題）という命題について、成立条件と失敗条件を判定する問題。

`K4 因果メカニズム` `T4 因果推論` `L3` `Trap-D 混同誘発`

**正解**: ウ

**必要知識**: [財政赤字と家計の貯蓄行動](/economics-and-economic-policy/macroeconomics-knowledge-consumption-and-investment-theory) — 恒常所得仮説とベルトラン・ゲーム

**解法の思考プロセス**:
- リカードの等価命題：赤字発行 ≡ 増税（総負担は変わらない）
- 成立条件：完全予見、完全市場、相続動機
- 失敗メカニズム：家計が「将来負担を過小割引」する心理、不確実性、流動性制約

各選択肢を「メカニズムの説明として正確か」で評価。

**誤答の落とし穴** `Trap-D 混同誘発`: 「赤字発行」と「増税」を単純に「効果が同じ」と混同。実際には短期的には需要効果が異なり（赤字発行は消費↑、増税は消費↓）、長期の定常状態のみで一致する。短期と長期の混同が多発。

**学習アドバイス**: 「等価」という概念は試験頻出ですが、その成立には強い前提条件（完全競争、完全情報、無限時間軸）が必要。「等価が成立しない現実的な理由」をリスト化しておくと応用問題にも対応可能です。

### 第9問　インフレーション指標（設問1）

**問題要旨**: インフレーションについて、「同一時点で成立する購買力平価」「異時点間の名目利子率と実質利子率の関係」など、インフレ指標の相互関係を判定する問題（2つの設問に分かれている）。

`K1 定義・用語` `T1 正誤判定` `L2` `Trap-D 混同誘発`

**正解**: ア と c （設問1・設問2の組み合わせ）

**必要知識**: [インフレーションと物価指標](/economics-and-economic-policy/macroeconomics-knowledge-ad-as-and-international-macro) — 名目利子率、実質利子率、予想インフレ率の関係

**解法の思考プロセス**:
- 設問1：同一時点での購買力平価は「相対的価格」の問題。異時点での名目金利と実質金利は $r = i - \pi^e$ の関係。
- 設問2：インフレが所得分配に与える影響（名目賃金 vs 実質賃金）を判定。

**誤答の落とし穴** `Trap-D 混同誘発`: 「名目利子率の上昇＝インフレ」と「期待インフレ率上昇＝実質利子率低下」を混同。実際には、期待インフレ率の上昇に伴い名目利子率も上昇することで、実質利子率への影響は曖昧になる場合がある。

**学習アドバイス**: インフレ関連の用語は「名目」「実質」「予想」の3次元を常に区別。各選択肢で「どの次元の話をしているか」を最初に明確にすると誤答が減ります。

### 第10問　インフレーション指標（設問2）

**問題要旨**: インフレが所得分配に与える影響として、「課税最低限が固定されている場合」「累進課税体系と実質負担」の相互作用を判定する問題。

`K4 因果メカニズム` `T4 因果推論` `L2` `Trap-B 条件すり替え`

**正解**: ウ

**必要知識**: [インフレーションと税負担](/economics-and-economic-policy/macroeconomics-knowledge-ad-as-and-international-macro) — インデックス化と課税のタイムラグ

**解法の思考プロセス**: インフレ下で名目所得↑ → 累進課税により実効税率↑ → 「課税の冷却効果」が発生 → 実質可処分所得の伸びが鈍化。

### 第15問　限界効用・価格弾性率と企業利潤

**問題要旨**: 需要曲線に沿ったABラインについて、傾きや弾性率の関係から、企業が直面する需要の価格感応度を計算する問題。

**必要知識**: [環境政策と国際協調](/economics-and-economic-policy/macroeconomics-knowledge-ad-as-and-international-macro) — 京都議定書の枠組み

**解法の思考プロセス**:
- 日本の約束：6%削減（1990年度比）
- 達成期間：2008〜2012年
- 達成メカニズム：国内対策が基本、補足的に国際取引を活用

選択肢で「数値目標」と「達成手段」の両者が正確に記述されているか確認。

**誤答の落とし穴** `Trap-C 部分正解`: 「京都議定書のテーマ＝環境保全」で概ね正しいが、「ビジネス・政治的な複雑性」（例：グローバルサプライチェーン下での排出権計算）を読み落とす。

**学習アドバイス**: 環境政策は国際政治の影響が大きく、試験出題後に「現実の政策が変更される」リスクがあります。最新の公式情報（環境省、UNFCCC）を参照してから本番に臨むことが重要です。

### 第25問　京都議定書・環境政策（設問2）

**問題要旨**: 前問の続編。日本の排出権の実績について、「2012年度に公表された値」に基づき、実際の削減達成状況を判定する統計読み取り問題。

`K5 制度・基準` `T1 正誤判定` `L1` `Trap-C 部分正解`

**正解**: ウ

**必要知識**: [環境統計と政策評価](/economics-and-economic-policy/macroeconomics-knowledge-ad-as-and-international-macro) — 京都議定書達成状況

**解法の思考プロセス**:
- 2004年時点での基準：1990年度比での削減状況
- 実績：単独の国内対策では不足 → 国際メカニズム活用
- 全体 90%超の削減率を達成（排出権購入含む）

選択肢で「数値が正確か」「フェーズ（第1約束期間）を特定しているか」を確認。

**誤答の落とし穴** `Trap-C 部分正解`: 「京都議定書の目標達成」という結論は正しいが、「達成の内訳」（国内措置 vs 国際メカニズム）を誤読する。特に「排出権購入の割合」を見落とすと、「実質削減が不十分か」という議論に至る。

**学習アドバイス**: 環境政策の成果は「定量的な数字」で語られます。本番では「グラフから数値を読み取る」スキルが必須。1問4点と軽んじず、確実に拾いましょう。
