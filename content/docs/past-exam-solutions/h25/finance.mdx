## 簿記・仕訳・帳簿

### 第1問　伝票式会計の記録基準

**問題要旨**: 伝票式会計に関する記録ルールで、最も適切な組み合わせを選択する。

`K5 会計基準` `T1 正誤判定` `L2` `Trap-D 類似混同`

**正解**: イ

**必要知識**: [簿記基礎・仕訳と帳簿](/finance-and-accounting/bookkeeping-basics-account-categories-and-journal-entries) — 伝票式会計は3種類の伝票（入金伝票・出金伝票・振替伝票）を用いて仕訳を記録する方式。各伝票と補助簿の連携ルールが重要。

**解法の思考プロセス**:
各選択肢を吟味し、伝票の記録対象と補助簿の使い分けを確認する。先生記帳（伝票が記帳済み）と後生記帳、入金・出金・振替伝票の役割を整理して、最も整合性のある組み合わせを選ぶ。

**誤答の落とし穴**:
先生記帳と後生記帳を混同しやすい。入金・出金・振替伝票の役割分担が曖昧だと、何がどの伝票で記録されるか判断できない。

**学習アドバイス**: 伝票式会計は実務的な方式。各伝票の「用途」と「記録対象」を表にして整理し、仕訳帳・補助簿との関係を図解化すること。

### 第2問　当座預金と小切手の勘定連携

**問題要旨**: 当座預金の残高が複数の関連勘定（入金伝票、受取手形、売上伝票）とどう結びついているか、最も適切な金額を計算選択する。

`K2 分類・表示` `T3 計算実行` `L2` `Trap-C 部分正解`

**正解**: ウ（100,000円）

**必要知識**: [簿記基礎・仕訳と帳簿](/finance-and-accounting/bookkeeping-basics-account-categories-and-journal-entries) — 当座預金は現金同等物。入金伝票・出金伝票・受取手形期日の回収が、すべて当座預金残高に反映される。

**解法の思考プロセス**:
提示データから、入金・出金伝票と当初残高を合算して、最終残高を計算する。タイミングのズレ（当座預金に反映されるのは「実際の現金受け取り時」）を意識して計算する。

**誤答の落とし穴**:
売上伝票と入金伝票のタイミングのズレ。振替預金と当座預金を混同する。

**学習アドバイス**: 当座預金を「全入金・全出金の合算」と考える。T字勘定（取引イメージ図）で整理し、どの伝票がどの資金流を表すか区別すること。

### 第3問　親会社・子会社の資本変動と当期純利益の推定

**問題要旨**: 対象期間の純資産変動表から、子会社の当期純利益と配当金を推定し、親会社が属する帳簿上の経営成績を計算する。

`K3 公式・推移` `T3 計算実行` `L3` `Trap-B 条件見落とし`

**正解**: ウ（60,900千円）

**必要知識**: [簿記基礎・調整仕訳](/finance-and-accounting/bookkeeping-basics-adjusting-entries) — 純資産変動表では「期末純資産 = 期首純資産 + 当期純利益 - 配当金」。親子関係がある場合、子会社の利益計上ルール（連結会計）も考慮が必要。

**解法の思考プロセス**:
提示された純資産変動表から期首・期末の数字を抽出し、当期純利益の有無を判定する。別途記載の配当予定が実際に計上されたか確認し、剰余金の整合性をチェックする。

**誤答の落とし穴**:
配当額の「予定」と「実績」の区別。資本再編と利益の混同。表の空欄部分を埋める際に仮定を立てすぎる。

**学習アドバイス**: 純資産変動表は「フロー」の視点。期首～期末のすべての変動要因を列挙し、残高差額から未知数を逆算する習慣をつけること。

## 財務計算・財務諸表

### 第4問　キャッシュ・フロー計算書における資金の分類

**問題要旨**: キャッシュ・フロー計算書が対象とする資金区分（現金・現金同等物など）について、最も適切な組み合わせを選択する。

`K5 会計基準` `T2 分類判断` `L2` `Trap-D 類似混同`

**正解**: ア（定期預金・普通預金）

**必要知識**: [簿記基礎・調整仕訳](/finance-and-accounting/bookkeeping-basics-adjusting-entries) — キャッシュ・フロー計算書では「現金および現金同等物」の定義が厳密。短期かつ流動性の高い金融商品に限定される。

**解法の思考プロセス**:
各選択肢の資産を評価し、現金同等物の要件（取得時から満期・償還期限が3ヶ月以内、価値変動リスクが無視できる）に照らして判定する。

**誤答の落とし穴**:
「短期」と「流動性が高い」を同じと考える。企業の決済目的と会計基準の定義を混同する。

**学習アドバイス**: キャッシュ・フロー計算書は「現金動態」を追跡する。現金同等物の定義を正確に覚え、具体例と結びつけること。

### 第5問　財務比率による収益性判定（当期純利益の計算）

**問題要旨**: 貸借対照表と損益計算書から、複数の収益性指標（売上高営業利益率・売上高営業利益率・総資本回転率）を計算し、前期比の改善・悪化を判定する。

`K3 公式` `T3 計算実行` `L3` `Trap-E 計算ミス誘発`

**正解**: イ（総資本営業利益率：悪化 / 売上高営業利益率：改善 / 総資本回転率：改善）

**必要知識**: [財務・会計 経営分析（効率性）](/finance-and-accounting/management-analysis-efficiency) — 売上高営業利益率、総資本営業利益率、総資本回転率の計算式と関係。

**解法の思考プロセス**:
提示データから各指標を計算し、分子・分母の両方の変化を追跡する。営業利益の絶対額が減少しても、総資本が減少していれば総資本回転率は改善し得ることに注意。

**誤答の落とし穴**:
分子の絶対額で判断する。分母である総資本の変動を見落とす。

**学習アドバイス**: 経営分析は「比率」であり「絶対額」ではない。分子・分母の両方の変化を追跡し、結果として比率がどう変わるか計算する習慣をつけること。

### 第6問　負債比率と企業の安全性評価

**問題要旨**: A社の貸借対照表から負債比率や流動比率などの安全性指標を計算し、負債処理（620,000千円で買収）を会計上正しく計上するにあたり、最も適切な処理を選択する。

`K3 公式` `T3 計算実行` `L3` `Trap-A 逆方向`

**正解**: イ（100,000千円）

**必要知識**: [財務・会計 法人税と租税効果](/finance-and-accounting/corporate-accounting-liabilities-equity-and-tax-effect) — 企業が持つ債務を減免する場合、持分変動として処理される。買収価格と帳簿価値の差は「負債免除益」として計上。

**解法の思考プロセス**:
A社の負債総額と買収価格、売却資産の実現可能価値を整理し、買収者が負担する金額と負債免除益を計算する。

**誤答の落とし穴**:
売上高と負債の混同。買収価格をそのまま「負債免除益」と計上する。

**学習アドバイス**: 負債処理は複雑。M&A や事業譲渡の文脈で、「持分変動」と「損益」を区別する習慣をつけること。

### 第7問　原価性と安全余裕（損益分岐点分析）

**問題要旨**: A社の売上高 20,000千円、費用内訳から安全余裕率を最も適切に計算する。

`K1 定義・概念` `T3 計算実行` `L2` `Trap-C 部分正解`

**正解**: ウ（25.0%）

**必要知識**: [経営分析 CVP分析と損益分岐点](/finance-and-accounting/management-analysis-cvp-and-break-even) — 安全余裕率 = 安全余裕 / 売上高 = (売上高 - 損益分岐点売上高) / 売上高。

**解法の思考プロセス**:
変動費・固定費を分類し、貢献利益率を計算して、損益分岐点売上高を求める。その後、安全余裕と安全余裕率を計算する。

**誤答の落とし穴**:
固定費と変動費の分類を誤る。貢献利益率の計算で分子を間違える。安全余裕と安全余裕「率」を混同する。

**学習アドバイス**: 損益分岐点分析は実務的で頻出。変動・固定費の分類を最初にチェックし、その後貢献利益率を計算する流れを定着させること。

## 財務分析

### 第8問　売上高を当期完成品の時価で評価するケース（実現主義）

**問題要旨**: 売上高 20,000千円が記録されている際、新期間における材料評価額や時価変動の影響を加味した最終売上高を計算する。

`K3 公式・評価` `T3 計算実行` `L4` `Trap-B 条件見落とし`

**正解**: ア（0千円）

**必要知識**: [簿記基礎・調整仕訳](/finance-and-accounting/bookkeeping-basics-adjusting-entries) — 実現主義では、売上は「商品の所有権移転時」に認識される。単なる生産完了では売上計上されない。

**解法の思考プロセス**:
既に計上済みの売上を修正することはできない（過去の取引は確定）。むしろ、新規の仕訳対象は「値入差分」や「評価損益」の計上。

**誤答の落とし穴**:
時価評価を「売上修正」と混同する。制度会計と評価会計の界線を曖昧にする。

**学習アドバイス**: 売上計上のタイミングと、その後の資産評価は別問題。実現主義と評価替えを区別すること。

### 第9問　労務費の分類（工賃は労務費に含まれない）

**問題要旨**: 給与、雑給、従業員給与手当、所得控除金などの費目のうち「労務費に含まれない」ものを選択する。

`K1 定義・概念` `T2 分類判断` `L1` `Trap-D 類似混同`

**正解**: ア（a と b）

**必要知識**: [原価計算 原価要素の分類](/finance-and-accounting/cost-accounting-job-order-and-process-costing) — 労務費：従業員の給与・賃金。非労務費：外注加工費（外部委託の工賃）。

**解法の思考プロセス**:
各選択肢を検討し、「誰への支払いか」を確認する。外注・外注加工費は別途の「購買費」である。

**誤答の落とし穴**:
「雑給」という名称から「マイナー」と判断する。外注加工費と従業員給与を混同する。

**学習アドバイス**: 労務費は「従業員」への支払いが基本。費目名だけでなく「誰への支払いか」を確認する習慣をつけること。

### 第10問　直接材料費の原価計算（月度の生産・在庫データから）

**問題要旨**: 月度の直接材料費（当月投入 3,100kg、材料消費価格 310円/kg）から、月末仕掛品の原価を計算する。

`K3 公式・推移` `T3 計算実行` `L3` `Trap-B 条件見落とし`

**正解**: ウ（61,000円）

**必要知識**: [原価計算 標準原価計算と分散分析](/finance-and-accounting/cost-accounting-standard-costing-and-variance-analysis) — 投入材料費の配分方法。完全生産量概念を用いた計算。

**解法の思考プロセス**:
生産数量の推移から完全生産量を算定し、投入量と単位あたり材料費を計算する。月末仕掛品の完成度（加工進捗度）に注意して配分する。

**誤答の落とし穴**:
月末仕掛品の完成度を見落とす。kg と単位の間の換算を誤る。

**学習アドバイス**: 原価計算は、生産量と投入量の換算が重要。加重平均法で計算する。特に「完全生産量」の概念を理解すること。

## 管理会計・原価計算

### 第11問　月末仕掛品の原価評価（先入先出法vs.加重平均法）

**問題要旨**: 数量データ（当月投入 800単位、当月完成 600単位、月末仕掛品 200単位・加工進捗度 0.5）と原価データから、月末仕掛品の原価を計算する。

`K3 公式・評価` `T3 計算実行` `L4` `Trap-B 条件見落とし`

**正解**: エ（2,280千円）

**必要知識**: [原価計算 標準原価計算と分散分析](/finance-and-accounting/cost-accounting-standard-costing-and-variance-analysis) — 直接材料費と加工費を分けて計算。加工進捗度が異なる場合の計算方法。

**解法の思考プロセス**:
加重平均法で完全生産量を算定し、材料費と加工費を分けて月末仕掛品原価を計算する。投入タイミングの違いに注意する。

**誤答の落とし穴**:
完全生産量の計算で加工進捗度を見落とす。直接材料費と加工費を統一してしまう。

**学習アドバイス**: プロセスコスティングは複雑。加重平均法の計算式を暗記ではなく、概念から導出する力をつけること。

| K4 | 手続・手順 | 原価配分、支払処理、投資評価プロセス |
| K5 | 制度・基準 | 会計基準（ファイナンスリース、実現主義）、金融理論（MM理論） |

### 思考法（T）

| タグ | 意味 |
| --- | --- |
| T1 | 正誤判定 |
| T2 | 分類判断 |
| T3 | 計算実行 |
| T4 | 条件整理 |
| T5 | 場合分け |

### 形式層（L）

| タグ | 意味 |
| --- | --- |
| L1 | 定義暗記で解ける |
| L2 | 構造理解が必要 |
| L3 | 因果連鎖・推論が必要 |
| L4 | 数式操作・応用が必要 |

### 罠パターン（Trap）

| タグ | 意味 | 対策 |
| --- | --- | --- |
| Trap-A | 逆方向 | 資金流の方向を矢印で書き出して確認 |
| Trap-B | 条件見落とし | 前提条件を問題文から最初に列挙 |
| Trap-C | 部分正解 | 複合選択肢は全要素をチェック |
| Trap-D | 類似混同 | 対比表で概念の違いを明確化 |
| Trap-E | 計算ミス誘発 | 検算・別ルート計算で確認 |
