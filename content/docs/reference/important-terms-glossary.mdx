## このページの役割

<Callout title="このページの役割">
  このページは、`科目をまたいで何度も出る用語` をまとめて引くための付録です。定義を丸暗記するのではなく、
  `どの文脈で使う言葉か` と `どのページへ戻るべきか` を短く確認する入口として使います。
</Callout>

## 学習のポイント

- まず `言葉の意味` と `使う場面` を分けて覚える
- 次に `1次での定義` と `2次での使い方` を対応させる
- 詳しい説明は用語集で完結させず、必ず元ページへ戻る

## 使い方

1. 問題文や答案で引っかかった語をこのページで探す
2. `意味` と `使う場面` を 1 行で確認する
3. `まず戻るページ` と `補助で戻るページ` を順に読む

## 問題文や答案の言い換えから引く

| 問題文や答案の言い換え | 先に引く用語 | 読み分けのポイント | まず戻るページ |
| --- | --- | --- | --- |
| `事実ベースで答える`、`与件のどこに書いてあるか` | `与件根拠` | 一般論と混ぜず、事例企業の事実へ戻る | [与件文の読み方](/docs/second-stage/reading-case-materials) |

| --- | --- | --- |
| `限界利益` と `営業利益` | `限界利益` は変動費まで引いた利益、`営業利益` は固定費まで含めた利益 | [重要公式・計算式一覧](/docs/reference/important-formulas) |
| `与件根拠` と `一般論` | `与件根拠` は事例企業の事実、`一般論` は助言を支える知識 | [与件文の読み方](/docs/second-stage/reading-case-materials) |

| `価格転嫁` と `値上げ` | `価格転嫁` はコスト上昇分の反映、`値上げ` はより広い価格戦略 | [白書の読み方と頻出テーマ](/docs/sme-management-and-policy/knowledge-sme-white-paper-reading) |

| `DX` と `IT化` | `DX` は事業変革、`IT化` は既存業務のデジタル化 | [IT戦略・BPR・DX](/docs/management-information-systems/knowledge-it-strategy-and-dx) |
| `CRM` と `販促` | `CRM` は顧客関係の継続管理、`販促` は接点を作る施策全般 | [サービスマーケティングとCRM](/docs/business-management-theory/knowledge-service-marketing-and-crm) |
| `BCP` と `防災マニュアル` | `BCP` は事業継続の優先順位と復旧手順まで含む | [BCPと災害復旧支援](/docs/sme-management-and-policy/policy-bcp-and-disaster-recovery-support) |

## 用語で迷ったとき

<Accordions type="single">
  <Accordion title="略語しか思い出せないときはどうするか">
    まず `略語から引く` で `正式名称` と `使う場面` を確認します。略語だけを覚えると、1次の知識整理と2次の答案で別の言葉に置き換わったときに止まりやすくなります。
  </Accordion>
  <Accordion title="意味は分かるが、どこで使うか曖昧なときはどうするか">
    まず `使う場面` の列を見て、1次の知識整理なのか、2次の答案処理なのかを切り分けます。意味だけで止めず、どの問題で使う言葉かを先に固定します。
  </Accordion>
  <Accordion title="1次と2次で使い方が違う言葉はどう読むか">
    `与件根拠` や `骨子` のように2次で色が強い語と、`限界利益` や `ROA` のように1次から入る語は分けて見ます。迷ったら、このページで入口を確認してから、対応する元ページへ戻ります。
  </Accordion>
  <Accordion title="似た言葉が混ざったときは何を比べるか">
    言い換えを増やす前に、`内部か外部か`、`本業の現金か投資後の現金か`、`コスト転嫁か価格戦略全体か` のように比較軸を一つに絞ります。比較軸が決まると戻り先も決めやすくなります。
  </Accordion>
</Accordions>

## 典型的なつまずき

- 用語の `定義` だけ覚えて、`どこで使うか` を結び付けない
- 略語だけ覚えて、`正式名称` と `使う場面` を結び付けない
- 同じ語が 1次と 2次で少し違う役割を持つことを見落とす
- 用語集だけ読んで終わり、元ページへ戻らない
- 日本語の言い換えだけ増えて、計算や設問処理へつながらない
