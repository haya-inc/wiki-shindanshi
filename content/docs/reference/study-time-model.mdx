## このページの役割

<Callout title="このページの役割">
    このページは、`どの時期に何へ時間を使うか` を整理するための付録です。万人向けの正解ではなく、
    `学習段階`、`得意不得意`、`直近の目標` に応じて配分を調整するための土台として使います。
</Callout>

## 学習のポイント

- まず `全体像を作る時間` と `解く時間` を分ける
- 次に `計算科目` と `暗記更新科目` の重さを分ける
- 2次対策は 1次知識と切り離さず、`設問処理` と `答案化` に時間を配る

## 段階別の時間配分モデル

| 段階 | 目的 | 週の時間配分の目安 | まず戻るページ |
| --- | --- | --- | --- |
| 導入期 | 7科目 / 4事例の地図を作る | `全体像 40% / 基礎知識 40% / 問題演習 20%` | [1次の7科目一覧](..)、[第2次試験](../second-stage) |
| 定着期 | 章ごとの論点を固める | `基礎知識 35% / 問題演習 45% / 復習 20%` | [過去問テーマ索引](./past-question-theme-index) |
| 直前期 | 捨て問判断と取り切る論点を固定する | `問題演習 55% / 復習 30% / 制度確認 15%` | [年度別出題傾向](./yearly-question-trends)、[受験ガイド](./exam-guide) |

## 背景別の調整例

| 背景 | 重点化すること | 時間を減らしすぎないこと |
| --- | --- | --- |
| 計算が苦手 | 財務・会計、事例Ⅳ、経済の図表問題 | 法務や政策の更新確認 |
| 暗記が苦手 | 用語集、比較表、更新論点の再確認 | 演習量の確保 |
| 社会人で時間が限られる | 章ハブ、要点整理、頻出演習に絞る | 直前期の制度確認 |
| 2次を見据えて進めたい | 企業経営理論、運営管理、財務・会計を厚くする | 1次の足切り回避科目 |

## 週次モデル

| 背景 | 平日の回し方 | 休日の回し方 | まず戻るページ |
| --- | --- | --- | --- |
| 働きながら 1次中心 | 平日は `章ハブ 30分 + 演習 30分` | 休日は `弱点科目 2時間 + 更新確認 30分` | [過去問テーマ索引](./past-question-theme-index) |
| 1次と2次を並行 | 平日は `1次知識 45分 + 2次骨子 30分` | 休日は `事例演習 2時間 + 1次補強 1時間` | [第2次試験](/docs/second-stage) |
| 直前期 | 平日は `頻出論点の確認 30分 + 演習 45分` | 休日は `年度差確認 30分 + 本番形式演習 2時間` | [年度別出題傾向](./yearly-question-trends) |

## 直前 2 週間の見方

| 時期 | 重点化すること | 残すべき確認 |
| --- | --- | --- |
| 試験 2 週間前 | 頻出論点の穴埋め、要点整理、間違い直し | 制度変更、持ち物、日程確認 |
| 試験 1 週間前 | 本番時間での演習、捨て問判断 | 更新論点の最終確認 |
| 前日 | 要約と付録だけを短く往復 | 新しい論点の追加学習をしない |

## 時間配分で迷ったとき

<Accordions type="single">
  <Accordion title="時間が足りないときは何を先に削るか">
    まず `読むだけの時間` を削り、章ハブ、要点整理、頻出演習に寄せます。制度確認や更新論点をゼロにすると直前で崩れやすいので、そこは細くても残します。
  </Accordion>
  <Accordion title="計算が苦手なときはどこを厚くするか">
    `財務・会計`、`事例Ⅳ`、`経済学・経済政策` の図表問題に時間を厚くし、用語確認は短い往復で回します。苦手だからといって計算だけに寄せすぎず、更新科目の確認も最低限は維持します。
  </Accordion>
  <Accordion title="2次を見据えるなら何を優先するか">
    `企業経営理論`、`運営管理`、`財務・会計` を厚めにしつつ、2次対策は `設問処理` と `答案化` に分けて時間を配ります。事例を読むだけで終わらせず、答案骨子まで書く前提で配分を考えます。
  </Accordion>
</Accordions>

## 典型的なつまずき

- 全科目を同じ密度で進めようとして失速する
- `読む時間` だけで満足し、`解く時間` を確保しない
- 更新論点の確認を後回しにして、直前に崩れる
- 2次対策を `事例を読むだけ` で済ませ、答案化の練習時間を取らない
