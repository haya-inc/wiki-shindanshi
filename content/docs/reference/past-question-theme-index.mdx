## このページの役割

このページは、`過去問で問われるテーマ` から戻る先をすぐ決めるための索引です。説明を読むページではなく、`科目を選ぶ → 表を見る → 戻る` ための道具として使います。

<Callout title="使い方の要点">
  問題文のテーマに近い科目を 1 つ選び、その表で `最初に読むページ`、`補助で読むページ`、`確認ページ` を決めます。年度差を見たいときだけ、最後に [年度別出題傾向](/docs/reference/yearly-question-trends) へ切り替えます。
</Callout>

## 迷ったときの分け方

| 問題文の見え方 | まず見る索引 | 代表キーワード |
| --- | --- | --- |
| 計算やグラフの処理が中心 | 経済学・経済政策、財務・会計 | 弾力性、IS-LM、CVP、NPV |
| 組織、人、顧客、販促の助言を考える | 企業経営理論、第2次試験 | SWOT、STP、権限移譲、販路 |
| 生産、在庫、品質、店舗運営を問う | 運営管理、第2次試験 | JIT、PERT、SKU、物流 |
| IT、データ、セキュリティ、DX を問う | 経営情報システム | SQL、ネットワーク、ISMS、クラウド |
| 法令、契約、知財、制度改正を問う | 経営法務、参照資料 | 会社法、特許、契約、個人情報 |
| 白書、定義、補助金、支援策を問う | 中小企業経営・政策 | 定義、白書、保証、補助金 |
| 与件読解、答案骨子、事例Ⅳの計算を問う | 第2次試験 | 与件文、設問型、経営分析、CF |

## 使い方

- 問題文のテーマに近い科目を 1 つ選び、その表へ飛びます。
- `最初に読むページ` で論点を確定し、必要なときだけ `補助で読むページ` を開きます。
- 最後に `確認ページ` で自力確認し、年度差が気になるときだけ [年度別出題傾向](/docs/reference/yearly-question-trends) を見ます。
