title: 参照資料
description: 制度・索引・付録
---

科目本文に固定しづらい `制度情報`、`索引`、`年度差`、`横断情報` を集めるセクションです。更新が必要な内容はここへ寄せ、学習本文と役割を分けます。

<Callout title="参照資料の使い方">
  ここは本文の代わりではなく、制度確認と横断参照のための面です。制度や年度差を確認したら、学習自体は各科目ページへ戻る前提で使います。
</Callout>

## 参照資料の役割

<Cards>
  <Card title="制度情報">
    日程、申込、手数料のように年度で動く情報を、本文から分離して管理します。
  </Card>
  <Card title="索引">
    テーマ名や年度差から、本文の章ページや事例ページへ戻る入口をまとめます。
  </Card>
  <Card title="横断付録">
    用語、公式、学習配分のように複数科目で引き直す情報を集約します。
  </Card>
</Cards>

## 参照資料の使い分け

<Tabs items={["制度確認", "逆引き", "横断付録"]}>
  <Tab value="制度確認">
    - 最新の制度、日程、申込、手数料を確認したいときは [受験ガイド](/docs/reference/exam-guide) を使います。
    - 年度更新が絡む情報は、本文より一次情報を優先して確認します。
  </Tab>
  <Tab value="逆引き">
    - 設問番号と設問型から戻りたいときは [第2次試験 設問索引](/docs/reference/second-stage-question-index) を起点にします。
    - テーマ名や過去問の切り口から戻りたいときは [過去問テーマ索引](/docs/reference/past-question-theme-index) を起点にします。
    - 年度差を見たいときは [年度別出題傾向](/docs/reference/yearly-question-trends) から該当章へ戻ります。
  </Tab>
  <Tab value="横断付録">
    - 用語の意味をまとめて見直すなら [重要用語集](/docs/reference/important-terms-glossary) を使います。
    - 計算式や学習配分を確認したいときは [重要公式・計算式一覧](/docs/reference/important-formulas) と [学習時間モデル](/docs/reference/study-time-model) を使います。
  </Tab>
</Tabs>

## 横断付録

<Cards>
  <Card title="重要用語集" href="/docs/reference/important-terms-glossary">
    科目横断で再利用する定義を、科目別索引と混同しやすい用語の両方から引けるように整理します。

## 運用方針

- 制度変更は参照資料へ、定義や解法は本文へ、横断で繰り返す知識は付録へ寄せます。
- 迷ったら、単独ページとして読ませたいか、辞書のように引かせたいかで本文と付録を分けます。

## よくある迷い

<Accordions type="single">
  <Accordion title="制度情報はどこに置くか">
    試験制度、申込、手数料、法令改正のように年度更新が前提の情報は、本文ではなく参照資料に寄せます。本文側には考え方と学習導線だけを残します。
  </Accordion>
  <Accordion title="年度差はどこで追うか">
    年度差そのものを見たいときは [年度別出題傾向](/docs/reference/yearly-question-trends) から確認し、具体的な学習は各章や事例ページへ戻ります。
  </Accordion>
  <Accordion title="第2次試験の設問番号から戻りたいときはどうするか">
    [第2次試験 設問索引](/docs/reference/second-stage-question-index) で `年度`、`問題番号`、`設問型`、`誤答パターン` を見て、対応する設問型ページか答案骨子ページへ戻ります。
  </Accordion>
  <Accordion title="本文と付録の境界はどう考えるか">
    定義、比較軸、解き方は本文に置き、科目横断で何度も引く内容だけを付録へ集約します。
  </Accordion>
</Accordions>
