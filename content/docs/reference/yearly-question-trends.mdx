このページは、`過去問テーマ索引` で論点を特定したあとに、`年度差` を読むための参照ページです。まずは `直近 5 年` を中心に、繰り返し出る論点と最近比重が上がる論点を切り分けます。

<Callout title="このページの役割">
  `論点そのもの` ではなく、`いつ比重が上がったか` をつかむためのページです。比較だけで終わらせず、必ず科目ページや事例ページへ戻ります。
</Callout>

## 使い方

<Steps>
  <Step>
    ### まず論点を特定する

    [過去問テーマ索引](/docs/reference/past-question-theme-index) で、何のテーマを追うかを先に決めます。
  </Step>
  <Step>
    ### 年度差を確認する

    このページで、`いつから比重が高いか` と `どこが最近の注意点か` を見ます。
  </Step>
  <Step>
    ### 必ず本文ページへ戻る

    法改正や白書更新が絡む論点は、科目ページの `更新確認メモ` とセットで使います。
  </Step>
</Steps>

## 年度別に見るときの注意点

<Cards>
  <Card title="安定論点">
    年度をまたいでも問われ方の型が大きく崩れにくい論点として扱います。
  </Card>
  <Card title="更新論点">
    古い年度の問題文と現行制度名がずれることがあるため、年度差を強く意識します。
  </Card>
  <Card title="白書と補助金">
    細かな数字や公募条件より、`何を示す資料か` と `何の目的の施策か` を優先して読みます。
  </Card>
</Cards>

## 第2次試験で年度差を見る観点

<Steps>
  <Step>
    ### 設問数と重さ

    `設問数` と `1問あたりの重さ` が変わると、時間配分の基準も動きます。
  </Step>
  <Step>
    ### 与件文の抽象度

    `根拠の拾いやすさ` が年度で変わるため、読み取り難度の差を先に見ます。
  </Step>
  <Step>
    ### 基本問題と変化球

    `基本問題` と `見慣れないテーマ` の比率を見ると、その年の難しさがつかみやすくなります。
  </Step>
  <Step>
    ### 問われる結論の型

    `助言内容` を問うのか、`狙い / 効果` を問うのかまで見ておくと、答案の組み方が安定します。
  </Step>
</Steps>

## 運用ルール

<Callout title="比較だけで終わらせない">
  年度別の比較は、まず `直近 5 年` を中心に行います。法改正が絡む科目は年度別ページだけで完結させず、必ず科目ページへ戻し、新しい年度の問題を見たら `繰り返し出るテーマ` と `直近で比重が上がるテーマ` のどちらかを先に判定します。
</Callout>

## 年度差を見るときの迷い

<Accordions type="single">
  <Accordion title="論点軸と年度軸はどう使い分けるか">
    まず `過去問テーマ索引` で論点を特定し、そのあとでこのページに戻って年度差を見ます。年度軸から入りすぎると、何の論点を追っているのかがぼやけやすいので、入口は論点軸を優先します。
  </Accordion>
  <Accordion title="安定論点と更新論点はどこで切り分けるか">
    図表、計算、基本理論のように毎年の型が大きく変わりにくいものは `安定論点` として扱い、法改正、白書、補助金、ガイドラインのように年度差が大きいものは `更新論点` として別に見ます。
  </Accordion>
  <Accordion title="年度差を見たあと、どこへ戻るべきか">
    年度差を確認したら、そのまま暗記を増やすのではなく、対応する科目ページや事例ページへ戻ります。このページは比較の入口であって、学習の本体ではありません。
  </Accordion>
</Accordions>
